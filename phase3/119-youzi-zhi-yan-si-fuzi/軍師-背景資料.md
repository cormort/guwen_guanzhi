# 〈有子之言似夫子〉背景資料

> **軍師分析 · 古文觀止卷三 · 禮記·檀弓上**

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《禮記·檀弓上》 |
| **選集** | 《古文觀止》卷三·周文類 |
| **體裁** | 語錄體／經典記言 |
| **時代** | 春秋末期（孔子去世後不久，約西元前 5 世紀初） |
| **核心事件** | 有子（有若）問曾子：「你聽過夫子說『喪欲速貧，死欲速朽』嗎？」曾子說聽過。有子認為這不像夫子說的話。後來問子游才知——孔子確曾說過這句話，但那是針對特定情境的反諷，而非普遍原則 |
| **關鍵人物** | 有子（有若，孔子晚年弟子，以相貌酷似孔子著稱）、曾子（曾參，以孝行聞名，孔子重要弟子）、子游（言偃，孔子弟子，文學科高徒） |
| **文本重點** | 這是一篇關於「文本解釋學」的範例——一句話放在不同的上下文中，意義可以完全相反。曾子只記住了孔子說了什麼，卻忘了孔子為什麼說；有子則敏銳地察覺到語境與字面之間的落差 |
| **思想譜系** | 出自《禮記·檀弓》篇。《禮記》是儒家禮學的經典彙編，「檀弓」是篇名，以首句「檀弓奔母之喪」得名，內容多為孔子與弟子及當時人物的喪禮對話。〈有子之言似夫子〉是其中極短卻極精彩的一則，堪稱中國最早的「語境論」宣言 |
| **一句話總結** | **理解聖人的話，不能只看字面，要看說話的對象、說話的場合、說話的意圖——「言各有當」是這篇文章留給後世最重要的解讀學原則。曾子記住了「說什麼」，有子記住了「為何說」，兩者的差距就是聖人弟子與普通記誦者的差距。** |

## 二、時代背景——孔子歿後，弟子們的傳承困境

### 2.1 孔子去世後的儒門情勢

孔子於魯哀公十六年（西元前 479 年）去世。身後留下一群弟子，各自整理、傳播、甚至「處理」孔子的遺言。孔子生前「述而不作」，他的話語散見於日常教學與論辯之中，弟子們各自記錄所聞——但問題在於：

- **孔子說話的場合不同**：對不同資質的學生，對不同的政治環境，孔子會說不同的話
- **弟子們的理解不同**：曾子忠厚老實，聽了就記住；子游聰明機敏，懂得分辨場合
- **口耳相傳的出入**：沒有錄音，沒有筆記本，一句話傳了幾手之後，可能已經失真

〈檀弓〉這則對話，正是這個「孔子話語傳承困境」的縮影。

### 2.2 有若的特殊地位

有若（有子）在孔子弟子中地位特殊。據《史記·仲尼弟子列傳》記載，有若「狀似孔子」——長得像孔子。孔子歿後，一群弟子因為思念老師，曾經想讓有若「升座」代理孔子，接受弟子們的禮拜（雖然後來因為有若答不出一個棘手的問題而作罷）。

> **軍師按**：有若長得像孔子這件事，在儒門傳承史上是一個極有意思的細節。弟子們思念老師到了一個程度，寧可找一個長得像的人來「替代」。但真正重要的是——有若不只是長得像，他對孔子思想的理解也比一般人深入。在這則對話中，正是他（而不是更負盛名的曾子）率先察覺到那句話的「不對勁」。

## 三、原文與結構

### 3.1 原文

> 有子問於曾子曰：「問喪於夫子乎？」
>
> 曰：「聞之矣：『喪欲速貧，死欲速朽。』」
>
> 有子曰：「是非君子之言也。」
>
> 曾子曰：「參也聞諸夫子也。」
>
> 有子又曰：「是非君子之言也。」
>
> 曾子曰：「參也與子游聞之。」
>
> 有子曰：「然。然則夫子有為言之也。」
>
> ——《禮記·檀弓上》

### 3.2 對話結構分析

| 回合 | 發言者 | 內容 | 功能 |
|------|--------|------|------|
| **第一回合** | 有子提問 → 曾子回答 | 有子問是否聽過夫子談喪事，曾子答：「喪欲速貧，死欲速朽」 | 揭示矛盾：曾子認為這是孔子的話 |
| **第二回合** | 有子質疑 × 2 | 有子兩次說：「這不是君子之言」 | 引出核心衝突——有子不相信孔子會說這種話 |
| **第三回合** | 曾子辯護 | 曾子強調自己是親耳從夫子那裡聽來的，而且是與子游一起聽的 | 加重矛盾——有子否定的是曾子親耳聽到的話 |
| **第四回合** | 有子讓步 | 有子說：「那麼，夫子一定是針對特定情境說的」 | 轉折——從全盤否定轉向「語境解釋」 |
| **第五回合（下文）** | 問子游 | 有子去找子游求證，子游確認：「有子說得對！夫子是在宋國看到司馬桓魋造石槨三年不成，感嘆奢華誤事，才這麼說的」 | 真相大白——語境決定意義 |

> **軍師按**：這篇對話的結構非常精巧，像一出三幕劇：
> - **第一幕**：有子提出問題（有問）
> - **第二幕**：矛盾展開——曾子引述孔子原話，有子否定其權威性（衝突）
> - **第三幕**：子游揭露語境，矛盾化解（解答）
>
> 如果沒有第三幕，這則對話就會變成「兩個弟子誰對誰錯」的懸案。子游的出現是解題的關鍵——他是唯一同時知道「孔子說了什麼」和「孔子為何而說」的人。

## 四、關鍵人物

### 4.1 有子（有若）—— 面相像孔子，腦子也像孔子

| 項目 | 內容 |
|------|------|
| **身份** | 孔子晚年弟子，魯國人，少孔子四十三歲 |
| **特長** | 對儒家思想的深層理解，善於推理和辨別 |
| **在這則對話中的角色** | 質疑者／語境覺察者 |
| **歷史評價** | 相貌酷似孔子，孔子歿後一度被推舉為「代理孔子」；《論語》中唯獨他與曾子被尊稱為「子」（有子、曾子），地位顯赫 |

**軍師評**：有子在這則對話中展示的能力，遠比「長得像孔子」重要——他擁有**語感**。當他聽到「喪欲速貧，死欲速朽」時，他的第一反應不是「這是老師說的話嗎？」，而是「這話聽起來不像老師會說的道理」。這種對言論風格與思想主旨的敏感度，是真正的理解者才具備的能力。曾子雖然更孝順、更忠實，但在「知言」這一點上，輸給了有子。

### 4.2 曾子（曾參）—— 忠實的記錄者，但不是最好的解釋者

| 項目 | 內容 |
|------|------|
| **身份** | 孔子晚年重要弟子，魯國人，以孝行聞名 |
| **特長** | 恪守禮法、篤行孝道，是孔子弟子中最嚴謹自持的人 |
| **在這則對話中的角色** | 被質疑者／忠實轉述者 |
| **歷史評價** | 《論語》中多次出現，孟子極力推崇；後世被尊為「宗聖」 |

**軍師評**：曾子的問題不在於他記錯了——他確實親耳聽孔子說過這句話。他的問題在於：他只記住了「說什麼」，卻沒有記住「為什麼說」。這在現代社會中是一個非常普遍的現象——我們常常引用權威的話來支持自己的論點，卻忽略了那些話原本是針對完全不同的情境說的。曾子是一個好學生，但作為「傳話者」，他需要補上「語境教育」這一課。

### 4.3 子游（言偃）—— 關鍵的解鈴人

| 項目 | 內容 |
|------|------|
| **身份** | 孔子晚年弟子，吳國人，孔子弟子中少數的南方人 |
| **特長** | 文學科（文獻學與禮儀學）的高徒，孔子曾讚許他「吾門有偃，吾道其南」 |
| **在這則對話中的角色** | 解答者／語境提供者 |
| **歷史評價** | 孔子弟子「十哲」之一；以精通禮儀著稱，對《禮記》的整理有重要貢獻 |

**軍師評**：子游是這則對話中的「關鍵證人」。他不是最早發言的人，但他的話一出來，所有矛盾瞬間化解。他的價值在於：他不僅記住了老師說了什麼，還記住了老師是在什麼情境下說的。這在資訊爆炸的現代社會尤其可貴——我們每天接收大量資訊，但能記得「來源」和「上下文」的人，才是真正理解資訊的人。

## 五、子游的解釋——解開謎團的兩則語境

### 5.1 第一則語境：在宋國，見司馬桓魋造石槨

> 有子問子游曰：「夫子有為言之乎？」
>
> 子游曰：「甚哉，有子之言似夫子也！昔者夫子居於宋，見桓司馬自為石槨，三年而不成。夫子曰：『若是其靡也，死不如速朽之愈也。』『死之欲速朽』，為桓司馬言之也。」

**語境還原**：孔子在宋國時，看到司馬桓魋給自己打造石棺（石槨），花了三年還沒做完。孔子感嘆：「奢侈到這種地步，死了還不如快點腐朽算了！」

孔子的意思很明確：他不是在提倡「死了快點爛掉」，而是在**反對奢侈浪費的厚葬之風**。桓魋這種人活著的時候貪婪，死了還要折騰一堆石頭——孔子說的反話，是一句「針對桓魋的諷刺」，而非普遍的「死之哲學」。

### 5.2 第二則語境：在衛國，見失位大夫無力簡葬

> 「南宮敬叔反，必載寶而朝。夫子曰：『若是其貨也，喪不如速貧之愈也。』『喪之欲速貧』，為敬叔言之也。」

**語境還原**：南宮敬叔（衛國大夫）被逐出衛國後回來，每次上朝都帶著財寶去賄賂權貴。孔子感嘆：「貪婪到這種地步，失去官位還不如快點窮了算了！」

孔子的意思同樣很清楚：他不是在提倡「當官要快點窮」，而是在**反對為了復職而不惜行賄的無恥行為**。南宮敬叔這種人，丟了官位已經夠丟臉了，還要帶著財寶到處送禮——孔子說的反話，是一句「針對敬叔的諷刺」。

### 5.3 兩則語境的對比

| 項目 | 桓司馬（宋國） | 南宮敬叔（衛國） |
|------|---------------|-----------------|
| **行為** | 自製石槨，三年不成 | 失位後載寶朝見，試圖復職 |
| **孔子的批評** | 死欲速朽 | 喪欲速貧 |
| **批評的核心** | 反對奢侈厚葬 | 反對貪婪行賄 |
| **修辭手法** | 反諷（說反話） | 反諷（說反話） |
| **誤解的後果** | 以為孔子反對喪禮 | 以為孔子主張早點失業 |

## 六、軍師核心分析——「言各有當」的解讀學原則

### 6.1 為什麼有子能看出問題？

有子聽到「喪欲速貧，死欲速朽」時，他的直覺告訴他：「這不像夫子會說的話。」

這直覺從哪裡來？——來自對孔子整體思想的把握。

孔子一生的核心關懷是「禮」。他主張「克己復禮為仁」，主張「慎終追遠」，主張「生事之以禮，死葬之以禮，祭之以禮」。這樣一個以禮為核心的思想家，怎麼可能主張「喪欲速貧、死欲速朽」？

有子的推理過程（雖然文中沒有明說，但可以還原）大概是：

1. **孔子主張厚葬嗎？** 不，孔子反對厚葬（他批評過顏回的厚葬）
2. **孔子主張薄葬嗎？** 也不完全，他認為葬禮應該符合禮法，不贊成簡陋到失禮
3. **那「速貧」「速朽」是什麼意思？** ——一定是在某種極端情境下的反話

這就是「以意逆志」的解讀方法——不是逐字逐句地死摳，而是從整體思想出發去理解局部言論。

### 6.2 曾子的錯誤是什麼？

曾子的錯誤不是「記憶錯誤」，而是 **「詮釋錯誤」**。他記對了（孔子確實說過這句話），但他用錯了地方——他把一句「針對特定對象的諷刺話」，當成了孔子對喪禮的「普遍教導」。

這在現代學術中有一個專有名詞：**脫離語境的引用（quote mining）**。把一句話從原來的上下文中抽出來，放到另一個完全不一樣的脈絡中，這句話的意義就會被扭曲。

### 6.3 子游的真正貢獻

子游的貢獻不只是「提供了解答」——他提供了解答的**方法論**。他告訴我們：

- 要理解一句話，你得知道：**誰說的、對誰說的、在哪裡說的、為什麼說**
- 同樣一句話，在不同的情境下，意義可以完全不同
- 孔子不是一個「教條主義者」，他的每個教導都是針對具體問題的「處方」，不是放諸四海而皆準的「定律」

> **軍師按**：子游的解釋模式，其實就是現代文本解釋學（Hermeneutics）的核心方法——**每一個文本都有其歷史語境（Sitz im Leben），脫離語境的解讀是危險的**。難怪美國漢學家赫伯特·芬格萊特（Herbert Fingarette）說，儒家典籍的解讀，最怕的就是「把孔子變成一個說格言的人」。

## 七、軍師總論

### 7.1 「似夫子」的三層意義

標題「有子之言似夫子」——子游讚嘆有子的話「像夫子」。這個「似」有三層意義：

| 層次 | 意義 | 解析 |
|------|------|------|
| **第一層** | 語氣像 | 有子說話的方式、語氣像孔子——這是最表面的一層 |
| **第二層** | 思維像 | 有子不是死記硬背，而是從孔子的整體思想出發去推理——這才是真正的「似」 |
| **第三層** | 方法像 | 孔子也經常根據不同的對象說不同的話，有子在這一點上學到了精髓——他是「活的繼承者」 |

### 7.2 這篇文章為什麼重要？

〈有子之言似夫子〉只有短短幾百字，但它所蘊含的思想方法論，在《古文觀止》全書中都是頂尖的：

1. **它提出了「語境決定意義」的原則**——這比西方現代語言哲學家奧斯汀（J.L. Austin）的「言語行為理論」早了兩千多年
2. **它示範了「以意逆志」的解讀方法**——孟子提出的「不以文害辭，不以辭害志」的解讀原則，在這篇文章中得到了完美的實踐
3. **它警告了「語錄式思維」的危險**——把名人的話當作放諸四海而皆準的金科玉律，是一種危險的思維懶惰

### 7.3 現代啟示

在網路時代，脫離語境的引用（quote mining）比比皆是。有人把孔子的一句話剪出來，配上一個完全無關的場景，就變成了「孔子說過……」。〈有子之言似夫子〉提醒我們：

- **看到一句話，先問：誰說的？對誰說的？為什麼說？**
- **別把反諷當成教條，別把針對特定情境的批評當成普遍原則**
- **真正理解一個人，不是記住他的話，而是理解他為什麼說那些話**

> **軍師總評**：〈有子之言似夫子〉是《古文觀止》卷三中一則容易被忽略的短文——因為它短、因為它不涉及重大歷史事件、因為它沒有華麗的辭藻。但它是全書中最具「思想密度」的文章之一。它教給我們的不是某個歷史知識，而是一種思維方式：**永遠不要只看一句話的表面，要看它的背後——站著什麼人、對著什麼事、戳著什麼病。** 有子做到了這一點，所以他「似夫子」。曾子沒有做到，所以他只是「記夫子」。而我們讀完這篇文章，能不能也學會「似夫子」——不只是記住學過的知識，而是學會用這些知識去判斷新遇到的情況？這才是這篇短文留給我們最深的功課。

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

- 《古文觀止》卷四·〈曾子易簀〉——曾子臨終的記事，展現曾子性格的另一面：守禮至死，與本文共讀可看見曾子的全貌
- 《古文觀止》卷四·〈子產不毀鄉校〉——同樣關於「如何理解他人的話」，子產主張讓人說話，與本文形成互文
- 《古文觀止》卷三·〈曾子不受賜〉——曾子的廉潔與堅守，可與本文對照

### 8.2 經典延伸

- 《論語·先進》：「子貢問：『師與商也孰賢？』子曰：『師也過，商也不及。』」——孔子對不同學生的評價本身就體現了「因材施教」的原則
- 《孟子·萬章上》：「故說詩者，不以文害辭，不以辭害志；以意逆志，是為得之。」——孟子提出的解讀方法，與有子的做法完全一致
- 《禮記·檀弓》全篇——包含大量孔子與弟子的對話，是理解孔子思想上課堂對話的重要材料

### 8.3 現代延伸

- Quentin Skinner 的「言語行為理論」（Speech Act Theory）——主張理解文本必須還原作者意圖與社會語境
- 徐復觀《中國思想史論集》——對中國經典解讀方法論的深刻討論
- 余英時《論天人之際》——對於如何理解古代思想家的「言論」與「行動」之間的關係，有精闢的分析

---

## 九、一句話總結（再錄）

> **理解聖人的話，不能只看字面，要看說話的對象、說話的場合、說話的意圖——「言各有當」是這篇文章留給後世最重要的解讀學原則。曾子記住了「說什麼」，有子記住了「為何說」，兩者的差距就是聖人弟子與普通記誦者的差距。**
