## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《韓昌黎集》——韓愈詩文合集，由其門人李漢編纂，共四十卷；〈獲麟解〉為韓愈雜說中的一篇短文 |
| **選集** | 《古文觀止》卷七·唐文——韓愈入選《古文觀止》共四篇（〈原道〉〈師說〉〈進學解〉〈送李愿歸盤谷序〉），〈獲麟解〉不在其中——此處作為延伸研讀篇目 |
| **體裁** | 雜說／論說文——「解」體——以解釋疑難、辨析義理為主——韓愈借麒麟之「惑」——來解人才不被識之「惑」 |
| **時代** | 中唐（約西元 9 世紀初）——韓愈身處安史之亂後的衰世——儒學不振——佛道昌熾——有識之士屢遭排擠——韓愈本人即因直言而屢遭貶謫 |
| **核心命題** | 麒麟為什麼不受歡迎？——「麟之所以為麟者，以德不以形」——真正的價值不在外表——而在本質——但世人卻只看外表——所以無法辨識真正的麒麟 |
| **關鍵人物** | 韓愈（唐代古文運動領袖）、孔子（「西狩獲麟」典故——魯哀公十四年獲麟——孔子絕筆《春秋》） |
| **文本重點** | 韓愈借用孔子「西狩獲麟」的典故——麒麟是祥瑞之獸——但沒有人認識牠——因為牠的外形不屬於任何已知動物分類——韓愈由此引申：真正有才華的人就像麒麟一樣——他們不符合世俗的標準——所以不被社會認可——但問題是——麒麟之所以是麒麟——靠的是牠的內在本質（德）——而不是外在形式（形）——然而——沒有外在形式可供辨認——人們又怎麼知道牠是麒麟還是怪物？ |
| **一段話總結** | **韓愈用麒麟的寓言說出了所有天才的困境——如果你和別人不一樣，別人不會認為你是天才——只會認為你是怪物。麒麟是祥瑞還是怪物——不在於麒麟本身——而在於看你的人——有沒有能力——看出麒麟。韓愈寫這篇文章——不是在談動物——而是在說他自己——也在說每一個——不被時代理解的——人。** |

## 二、時代背景——中唐的麒麟困境

### 2.1 中唐：一個「認不出麒麟」的時代

韓愈〈獲麟解〉的寫作背景——與他個人仕途的坎坷——以及中唐社會的普遍困局——息息相關：

| 面向 | 說明 |
|------|------|
| **政治** | 安史之亂後——中央權威大減——藩鎮割據——宦官專權——黨爭激烈（牛李黨爭）——人才選拔——完全被權力鬥爭綁架——真正有才能的人——往往得不到重用——甚至被排擠、打壓 |
| **韓愈的個人處境** | 韓愈本人——就是一個典型的「不被認識的麒麟」——他三歲喪父——刻苦讀書——貞元八年（792 年）進士及第——此後仕途坎坷——屢次上書自薦——卻屢遭冷遇——先後擔任四門博士、監察御史等低微官職——更因上〈論天旱人飢狀〉得罪權貴——被貶連州陽山令——後來又因〈諫迎佛骨表〉——被貶潮州——他的一生——就是不斷被「誤認為怪物」的一生 |
| **社會—思想** | 中唐社會——價值觀混亂——佛道盛行——儒學衰微——社會缺乏統一的「識人標準」——世人——只能根據外表、出身、關係——來判斷一個人的價值——真正有德行、有才華的人——反而因為——不符合主流標準——而被邊緣化 |
| **文學中的反映** | 〈獲麟解〉——就是韓愈對這個時代——最深刻的「自畫像」——他把自己——比作不被世人認識的麒麟——他寫的不是「麒麟」——他寫的是自己的命運——「麟——你們不認識它——但它是祥瑞——你們認識的——未必是真的——這就是我的困境」 |

> **軍師按**：中唐的韓愈——是中國歷史上——最偉大的「孤獨者」之一。他推動古文運動——反對流行文風——他排斥佛老——對抗主流信仰——他提倡儒學道統——試圖挽回衰敗的思想——每一件事——他都在「逆流而上」——他寫〈獲麟解〉——不是為了討論麒麟——而是為了告訴這個世界——「你們不認識我——不是因為我不值得認識——而是因為你們——沒有能力認識我——我是一隻麒麟——不是一隻怪物——你們——才是問題。」

### 2.2 「西狩獲麟」——孔子與麒麟的千年因緣

| 面向 | 說明 |
|------|------|
| **歷史典故** | 「西狩獲麟」——出自《春秋·哀公十四年》——「春——西狩獲麟」——五個字——是《春秋》的最後一條記載——據說——孔子得知麒麟被獵獲——大哭——「吾道窮矣！」——從此絕筆——不再修《春秋》——因為——麒麟是祥瑞——只應在太平盛世出現——卻在亂世被獵殺——這是天意——告訴孔子——他的「道」——已經無法推行 |
| **麒麟的文化意義** | 在中國傳統中——麒麟——與龍、鳳、龜——並稱「四靈」——是「仁獸」——牠「不履生蟲——不折生草」——性情溫和——只出現在聖王在位——或天下太平的時候——是一種「政治祥瑞」——孔子以「獲麟」作為《春秋》的終點——寓意深刻——「祥瑞被殺」——象徵「盛世不再——大道不行」 |
| **韓愈對典故的運用** | 韓愈在〈獲麟解〉中——沒有直接引用「西狩獲麟」的完整故事——但他抓住了孔子最核心的困惑——「麟——之所以為麟——以德不以形」——牠不是靠長相——來證明自己是祥瑞的——那——沒有長相——人們怎麼知道牠是祥瑞？——韓愈把孔子的「絕筆之痛」——轉化為一個永恆的哲學問題——「如果真正有價值的東西——沒有標準的外形——你怎麼知道——它是有價值的？」 |

> **軍師按**：孔子和韓愈——跨越了一千多年的時間——被同一隻麒麟——聯繫在一起。孔子看到麒麟被獵——哭了——因為他覺得——天意已絕——他的道——無法實現。韓愈寫麒麟不被認識——笑了——帶著苦笑——因為他知道——他自己——就是那隻——在亂世中被誤認為怪物的——麒麟。哭——和笑——相隔千年——但答案——是一樣的——「吾道窮矣——但吾道不變。」

## 三、文本結構——從「認識麒麟」到「認識自己」

### 3.1 全文結構一覽

〈獲麟解〉全文極短——約一百五十餘字——但邏輯層次極為縝密——可分為三個層次：

| 層次 | 核心內容 | 情感基調 |
|------|---------|---------|
| **第一層：麒麟是什麼？** | 麒麟是祥瑞——但牠的外形不屬於任何已知動物——你不能用「像鹿像牛」來定義牠——牠就是牠自己——「麟之為麟——以德不以形」 | 冷靜——定義——建立命題 |
| **第二層：為什麼沒人認識麒麟？** | 因為沒有外形標準——世人無法辨認——如果牠永遠不出現——大家還能說「麒麟是祥瑞」——一旦牠真的出現了——大家反而說「這是什麼怪物？」——這是一個荒謬的悖論 | 轉折——從定義轉向批判——語調逐漸尖銳 |
| **第三層：到底是誰的問題？** | 不是麒麟的問題——是「識者」的問題——「雖有麟——不可知其為麟也」——如果一個時代——沒有人能認出麒麟——那不是麒麟的錯——是那個時代——太差了——「麟——不祥之物也——雖然——麟之出——必有聖人在乎位」 | 結論——帶著深深的無奈——但骨子裡——是對時代的無情嘲諷 |

### 3.2 開篇——「麟之所以為麟者，以德不以形」

> 「麟之為靈，昭昭也。詠於《詩》，書於《春秋》，雜出於傳記百家之書。雖婦人小子，皆知其為祥也。」

— 翻譯：麒麟是靈獸——這是清清楚楚的——《詩經》歌詠過牠——《春秋》記載過牠——各種傳記和諸子百家的書中——也都提到過牠——就算是婦女和小孩子——都知道麒麟是祥瑞。

> 「然麟之為物，不畜於家，不恆有於天下。其為形也不類，非若馬牛犬豕豺狼麋鹿然。然則雖有麟，不可知其為麟也。」

— 翻譯：但——麒麟這種動物——不養在家裡——也不是常常出現在世上——牠的外形——不屬於任何類別——不像馬、牛、狗、豬、豺狼、麋鹿——所以——就算有一隻麒麟在面前——你也不知道——牠就是麒麟。

| 層次 | 分析 |
|------|------|
| **「麟之為靈——昭昭也」** | 韓愈先做了一個「鋪墊」——大家都說麒麟是祥瑞——每個人都知道——這是「常識」——但——問題來了 |
| **「然……」** | 一個轉折——把前面的「常識」——徹底推翻——大家都知道麒麟是祥瑞——但——沒有人見過麒麟——也沒有人知道麒麟長什麼樣子——「常識」——變成了「空洞的知識」——大家都知道麒麟很厲害——但麒麟真的來了——沒有人認得出來 |
| **「其為形也不類」** | 這是全篇的關鍵——「不類」——就是不屬於任何類別——韓愈用了一個非常精準的詞——麒麟——沒有「類別」——馬可以歸類為馬——牛可以歸類為牛——但麒麟——無法歸類——牠就是——「除外」的存在 |

> **軍師按**：開篇這一段——表面上是「客觀描述」——但韓愈的筆——已經在笑了——「你們都說麟是祥瑞——但你們從來沒見過——那——你們怎麼知道你們說的『祥瑞』——就是真正的祥瑞？」——他問的——不是麒麟——他問的是——「你們——到底有沒有能力——分辨真正的價值？」

### 3.3 結尾——「麟，不祥之物也」

> 「麟之為靈，昭昭也。……雖然，麟之出，必有聖人在乎位。麟為聖人出也。聖人者，必知麟。麟之果不為不祥也。」

— 翻譯：麒麟之為靈獸——是清清楚楚的……雖然如此——麒麟的出現——一定是因為有聖人在位——麒麟是為了聖人而出現的——聖人——一定認識麒麟——所以——麒麟——終究不能算是——不祥之物。

| 層次 | 分析 |
|------|------|
| **表面矛盾** | 韓愈說——麒麟 ——「不可知其為麟」——所以——「麟——不祥之物也」——但緊接著——他又說——「雖然——麟之出——必有聖人在乎位」——「麟之果不為不祥也」——自問自答——自相矛盾——但這種「矛盾」——正是韓愈的深意 |
| **真正的結論** | 韓愈真正想說的是——麒麟是不是祥瑞——不是麒麟自己決定的——是看「有沒有聖人」——如果有聖人——麒麟就是祥瑞——如果沒有聖人——麒麟出現也沒人認得——那牠——就「等於」不祥之物——但——這不是麒麟的錯——是那個時代——沒有聖人——的錯 |
| **韓愈的自況** | 韓愈把「麒麟的存在」——與「聖人的存在」——綁定在一起——這是他對自己命運的終極回答——「我不是怪物——我是麒麟——但這個時代——沒有聖人——所以——沒有人認識我——但——沒有關係——終究會有的（或者——至少——有像我韓愈這樣的人——認識自己——就夠了）」 |

> **軍師按**：結尾是全文最精彩的部分——韓愈繞了一大圈——其實只說了一句話——「是我不祥——還是你們不幸？」——他不敢明說——但他用「麟之出——必有聖人在乎位」——暗示了——「我韓愈——是一隻麒麟——而這個時代——沒有聖人」——這句話——才是〈獲麟解〉——真正的——殺招。

## 四、關鍵人物

### 4.1 韓愈——被時代誤認的「麒麟」

韓愈（768–824 年）——字退之——唐代文學家、思想家、政治家——古文運動領袖——「唐宋八大家」之首——蘇軾稱他「文起八代之衰——道濟天下之溺」——但他的一生——就是一篇活生生的〈獲麟解〉：

| 面向 | 內容 |
|------|------|
| **不被理解的先驅** | 韓愈提倡「古文」——反對駢文——提倡「儒學」——反對佛老——這些主張——在他所處的時代——都是「少數派」——他是那個時代——最孤獨的聲音——他的〈原道〉——沒有立刻改變世界——他的〈師說〉——被嘲笑為「狂人」——他的一生——都在被「誤認」——正如他筆下的麒麟——「不可知其為麟也」 |
| **屢遭貶謫的命運** | 韓愈因為上書言事——被貶陽山——因為〈諫迎佛骨表〉——被貶潮州——「一封朝奏九重天——夕貶潮陽路八千」——這不是一次性的挫折——這是他的命運——他每一次「出現」——都被人當作「怪物」——正如麒麟——一出現——就被人獵殺 |
| **道統的自我定位** | 韓愈在〈原道〉中自居為「道統」的繼承人——「軻之死——不得其傳焉」——這句話的意思是——「孟子之後——就沒有人了——我是第一個——重新接上道統的人」——這是一種極其大膽的自我定位——正如麒麟自認是祥瑞——但——問題是——沒人認識牠——也沒人認識他——韓愈的自信和孤獨——是同一件事的兩個側面 |

**軍師評**：韓愈——是中國歷史上——最知道自己「是誰」的人——他知道自己是麒麟——他也知道——人們會把他當作怪物——但他不在乎——他照樣寫他的文章——說他的道理——做他認為對的事——因為——「麟之所以為麟者——以德不以形」——他不是為了被人認識——才成為麒麟的——他成為麒麟——是因為——他本來就是。

### 4.2 孔子——第一個為了麒麟哭泣的人

| 項目 | 內容 |
|------|------|
| **「西狩獲麟」與《春秋》絕筆** | 魯哀公十四年（西元前 481 年）——魯國人在西郊狩獵——捕獲了一隻麒麟——孔子——看到被獵殺的麟——大哭——「吾道窮矣！」——從此絕筆——不再修《春秋》——因為——麒麟是祥瑞——應在太平盛世出現——卻在亂世中——被獵人殺死——這是「天意」——告訴孔子——他的政治理想——無法實現了 |
| **麒麟與孔子的象徵關係** | 在中國文化中——孔子——與麒麟——有極深的象徵關聯——據說——孔子出生前——有「麟吐玉書」的異象——孔子去世前——有「西狩獲麟」的凶兆——麒麟——彷彿是孔子的「化身」——麟的死——象徵了「聖人不出——天下無道」 |
| **韓愈筆下的孔子** | 韓愈在〈獲麟解〉中——雖然沒有直接提到孔子——但「麟之出——必有聖人在乎位」這句話——處處指向孔子——聖人認識麒麟——因為聖人自己——就是一隻麒麟——韓愈說「聖人者——必知麟」——這句話的潛台詞是——「我——韓愈——以聖人為榜樣——我也認識麒麟——因為——我就是那隻麒麟」 |

**軍師評**：孔子和韓愈——相隔約一千三百年——但他們面臨的問題——一模一樣——「我是一個有德之人——但這個時代——不認識我。」孔子的選擇——是哭——然後絕筆——韓愈的選擇——是寫一篇文章——用文字——為自己辯護——也為所有——不被時代認識的人——辯護。一個哭——一個寫——但他們都在說——同一句話——「不是我錯了——是這個時代錯了。」

## 五、核心分析——「以德不以形」的哲學命題

### 5.1 為什麼「德」比「形」重要？

韓愈在〈獲麟解〉中——提出了一個極其深刻的命題——「麟之所以為麟者——以德不以形」——這句話——可以說是整篇文章——乃至整個韓愈思想的——核心：

| 面向 | 分析 |
|------|------|
| **「德」是什麼？** | 韓愈在〈原道〉中定義了「仁」和「義」——「博愛之謂仁——行而宜之之謂義」——「德」——就是「足乎己無待於外」——內心充實——不需要向外尋求——麒麟的「德」——就是牠的「仁」——牠不傷害生靈——不踐踏草木——是「仁獸」——所以——牠是祥瑞——不是因為牠長得好看——是因為牠的「本性」——是「仁」的 |
| **「形」為什麼不可靠？** | 外在形式——可以偽裝——可以模仿——一隻普通的鹿——長得再奇怪——也不會成為麒麟——同樣——一個人——就算穿得再好——出身再高貴——如果內心沒有「德」——也只是徒有其表——韓愈用這個觀點——狠狠地諷刺了那些「以貌取人」的時代——你們只看外表——所以——你們永遠——認不出真正的麒麟 |
| **對「以德不以形」的現實意義** | 韓愈生活在一個——「以形取人」的時代——科舉考試——要看門第——在官場——要看關係——社會——只看你的「外形」——你的出身、你的官位、你的財富——而沒有人看你的「內在」——你的品德、你的才華、你的思想——韓愈的〈獲麟解〉——就是對這種「以形取人」的社會——最深刻、最優雅的——抗議 |

### 5.2 一個永恆的悖論——如果沒有「形」——「德」如何被看見？

韓愈的〈獲麟解〉——最精彩的地方——在於——他提出了一個——無法完全解決的——悖論：

> 「麟之所以為麟者，以德不以形。若麟之出不待聖人，則謂之不祥也亦宜。」

— 如果麒麟的出現——不是在聖人在位的時候——那說牠是不祥之物——也不算冤枉。

| 面向 | 分析 |
|------|------|
| **悖論的核心** | 如果麒麟——只靠「德」——不靠「形」——來證明自己——那麼——在沒有聖人的時代——世人——沒有辦法——根據「形」來認識麒麟——就必然——把牠誤認為——怪物——而麒麟——如果不被認識——牠的「德」——就無法「顯現」——那牠——在實際效果上——就和「不祥之物」——沒有區別——這是一個「認識論的陷阱」——你——無法證明——你是一隻麒麟——除非——有人——有能力認識你 |
| **韓愈的「解法」** | 韓愈的「解法」——不是「降低麒麟的標準」——而是「等待聖人」——或者——更準確地說——「成為聖人」——因為——在韓愈的思想中——聖人——是可以通過學習——來達到的——「聖人者——必知麟」——如果你——像聖人一樣——努力修養自己的德行——你——就能認出麒麟——也能——成為麒麟 |
| **這不是一個「完美」的答案** | 韓愈沒有解決這個悖論——因為——它本來就無法「解決」——這是人類認識論的——永恆困境——「真正有價值的東西——往往無法被普遍標準衡量」——韓愈的偉大——在於——他看到了這個困境——並且——用最優雅的語言——把這個困境——說了出來——而不是假裝——它不存在 |

> **軍師按**：〈獲麟解〉——是韓愈所有文章中最「哲學」的一篇——因為它問了一個——沒有人能完全回答的問題——「如果我是對的——但全世界都說我錯了——那——到底是我錯了——還是全世界錯了？」——韓愈的回答是——「都沒錯——只是——沒有人——擁有——辨別真理的眼睛。」——而這——比說「我是對的——你們都是錯的」——要深刻得多——因為——它承認了——「認出真理」——本身——就是一門——需要修養的學問。

## 六、軍師核心分析——「所有天才的困境」

### 6.1 獲麟解 vs. 原道——韓愈的「柔」與「剛」

把〈獲麟解〉和〈原道〉放在一起——可以看到韓愈的兩面：

| 面向 | 〈原道〉 | 〈獲麟解〉 |
|------|---------|-----------|
| **主題** | 什麼是正確的「道」？——我告訴你 | 什麼是真正的「麟」？——你們猜 |
| **態度** | 斬釘截鐵——「人其人——火其書——廬其居」 | 迂迴反覆——「不可知其為麟也」——「然則雖有麟——不可知其為麟也」 |
| **語氣** | 戰鬥——強硬——不容置疑 | 無奈——自嘲——帶著苦笑 |
| **目標** | 消滅錯誤——建立真理 | 讓人認識——真正的價值 |
| **韓愈的角色** | 戰士——拿著刀——衝鋒陷陣 | 哲人——坐在角落——對著牆壁說話 |

**軍師評**：同一個韓愈——在〈原道〉中——是「戰鬥姿態」——他要改造世界——在〈獲麟解〉中——是「沉思姿態」——他在理解世界——為什麼——我在〈原道〉中是對的——卻不被接受？——兩篇文章——放在一起——才是完整的韓愈——一個——在戰鬥中——思考——在思考中——戰鬥——的中國知識分子。

### 6.2 〈獲麟解〉的現代意義——每個時代都有自己的麒麟

〈獲麟解〉——雖然寫於一千多年前——但它的主題——在今天——依然極有共鳴：

| 面向 | 說明 |
|------|------|
| **不被理解的創作者** | 每一個時代——都有「不被理解」的藝術家、作家、思想家——他們的風格——不屬於任何「類別」——他們的作品——賣不出去——不被看好——被嘲笑——但——時間證明了——他們是「麒麟」——梵谷生前——只賣出一幅畫——卡夫卡——要求朋友燒掉他的手稿——哥白尼——被教會審判——他們——都是「不被認識的麒麟」——而韓愈——在他們之前的八百年——就已經——替他們說話了 |
| **「以德不以形」的現代解讀** | 在社交媒體時代——我們——比任何時代——都更「以形取人」——外表、粉絲數、點讚數、頭銜——這些「形式」——決定了誰值得被關注——而真正有思想、有品德的人——往往因為——沒有華麗的「包裝」——被淹沒在資訊的洪流中——韓愈的「以德不以形」——在今天——是一種「反社交媒體」的哲學——它提醒我們——真正的價值——不在於「看起來怎麼樣」——而在於——「本質是什麼」 |
| **「聖人者——必知麟」——對識人者的考驗** | 韓愈說——「聖人者——必知麟」——這句話——把責任——從「被認識的人」——轉移到了「認識的人」身上——如果你——認不出真正的價值——那是因為——你自己的修養——還不夠——你還沒有——成為「聖人」——這是一個極其嚴格的「自我要求」——它要求每一個人——都努力提升自己的「識別能力」——不要——只會——以貌取人 |

> **軍師按**：〈獲麟解〉最動人的地方——不是韓愈為自己辯護——而是他為所有「不被認識的人」——建立了一個尊嚴——「如果你不被認識——不是因為你不值得——而是因為——這個時代——還沒有聖人——而你——是一隻——屬於未來的麒麟。」這——是給所有孤獨者的——一封——來自一千二百年前的情書。

## 七、一句話總結（再錄）

> **韓愈用麒麟的寓言說出了所有天才的困境——如果你和別人不一樣，別人不會認為你是天才——只會認為你是怪物。麒麟是祥瑞還是怪物——不在於麒麟本身——而在於看你的人——有沒有能力——看出麒麟。韓愈寫這篇文章——不是在談動物——而是在說他自己——也在說每一個——不被時代理解的——人。而〈獲麟解〉——用短短一百五十餘字——告訴了我們一個——比千年還長的道理——「麟之所以為麟者——以德不以形」——你不是怪物——你只是——生錯了時代——但——沒有關係——因為——真正有德的人——終究——會被認識——不是被這個時代認識——就是被——下一個時代認識。而韓愈——用他的一生——證明了——一個「不被認識的麒麟」——最終——會成為——那個——讓後人——認識——什麼是麒麟的人。因為——他一早就知道——他——不是怪物。**

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷七·〈原道〉（韓愈） | 韓愈的「戰鬥姿態」與〈獲麟解〉的「沉思姿態」——構成韓愈思想的兩個側面——一個是對外的攻擊——一個是對內的反思——兩者合讀——才能看到完整的韓愈 |
| 卷八·〈師說〉（韓愈） | 「師者——所以傳道授業解惑也」——韓愈為「師道」正名——正如他為「麒麟」正名——都在為「不被理解的東西」——爭取一個——應有的位置 |
| 卷八·〈進學解〉（韓愈） | 韓愈以自嘲的筆法——寫自己「學有所成——卻不被重用」——與〈獲麟解〉的「不被認識」——異曲同工——都是韓愈對自己命運的——自我解嘲 |
| 卷六·〈報任安書〉（司馬遷） | 司馬遷說「人固有一死——或重於泰山——或輕於鴻毛」——和韓愈說「麟之所以為麟者——以德不以形」——都在討論一個核心問題——「人——如何證明自己的價值？」 |
| 卷九·〈愚溪詩序〉（柳宗元） | 柳宗元說自己「以愚觸罪」——把一條溪水命名為「愚溪」——和韓愈的「獲麟解」——都是「自嘲」——但柳宗元的「愚」——是對自己的貶低——韓愈的「麟」——是對自己的肯定——一個是「自傷」——一個是「自傲」 |

### 8.2 經典延伸

- 《韓昌黎集》（韓愈著，唐·李漢編）——韓愈詩文集——〈獲麟解〉收錄於集中——為韓愈雜說中的代表作
- 《春秋·哀公十四年》（孔子編）——「西狩獲麟」的出處——全文僅五字——卻是中國文學史上——最意味深長的結尾之一
- 《史記·孔子世家》（司馬遷）——司馬遷詳細記載了孔子與「西狩獲麟」的故事——以及孔子「絕筆」的經過——是了解這個典故最權威的史料
- 《公羊傳·哀公十四年》——對「西狩獲麟」的解釋——公羊學派認為——「麟——仁獸也——有王者則至——無王者則不至」——韓愈的〈獲麟解〉——明顯受到公羊學的影響
- 《荀子·勸學》（荀子）——「青——取之於藍——而青於藍」——荀子討論「學習」的意義——而韓愈的師道和麒麟——都在討論——「人——如何超越自己的『出身』——成為一個真正有價值的人」

### 8.3 現代延伸

- 余秋雨《中國文脈》——余秋雨論述中國文學的精神脈絡——其中對韓愈的評價——特別強調了他「孤獨的先驅」形象——與〈獲麟解〉的主題——完全契合
- 木心《文學回憶錄》——木心說韓愈的文章「像一把刀」——但〈獲麟解〉——是這把刀最「溫柔」的一次出鞘——它不砍人——它只是——輕輕地——劃開了一道——一千二百年來——無數人——都感到的——傷口
- 王小波《沉默的大多數》——王小波討論「特立獨行的豬」——那隻豬——不屬於任何類別——不被主流接受——但活得很自由——這和韓愈的麒麟——有著驚人的相似——「不被歸類——才是真正的——自由」

## 九、背景補遺

### 9.1 「解」體——韓愈的發明？

「解」——是唐代才興起的一種文體——意為「解釋」「辨析」——韓愈、柳宗元——都寫過大量的「解」——如〈獲麟解〉、〈師說〉（亦稱〈師解〉）——這種文體——特點是——設一個具體的「疑問」——然後層層分析——最後給出「解答」——〈獲麟解〉——就是一個完美的例子——「麒麟是什麼？」→「麒麟為什麼不被認識？」→「這說明了什麼？」——層層推進——最後——繞了一圈——回到起點——「麟——果不為不祥也」——但——你讀完——已經——和開始時——完全不一樣了——這就是「解」的魅力——把讀者——從「常識」——帶到——「新的理解」——中。

### 9.2 韓愈與柳宗元的「麒麟」對比

柳宗元在〈三戒〉中也寫過動物寓言——〈臨江之麋〉〈黔之驢〉〈永某氏之鼠〉——和韓愈的〈獲麟解〉——表面上——都是用動物說人事——但態度完全不同：

| 面向 | 韓愈〈獲麟解〉 | 柳宗元〈三戒〉 |
|------|---------------|----------------|
| **動物主角** | 麒麟——祥瑞之獸 | 麋鹿——被狗吃掉、驢子——被老虎吃掉、老鼠——被貓消滅 |
| **人的態度** | 人——不認識麒麟——是人的問題——不是麒麟的問題 | 動物——因為愚蠢或依賴他人——自取滅亡——是動物自己的問題 |
| **韓愈/柳宗元的自況** | 韓愈 = 不被認識的麒麟——「不是我錯了——是你們不認識我」 | 柳宗元 = 被現實擊敗的動物——「是我太天真了——世界就是這麼殘酷」 |
| **態度** | 自傲——自信——帶著不屈的驕傲 | 自嘲——自省——帶著對人性的悲觀 |

**軍師評**：同樣是「被貶」的文人——同樣在寫動物寓言——韓愈寫的是「我是麒麟——你們不認識我——是你們的問題」——柳宗元寫的是「我是那隻愚蠢的驢子——我不該出現在不屬於我的地方」——一個是「不低頭的天才」——一個是「低頭後再抬頭的哲人」——兩種態度——都偉大——但韓愈——更「霸氣」——也更——孤獨。

### 9.3 〈獲麟解〉在韓愈作品中的位置

〈獲麟解〉——不是韓愈最重要的作品——比不上〈原道〉的思想深度——比不上〈師說〉的影響力——比不上〈進學解〉的文學成就——但——它可能是——韓愈所有作品中——最「韓愈」的一篇——因為——它直接說出了——韓愈對自己命運的——全部感受——「我——是一隻麒麟——我知道——你們不認識我——但——沒有關係——因為——麟之所以為麟者——以德不以形——我——知道——我是一隻麒麟——這——就夠了。」
