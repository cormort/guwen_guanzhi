# 〈報任安書〉背景資料

> **軍師分析 · 古文觀止卷五 · 司馬遷**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《漢書·司馬遷傳》收錄——司馬遷寫給朋友任安的私人書信 |
| **選集** | 《古文觀止》卷五·漢文類 |
| **體裁** | 書信體／抒情議論散文——中國文學史上最偉大的一封信 |
| **時代** | 西漢武帝太始四年（約西元前 93 年），司馬遷任中書令時 |
| **核心命題** | 一個人的尊嚴可以被人奪走嗎？——「人固有一死，或重於泰山，或輕於鴻毛」 |
| **關鍵人物** | 司馬遷（寫信者）、任安（司馬遷的朋友，時任益州刺史）、漢武帝（隱形的主角——權力的象徵）、李陵（事件的起點——因他而降的將軍） |
| **文本重點** | 這是中國文學史上最偉大的一封信。司馬遷因李陵之禍被判死刑，因家貧無錢贖罪，最終選擇接受宮刑以求活命完成《史記》。任安寫信要他「推賢進士」——推薦人才給朝廷。司馬遷在回信中坦白了自己的屈辱——「詬莫大於宮刑」——但他之所以不死，是因為《史記》未完成。信中充滿了憤怒、悲傷、不屈和終極的決心。他告訴任安：我活下來不是為了苟且偷生，我是為了完成一部足以傳之後世的著作 |
| **思想譜系** | 出自《漢書·司馬遷傳》。司馬遷在遭遇宮刑之後，將所有的憤怒、屈辱、不甘與決心都灌注到了這封信中。這封信在某種意義上是《史記》的「創作宣言」——它說明了司馬遷為什麼在遭受了最極端的羞辱之後，仍然選擇活下去。信中提到的「究天人之際，通古今之變，成一家之言」——成為《史記》最經典的定義，也是中國史學精神最核心的表述 |
| **一句話總結** | **〈報任安書〉是一個人用最屈辱的方式活下去的理由——當尊嚴被奪走時，你還有最後一件事可以守護，那就是你的作品。司馬遷告訴我們：死亡不是最可怕的，最可怕的是你還沒有完成你該做的事就死了。** |

## 二、時代背景——李陵之禍與司馬遷的命運轉折

### 2.1 李陵之禍——司馬遷一生的分水嶺

司馬遷的人生，以「李陵之禍」為界，分成截然不同的兩半：

| 面向 | 說明 |
|------|------|
| **事件經過** | 西元前 99 年（天漢二年），漢武帝派貳師將軍李廣利征伐匈奴，李陵（李廣之孫）主動請纓率五千步兵出擊。李陵行軍三十日，與匈奴單于主力八萬騎兵遭遇，以寡擊眾，殺敵過萬。最終因箭盡援絕、後無補給——距離邊塞僅百餘里——被迫投降。漢武帝原以為李陵戰死，後來得知李陵投降，大怒。朝中大臣紛紛指責李陵 |
| **司馬遷的辯護** | 司馬遷與李陵並無深交，但他在漢武帝面前為李陵說了公道話——他認為李陵以五千步兵對抗八萬騎兵，殺敵過萬，已經盡了全力；李陵投降可能是權宜之計，日後或有機會報效漢朝。這番話觸怒了漢武帝——漢武帝認為司馬遷在批評李廣利（漢武帝的寵妃李夫人之兄，當時的統帥）無能 |
| **司馬遷的刑罰** | 漢武帝將司馬遷下獄，判處死刑。在漢律中，死刑有兩個替代方案：一是以錢贖罪（需要五十萬錢），二是接受宮刑（腐刑）。司馬遷「家貧，財賂不足以自贖」——他沒有錢贖罪——「交遊莫救，左右親近不為一言」——他的朋友們誰也不幫他說話。最終，他選擇了宮刑。這在西漢社會是比死刑更大的羞辱——因為宮刑剝奪的是一個人的基本尊嚴 |
| **司馬遷的處境** | 宮刑之後，司馬遷被任命為「中書令」——名義上是皇帝的機要秘書，實際上是一個「閹人」才能擔任的官職。他每天出入宮廷，面對那些曾經的同僚——他寫道：「每念斯恥，汗未嘗不發背沾衣也」——每次想到這個恥辱，汗就從後背流下來，沾濕了衣服。他活著，但每一天都在承受比死亡更痛苦的煎熬 |

> **軍師按**：理解〈報任安書〉的關鍵，在於理解司馬遷所承受的「雙重羞辱」——第一重羞辱是宮刑本身，它剝奪了他作為男人的尊嚴；第二重羞辱是「中書令」這個職位——它讓他每天都要面對那些知道他受過宮刑的人。他不是在一個偏遠的地方默默承受——他就在權力核心，每天都要在那些曾經的同僚面前，以一個「閹人」的身份活著。寫信給任安的時候，他已經忍受了六年這樣的生活。

### 2.2 任安——寫信的對象

任安是司馬遷的朋友，也是漢武帝朝的重要官員。這封信是因任安的來信而起：

| 項目 | 內容 |
|------|------|
| **身份** | 任安，字少卿，滎陽人。漢武帝朝官員，曾任益州刺史（今天四川一帶的最高地方長官）。在巫蠱之禍中（西元前 91 年），任安因太子劉據事件而被判死刑 |
| **與司馬遷的關係** | 司馬遷與任安是朋友。任安知道司馬遷的遭遇——也知道司馬遷現在擔任中書令。他寫信給司馬遷，希望司馬遷利用中書令的職位「推賢進士」——向朝廷推薦人才。這封信本身沒有流傳下來，但從司馬遷的回覆中可以推斷其內容 |
| **任安的「不識時務」** | 任安寫這封信的時候，顯然沒有意識到——或者選擇性地忽略了——司馬遷的真實處境。他以為司馬遷當了中書令，就是皇帝身邊的紅人——可以推薦人才了。但實際上，司馬遷在宮中只是一個「掃除之隸」（打掃衛生的小官），一個被剝奪了尊嚴的閹人。任安的請求，無意中刺痛了司馬遷最深的傷口——你不是要我做一個正常的官員嗎？但我已經不是一個正常的人了 |
| **司馬遷的回應** | 司馬遷沒有直接拒絕任安——他寫了這封長信，坦白了一切。他告訴任安：我不是不願意推賢進士，而是我已經沒有資格做這件事了——我是一個受過宮刑的人，我是一個被社會拋棄的人。我的尊嚴已經被剝奪了——我唯一活下去的理由，是完成我的書。等書完成了，我就可以死了 |

> **軍師按**：任安這封信來得正是時候——它給了司馬遷一個機會，把自己六年來積壓在心中的一切——憤怒、屈辱、悲傷、決心——全部傾瀉出來。如果沒有任安的來信，司馬遷可能永遠不會寫出這封被稱為「中國文學史上最偉大的一封信」的作品。從這個意義上說，任安雖然「不識時務」，但他的無心之問，催生了中國文學史上最壯麗的生命宣言。

### 2.3 司馬遷的處境——為什麼他選擇了活著？

司馬遷選擇宮刑而不是死刑，是中國歷史上最令人心碎、也最令人敬佩的選擇之一：

| 面向 | 說明 |
|------|------|
| **他為什麼不選擇死？** | 司馬遷說：「人固有一死，或重於泰山，或輕於鴻毛。」他認為，如果他就這樣死了——以一個犯人的身份被處死——他的死就「輕於鴻毛」，毫無意義。他的父親司馬談臨終前囑咐他要完成《史記》——這項事業還沒有完成，他還不能死 |
| **他為什麼選擇宮刑？** | 宮刑是比死刑更痛苦的選擇——不是因為肉體上的痛苦（雖然確實極其痛苦），而是因為尊嚴的喪失。在漢代（以及整個古代中國），宮刑是對一個人最大的羞辱——它不僅剝奪了你的生育能力，更剝奪了你作為「人」的基本尊嚴。選擇宮刑，意味著你願意承受最極端的羞辱——只為了可以繼續活下去，完成你的事業 |
| **他活著的日子有多痛苦？** | 司馬遷在信中寫道：「是以腸一日而九迴，居則忽忽若有所亡，出則不知其所往。每念斯恥，汗未嘗不發背沾衣也。」——他的腸子一天要轉九個彎（極度痛苦），待在家裡恍恍惚惚像丟了魂一樣，出門不知道要去哪裡——每次想到這份恥辱，汗就從後背流下來。他不是在「活著」——他是在「忍受活著」 |
| **他為什麼還活著？** | 只有一個原因——「恨私心有所不盡，鄙陋沒世而文采不表於後也」——遺憾的是我的心願還沒有完成，如果就這樣死去，我的文章就不能流傳於後世了。他活著是為了完成《史記》——這是他活著唯一的理由 |

> **軍師按**：司馬遷的選擇，在某種意義上比慷慨赴死更需要勇氣。慷慨赴死只需要一瞬間的勇氣——而忍受每一天的羞辱活下去、在所有人的鄙視中繼續寫作——需要數年、數十年的勇氣。司馬遷用宮刑換來了十年的生命——在這十年中，他完成了《史記》。這不是懦夫的選擇——這是英雄的選擇。正如他自己所說：「所以隱忍苟活，幽於糞土之中而不辭者，恨私心有所不盡，鄙陋沒世而文采不表於後也。」——我之所以忍辱負重地活著，生活在糞土之中也不逃避——只是因為我的心願還沒有完成。

## 三、原文結構與核心論點

### 3.1 〈報任安書〉的結構

〈報任安書〉是一封長信，全文將近兩千字。它絕不是一封普通的回信——而是一篇精心結構的生命宣言：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：開場——為什麼回信這麼晚？** | 司馬遷先向任安道歉——你的來信我早就收到了，但我一直沒有回信。為什麼？因為我「罷癈不用」（被廢棄了不被重用），所以不敢與人交往——「闕然久不報，幸勿為過」（很久沒有回信，請不要責怪） | 一句話就讓任安知道——我的人生已經完全不同了。我不是那個你可以平等交往的朋友了——我是一個被社會拋棄的人 |
| **第二層：解釋——為什麼我不能推賢進士？** | 任安要他推賢進士，司馬遷的回答是：我今天已經不是一個正常的人了。他說自己「大質虧缺」——身體最重要的部分已經殘缺了——這樣的人，在社會上是「刑餘之人」，不受尊重。他說：「鄉者，僕亦嘗廁下大夫之列，陪外庭末議」——過去我也曾忝居下大夫之列，參與朝廷的議論——但現在，「今已虧形為掃除之隸」——我已經是一個打掃衛生的小吏——沒有資格推薦人才了 | 關鍵轉折——司馬遷告訴任安：你以為我還是原來的司馬遷嗎？我不是了。我被宮刑了。我已經不是一個完整的人了 |
| **第三層：坦白——說出宮刑的屈辱** | 這是全信最重要的段落之一。司馬遷說：「故禍莫慘於欲利，悲莫痛於傷心，行莫醜於辱先，詬莫大於宮刑」——禍沒有比貪圖利益更慘的，悲沒有比傷心更痛的，行為沒有比侮辱祖先更醜的，恥辱沒有比宮刑更大的。他坦白了宮刑對他的摧殘——「刑餘之人，無所比數」——受過刑的人，什麼都不是，在任何地方都排不上號。他說自己「介推」（介子推——不要封賞的隱士）不願讓人知道自己受過刑——因為太丟臉了 | 全信的情感爆發點——司馬遷不再掩飾。他直接告訴任安：我被宮刑了——我是個「閹人」——「詬莫大於宮刑」——這是世界上最大的恥辱 |
| **第四層：追憶——李陵之禍的始末** | 司馬遷開始回憶李陵之禍的經過。他詳細描述了李陵的英勇——以五千步兵對抗八萬匈奴騎兵，「連戰十餘日，所殺過當」——殺敵人數超過己方人數。但因為「救兵不至」（沒有援軍），「矢盡道窮」（箭完了、路沒了），「張空弮，冒白刃」（拉著空弓，迎著刀刃）——最終投降。司馬遷說：我為李陵說公道話——「以為李陵素與士大夫絕甘分少，能得人之死力，雖古之名將，不能過也」——我認為李陵平時與士大夫同甘共苦，能讓別人為他賣命——即使是古代的名將，也不過如此。但是——「明主不曉」——陛下（漢武帝）不明白——「以為僕沮貳師，而為李陵遊說」——以為我是在批評李廣利，為李陵開脫——於是把我下了監獄 | 司馬遷把整個事件的來龍去脈告訴任安——他不是因為做錯了什麼事而被處罰的——他為一個冤屈的人說了公道話，然後他付出了慘痛的代價。這個敘述本身就是一種控訴：我沒有錯——錯的是這個世界 |
| **第五層：升華——人為什麼活著？** | 這是全文的核心——中國文學史上最著名的段落之一。司馬遷說：「人固有一死，或重於泰山，或輕於鴻毛，用之所趨異也」——人終究會死，但有的死比泰山還重，有的死比鴻毛還輕——差別在於你為什麼而死。他說自己如果現在死了——作為一個「刑餘之人」——「若九牛亡一毛，與螻蟻何以異？」——像九頭牛身上少了一根毛，和螻蟻有什麼區別？他之所以忍辱負重地活著，是因為他要完成《史記》——這不是普通的歷史書，這是他一生的心血——「僕竊不遜，近自託於無能之辭，網羅天下放失舊聞，略考其行事，綜其終始，稽其成敗興壞之理……亦欲以究天人之際，通古今之變，成一家之言」——我想探討天人之間的關係，通曉古今變化的規律，形成一家之說——這就是《史記》的核心理念，也是中國史學最經典的定義 | 從個人悲劇上升到普遍意義——司馬遷從自己的屈辱中提煉出一個永恆的命題：什麼樣的死值得，什麼樣的死不值得？他的答案是：為了你應該做的事業而死——那才是值得的。在完成《史記》之前，他的死比鴻毛還輕——所以他選擇活著，承受一切屈辱 |
| **第六層：決心——完成《史記》之前，我不會死** | 司馬遷用一系列古代聖賢的例子來證明自己的決心——「文王拘而演《周易》，仲尼厄而作《春秋》；屈原放逐，乃賦《離騷》；左丘失明，厥有《國語》；孫子臏腳，《兵法》修列；不韋遷蜀，世傳《呂覽》；韓非囚秦，《說難》《孤憤》」——這些人都曾在最困厄的處境中創作了不朽的作品——司馬遷說，我也要這樣做。他告訴任安——「草創未就，會遭此禍。惜其不成，是以就極刑而無慍色」——我的書還沒有完成，就遭遇了這場災禍——我捨不得讓它半途而廢——所以接受最嚴酷的刑罰而沒有怨恨的表情。他最後說：「僕誠已著此書，藏之名山，傳之其人」——等我真正完成了這部書，把它藏在名山之中，傳給那些真正懂它的人——到那時，我就可以死了——「雖萬被戮，豈有悔哉！」——即使被殺一萬次，我還有什麼好後悔的！ | 全文的高潮——司馬遷宣告了他的終極決心：我之所以活著，不是因為我怕死——而是因為我的書還沒有完成。當書完成的那一刻，我的使命就結束了——那個時候，死亡不過是回家的路 |
| **第七層：結語——告別** | 信的最後，司馬遷對任安說：「今少卿乃教之以推賢進士，無乃與僕私心刺謬乎？」——你現在卻勸我推賢進士——這不是和我的心意完全相反嗎？他委婉地告訴任安：你同我講這些，是因為你還不了解我的真實處境。最後他說：「書不能悉意，故略陳固陋。謹再拜。」——一封信無法說盡我的心意，所以簡略地陳述了我這愚執的想法。謹此再拜。 | 信的結尾平靜而沉重——司馬遷已經說完了所有要說的話。他不會去推賢進士——他也不會去死——他會活著，把《史記》寫完 |

> **軍師按**：〈報任安書〉的結構是一層層推進的——從道歉到解釋，從解釋到坦白，從坦白到回憶，從回憶到升華，從升華到決心——每一層都比前一層更深刻、更沉重。司馬遷像剝洋蔥一樣，一層層地剝開自己的內心——從表面的「抱歉」到深處的「決心」——讓任安（也讓兩千年來的讀者）一步一步地走進他的世界。到最後一層——「藏之名山，傳之其人」——他已經不是在寫信了，他是在向整個世界宣告他的使命。這封信之所以偉大，不是因為它寫得好——而是因為它是一個人用整個生命寫出來的。

### 3.2 核心段落節錄

> 人固有一死，或重於泰山，或輕於鴻毛，用之所趨異也。

> ——中國文學史上最著名的句子之一。司馬遷用這一句話，把從個人悲劇中提煉出來的哲學思考變成了永恆的格言。你為什麼而死——決定了你的死有多重。為理想而死——重於泰山。為私慾而死——輕於鴻毛。

> 所以隱忍苟活，幽於糞土之中而不辭者，恨私心有所不盡，鄙陋沒世而文采不表於後也。

> ——司馬遷活著的全部理由。他忍辱負重地活在糞土之中——不是因為他貪生怕死——而是因為他還有沒有完成的心願。這句話裡，有最深沉的悲傷，也有最不可動搖的決心。

> 古者富貴而名摩滅，不可勝記，唯倜儻非常之人稱焉。蓋文王拘而演《周易》；仲尼厄而作《春秋》；屈原放逐，乃賦《離騷》；左丘失明，厥有《國語》；孫子臏腳，《兵法》修列；不韋遷蜀，世傳《呂覽》；韓非囚秦，《說難》《孤憤》。《詩》三百篇，大底聖賢發憤之所為作也。

> ——司馬遷用中國歷史上所有在困厄中創作出偉大作品的聖賢——來證明自己的選擇是對的。他不是孤獨的——從文王到孔子、從屈原到韓非——每一部偉大的作品都是從痛苦中誕生的。司馬遷把自己的命運納入這個偉大的傳統——他告訴我們：苦難，是偉大作品的必要條件。

> 僕竊不遜，近自託於無能之辭，網羅天下放失舊聞，略考其行事，綜其終始，稽其成敗興壞之理……亦欲以究天人之際，通古今之變，成一家之言。

> ——《史記》的「創作宣言」。司馬遷用三句話定義了《史記》的使命——「究天人之際」：探索天道與人事之間的關係；「通古今之變」：通曉從古到今的變化規律；「成一家之言」：形成自己獨立的學術體系。這三句話，成為中國史學的最高理想——影響了兩千年來所有的歷史學家。

> 僕誠已著此書，藏之名山，傳之其人，通邑大都，則僕償前辱之責，雖萬被戮，豈有悔哉！然此可為智者道，難為俗人言也。

> ——信的結尾前的終極宣言。司馬遷說：等我把這部書完成了——藏到名山裡，傳給那些懂它的人——我就償還了之前所受屈辱的債務。到那個時候——即使被殺一萬次——我也沒有什麼好後悔的了。最後一句「然此可為智者道，難為俗人言也」——這些話只能對聰明人說，很難對世俗之人說——既是對任安的認可（你是那個能理解我的人），也是對這個世界的不屑（你們這些俗人，不會懂的）。

> ——《漢書·司馬遷傳》

## 四、關鍵人物

### 4.1 司馬遷——〈報任安書〉的寫作者

| 項目 | 內容 |
|------|------|
| **身份** | 西漢太史令（前），中書令（後）。《史記》作者——中國最偉大的歷史學家。字子長，夏陽（今陝西韓城）人。生於西元前 145 年（或前 135 年），卒年不詳（約西元前 86 年） |
| **生平核心** | 司馬遷出身史官世家——父親司馬談是漢武帝的太史令。司馬遷自幼好學，二十歲後壯遊天下——足跡遍及中國各地，為《史記》的寫作積累了大量的第一手資料。父親臨終前囑咐他要完成一部偉大的史書——「余死，汝必為太史；為太史，無忘吾所欲論著矣」。司馬承繼父志，開始寫作《史記》。但西元前 99 年的李陵之禍改變了一切——他因為李陵辯護而被判死刑，因家貧無錢贖罪，最終選擇接受宮刑。宮刑之後，他沒有放棄寫作——反而以更大的決心投入《史記》的創作。〈報任安書〉寫於西元前 93 年左右——此時《史記》已經接近完成 |
| **〈報任安書〉中的司馬遷** | 這封信中的司馬遷，不是那個客觀冷靜的歷史學家——而是一個傷痕累累的人。他憤怒、悲傷、不甘、同時又充滿了不可動搖的決心。他在信中向任安（也向後世所有的讀者）坦白了一切——他的屈辱、他的痛苦、他活著的理由。這封信讓我們看到了一個「脫下史官長袍」的司馬遷——一個有血有肉、會痛會哭、但絕不屈服的人 |
| **歷史定位** | 司馬遷不僅是中國歷史上最偉大的歷史學家——他還是中國文學史上最偉大的散文家之一。〈報任安書〉是中國書信文學的巔峰——它將個人情感與普遍真理完美地融合在一起，成為中國文學史上最能觸動人心的作品之一。魯迅評價《史記》為「史家之絕唱，無韻之離騷」——而〈報任安書〉，就是這部「絕唱」背後的心跳聲 |

**軍師評**：司馬遷最偉大的地方，不是他寫了《史記》——而是他是在什麼樣的條件下寫的。他不是在舒適的書齋中從容地寫作——他是在被宮刑之後、在所有人鄙視的目光中、在每一天都在承受羞辱的情況下——仍然堅持完成了一部五十二萬字的巨著。當我看到〈報任安書〉中那句「每念斯恥，汗未嘗不發背沾衣也」時，我想到的不是一個可憐的人——而是一個真正偉大的人。因為真正的偉大，不是在順境中取得的成就——而是在逆境中仍然堅持了應該堅持的事。

### 4.2 任安——收信的對象

| 項目 | 內容 |
|------|------|
| **身份** | 任安，字少卿，滎陽人。漢武帝朝官員。出身貧寒，曾為大將軍衛青的門客，後官至益州刺史 |
| **與司馬遷的關係** | 司馬遷的朋友。他寫信給司馬遷——希望司馬遷利用中書令的職位「推賢進士」。這封信本身沒有流傳下來，但從司馬遷的回覆中可以推斷——任安的來信是善意的，他希望司馬遷在宮中有所作為 |
| **任安的結局** | 任安後來在「巫蠱之禍」（西元前 91 年）中因太子事件被牽連——太子劉據被誣陷謀反，領兵逃到長安城外，任安當時擔任北軍使者護軍——太子曾召他出兵相助。任安雖然沒有答應太子，但漢武帝認為他「坐觀成敗」，將其判處腰斬。任安的命運與司馬遷有相似之處——都是在權力的陰影下、被誤解而付出生命的代價 |
| **任安在這封信中的角色** | 如果沒有任安的來信，就沒有〈報任安書〉。任安的來信——雖然「不合時宜」——卻給了司馬遷一個機會，將壓抑在心中六年的情感全部傾洩出來。從這個意義上說，任安是〈報任安書〉的「催生者」——他的一封不經意的信，催生了中國文學史上最偉大的一封信 |

**軍師評**：任安在歷史上可能不是一個特別重要的人物——但他的名字卻因為收了一位朋友的來信而不朽。這本身就像是一則寓言：有時候，你的人生中最重要的事——就是你無意中問了一個對的問題。任安問司馬遷「為什麼不推賢進士」——這個問題在今天看來似乎有些不合時宜甚至殘忍——但正因為這個問題，司馬遷才能夠寫出那句「人固有一死，或重於泰山，或輕於鴻毛」。從這個意義上說，任安是幸運的——他問了一個讓千古動容的問題。

### 4.3 李陵——事件的起點

| 項目 | 內容 |
|------|------|
| **身份** | 字少卿，隴西成紀（今甘肅秦安）人。「飛將軍」李廣之孫。西漢名將，擅長騎射，愛護士卒 |
| **核心事件** | 西元前 99 年，李陵率五千步兵出征匈奴，遭遇單于主力八萬騎兵。李陵以寡敵眾，殺敵過萬。但因箭盡援絕，最終投降。漢武帝大怒，將其滿門抄斬。李陵從此留在匈奴，娶單于之女為妻，被封為右校王 |
| **與司馬遷的關係** | 李陵與司馬遷並無深交——但司馬遷認為李陵是一個「國士」（國家的優秀人才）。他在朝堂上為李陵辯護，認為李陵投降是權宜之計。這番話導致了司馬遷的宮刑。李陵後來得知司馬遷為他辯護而受刑的消息，曾說：「此吾所以為太史公也。」——但兩人從未再見面 |
| **歷史爭議** | 李陵投降匈奴後，是否真的「另有苦衷」？史料顯示——李陵最初投降時，可能確實想找機會回到漢朝——但漢武帝殺了他的全家，斷絕了他的歸路。他最終接受匈奴的官職，娶了匈奴公主——從漢朝的角度看，他是叛徒；從個人的角度看，他是被漢武帝逼上絕路的悲劇人物 |

**軍師評**：李陵是〈報任安書〉中不能不出現的名字——因為他是整個事件的起點。但司馬遷在信中寫李陵，不是為了替李陵辯護（這件事的後果他已經承受了）——而是為了向任安證明：我沒有做錯事。我只是說了一句公道話。他說李陵「雖古之名將，不能過也」——這不是在誇獎李陵——這是在告訴任安：我說的是實話。而說實話——就應該付出這樣的代價嗎？

### 4.4 漢武帝——信中的「隱形主角」

| 項目 | 內容 |
|------|------|
| **身份** | 西漢第七位皇帝，在位五十四年（西元前 141–87 年）。「秦皇漢武」——中國歷史上權力最大的皇帝之一 |
| **與司馬遷的關係** | 司馬遷的父親司馬談曾為漢武帝的太史令，司馬遷繼任太史令——他原本是漢武帝的臣子，忠誠地為朝廷服務。但李陵之禍後，兩人的關係變成了「施害者」與「受害者」的關係——司馬遷從未在〈報任安書〉中直接批評漢武帝，但他所有的痛苦，都源於漢武帝的判決 |
| **信中的漢武帝** | 司馬遷在信中提到漢武帝時只說「明主」（聖明的君主）——但他用「明主不曉」（陛下不明白）這個詞時，已經包含了極深的諷刺。他不是真的在說漢武帝是「明主」——他只是在用這個詞來表達一種無奈：您是「明主」，但您卻不明白我說的是實話。整封信中，漢武帝是一個「隱形的主角」——他不在信中，但他無處不在。司馬遷所有的痛苦，都源於他的一句話——而司馬遷在信中，用沉默的方式表達了對這個判決的質疑 |
| **歷史評價** | 漢武帝是一個偉大的皇帝——他拓展了中國的疆域、建立了大一統的帝國——但他也是一個暴君——他濫用權力、喜歡聽奉承話、打壓異己。司馬遷的悲劇——對漢武帝來說只是一個微不足道的插曲——但對司馬遷和中國歷史來說，卻是決定性的轉折點 |

**軍師評**：漢武帝是中國歷史上最有爭議的皇帝之一。他偉大——因為他開拓了疆域、建立了制度。但他也冷酷——因為他殺了多少人、毀了多少家庭，他自己也不知道。司馬遷只是他無數犧牲品中的一個——但這個犧牲品，卻用《史記》完成了對漢武帝（以及整個時代）最深刻的審判。兩千年後，漢武帝的名字還被人記得——很大程度是因為《史記》中對他的記錄。而這些記錄的作者——一個被他處以宮刑的人——用他的筆，決定了一個皇帝在歷史中的形象。這是報復嗎？不——這是歷史的力量。權力可以傷害一個人的身體——但無法阻止他寫下真相。

## 五、核心分析——「人固有一死，或重於泰山，或輕於鴻毛」

### 5.1 「死」的哲學——司馬遷對死亡的重新定義

司馬遷在〈報任安書〉中給出了中國文學史上最著名的死亡觀：

| 面向 | 分析 |
|------|------|
| **死亡的重量不是天生的** | 「人固有一死，或重於泰山，或輕於鴻毛」——死亡本身是中性的，但死亡的意義取決於你為什麼而死。為理想而死——重於泰山；為私慾而死——輕於鴻毛。司馬遷在這裡做了一個革命性的劃分：不是「什麼時候死」決定了你的價值，而是「為什麼而死」決定了你的價值 |
| **活著的尊嚴 vs. 死亡的尊嚴** | 一般人認為「慷慨赴死」比「忍辱活著」更有尊嚴——但司馬遷翻轉了這個觀念。他認為：如果你的死沒有完成你應該做的事——那你的死就是「輕於鴻毛」。反之，如果你活著是為了完成一件重要的事——即使你在活著的過程中承受了最極端的羞辱——你的生命仍然是「重於泰山」的。活著的尊嚴，不是由你「如何活著」來決定的——而是由你「為什麼活著」來決定的 |
| **司馬遷的選擇——最勇敢的活法** | 司馬遷選擇了「隱忍苟活」——不是因為他怕死，而是因為他認為自己的生命還有更重要的使命。他承擔了最大的屈辱，完成了《史記》——然後把這部著作「藏之名山，傳之其人」——讓它影響了兩千年。如果他當年選擇了死刑——他會成為一個為李陵辯護而死的烈士——但那樣的死，對歷史的影響遠不如他活下來完成《史記》。司馬遷用他的人生證明了：有時候，活著比死亡更需要勇氣——因為你要承受的是每一天的羞辱，而不只是那一瞬間的疼痛 |

> **軍師按**：司馬遷的死亡觀影響了整個中國文化。「重於泰山，輕於鴻毛」——這八個字——成為中國人判斷生命價值的基本標準。文天祥的「人生自古誰無死，留取丹心照汗青」——是這個觀念的延續；譚嗣同的「我自橫刀向天笑，去留肝膽兩崑崙」——也是這個觀念的變體。司馬遷用一句話，定義了中國知識分子的死亡哲學：死不是終點——死後的價值才是。

### 5.2 「發憤著書」——苦難作為創造的動力

司馬遷在信中列舉了中國歷史上所有在困厄中創作出偉大作品的人物——這個「發憤著書」的傳統，是〈報任安書〉中最核心的創作論：

| 人物 | 困境 | 作品 | 啟示 |
|------|------|------|------|
| **文王** | 被紂王囚禁於羑里 | 推演《周易》（六十四卦） | 肉體的囚禁不能禁錮思想——被關起來的時候，可以思考天地變化的道理 |
| **孔子** | 周遊列國不被重用，困於陳蔡之間 | 修訂《春秋》 | 當現實世界拒絕你的時候——你可以用筆創造一個理想的世界 |
| **屈原** | 被楚王放逐，徘徊於湘江畔 | 創作《離騷》 | 當你被國家拋棄的時候——你的靈魂可以在詩歌中找到歸宿 |
| **左丘明** | 雙目失明 | 撰寫《國語》 | 當你看不見這個世界的時候——你可以用心靈去看歷史 |
| **孫臏** | 被龐涓陷害，遭受臏刑（挖去膝蓋骨） | 撰寫《孫臏兵法》 | 當你的身體被摧殘的時候——你的智慧仍然是完整的 |
| **呂不韋** | 被秦始皇貶謫到蜀地 | 《呂氏春秋》流傳於世 | 當權力拋棄你的時候——你的作品仍然可以流傳 |
| **韓非** | 被秦始皇囚禁於秦國 | 創作〈說難〉〈孤憤〉 | 當你的理想無法實現的時候——你可以把它們寫下來，傳給後人 |

**軍師評**：司馬遷列舉這七個人，不是為了炫耀自己的學問——他是為了告訴任安（以及後世所有的讀者）：我不是瘋了才選擇活著——我是走在一條偉大的路上。從文王到韓非——每一部偉大的作品都是在痛苦的土壤中長出來的。我司馬遷——也將追隨他們的腳步——用我的痛苦，孕育出《史記》這樣一部不朽的作品。這不是自我安慰——這是對命運的回應：你摧毀了我的身體，但你無法摧毀我的靈魂。

### 5.3 「究天人之際，通古今之變，成一家之言」——《史記》的靈魂

這三句話是〈報任安書〉中最核心的「創作宣言」，也是《史記》的靈魂：

| 命題 | 含義 | 意義 |
|------|------|------|
| **究天人之際** | 探索天道（自然規律、命運）與人事（人的行為、選擇）之間的關係。司馬遷想問的是：這個世界有公正嗎？好人為什麼沒有好報？——〈伯夷列傳〉中的困惑就是「天人之際」的具體體現 | 這是《史記》最深刻的哲學問題。司馬遷不滿足於記錄歷史事實——他想理解歷史背後的法則。他對「天道無親，常與善人」的質疑——是中國思想史上最深刻的懷疑論之一 |
| **通古今之變** | 通曉從古到今歷史變化的規律。司馬遷想找出：國家為什麼興盛？為什麼衰亡？——他從黃帝寫到漢武帝，橫跨三千年的歷史，就是要找到變化背後不變的法則 | 這是《史記》的史學方法——不滿足於編年記事，而是追求對歷史的整體理解。司馬遷的史學是「通史」——他不把歷史分割成片段，而是把它作為一個整體來理解 |
| **成一家之言** | 形成自己獨立的學術體系。司馬遷不滿足於做一個「記錄者」——他要做一個「思想者」。《史記》不僅是歷史——它是司馬遷對整個世界的理解——從人類歷史到社會制度到人性本質 | 這是《史記》的終極目標。司馬遷不是任何學派的門徒——他是「自成一家」——他的思想超越了儒家、道家、法家——形成了一個獨立的、包羅萬象的思想體系 |

> **軍師按**：這三句話是理解《史記》的鑰匙——也是理解司馬遷一生追求的總綱。他之所以能夠忍受宮刑活下去——就是因為這三句話還沒有實現。他還沒有「究天人之際」——所以他不能死。他還沒有「通古今之變」——所以他不能死。他還沒有「成一家之言」——所以他不能死。這三句話，是他活著的全部理由——也是他寫出《史記》的全部動力。當我們讀《史記》時——我們不是在讀一部冰冷的歷史書——我們是在讀一個人在極端痛苦中，用生命寫出的答案。

## 六、軍師核心分析——當尊嚴被奪走時，你還有最後一件事可以守護

### 6.1 為什麼〈報任安書〉是中國文學史上最偉大的一封信？

〈報任安書〉的特殊之處在於——它不只是一封信，它是一份「生命宣言」：

| 面向 | 分析 |
|------|------|
| **私人信件中的哲學高度** | 這是一封私人信件——寫給一個朋友的——但它從個人的痛苦出發，上升到普遍的人類困境。它討論了死亡、尊嚴、使命、痛苦、創造——這些永恆的主題。它的偉大之處在於：它用最個人化的痛苦，表達了最普遍的真理。每一個人在讀〈報任安書〉的時候，都能從司馬遷的痛苦中看到自己的影子——不是因為我們都受過宮刑——而是因為我們都曾經面對過「該怎麼活下去」這個問題 |
| **真實的力量** | 這封信不是為了發表而寫的——它是寫給一個朋友的——所以它沒有任何修飾、沒有任何保留。司馬遷把自己最深的痛苦——「腸一日而九迴」「汗未嘗不發背沾衣」——全都寫了出來。這種真實，是任何文學技巧都無法達到的。因為它不是「文學」——它是一個人用生命寫出的文字 |
| **從絕望中開出的希望** | 整封信充滿了痛苦和憤怒——但它的最終結論不是絕望。司馬遷告訴我們：即使在最絕望的處境中，你仍然可以找到活下去的理由。不是為了復仇——不是為了證明什麼——而是為了完成你應該做的事。這個結論，比任何安慰的話都更有力量——因為它不是空話，是一個從絕望深處爬出來的人用自己的生命證明的真理 |

### 6.2 司馬遷的選擇對後世的影響

司馬遷的「隱忍苟活」——選擇活著完成《史記》——對後世的中國知識分子產生了深遠的影響：

| 時期 | 影響 |
|------|------|
| **漢代** | 司馬遷完成《史記》後不久去世——但《史記》迅速流傳開來，成為中國史學的典範。後世的史學家——從班固到司馬光——都受到《史記》的深刻影響。司馬遷「成一家之言」的理想，成為中國知識分子的最高追求 |
| **魏晉南北朝** | 在動盪的時代中，司馬遷的選擇被反覆提及。那些在亂世中仍然堅持寫作的知識分子——如陳壽寫《三國志》——都在精神上與司馬遷相呼應 |
| **唐宋** | 韓愈、柳宗元、歐陽修、蘇軾——這些唐宋八大家——都深受〈報任安書〉的影響。韓愈的「不平則鳴」——文章是從不平中產生的——直接繼承了司馬遷「發憤著書」的傳統。蘇軾一生多次被貶——他能在困境中保持豁達——也與司馬遷的精神有關 |
| **明清** | 在文字獄盛行的時代——司馬遷「藏之名山，傳之其人」的態度——成為許多知識分子的精神寄託——他們把自己的作品藏起來，等待後人來發現 |
| **近代** | 魯迅說《史記》是「史家之絕唱，無韻之離騷」——他將〈報任安書〉與〈離騷〉並列——認為兩者都是中國文學史上最能表達「悲憤」的作品。在抗日戰爭期間——司馬遷「隱忍苟活」的精神——被賦予了新的意義——在國家危亡的時候，知識分子應該活著、堅持寫作、保存文化的火種 |

### 6.3 與其他篇目的互文

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈滑稽列傳〉（司馬遷） | 〈滑稽列傳〉中的淳于髡、優孟、優旃——都是用幽默在權力面前說真話的人——與司馬遷在〈報任安書〉中的「直諫悲劇」形成了強烈的對照。司馬遷就是因為太「正經」地說真話而付出代價——如果他像滑稽人物一樣用笑話來表達——也許就不會遭受宮刑。〈滑稽列傳〉是司馬遷對「另一種活法」的想像 |
| 卷五·〈游俠列傳序〉（司馬遷） | 游俠「言必信，行必果」——用個人的力量維護公道——與司馬遷在〈報任安書〉中的「為李陵說公道話」——某種程度上也是一種「俠」的行為——只是他為此付出了慘痛的代價。游俠是司馬遷筆下的理想人格——而他自己，在現實中做了一回「俠」，卻被權力碾壓 |
| 卷五·〈伯夷列傳〉（司馬遷） | 〈伯夷列傳〉中對「天道無親」的困惑——好人為什麼沒有好報？——在〈報任安書〉中得到了最直接的回答：因為世界上就沒有所謂的「天道」。司馬遷不再困惑了——他接受了這個不公平的世界——然後決定用自己的作品來對抗它 |
| 卷五·〈屈原列傳〉（司馬遷） | 屈原與司馬遷的命運有著驚人的相似——兩人都因為說真話而被放逐（或受刑）。但屈原選擇了自殺——司馬遷選擇了活著。兩者都是對不公平命運的回應——只是回應的方式不同。屈原用死亡完成了他的詩歌——司馬遷用活著完成了他的史書 |
| 卷四·〈漁父〉（屈原） | 漁父勸屈原「與世推移」——隨波逐流——司馬遷的選擇在某種程度上也是「與世推移」——但他不是為了討好誰——他是為了活下來完成《史記》。他的「與世推移」是為了更大的目標——而不是為了苟且偷生 |
| 卷三·〈卜居〉（屈原） | 屈原在〈卜居〉中問了一系列問題——「寧正言不諱以危身乎？將從俗富貴以偷生乎？」——司馬遷在〈報任安書〉中給出了他的答案：我不會「從俗富貴以偷生」——但我也不會「正言不諱以危身」——我會「隱忍苟活」，活下來完成我的使命 |

### 6.4 對當代讀者的啟示

〈報任安書〉雖然寫於兩千年前，但它對當代讀者仍然有深刻的啟示：

| 面向 | 啟示 |
|------|------|
| **面對不公時** | 司馬遷告訴我們：當你遇到不公平的對待時，你可以憤怒——你甚至可以徹底崩潰——但最後——你必須找到一個理由讓自己活下去。這個理由——不應該是復仇——而應該是完成一件比你的痛苦更偉大的事 |
| **尊嚴的重新定義** | 司馬遷顛覆了我們對「尊嚴」的傳統理解。尊嚴不是「不受羞辱」——而是在受到羞辱之後，仍然堅持你應該做的事。真正的尊嚴，不是來自於別人怎麼看你——而是來自於你自己對自己的要求 |
| **痛苦與創造的關係** | 司馬遷的「發憤著書」傳統告訴我們：痛苦不是毫無意義的——它可以成為創造的動力。當你經歷了最深的痛苦時——如果你能夠把這種痛苦轉化為創造的能量——你可以做出平常無法做出的成就。《史記》就是從司馬遷的宮刑中誕生的——沒有宮刑，就沒有《史記》 |
| **作品的意義** | 司馬遷告訴我們：一個人可以被打倒——但他的作品會替他說話。即使你的身體被摧殘、你的尊嚴被剝奪——只要你的作品還在——你就沒有被徹底打敗。《史記》是司馬遷留給世界最偉大的遺產——它是他用生命換來的 |

## 七、一句話總結（再錄）

> **〈報任安書〉是一個人用最屈辱的方式活下去的理由。當李陵之禍將司馬遷打入深淵——當宮刑剝奪了他作為人的基本尊嚴——當他每一天都生活在「汗未嘗不發背沾衣」的痛苦中——他找到了一個活下去的理由：完成《史記》。這封信是他向朋友任安（也是向整個世界）的告白——他告訴我們，為什麼他選擇了最艱難的路：不是因為他怕死，而是因為他還有沒有完成的事。他用「人固有一死，或重於泰山，或輕於鴻毛」重新定義了死亡的意義——用「究天人之際，通古今之變，成一家之言」定義了《史記》的使命——用「藏之名山，傳之其人」宣告了他的終極決心。這封信之所以偉大，不是因為它寫得好——而是因為它是一個人用整個生命寫出來的。兩千年來，每一個在困境中掙扎的人——從被貶的官員到失意的文人——都能從〈報任安書〉中找到力量。因為司馬遷證明了：當尊嚴被奪走時，你還有最後一件事可以守護——那就是你的作品。而只要你的作品還在——你就沒有輸。** |

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈滑稽列傳〉（司馬遷） | 司馬遷因為太「正經」地說真話而付出宮刑的代價——〈滑稽列傳〉中的淳于髡、優孟、優旃則用幽默在權力面前說真話——兩者對照，可以看到司馬遷對「說真話的方式」的深刻反思 |
| 卷五·〈游俠列傳序〉（司馬遷） | 司馬遷為游俠辯護——游俠「言必信，行必果」——而司馬遷自己也因為「說公道話」而成為了某種意義上的「俠」——只是他為此付出了比任何人都慘痛的代價 |
| 卷五·〈伯夷列傳〉（司馬遷） | 司馬遷在〈伯夷列傳〉中質疑「天道無親」——在〈報任安書〉中，他不再質疑了——他知道這個世界是不公平的——但他用行動證明了：即使世界不公平，你仍然可以完成一件偉大的事 |
| 卷五·〈屈原列傳〉（司馬遷） | 屈原與司馬遷的命運對照——兩人都因說真話而受苦——屈原選擇了死亡，司馬遷選擇了活著。兩者沒有高下之分——但兩者都是對不公正命運的回應 |
| 卷四·〈漁父〉（屈原） | 漁父的「與世推移」——與司馬遷的「隱忍苟活」——兩者都是為了生存而做出的妥協——但司馬遷的妥協有一個更高的目標 |
| 卷三·〈卜居〉（屈原） | 屈原的「寧正言不諱以危身乎？將從俗富貴以偷生乎？」——司馬遷給出了第三個選擇：不正言不諱以危身，也不從俗富貴以偷生——而是隱忍苟活，完成使命 |

### 8.2 經典延伸

- 《史記·太史公自序》（司馬遷）——司馬遷對自己寫作《史記》的完整說明——《報任安書》中提到的許多觀念，在〈太史公自序〉中都有更詳細的闡述。兩篇合讀，可以完整地理解司馬遷的精神世界
- 《史記·李將軍列傳》（司馬遷）——李廣、李陵祖孫的傳記——是理解李陵之禍背景的第一手資料。司馬遷在傳中對李陵的描寫，與〈報任安書〉中的敘述互為補充
- 《漢書·司馬遷傳》（班固）——〈報任安書〉最早出於此——班固在〈司馬遷傳〉中全文收錄了這封信。班固對司馬遷的評價（「是非頗謬於聖人」——是非標準與聖人頗有出入）引發了後世對司馬遷思想的長期討論
- 《漢書·李陵傳》（班固）——從漢朝官方史學的角度記錄李陵事件——可與司馬遷的個人敘述互相參照
- 《史記》（司馬遷）——全文五十二萬字——〈報任安書〉是理解《史記》的「鑰匙」——讀完這封信再讀《史記》，你會覺得每一篇都帶著司馬遷的血淚
- 《文選》（蕭統編）——〈報任安書〉亦收入《昭明文選》卷四十一——蕭統將此信列為「書」類的代表作
- 錢穆《司馬遷與〈史記〉》——對司馬遷的生平與思想有深入的分析——特別是他對司馬遷「發憤著書」的解讀
- 李長之《司馬遷之人格與風格》——從人格心理學的角度分析司馬遷——認為司馬遷的悲劇塑造了他的史學風格——是一部極有深度的《史記》導讀

### 8.3 現代延伸

- 木心《文學回憶錄》——木心對司馬遷有極高的評價——他認為〈報任安書〉是中國文學中最偉大的「抒情散文」——「讀〈報任安書〉，不流淚的不是人」
- 余秋雨《中國文脈》——余秋雨在論述中國知識分子精神時——以司馬遷為核心案例——分析「苦難與創造」的關係
- 維克多·弗蘭克《活出意義來》——弗蘭克在集中營中發現了「意義療法」——認為人即使在最極端的苦難中——只要找到了「意義」——就能夠活下去。這與司馬遷「隱忍苟活」以完成《史記》的精神完全相通——只不過一個是被關在納粹集中營，一個是被困在宮刑的恥辱中——但他們找到的答案是一樣的：活下去，因為你還有未完成的使命
- 米蘭·昆德拉《生命中不能承受之輕》——關於生命中的「輕」（沒有意義的自由）與「重」（承擔責任的負荷）——司馬遷選擇了「重」——他把《史記》扛在肩上——用這個「重」來對抗「輕」（死亡）的誘惑
- 司馬遼太郎《史記的世界》——日本作家司馬遼太郎對《史記》的解讀——從一個外國人的視角重新審視司馬遷的偉大
