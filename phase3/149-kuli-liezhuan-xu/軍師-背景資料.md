# 〈酷吏列傳序〉背景資料

> **軍師分析 · 古文觀止卷五 · 史記·司馬遷**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《史記·酷吏列傳》序——非傳記正文，而是列傳前的序論 |
| **選集** | 《古文觀止》卷五·史記文類 |
| **體裁** | 史記列傳序論／政論散文 |
| **時代** | 西漢武帝時期（約西元前 2 世紀末），酷吏政治最為熾盛的時代 |
| **核心命題** | 法律越多，犯罪越多？——「法令者，治之具，而非制治清濁之源也」——酷吏不是問題的原因，而是問題的症狀 |
| **關鍵人物** | 司馬遷（太史公，本序作者）、孔子（引用「導之以政，齊之以刑，民免而無恥」）、老子（引用「法令滋彰，盜賊多有」）、漢代酷吏代表——郅都、寧成、張湯、杜周、趙禹等 |
| **文本重點** | 司馬遷為酷吏作傳，但他並未一味謾罵酷吏，而是冷靜分析為什麼會出現酷吏——他引用老子「法令滋彰，盜賊多有」和孔子「導之以政，齊之以刑，民免而無恥」，指出當社會的道德基礎崩潰時，再嚴厲的法律也無法維持秩序。酷吏的出現不是他們的個人問題，而是時代的產物 |
| **思想譜系** | 出自《史記》七十列傳中的〈酷吏列傳〉第六十二。司馬遷將西漢前期十餘名酷吏合為一傳，並在開頭添加了這篇序文，闡述自己對「以法治國」和「道德治國」的總體看法。本篇在《古文觀止》卷五中，與〈伯夷列傳〉、〈屈原列傳〉、〈管晏列傳〉等並列，是司馬遷以序論形式表達核心史觀的重要篇章。相較於其他列傳的「敘事推動議論」，〈酷吏列傳序〉是典型的「議論導入敘事」——司馬遷先亮出自己的觀點，再用具體的酷吏案例來證明 | |
| **一句話總結** | **司馬遷對酷吏的態度是「哀矜而勿喜」——他們不是壞人，而是制度失靈的症狀。一個時代之所以出現大量酷吏，不是因為有壞人在當官，而是因為這個時代的道德基礎已經崩潰了。** |

## 二、時代背景——酷吏的「必要之惡」

### 2.1 為什麼西漢武帝時期會出現大量酷吏？

酷吏在中國歷史上並非每個時代都有——它的出現與特定政治條件密切相關。西漢武帝時期（約西元前 141–87 年在位）是酷吏最為活躍的時代，原因如下：

| 面向 | 說明 |
|------|------|
| **帝國轉型** | 漢武帝在位五十四年，將西漢從「黃老之治」（無為而治）轉向「獨尊儒術」和「積極有為」。但儒術只是表面的意識形態——實際的統治手段卻是嚴刑峻法。外儒內法，是武帝時代統治的真正底色 |
| **權力集中** | 漢武帝打擊諸侯王（推恩令）、削弱相權（設立內朝）、加強中央集權。這個過程中需要一批「不講情面」的官員來執行皇帝的意志——酷吏就是這批人。他們不依附於地方豪強、不被貴族左右、只忠於皇帝一人 |
| **戰爭與財政壓力** | 武帝連年征伐匈奴、開疆拓土，耗費巨大——需要更高效的稅收和更嚴格的財政管理。酷吏被派去整頓地方財政、打擊豪強兼併、嚴懲偷稅漏稅——手段越嚴厲越好 |
| **社會矛盾激化** | 在皇權擴張的過程中，地方豪強、諸侯王、官僚集團與皇帝之間的利益衝突日益尖銳。酷吏是皇帝用來「清理地方勢力」的工具——他們以嚴酷的手段打擊豪強，維護中央的權威 |

> **軍師按**：酷吏不是一個「道德敗壞」的類別，而是一個「功能類別」——他們是皇帝用來完成特定政治任務的「工具」。漢武帝需要他們來打擊豪強、整頓財政、加強中央集權——所以他們出現了。當任務完成後，或者當他們的嚴酷手段引起社會反彈時，皇帝又會毫不猶豫地拋棄他們——工具用完了就扔，這就是酷吏的命運。司馬遷之所以要為酷吏作傳，不是因為他同情酷吏，而是因為他看到了一個更深的問題：皇帝的統治方式，決定了官吏的行為方式。

### 2.2 司馬遷對酷吏的矛盾態度

司馬遷對酷吏的態度是極其複雜的——他不喜歡酷吏的嚴酷手段，但他也不認為酷吏是「壞人」：

| 面向 | 說明 |
|------|------|
| **對酷吏手段的批評** | 司馬遷在〈酷吏列傳〉正文中，詳細記錄了酷吏的殘酷手段——如張湯以「腹誹」之罪株連大臣、杜周以「捕獄多至數千人」為能——這些記錄本身就帶有強烈的批評態度 |
| **對酷吏存在的理解** | 但司馬遷在序言中清楚地表明：酷吏不是孤立的個人問題，而是時代的產物——「法令者，治之具，而非制治清濁之源也」——法律只是工具，不能解決根本的社會問題。當社會本身出了問題時，就會出現兩種人：一種是酷吏（用嚴刑峻法來維持秩序），一種是貪官（趁亂謀取私利）——兩者都是病灶的症狀 |
| **「哀矜而勿喜」的態度** | 司馬遷對酷吏的態度，可以概括為《論語》中的一句話——「哀矜而勿喜」——對他們的遭遇感到悲哀和同情，而不是為他們的失敗感到高興。酷吏的悲劇在於：他們自以為是皇帝的忠臣，但在皇帝眼中他們只是用完就丟的工具——當他們的嚴酷手段引起民怨時，皇帝會毫不猶豫地殺掉他們來平息眾怒 |

## 三、原文結構與核心論點

### 3.1 〈酷吏列傳序〉的結構

〈酷吏列傳序〉是司馬遷為〈酷吏列傳〉撰寫的開篇序論，全文以議論為主，層層推進：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：引用孔子——禮教優於刑罰** | 開篇引用孔子：「導之以政，齊之以刑，民免而無恥；導之以德，齊之以禮，有恥且格。」——用政令引導、用刑罰整頓的社會，人民只會想辦法逃避刑罰，而不會有羞恥心；用道德引導、用禮儀整頓的社會，人民既有羞恥心，又會自覺遵守秩序 | 為全文奠定基調——刑罰不是最好的治理方式。最好的治理是德教和禮儀 |
| **第二層：引用老子——法令越多，犯罪越多** | 接著引用老子：「法令滋彰，盜賊多有。」——法令越繁雜、越嚴密，盜賊反而越多。這與孔子之說形成了雙重印證：儒道兩家雖然思想不同，但在「法律不是萬能」這一點上達成了驚人的一致 | 用兩家聖賢之言撐起核心論點——法律的擴張不能解決社會問題，反而可能使問題惡化 |
| **第三層：提出核心命題——法律只是工具** | 司馬遷自己下結論：「法令者，治之具，而非制治清濁之源也。」——法律只是治理國家的工具，而不是判斷社會好壞的根本標準。一個社會的清明與否，根源在道德風氣，不在法律的嚴密程度 | 全文的核心論點——法律是「具」（工具），不是「源」（根本） |
| **第四層：歷史回顧——秦朝嚴刑峻法的教訓** | 司馬遷回顧秦朝的歷史——「昔天下之網嘗密矣，然姦偽萌起，其極也，上下相遁，至於不振」——秦朝的法律極其嚴密（網密），但犯罪反而層出不窮，到了極點時，從上到下都在互相欺騙、逃避責任，導致國家無法運轉。他特別指出了「吏治若救火揚沸，非武健嚴酷，惡能勝其任而愉快乎？」——當社會已經亂到像火災一樣的時候，不用強硬嚴酷的手段，根本無法控制局面 | 用秦朝的歷史教訓來證明：嚴刑峻法不但不能解決問題，反而會讓問題更糟。酷吏的出現，本身是社會失序的結果，不是原因 |
| **第五層：轉入正文——為酷吏立傳的緣由** | 在序言的結尾，司馬遷說：「漢興，破觚而為圜，斫琱而為樸，網漏於吞舟之魚，而吏治烝烝，不至於姦，黎民艾安。由是觀之，在彼不在此。」——漢朝初年，廢除了秦朝的嚴刑峻法（破觚為圜），以寬鬆簡樸的方式治理，法律網眼大到連吞船的大魚都能漏過去——但吏治反而清明，百姓安居樂業。這證明：治理國家的關鍵在於道德（彼），不在於法律（此） | 以漢初的「寬鬆治理」與武帝時代的「嚴刑峻法」進行對比，為〈酷吏列傳〉整篇正文拉開序幕——接下來就要寫那些在「嚴刑峻法」時代出現的酷吏了 |

> **軍師按**：這篇序文的結構，是典型的「古為今用」——司馬遷先引古聖賢之言（孔子、老子），再回顧歷史（秦朝、漢初），最後才引出自己的寫作對象（酷吏）。這種寫法給後文（酷吏列傳正文）提供了極其有力的批判框架：不是批判酷吏個人——因為酷吏只是工具——而是批判「為何這個時代需要工具」。

### 3.2 核心段落節錄

> 孔子曰：「導之以政，齊之以刑，民免而無恥；導之以德，齊之以禮，有恥且格。」老氏稱：「法令滋彰，盜賊多有。」

> ——序文的開篇總綱。以孔子和老子的話並列，儒道合一，共同指向同一個結論：法律和刑罰的擴張，無法帶來真正的社會秩序。司馬遷選擇用老子之言作為全文的出發點——這在「獨尊儒術」的漢武帝時代，是一個意味深長的選擇。

> 法令者，治之具，而非制治清濁之源也。

> ——全篇最核心的一句話。司馬遷把法律明確定位為「工具」（具），而不是「根本」（源）。一個社會的清濁好壞，根源在於道德風氣和人心向背——法律再嚴密，也無法創造一個有羞恥心的社會。這句話既是對酷吏現象的解釋，也是對漢武帝「以法為治」的委婉批評。

> 昔天下之網嘗密矣，然姦偽萌起，其極也，上下相遁，至於不振。

> ——秦朝的教訓。秦朝的法律網羅極其嚴密（「網密」），但腐敗和犯罪（「姦偽」）就像野草一樣不斷冒出。到了最極端的時候（「其極也」），從上到下都在互相欺騙、推卸責任（「上下相遁」），整個國家都無法正常運轉。這是司馬遷對「嚴刑峻法」最有力的批判：法律越嚴，犯罪越多；懲罰越重，欺騙越深。

> 漢興，破觚而為圜，斫琱而為樸，網漏於吞舟之魚，而吏治烝烝，不至於姦，黎民艾安。

> ——漢初的成功經驗。漢高祖和文帝、景帝廢除了秦朝的嚴刑峻法（「破觚為圜」——把方角磨圓了），法律的網眼大到連吞船的大魚都能漏過去——但沒有人利用這個漏洞犯罪，因為吏治清明、百姓安樂。這段話與前文的秦朝形成了最強烈的對比：秦朝法網嚴密卻犯罪橫行，漢初法網寬鬆卻天下太平——根本差別不在法律本身，而在統治者的道德和社會的風氣。

> ——《史記·酷吏列傳》

## 四、關鍵人物

### 4.1 司馬遷——酷吏時代的清醒觀察者

| 項目 | 內容 |
|------|------|
| **身份** | 西漢太史令，《史記》作者——中國最偉大的歷史學家 |
| **與酷吏的關係** | 司馬遷生活在漢武帝時代——這正是酷吏政治的巔峰時期。他親眼目睹了張湯、杜周等人如何以嚴刑峻法來執行皇帝的意志，也看到了這些酷吏最終如何被皇帝拋棄——他們的命運就是他寫作〈酷吏列傳〉的素材 |
| **寫作態度** | 司馬遷對酷吏的態度是「客觀冷靜但內心複雜」——他記錄酷吏的嚴酷手段時，沒有刻意渲染恐怖；他分析酷吏出現的原因時，沒有簡單地歸咎於酷吏個人。他始終把焦點放在更宏大的問題上：為什麼一個時代會需要酷吏？——這將批評的矛頭，從酷吏的上司（皇帝），指向了時代本身 |

**軍師評**：司馬遷寫〈酷吏列傳〉最大的貢獻，不是讓人們知道了酷吏有多殘酷——這在當時是人人皆知的事實。他的貢獻在於提出了兩個問題：第一，酷吏是怎麼產生的？第二，如果不靠酷吏，我們該如何治國？第一個問題的答案是「時代需要」，第二個問題的答案是「德教」——這兩個答案，在兩千年後仍然值得深思。

### 4.2 老子與孔子——序文的理論權威

| 項目 | 內容 |
|------|------|
| **孔子之言** | 「導之以政，齊之以刑，民免而無恥；導之以德，齊之以禮，有恥且格」——來自《論語·為政》。孔子認為刑罰只能讓人民害怕，不能讓人民知恥；只有德教和禮儀，才能創造真正的社會秩序 |
| **老子之言** | 「法令滋彰，盜賊多有」——來自《道德經》第五十七章。老子認為法律越繁雜，犯罪反而越多。因為法律本身會創造新的犯罪（比如新法規讓原本合法的事變成了非法），同時也會激發人民鑽法律漏洞的聰明才智 |
| **在序中的功能** | 司馬遷將儒道兩家的聖賢之言並列，不是為了調和儒道（這在學術上是不可能的），而是為了形成「雙重權威」的論證效果——即使立場完全不同的孔子和老子，都在「法律不是萬能」這個問題上達成一致了，你們還能說什麼？ |

**軍師評**：這篇序文最巧妙的地方，就是司馬遷把「獨尊儒術」時代的官方權威（孔子）和道家思想（老子）並列使用。他沒有直接說漢武帝的統治有問題——他不可能這麼說——而是讓孔子和老子的話幫他說：如果孔子和老子都說法律不是萬能的，那為什麼現在的皇帝卻把法律當作唯一的工具呢？這是極為高明的政治修辭——用聖賢的話來委婉地批評當代的政策。

### 4.3 漢代酷吏代表（列傳正文人物）

司馬遷在〈酷吏列傳〉正文中記錄了十餘位酷吏，以下是其中最有代表性的幾位：

| 人物 | 事蹟要點 | 司馬遷的態度 |
|------|---------|-------------|
| **郅都** | 西漢早期酷吏的代表，漢文帝、景帝時人，以嚴厲打擊豪強聞名。號稱「蒼鷹」——像鷹一樣銳利冷酷。他執法不避權貴，甚至在朝堂上當面責備諸侯王。但他同時也是一個清廉正直的人——「行法不避貴戚，列侯宗室見都側目而視，號曰『蒼鷹』」 | 司馬遷對郅都有保留的肯定——他雖然嚴酷，但至少清廉正直，不是為了私利 |
| **寧成** | 漢景帝、武帝時人，以殘酷聞名。他做濟南都尉時，連號稱「蒼鷹」的郅都都被他壓制。後來被免官後回到家鄉，自稱「仕不至二千石，賈不至千萬，安可比人乎」——然後用酷烈手段經營產業，成為地方豪強 | 司馬遷對寧成持批評態度——他從酷吏變成了地方豪強，證明了酷吏其實和豪強是一體兩面 |
| **張湯** | 漢武帝時期最著名的酷吏。他善於揣摩皇帝的心意——皇帝想要嚴懲的人，他就往重裡判；皇帝想要寬恕的人，他就往輕裡判。他發明了「腹誹」之罪（心中誹謗朝廷也是罪），株連了許多人。但他同時也是一個極有才幹的人——他參與了漢武帝的幣制改革、鹽鐵專賣等重大政策。最終他因為政治鬥爭被誣告自殺——死後被抄家，家產只有五百金（對於一個做了多年高官的人來說，少得可憐） | 司馬遷對張湯的態度最為複雜——他記錄了張湯的殘酷（「所治即豪，必舞文巧詆」），但也記錄了他的清廉（死後家無餘財）——張湯不是一個簡單的「壞人」，而是一個充滿矛盾的時代產物 |
| **杜周** | 漢武帝後期的酷吏，以善於審訊、株連極廣聞名。他在擔任廷尉期間，每年審理的案件多達數千件，株連的人動輒數百人。他的辦案原則是「上所欲擠者，因而陷之；上所欲釋者，久繫待問而微見其冤狀」——皇帝想要打擊的人，就想辦法定罪；皇帝想要寬恕的人，就故意拖延審理等待皇帝暗示。有人問他：「你斷案不依法，只依皇帝的臉色——這是對的嗎？」杜周回答：「法令從哪裡來的？是前代的皇帝制定的。你說的『法律』，難道比當今聖上的『旨意』更權威嗎？」——這句話暴露了酷吏政治的本質：法律只是權力的工具，皇帝才是真正的法律 | 司馬遷對杜周的批評最為直接——他記錄了杜周那句「何謂法？」的辯解，展現了酷吏制度的核心問題：當法律完全服從於權力時，司法就變成了迫害的工具 |

## 五、核心分析——「酷吏：制度失靈的症狀」

### 5.1 司馬遷的「工具論」——法律與道德的關係

〈酷吏列傳序〉的核心論點，可以概括為一句話：法律是工具，不是根本。

| 層面 | 法律（具） | 道德（源） |
|------|-----------|-----------|
| **功能** | 懲罰犯罪、維持表面的秩序 | 培養羞恥心、創造真正的秩序 |
| **極限** | 只能嚇阻，不能教化 | 可以改變人心，不需要外在強制 |
| **過度的後果** | 產生更多的犯罪（人民的智力用來鑽法律漏洞），以及更多的酷吏（皇帝要用嚴刑來維持統治） | 產生自覺的守法公民——不需要監督也能遵守規則 |
| **秦朝的教訓** | 法網嚴密，但「姦偽萌起」——犯罪層出不窮 | 從上到下都習慣欺騙和推卸責任，國家最終無法運轉 |
| **漢初的成功** | 法網寬鬆——「網漏於吞舟之魚」 | 但吏治清明、百姓安樂——「黎民艾安」 |

### 5.2 酷吏的悲劇——皇帝的工具，用完即丟

司馬遷在〈酷吏列傳〉中記錄了一個極其諷刺的規律：幾乎每一個著名的酷吏，最終都沒有好下場。

| 酷吏 | 結局 | 原因 |
|------|------|------|
| 張湯 | 被誣告自殺 | 政治鬥爭中失勢——皇帝需要一個替罪羊來平息眾怒 |
| 杜周 | 病死（少數善終者） | 他極其善於揣摩皇帝的心意，在政治鬥爭中存活了下來——但他死後不久，他家族就被滅門 |
| 郅都 | 被殺 | 觸怒了竇太后（漢景帝的母親）——太后認為他太殘酷，皇帝無法保護他 |
| 寧成 | 被免官 | 皇帝認為他太嚴酷，失去了利用價值——免官後他變成了地方豪強 |

> **軍師按**：酷吏的命運，是極權政治中「工具人」的典型寫照。當皇帝需要打擊豪強、整頓財政、鎮壓異議時——酷吏就是最佳工具。但當酷吏的嚴酷手段引起社會反彈、或者他們自己陷入權力鬥爭時——皇帝會毫不猶豫地拋棄他們。因為在皇帝眼中，酷吏從來不是人的問題：他們只是工具，用完了就換新的。這就是為什麼司馬遷說「法令者，治之具」——工具這個詞，用在酷吏身上也同樣精確。

### 5.3 與其他篇目的互文對照

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈伯夷列傳〉（司馬遷） | 伯夷列傳質疑「天道無親，常與善人」——為什麼好人沒有好報？〈酷吏列傳序〉提出了一個更尖銳的問題：如果法律和道德都失靈了，好人該怎麼辦？酷吏的出現，正是「天道失靈」在政治領域的表現 |
| 卷五·〈孔子世家贊〉（司馬遷） | 孔子主張「德治」——〈酷吏列傳序〉開篇就引用孔子來支持「德治優於法治」的論點。但孔子世家贊中司馬遷說孔子雖然偉大卻不被當世重用——這與酷吏序形成了呼應：孔子的「德治」理想很美，但在殘酷的現實政治中，統治者需要的卻是「嚴刑峻法」 |
| 卷五·〈屈原列傳〉（司馬遷） | 屈原是被讒臣陷害的忠臣——而讒臣的存在，與酷吏的存在是同一種政治生態的產物：在一個法治不彰、人治為上的時代，忠臣和酷吏是同一枚硬幣的兩面——忠臣被權貴打壓，酷吏被皇帝利用——兩者都是權力遊戲中的犧牲品 |
| 卷四·〈諫逐客書〉（李斯） | 李斯主張廣納賢才，反對逐客——但他後來和趙高合謀，成了秦朝酷政的一部分。李斯的故事證明：一個主張開放的人，在權力面前也可能變成酷吏——制度和環境對人的塑造，遠比人的意志更強大 |

## 六、軍師核心分析——「法令滋彰，盜賊多有」的現代意義

### 6.1 一個永恆的政治悖論

〈酷吏列傳序〉提出的問題——「法律越多，犯罪越多？」——是一個跨越兩千年的政治悖論。它在現代世界中的表現形式包括：

- **法律膨脹**：法律條文越多、越細密，社會就越需要更多的執法者（警察、法官、檢察官），而這些執法者本身也可能變成新的權力濫用者（古代的酷吏，現代的濫權執法）
- **道德滑坡**：當法律試圖取代道德來規範一切行為時，社會的道德感反而會下降——因為人們不再需要自己的道德判斷，只要「依法辦事」就好。這就是孔子說的「民免而無恥」——只要不觸法，什麼都可以做
- **制度性暴力**：一個過度依賴法律和刑罰的社會，必然會產生「制度性暴力」——不是某個官員特別壞，而是整個系統需要暴力來維持運轉。酷吏，就是這種制度性暴力的人格化表現

### 6.2 司馬遷給後世的警示

司馬遷寫〈酷吏列傳序〉，不是為了批判某一朝、某一人——而是為了揭示一個普遍的歷史規律：

> 一個時代的道德風氣越好，法律就越不需要嚴苛；一個時代的道德風氣越差，法律就越嚴苛——但法律越嚴苛，道德風氣反而越差。這是一個惡性循環，而酷吏是這個循環中最顯眼的標誌。

他的警示在兩千年後仍然有效：一個社會如果只靠法律和刑罰來維持秩序，而不去關注道德建設和社會信任——那麼它一定會走向酷吏化——用越多的酷吏來鎮壓越多的犯罪，而犯罪永遠不會減少，因為問題的根源從來不在法律的不夠嚴密。

## 七、一句話總結（再錄）

> **〈酷吏列傳序〉是司馬遷對「刑罰治國」的深刻批判。他引用孔子和老子的話，指出法律只是「治之具」而非「制治清濁之源」——當一個社會的道德基礎崩塌，再嚴厲的法律也無法創造真正的秩序。酷吏不是問題的原因，而是問題的症狀——他們的出現，標誌著一個時代已經失去了依靠道德來維持秩序的能力。司馬遷不恨酷吏——他對酷吏的態度是「哀矜而勿喜」——因為他知道，酷吏和他們所懲罰的那些罪犯一樣，都是這個道德淪喪時代的產物。當一個社會需要用嚴刑峻法來維持表面的平靜時，真正的問題從來不在法律本身，而在產生法律需求的那些社會病灶之中。兩千年後，我們仍然面臨同樣的問題——好的治理，需要的不只是更好的法律，更需要更好的社會。**

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈伯夷列傳〉（司馬遷） | 伯夷列傳質疑天道公平——〈酷吏列傳序〉則質疑法律的有效性——兩篇都是司馬遷對「社會秩序如何建立」的終極追問 |
| 卷五·〈孔子世家贊〉（司馬遷） | 孔子世家贊肯定德治的理想——與〈酷吏列傳序〉中對德治的讚美形成直接呼應——但現實中的政治卻走向了法治和酷吏——這是理想與現實的巨大落差 |
| 卷五·〈屈原列傳〉（司馬遷） | 屈原是被權力拋棄的忠臣——他代表了權力遊戲中的「受害者」；酷吏則是權力遊戲中的「工具人」——兩者都是極權政治的不同面孔 |
| 卷四·〈諫逐客書〉（李斯） | 李斯主張開放和包容——但秦朝的實踐卻是嚴刑峻法——李斯的矛盾正說明了「理論上的法治」與「現實中的酷吏」之間的差距 |
| 卷二·〈子產不毀鄉校〉（左傳） | 子產在鄭國推行寬鬆的治理——允許人們在鄉校中議論國政——這是「德治」的典範——與酷吏的嚴刑峻法形成了鮮明對比 |

### 8.2 經典延伸

- 《史記·酷吏列傳》（司馬遷）——全文閱讀——〈酷吏列傳序〉只是開篇，正文中對十餘位酷吏的詳細記述才是這篇列傳的主體——讀完序文後再讀正文，可以清楚地看到司馬遷如何用具體案例來證明自己在序文中提出的論點
- 《史記·太史公自序》（司馬遷）——司馬遷對自己撰寫〈酷吏列傳〉動機的說明——他為什麼選擇為這些「不光彩」的人物立傳
- 《論語·為政》（孔子）——「導之以政，齊之以刑」一段的出處——理解孔子對「政」「刑」「德」「禮」四者關係的完整思想
- 《道德經》第五十七章（老子）——「法令滋彰，盜賊多有」的出處——老子對「以法治國」的根本性批判
- 《漢書·酷吏傳》（班固）——班固沿襲司馬遷的體例，在《漢書》中繼續記錄西漢中後期的酷吏——可以對照兩者的不同態度
- 《報任安書》（司馬遷）——司馬遷自己的生命宣言——他之所以能夠冷靜地分析和記錄酷吏，是因為他自己就是權力暴力的親身受害者——他懂得「被權力碾壓」是什麼感覺

### 8.3 現代延伸

- 傅柯《規訓與懲罰》——對監獄和刑罰制度的現代批判——與司馬遷對「法治的局限」的思考形成了跨越兩千年的對話——傅柯認為現代社會的刑罰體系不僅沒有減少犯罪，反而創造了新的犯罪形式（如「犯罪學」對人的分類和標籤化）
- 漢娜·鄂蘭《極權主義的起源》——關於「制度性暴力」如何產生——鄂蘭的分析與司馬遷對酷吏政治的分析有著驚人的相似：當一個系統不再受法律和道德的約束時，就會產生「制度化」的暴力——而執行這種暴力的人（酷吏／納粹官僚），往往不是禽獸，而是普通的「盡責的公務員」
- 米爾格倫《服從權威實驗》——著名的心理學實驗，證明普通人在權威的指示下可以做出極其殘酷的行為——這與司馬遷在〈酷吏列傳〉中記錄的現象完全一致：大多數酷吏不是天生的壞人，而是「服從上級命令」執行了一項他們自己也知道有問題的政策
- 李長之《司馬遷的人格與風格》——對司馬遷思想體系的深入分析——其中對司馬遷「反酷吏」立場的討論，對於理解〈酷吏列傳序〉的政治背景極有幫助
