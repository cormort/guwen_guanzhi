## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《駱賓王集》／《文苑英華》卷六百四十六——「檄」類代表作；駱賓王文集最早的編本已散佚，今傳《駱臨海集》為後人輯錄 |
| **選集** | 《古文觀止》卷七·唐文——唐代駢文與檄文的雙重典範；也是「初唐四傑」中唯一入選《古文觀止》的檄文作品 |
| **體裁** | 檄文——古代征戰或起兵時發布的討伐文書，目的是動員人心、聲討敵人、宣示正義立場。本文是中國歷史上最著名的檄文之一——被譽為「古今第一檄文」——與〈討王莽檄〉並稱 |
| **時代** | 唐睿宗光宅元年（西元 684 年）——武則天廢中宗李顯、立睿宗李旦為傀儡皇帝，自己以太后身份臨朝稱制，改元「光宅」——距離大唐開國僅六十六年 |
| **核心命題** | 一篇檄文能否推翻一個女皇帝？——「一抔之土未乾，六尺之孤何託」——以君臣大義為劍——以情感動員為盾——試圖用文字的力量——發動一場推翻武則天的軍事政變 |
| **關鍵人物** | 駱賓王（初唐四傑之一，徐敬業的幕僚與軍師——本文的作者）、武則天（本文的討伐對象——中國歷史上唯一的女皇帝——當時以太后身份臨朝）、徐敬業（唐初名將徐世勣之孫——稱「匡復府上將」——在揚州起兵反武——失敗被殺）、李孝逸（受武則天命率軍討伐徐敬業的唐將） |
| **文本重點** | 徐敬業在揚州起兵反對武則天臨朝稱制——駱賓王為他寫了這篇討伐武則天的檄文。全文以駢文寫成——氣勢磅礴——層層進逼——先斥責武則天的「出身微賤」與「個人品德敗壞」——然後指責她的種種「罪行」——「穢亂春宮」「殘害忠良」「殺姊屠兄，弒君鴆母」——然後轉而歌頌徐敬業的正義——最後以「試看今日之域中，竟是誰家之天下」作結——全篇情感如江河奔騰——一氣呵成。據《新唐書》記載——武則天讀到「一抔之土未乾，六尺之孤何託」時——感嘆說：「宰相安得失此人！」——可見此文的震撼力——連被罵的人——都忍不住為作者的才華鼓掌 |
| **一段話總結** | **〈爲徐敬業討武曌檄〉是中國歷史上最著名的檄文——沒有之一。它用駢文中最華美鏗鏘的語言——寫了一封給女皇帝的宣戰書——從「穢亂春宮」到「一抔之土未乾，六尺之孤何託」——從「南連百越，北盡三河」到「請看今日之域中，竟是誰家之天下」——每一句都是戰鼓——每一段都是號角。這篇文章的力量在於——連被罵的人（武則天）都忍不住為作者的才華鼓掌——感嘆「宰相安得失此人」。檄文沒有推倒武則天——徐敬業的起兵失敗了——但這篇文章——卻永遠地留在了中國文學史上——成為「文字的力量」最經典的證明——也成為「才華可以超越政治立場」——最震撼的案例。** |

## 二、時代背景——武則天臨朝與徐敬業揚州起兵

### 2.1 武則天的崛起——從才人到女皇之路

要理解這篇檄文——必須先理解武則天是一個什麼樣的人。她的一生——是中國歷史上最具傳奇色彩、也最具爭議的政治生涯之一：

| 面向 | 說明 |
|------|------|
| **出身** | 武則天（西元 624–705 年）——并州文水（今山西文水）人——父親武士彠是唐朝開國功臣——官至工部尚書。十四歲入宮——成為唐太宗的「才人」（低級嬪妃）——在太宗身邊十二年——地位始終不高。太宗去世後——按規定——她入感業寺為尼 |
| **重返皇宮** | 唐高宗李治（太宗的兒子）在做太子的時候——就和武則天建立了感情。太宗駕崩後——高宗在感業寺與她重逢——舊情復燃——將她重新接入皇宮——先封為「昭儀」——最終在慘烈的宮廷鬥爭後——於永徽六年（655 年）被立為皇后 |
| **二聖臨朝** | 高宗體弱多病——患「風疾」（頭痛、視力衰退）——無法正常處理政務——武則天以皇后的身份協助批閱奏章——逐漸掌握了實際權力。從麟德元年（664 年）開始——高宗與武則天一起上朝——「天下大權，悉歸中宮」——史稱「二聖臨朝」 |
| **廢立皇帝** | 弘道元年（683 年）——高宗駕崩——太子李顯即位（唐中宗）——武則天以太后身份臨朝稱制。嗣聖元年（684 年）二月——武則天以中宗「想把天下讓給岳父」為由——廢中宗為廬陵王——改立第四子李旦為帝（唐睿宗）——但睿宗被軟禁——不得參與朝政——武則天完全控制了朝政大權 |
| **光宅改元** | 廢中宗後——武則天改元「光宅」——更改百官名稱——改東都洛陽為「神都」——大量提拔支持自己的官員——從太后走向女皇的意圖——已經非常明顯。這一年——就是徐敬業起兵的那一年——也是這篇檄文誕生的那一年 |

> **軍師按**：武則天——不是一個「普通的女皇帝」。她是中國歷史上——唯一一個——以女性身份——正式稱帝的人。在她之前——漢代呂后權傾朝野——但沒有稱帝。在她之後——清代慈禧太后垂簾聽政——但也沒有稱帝。武則天的稱帝——是對整個中國傳統政治秩序——對「男尊女卑」的儒家禮教——最徹底的挑戰。這也正是駱賓王這篇檄文的「關鍵張力」所在——他不僅在罵一個掌權者——他在罵一個「違反了天地秩序」的人——一個「牝雞司晨」（母雞打鳴——比喻女性掌權）的女人。

### 2.2 徐敬業——名將之後的「匡復」之舉

徐敬業——是唐初傳奇名將李勣（徐世勣，字懋功——凌煙閣二十四功臣之一）的孫子。這個家族——本身就是一個濃縮的唐初政治史：

| 面向 | 說明 |
|------|------|
| **家族背景** | 徐敬業的祖父徐世勣——最初是瓦崗寨的將領——後來追隨李世民——在唐朝開國戰爭中立下赫赫戰功——被賜姓「李」——改名李勣——位列凌煙閣二十四功臣。他的父親李震早逝——徐敬業繼承了祖父的爵位——襲封英國公——任眉州刺史——後被貶為柳州司馬 |
| **起兵的契機** | 684 年——武則天廢中宗、臨朝稱制——大批李唐宗室和忠於李唐的官員不滿。徐敬業和他的弟弟徐敬猷——以及唐之奇、駱賓王、杜求仁等人——在揚州密謀起兵——打著「匡復廬陵王」（恢復中宗李顯的皇位）的旗號——號召天下人起來反對武則天 |
| **揚州起兵的規模** | 光宅元年（684 年）九月——徐敬業在揚州正式起兵——自稱「匡復府上將」——以駱賓王為「藝文令」（相當於秘書長兼宣傳部長）——以魏思溫為軍師——十天之內——聚集了十多萬人馬——佔據了揚州、潤州（鎮江）等地——聲勢浩大 |
| **失敗的原因** | 徐敬業的起兵——從一開始就有嚴重的戰略分歧。軍師魏思溫建議——「直取洛陽」——以迅雷不及掩耳之勢打擊武則天的核心統治區。但徐敬業猶豫不決——選擇了先攻取潤州——鞏固根據地——這給了武則天調兵遣將的時間。武則天命李孝逸率三十萬大軍征討——徐敬業在戰略上的猶豫——加上部下叛變——最終導致了失敗 |
| **結局** | 十一月——徐敬業兵敗——在逃亡途中被部下所殺——首級被送往洛陽。徐敬業和徐敬猷的家族——被武則天下令屠滅——連祖父李勣的墳墓——都被掘開——戮屍滅籍——可謂極其慘烈 |

> **軍師按**：徐敬業的起兵——是一場典型的「忠臣義士」的失敗。他的旗號是正義的——「匡復李唐」——但他本人——並非一個優秀的軍事統帥。他的猶豫不決——他的戰略失誤——他的部下叛變——最終讓這場聲勢浩大的起兵——在短短兩個月內——就灰飛煙滅。但——這篇檄文——卻因為徐敬業的失敗——反而更加有名——因為它代表了「雖然失敗了——但精神永存」——這正是中國文化中最推崇的悲劇英雄主義。

### 2.3 684 年的政治版圖——站在起兵十字路口的大唐

684 年——是唐朝歷史上最動盪的一年之一。讓我們看看當時的政治格局：

| 面向 | 說明 |
|------|------|
| **中央政局** | 武則天已經完全控制了朝廷——反對她的宰相裴炎被她處死——支持她的一批新進官員（如李孝逸、崔詧等）被提拔到關鍵崗位。朝廷中——沒有人敢公開反對她 |
| **地方反應** | 徐敬業在揚州起兵後——江南地區一片震動——但地方官員大多持觀望態度——沒有大規模響應。原因有二：一是武則天的鎮壓行動極其迅速——二是徐敬業的戰略失誤讓人對他缺乏信心 |
| **宗室態度** | 李唐宗室——諸王們——對武則天臨朝當然不滿——但他們大多選擇了沉默或消極抵抗。徐敬業起兵後——宗室諸王沒有及時響應——這使得徐敬業成了一支「孤軍」——這也是他失敗的重要原因 |
| **民心向背** | 從理論上說——唐朝開國不到七十年——李唐皇室在民間還有很高的威望——武則天以女性身份掌權——確實違反了傳統觀念——應該會引起大規模的反對。但實際上——武則天在執政初期——施政相當清明——減輕賦稅、提拔寒門——得到了底層百姓和中下層官員的支持——所以——民心並非一面倒地反對她 |

> **軍師按**：684 年——大唐站在一個十字路口。一邊是武則天——一個能力超群、手段凌厲的女性政治家——她想要稱帝——而且確實有稱帝的能力。另一邊是徐敬業——一群忠於李唐的將領和文人——他們想要恢復李唐的正統——但實力不足、戰略混亂。駱賓王的這篇檄文——就是在這個十字路口上——用文字點燃的一把火——這把火——最終沒有燒掉武則天——但它照亮了中國文學史上——最燦爛的一頁。

## 三、文本結構——從「罵人」到「戰爭動員」的層層推進

### 3.1 全文三段式結構

〈爲徐敬業討武曌檄〉全文約 500 餘字——以駢文寫成——氣勢磅礴——可以分為三個層次：

| 層次 | 核心內容 | 情感基調 |
|------|---------|---------|
| **第一段：揭發罪狀——痛罵武則天** | 開門見山——直接揭露武則天的「出身微賤」與「罪惡滔天」——從她入宮為才人——到「穢亂春宮」——到「殘害忠良」「殺姊屠兄，弒君鴆母」——一層層剝開——極盡批判之能事 | 凌厲、憤怒——如刀如劍——字字見血 |
| **第二段：宣示正義——歌頌徐敬業** | 轉折——從「罵敵」轉向「揚己」——歌頌徐敬業的忠義——家族的功勳——軍隊的聲勢——「南連百越，北盡三河」——以及「匡復」的正義性——給予天下人信心 | 雄壯、豪邁——如戰鼓——如號角——鼓舞士氣 |
| **第三段：號召天下——最後的動員令** | 以「一抔之土未乾，六尺之孤何託」為情感高潮——然後向天下人發出號召——無論你是什麼身份——都應該站出來——共同討伐武則天——以「試看今日之域中，竟是誰家之天下」結尾——氣吞山河 | 激昂、悲壯——既有情——有理——更有不可抗拒的號召力 |

### 3.2 第一段：揭發罪狀——「穢亂春宮，殘害忠良」

檄文的第一段——是最直接、最凌厲的攻擊——駱賓王沒有任何鋪墊——直接開罵：

> 「偽臨朝武氏者——性非和順——地實寒微。昔充太宗下陳——曾以更衣入侍。洎乎晚節——穢亂春宮。」

— 翻譯：這個非法臨朝聽政的武氏——本性不溫順——出身又低微。當年不過是太宗的才人——曾經靠著侍奉更衣的機會接近皇帝。到了後來——竟然在太子的東宮裡——淫亂後宮。

然後——他列舉了武則天的「罪行」：

> 「踐元后於翬翟——陷吾君於聚麀。加以虺蜴為心——豺狼成性。近狎邪僻——殘害忠良。殺姊屠兄——弒君鴆母。人神之所同嫉——天地之所不容。」

— 翻譯：她竊取了皇后的位子——陷害君王幹了亂倫之事。加上她毒蛇般的心腸——豺狼般的本性——親近小人——殘害忠良——殺害姐姐、屠戮兄長——謀殺君王——毒死母親。這是人和神共同痛恨的——是天地所不能容忍的。

| 句子 | 翻譯 | 批判策略 |
|------|------|---------|
| **「性非和順，地實寒微」** | 本性不溫順，出身低微 | 先打壓對方的「道德」和「身份」——出身「寒微」在門閥觀念濃厚的唐代——是殺傷力極強的標籤 |
| **「穢亂春宮」** | 在太子宮中淫亂 | 直指武則天與高宗李治的關係——說她在太宗還在世時就與太子（後來的高宗）私通——這是禮教大防中最嚴重的指控 |
| **「殘害忠良」** | 殺害忠臣良將 | 武則天在立后之爭中——確實處死或流放了長孫無忌、褚遂良等顧命大臣——這個指控——有相當的事實基礎 |
| **「殺姊屠兄」** | 殺害姐姐和哥哥 | 武則天確實與異母兄長武元慶、武元爽不睦——兩人被流放而死——但「殺姊」的指控——史料中缺乏明確證據——存在誇大 |
| **「弒君鴆母」** | 殺死皇帝、毒死母親 | 「弒君」指控高宗被武則天害死——「鴆母」指控她毒死了自己的母親楊氏——這兩個指控——在正史中——完全沒有證據——是駱賓王為了製造憤怒情緒而做的「極限指控」 |
| **「人神之所同嫉，天地之所不容」** | 人和神都痛恨，天地都不容忍 | 將個人的罪行——上升到「違反天地秩序」的高度——為後文的「討伐正當性」——鋪墊神學基礎 |

> **軍師按**：第一段的策略——是「先定罪再審判」。駱賓王不需要武則天真的犯了這些罪——他只需要讓讀者「相信」她犯了這些罪。而且——這些指控——從「出身寒微」到「穢亂春宮」到「殘害忠良」——是層層遞進的——從身份——到道德——到政治——一層比一層嚴重——最後推到「弒君鴆母」——這個在歷史學上完全站不住腳的指控——但對當時的讀者來說——情緒已經被前面的指控帶動起來了——這個「終極指控」——正好給情感一個爆發點。這就是檄文的「修辭暴力」——它不是司法文書——不需要證據——它只需要讓人憤怒——讓人願意拿起武器。

### 3.3 第二段：宣示正義——「南連百越，北盡三河」

罵完武則天後——檄文轉向——開始歌頌徐敬業和他的軍隊：

> 「敬業——皇唐舊臣——公侯冢子。奉先君之成業——荷本朝之厚恩。…… 用班聲以動北塞——發怒於南浦。南連百越——北盡三河——鐵騎成群——玉軸相接。海陵紅粟——倉儲之積靡窮——江浦黃旗——匡復之功何遠。」

— 翻譯：徐敬業——是大唐的老臣——公爵侯爵的嫡系子孫。繼承先輩的功業——承受本朝的厚恩。…… 戰馬在北塞嘶鳴——怒氣在南浦爆發。南方連接百越——北方直到三河——鐵騎成群——戰車相接。海陵的糧倉——堆積的糧食吃不完——江浦的戰旗——匡復的功業還有多遠呢？

| 層次 | 分析 |
|------|------|
| **強調家世** | 「皇唐舊臣——公侯冢子」——徐敬業是李勣的孫子——李勣是凌煙閣功臣——這個身份——本身就代表了「正統」——代表了大唐的合法繼承人之一 |
| **渲染軍威** | 「鐵騎成群——玉軸相接」——這是極具畫面感的戰爭描寫——雖然實際上徐敬業的軍隊——沒有這麼強大——但檄文需要誇大——需要讓讀者覺得——這是一支不可戰勝的力量 |
| **儲備充足** | 「海陵紅粟——倉儲之積靡窮」——海陵（今江蘇泰州）的糧倉——儲糧豐富——象徵後勤充足——戰爭有保障——這是給潛在的響應者信心 |
| **正義必勝** | 「匡復之功何遠」——匡復李唐的功業——離我們不遠了——這是預言式的信心喊話——告訴大家——勝利在望——快來加入我們 |

> **軍師按**：第二段——從「破」轉向「立」。第一段把武則天罵得體無完膚——等於拆掉了對手的合法性。第二段則要建立己方的正義性和實力——告訴讀者——「罵完了——來看看我們——忠臣之後——兵強馬壯——糧草充足——勝利在望——你還在猶豫什麼？」——這個轉折——非常自然——也非常有力——從「憤怒」到「希望」——情緒的轉換——恰到好處。

### 3.4 第三段：動員令——「一抔之土未乾，六尺之孤何託」

這是全文最著名的段落——也是中國文學史上最有名的情感動員之一：

> 「公等或居漢地——或協周親——或膺重寄於話言——或受顧命於宣室。言猶在耳——忠豈忘心。一抔之土未乾——六尺之孤何託？」

— 翻譯：諸位——有些人是漢室的舊臣——有些人是皇室的親族——有些人曾經受過先帝的重託——有些人在宣室（皇宮）接受過遺命。先帝的話還在耳邊——你們的忠心豈能忘記？先帝陵墓的黃土還沒有乾——年僅六尺的孤兒（中宗李顯）——又能託付給誰呢？

然後——是全文的最高潮：

> 「試看今日之域中——竟是誰家之天下！」

— 翻譯：請看看——今天的天下——到底是誰家的天下！

| 句子 | 分析 |
|------|------|
| **「一抔之土未乾」** | 高宗的陵墓——剛蓋上黃土——還沒有乾——這是極具畫面感的修辭——暗指高宗剛死不久（683 年駕崩——684 年起兵——確實只有一年左右的時間）——先帝屍骨未寒——你們就把他的兒子廢了——把天下交給了一個女人——你們對得起先帝嗎？ |
| **「六尺之孤何託」** | 中宗李顯——被廢為廬陵王——軟禁在均州（今湖北）——他是一個「六尺之孤」——一個無依無靠的孤兒——誰來保護他？這個修辭——把政治鬥爭——轉化為一種「保護孤兒」的道德召喚——極具感染力 |
| **「試看今日之域中，竟是誰家之天下」** | 這是最經典的結尾——駱賓王把筆鋒指向每一個人——請你看看——今天的天下——到底是李家的——還是武家的？——如果你覺得是李家的——你就應該站出來——如果你不出來——武家就會拿走它——這句話的威力——在於它讓每個人——都無法逃避——你必須做出選擇 |

| 面向 | 分析 |
|------|------|
| **情感層面** | 「一抔之土未乾——六尺之孤何託」——從感性入手——激發人們對先帝的懷念、對孤兒的同情、對武則天的憤怒——這種情感——比理性的政治論證——更加直接——也更加有力 |
| **道德層面** | 駱賓王提醒「受顧命」的大臣——你們曾在先帝面前發誓——要保護李唐的江山——現在——你們的誓言還在耳邊——你們難道要背叛先帝嗎？——這是道德綁架——但——對當時的士大夫——這種綁架——非常有效 |
| **行動層面** | 最後的「試看今日之域中——竟是誰家之天下」——是一次「選擇題」——你要麼選擇「李家」——加入我們——要麼選擇「武家」——成為不忠不義之人——沒有中立空間——這種「非此即彼」的修辭——是最有效的動員手段 |

> **軍師按**：全文最精彩的部分——就是「一抔之土未乾，六尺之孤何託」這兩句。據《新唐書·駱賓王傳》記載——武則天讀到這裡的時候——大吃一驚——說：「宰相安得失此人！」——意思是——宰相（指當時的宰相裴炎）怎麼沒有把這樣的人才拉攏過來？這句話——是對駱賓王才華的最高讚美——也是對這篇檄文力量的最高證明——連被罵的人——都忍不住為罵她的人鼓掌。從文學的角度說——「一抔之土」和「六尺之孤」——一個是「土」——一個是「人」——一個是「過去」（先帝已死）——一個是「未來」（孤兒尚在）——兩個意象——壓縮在兩句之中——產生了極強的情感衝擊力。這是駢文最極致的技巧——用最少的字——說最多的話——打最痛的心。

## 四、關鍵人物

### 4.1 駱賓王——初唐四傑中最悲劇的文人

| 項目 | 內容 |
|------|------|
| **身份** | 唐代詩人、文學家，「初唐四傑」之一（王勃、楊炯、盧照鄰、駱賓王）。字觀光，婺州義烏（今浙江義烏）人。生於約西元 619 年——卒年不詳（約 684 年後下落不明）——一生坎坷——命運多舛 |
| **生平** | 駱賓王少負才名——七歲能詩——最著名的〈詠鵝〉就是他七歲時所作——「鵝鵝鵝——曲項向天歌——白毛浮綠水——紅掌撥清波」——千百年來——是中國兒童啟蒙詩歌中的經典。成年後——他先後做過一些小官——但性格耿直——不善逢迎——屢遭貶謫——最後擔任長安主簿——又被貶為臨海丞（今浙江台州）——所以世稱「駱臨海」 |
| **加入徐敬業** | 684 年——徐敬業在揚州起兵——聘請駱賓王擔任「藝文令」——負責起草文書——這篇〈爲徐敬業討武曌檄〉——就是他在此時寫下的傑作。徐敬業失敗後——駱賓王下落不明——有的說他被殺——有的說他出家為僧——還有的說他隱居浙江——成了一個永久的謎 |
| **文學成就** | 駱賓王以駢文和詩歌聞名——他的駢文氣勢雄渾——情感奔放——格律工整——突破了六朝以來駢文「華而不實」的格局——將駢文從「宮廷裝飾品」——變成了「戰鬥的武器」。〈爲徐敬業討武曌檄〉——是他最著名的作品——也是中國駢文史上——運用最純熟、效果最震撼的篇章 |
| **「初唐四傑」中的定位** | 在「初唐四傑」中——王勃以才情勝——楊炯以氣骨勝——盧照鄰以情感勝——駱賓王以氣勢勝。他的文章和詩歌——都有一種「不可阻擋的力量」——〈在獄詠蟬〉的「露重飛難進——風多響易沉」——〈於易水送人〉的「此地別燕丹——壯士髮衝冠」——以及這篇檄文的「試看今日之域中——竟是誰家之天下」——都充滿了英雄氣概和悲壯情懷 |

**軍師評**：駱賓王是中國文學史上——最典型的「才高命蹇」的文人。他七歲就能寫出「鵝鵝鵝」——但一輩子沒有得到與才華相匹配的官位。直到徐敬業起兵——他才終於有了一個施展才華的舞台——寫出了這篇千古絕唱。但他押對了才華——押錯了政治——徐敬業失敗了——駱賓王也從此消失在歷史的迷霧中。然而——這篇檄文——讓他的名字永遠留在中國文學史上——比那些打敗他的將軍們——更加不朽。

### 4.2 武則天——中國歷史上唯一的女皇帝

| 項目 | 內容 |
|------|------|
| **身份** | 中國歷史上唯一的女皇帝——唐高宗李治的皇后——唐中宗李顯和唐睿宗李旦的母親——也是武周王朝的開國皇帝（在位 690–705 年）——死後被廢去帝號——與高宗合葬乾陵——立「無字碑」——是非功過——任後人評說 |
| **與檄文的關係** | 她是這篇檄文的討伐對象——也是這篇檄文最著名的「見證者」。據《新唐書》記載——武則天讀到這篇檄文時——「讀之——矍然而歎」——特別是在「一抔之土未乾——六尺之孤何託」處——她感嘆「宰相安得失此人」——這說明——她不僅不生氣——反而對駱賓王的才華——充滿了欣賞——甚至有一點惋惜——這樣的人才——為什麼不能為我所用？ |
| **政治能力** | 武則天是中國歷史上最傑出的政治家之一——她在位期間——推行科舉——提拔寒門——打壓關隴集團——發展經濟——穩定邊疆——為後來的「開元盛世」奠定了基礎。但她也以殘酷的手段——剷除異己——包括殺害自己的兒女（存疑）和孫輩——使用酷吏（來俊臣、周興等）——製造了大量冤案——是一個極具爭議的人物 |
| **文學修養** | 武則天本人——也是一個有相當文學修養的人——她寫過不少詩文——最著名的〈如意娘〉——「看朱成碧思紛紛——憔悴支離為憶君——不信比來長下淚——開箱驗取石榴裙」——情感真摯——文采斐然。正因為她懂文學——她才更能欣賞駱賓王的才華——也正因為她懂文學——她才會在看到一篇罵自己的文章時——感嘆作者沒有被自己所用 |

**軍師評**：武則天對這篇檄文的反應——是中國文學史上最有趣的「讀者反饋」之一。一般的被罵者——要麼憤怒——要麼不屑——要麼無視。但武則天——這位被罵的女皇帝——她選擇了——欣賞。她不是不生氣——她是太懂文學了——她知道這篇文章寫得多好——她更知道——這樣的人才——不能為我所用——是一種巨大的損失。這種「超越個人恩怨——對才華的尊重」——正是武則天之所以能成為一代女皇的氣度之一。她輸掉了這場「罵戰」——但贏得了後世對她氣度的——另眼相看。

### 4.3 徐敬業——失敗的起義軍領袖

| 項目 | 內容 |
|------|------|
| **身份** | 唐代將領——英國公李勣（徐世勣）之孫——襲封英國公爵位——在揚州起兵反對武則天——自稱「匡復府上將」——失敗被殺 |
| **起兵的「正當性」** | 從李唐皇室的角度看——徐敬業的起兵——是「勤王」——是「匡復」——是具有正當性的。但從武則天的角度看——他是一個「叛逆」——一個「亂臣賊子」。歷史的評價——取決於你站在哪一邊 |
| **失敗的原因** | 徐敬業的失敗——有多方面的原因：第一——戰略失誤——軍師魏思溫建議直取洛陽——但徐敬業選擇了先攻潤州——錯失了戰機；第二——缺乏外部支援——李唐宗室沒有響應——他成了孤軍；第三——武則天的鎮壓極其迅速冷酷——派李孝逸率三十萬大軍——一舉擊潰了徐敬業的軍隊；第四——部下叛變——最後徐敬業是被自己的部下殺死的——可見他的領導力和凝聚力——並不強 |
| **文學上的「幸運」** | 雖然徐敬業在政治和軍事上是失敗者——但他因為駱賓王這篇檄文——而獲得了文學上的「永生」。如果沒有駱賓王——如果沒有這篇檄文——徐敬業可能只是一個——在史書角落裡的一個小叛亂——和他的許多失敗者一樣——被歷史遺忘。但因為這篇檄文——他的名字——和「一抔之土未乾——六尺之孤何託」——一起——被永遠地記住了 |

**軍師評**：徐敬業——是一個「被文學拯救的失敗者」。他的失敗——是軍事和政治的失敗——但他的名字——因為駱賓王的文字——而成為了一種「精神象徵」——象徵那些「知其不可而為之」——為了信念而戰——即使失敗——也在所不惜的人。從這個意義上說——徐敬業雖然輸了戰爭——但贏得了歷史的敬重——而這一切——都要歸功於——那篇連敵人都忍不住叫好的檄文。

## 五、核心分析——「檄文的力量」與「文字的政治」

### 5.1 什麼是「檄文」？——古代戰爭的「宣傳武器」

「檄文」——是中國古代戰爭中——發布的討伐性文書——作用類似於今天的「宣戰書」+「動員令」+「宣傳材料」：

| 面向 | 說明 |
|------|------|
| **定義** | 《文心雕龍·檄移》說——「檄者——皦也——宣露於外——皦然明白也」——檄就是「明明白白地告訴大家」的意思——把敵人的罪行公之於眾——把我方的正義宣告於天下 |
| **功能** | 檄文有三大功能：第一——聲討敵人的罪行（讓對方失去道德正當性）；第二——宣示我方的正義（讓己方獲得道德號召力）；第三——動員天下人（號召更多人加入我方）。簡而言之——檄文是一場「道德戰爭」——在真正的軍事行動開始之前——先用文字打一場 |
| **寫作要求** | 好的檄文——必須做到——「事昭而理辨——氣盛而辭斷」——事實清楚——道理明白——氣勢充沛——語言果斷。它不需要客觀——不需要平衡——它只需要——讓人憤怒——讓人熱血沸騰——讓人願意拿起武器 |
| **經典檄文** | 中國歷史上的經典檄文——包括——漢代陳琳的〈爲袁紹檄豫州〉（罵曹操——說曹操的祖父是宦官——父親是乞丐——後來曹操讀了——頭風病都好了——說「這文章治好了我的病」——又是一個被罵的人讚賞罵人者的案例）——唐代駱賓王的〈爲徐敬業討武曌檄〉——以及近代的〈討粵匪檄〉（曾國藩討伐太平天國） |

### 5.2 這篇檄文為什麼如此有力？——四大修辭武器

駱賓王的這篇檄文——之所以能成為千古絕唱——是因為他同時使用了四種強大的修辭武器：

| 修辭武器 | 示例 | 分析 |
|---------|------|------|
| **1. 句句對仗——節奏力量** | 「南連百越，北盡三河」——「鐵騎成群，玉軸相接」——「言猶在耳，忠豈忘心」 | 駢文的對仗——產生了一種類似「戰鼓」的節奏感——三字句、四字句、六字句——交錯使用——時而急促——時而舒緩——讓讀者的情緒——隨著節奏起伏——最終推向高潮 |
| **2. 情感堆疊——層層加壓** | 從「性非和順」→「穢亂春宮」→「殘害忠良」→「殺姊屠兄」→「弒君鴆母」——一層比一層嚴重——最後達到「人神之所同嫉——天地之所不容」 | 這是一種「遞進式」的情感動員——不是一次性把所有的指控都倒出來——而是一層一層地加碼——每一層都讓憤怒增加一分——等到最後——讀者的情緒已經被推到了頂點——「天地之所不容」——這句話就不再是修辭——而成了一種「必然的結論」 |
| **3. 畫面感——讓人看到** | 「一抔之土未乾」——先帝陵墓的土還沒有乾——這個畫面——極其具體——極其可感——讓讀者彷彿看到了高宗的墳墓——看到了那個孤兒——站在墳前——無人保護 | 好的修辭——不是告訴人們「應該憤怒」——而是讓人們「看到讓人憤怒的畫面」。駱賓王不直接說「武則天對不起先帝」——他讓你看到高宗的墳——讓你看到那個孤兒——你自己——就會憤怒 |
| **4. 結尾的「選擇題」——讓人無法逃避** | 「試看今日之域中——竟是誰家之天下」 | 這是最厲害的一招——它不是一個陳述句——它是一個問題——它問每一個人——你覺得今天的天下——是誰的？如果你覺得是李家的——你就必須站出來——如果你不出來——你就是背叛了李家。這個問題——讓每個人——都無法保持沉默——都必須做出自己的選擇 |

> **軍師按**：駱賓王的檄文——之所以超越了一般的「罵人文」——是因為它不是簡單的辱罵——它是一個完整的「政治動員系統」——有「罪狀」——有「正義」——有「情感」——有「行動呼籲」——從頭到尾——一氣呵成——讓讀者從「理智上的同意」——到「情感上的共鳴」——到「行動上的響應」——層層遞進——不可阻擋。這就是為什麼——一千三百多年後——我們仍然在讀這篇文章——不是因為我們關心徐敬業——而是因為——我們在它裡面——看到了文字可以擁有的——最強大的力量。

### 5.3 「一抔之土未乾，六尺之孤何託」——為什麼這兩句是全文的靈魂？

這兩句——是全文的靈魂——也是中國文學史上最著名的修辭之一。為什麼？

| 面向 | 分析 |
|------|------|
| **意象的對比** | 「一抔之土」——一小捧土——象徵先帝的墳墓——象徵死亡——象徵過去——「六尺之孤」——一個身高六尺的孤兒——象徵年輕的中宗——象徵生命——象徵未來——死亡與生命的對比——過去與未來的對比——壓縮在兩句之中 |
| **情感的衝擊** | 「未乾」——兩個字——充滿了「來不及」的緊迫感——先帝剛死——黃土還沒有乾——你們就把他的兒子廢了——這種「來不及」——最容易激發人們的哀傷和憤怒——「何託」——兩個字——充滿了「無助感」——那個孤兒——誰來保護他？——這種「無助」——最容易激發人們的同情和保護慾 |
| **道德的力量** | 這兩句——把一個複雜的政治問題——簡化成了一個最簡單的道德問題——「你忍心看著一個孤兒——被一個惡毒的婦人欺負嗎？」——這個問題——不需要任何政治知識——任何一個有良心的人——都會回答「不忍心」——這就是道德的力量——它繞過了所有的政治分析——直接擊中了人們的良心 |
| **武則天的反應** | 武則天讀到這兩句時——「矍然而嘆」——吃驚地感嘆——說「宰相安得失此人」——這說明——連她——這個被罵的人——也無法抗拒這兩句話的情感力量——她不是在評價文章的好壞——她是在感嘆——「這樣的人才——怎麼沒有為我所用？」——一個被罵的人——對罵人者的最高讚美——就是「我應該把你拉攏過來」 |

> **軍師按**：「一抔之土未乾——六尺之孤何託」——之所以能成為千古名句——是因為它用最少的字——觸動了人類最深的情感——「對死者的懷念」——「對弱者的同情」——「對不公的憤怒」——這三種情感——被壓縮在十四個字裡——每一種都精準到位——每一種都不可抗拒。這就是文學最偉大的力量——它不是用邏輯說服你——它是用情感——直接擊中你的心——讓你在理智還沒有反應過來的時候——眼淚已經流下來了。

## 六、軍師核心分析——「文字可以成就什麼？」

### 6.1 這篇檄文改變了什麼？

這篇檄文——最終沒有改變歷史的走向——徐敬業起兵失敗了——武則天仍然在 690 年稱帝——建立了武周王朝。但——它改變了一些——更重要的東西：

| 面向 | 改變 |
|------|------|
| **中國文學史** | 它證明了——駢文——這種被認為只適合寫「宮廷應酬」「風花雪月」的文體——也可以用來寫最凌厲、最激昂、最有戰鬥力的政治文章——它打破了「駢文＝華而不實」的刻板印象——為駢文開闢了新的可能 |
| **檄文的藝術** | 它將「檄文」這種功能性文體——提升到了「文學經典」的高度。在〈爲徐敬業討武曌檄〉之前——檄文是一種「用完就扔」的實用文書——在這之後——檄文也可以成為「千古傳誦」的文學作品——它為後世的「文學性檄文」——樹立了最高的標杆 |
| **「失敗者文學」的傳統** | 駱賓王和徐敬業——是失敗者——他們在政治上輸了——但他們在文學上贏了。這種「失敗者的勝利」——開啟了中國文學史上一個重要的傳統——「失敗者文學」——那些在政治上失敗、在歷史上被遺忘的人——因為一篇詩文——而被永遠記住——這個傳統——從屈原——到駱賓王——到文天祥——一直延續到現代 |
| **對武則天的「間接影響」** | 雖然這篇檄文沒有推翻武則天——但它讓武則天意識到——反對她的人——有這樣的人才——有這樣的力量。她後來大肆提拔文學之士——大規模推廣科舉制度——某種程度上——也是為了把這樣的人才——都收歸己用——不再讓「駱賓王式」的人才——流落到敵對陣營——這是檄文對政治——最間接——但可能最深的影響 |

### 6.2 駱賓王與陳琳——兩個「罵人者」的不同命運

把駱賓王〈爲徐敬業討武曌檄〉——和陳琳〈爲袁紹檄豫州〉放在一起看——會發現一個有趣的對比：

| 面向 | 駱賓王 | 陳琳 |
|------|--------|------|
| **被罵的人** | 武則天——中國唯一的女皇帝 | 曹操——三國的梟雄 |
| **罵的內容** | 指責武則天「穢亂春宮」「殺姊屠兄」「弒君鴆母」 | 指責曹操的祖父是宦官——父親是乞丐——曹操本人「贅閹遺醜」 |
| **被罵者的反應** | 武則天讀後感嘆「宰相安得失此人」——欣賞才華——但沒有拉攏——因為駱賓王是敵人 | 曹操讀後——「頭風病豁然而愈」——事後不僅沒有殺陳琳——反而重用了他——因為陳琳後來歸順了曹操 |
| **罵人者的結局** | 徐敬業失敗——駱賓王下落不明——生死未卜 | 袁紹失敗——陳琳投降曹操——成為曹操的幕僚——善終 |
| **文學影響** | 駱賓王的檄文——比徐敬業的起兵更有名——成為千古絕唱 | 陳琳的檄文——也很有名——但影響力——略遜於駱賓王 |

| 面向 | 分析 |
|------|------|
| **相似之處** | 兩篇檄文——都是歷史上最著名的「罵人文」——而且——被罵的人（曹操和武則天）——都是雄才大略的統治者——他們都展現了對罵人者才華的欣賞——曹操說「這文章治好了我的病」——武則天說「宰相安得失此人」——這兩句話——成了中國文學史上——關於「才華的力量」——最經典的兩個案例 |
| **不同之處** | 最大的不同——是結局。陳琳後來投降了曹操——做了曹魏的官員——得以善終。駱賓王——徐敬業失敗後——他沒有投降——選擇了逃亡——結局成謎。陳琳的「投降」——讓他活下來了——但也讓他的人格——受到了一些質疑。駱賓王的「不投降」——讓他成了一個悲劇英雄——也更符合中國文化中「忠臣不事二主」的理想——但代價——是他的生命。這兩個選擇——沒有對錯——但反映了兩種不同的人生態度——「活下去」和「忠於信念」——哪一個更重要——每個人都有自己的答案 |

> **軍師按**：把駱賓王和陳琳放在一起看——會發現一個殘酷的事實：在政治鬥爭中——「才華」往往不如「立場」重要。陳琳的才華不如駱賓王嗎？未必。但陳琳投降了——所以他活下來了。駱賓王沒有投降——所以他消失了。但正因為他沒有投降——他的名字——和他「消失在歷史迷霧中」的結局——反而更增添了一層傳奇色彩——這或許是——命運對他——最後的公平。

### 6.3 「武則天讀檄」——一個象徵性的場景

「武則天讀檄」——這個場景——在中國文化中——已經超越了歷史事實——成為了一個「象徵」——一個關於「才華與氣度」的象徵：

| 層次 | 解讀 |
|------|------|
| **歷史的真實** | 《新唐書》的記載——「太后讀之——矍然而歎曰：『宰相安得失此人！』」——這個記載——可能經過了文學渲染——未必是百分之百的歷史事實——但它反映了唐代人對武則天和駱賓王的一種「集體記憶」——他們願意相信——武則天是這樣的反應——因為這符合他們對「女皇的氣度」和「文人的才華」的想像 |
| **象徵的意義** | 「武則天讀檄」——象徵的是「才華超越了政治立場」——一個人可以反對另一個人——但可以欣賞那個人的才華。武則天和駱賓王——是敵人——但武則天——作為一個懂文學的政治家——她能夠超越個人的憤怒——看到對手的才華——這是一種極其高貴的品質——也是一個統治者——最需要的品質之一 |
| **對後世的影響** | 「武則天讀檄」這個故事——成為後世「惜才」的典範——無論是韓愈的〈馬說〉——「世有伯樂——然後有千里馬」——還是蘇軾的「天下未嘗無才」——都可以在這個故事中找到源頭。武則天——一個被罵的人——卻能欣賞罵她的人的才華——這種「超越個人恩怨的格局」——是中國文化中——對「君王」的最高要求之一 |

> **軍師按**：「武則天讀檄」——比「徐敬業起兵」——對後世的影響——更加深遠。人們記住這篇檄文——不是因為徐敬業打了一場多麼漂亮的仗——而是因為——連被罵的武則天——都忍不住為它叫好。這種「敵人的讚美」——是對這篇文章——最高的評價。它告訴我們——在中國文化中——才華——最終可以超越立場——文字——最終可以超越政治——而一個真正有氣度的人——即使在被罵的時候——也能夠看到——罵他的人的——才華。

## 七、一句話總結（再錄）

> **〈爲徐敬業討武曌檄〉是中國歷史上最著名的檄文——沒有之一。它用駢文中最華美鏗鏘的語言——寫了一封給女皇帝的宣戰書——從「穢亂春宮」到「一抔之土未乾，六尺之孤何託」——從「南連百越，北盡三河」到「請看今日之域中，竟是誰家之天下」——每一句都是戰鼓——每一段都是號角。這篇文章的力量在於——連被罵的人（武則天）都忍不住為作者的才華鼓掌——感嘆「宰相安得失此人」。檄文沒有推倒武則天——徐敬業的起兵失敗了——但這篇文章——卻永遠地留在了中國文學史上——成為「文字的力量」最經典的證明——也成為「才華可以超越政治立場」——最震撼的案例。——因為——在政治的戰場上——你可以贏——也可以輸——但在文學的戰場上——一篇好文章——比一場勝仗——活得長久得多。徐敬業的軍隊——兩個月就灰飛煙滅了——但駱賓王的檄文——一千三百年後——我們還在讀。誰贏了？——文字贏了。**——**「試看今日之域中——竟是誰家之天下」——這不是一個問題——這是一個預言——它預言了——無論誰統治天下——最終——能夠統治人心的——永遠是文字——而不是刀劍。而駱賓王——用他最後的力氣——用他所有的才華——寫下的這篇檄文——就是對這個預言——最完美的證明。**——**從「一抔之土未乾」到「試看今日之域中」——駱賓王用長短不齊的駢文句式——模擬了一場戰爭的節奏——先是衝鋒（短句——急促）——然後是鋪開陣勢（長句——舒展）——最後是總攻（最激烈的號召）——這不是一篇「文章」——這是一場——用文字完成的——戰爭。而這場戰爭——雖然在現實中輸了——但在文學中——贏得徹徹底底。**

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷六·〈報任安書〉（司馬遷） | 司馬遷以「忍辱負重」完成史記——駱賓王以「慷慨赴死」寫下檄文——一個在屈辱中寫——一個在戰鬥中寫——都是「用文字來對抗命運」的極致案例 |
| 卷七·〈北山移文〉（孔稚珪） | 孔稚珪用「移文」罵假隱士——駱賓王用「檄文」罵武則天——兩篇都是中國諷刺文學的經典——一個諷刺個人品德——一個批判政治權力——文體相近——但格局不同 |
| 卷七·〈諫太宗十思疏〉（魏徵） | 魏徵以「諫疏」勸誡唐太宗要居安思危——駱賓王以「檄文」號召天下推翻武則天——一個是「勸諫」——一個是「討伐」——一個是「臣子對君王的話」——一個是「天下人對篡位者的宣戰」—反映的是唐朝從貞觀到武周——巨大的政治變化 |
| 卷九·〈送李愿歸盤谷序〉（韓愈） | 韓愈送友人歸隱——對官場的厭倦與批判——與駱賓王在檄文中對武則天朝廷的批判——在「對權力的態度」上——形成了有趣的對照 |
| 卷十一·〈潮州韓文公廟碑〉（蘇軾） | 蘇軾讚美韓愈——「文起八代之衰——而道濟天下之溺」——這種「用文字救天下」的精神——與駱賓王用一篇檄文——試圖改變天下格局——在精神上是相通的 |

### 8.2 經典延伸

- 《新唐書·駱賓王傳》（歐陽修、宋祁）——正史中關於駱賓王最完整的傳記——記錄了他七歲詠鵝的才華——以及參與徐敬業起兵、下落不明的結局——是了解駱賓王的第一手文獻
- 《新唐書·武則天本紀》（歐陽修、宋祁）——正史中關於武則天的官方記載——雖然帶著傳統史家的偏見——但仍然是了解武則天政治生涯最重要的文獻
- 《文心雕龍·檄移》（劉勰）——中國最早的文學批評專著中——專門討論了「檄」和「移」兩種文體的寫作要求——「事昭而理辨——氣盛而辭斷」——是理解檄文寫作理論的必讀文獻
- 《資治通鑑》卷二百三（司馬光）——司馬光在《資治通鑑》中——詳細記載了徐敬業起兵的經過——從揚州起兵到失敗被殺——可以與檄文對照閱讀——了解歷史與文學的差異
- 《唐才子傳·駱賓王》（辛文房）——唐代文人傳記——對駱賓王的一生有更具文學性的描寫——特別是他參與徐敬業起兵的細節
- 陳琳〈爲袁紹檄豫州〉——《昭明文選》收錄的經典檄文——與駱賓王的檄文並列——是中國檄文史上最著名的兩篇作品——曹操讀後「頭風豁然而愈」——與武則天讀後「宰相安得失此人」——形成了完美的對照

### 8.3 現代延伸

- 喬治·奧威爾《一九八四》——奧威爾對「語言與權力」的深刻剖析——特別是「雙重思想」對現實的扭曲——與駱賓王用檄文來揭露武則天「偽臨朝」的虛偽——在「用語言對抗權力」這個主題上——完全一致
- 米歇爾·福柯《話語與權力》——福柯認為——「話語即權力」——誰掌握了話語——誰就掌握了權力。駱賓王的檄文——就是在用「話語」——去挑戰武則天的「權力」——這是一場「話語的戰爭」——雖然駱賓王輸了——但他證明了——話語——確實有挑戰權力的力量
- 亞里斯多德《修辭學》——亞里斯多德提出的修辭三要素——「訴諸理性（Logos）」「訴諸情感（Pathos）」「訴諸人格（Ethos）」——在駱賓王的檄文中——全部得到完美運用——第一段對武則天的指控是「理性說服」（雖然是歪曲的）——「一抔之土未乾」是「情感訴求」——對徐敬業家世的歌頌是「人格背書」——這篇檄文——本身就是一部修辭學的完美教材
- 文森特·魯吉羅《超越感覺：批判性思考指南》——這本書教你如何識別「修辭操控」——讀駱賓王的檄文時——不妨同時運用批判性思維——問自己——駱賓王說的——哪些是事實——哪些是誇大——哪些是捏造——這是一個極好的「批判性閱讀」練習——因為——這篇檄文——既是文學經典——也是「修辭操控」的極致案例——學會欣賞它的美——同時保持清醒——才是最好的閱讀方式

## 九、背景補遺

### 9.1 「檄文」與「移文」的異同

| 文體 | 功能 | 對象 | 語氣 | 經典示例 |
|------|------|------|------|---------|
| **檄文** | 討伐——聲討敵人的罪行——宣戰——動員 | 敵人——叛亂者——侵略者 | 激烈凌厲——充滿戰鬥性——毫不留情——目的是激發仇恨和戰鬥意志 | 駱賓王〈爲徐敬業討武曌檄〉——陳琳〈爲袁紹檄豫州〉 |
| **移文** | 責備——通告——協商——諷刺 | 同級機關——個人——某種社會現象 | 相對溫和——但也有嚴厲的——可以用來諷刺——如〈北山移文〉是「用溫和的文體來進行最嚴厲的批評」 | 孔稚珪〈北山移文〉——官方往來的各種移文 |

**關鍵區別**：檄文的目標是「消滅敵人」——移文的目標是「責備或糾正」。所以檄文的語言——更激烈——更充滿殺氣——更「不共戴天」。駱賓王的檄文——就是「檄文」的極致——它不僅要罵武則天——它要讓所有人——站起來——拿起武器——去消滅她。

### 9.2 駱賓王的下落之謎

徐敬業兵敗後——駱賓王的下落——成了中國文學史上一個著名的「謎」：

| 說法 | 來源 | 內容 |
|------|------|------|
| **被殺說** | 《新唐書》《資治通鑑》等正史 | 徐敬業兵敗後——駱賓王被殺——首級被送往洛陽 |
| **逃亡說** | 唐代民間傳說——筆記小說 | 駱賓王沒有被殺——而是逃走了——隱姓埋名——出家為僧——雲遊四方 |
| **靈隱寺說** | 宋代《唐才子傳》等 | 相傳宋之問遊靈隱寺——遇到一個老僧——老僧為他續了兩句詩——「樓觀滄海日——門聽浙江潮」——宋之問後來才知道——這個老僧——就是駱賓王——這是中國文學史上最著名的「隱居結局」之一 |
| **浙江隱居說** | 地方志和民間傳說 | 駱賓王逃回浙江義烏老家——隱居終老——死後葬於當地 |

> **軍師按**：哪個說法是真的？——從歷史學的角度——「被殺說」——最接近事實——因為正史記載較為一致。但從文學的角度——「逃亡說」——尤其是「靈隱寺說」——是最美麗的結局——一個寫過「鵝鵝鵝」的七歲神童——一個寫過「試看今日之域中」的戰鬥文人——在兵敗之後——遁入空門——在靈隱寺的晨鐘暮鼓中——度過了餘生——這是一個——多麼適合駱賓王的結局。或許——真相已經不重要了——駱賓王的下落——本身就是一個「文學的謎」——而謎——比答案——更有魅力。

### 9.3 〈爲徐敬業討武曌檄〉的版本問題

這篇檄文的題目——在不同版本中——略有不同：

| 題目 | 出處 | 說明 |
|------|------|------|
| **〈代李敬業討武氏檄〉** | 《文苑英華》 | 徐敬業本姓徐——因祖父徐世勣被賜姓李——所以也稱「李敬業」——「武氏」指武則天——因為當時武則天尚未稱帝——以「武氏」稱之——符合身分 |
| **〈為徐敬業討武曌檄〉** | 《古文觀止》 | 「徐敬業」恢復本姓——因為唐代正統史書不承認武則天賜姓——「武曌」——「曌」是武則天自造的字——「日月當空」之意——是她稱帝後使用的名字——《古文觀止》使用這個題目——可能反映了後世的立場 |
| **〈討武曌檄〉** | 民間簡稱 | 最簡潔的稱呼——去掉「爲徐敬業」——直接聚焦於「討武曌」——但這樣會忽略徐敬業的角色 |

> **軍師按**：題目的變化——本身就反映了歷史立場的變化。用「李敬業」——呼應的是唐朝官方的賜姓。用「徐敬業」——暗示恢復本姓——帶有不承認武周政權的意味。用「武曌」——用武則天自造的字——強調她「僭越」的身份——而用「武氏」——則相對中性——只是稱呼她的姓氏。一個題目——幾個字的變動——背後是幾百年的政治和文化立場變化——這就是「版本學」的趣味所在。

### 9.4 武則天的「無字碑」——最後的沉默

武則天死後——與高宗合葬於乾陵（今陝西咸陽）——她的墓碑——是一座「無字碑」——上面沒有任何文字：

| 面向 | 說明 |
|------|------|
| **由來** | 武則天臨終遺命——「去帝號——稱則天大聖皇后」——與高宗合葬——墓碑上不留文字——讓後人評說 |
| **後世的傳說** | 民間傳說——武則天說——「我的功過——讓後人來評價」——所以立了無字碑。但也有人說——因為她的功過——實在太難以評說了——寫什麼都不合適——不如不寫 |
| **與駱賓王的關聯** | 非常諷刺——駱賓王用最華麗的文字——罵她——她卻用「無字」——來回應一切——文字的最高境界——是無字——武則天在最後——用「沉默」——戰勝了所有罵她的人——包括駱賓王——因為——無字碑——讓一切文字——都失去了意義——你可以罵她——但她選擇了——不回答 |

> **軍師按**：武則天的無字碑——和駱賓王的檄文——形成了中國歷史上最絕妙的「對位」。一邊是「用盡一切文字來罵」——一邊是「一個字都不留」。罵的人——用了所有他能用的——華美的、憤怒的、尖刻的、悲壯的文字——被罵的人——最後選擇了——什麼都不說。這兩種極端的選擇——一個是「文字的抗爭」——一個是「沉默的統治」——誰更高明？——也許——兩個都對——因為——駱賓王用文字——讓自己不朽——武則天用沉默——讓自己超脫——兩種方式——都達到了各自的目的。
