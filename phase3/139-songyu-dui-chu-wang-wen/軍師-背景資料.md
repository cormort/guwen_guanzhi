# 〈宋玉對楚王問〉背景資料

> **軍師分析 · 古文觀止卷四 · 楚辭·宋玉**

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《楚辭》相傳為宋玉所作（一說見於《新序·雜事》） |
| **選集** | 《古文觀止》卷四·楚辭類 |
| **體裁** | 對問體／辯護散文 |
| **時代** | 戰國末期（約西元前 3 世紀，楚頃襄王時期） |
| **核心命題** | 高處不勝寒——「曲高和寡」的經典出處 |
| **關鍵人物** | 宋玉（楚國辭賦家，屈原的繼承者）、楚頃襄王（提問者） |
| **文本重點** | 有人在楚王面前批評宋玉行為不檢點，楚王問宋玉。宋玉沒有直接為自己辯護，而用了三個比喻——有人在市集唱歌，唱《下里》《巴人》時數千人跟著唱，唱《陽春》《白雪》時只有幾十人跟著唱（「曲高和寡」出處）；鳥中有鳳凰，魚中有鯤魚，聖人與凡人的差距也一樣；所以不是我有問題，而是那些批評我的人不夠格理解我 |
| **思想譜系** | 出自《楚辭》，是中國文學史上最早的「自辯文本」之一。宋玉開創了一種獨特的辯護模式——不正面回應批評，而以比喻迂迴說明「被誤解是高人一等的證明」。這種「曲高和寡」的敘事，後來成為中國知識分子面對世俗批評時的標準回應策略，影響深遠：從魏晉名士的「越名教而任自然」到蘇東坡的「高處不勝寒」——後者更是直接化用了宋玉的文章意境 |
| **一句話總結** | **「曲高和寡」不僅是藝術的真理，也是知識分子的宿命——當你的境界越高，能理解你的人就越少，而這不是你的錯，是世界的常態。** |

## 二、時代背景——戰國末期的楚國與宋玉

### 2.1 楚國的衰落與傾襄王時代

戰國後期，楚國的國勢已從鼎盛急速下滑。西元前 299 年，楚懷王被秦昭王誘騙至武關會盟，遭秦國扣押，最終客死咸陽。懷王之子頃襄王（熊橫）即位後，楚國面對秦國步步進逼，國力日衰。

| 面向 | 說明 |
|------|------|
| **楚懷王被扣** | 懷王輕信秦國，入武關被扣留，楚國群龍無首，國勢急轉直下 |
| **頃襄王即位** | 懷王之子，在位期間（前 298—前 263 年）秦國不斷侵吞楚國領土——西元前 278 年白起攻破郢都，楚國被迫遷都陳（今河南淮陽） |
| **楚國文化氣候** | 儘管國勢衰落，楚國宮廷仍保留濃厚的辭賦傳統——屈原之後，宋玉、唐勒、景差等辭賦家活躍於頃襄王周圍 |
| **宋玉的地位** | 宋玉是屈原之後最著名的楚辭作家，但地位遠不如屈原崇高——他在宮廷中更像「文學侍從」或「辭賦弄臣」，常被同僚嫉妒和攻擊 |

> **軍師按**：理解〈宋玉對楚王問〉的關鍵，在於理解宋玉在楚國宮廷中的微妙處境。他繼承了屈原的辭賦傳統，但他不具備屈原的政治理想與勇氣——他是個文人，不是個政治家。有人批評他「行為不檢點」（「願勿與之出入後宮」或類似指控），他不像屈原那樣以死明志，而是用美麗的比喻來為自己開脫。這不是軟弱，這是一種屬於文人的生存智慧——你不是要說我不對嗎？我告訴你，那是因為你不懂我。

### 2.2 宋玉其人

宋玉是中國文學史上最神秘的作家之一——關於他的生平，史料極少，大部分來自他的作品和他人的記述。

| 面向 | 說明 |
|------|------|
| **身份** | 楚國辭賦家，活動於頃襄王時期。據傳為屈原的學生或後輩 |
| **作品** | 相傳為宋玉所作的作品包括〈九辯〉（收入《楚辭》）、〈風賦〉、〈高唐賦〉、〈神女賦〉、〈登徒子好色賦〉、〈對楚王問〉等 |
| **歷史定位** | 屈原之後最重要的楚辭作家——「屈宋」並稱，但宋玉的風格與屈原截然不同：屈原是熱烈奔放的理想主義者，宋玉是清醒冷靜的唯美主義者 |
| **文學貢獻** | 宋玉將楚辭從「政治抒情」轉向「宮廷辭賦」，開啟了漢賦的先河。後世司馬相如、揚雄等漢賦大家，皆受宋玉影響 |
| **人格特質** | 才華出眾但孤傲——他的作品反覆出現「懷才不遇」和「被誤解」的主題，說明他在宮廷中人緣不佳，常遭同僚讒言 |

> **軍師按**：關於宋玉是否真的是屈原的學生，學界並無定論。但從文學傳承來看，宋玉確實繼承了屈原的辭賦形式，卻完全拋棄了屈原的政治熱情——屈原的作品充滿了「雖九死其猶未悔」的悲壯激情，宋玉的作品則充滿了「高處不勝寒」的孤芳自賞。如果說屈原是用生命在寫作，那麼宋玉就是用才華在寫作——前者是烈士，後者是才子。

## 三、原文結構與核心比喻

### 3.1 文章的結構

〈宋玉對楚王問〉結構極為精簡，全文約三百字左右，分為「提問 → 三個比喻 → 結論」四個層次：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：楚王的質問** | 有人在楚王面前說宋玉的行為有問題（具體指控不詳，或謂其「好色」或「出入後宮」），楚王問宋玉：「先生其有遺行與？何士民眾庶不譽之甚也？」——你是否有品行上的瑕疵？為什麼這麼多人說你不好？ | 建立衝突——眾口鑠金，宋玉面臨輿論壓力 |
| **第二層：第一個比喻——曲高和寡** | 宋玉說：有一位外地的歌者在郢中唱歌——他開始唱《下里》《巴人》時，跟著唱的有數千人；唱《陽阿》《薤露》時，幾百人跟著唱；唱《陽春》《白雪》時，只有幾十人跟著唱。不是他越唱越差，而是曲調越高，能跟著唱的人就越少。最後他「引商刻羽，雜以流徵」——那是最高級的音樂，能跟著唱的只有幾個人了。 | 暗示：我不是行為有問題，而是我的層次太高，一般人無法理解我。批評我的人越多，越證明我超越大眾 |
| **第三層：第二個比喻——鳥有鳳而魚有鯤** | 宋玉接著說：鳥中有鳳凰，魚中有鯤魚。鳳凰可以飛上九千里高空，穿過雲霄，背負蒼天，翱翔於宇宙間——籬笆間的麻雀怎麼可能理解鳳凰的志向？鯤魚可以游入大海，橫渡汪洋——水塘中的小魚怎麼可能理解鯤魚的壯闊？ | 進一步強化「聖凡之別」——拿我與普通人比較，就像拿鳳凰與麻雀、鯤魚與泥鰍比較，根本不在一個層次上 |
| **第四層：結論——聖人與凡人的鴻溝** | 宋玉總結：「故非獨鳥有鳳而魚有鯤也，士亦有之。夫聖人瑰意琦行，超然獨處；夫世俗之民，又安知臣之所為哉？」——不只是鳥有鳳凰、魚有鯤魚，士人之中也有這樣的聖人。聖人有卓越的思想和非凡的行為，超然獨立於世俗之上；那些凡夫俗子，又怎麼可能理解我的所作所為呢？ | 直接點題——我沒有錯，錯的是那些批評我的人層次不夠。這種自信不是傲慢，而是來自對自己價值的確認 |

> **軍師按**：這篇文章最精彩的地方，是宋玉完全迴避了楚王的問題。楚王問的是「你是否有行為不檢點的地方？」——這是個事實問題（有或沒有）。但宋玉完全不回答事實，而是直接把對話拉到「層次問題」：你問我有沒有做錯事？我先告訴你——那些批評我的人根本不配評價我。這是一個高明的辯論技巧：當你無法在對方的戰場上獲勝時，就換一個戰場。宋玉換的這個戰場，讓他永遠立於不敗之地——因為沒有人可以反駁「你層次不夠」這個論點。

### 3.2 核心段落節錄

> 客有歌於郢中者，其始曰《下里》《巴人》，國中屬而和者數千人；其為《陽阿》《薤露》，國中屬而和者數百人；其為《陽春》《白雪》，國中屬而和者不過數十人；引商刻羽，雜以流徵，國中屬而和者不過數人而已。是其曲彌高，其和彌寡。

> ——「曲高和寡」的出處——這段話用音樂的層次來說明認知的差距：越是高深的藝術，能欣賞的人就越少。這既是音樂的真理，也是人生的真理。

> 故鳥有鳳而魚有鯤。鳳凰上擊九千里，絕雲霓，負蒼天，足亂浮雲，翱翔乎杳冥之上；夫籓籬之鷃，豈能與之料天地之高哉？

> ——第二個比喻——用鳳凰與麻雀的對比，說明「聖人」與「凡夫」之間的巨大差距。鳳凰飛上九千里高空，籬笆間的麻雀連想都不敢想。

> 夫聖人瑰意琦行，超然獨處；夫世俗之民，又安知臣之所為哉？

> ——全文的結論——直接點題：聖人的思想與行為超越世俗，凡夫俗子沒有資格評判我。

> ——《楚辭·宋玉》

## 四、關鍵人物

### 4.1 宋玉——孤高的才子

| 項目 | 內容 |
|------|------|
| **身份** | 楚國辭賦家，屈原的文學後繼者 |
| **核心貢獻** | 開創楚辭向辭賦的轉變，影響漢代大賦的形成 |
| **代表作品** | 〈九辯〉、〈風賦〉、〈高唐賦〉、〈神女賦〉、〈登徒子好色賦〉、〈對楚王問〉 |
| **人格特質** | 才華橫溢但孤傲不合群——他的作品充滿了「懷才不遇」的哀傷和「孤芳自賞」的自信 |
| **歷史定位** | 中國文學史上第一位「純文人」——不具備屈原的政治理想，純以辭賦才華立足於宮廷。他的存在本身就是一種「文人意識」的覺醒：我不需要是政治家，我只需要是文學家 |

**軍師評**：宋玉最了不起的地方，在於他發明了一種「被誤解的快樂」。一般人在被誤解時會委屈、會憤怒、會急於辯解。但宋玉不這樣——他用「曲高和寡」說服自己：被誤解不是我的損失，而是那些誤解我的人的損失。這種心態轉換，對於任何時代的知識分子來說，都是一種強大的心理防禦機制——當全世界都在批評你時，你可以告訴自己：不是我有問題，是我的層次太高。

### 4.2 楚頃襄王——提問者

| 項目 | 內容 |
|------|------|
| **身份** | 楚國國君，楚懷王之子，在位約 36 年（前 298—前 263 年） |
| **與本文的關係** | 他聽說宋玉的行為不檢點，當面質問宋玉——是本文的提問者 |
| **性格側寫** | 平庸之主——既無懷王的雄圖，也無屈原的忠誠骨幹。他在位時楚國不斷被秦國打壓，但他似乎更關心宮廷中的閒事（如宋玉的行為） |
| **歷史定位** | 楚國走向衰亡時期的君主——他的平庸，讓屈原的悲劇和楚國的衰落顯得更加令人惋惜 |

**軍師評**：楚頃襄王在這篇文章中只說了幾句話（甚至原文中他只問了一個問題），但他的角色極為重要——他是「世俗輿論」的人格化。他代表的是那個「永遠無法理解天才」的普羅大眾。而宋玉的回答，是對所有「不理解天才的普通人」的一次經典回擊。

## 五、核心分析——「曲高和寡」的修辭與心理

### 5.1 一個天才的辯護策略

〈宋玉對楚王問〉之所以成為千古名篇，不是因為它解決了問題，而是因為它發明了一種解決問題的方式——「昇華」。

| 策略 | 宋玉的做法 | 效果 |
|------|-----------|------|
| **不回答事實** | 宋玉完全不理會楚王的具體指控 | 避免陷入「有沒有做錯事」的事實辯論——一旦陷入那個戰場，無論他說什麼都是為自己辯解 |
| **重新定義問題** | 把「我有沒有錯？」變成「大眾有沒有資格評判我？」 | 把問題從事實層面拉到哲學層面——從「我做了什麼」變成「誰有資格說我」 |
| **用比喻取代論證** | 三個比喻層層遞進——唱歌、鳳凰、鯤魚 | 比喻的威力在於：你不必證明自己是對的，你只需要讓對方覺得「批評你的人確實層次太低」 |
| **把缺點變成優點** | 被批評不是因為我有問題，而是因為我太高級 | 這是心理上的驚天一擊——把「被討厭」從劣勢轉化為優勢 |

> **軍師按**：這篇文章展示了「辯論中的最高境界」——不是說服對方你是對的，而是讓對方覺得完全不值得跟你爭論。宋玉沒有回答楚王的問題，但他讓楚王無話可說。因為一旦楚王承認了「曲高和寡」這個邏輯，他就等於承認了自己和那些批評宋玉的人一樣——層次太低，聽不懂宋玉的「陽春白雪」。

### 5.2 「曲高和寡」的兩面性

「曲高和寡」這個成語，在中國文化中有兩層完全不同的含義：

| 層面 | 含義 | 應用場景 |
|------|------|----------|
| **正面（自省）** | 如果沒有人理解你，請反省自己——是否是因為自己的表達方式太晦澀？是否真的是曲太高，而不是人太俗？ | 用來提醒自己：不要把自己的孤獨當成高明，你的「陽春白雪」可能只是別人不願意聽的噪音 |
| **負面（自傲）** | 如果沒有人理解你，那一定是他們太笨——我的境界高，他們不懂我是正常的 | 用來安慰自己：保持清高，不要與庸俗為伍。宋玉在本文中的用法正是這一種 |

**軍師評**：這兩種解讀都沒有錯，關鍵在於你身處什麼位置、面對什麼困境。如果你是一個真正的天才，你的確需要「曲高和寡」的信念來抵禦世俗的壓力——李白的「天生我材必有用」、杜甫的「爾曹身與名俱滅，不廢江河萬古流」、蘇軾的「高處不勝寒」，都是這個信念的變奏。但如果你只是不夠努力或不夠接地氣，「曲高和寡」就會成為你拒絕成長的藉口。宋玉的這篇文章，提供了一個永恆的命題：當你被世界誤解時，你該如何與自己和解？

### 5.3 三個比喻的層層遞進

宋玉的三個比喻不是隨便排列的，它們形成了一個完美的遞進結構：

| 比喻 | 對象 | 差距 | 效果 |
|------|------|------|------|
| **唱歌（雅俗）** | 《下里》《巴人》→《陽春》《白雪》 | 從數千人到數人 | 雅俗之別——越高級的藝術，欣賞者越少 |
| **鳥（大小）** | 籬笆間的麻雀 vs. 翱翔九天的鳳凰 | 從地面到九千里高空 | 格局之別——麻雀只能看見籬笆，鳳凰看見的是蒼天 |
| **魚（深淺）** | 水塘中的小魚 vs. 橫渡汪洋的鯤魚 | 從淺塘到大海 | 境界之別——蠟燭只能照亮房間，太陽照亮的是整個世界 |

> **軍師按**：這三個比喻從「藝術鑑賞力」（唱歌）到「物理空間」（鳳凰和鯤魚），從文化到自然——宋玉用了一整個人類經驗的跨度來說明一件事：我和你們的差距，是全方位的——你們聽不懂我的音樂，看不見我的高度，游不進我的深度。當一個人能用三個完全不同的維度來證明你跟他的差距時，你還能說什麼呢？

## 六、軍師核心分析——「曲高和寡」的永恆命題

### 6.1 為什麼這篇文章到今天仍然動人？

〈宋玉對楚王問〉雖然只有三百字，但它觸及了一個人類永恆的困境：**當你的認知超越了你的環境時，你如何自處？**

| 時代 | 「曲高和寡」的表現 |
|------|-------------------|
| **戰國時代的宋玉** | 辭賦家在宮廷中被同僚嫉妒，被世俗批評——用「曲高和寡」來自我保護 |
| **魏晉時代的名士** | 嵇康、阮籍等竹林七賢——越名教而任自然，以「不屑與俗人為伍」的姿態對抗司馬氏的政治壓力 |
| **唐宋時代的文人** | 蘇軾被貶黃州——「揀盡寒枝不肯棲，寂寞沙洲冷」——高處不勝寒的意境 |
| **現代社會** | 在任何領域中，先鋒者的孤獨——科學家（哥白尼、伽利略）、藝術家（梵谷）、思想家（尼采）——無一不是「曲高和寡」的現代版 |

**軍師評**：宋玉的文章之所以穿越兩千年仍然讓人共鳴，不是因為它的文采，而是因為它說出了一個無人願意承認但人人都有體會的事實：**在這個世界上，懂得你的人永遠是少數。** 如果你追求真理、追求卓越、追求超越世俗的境界，你就必須接受一個殘酷的現實——你會在很長的時間裡、甚至一生，都是一個孤獨的人。宋玉沒有解決這個問題，但他提供了一個答案：與其委屈自己去適應世界，不如驕傲地守護自己的高度。

### 6.2 與其他篇目的互文

| 篇目 | 關聯點 |
|------|--------|
| 卷四·〈屈原列傳〉（司馬遷） | 屈原是宋玉的精神先驅——同樣被小人讒言、被楚王疏遠。但屈原選擇了「寧赴湘流，葬於江魚之腹中」，宋玉選擇了「超然獨處，笑對世俗」——一個是殉道者，一個是哲人 |
| 卷三·〈漁父〉（屈原） | 漁父對屈原說：「聖人不凝滯於物，而能與世推移」——勸屈原放下執著、與世浮沉。宋玉的「曲高和寡」在某種意義上就是漁父的態度——如果在這個世界上找不到知音，那就與自己和解，不要像屈原那樣以死明志 |
| 卷五·〈赤壁賦〉（蘇軾） | 蘇軾在赤壁之下面對人生無常，說「惟江上之清風，與山間之明月，耳得之而為聲，目遇之而成色」——這是一種與天地和解的態度，與宋玉的「超然獨處」一脈相承 |
| 卷五·〈岳陽樓記〉（范仲淹） | 「先天下之憂而憂，後天下之樂而樂」——范仲淹的入世情懷與宋玉的超然姿態形成對比：一個為天下憂，一個為自己守 |

> **軍師按**：宋玉與屈原的對比，是中國文學史上最深刻的一個對比——同樣被誤解、被讒言、被排擠，屈原選擇了「寧可玉碎，不為瓦全」，宋玉選擇了「我不屑於向你們解釋」。這兩種態度沒有高下之分——它們代表了面對不公的兩種終極可能：你要麼奮不顧身地抗爭到底（即使代價是死亡），要麼超然物外地保持高度（即使代價是孤獨）。而宋玉的〈對楚王問〉，就是後一種態度的最經典宣言。

### 6.3 對後世的深遠影響

「曲高和寡」這個成語出自本文，但它對後世的影響遠不止於此：

| 領域 | 影響 |
|------|------|
| **文學** | 宋玉開創的「對問體」——以問答形式進行的自我辯護——成為後世文人喜愛的文體。東方朔〈答客難〉、揚雄〈解嘲〉、韓愈〈進學解〉皆為此體 |
| **藝術** | 「陽春白雪」與「下里巴人」成為雅俗分野的代名詞——高雅藝術永遠面對「和者寡」的困境 |
| **人生哲學** | 「曲高和寡」成為中國知識分子面對世俗壓力時的心理防禦機制——後世文人喜歡用「髙處不勝寒」或「知音難覓」來表達自己的孤獨與自傲 |
| **知識分子的自我認同** | 宋玉提供了一個極為強烈的「知識分子身份認同」——我與普通人不同，我不需要被他們理解。這種認同在後世不斷被強化——從魏晉名士到唐宋文人，從明清遺民到現代知識分子，始終延續 |

## 七、一句話總結（再錄）

> **〈宋玉對楚王問〉是中國文學史上最美麗的自辯——它不說「我沒錯」，而是說「你沒有資格評判我」。宋玉用「曲高和寡」這個比喻，為後世所有被誤解的天才提供了一個退路：你的孤獨，恰恰是你超越常人的證明。兩千年來，無數人在被誤解時用這篇文章安慰自己——不是我有問題，是這個世界還沒準備好理解我。而這種自我安慰，有時候比真相還要重要。因為真相可能讓人絕望，但「曲高和寡」給人尊嚴。宋玉沒有讓楚王相信他是清白的——但他讓楚王相信了一件事：批評宋玉的人，都是層次不夠的人。這或許不是事實，但這是一場完美的辯論。**

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷三·〈漁父〉 | 同為楚辭中的對話體——漁父勸屈原「與世推移」，宋玉選擇「超然獨處」——兩篇共同探討「聖人如何在濁世自處」 |
| 卷四·〈屈原列傳〉（司馬遷） | 屈宋並稱——兩篇對照閱讀，可以理解中國最早的一對「師徒文學家」如何面對不公：一個以死抗爭，一個以文自救 |
| 卷五·〈赤壁賦〉（蘇軾） | 蘇軾在困境中的「物我兩忘」與宋玉的「超然獨處」——兩者都是面對孤獨的文人姿態 |
| 卷五·〈進學解〉（韓愈） | 韓愈的〈進學解〉直接繼承了宋玉「對問體」的形式——以自問自答的方式為自己辯護，成為古文運動中的經典 |

### 8.2 經典延伸

- 《楚辭·九辯》（宋玉）——宋玉最著名的作品，以秋天的悲涼抒發懷才不遇之苦，是〈對楚王問〉的精神先聲
- 《楚辭·漁父》（屈原）——屈原與漁父的對話，與宋玉的對問體形成絕佳對照
- 東方朔〈答客難〉——漢代對問體的代表作，繼承宋玉的辯護模式
- 揚雄〈解嘲〉——漢代對問體的進一步發展，以自嘲化解批評
- 韓愈〈進學解〉——唐代古文運動中的對問體傑作——韓愈以師生問答的方式為自己的「不合時宜」辯護

### 8.3 現代延伸

- 米哈里·契克森米哈賴《心流》——關於「專注與孤獨」的創造力心理學——宋玉在「曲高和寡」中所展現的心態，正是典型的心流狀態：當一個人完全沉浸在超越世俗的境界中時，外界的批評便無法干擾他
- 羅洛·梅《創造的勇氣》——關於藝術家與社會的衝突——宋玉的文章是對「創作者被誤解」這一永恆現象的經典回應
- 蘇珊·桑塔格《反對詮釋》——關於高雅藝術與大眾認知的鴻溝——宋玉兩千年前提出的「曲高和寡」，在現代藝術批評中依然是最核心的命題之一
