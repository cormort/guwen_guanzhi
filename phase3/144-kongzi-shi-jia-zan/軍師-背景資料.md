## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《史記·孔子世家》篇末「太史公曰」贊語 |
| **選集** | 《古文觀止》卷五·史記文類 |
| **體裁** | 史論體／人物贊論——世家贊 |
| **時代** | 西漢武帝時期（約西元前 91 年，《史記》成書前後） |
| **核心命題** | 至高無上的讚美——孔子作為「至聖」的歷史定位 |
| **關鍵人物** | 司馬遷（太史公，評論者）、孔子（至聖先師，被評論者） |
| **文本重點** | 司馬遷對孔子的讚美達到了整部《史記》的最高點——他到魯國看到孔子的遺跡，「祗回留之不能去」；讀孔子的書，「想見其為人」。他感嘆：「天下君王至於賢人眾矣，當時則榮，沒則已焉。孔子布衣，傳十餘世，學者宗之。自天子王侯，中國言六藝者折中於夫子，可謂至聖矣！」這是一篇飽含深情的讚辭——權力擁有的是一時的榮耀，而孔子擁有的是永恆的影響力 |
| **思想譜系** | 出自《史記》三十「世家」之一的〈孔子世家〉篇末。孔子在《史記》中被列入「世家」——這是諸侯的等級——然而孔子一生從未封侯，司馬遷卻以諸侯的體例為他立傳。這本身就是一個極具象徵意義的舉動：在司馬遷看來，孔子的歷史地位超越了一般的諸侯王。司馬遷引用《詩經》中的「高山仰止，景行行止」表達對孔子的敬仰——「雖不能至，然心鄉往之」——雖然我無法達到他的境界，但我心中永遠嚮往。這篇贊論收錄於《古文觀止》卷五，與〈項羽本紀贊〉、〈五帝本紀贊〉、〈秦楚之際月表〉等並列，是司馬遷最著名的「太史公曰」之一 |
| **一句話總結** | **司馬遷在〈孔子世家贊〉中做了整部《史記》中最驚人的論斷——那些擁有至高權力的君王，死後就被遺忘了；而一個沒有權力的布衣，卻讓後世所有的皇帝都必須向他學習。這是對「權力」與「影響力」最深刻的比較：權力只能統治一時，思想可以統治千年。** |

## 二、時代背景——司馬遷眼中的孔子

### 2.1 司馬遷與孔子的精神連結

司馬遷寫〈孔子世家〉時，孔子已去世約四百年。但司馬遷對孔子的感情，超越了時間與空間——他將孔子視為自己的精神導師和寫作靈感的來源。

| 面向 | 說明 |
|------|------|
| **司馬遷與孔子的時代差距** | 孔子生於西元前 551 年，卒於前 479 年；司馬遷生於約前 145 年，卒於前 86 年。兩人相距約四百年。但司馬遷對孔子的理解，比任何同時代的人都要深刻 |
| **司馬遷的遠遊與尋訪** | 司馬遷在二十歲時曾漫遊天下——「適魯，觀仲尼廟堂車服禮器」——他親自到魯國（今山東曲阜）瞻仰孔子的遺跡，看到孔子的廟堂、車服、禮器，看到「諸生以時習禮其家」——這是〈孔子世家贊〉中的第一手觀察，直接來自他的親身經歷 |
| **司馬遷的歷史使命感** | 司馬遷以繼承孔子《春秋》為己任。他在《太史公自序》中說：「先人有言：『自周公卒五百歲而有孔子。孔子卒後至於今五百歲，有能紹明世、正易傳、繼春秋、本詩書禮樂之際？』意在斯乎！意在斯乎！小子何敢讓焉。」他認為自己就是孔子之後五百年，那個繼承文化使命的人 |
| **孔子在《史記》中的特殊地位** | 《史記》三十「世家」中，孔子是唯一一個生前沒有諸侯爵位卻被列入世家的人。司馬遷為此在〈孔子世家〉最後的贊語中給出了自己的理由——孔子的影響力超越了任何諸侯王，他雖然是「布衣」，但「學者宗之」 |

> **軍師按**：司馬遷對孔子的感情，不僅僅是一個史家對歷史人物的尊重——它更接近於一個弟子對師長的敬愛。司馬遷在〈孔子世家贊〉中說「余讀孔氏書，想見其為人」——這句話充滿了情感。他讀孔子的書，不是為了做學問，而是為了「見」到孔子這個人——他想穿過文字，觸摸到那個在兩百年前就去世了的人的心靈。這種「與古人神交」的能力，正是司馬遷成為最偉大史家的關鍵——他不只是記錄歷史人物，他是與歷史人物對話。

### 2.2 漢武帝時代的儒學與孔子

司馬遷寫〈孔子世家〉的時代，正是漢武帝「獨尊儒術」的時代。儒學在漢初經過叔孫通、董仲舒等人的改造，從先秦的百家之一上升為官方意識形態。

| 面向 | 說明 |
|------|------|
| **獨尊儒術** | 漢武帝採納董仲舒「罷黜百家，獨尊儒術」的建議（約西元前 134 年），儒學成為官方正統思想 |,
| **孔子的官方化** | 孔子從一個布衣學者，被漢帝國包裝成「素王」（沒有王位的王者）。漢武帝時代開始對孔子的後代進行封賞（封孔安國為大夫），孔子的地位在官方層面上不斷提升 |
| **司馬遷的獨立立場** | 但司馬遷對孔子的讚美，與漢武帝的官方宣揚不同——司馬遷讚美的不是「被權力馴化的孔子」，而是那個「在絕境中仍然堅持理想的孔子」。司馬遷在〈孔子世家〉中詳細記錄了孔子周遊列國的困頓——「累累若喪家之狗」——但這正是司馬遷最敬佩孔子之處：他可以在最艱難的時候仍然不放棄 |
| **司馬遷與官方儒學的微妙關係** | 司馬遷對儒學的態度比官方複雜得多。他在《史記·太史公自序》中將儒家與陰陽、墨、法、名、道並列為「六家」，並分別評其優劣——他不是一個純粹的儒者，但他對孔子本人的尊敬是毫無保留的 |

> **軍師按**：司馬遷對孔子的讚美，在漢武帝「獨尊儒術」的時代背景下，具有特殊的意義。一方面，他的讚美符合官方的意識形態——讚美孔子不會觸怒權力；另一方面，司馬遷對孔子的理解超越了官方儒學的表面——他看到的孔子不是一個完美的聖人像，而是一個在現實中屢屢碰壁、但仍然堅持理想的真人。這與他在〈項羽本紀贊〉中對失敗者的同情，是同一種史學態度——他關注的不是成功者，而是那些在困境中仍然堅持自己信念的人。

## 三、原文結構與關鍵評語

### 3.1 〈孔子世家贊〉的結構

全文極為短小精悍（約兩百字），但結構完整——從個人感受出發，到歷史比較，再到最終的讚美：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：個人感觸——讀其書，想見其人**（開頭） | 司馬遷引用《詩經》「高山仰止，景行行止」——表示自己對孔子的敬仰之情。他說「雖不能至，然心鄉往之」——雖然我達不到他的境界，但我心中嚮往。接著敘述自己「讀孔氏書，想見其為人」——讀孔子的書，就彷彿見到了孔子本人。最後說自己到魯國時，「觀仲尼廟堂車服禮器，諸生以時習禮其家」，他「祗回留之不能去」——徘徊不忍離去 | 從個人體驗出發——這不是一個冷漠的史家在寫一篇客觀的傳記，而是一個被感動的人在他所崇敬的人面前表達自己的心聲 |
| **第二層：歷史比較——權力的虛幻與思想的永恆**（中間） | 「天下君王至於賢人眾矣，當時則榮，沒則已焉。孔子布衣，傳十餘世，學者宗之。」——天下那些君王和賢人太多了，他們活著的時候榮耀，死後就煙消雲散了。而孔子只是一個「布衣」（平民），但他的學說流傳了十幾代，學者都以他為宗師 | 核心對比——君王擁有的權力是暫時的，孔子擁有的影響力是永恆的。這是司馬遷對「何者為不朽」的終極回答 |
| **第三層：蓋棺論定——「可謂至聖矣」**（結尾） | 「自天子王侯，中國言六藝者折中於夫子，可謂至聖矣！」——從天子王侯，到全天下研究六藝的人，都以孔子的學說為判斷是非的標準——孔子可以稱為「至聖」了！ | 最終定論——「至聖」是司馬遷對一個人的最高評價，在整部《史記》中，他只給了孔子這個稱號 |

> **軍師按**：這篇贊論最動人之處在於它的「個人性」。司馬遷沒有用官方口吻來讚美孔子——他沒有說「孔子是聖人，萬世師表」——他說的是一個具體的人的真實感受：「我讀孔子的書，好像見到他本人一樣」；「我到魯國，看到他的遺跡，捨不得離開」。這種發自內心的敬仰，比任何官方的讚美都更有力量——因為它是真實的。

### 3.2 核心段落節錄

> 太史公曰：《詩》有之：「高山仰止，景行行止。」雖不能至，然心鄉往之。余讀孔氏書，想見其為人。

> ——開篇第一段話就奠定了全文的基調——這不是一篇冷靜的評論，而是一篇充滿情感的禮讚。司馬遷借用《詩經·小雅·車舝》中的「高山仰止，景行行止」——高山人人仰望，大路人人都走——來比喻孔子的人格和學問的崇高。「想見其為人」四個字，寫出了一個史家對歷史人物最深沉的敬意——他不是在研究孔子的思想，而是想「見到」孔子這個人

> 適魯，觀仲尼廟堂車服禮器，諸生以時習禮其家。余祗回留之，不能去云。

> ——司馬遷在魯國親眼看到了孔子的遺跡——廟堂、車服、禮器——以及孔子的弟子們「以時習禮其家」——按時在孔子的家中演習禮儀。他看到這一切，「祗回留之不能去」——徘徊流連，不忍離去。這是一個極為生動的畫面：一個兩千年來最偉大的史家，站在一個更偉大的人的故居前，不願離開。這段話中的情感之濃烈，在整部《史記》中都極為罕見

> 天下君王至於賢人眾矣，當時則榮，沒則已焉。孔子布衣，傳十餘世，學者宗之。自天子王侯，中國言六藝者折中於夫子，可謂至聖矣！

> ——這是全文最核心、最有力的一段話，也是整部《史記》中最著名的論斷之一。司馬遷將「君王」與「布衣」進行了尖銳的對比——天下君王那麼多，活著的時候榮耀，死了就被人遺忘了；而孔子只是一個平民，他的學說卻流傳了十幾代，學者都以他為宗師。從天子到王侯，全天下議論六藝（儒家的六部經典）的人，都以孔子的話為判斷標準——這就是「至聖」！

> ——《史記·孔子世家》

## 四、關鍵人物

### 4.1 孔子——布衣之中，出至聖

| 項目 | 內容 |
|------|------|
| **身份** | 名丘，字仲尼，魯國陬邑（今山東曲阜）人。春秋末期思想家、教育家、政治家，儒家學派創始人 |
| **生平核心** | 早年做過管理倉庫和畜牧的小官；三十歲左右開始講學；五十一歲出任魯國中都宰，後升任大司寇；五十五歲開始周遊列國十四年——到處推銷自己的政治理想，卻處處碰壁；六十八歲返回魯國，專心整理六經（《詩》《書》《禮》《樂》《易》《春秋》）；西元前 479 年去世，享年七十三歲 |
| **核心思想** | 「仁」——「己所不欲，勿施於人」；「禮」——「克己復禮為仁」；「正名」——「君君、臣臣、父父、子子」；「有教無類」——不分貴賤，人人可以受教育 |
| **生前處境** | 孔子在世時，他的政治理想幾乎完全落空——沒有哪個諸侯國真正採納了他的主張。他甚至被嘲笑為「累累若喪家之狗」——像一隻無家可歸的流浪狗。但他在最困頓的時候仍然堅持——「君子固窮，小人窮斯濫矣」 |
| **死後的影響力** | 孔子死後，他的學說逐漸擴散——戰國時孟子繼承並發揚儒家，漢代董仲舒將儒學推向官方意識形態。從漢代到清代，孔子的地位不斷上升——從「先師」到「至聖先師」，再到「大成至聖文宣王」。而這一切的起點，就是一個生前處處碰壁的布衣 |

**軍師評**：孔子一生最大的悖論在於——他生前幾乎是一個「失敗者」。他夢想恢復周禮，重建一個有秩序的社會，但春秋時代禮崩樂壞的趨勢不是一個人的努力可以逆轉的。他周遊列國十四年，幾乎沒有得到任何一個諸侯的真正重用。他的弟子們都為他感到難過——但他們不知道，他們的老師正在創造中國歷史上下兩千年最偉大的精神遺產。司馬遷在〈孔子世家贊〉中看到了這個悖論——這就是為什麼他把「天下君王」與「孔子布衣」放在一起比較——君王擁有的權力是「當時則榮」，孔子擁有的影響力是「傳十餘世」。什麼是真實的「成」與「敗」？這是一個需要兩千年來回答的問題——而司馬遷提前給出了答案。

### 4.2 司馬遷——孔子的知音與繼承者

| 項目 | 內容 |
|------|------|
| **身份** | 西漢太史令，《史記》作者——中國最偉大的歷史學家 |
| **與孔子的關係** | 司馬遷以繼承孔子《春秋》為己任。他自己說過：「自周公卒五百歲而有孔子。孔子卒後至於今五百歲……意在斯乎！」他把自己當作孔子之後五百年，那個繼承孔子文化使命的人 |
| **寫作態度** | 司馬遷對孔子的感情，是所有《史記》人物中最深沉的——他對項羽是「讚嘆與惋惜」，對屈原是「同情與共鳴」，但對孔子是「敬畏與嚮往」——他把孔子當成自己永遠無法達到、但永遠嚮往的精神目標 |
| **〈孔子世家〉的特殊地位** | 在《史記》一百三十篇中，〈孔子世家〉是極特殊的一篇——司馬遷用了極長的篇幅來寫孔子（全文約一萬字），遠遠超過了一般世家的長度。更重要的是，他將孔子置於世家之中，與諸侯王同列——這是他對孔子的最高禮遇 |
| **精神共鳴** | 司馬遷與孔子有一個根本的共同點——兩人都經歷了人生的巨大挫敗。孔子周遊列國處處碰壁，司馬遷因李陵之禍受宮刑——兩人都從困境中走出了超越自身苦難的道路：孔子整理了六經，司馬遷完成了《史記》。這種「苦難中見精神」的共鳴，是司馬遷能寫出〈孔子世家贊〉如此深情文字的根本原因 |

**軍師評**：司馬遷讚美孔子，其實在某種程度上是在讚美他自己選擇的道路。孔子在絕望中仍然堅持整理古籍、傳授弟子——因為他知道有些東西比當下的成敗更重要。司馬遷在遭受宮刑後仍然選擇活下去、完成《史記》——因為他也相信有些東西比個人的尊嚴更重要。〈孔子世家贊〉中那句「雖不能至，然心鄉往之」——雖然我達不到孔子的境界，但我心中嚮往——或許也是司馬遷對自己說的：我可能永遠無法成為像孔子那樣的人，但我可以在我的時代，做孔子做過的事——把文化傳下去。

## 五、核心分析——「至聖」的定義：權力 vs. 影響力

### 5.1 司馬遷的最大膽論斷

〈孔子世家贊〉中最驚人的一句話是：「天下君王至於賢人眾矣，當時則榮，沒則已焉。孔子布衣，傳十餘世，學者宗之。」

司馬遷這句話的批判對象，不只是春秋戰國的那些諸侯——在某種程度上，它也指向了漢武帝時代的權力者：皇帝們擁有權力，看起來至高無上，但他們死後會被人遺忘；而一個布衣學者，卻能讓後世所有的皇帝都必須向他學習。

| 比較維度 | 天下君王 | 孔子 |
|---------|---------|------|
| **出身** | 諸侯、王者——擁有血統、權力、軍隊 | 布衣——沒有任何權力資源 |
| **生前** | 「當時則榮」——權力帶來短暫的榮耀 | 「累累若喪家之狗」——處處碰壁 |
| **死後** | 「沒則已焉」——隨著死亡，一切歸於虛無 | 「傳十餘世，學者宗之」——影響力跨越了時間 |
| **影響範圍** | 局限於自己的時代和領地 | 「自天子王侯」——連皇帝都要向他學習 |
| **權力類型** | 強制性權力（靠武力維持） | 精神性權力（靠思想傳承） |
| **永恆性** | 短暫——一代人的榮耀 | 永恆——兩千年不斷被學習和傳承 |

> **軍師按**：這不是一篇普通的致敬文章——它是一篇有著鋒利批判力量的歷史哲學論文。司馬遷用一個尖銳的對比質問了所有當權者：你的權力能持續多久？你死了以後，誰還記得你？與此同時，他也給出了一個充滿希望的答案：世界上存在著一種比權力更強大的力量——那就是思想的力量。孔子沒有軍隊、沒有領土、沒有金錢——但他有學問、有弟子、有寫下來的書——而這就是以讓後世所有的皇帝都低頭。

### 5.2 「至聖」的標準——為什麼是孔子？

司馬遷稱孔子為「至聖」——「至」是「極致」的意思，「至聖」就是「聖人中的聖人」。司馬遷為什麼把這個稱號給了孔子？

| 標準 | 孔子的符合度 | 說明 |
|------|-------------|------|
| **學術成就** | ✅ | 整理六經——《詩》《書》《禮》《樂》《易》《春秋》。司馬遷說「中國言六藝者折中於夫子」——所有人談論六經，都以孔子的解釋為標準 |
| **教育貢獻** | ✅ | 有教無類——孔子打破了「學在官府」的傳統，開創了私人講學的風氣。他的弟子來自各國、各階層——這就是為什麼「學者宗之」——後世的學者都是他的徒子徒孫 |
| **思想影響** | ✅ | 儒家思想成為中國兩千年政治、社會、文化的核心——從漢代的「獨尊儒術」到唐宋的科舉制度，從日常的人際關係到國家的治理哲學 |
| **道德感召** | ✅ | 孔子的人格魅力——「高山仰止」——使後人不斷向他學習。即使在他生前最困頓的時候，他的弟子們仍然緊緊跟隨他 |
| **歷史意義** | ✅ | 孔子是中國歷史上第一位系統性整理古代文獻的人——沒有孔子，我們對上古歷史的了解將大大減少。某種程度上，司馬遷能寫出《史記》，也離不開孔子打下的學術基礎 |

**軍師評**：司馬遷稱孔子為「至聖」，不是因為孔子是完美無缺的聖人——事實上，司馬遷在〈孔子世家〉的正文中忠實記錄了孔子的缺點和困頓（包括他對弟子的發脾氣、他在政治上的天真、以及他被世人嘲笑為「喪家之狗」的狼狽）。但正是因為孔子不是一個「完美的聖人」，他的「至聖」地位才更加真實——他是一個有血有肉的人，他曾經失敗過、痛苦過、迷茫過——但他從未放棄過。這種「在困境中仍然堅持」的精神，才是司馬遷最敬佩的——因為這正是他自己的人生寫照。

### 5.3 與其他《史記》贊論的對照

〈孔子世家贊〉與〈項羽本紀贊〉、〈五帝本紀贊〉、〈秦楚之際月表〉等篇可以互相對照：

| 篇目 | 核心論斷 | 與孔子世家贊的關係 |
|------|---------|-------------------|
| 〈項羽本紀贊〉 | 「豈不謬哉」——項羽的失敗是他自己的錯 | 項羽代表了「權力的悲劇」——擁有最強的武力，卻無法守住勝利。孔子代表了「思想的勝利」——沒有任何權力，卻擁有永恆的影響力 |
| 〈五帝本紀贊〉 | 「信以傳信，疑以傳疑」——史學方法論 | 五帝是傳說中的聖王，孔子是真實世界中的聖人。司馬遷對五帝「考而後信」，對孔子則是「讀其書，想見其人」——前者是理性的考證，後者是情感的嚮往 |
| 〈秦楚之際月表〉 | 「王跡之興，起於閭巷」——平民也能當皇帝 | 劉邦從亭長變成了皇帝，孔子從布衣變成了至聖——兩者都在證明：出身不重要，重要的是你的作為。但劉邦靠的是權力，孔子靠的是思想 |

> **軍師按**：如果把〈項羽本紀贊〉、〈秦楚之際月表〉和〈孔子世家贊〉放在一起讀，可以看到司馬遷對「權力」與「真正的不朽」的完整思考。項羽代表「權力的悲劇」——你有最強的權力，但你不會用，結果就是灰飛煙滅。劉邦代表「權力的喜劇」——你出身卑微，但你會用權力，結果你成了皇帝。孔子代表了第三條路——「超越權力的不朽」——你不是靠權力，而是靠思想來影響世界。這三篇讚論合在一起，就是司馬遷對「人該如何活著」的三種答案——而他的選擇是第三種。

## 六、軍師核心分析——思想的力量大於權力的力量

### 6.1 為什麼「至聖」是一個顛覆性的概念？

在漢武帝的時代，「君權神授」是官方意識形態——皇帝的權力來自上天，是不可挑戰的。但在〈孔子世家贊〉中，司馬遷做了一個極為大膽的論述：權力最高的「君王」死了就被人忘記了，而沒有權力的「布衣」卻讓後世所有人——包括那些皇帝——必須向他學習。

這個論述的顛覆性在於：

| 層面 | 傳統觀念 | 司馬遷的挑戰 |
|------|---------|-------------|
| **權力來源** | 權力來自血統和天命——皇帝是天之子 | 權力的來源不是天命——孔子的影響力不是上天給的，是他自己掙來的 |
| **誰是真正的「聖」** | 聖人應該是堯舜禹湯這樣的帝王 | 真正的聖人不是帝王，而是一個推廣教育的平民學者 |
| **什麼是「不朽」** | 建立功業、統一天下才是不朽 | 思想傳播、文化傳承才是不朽——比權力更永恆 |
| **誰說了算** | 權力決定真理——皇帝說的話就是法律 | 真理決定權力——連皇帝都要以孔子的話為標準 |

> **軍師按**：這個論斷在漢武帝時代是大膽的，但它不是對皇權的直接挑戰——因為孔子已經是官方承認的聖人。司馬遷聰明的地方在於，他利用了官方儒學的話語空間，做了一個「合法」但「鋒利」的論證：如果你們說孔子是聖人，那麼你們就必須承認——思想的力量高於權力的力量。這個結論的邏輯是無懈可擊的——但它的政治含義，在每一個時代都讓人深思。

### 6.2 司馬遷對孔子態度的雙重性

司馬遷對孔子的態度有一個有趣的雙重性——一方面，他對孔子的人格充滿敬畏和嚮往；另一方面，他對孔子留下的學術遺產保持著獨立思考：

| 面向 | 司馬遷的態度 | 體現 |
|------|-------------|------|
| **對孔子的人格** | 無限敬仰——「雖不能至，然心鄉往之」 | 這是一個讀者對作者的愛——司馬遷讀孔子的書，就想見孔子這個人。他到魯國的孔子故居，徘徊不捨得離開 |
| **對儒家的學術** | 獨立批判——不是盲目尊崇 | 在《史記·太史公自序》中，司馬遷批評儒家「博而寡要，勞而少功」——他尊敬孔子個人，但他不認為儒家學說完美無缺 |
| **對孔子的歷史評價** | 極高定位——「至聖」 | 用「至聖」來稱呼孔子，是司馬遷在整部《史記》中給出的最高評價——他沒有用這個詞稱呼任何人，包括黃帝、堯、舜 |
| **對孔子的人生選擇** | 深刻共鳴——「知其不可而為之」 | 司馬遷在〈孔子世家〉中詳細記錄了孔子周遊列國的困頓，但從中提煉出一個核心精神：即使知道做不到，也要去做——這也是司馬遷自己選擇的人生態度 |

**軍師評**：司馬遷對孔子的態度，可以總結為「敬其人而不從其學」——他尊敬孔子這個人的人格和精神，但他不盲從儒家學說。這種態度在後世被很多知識分子繼承：你可以不認同儒家的一切主張，但你無法否認孔子的偉大——因為他的偉大不在於他說了什麼，而在於他是一個「知其不可而為之」的人。這種「知其不可而為之」的精神，是中國知識分子最寶貴的遺產之一。

### 6.3 對後世的深遠影響

〈孔子世家贊〉對中國文化和歷史的影響不可估量：

| 領域 | 影響 |
|------|------|
| **史學** | 「太史公曰」體例——本篇是《史記》世家中最著名的贊語之一。後世史家在為人物作傳後寫贊語時，經常引用司馬遷對孔子的評價作為典範 |
| **儒學地位** | 司馬遷稱孔子為「至聖」，是中國歷史中最早以「至聖」稱孔子的文獻之一——這個稱號後來被歷代王朝沿用，最終成為孔子的標準稱號：「大成至聖先師」 |
| **文學** | 本篇贊語中「高山仰止，景行行止」的引用，以及「雖不能至，然心鄉往之」的表達——成為後世讚美他人時最常被引用的經典句式 |
| **教育理念** | 司馬遷對「學者宗之」的強調——強調教育與傳承的重要性——影響了後世對「師道尊嚴」的理解。孔子是「萬世師表」——這個定位與司馬遷的評價密不可分 |
| **政治哲學** | 「天下君王至於賢人眾矣，當時則榮，沒則已焉」這句話——成為後世對「權力的虛幻」最經典的表達之一。它提醒每一個掌權的人：你的權力只是一時的，真正持久的力量來自於你為這個世界留下了什麼 |

## 七、一句話總結（再錄）

> **〈孔子世家贊〉是司馬遷對孔子最深沉的致敬，也是《史記》中最溫暖、最動人的一篇贊語。司馬遷用「高山仰止，景行行止」開篇——他說自己讀孔子的書，就想見孔子這個人；他到魯國看到孔子的遺跡，徘徊不忍離去。然後他做了一個整部《史記》中最驚人的論斷：那些擁有至高權力的君王死了就被人遺忘了，而一個沒有權力的布衣——孔子——卻讓後世所有的學者、王侯、甚至天子都必須以他的話為標準。他稱孔子為「至聖」——這是司馬遷對一個人的最高評價。兩千年後，這個論斷仍然成立：那些曾經統治中國的皇帝們，大多數人的名字已經被遺忘；而那個被嘲笑為「喪家之狗」的布衣，他的生日（孔子誕辰）至今仍然被紀念，他的話語至今仍然被傳誦。這是思想的力量，這是文化的力量——這是司馬遷在兩千年前就告訴我們的真相：真正的不朽，不屬於權力，而屬於那些為世界留下了精神財富的人。**

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈項羽本紀贊〉（司馬遷） | 司馬遷對兩種「失敗者」的不同態度——項羽是「有權力但不會用」的失敗，孔子是「無權力卻不朽」的成功。兩篇對照，可以看到司馬遷對「什麼才是真正的不朽」的完整思考 |
| 卷五·〈五帝本紀贊〉（司馬遷） | 五帝贊講的是史學方法——如何從殘缺的史料中重建上古歷史；孔子世家贊講的是歷史評價——如何對一個偉大人物做出最終判斷。兩篇合在一起，是司馬遷史學「方法」與「價值」的完整體現 |
| 卷五·〈秦楚之際月表〉（司馬遷） | 「王跡之興，起於閭巷」——劉邦從平民到皇帝，孔子從布衣到至聖——兩者都在證明出身不重要，但劉邦靠的是權力，孔子靠的是思想 |
| 卷四·〈屈原列傳〉（司馬遷） | 同為司馬遷筆下的悲劇人物——屈原和孔子都經歷了人生的巨大挫折。但屈原選擇了投江，孔子選擇了繼續教書。兩者的不同選擇，展示了中國知識分子面對困境的兩種路徑 |
| 卷三·〈漁父〉（屈原） | 漁父問屈原：「安能以身之察察，受物之汶汶者乎？」——〈孔子世家贊〉中的孔子給出了另一種回答：即使世界不完美，也要堅持教書育人、整理文獻——因為文化比政治更持久 |

### 8.2 經典延伸

- 《史記·孔子世家》（司馬遷）——本文的出處，是了解孔子一生最重要的文獻。司馬遷以極其生動的筆法記述了孔子從出生到死亡的完整故事——包括他周遊列國的困頓、與弟子的互動、以及面對困境時的態度
- 《史記·仲尼弟子列傳》（司馬遷）——孔子弟子的傳記——司馬遷記錄了七十多位弟子的生平，是了解孔子教育思想的重要材料
- 《史記·太史公自序》（司馬遷）——司馬遷對自己寫作《史記》的完整說明，其中明確表達了他以繼承孔子《春秋》為使命的態度
- 《論語》（孔子弟子輯錄）——孔子思想的第一手文獻——司馬遷撰寫〈孔子世家〉時最重要的史料來源
- 《孟子·滕文公下》（孟子）——孟子對孔子的評價——「孔子成《春秋》而亂臣賊子懼」——與司馬遷對孔子的評價互相呼應
- 《報任安書》（司馬遷）——司馬遷自己的生命宣言——「人固有一死，死有重於泰山，或輕於鴻毛」——這封信與〈孔子世家贊〉共同展示了司馬遷的核心價值觀：有些東西比個人的生命和尊嚴更重要

### 8.3 現代延伸

- 錢穆《孔子傳》——現代史學家錢穆對孔子的系統性研究，結合史料與考證，還原了一個真實的孔子形象
- 李長之《司馬遷的人格與風格》——對司馬遷性格與寫作風格的深入分析——提到司馬遷對孔子的崇拜與傳承關係
- 李零《喪家狗：我讀〈論語〉》——以「喪家狗」這個孔子自嘲的形象為切入點——還原孔子作為一個真實的人的樣貌，與司馬遷〈孔子世家〉中的孔子形象互相發明
- 韓兆琦《史記箋證》——對《史記》最詳細的現代注釋，其中對〈孔子世家贊〉的分析極具參考價值
- 余英時《論天人之際：中國古代思想起源試探》——從思想史的角度分析孔子的歷史地位——與司馬遷在〈孔子世家贊〉中的評價互相參照
