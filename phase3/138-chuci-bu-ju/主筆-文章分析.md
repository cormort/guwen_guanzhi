## 1. 文章結構分析

**第一段：屈原被放——三年不得復見**

> 「屈原既放，三年不得復見。竭智盡忠，而蔽障於讒。心煩慮亂，不知所從。」

屈原被流放三年——忠心耿耿卻被讒言阻隔。他心煩意亂，不知該何去何從。

**第二段：往見太卜鄭詹尹——請決於疑**

> 「乃往見太卜鄭詹尹曰：『余有所疑，願因先生決之。』」

屈原去找太卜（占卜官）鄭詹尹——我有疑惑，請先生幫我決斷。

**第三段：八個對比——正邪之間的選擇**

> 「寧正言不諱以危身乎？將從俗富貴以偷生乎？……」

屈原一連提出八組對比——全是正邪之間的選擇：

| 正面（寧可） | 反面（將要） |
|:----------|:----------|
| 正言不諱以危身 | 從俗富貴以偷生 |
| 超然高舉以保真 | 哫訾慄斯、喔咿儒兒以事婦人 |
| 廉潔正直以自清 | 突梯滑稽、如脂如韋以潔楹 |
| 昂昂若千里之駒 | 汎汎若水中之鳧、與波上下偷以全吾軀 |
| 與騏驥亢軛 | 隨駑馬之跡 |
| 與黃鵠比翼 | 與雞鶩爭食 |
| 誅鋤草茅以力耕 | 游大人以成名 |
| 正言不諱 | 從俗富貴 |

**第四段：詹尹的回答——尺有所短，寸有所長**

> 「詹尹乃釋策而謝曰：『夫尺有所短，寸有所長；物有所不足，智有所不明；數有所不逮，神有所不通。用君之心，行君之意。龜策誠不能知此事。』」

鄭詹尹放下蓍草說：尺有時短、寸有時長——用您的心、行您的意吧——占卜解決不了您的問題。

## 2. 寫作技巧分析

### 八組對比——正邪之辨的完整呈現
屈原用八個對比句，把正義與邪惡的所有選項排列出來。每一組都是一個選擇題：

| 對象 | 正面 | 反面 |
|:----:|:----|:----|
| 自己 | 正言不諱 | 從俗富貴 |
| 為人 | 廉潔正直 | 如脂如韋 |
| 志向 | 若千里駒 | 若水中鳧、與波上下 |
| 格局 | 與黃鵠比翼 | 與雞鶩爭食 |

## 答案不在占卜——在於心
鄭詹尹最後的回答是全文的點睛之筆：「尺有所短，寸有所長……用君之心，行君之意。」——占卜無法幫你選擇道德，因為正邪的選擇在心，不在神。

### 修辭手法一覽

| 手法 | 原文 | 效果 |
|:----|------|------|
| **排比** | 八組「寧……將……」 | 節奏強烈、氣勢磅礡 |
| **比喻** | 千里駒 vs 水中鳧、黃鵠 vs 雞鶩 | 高下立判 |
| **對比** | 正與邪的選擇 | 不容模糊 |
| **反問** | 「此孰吉孰凶？何去何從？」 | 問蒼天，問自己 |
| **留白** | 詹尹不占——把答案還給屈原 | 最高級的哲學回答 |

## 3. 佳句賞析

### 「寧正言不諱以危身乎？將從俗富貴以偷生乎？」

寧可直言不諱讓自己身處危險？還是隨波逐流貪圖富貴苟且偷生？這是屈原對自己靈魂的拷問——每一個有良知的人都要面對的問題。

### 「尺有所短，寸有所長；物有所不足，智有所不明。」

尺有時反而短，寸有時反而長——萬物都有不足，智慧也有不明。這是中國哲學中最早關於「相對性」的優美表達。

### 「用君之心，行君之意。龜策誠不能知此事。」

用您的心，走您的路——占卜確實無法知道這種事。鄭詹尹的回答告訴屈原（也告訴我們）：道德選擇沒有捷徑，占卜無法替你活。
