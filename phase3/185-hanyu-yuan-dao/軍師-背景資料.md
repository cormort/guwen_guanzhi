# 〈原道〉背景資料

> **軍師分析 · 古文觀止卷七 · 唐文 · 韓愈**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《韓昌黎集》——韓愈詩文合集，由其門人李漢編纂，共四十卷；〈原道〉為韓愈論說文的代表作，列於全書卷十一 |
| **選集** | 《古文觀止》卷七·唐文——韓愈入選《古文觀止》共四篇（〈原道〉〈師說〉〈進學解〉〈送李愿歸盤谷序〉），〈原道〉居首，是其古文運動的理論總綱 |
| **體裁** | 論說文（「原」體）——以「原」為名的論說文，意在「推究本源」——韓愈以此文系統闡述儒家的「道」——並對佛教、道教展開全面批判 |
| **時代** | 中唐（約西元 8–9 世紀之際）——安史之亂後——藩鎮割據、佛道盛行、儒學衰微——韓愈在此背景下推動古文運動，〈原道〉是其思想宣言 |
| **核心命題** | 儒家之道是什麼？——「博愛之謂仁，行而宜之之謂義，由是而之焉之謂道」——韓愈重新定義了儒家「道」的內涵，並以此為武器批判佛老 |
| **關鍵人物** | 韓愈（唐代古文運動領袖、「文起八代之衰」）、佛老（佛教與道教——批判對象——被韓愈指為破壞社會秩序的根源） |
| **文本重點** | 韓愈在這篇文章中系統地論述了儒家「道」的內涵——「博愛之謂仁，行而宜之之謂義，由是而之焉之謂道」——並猛烈批判佛教和道教——佛道讓人出家、不事生產——「焚頂燒指，斷臂臠身」以求福報——導致社會勞動力減少、經濟衰退——他主張恢復儒家的「正統」——重新確立君臣父子、士農工商的社會秩序——文章言辭激烈——邏輯嚴密——是中國思想史上「排佛」的經典文獻 |
| **一段話總結** | **韓愈的〈原道〉不是一篇哲學論文——而是一份文化戰爭的宣言——他要奪回被佛教和道教佔領的思想陣地——重新定義「什麼是中國人的正確生活方式」。他用最簡單的語言——說最明白的道理——仁就是愛人——義就是做對的事——道就是走在這條路上——然後——他轉過身——對著寺廟裡的和尚說——你們——不是中國人該有的樣子。** |

## 二、時代背景——中唐的思想危機

### 2.1 安史之亂後的帝國困局

〈原道〉寫於中唐時期——約在唐憲宗元和年間（806–820 年）——安史之亂（755–763 年）過去約半個世紀——但帝國的創傷遠未癒合：

| 面向 | 說明 |
|------|------|
| **政治** | 安史之亂後——唐帝國中央權力大幅衰弱——藩鎮割據——節度使各自為政——朝廷僅能控制關中、江南等少數地區——唐憲宗雖力圖中興——但根基已動——危機四伏 |
| **經濟** | 戰爭破壞了北方的生產力——大量人口南遷——而佛寺道觀卻佔有大量土地和勞動力——「十分天下之財而佛有七八」——僧侶不納稅、不服役——成為國家財政的巨大漏洞 |
| **思想** | 佛教自東漢傳入——經過魏晉南北朝的發展——到唐代已成為主流信仰——禪宗、淨土宗、天台宗等宗派各擁信徒——道教因李唐皇室尊老子為始祖——也享有特權——儒學反而相對衰落——「儒門淡薄——收拾不住」 |
| **韓愈的憂慮** | 韓愈認為——佛教和道教的流行——不僅侵蝕了國家的經濟基礎——更破壞了儒家的倫理秩序——君臣父子、士農工商——這些社會基本結構——正在被「出家」「修道」的風氣——瓦解——這是〈原道〉寫作的直接動因 |

> **軍師按**：中唐的思想格局——用一個比喻——就像三國鼎立——儒家——是名義上的正統——但實際影響力最小——佛教——信徒最多——財力最雄厚——道教——有皇室的加持——也穩穩佔據一方。韓愈——就是在這個格局中——站出來——用一篇〈原道〉——向佛教和道教——同時宣戰。

### 2.2 「古文運動」的理論基礎——〈原道〉的寫作意圖

〈原道〉不僅是一篇排佛文章——它更是韓愈「古文運動」的理論宣言：

| 面向 | 說明 |
|------|------|
| **古文運動的目標** | 韓愈反對魏晉以來盛行的駢文——主張恢復先秦兩漢的「古文」——用樸實、自由的散文——來承載儒家的「道」——「文以載道」——這就是古文運動的核心綱領 |
| **〈原道〉的角色** | 〈原道〉是古文運動的「理論總綱」——韓愈在這裡——不僅闡明了什麼是「道」——更回答了為什麼要用「古文」來載「道」——因為——駢文華而不實——只適合「吟風弄月」——不適合「說理明道」 |
| **排佛與古文運動的關係** | 對韓愈來說——排佛和古文運動——是同一件事的兩個側面——佛教——用梵文的經書——講述外來的思想——韓愈——要用純正的漢語——講述純正的儒家之道——「古文」——就是「道」的載體——「道」——就是「古文」的靈魂 |

> **軍師按**：韓愈的偉大——在於他同時做了兩件事——在思想層面——他重新定義了儒家之道——讓它變得簡單、明白、可操作——在文學層面——他創造了一種新的散文風格——讓「道」可以用最美、最有力的語言——傳達給每一個人。〈原道〉——就是這兩件事的——完美結合。

## 三、文本結構——「原」道的三層論證

### 3.1 全文結構一覽

〈原道〉全文約一千餘字——邏輯嚴密——層層推進——可分為三大層次：

| 層次 | 核心內容 | 論證策略 |
|------|---------|---------|
| **第一層：正名——什麼是道？** | 從儒家的仁義出發——定義「道」的內涵——「博愛之謂仁——行而宜之之謂義——由是而之焉之謂道」 | 正面立論——先樹立標準——再以這個標準——衡量一切 |
| **第二層：批判——佛老哪裡錯了？** | 批判老子「去仁去義」的「道」——以及佛教「出家棄世」的「法」——指出它們都是對「道」的歪曲——導致社會混亂 | 反面論證——用儒家標準——逐一駁斥佛老 |
| **第三層：重建——儒家的正統是什麼？** | 提出儒家的「道統」——從堯舜禹湯——到文武周公——再到孔子孟子——這是一條一脈相承的「道」——韓愈自居為這一「道統」的繼承人 | 歷史論證——用「道統」——為儒家正名——也為自己——賦予話語權 |

### 3.2 開篇定調——「博愛之謂仁，行而宜之之謂義」

> 「博愛之謂仁，行而宜之之謂義，由是而之焉之謂道。足乎己無待於外之謂德。仁與義為定名，道與德為虛位。」

— 翻譯：廣泛地愛一切人——叫做「仁」——做什麼事都做得恰當——叫做「義」——從仁義出發往前走——叫做「道」——內心充實——不需要向外尋求——叫做「德」。仁和義——是有具體內容的名詞——道和德——是虛位——可以裝不同的內容。

| 層次 | 分析 |
|------|------|
| **重新定義「道」** | 韓愈開篇所做的第一件事——就是「搶奪話語權」——佛教和道教——都在講「道」——但什麼是「道」？——韓愈說——讓我來告訴你——「道」——就是從仁義出發——往前走的道路——不是虛無縹緲的「涅槃」——也不是「無為而治」——而是——實實在在地——愛人——做正確的事 |
| **「仁與義為定名——道與德為虛位」** | 這是全文最關鍵的哲學論證——韓愈說——「道」和「德」——是空盒子——誰都可以往裡面裝東西——佛教裝「涅槃」——道教裝「虛無」——儒家裝「仁義」——而只有儒家——裝對了東西——因為——仁義——是有具體內容的——不是空的——是實實在在的「愛人」和「行宜」 |
| **一句話掌握全文** | 韓愈的整個論證——就是一句話——「你們的『道』——不是真正的『道』——因為——你們的『道』——不講『仁義』」——所以——你們都是邪說——只有儒家——才是正統 |

> **軍師按**：韓愈的論證策略——非常高明——他不說「佛教是錯的」——他說的是「你們說的『道』——和我們說的『道』——不是同一個東西——而只有我們這個——才是對的」——他用定義——來排除異己——不給對方——任何辯駁的空間。

### 3.3 批判佛老——「焚頂燒指，斷臂臠身」

> 「今其法曰：必棄而君臣，去而父子，禁而相生養之道，以求其所謂清淨寂滅者。……然則如之何而可也？曰：不塞不流，不止不行。人其人，火其書，廬其居，明先王之道以道之。」

— 翻譯：現在他們的教義說——必須拋棄你的君臣關係——離開你的父子關係——禁止你們相互扶養的生活方式——去追求他們所謂的「清淨寂滅」……那麼——怎麼辦呢？——答案是：不堵塞（邪說）——（正道）就不能流傳——不止住（異端）——（正義）就不能推行——把那些僧道——還俗為民——把他們的經書——燒掉——把他們的寺觀——改為民宅——用先王之道——來教導他們。

| 層次 | 分析 |
|------|------|
| **對佛教的批判** | 韓愈指出佛教的核心問題——「棄而君臣——去而父子——禁而相生養之道」——出家人——不敬皇帝（棄君臣）——不養父母（去父子）——不結婚生子（禁相生養）——這在韓愈看來——是對儒家倫理——最徹底的破壞——如果每個人都出家——誰來種地？誰來打仗？誰來傳宗接代？——社會——會直接崩潰 |
| **對道教的批判** | 韓愈也批評了道教——追求「長生不死」——「清淨寂滅」——同樣是以「棄世」為代價——他認為——老子提出的「聖人之道」——本身就是錯誤的——「其所謂道——道其所道——非吾所謂道也」 |
| **激烈的解決方案** | 韓愈提出的方案——極其激烈——「人其人——火其書——廬其居」——把和尚道士還俗——燒掉佛經道藏——把寺廟道觀改成民宅——這在今天看來——像是「文化清洗」——但在當時——韓愈認為——這是唯一能拯救社會的方案——因為——佛老已經猖獗到了——威脅國家存亡的地步 |

> **軍師按**：韓愈在〈原道〉中的態度——是「零容忍」——他不說「佛教也有好的地方」——也不說「我們可以和平共處」——他說的是——「你是錯的——我要消滅你」——這種態度——讓〈原道〉——成為一篇極具「戰鬥性」的文章——也讓韓愈——在後世——既被尊為「道統」捍衛者——也被批評為「文化偏狹」的代表。但——回到中唐的歷史語境——韓愈的激烈——不是沒有理由的——當時——寺廟佔了全國最肥沃的土地——僧侶不納稅——不服役——勞動力大量流失——國家財政瀕臨崩潰——韓愈寫〈原道〉——不是學術討論——是一場——救亡運動。

## 四、關鍵人物

### 4.1 韓愈——「文起八代之衰」的孤獨鬥士

| 項目 | 內容 |
|------|------|
| **身份** | 唐代文學家、思想家、政治家——字退之——自稱「昌黎韓愈」——世稱「韓昌黎」——生於唐代宗大曆三年（768 年）——卒於唐穆宗長慶四年（824 年）——五十七歲——諡號「文」——故又稱「韓文公」 |
| **生平經歷** | 韓愈三歲喪父——由兄長韓會撫養——兄長早逝後——隨嫂嫂鄭氏生活——幼年刻苦讀書——「焚膏油以繼晷——恆兀兀以窮年」——貞元八年（792 年）進士及第——但仕途坎坷——屢遭貶謫——最著名的是——因上〈諫迎佛骨表〉——觸怒唐憲宗——被貶潮州——「一封朝奏九重天——夕貶潮陽路八千」 |
| **古文運動領袖** | 韓愈是中唐古文運動的領袖——他反對六朝以來的駢文——主張「文以載道——文道合一」——他的散文——氣勢磅礴——語言流暢——結構嚴謹——後世將其與柳宗元並稱「韓柳」——並列「唐宋八大家」之首——蘇軾評價他「文起八代之衰——道濟天下之溺」——這是中國文學史上——對一位作家——最高的讚譽之一 |
| **思想遺產** | 韓愈是宋明理學的先驅——他在〈原道〉中提出的「道統」說——「堯以是傳之舜——舜以是傳之禹——禹以是傳之湯——湯以是傳之文武周公——文武周公傳之孔子——孔子傳之孟軻——軻之死——不得其傳焉」——韓愈自居為這個「道統」的繼承人——這一觀念——直接影響了宋代的程朱理學——可以說——沒有韓愈——就沒有後來的理學傳統 |

**軍師評**：韓愈是一個「孤獨的鬥士」——在他所處的時代——佛教和道教是主流——儒學是邊緣——他站出來——用一篇〈原道〉——向整個時代的「共識」——發起挑戰。他提倡古文——反對駢文——提倡儒學——反對佛老——每一件事——都是在對抗時代的潮流。但他——最終——贏了——不僅贏了——而且——改變了——中國文化的一千年走向。這就是韓愈——「文起八代之衰」——不是一句空話——是一個人在面對整個時代時——做出的選擇。

### 4.2 佛教——被批判的「外來文化」

| 項目 | 內容 |
|------|------|
| **唐代佛教的盛況** | 唐代佛教達到鼎盛——全國有寺廟四千多所——僧尼十多萬人——寺院經濟發達——擁有大量田產——敦煌莫高窟的壁畫——龍門石窟的佛像——都是唐代佛教繁榮的見證 |
| **韓愈反對佛教的原因** | 韓愈反對佛教——主要有三個層面——（一）經濟層面——僧侶不納稅、不服役——寺院佔有大量土地——「十分天下之財而佛有七八」——（二）文化層面——佛教是外來宗教——它的教義——與儒家倫理衝突——特別是出家棄世——不拜君王——不敬父母——（三）社會層面——大量勞動力進入寺廟——導致農業生產萎縮——人口增長停滯——國家稅收減少 |
| **〈諫迎佛骨表〉** | 韓愈後來寫了〈諫迎佛骨表〉——這是〈原道〉的「實戰版」——他反對唐憲宗將佛骨迎入宮中供奉——說佛骨是「朽穢之物」——「凶穢之餘」——應「投諸水火」——這份奏表——讓韓愈差點被處死——最終被貶潮州——但他也因此——成為中國歷史上——「排佛」最著名的代表人物 |

**軍師評**：韓愈對佛教的態度——在今天是「不正確」的——但在中唐的歷史語境中——他的擔憂——是真實的——是合理的——他看到的——不是佛教的「慈悲」——而是寺院的「財富」——不是僧侶的「修行」——而是勞動力的「流失」——他不是在反對「宗教」——他是在反對——一個正在侵蝕國家根基的——經濟制度。

## 五、核心分析——〈原道〉為什麼是一份「文化戰爭的宣言」？

### 5.1 「道統」的發明——韓愈如何為自己賦權？

〈原道〉中——韓愈提出了一個極具影響力的概念——「道統」：

| 面向 | 分析 |
|------|------|
| **什麼是道統？** | 韓愈說——儒家的「道」——不是憑空產生的——它有「傳承」——從堯→舜→禹→湯→文武周公→孔子→孟子——這是一條——由聖賢組成的——「道」的傳遞鏈——孟子死後——這個「道統」——斷了——一千多年來——沒有人——真正繼承了這個「道」 |
| **韓愈的自我定位** | 韓愈說「軻之死——不得其傳焉」——這句話——的真正用意——是——「孟子之後——就沒有人了——而現在——我——韓愈——要來接上這條斷掉的線」——他把自己——放在了「道統」的繼承人位置上——這是一個極其大膽的——自我賦權 |
| **道統的歷史作用** | 「道統」的提出——在中國思想史上——具有劃時代的意義——它創造了一個「儒家內部」的傳承譜系——讓儒家——從一個「學派」——變成了一個「信仰」——後來的宋明理學家——程頤、朱熹——都自稱繼承了「道統」——而這個概念——正是從韓愈開始的 |

> **軍師按**：韓愈發明「道統」——是為了回答一個根本問題——「我們儒家——有什麼資格——要求所有人都聽我們的？」——韓愈的回答是——因為我們是堯舜禹湯文武周公孔孟的繼承人——我們的「道」——是從最古的聖人——一路傳下來的——而你們佛老——只是「外來」或「旁出」的異端——沒有這個「血統」——這是一個——無法反駁的——「身分政治」論證。

### 5.2 「人其人，火其書，廬其居」——韓愈的激烈與他的時代

韓愈在〈原道〉結尾提出的「解決方案」——極其激烈——甚至粗暴——但——需要理解他的時代語境：

| 面向 | 分析 |
|------|------|
| **韓愈的立場** | 韓愈認為——對佛老——沒有「共存」的空間——因為——它們的「道」——從根本上——與儒家之道——是對立的——你不可能同時信奉「君臣父子」和「出家棄世」——這是零和博弈——不是你死——就是我亡 |
| **時代的緊迫性** | 中唐時期——國家財政的困境——已經到了非常嚴重的地步——土地兼併——人口流失——藩鎮割據——朝廷的權威——岌岌可危——在韓愈看來——佛老——不是「文化多樣性」的問題——而是「國家生死存亡」的問題——他的激烈——來自於他的——焦慮 |
| **歷史的後續** | 唐武宗會昌五年（845 年）——距離韓愈去世僅二十一年——唐朝發動了「會昌滅佛」——拆毀寺廟四千六百多所——強制還俗僧尼二十六萬人——韓愈在〈原道〉中的主張——某種意義上——被實現了——雖然——佛教後來又復興——但——中國再也沒有回到——「佛教凌駕於國家之上」的狀態 |

> **軍師按**：韓愈的「激烈」——在今天看來——是「不寬容」——但在他的時代——這是「負責任」。他看到了帝國的危機——他選擇了——用最直接的方式——去解決這個危機。他不跟你談包容——不跟你談對話——他告訴你——這是戰爭——而戰爭——沒有妥協。

### 5.3 〈原道〉的文學成就——為什麼它是「古文」的典範？

〈原道〉不僅是一篇重要的思想文獻——它也是一篇——極高水平的——散文：

| 面向 | 分析 |
|------|------|
| **語言風格** | 韓愈的語言——簡潔——有力——沒有駢文的華麗雕琢——每一句話——都像一把刀——直插要害——「博愛之謂仁——行而宜之之謂義」——六個字完成定義——「人其人——火其書——廬其居」——三個動詞——完成了整篇文章的結論——這樣的語言——在駢文盛行的時代——是革命性的 |
| **邏輯結構** | 〈原道〉的論證——極其嚴密——先定義——再批判——再重建——每一步——都建立在前面論證的基礎上——層層遞進——不可反駁——這是「古文」說理的最高境界——用最樸實的語言——說最堅不可摧的道理 |
| **情感力量** | 雖然〈原道〉是一篇「說理文」——但它充滿了——韓愈的「情感」——他對儒家的熱愛——對佛老的憤怒——對國家未來的憂慮——貫穿全文——讓讀者——在讀理——同時——也被感染——這是韓愈超越一般說理文的地方——他不是在「說理」——他是在「佈道」 |

> **軍師按**：〈原道〉作為「古文」的典範——證明了韓愈最核心的主張——「文以載道」——文章——不是用來炫耀辭藻的——它的最高使命——是承載思想——傳播真理。〈原道〉——用最質樸的語言——說了最深刻的道理——這就是——「文起八代之衰」——最完美的——註腳。

## 六、軍師核心分析——「道」的戰爭：韓愈與他的時代

### 6.1 韓愈與李白——兩種「狂」

把韓愈的〈原道〉和李白〈與韓荊州書〉放在一起——可以看到唐代最偉大的兩個文學家——面對「主流」——做出了完全不同的選擇：

| 面向 | 李白〈與韓荊州書〉 | 韓愈〈原道〉 |
|------|-------------------|-------------|
| **面對的對象** | 一個具體的人（韓朝宗）——我要求你推薦我 | 一個抽象的時代（佛老之風）——我要改變這股潮流 |
| **姿態** | 狂——但狂的是「個人」——我是天才——你應該幫我 | 狂——但狂的是「真理」——我是對的——你們都錯了 |
| **策略** | 捧對方——同時捧自己——「雙贏」的求職術 | 先定義——再批判——「不塞不流——不止不行」——零容忍 |
| **結局** | 信成了千古名篇——但李白——可能沒有被推薦 | 文成了思想宣言——韓愈——被貶了——但他的思想——影響了千年 |

> **軍師按**：李白和韓愈——代表了「狂」的兩種極致——李白的狂——是「個人」的狂——「我是天才——你們都應該來幫我」——韓愈的狂——是「思想」的狂——「真理在我這裡——你們都錯了——我要消滅你們」——李白的狂——讓人喜歡他——韓愈的狂——讓人敬畏他——但——兩者——都配得上——「偉大」二字。

### 6.2 〈原道〉的歷史迴響——從韓愈到宋明理學

〈原道〉的影響——遠遠超出了唐代——它成為宋明理學的——思想起點：

| 面向 | 說明 |
|------|------|
| **對宋代儒學復興的影響** | 宋代的理學家——如周敦頤、張載、程顥、程頤、朱熹——都繼承了韓愈的「道統」說——他們把韓愈視為——從孟子之後——第一位真正理解儒家之道的人——朱熹甚至將韓愈的〈原道〉——收入《近思錄》——作為理學入門的必讀文獻 |
| **「道統」與中國政治文化** | 韓愈發明的「道統」——在中國後來的政治文化中——扮演了重要角色——它為儒家知識分子——提供了一個對抗皇權的「精神武器」——皇帝——是「政統」的代表——但儒家學者——可以自稱繼承了「道統」——從而在道德上——高於皇帝——這是宋明士大夫「以天下為己任」的思想根源 |
| **排佛的遺產** | 韓愈的排佛思想——在宋代——被歐陽修、二程、朱熹等人繼承和發展——但——宋代儒者對佛教的態度——比韓愈更加「精緻」——他們不是簡單地「排斥」——而是「吸收」——把佛教的思辨哲學——融入到儒學體系中——形成了「理學」——可以說——韓愈是「破」——宋儒是「立」——沒有韓愈的「破」——就沒有宋儒的「立」 |

> **軍師按**：韓愈的偉大——不在於他「說對了」什麼——而在於他「做了一個開頭」——他站在中唐的思想廢墟上——大聲說了一句——「儒家的道——是對的——我要恢復它」——這句話——在當時——幾乎沒有人聽——但——一百年後——兩百年後——三百年後——越來越多的人——開始聽到了——並沿著他開闢的道路——繼續走了下去——這就是——「文起八代之衰」——真正的含義——他不是一個人打贏了戰爭——他是一個人——發起了——一場戰爭。

## 七、一句話總結（再錄）

> **韓愈的〈原道〉不是一篇哲學論文——而是一份文化戰爭的宣言——他要奪回被佛教和道教佔領的思想陣地——重新定義「什麼是中國人的正確生活方式」。他用最簡單的語言——說最明白的道理——仁就是愛人——義就是做對的事——道就是走在這條路上——然後——他轉過身——對著寺廟裡的和尚說——你們——不是中國人該有的樣子。這篇文章——在中唐——是一聲孤獨的吶喊——在後世——是一面不倒的旗幟——它告訴每一個中國知識分子——真正的「道」——不在寺廟裡——不在山林中——而在君臣父子之間——在士農工商之中——在每一個——踏踏實實——過好自己生活的人——心中。而韓愈——用他的一生——證明了——一個人——可以面對整個時代——仍然——不退縮——不妥協——不沉默——因為——他相信——他所守護的——是「道」本身——而「道」——終將——戰勝一切。這是韓愈教給我們最重要的一件事——當真理在你這邊——你不必——向任何人——低頭。當全世界都告訴你——「大家都在信佛」——而你依然可以站出來說——「不對」——因為——你知道——「道」——是什麼樣的——而你知道——你——是對的。**

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷八·〈師說〉（韓愈） | 韓愈另一篇「原體」論說文——同樣是為儒家之道正名——〈師說〉講的是「從師之道」——〈原道〉講的是「儒家之道」——兩篇合讀——可以全面理解韓愈的思想體系 |
| 卷八·〈進學解〉（韓愈） | 韓愈以自嘲的方式——講述自己「學有所成——卻未被重用」的困境——與〈原道〉的慷慨激昂——形成「一文一諧」的對照——一個是出征前的宣言——一個是歸來後的感嘆 |
| 卷八·〈送李愿歸盤谷序〉（韓愈） | 韓愈對官場的厭倦——「與其有譽於前——孰若無毀於其後」——與〈原道〉的入世精神——形成辯證對照——韓愈不僅會「戰」——他也會「退」——這才是完整的人 |
| 卷七·〈滕王閣序〉（王勃） | 王勃的華麗駢文——是韓愈要「革」的對象——對比閱讀——可以看到古文運動前後——文章風格的劇變——從「落霞與孤鶩齊飛」——到「博愛之謂仁——行而宜之之謂義」——這是中國散文史上——最大的一次風格革命 |
| 卷九·〈種樹郭橐駝傳〉（柳宗元） | 柳宗元與韓愈同為古文運動領袖——但風格不同——柳宗元以寓言說理——韓愈以論說立論——〈種樹郭橐駝傳〉的「順其自然」——與〈原道〉的「不塞不流」——恰恰代表了兩種——對世界的不同態度 |

### 8.2 經典延伸

- 《韓昌黎集》（韓愈著，唐·李漢編）——韓愈詩文的最完整結集——四十卷——〈原道〉收錄於卷十一——是研究韓愈思想與文學的第一手資料
- 《新唐書·韓愈傳》（歐陽修、宋祁）——正史中韓愈最完整的傳記——評價他「其道——蓋自比孟軻」——肯定了他在儒家道統中的地位
- 《舊唐書·韓愈傳》（劉昫等）——比《新唐書》更早的韓愈傳記——對韓愈的評價較為中立——甚至有所保留——反映了唐代官方對韓愈的真實態度
- 《進學解》（韓愈）——韓愈以自嘲筆法寫成的奇文——「業精於勤——荒於嬉——行成於思——毀於隨」——與〈原道〉的嚴肅——構成鮮明對比——展現了韓愈性格的另一面
- 《諫迎佛骨表》（韓愈）——韓愈因為這篇文章——被貶潮州——是〈原道〉的「實戰版」——讀完〈原道〉——一定要讀〈諫迎佛骨表〉——才能理解韓愈——有多大的膽量——以及——他為自己的信念——付出了多大的代價
- 《近思錄》（朱熹、呂祖謙編）——宋代理學的入門書——收錄了韓愈的〈原道〉——作為理學思想的源頭——可以從這本書中——看到韓愈對宋儒的深遠影響
- 《韓愈志》（錢基博）——現代學者錢基博對韓愈的系統研究——從文學、思想、政治三個角度——全面評析韓愈的成就與局限

### 8.3 現代延伸

- 余英時《朱熹的歷史世界》——余英時在這本書中——詳細論述了韓愈「道統說」對宋代政治的影響——他認為——韓愈創造的「道統」——為宋代士大夫提供了一個「超越皇權」的道德立足點——這是理解中國政治思想史、宋明理學的重要著作
- 陳寅恪《論韓愈》——陳寅恪對韓愈的經典研究——他認為——韓愈的〈原道〉——不僅是儒學復興的宣言——更是唐代「民族主義」的體現——韓愈反對佛教——本質上是反對「外來文化」對中國的侵蝕——這個觀點——在學術界——引發了長期的討論
- 木心《文學回憶錄》——木心在講到韓愈時說——「韓愈是一個『兇猛』的文學家——他的文章——像一把刀——砍下去——不給你反駁的機會」——這個比喻——精準地抓住了韓愈的寫作風格——也抓住了〈原道〉的精神內核
- 王小波《沉默的大多數》——王小波在〈知識分子的不幸〉一文中——反思了中國知識分子的「使命感」——韓愈——就是這種「使命感」的典型代表——他認為自己掌握了真理——所以——有義務——讓所有人——都接受這個真理——這種態度——在今天——值得反思——但在韓愈的時代——它是——知識分子的最高光榮

## 九、背景補遺

### 9.1 「原」體——韓愈的發明

「原」——是韓愈創造的一種論說文體——意為「推究本源」——「原道」——就是「推究道的本源」——韓愈用這個題目——告訴讀者——我不是在講自己的觀點——我是在「追溯」——那個最古老、最正確的「道」——這種「溯源」的姿態——賦予了文章——一種無可辯駁的權威感——因為——「道」——從堯舜時代就有了——不是我韓愈發明的——我只是——把它——重新——告訴你們。

### 9.2 〈原道〉與韓愈的命運

韓愈寫〈原道〉的時候——大約三十多歲——還在長安——擔任四門博士（一個低級教職）——他的古文運動——才剛剛起步——他的思想——還沒有被廣泛接受——可以說——〈原道〉——是韓愈——在「什麼都不是」的時候——寫下的「宣言」——他當時——沒有任何政治權力——沒有任何學術權威——他只有——他的筆——和——他的信念——而正是這支筆和這個信念——最終——改變了——中國文化的走向。

### 9.3 「文起八代之衰」的真義

蘇軾對韓愈的評價——「文起八代之衰——道濟天下之溺」——「八代」指東漢、魏、晉、宋、齊、梁、陳、隋——這八個時代——駢文盛行——文章追求形式美——而忽略了內容——韓愈的「古文」——讓文章——重新回到了「承載思想」的本位——「道濟天下之溺」——指韓愈的儒學思想——把沉溺在佛老中的天下人——拯救了出來——這兩個評價——放在一起——就是韓愈一生的——最佳總結——他用「古文」——載「古道」——用「古道」——救「天下」——而這一切——都從一篇——只有一千多字——的〈原道〉——開始。
