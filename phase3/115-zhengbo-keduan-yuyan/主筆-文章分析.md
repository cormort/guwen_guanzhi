## 1. 文章結構分析

穀梁傳對「鄭伯克段於鄢」的註解，是春秋三傳中最短、最精煉的之一——全篇僅十數句，卻完成了對「經文用字→動機剖析→道德判斷」的完整推理鏈。

### 第一段：破題——「克者何？殺也。」

> 「克者何？殺也。何為殺之？段弟也而弗謂弟，公子也而弗謂公子，是段之失子弟之道也。」

穀梁傳開門見山，以設問破題：什麼叫「克」？就是殺。但經文為什麼不用「殺」而用「克」？因為段雖然是弟弟卻不被稱作弟弟，雖然是公子卻不被稱作公子——他已經失去了作為弟弟和公子的本分。一個「克」字，同時指控了兩個人的錯誤。

### 第二段：轉折——從段的失道到鄭伯的失教

> 「段失子弟之道，則鄭伯失教之道明矣。」

這是全文最有力量的邏輯轉折——穀梁傳不只罵段，更把矛頭指向鄭莊公：段之失道，根源在鄭伯之失教。作為兄長和國君，莊公對段的墮落負有不可推卸的責任。

### 第三段：深化——「處心積慮」的指控

> 「甚鄭伯之處心積慮，成於殺也。」

穀梁傳進一步指控：鄭莊公的殺心不是一時衝動，而是長期算計、精心策劃——「處心積慮」四個字把莊公從「失教的哥哥」升級為「算計弟弟的兇手」。

### 第四段：收束——「于鄢」的倫理意義

> 「于鄢，遠也。猶曰取之其母之懷中而殺之云爾，甚之也。」

最後落到「于鄢」二字——經文記載地點，不是為了地理標記，而是為了說明追殺之遠、趕盡殺絕之狠。把弟弟從偏遠的鄢地抓回來殺掉，就像從母親懷裡奪走嬰兒殺掉一樣殘忍。這是全文的倫理高潮。

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| **起** | 克者何？殺也——以設問破題，定義用字背後的殺意 |
| **承** | 段失子弟之道——指出段的錯誤（不配為弟） |
| **轉** | 鄭伯失教之道——最高明的轉折：從段的錯轉到莊公的錯 |
| **合** | 于鄢——取之其母之懷中而殺之——以母親視角作倫理收束 |

## 2. 寫作技巧分析

### 穀梁傳的「逐字解經法」——從用字推出道德判斷

穀梁傳的核心方法論，是對經文每一個字的獨立分析——不靠敘事補充、不靠歷史背景，單從用字的選擇來推斷《春秋》作者的道德立場。這種方法可以稱作「語言哲學式的解經」。

#### 經文字詞→穀梁傳解讀→背後判斷

| 經文字詞 | 穀梁傳解讀 | 背後判斷 |
|:---------|:-----------|:---------|
| **鄭伯** | 稱爵不稱人——「鄭伯」是諸侯稱號，不是「鄭人」（貶稱）也不是字（褒稱） | 中性稱謂暗示莊公未做值得褒貶之事，本身已是批評 |
| **克** | 克者何？殺也。用「克」不用「殺」——因為這段關係不是刑罰而是戰爭 | 莊公把弟弟當敵人打——已失兄長之道 |
| **段** | 不稱「弟」不稱「公子」——直接寫名字 | 段不配被稱為弟弟和公子——失子弟之道 |
| **于鄢** | 于鄢，遠也——特意標出遙遠的地點 | 莊公追殺至遠方，趕盡殺絕——過分 |
| **（段之名）** | 段失子弟之道 | 段先犯錯，但不等於莊公無罪 |
| **（處心積慮）** | 甚鄭伯之處心積慮，成於殺也 | 不是一時憤怒而是長期算計——更惡 |
| **（母懷之喻）** | 猶曰取之其母之懷中而殺之云爾 | 莊公的行為等同在母親面前殺她的孩子——不孝不仁 |

### 問答體——設問即立場

穀梁傳採用「問→答」的形式，但每一次設問其實都已經內含了立場：

| 設問 | 引導方向 | 目的 |
|:---:|:---------|:-----|
| 「克者何？」 | 迫使讀者思考為什麼不用更直接的「殺」 | 引向「用字即判斷」的解經前提 |
| 「何為殺之？」 | 追問殺人的正當性 | 引向對段的批評（段有罪在先） |
| （隱含）「為什麼不稱弟？」 | 迫使讀者思考稱謂的意義 | 引向對段失道的批評 |
| （隱含）「為什麼記于鄢？」 | 迫使讀者思考地點的象徵意義 | 引向對莊公趕盡殺絕的批評 |

### 穀梁傳的三傳差異分析——三種提問方式

「鄭伯克段於鄢」是春秋三傳共同重點註解的篇章。同一條經文，三傳的問法截然不同，因此得出截然不同的結論：

| 維度 | 左傳 | 公羊傳 | 穀梁傳 |
|:----|:-----|:-------|:-------|
| **核心問題** | 發生了什麼事？（敘事補充） | 誰對誰錯？（褒貶判斷） | 為什麼這樣寫？（語言分析） |
| **解法** | 補充史實——姜氏愛段、莊公縱段、段叛、莊公伐之 | 以一字定褒貶——用「克」是為了「大鄭伯之惡」 | 以用字推義理——克≠殺，弟≠段，于鄢≠無意義 |
| **對莊公態度** | 隱忍算計，但最後勝出 | 惡——「處心積慮」殺弟 | 失教＋狠毒——雙重批評 |
| **對段態度** | 咎由自取 | 未深究 | 有責——「失子弟之道」 |
| **對姜氏態度** | 偏愛幼子釀禍 | 未多著墨 | 以母親比喻隱含同情 |
| **文風** | 故事精彩，人物立體 | 斬釘截鐵，判斷明確 | 邏輯推演，層層剝繭 |

#### 同一個「克」字，三種解讀

| 傳別 | 「克」的解釋 | 道德歸責 |
|:----|:------------|:---------|
| **左傳** | 未特別解釋「克」字——直接用敘事取代經文 | 以「多行不義必自斃」暗批段，以「遂寘姜氏於城潁」托出莊公算計 |
| **公羊傳** | 克之者何？殺之也。殺之則曷為謂之克？大鄭伯之惡也 | 全部歸咎莊公——用「克」是為了放大莊公的罪惡 |
| **穀梁傳** | 克者何？殺也。何為殺之？段弟也而弗謂弟……段失子弟之道也 | 雙方都有責——先批段失道，再批莊公失教 |

> **軍師按**：三傳之別在同一字上看得最清楚——左傳不管「克」，只管講故事；公羊說「克=放大罪惡」；穀梁說「克=殺，但殺人的理由不充分」。同一本教科書，三種上課方式。

### 均衡倫理——穀梁傳的標誌性手法

穀梁傳與公羊傳最大的不同是：公羊贊成一邊倒的褒貶，穀梁堅持各打五十大板。

| 層次 | 公羊傳的判斷 | 穀梁傳的判斷 |
|:----|:-------------|:-------------|
| 誰的錯？ | 全是鄭伯的錯（大鄭伯之惡） | 兩人都有錯 |
| 段該不該被殺？ | 未正面回答 | 段有罪，但不該被這樣殺——莊公失教在先 |
| 莊公的問題是？ | 天性邪惡 | 失職（不教）＋殘忍（追殺到底） |
| 春秋經文的態度 | 貶斥莊公 | 同時貶斥兄弟二人 |

### 修辭手法一覽

| 手法 | 原文 | 效果 |
|:----|------|------|
| **設問** | 「克者何？」「何為殺之？」 | 以問句強迫讀者思考每個字的意義 |
| **定義** | 「克者何？殺也」「于鄢，遠也」 | 先給明確語義再展開論證 |
| **遞進** | 段失道→鄭伯失教→處心積慮→取之母懷 | 層層升級，從客觀記述到倫理控訴 |
| **比喻** | 「猶曰取之其母之懷中而殺之云爾」 | 將抽象的道德批評具象化為母親的悲劇 |
| **反覆強調** | 兩個「甚」（甚……處心積慮、甚之也） | 以「甚」字重複強化批評力度 |
| **留白** | 不交代史實細節（不敘述築城、請邑、叛亂等情節） | 迫使讀者聚焦於用字本身的道德意義 |

## 3. 佳句賞析

### 「克者何？殺也。」

穀梁傳的第一句——也是最有力的一句。直接定義「克＝殺」，但又不只定義——它同時在問：既然是殺，為什麼不用「殺」字？這個問號背後藏著整個穀梁學派的方法論：每一個字都不是隨便選的，經文的用字本身就是一種判斷。

### 「段失子弟之道，則鄭伯失教之道明矣。」

這是全文邏輯的關鍵轉折。穀梁傳先批評段（你不配當弟弟），但立刻補上最強的一刀：段之所以會變成這樣，是因為你這個哥哥沒有教好。這不是為段開脫，而是在追究更深層的責任——誰創造了讓段可以走上反叛之路的環境？

### 「甚鄭伯之處心積慮，成於殺也。」

「處心積慮」四個字是穀梁傳對莊公最嚴厲的指控。不是一時憤怒，不是戰場上的衝動——是長期以來一直在心裡算計，最終把自己逼到非殺不可的局面。這個詞後來成為成語，幾乎成了鄭莊公在中國文化中的標籤。

### 「猶曰取之其母之懷中而殺之云爾，甚之也。」

全文最動人的一句。穀梁傳突然插入一個母親的形象——姜氏站在遠方，眼睜睜看著自己的大兒子殺了自己的小兒子。穀梁傳沒有寫姜氏的反應，但這一句話已經把母親的悲痛寫盡了。在三傳之中，只有穀梁傳用了這個視角。

### 全文一覽——三傳核心文句對照

| 傳別 | 核心文句 | 核心關懷 |
|:----|:---------|:---------|
| **左傳** | 「多行不義必自斃，子姑待之。」 | 莊公的隱忍與算計 |
| **公羊傳** | 「克之者何？殺之也。殺之則曷為謂之克？大鄭伯之惡也。」 | 以「克」字彰顯莊公之惡 |
| **穀梁傳** | 「克者何？殺也。……段失子弟之道矣，鄭伯失教之道明矣。」 | 雙方都有責任——失教與失道 |

## 4. 白話改編

《春秋》經文寫道：「夏五月，鄭伯克段於鄢。」

穀梁傳解釋說：

什麼叫「克」？就是殺。為什麼說他殺？段明明是弟弟卻不稱他為弟弟，明明是公子卻不稱他為公子——這表示段已經失去了身為弟弟和公子的本分。

段既然失去了子弟之道，那麼鄭伯失於教導的責任也就很清楚了。

太可惡了——鄭伯在心裡算計了很久，最終就是要殺他。

經文特意寫「於鄢」，意思是說段逃到了很遠的地方。但鄭伯仍然追過去殺掉他——這就好像從母親懷裡把孩子搶出來殺掉一樣。太過分了。
