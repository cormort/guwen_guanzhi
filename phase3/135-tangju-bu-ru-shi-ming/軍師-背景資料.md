## 二、時代背景——秦滅六國前夕的強權政治

### 2.1 秦王政的擴張態勢

西元前 230 年起，秦國開始大規模吞併六國：先滅韓（前 230 年），再滅趙（前 228 年），兵鋒直指魏國。安陵是魏國的附庸小國，國土僅方圓五十里，在秦國即將滅魏的局勢下，秦王政提出「以五百里之地易安陵」——這顯然不是公平交易，而是藉「交換」之名行「吞併」之實。

| 面向 | 說明 |
|------|------|
| **秦國戰略** | 韓、趙已滅，秦國的下一個目標是魏國。先取安陵（魏國附庸），意在試探魏國抵抗意志，同時拔除魏國南方的屏障 |
| **安陵處境** | 方圓五十里的小國，無兵可戰、無險可守。在秦國面前，安陵連「以卵擊石」的資格都沒有——卵還有機會砸碎石頭，安陵連這個機會都沒有 |
| **交換的真相** | 「五百里之地」是空頭支票——秦國的土地都在自己掌控中，給了你也能隨時拿回去。安陵君的拒絕，說明了所有人都看得穿這個騙局，只是沒有人敢在秦王面前說破 |

### 2.2 為什麼秦王要「易地」——強權的偽裝與算計

秦王政的「易地」之說，本質上是一場政治恐嚇：表面上是「用大換小」的公平交易，實際上是在測試安陵君的忠誠。如果安陵君同意，秦國不費一兵一卒就吞併了安陵；如果安陵君拒絕，秦國就有了「你不給我面子」的藉口，可以名正言順出兵。

> **軍師按**：秦王政的「五百里易五十里」，和現代談判中的「虛張聲勢的開價」（anchoring）一模一樣——先拋出一個荒謬的提議，對方拒絕的理由就會顯得「小氣」，從而讓對方陷入被動。安陵君看穿了這個陷阱，但他不敢直接拒絕秦國——他只能派唐雎去「解釋」。唐雎的任務，就是用外交辭令說出安陵君真正的意思：「我們不想換，但我們不敢說『不』，所以請您自己放棄。」

## 三、原文結構與關鍵情節

### 3.1 原文結構總覽

| 層次 | 情節 | 寓意 |
|------|------|------|
| **第一層：秦王出題** | 秦王對唐雎說：秦國用五百里之地換安陵，安陵君竟然拒絕？秦國滅韓亡趙，你不要自誤 | 秦王先展示實力，再施壓——典型的「胡蘿蔔＋棒子」策略。用大國威勢為「交換」背書，讓對方覺得拒絕就是找死 |
| **第二層：唐雎婉拒** | 唐雎說：安陵君受地於先王而守之，即使拿千里之地也不敢換，何況五百里？ | 唐雎避開了「敢不敢拒絕秦國」的問題，而把拒絕的理由歸結為「先王所賜」——這是儒家「孝」的邏輯，秦王無從反駁 |
| **第三層：秦王大怒** | 秦王勃然大怒——「你聽過天子之怒嗎？天子一怒，伏屍百萬，流血千里。」 | 秦王見軟的不行，直接亮出軍事威脅——「百萬伏屍」就是秦國實力的寫照 |
| **第四層：唐雎反擊** | 唐雎說：「你聽過布衣之怒嗎？布衣一怒，也不過是摘掉帽子、用頭撞地罷了。」——先貶低自己，製造反差。然後話鋒一轉：專諸刺吳王僚、聶政刺韓相俠累、要離刺慶忌——「此三子者，皆布衣之士也」——「若士必怒，伏屍二人，流血五步，天下縞素。今日是也！」說罷拔劍而起 | 這是全文最高潮——唐雎先用「布衣之怒＝撞頭」的假象麻痺秦王，然後瞬間翻轉：真正的布衣之怒，不是拿自己的命發洩，而是拿你的命來陪葬——「伏屍二人，流血五步」——你（秦王）和我（唐雎），兩個人。你死了，天下縞素；我死了，只是一個老人家。我的命不值錢，你的命很值錢——你要不要賭？ |
| **第五層：秦王屈服** | 秦王臉色大變，長跪道歉——「先生坐！何至於此！寡人明白了——韓魏滅亡，但安陵以五十里之地存活，是因為有先生啊！」 | 秦王從「天子之怒」到「長跪而謝」——一百八十度大轉彎。這不是秦王突然變講道理了，而是他發現：眼前這個九十多歲的老人，是真的不怕死 |

> **軍師按**：這篇外交對話的結構堪稱完美——秦王先攻（以強凌弱），唐雎先守（禮貌拒絕）；秦王再攻（武力威脅），唐雎反殺（以命相搏）。層層遞進，最後以「拔劍而起」作為話語的最高點——所有語言的力量，最終凝聚在一個具體的動作上。這比任何修辭都更有說服力。

### 3.2 核心段落節錄

> 秦王使人謂安陵君曰：「寡人欲以五百里之地易安陵，安陵君其許寡人？」
> 安陵君曰：「大王加惠，以大易小，甚善。雖然，受地於先王，願終守之，弗敢易。」

> ——第一回合：秦王以「大易小」施壓，唐雎以「受地先王」婉拒

> 秦王怫然怒，謂唐雎曰：「公亦嘗聞天子之怒乎？」
> 唐雎對曰：「臣未嘗聞也。」
> 秦王曰：「天子之怒，伏屍百萬，流血千里。」
> 唐雎曰：「大王嘗聞布衣之怒乎？」
> 秦王曰：「布衣之怒，亦免冠徒跣，以頭搶地爾。」
> 唐雎曰：「此庸夫之怒也。若士必怒，伏屍二人，流血五步，天下縞素。今日是也！」

> ——第二回合：秦王展現天子威嚴，唐雎以「士之怒」反殺

> 秦王色撓，長跪而謝之曰：「先生坐！何至於此！寡人諭矣——夫韓、魏滅亡，而安陵以五十里之地存者，徒以有先生也。」

> ——第三回合：秦王徹底屈服，長跪道歉

## 四、關鍵人物

### 4.1 唐雎——九十歲的「布衣刺客」

| 項目 | 內容 |
|------|------|
| **身份** | 魏國老臣，以善說與膽識聞名。此時年紀已逾九十 |
| **在《戰國策》中的出場** | 至少出現三次：〈唐雎說信陵君〉（勸信陵君謙遜）、本文（出使秦國捍衛尊嚴）、以及其他魏國外交活動 |
| **核心貢獻** | 以一己之力，讓秦國放棄吞併安陵的企圖——不費一兵一卒，單靠一句「伏屍二人，流血五步」 |
| **性格特質** | 冷靜、沉著、不怒自威。他是少數敢在秦王面前拔劍的人——而且不是因為衝動，而是經過精心計算 |
| **外交風格** | 先禮後兵——前面恭恭敬敬、禮貌周到，讓秦王覺得他好欺負；然後在最關鍵的時刻，瞬間切換為「以命相搏」模式，讓秦王措手不及 |

**軍師評**：唐雎的高明，不在於他敢於拔劍，而在於他精準計算了「拔劍的時機」和「拔劍的價值」。

1. **先示弱，後打反差**——唐雎前面表現得越謙卑、越禮貌，當他拔劍時造成的衝擊力就越大。如果他一開始就強硬，秦王會有心理準備；但他前面完全是一副「老實人」的樣子，突然間變成一個不要命的刺客——這才是真正的震懾。

2. **用他人的案例替自己背書**——唐雎沒有直接說「我會殺你」，而是先說了專諸、聶政、要離三人的故事。這三個都是歷史上有名的刺客，而且都成功了。唐雎講這三個故事，等於是在說：「這些人能做到的事，我也能做到——而且我已經站在你面前了。」

3. **一句話把秦王拉下水**——「伏屍二人，流血五步」——注意，是「二人」，不是「一人」。唐雎明確告訴秦王：我要殺的是你。這不是同歸於盡，這是「我的命換你的命」——對唐雎來說，九十多歲的老命換三十多歲秦王的命，太划算了。

### 4.2 秦王嬴政——從狂妄到長跪的權力者

| 項目 | 內容 |
|------|------|
| **身份** | 秦國國君，後來的秦始皇。此時已滅韓、趙，正準備吞併六國 |
| **初始姿態** | 以「天子」自居——「天子之怒，伏屍百萬，流血千里」——這句話充滿了權力者的傲慢 |
| **態度轉變** | 從「天子之怒」到「長跪而謝」——這個轉變說明了唐雎的震懾是真實的 |
| **最後一句話** | 「夫韓、魏滅亡，而安陵以五十里之地存者，徒以有先生也」——這不是客套話，是秦王真正被說服了的證明 |

**軍師評**：秦王的反應，暴露了一個重要的權力真相——**看似最強的人，往往最怕死。** 秦始皇統一六國後，派徐福求長生不老藥、修建驪山陵墓、築造阿房宮——他對「死亡」的恐懼比任何人都強烈。唐雎正是抓住了這一點：你用「百萬伏屍」嚇我，我就用「你一個人死」反過來嚇你。百萬人死跟你無關，你一個人死，天下都跟你無關了。

### 4.3 安陵君——藏在幕後的「看不見的手」

安陵君在本文中沒有親自出場（所有對話由唐雎代理），但他才是真正的外交操盤手。他派唐雎出使秦國，說明了三件事：
1. **他知道唐雎的價值**——在魏國即將滅亡的關頭，他派的是全國最有膽識、最有謀略的老臣
2. **他算準了秦王的心理**——秦王是天之驕子，最怕的不是兵強馬壯的對手，而是一個不怕死的老人
3. **他給了唐雎充分的授權**——「若士必怒，伏屍二人」——這代表唐雎有權在秦王面前做出「最後決定」，這不是一個普通使節擁有的權力

## 五、核心分析——「布衣之怒」為何能勝過「天子之怒」？

### 5.1 「天子之怒」與「布衣之怒」的對話分析

秦王和唐雎的這一段對話，是人類外交史上最精彩的辯論之一。讓我們逐句拆解：

**秦王：「天子之怒，伏屍百萬，流血千里。」**
→ 秦王的論證邏輯：我是天子，我生氣了就會發動戰爭，百萬人會死。你承受得起嗎？
→ 這是典型的「嚇阻力」（deterrence）——用毀滅性後果阻止對方反抗。

**唐雎：「布衣之怒，亦免冠徒跣，以頭搶地爾。」**
→ 唐雎故意曲解「布衣之怒」的含義——把「布衣之怒」描述成一個可笑的形象（摘帽子、光腳、撞頭）。
→ 這是在麻痺秦王，讓秦王覺得「這個老頭也就這點本事」。

**秦王顯然鬆懈了——他笑了？或者鄙夷地看著唐雎？**
→ 這正是唐雎要的效果——讓秦王的防備降到最低。

**唐雎（忽然話鋒一轉）：「此庸夫之怒也，非士之怒也。夫專諸之刺王僚也，彗星襲月；聶政之刺韓傀也，白虹貫日；要離之刺慶忌也，倉鷹擊於殿上。此三子者，皆布衣之士也——若士必怒，伏屍二人，流血五步，天下縞素。今日是也！」挺劍而起。**
→ 這是全文最重要的翻轉。唐雎用三個歷史刺客的故事，重新定義了「布衣之怒」的含義。
→ 「伏屍二人」——對比秦王的「伏屍百萬」。秦王的百萬是別人的命，唐雎的二人是「你和我」——其中一個是你。

### 5.2 為什麼秦王投降了？

秦王怕的不是唐雎這個人，而是怕「一個九十歲老人不怕死」這件事：

1. **唐雎沒有任何可失去的東西**——他已經九十多歲了，人生走完了，家人也不見得還能帶走（魏國即將滅亡）。這樣的人，你拿什麼威脅他？沒有。
2. **秦王有太多可失去的東西**——他才三十多歲，剛剛滅了韓趙，正準備統一天下。他的人生才剛開始。如果死在這裡，一切都完了。
3. **唐雎的刺殺是「可執行的」**——他已經站在秦王面前，距離只有幾步。他有劍。秦王有護衛，但護衛來不及在唐雎拔劍刺出之前制服他。

> **軍師按**：唐雎的「布衣之怒」之所以能贏秦王的「天子之怒」，不是因為勇氣比權力更大，而是因為**「沒有退路的人」永遠比「有退路的人」更可怕**。秦王有天下要統治，有六國要征服，有長生不老藥要追求——他輸不起。唐雎什麼都沒有，只剩下這條命——他輸得起。在談判中，最強的王牌永遠是「我沒有什麼可失去的」。

### 5.3 與〈唐雎說信陵君〉的參照閱讀

| 面向 | 〈唐雎說信陵君〉 | 〈唐雎不辱使命〉 |
|------|---------------|----------------|
| **對象** | 信陵君（盟友）——溫和、含蓄 | 秦王（敵人）——剛烈、決絕 |
| **說服策略** | 設懸念→給框架→套用現實 | 禮貌→反諷→威脅→拔劍 |
| **核心命題** | 忘記自己的功勞 | 不怕失去自己的生命 |
| **唐雎的角色** | 智慧長者——「冷卻劑」 | 布衣刺客——「引爆點」 |
| **說話風格** | 極簡含蓄、點到為止 | 層層遞進、驚天翻轉 |

兩篇合看，展現了一個完整的唐雎——**對朋友溫和、對敵人剛烈；能伸能屈、能柔能剛。**

## 六、歷史影響與文學地位

### 6.1 弱國外交的典範

「唐雎不辱使命」在中國文化中已經成為一個成語——代表**用智慧和勇氣完成不可能的外交任務**。在戰國策中，這是少數「弱國靠一句話就讓強國退讓」的篇章——大多數時候，弱國在強權面前只能妥協，但唐雎打破了這個定律。

### 6.2 現代啟示

1. **談判中的終極籌碼**——當你什麼都沒有了的時候，你的命就是你最後的籌碼。這句話在商業談判中也適用：當你已經沒有任何東西可以失去時，你反而獲得了最大的談判權——因為對方無法懲罰你，也無法威脅你。

2. **用對比製造翻轉**——唐雎先假裝「布衣之怒＝撞頭」的庸夫形象，然後突然翻轉為「士之怒＝殺你」。這個強烈的對比，讓秦王的心理防線在瞬間崩塌。這在演講和談判中是一個重要技巧：先建立一個錯誤的期待，再用正確的事實摧毀它。

3. **權力者的最大弱點**——秦王有天子的權力，但他有最大的弱點：怕死。唐雎看得比任何人都清楚。找到權力者的弱點，往往比對抗權力本身更有效。

> **軍師總評**：〈唐雎不辱使命〉是《古文觀止》中最具戲劇張力的一篇外交對話。從「五百里易五十里」的荒謬提議，到「天子之怒」的武力威脅，再到「布衣之怒」的以命相搏——全文在層層推進中達到令人窒息的緊張，最後以秦王「長跪而謝」的徹底屈服作結。唐雎用自己的生命，證明了中國士人文化中的一個永恆命題——**士可殺不可辱。** 當秦國認為可以用實力壓迫一個小國時，一個九十歲的老人站出來說：你可以殺我，但你不能侮辱我——而如果你逼我選擇，我會選擇拉你一起走。**最強大的帝國也怕一個不怕死的人——因為權力只能支配活著的人，而無法支配一個已經準備好去死的人。**
