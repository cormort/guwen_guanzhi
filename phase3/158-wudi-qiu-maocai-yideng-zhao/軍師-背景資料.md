# 〈武帝求茂材異等詔〉背景資料

> **軍師分析 · 古文觀止卷六 · 西漢文 · 劉徹（漢武帝）**

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《漢書·武帝紀》——漢武帝晚年求賢的著名詔書 |
| **選集** | 《古文觀止》卷六·西漢文類 |
| **體裁** | 詔令文——皇帝徵召人才的官方命令，但口吻中充滿了對「非常之人」的渴望 |
| **時代** | 西漢中期，漢武帝元封五年（約西元前 106 年）——武帝在位已近四十年（前 141–前 87 年），帝國如日中天，但也面臨人才枯竭的隱憂 |
| **核心命題** | 非常之功，必待非常之人——當帝國需要開拓時，常規的人才選拔標準遠遠不夠，必須突破框架，不拘一格用人才 |
| **關鍵人物** | 漢武帝劉徹（發布詔書者）、州郡地方官（被要求舉薦人才的執行人）、各級「跅弛之士」（被詔書鎖定的非常規人才） |
| **文本重點** | 這是一道極為簡短但影響深遠的求賢詔。武帝開宗明義：「蓋有非常之功，必待非常之人」——想要建立不平凡的事業，就必須依靠不平凡的人。他特別強調「馬或奔踶而致千里，士或有負俗之累而立功名」——有些馬會踢人但能日行千里，有些人被世俗議論但能建功立業——朝廷不該用常規標準來衡量特殊人才。著名的「跅弛之士」一詞即出於此——指那些不受拘束、不遵常軌的狂放之士 |
| **思想譜系** | 延續先秦「尚賢」傳統——從孔子「舉賢才」到劉邦〈高帝求賢詔〉，再到這道詔書，求賢是中國政治的一條主線。但武帝的特殊之處在於：他不是泛泛地求賢，而是有意識地要突破道德與行為常規的界限——他要的不是模範公民，而是能建功立業的「非常之人」。這是中國歷史上第一次由最高統治者公開宣稱：「不完美的人才」也可以被重用——甚至更應該被重用 |
| **一句話總結** | **漢武帝是中國歷史上最懂「突破常規用人」的皇帝——他寧可用一個有缺點的天才，也不用一個完美的庸才。〈武帝求茂材異等詔〉就是這份用人哲學的宣言書：它告訴天下——你的缺點不重要，你的才能才重要。** |

## 二、時代背景——帝國巔峰的人才焦慮

### 2.1 漢武帝在位近四十年的帝國格局

西元前 106 年，漢武帝頒布這道求賢詔時，他已經統治了這個帝國三十五年：

| 面向 | 說明 |
|------|------|
| **武功鼎盛** | 武帝即位後，改變了漢初對匈奴的「和親」政策，轉為大規模主動進攻——衛青、霍去病等名將屢次出擊匈奴，奪取河套、河西走廊，迫使匈奴北遁。張騫通西域（前 138 年首次出使），帝國的視野從黃河流域擴展到中亞——這是中國歷史上第一次真正意義上的「世界眼光」 |
| **內政改革** | 武帝推行了一系列加強中央集權的措施——頒行「推恩令」（前 127 年），削弱諸侯王勢力；設立「中朝」（內朝），削弱丞相權力；創立「刺史」制度（前 106 年——正是這道詔書的同一年），監察地方官員；確立「察舉孝廉」制度，從民間選拔人才 |
| **經濟控制** | 武帝將鹽、鐵、酒等重要物資收歸國營（鹽鐵專賣），發行「五銖錢」統一貨幣，設置「均輸平準」調控物價——這些政策為他的對外征伐提供了巨額財政支持 |
| **文化大一統** | 武帝採納董仲舒「罷黜百家，獨尊儒術」的建議（前 134 年），將儒家確立為官方意識形態；設立五經博士，推廣儒家教育。從此，儒學成為中國兩千年帝制時代的正統思想 |

> **軍師按**：理解〈武帝求茂材異等詔〉的關鍵，在於理解武帝時代的「雙重性格」。一方面，這是中國歷史上最輝煌的時代之一——開疆拓土、打通西域、獨尊儒術，帝國前所未有地強大；另一方面，長期的征戰（對匈奴、對南越、對朝鮮、對西南夷）耗費了巨大的人力財力——百姓負擔沉重，社會矛盾加劇。武帝在晚年頻繁頒布求賢詔——是因為他隱約感到：帝國的擴張已經到了極限，而最稀缺的資源不是土地、不是金錢——是人才。他需要新的人才來處理前所未有的複雜局面。

### 2.2 為什麼需要「非常之人」？

武帝在元封五年頒布這道求賢詔，有其特定的歷史契機：

| 面向 | 說明 |
|------|------|
| **開疆拓土後的治理需求** | 到元封五年，漢朝的疆域已經遠遠超過了秦始皇時代——新設的郡縣（如河西四郡、交趾、日南等）需要大量官員去治理。但這些新領土地處偏遠、環境惡劣，正規的士大夫不願去——需要的是那些「不守常規」、敢於冒險的人 |
| **舊臣凋零，急需新血** | 武帝早年的一批名臣——衛青（前 106 年去世）、霍去病（前 117 年去世）、張騫（前 114 年去世）、主父偃（前 126 年被殺）——先後凋零。帝國的人才庫出現了明顯的斷層。武帝需要一批新的能人，來填補這些傳奇人物的空缺 |
| **對「察舉制」的反思** | 武帝於前 134 年推行「察舉孝廉」，這是一種由地方官推薦人才的制度。但這個制度有一個根本問題：「孝廉」以道德為標準——推薦的都是老實人、好人——但開拓帝國需要的是有膽識、有謀略、敢於打破規則的「能人」。〈武帝求茂材異等詔〉就是針對這個問題而發——武帝在公開說：不要只推薦乖孩子——給我推薦那些有本事、即使有缺點也沒關係的人 |

> **軍師按**：這道詔書還有一個微妙的時代背景——元封五年正好是武帝「封禪泰山」（前 110 年）之後的第四年。封禪泰山是中國古代帝王最高的榮耀——它意味著皇帝已經完成了「受命於天」的全部儀式，達到了權力的巔峰。但在巔峰之後，武帝面對的是一個更現實的問題：我已經證明了我是「受命於天」的皇帝——但我的帝國需要誰來治理？他把目光從「天」轉回了「人」——從「封禪」轉向了「求賢」。這種從最高榮耀回歸務實的轉變，本身就說明了武帝的清醒。

## 三、原文結構與核心分析

### 3.1 〈武帝求茂材異等詔〉的結構

這道詔書極簡——全文不足兩百字——但信息密度極高，層層推進、一氣呵成：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：提出命題** | 「蓋有非常之功，必待非常之人」——要建立不平凡的功業，必須依靠不平凡的人。開篇即為全詔的靈魂——武帝不談道德、不談禮法，直接切入主題：我要做大事，我需要能人 | 一句話就確立了全文的基調——這不是一次普通的官員選拔，這是一次打破常規的人才狩獵。武帝把話說得非常清楚：常規標準（德行、禮儀、出身）在面對「非常之功」時，都要讓路 |
| **第二層：說明標準** | 「故馬或奔踶而致千里，士或有負俗之累而立功名」——有些馬會踢人、會咬人，但能日行千里；有些人被世俗議論、行為不檢點，但能建功立業。武帝用了一個非常形象的比喻——好馬會踢人——來為那些「有缺點的人才」辯護 | 這是非常大膽的用人宣言。在儒家的傳統中，「修身」是用人的首要標準——一個「修身」不好的人，不值得重用。武帝卻公開說：不一定——一個人可能在私德上有瑕疵，但這不影響他建立卓越的功業。這在中國政治思想史上是一個關鍵的轉折點 |
| **第三層：下達命令** | 「其令州郡察吏民有茂材異等，可為將相及使絕國者」——命令各州郡考察官吏和百姓——那些有傑出才能、可以擔任將相或出使遙遠國家的人——全部推薦上來 | 最後一句點明了「非常之人」的具體用途——「將相」（治理國家）、「使絕國」（出使偏遠國家，即外交使命）。注意「使絕國」三個字——這說明武帝心中的人才是有「國際視野」的——他要的不只是治國的文官，還有敢於深入西域、開拓未知世界的冒險家 |

> **軍師按**：這道詔書雖然簡短，但它的結構非常精妙——從「論」到「喻」再到「令」：先給出核心論點（非常之功需要非常之人），再用比喻說明理由（好馬會踢人、好人有缺點），最後下達具體命令（各州郡考察推薦）。整篇詔書在不到兩百字中完成了從「說服」到「執行」的全部過程——沒有一句廢話。

### 3.2 核心段落節錄

> 蓋有非常之功，必待非常之人。

——全詔書的靈魂。十個字，說透了一切。武帝不講出身、不講資歷、不講道德——他只講「才能」與「功業」的對應關係。你想要了不起的成就嗎？那就別用普通人。

> 故馬或奔踶而致千里，士或有負俗之累而立功名。

——武帝用了一個非常生動的類比：駿馬會踢人，但能日行千里；能人會被人議論，但能建功立業。這句話是對儒家中庸之道的挑戰——在儒家看來，「修身」是一切的前提——一個連自己都管不好的人，怎麼能治理國家？武帝說：不一定。那些「負俗之累」（被世俗指責的缺點）的人，恰恰可能是最能成事的人。

> 夫泛駕之馬，跅弛之士，亦在御之而已。

——「跅弛之士」這個成語的出處。「泛駕之馬」指不馴服的馬——喜歡亂跑、不聽指揮；「跅弛之士」指不拘小節、行為放縱的人。武帝說：這樣的人，關鍵在於「御之而已」——看你怎麼駕馭、怎麼用。他不是否定缺點——他是說：缺點可以被管理——只要你懂得怎麼用他。

> 其令州郡察吏民有茂材異等，可為將相及使絕國者。

——詔書的執行指令。「使絕國」是一個關鍵詞——它揭示了武帝對「國際人才」的需求。打通西域之後，漢朝需要大量能出使偏遠國家的人才——這些人要有膽識、要有口才、要敢於面對未知的危險。一般循規蹈矩的士大夫做不了這種事——只有「跅弛之士」才敢去。

> ——《漢書·武帝紀》

## 四、關鍵人物

### 4.1 漢武帝劉徹——帝國的野心與用人哲學

| 項目 | 內容 |
|------|------|
| **身份** | 西漢第七位皇帝（在位：西元前 141–87 年），漢景帝劉啟之子。即位時年僅十六歲，在位五十四年——是中國歷史上在位時間最長的皇帝之一 |
| **生平核心** | 武帝是中國歷史上最具開拓精神的皇帝之一。他即位後，改變了漢初「無為而治」的國策，轉為積極進取——對外征伐匈奴、開通西域、征服南越、東越、朝鮮、西南夷——將漢帝國的疆域擴大了將近一倍。對內加強中央集權——推恩令削弱諸侯、獨尊儒術統一思想、鹽鐵專賣控制經濟。他的一生，就是一部帝國的擴張史 |
| **用人風格** | 武帝是中國歷史上最「不拘一格」用人的皇帝之一——衛青原是平陽公主家的騎奴、霍去病是衛青的外甥（私生子出身）、金日磾是匈奴休屠王太子（俘虜出身）、張騫是郎官（冒險家型人才）、桑弘羊是商人子弟（理財專家）——武帝用人，不看門第、不看出身、不看私德——只看你能不能成事。這道求賢詔，就是這種用人哲學的理論總結 |
| **晚年的反思** | 武帝晚年（征和年間，前 92–前 87 年）由於連年征戰、巫蠱之禍（前 91 年）等事件，帝國陷入危機。他在去世前兩年頒布了著名的「輪臺詔」（前 89 年）——深刻檢討自己的過失，宣布停止征伐、與民休息。從〈求茂材異等詔〉（前 106 年）到〈輪臺詔〉（前 89 年）——這十七年間——武帝從一個「開拓者」變成了一個「反思者」——這是中國歷史上最偉大的人物蛻變之一 |

**軍師評**：漢武帝是一個極其複雜的人。他雄才大略，但也窮兵黷武；他識人善任，但也殘忍多疑（晚年殺了不少人，包括自己的太子劉據）。他的求賢詔寫得漂亮——但歷史證明，他晚年對「人才」的態度發生了巨大的轉變：前期的「非常之人」被他重用（衛青、霍去病），晚期的「非常之人」卻被他懷疑（江充就是一個「跅弛之士」——利用武帝的信任製造了巫蠱之禍）。這說明了一個深刻的道理：用人的哲學再漂亮，最終還是取決於用人者自己的心態——當一個開拓者變得焦慮時，他曾讚美的「非常之人」就變成了他恐懼的對象。

### 4.2 「跅弛之士」——被詔書召喚的非常規人才

| 項目 | 內容 |
|------|------|
| **字面含義** | 「跅」——放縱；「弛」——鬆懈。「跅弛之士」即行為放縱、不拘禮法的人。武帝用這個詞來指代那些不被主流社會接受、但實際上非常有能力的人——他對這些人喊話：你們的缺點不重要，來為朝廷效力 |
| **在詔書中的定位** | 武帝將「跅弛之士」比喻為「泛駕之馬」——難以駕馭、不受控制的馬——但這樣的馬一旦被駕馭，就能日行千里。這個比喻極富洞察力——天才往往難以管理，但他的產出可以遠超常人。關鍵不是抱怨天才的缺點——而是找到駕馭他的方法 |
| **歷史上的代表人物** | 衛青（騎奴出身，私生子）、霍去病（年輕氣盛，不學兵法）、主父偃（早年貧困，為人刻薄）、張騫（萬里出使，被匈奴扣留十餘年）——這些人如果按「孝廉」標準來選拔——一個都選不上。但正是這些「跅弛之士」，成就了漢武帝的帝國偉業 |
| **文化影響** | 「跅弛之士」後來成為中國文化中的一個經典詞語——用來形容那些不受世俗束縛、有特殊才能的人。在後世文學中，這個詞經常出現在對「狂士」「奇才」的描述中——從魏晉名士到唐代詩人李白，都可以被歸入「跅弛之士」的範疇 |

**軍師評**：「跅弛之士」這個概念之所以重要——因為它代表了一種對「人才」的全新理解。在此之前，中國社會衡量人才的標準是「德」——忠孝節義、謹言慎行。武帝說：不夠。在某些特殊的時代——比如帝國擴張的時代——「能」比「德」更重要。一個能打勝仗但驕傲自滿的將軍，比一個老實本分但平庸的將軍更有價值。這不是說道德不重要——而是說在不同的歷史階段，優先級不同。武帝的深刻之處在於：他知道自己的時代需要什麼樣的人，所以他敢於打破常規去尋找這樣的人。

## 五、軍師核心分析——「非常之功」與「常規標準」的對抗

### 5.1 「非常之功，必待非常之人」——一個悖論的揭示

武帝這道詔書的核心，在於揭示了一個深刻的悖論：

| 面向 | 分析 |
|------|------|
| **常規的用人標準** | 在任何穩定的社會中，人才選拔都傾向於選擇那些「符合規範」的人——品行端正、言行一致、遵守規則。這種標準的優點是「可預測」——你大體知道這個人會怎麼做。缺點是「平庸」——符合規範的人通常不會做出驚人之舉 |
| **非常之功的邏輯** | 但「非常之功」——開疆拓土、推翻舊制、創造前所未有的局面——恰恰需要那些「不符合規範」的人。因為「做大事」本身就意味著——你需要打破規則、挑戰常識、承受別人無法承受的風險。一個循規蹈矩的人不可能開創前所未有的事業——因為前所未有的事業本身就意味著你必須走出一條前人沒有走過的路 |
| **武帝的解法** | 武帝清楚地意識到這個悖論——所以他說：「馬或奔踶而致千里。」好馬之所以是好馬，不是因為它聽話——而是因為它跑得快。如果你用「聽話」的標準來選馬——你會選到一匹溫馴的劣馬；如果你用「速度」的標準來選馬——你才會選到一匹可能踢人但能日行千里的駿馬。用人也一樣——用「道德」的標準選人，你只能選到安分守己的庸人；用「才能」的標準選人，你才有可能選到能建功立業的能人 |

> **軍師按**：武帝的這個洞察——在兩千年後——仍然深深困擾著所有的組織。每一個企業在招聘時都面臨同樣的選擇：你是要一個「聽話」的人，還是要一個「能幹」的人？聽話的人穩定但平庸——能幹的人出色但難以管理。大多數組織的選擇是「中庸」——找一個既聽話又能幹的人。但問題是——在真正需要開拓的時候——這樣的人往往不存在。你必須在「聽話」和「能幹」之間做選擇。漢武帝的答案是堅定的——他選擇「能幹」。他的這道詔書，就是這個選擇的宣言。

### 5.2 武帝求賢與劉邦求賢——兩位創業者的對話

〈武帝求茂材異等詔〉與卷六中收錄的〈高帝求賢詔〉（前 196 年）——相隔近九十年——可以形成有趣的對照：

| 面向 | 劉邦〈求賢詔〉 | 武帝〈求茂材異等詔〉 |
|------|--------------|-------------------|
| **時代背景** | 開國初期——天下初定，百廢待興 | 帝國巔峰——疆域遼闊，制度完備 |
| **核心訴求** | 「賢士大夫有肯從我游者」——來跟我一起治理國家 | 「茂材異等，可為將相及使絕國者」——來幫我做大事 |
| **人才標準** | 賢士大夫——以德行和才能並重 | 茂材異等——偏重才能，不拘小節 |
| **口吻** | 懇切、真誠——「吾能尊顯之」——像創業者招募合夥人 | 果斷、直接——「蓋有非常之功，必待非常之人」——像 CEO 發布戰略指令 |
| **情感底色** | 悲涼——劉邦知道時日無多，為兒子留後路 | 自信——武帝知道帝國如日中天，但他需要更多的人來撐起這個局面 |
| **共同點** | 兩人都清楚地知道：事業的成功，最終取決於人才——而不是權力或財富 |

> **軍師按**：從劉邦到漢武帝——從「賢士大夫」到「跅弛之士」——從「肯從我游者」到「使絕國者」——九十年間，漢朝發生了巨大的變化。劉邦時代的求賢詔，帶有濃厚的「創業」色彩——他需要的是能幫他穩住江山的人；武帝時代的求賢詔，帶有濃厚的「擴張」色彩——他需要的是能幫他開拓世界的人。兩道詔書的對比——既是兩位皇帝個人風格的對比，也是「開國」與「盛世」兩個時代的對比。

### 5.3 「亦在御之而已」——天才的管理學

武帝在詔書中特別強調了一句話——「亦在御之而已」——關鍵在於怎麼駕馭。這四個字揭示了武帝對「人才管理」的深刻理解：

| 面向 | 分析 |
|------|------|
| **「御」的雙重含義** | 「御」——駕馭馬匹。武帝把管理人才比喻為駕馭烈馬。烈馬難馴——但一旦被馴服，它的速度是普通馬無法比擬的。同樣的道理——天才難管——但一旦找到正確的管理方式——他的產出是普通人無法比擬的。武帝不迴避天才的缺點——他正面回應：這些缺點是可以被管理的 |
| **知人善任——武帝的核心能力** | 武帝之所以敢用「跅弛之士」，不是因為他天真地認為這些人都很完美——而是因為他相信自己有能力「駕馭」他們。歷史證明：衛青謙虛謹慎（需要鼓勵），霍去病驕傲果敢（需要放權），主父偃尖刻狠辣（需要警惕）——武帝對每一個人都採取了不同的管理方式——這就是「御之而已」的實踐 |
| **「御」的風險** | 但「御之而已」也隱含了巨大的風險——如果「御」的人本身判斷失誤，或者「御」的力量減弱——烈馬就會反噬。武帝晚年對江充的信任就是一個反面案例——江充也是一個「跅弛之士」，但武帝失去了對他的控制——最終釀成了巫蠱之禍 |
| **對後世的啟發** | 「亦在御之而已」——這六個字是所有管理者的箴言。一個好的管理者不應該抱怨「這個人太難管」——他應該問自己：我有沒有找到正確的方式來管他？天才之所以難管——不是因為天才錯了——而是因為管理者的方法還沒有調整到適合這個天才 |

> **軍師按**：武帝的「御之而已」——與劉邦的「吾不如」——形成了一個有趣的對照。劉邦用人的方式是不如——「我不如你，所以我聽你的」；武帝用人的方式是御——「我能駕馭你，所以你聽我的」。兩者都成功了——但他們成功的路徑不同。劉邦更像一個團隊的「精神領袖」——他用自己的謙虛吸引人才；武帝更像一個帝國的「CEO」——他用自己的能力管理人才。沒有哪一種方式更優——關鍵在於找到適合自己的方式。

## 六、相關篇目與關聯

### 6.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷六·〈高帝求賢詔〉（劉邦） | 同一卷中的兩道求賢詔——相隔九十年——展示了從「創業」到「盛世」的人才需求變化。劉邦的詔書更像一個父親的遺囑——武帝的詔書更像一個企業家的宣言 |
| 卷六·〈文帝議佐百姓詔〉（漢文帝） | 文帝的詔書是「自省」——他在問自己哪裡做得不夠好；武帝的詔書是「進取」——他在尋找能幫他做得更多的人。兩位皇帝的心態截然不同——一個謹慎、一個自信 |
| 卷六·〈武帝本紀贊〉（司馬遷） | 司馬遷對漢武帝的一生的評價——雖然〈武帝本紀贊〉篇幅不長——但其中對武帝的批評與肯定，可以與這道求賢詔相互參照 |
| 卷四·〈諫逐客書〉（李斯） | 李斯說「太山不讓土壤，故能成其大」——人才應該不分國籍、不分出身。武帝的求賢詔——「不問出身、不拘小節」——與李斯的觀念完全一致 |
| 卷五·〈報任安書〉（司馬遷） | 司馬遷為李陵辯護——自己卻因此受宮刑——而李陵正是「跅弛之士」的典型（一個有缺點但勇敢的將軍）。武帝對李陵的態度——從重用（信任他率五千步兵出征）到滅族（認為他投降匈奴不可原諒）——反映了武帝對「非常之人」的愛恨交織 |

### 6.2 經典延伸

- 《漢書·武帝紀》（班固）——〈武帝求茂材異等詔〉的出處——全文通讀，特別是武帝晚年的詔書與政策變化
- 《史記·衛將軍驃騎列傳》（司馬遷）——衛青與霍去病的傳記——是理解武帝「用非常之人」的最佳案例
- 《史記·大宛列傳》（司馬遷）——張騫通西域的故事——是「使絕國」的具體實踐
- 《資治通鑑·漢紀》（司馬光）——對武帝時代的編年記載——可以幫助理解這道詔書的具體歷史背景
- 《鹽鐵論》（桓寬）——記錄了武帝死後關於鹽鐵專賣等政策的大辯論——可以從中看到武帝時代的經濟政策對社會的深遠影響
- 曹操〈求賢令〉（三國）——曹操「唯才是舉」的口號直接繼承了武帝「不拘一格用人才」的精神。曹操與武帝一樣——都是在亂世與征戰中尋找「非常之人」——他們的求賢令也因此有著驚人的相似

### 6.3 現代延伸

- 彼得·德魯克《卓有成效的管理者》——德魯克說：「有效的管理者知道——他們不應該問『這個人跟我合得來嗎？』——而應該問『這個人能做出什麼貢獻？』」——與武帝「馬或奔踶而致千里」的邏輯完全一致
- 吉姆·柯林斯《從優秀到卓越》——柯林斯提出「第五級領導人」的概念——謙虛而堅定——這與劉邦的「吾不如」更接近，而武帝的「御之而已」則更像「第四級領導人」（高效的執行者）
- 傑克·韋爾奇《贏》——韋爾奇在 GE 的「活力曲線」管理——把人才分為 A、B、C 三級——與武帝「選茂材異等」的邏輯有異曲同工之妙：真正優秀的組織，不會被「人情」或「資歷」束縛——而是敢於重用那些真正有能力的人
- 納西姆·塔勒布《反脆弱》——塔勒布提出「反脆弱」的概念——有些系統在混亂中受益——武帝的「非常之人」在某種意義上就是「反脆弱」的人才——他們在動盪與不確定中表現最好，在穩定與常規中反而平庸

## 七、一句話總結（再錄）

> **〈武帝求茂材異等詔〉——中國歷史上最簡短、最直接、最大膽的求賢令。十個字——「蓋有非常之功，必待非常之人」——就講透了人才選拔的最高機密：你想要不平凡的成就，就別用平凡的人。武帝不怕「跅弛之士」有缺點——他怕的是沒有人才可用。他不在乎你的出身、不在乎你的私德、不在乎別人怎麼看你——他只在乎一件事：你能不能幫我建功立業。這道詔書的偉大之處不在於它的文字——而在於它對「人才」的理解：天才的缺點，是天才的一部分。如果你因為一個人有缺點就拒絕他——你也同時拒絕了他可能帶來的偉大成就。兩千年後，當我們還在用「這個人脾氣不好」「那個人心高氣傲」來否定一個人才時——我們應該記起武帝的這句話：「馬或奔踶而致千里。」——好馬會踢人，但能致千里；能人有缺點，但能建功名。關鍵不在於他有沒有缺點——關鍵在於——你會不會用他。**
