## 二、時代背景——鄭國間諜案與逐客令

### 2.1 鄭國間諜案的始末

西元前 246 年，秦王政（未來的秦始皇）即位。韓國為了消耗秦國的國力、使其無力東征，派出了一位著名的水利工程師——鄭國——前往秦國，建議秦國修建一條大型灌溉渠道（即後來的「鄭國渠」）。韓國的如意算盤是：秦國一旦投入大量人力物力修渠，就沒有餘力攻打韓國了。

| 面向 | 說明 |
|------|------|
| **韓國的計謀** | 派遣水利專家鄭國入秦，建議秦王修建從涇水到洛水的大型灌溉渠道。韓國希望此工程能拖垮秦國國力，使其無暇東征 |
| **秦國的發現** | 工程進行到一半時，鄭國的間諜身份暴露。秦國宗室貴族大為震怒——原來韓國人是用水利工程當幌子，意在疲秦 |
| **貴族的反彈** | 秦國宗室大臣趁機向秦王進言：「諸侯人來事秦者，大抵為其主游間於秦耳。」——所有外國人來秦國，都是來當間諜的！請求將所有外國客卿驅逐出境 |
| **李斯的處境** | 李斯是楚國上蔡人，作為「客卿」（外國人在秦國任高官），也在被驅逐的名單之中 |

> **軍師按**：鄭國渠事件是一個完美的「陰謀論引爆點」——一個真實的間諜案（鄭國確實是韓國派來的），被用來攻擊所有「長得像間諜的人」（所有外國客卿）。這種「以個案否定全體」的邏輯，在人類歷史上反覆上演——只要有幾個人做了壞事，就有人會說「他們那一類人都是壞的」。李斯要反駁的，正是這個看似有理、實則荒謬的邏輯。

### 2.2 秦國的兩難選擇

秦王政當時面臨的困境，本質上是一個「開放」與「安全」的兩難：

| 選項 | 主張 | 優點 | 缺點 |
|------|------|------|------|
| **驅逐客卿** | 秦國宗室貴族 | 消除間諜威脅，純化秦國政治 | 失去大量優秀人才——秦國的崛起靠的就是外國人 |
| **保留客卿** | 李斯等外國人才 | 繼續吸引天下英才為秦所用 | 可能仍有間諜潛伏，存在安全隱患 |

> **軍師按**：這個兩難到今天仍然存在——每一個國家、每一個企業、每一個組織都在「開放以吸引人才」和「封閉以確保安全」之間掙扎。李斯給出的答案，是兩千年來最經典的回答：**如果你因為害怕風險而拒絕所有外國人，你拒絕的不只是風險，而是你自己的力量來源。** 因為秦國的霸業，從來就不是靠秦國人自己完成的。

## 三、關鍵人物

### 3.1 李斯（約西元前 280—前 208 年）

| 項目 | 內容 |
|------|------|
| **身份** | 楚國上蔡人，秦國客卿，後為丞相。師從荀子學帝王之術 |
| **核心貢獻** | 輔佐秦始皇統一六國，制定郡縣制、統一文字、統一度量衡 |
| **與本文的關係** | 本文是李斯政治生涯的「生死轉折點」——若此文失敗，他被逐出秦國，就沒有後來的秦相李斯 |
| **性格特質** | 格局宏大、思路清晰、文采出眾、但不甘平庸——少年時見廁中老鼠與倉中老鼠的處境差別，感嘆「人之賢不肖，譬如鼠矣，在所自處耳」，從此立下出人頭地之志 |
| **歷史定位** | 秦帝國制度的總設計師——沒有李斯，秦國的統一可能是另一種面貌 |

**軍師評**：李斯這一生最傳奇的事蹟，不是他做了秦相，而是他在被驅逐的路上寫了一封信，然後這封信讓他變成了秦相。〈諫逐客書〉是「以文改變命運」的最高典範——秦王讀完後說：「把李斯追回來。」——這個決定改變了李斯的命運，也改變了中國的歷史。

李斯在〈諫逐客書〉中展現的，不僅是文采和邏輯，更是一種「大局觀」——他沒有為自己求情（「我可憐，請不要趕我走」），而是為秦國的利益辯論（「趕走外國人，是秦國自己的損失」）。這種把自己放在國家利益之下的論證方式，讓秦王無法拒絕——因為拒絕它就等於承認自己不要國家利益。

### 3.2 秦王嬴政（西元前 259—前 210 年）

| 項目 | 內容 |
|------|------|
| **身份** | 秦國國王（後為秦始皇），中國歷史上第一位皇帝 |
| **與本文的關係** | 本文的讀者和決策者——他讀完後收回逐客令，恢復李斯官職 |
| **性格側寫** | 雄才大略但多疑——他後來焚書坑儒、廢除分封制，但其早年卻是一個能聽進逆耳忠言的「開放型」領導者 |
| **歷史評價** | 中國歷史上最具爭議的帝王之一——統一中國、書同文車同軌的功績與焚書坑儒的暴行並存 |

**軍師評**：秦王政在本文中的角色，與其說是「獨裁者」，不如說是「開明的決策者」。他下了逐客令（受貴族影響），但讀了李斯的信後立刻收回成命——這說明他是一個能「從諫如流」的領導者。一個能聽得進跟自己決策相反的意見、並且願意公開改變決定的領導者，在任何時代都是稀缺資源。

### 3.3 鄭國——引爆逐客事件的關鍵人物

鄭國是來自韓國的水利專家。他的故事有兩個版本的結局：

| 面向 | 內容 |
|------|------|
| **初始身份** | 韓國派往秦國的間諜——名義上幫助秦國修渠，實則消耗秦國國力 |
| **身份暴露後** | 秦國貴族要求殺鄭國。但鄭國辯稱：渠修成了對秦國有百世之利。秦王權衡後釋放了鄭國，讓他繼續修渠 |
| **最終成果** | 鄭國渠全長約 150 公里，灌溉關中平原四萬餘頃農田，使關中成為沃野。秦國因此國力大增，為統一奠定了物質基礎 |
| **歷史諷刺** | 韓國原本想用鄭國「疲秦」，結果反而「強秦」——鄭國渠讓秦國糧食產量大增，最終加速了韓國的滅亡 |

> **軍師按**：鄭國渠的故事是中國歷史最大的一個「反向間諜案」——韓國派去的間諜，最後幫秦國修了一條讓秦國更強大的渠。這與〈諫逐客書〉的核心命題完全一致：你不應該因為一個外國人可能有害，就拒絕所有外國人——因為有些外國人，會幫你變得比你原本更強。

## 四、核心論證——李斯如何說服秦王

### 4.1 原典節錄

> 臣聞吏議逐客，竊以為過矣。昔穆公求士，西取由余於戎，東得百里奚於宛，迎蹇叔於宋，來邳豹、公孫支於晉。此五子者，不產於秦，而穆公用之，并國二十，遂霸西戎。……
>
> 今取人則不然：不問可否，不論曲直，非秦者去，為客者逐。然則是所重者在乎色樂珠玉，而所輕者在乎人民也。此非所以跨海內、制諸侯之術也。
>
> 臣聞地廣者粟多，國大者人眾，兵強則士勇。是以泰山不讓土壤，故能成其大；河海不擇細流，故能就其深；王者不卻眾庶，故能明其德。……
>
> 夫物不產於秦，可寶者多；士不產於秦，而願忠者眾。今逐客以資敵國，損民以益讎，內自虛而外樹怨於諸侯，求國無危，不可得也。

> ——《史記·李斯列傳》

### 4.2 李斯的論證結構

〈諫逐客書〉的論證結構極其精妙，分為「以史為證 → 以物為喻 → 以理推演」三個層次：

| 層次 | 內容 | 分析 |
|------|------|------|
| **第一層：歷史論證** | 列舉秦國四位先王重用外國人的史實——秦穆公用由余、百里奚、蹇叔；秦孝公用商鞅；秦惠王用張儀；秦昭王用范雎 | 用秦國自己的歷史反駁秦國貴族：你們說外國人不可靠？可秦國的霸業正是靠這些外國人建立的。如果當年驅逐外國人，秦國至今還是西陲小國 |
| **第二層：類比論證** | 以「泰山不讓土壤，故能成其大；河海不擇細流，故能就其深」為喻，說明秦王不應排斥外國人才 | 用自然界最直觀的道理來比喻政治——泰山之所以高大，是因為它不拒絕任何一粒泥土；大海之所以深邃，是因為它不拒絕任何一條小溪。統一天下一樣需要吸納各國人才 |
| **第三層：邏輯推演** | 驅逐客卿等於「藉寇兵而齎盜糧」——把武器和糧食借給敵人 | 從反面論證：如果你把這些人才趕走，他們不會回國種田——他們會去為你的敵國效力。你親手把最優秀的人才送給你的敵人，這不是愚蠢，這是自殺 |

> **軍師按**：李斯最精采的一步棋，是他把論證的戰場從「我該不該被趕走」轉移到了「秦國該不該強大」。他沒有說「我是無辜的」（那只是個人求情），他說的是「你趕走外國人，秦國會變弱」（這是國家戰略）。一旦他把論證拉到了這個層次，秦王就不得不認真考慮——因為秦王關心的不是李斯的個人命運，而是秦國的命運。

### 4.3 「泰山不讓土壤」——一個比喻的力量

「泰山不讓土壤，故能成其大；河海不擇細流，故能就其深」——這兩句話是全文最著名的句子，也是中國文學史上最有力量的比喻之一。

| 比喻元素 | 對應的現實 |
|----------|-----------|
| **泰山** | 秦國——想要成為天下霸主 |
| **土壤** | 各國人才——不論來自哪個國家，都是秦國的養分 |
| **河海** | 秦國的國力——吸納越多，實力越強 |
| **細流** | 小國的人才——即使是來自弱國的人才，也能匯聚成秦國的實力 |

這個比喻的力量在於：它用的是「無可爭辯的物理事實」——沒有人會否認泰山是由土壤堆積而成的、河海是由細流匯聚而成的。一旦你接受了這個前提，你就無法否認「秦國的強大也需要各國人才的匯聚」。這就是頂級的類比論證——**用一個無法否認的物理事實，推導出一個無法否認的政治結論。**

## 五、軍師核心分析——開放作為一種國家戰略

### 5.1 為什麼秦國必須依靠外國人？

秦國崛起於西陲，在春秋戰國時期一直被中原諸侯視為「戎狄」。秦國的立國根基，從一開始就不是「秦人」，而是「制度」——商鞅變法為秦國建立了高效的農戰體制，而商鞅本人是衛國人。

| 時期 | 關鍵人才 | 來自 | 貢獻 |
|------|----------|------|------|
| 秦穆公 | 由余（戎人）、百里奚（宛人）、蹇叔（宋人） | 非秦國 | 稱霸西戎，奠定秦國大國地位 |
| 秦孝公 | 商鞅 | 衛國 | 變法圖強，使秦國富國強兵 |
| 秦惠王 | 張儀 | 魏國 | 連橫破縱，瓦解六國合縱 |
| 秦昭王 | 范雎 | 魏國 | 遠交近攻，奠定統一戰略 |
| 秦王政 | 李斯 | 楚國 | 統一天下，建立帝國制度 |

李斯的論證告訴我們一個殘酷但真實的事實：**秦國的「核心競爭力」從來就不是秦國人——而是秦國吸引外國人才的能力。** 秦國的體制之所以優越，是因為它能夠讓每一個來到秦國的人，都發揮最大的作用。如果你把外國人趕走，秦國就只剩下一個空殼——制度再優越，沒有人才來執行，也是白搭。

### 5.2 「藉寇兵而齎盜糧」——最狠的比喻

「藉寇兵而齎盜糧」——把武器借給強盜，把糧食送給小偷。李斯用這個比喻來形容驅逐客卿的後果：

| 秦國驅逐客卿後的結果 | 等同於 |
|---------------------|-------|
| 外國人才被迫離開秦國 | 秦國親手削弱自己 |
| 這些人才回到各國（或投奔敵國） | 秦國親自為敵國輸送人才 |
| 他們在各國用他們的才能對抗秦國 | 秦國把自己的人才變成了敵人的武器 |

這個比喻的力量在於它的「具象化」——它把一個抽象的國家政策（驅逐客卿）變成了一個具體的、荒謬的行為（把武器借給強盜）。沒有人會同意「把武器借給強盜」——所以你也不能同意「驅逐客卿」。

**軍師評**：「藉寇兵而齎盜糧」是中國政治文章中最早的「成本效益分析」——李斯不是在講道德，他是在算帳：你趕走這些人，他們去了敵國，用他們的聰明才智來幫敵國打你——你損失了人才，敵國得到了人才，你的安全狀況不但沒有改善，反而更糟了。這不是道德問題，這是一個數學問題。

### 5.3 〈諫逐客書〉的歷史影響

| 時間 | 影響 |
|------|------|
| **秦王政十年（前 237 年）** | 秦王讀後立即收回逐客令，恢復李斯官職。秦國繼續大量起用外國人才——王翦（頻陽東鄉人，非秦國核心貴族）、蒙恬（齊人後裔）等皆為秦所用 |
| **秦統一後** | 李斯協助秦始皇建立郡縣制、統一文字、統一車軌、統一度量衡——這些政策的本質都是「打破壁壘、促進融合」，與〈諫逐客書〉的開放精神一脈相承 |
| **後世影響** | 〈諫逐客書〉成為中國歷史上論證「開放」的經典文本。歷代政治家在討論人才政策時，幾乎必引「泰山不讓土壤」之喻 |

## 六、一句話總結（再錄）

> **〈諫逐客書〉之所以偉大，不在於它幫李斯保住了官位，而在於它論證了一個跨越兩千年的真理——任何組織的強大，都不是因為它拒絕了什麼，而是因為它吸納了什麼。李斯用他完美的邏輯和磅礡的文采，證明了「開放」不僅是一種胸懷，更是一種戰略。而「泰山不讓土壤，故能成其大；河海不擇細流，故能就其深」，也因此成為中國文化中最具包容精神的經典表達。**

## 七、關聯篇目與延伸閱讀

### 7.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷四·〈范雎說秦王〉 | 同為秦國對待外國人才的經典案例——范雎本人就是外國人（魏人），說服秦王採用「遠交近攻」。李斯延續了這個傳統 |
| 卷四·〈蘇秦以連橫說秦〉 | 蘇秦和張儀都是外國人（蘇秦東周洛陽人、張儀魏人），都在秦國尋求發展。〈諫逐客書〉可視為這些「外國人入秦」現象的終極辯護 |
| 卷四·〈魯仲連義不帝秦〉 | 從反面展現了秦國擴張引發的恐懼——而秦國的擴張速度，正是因為它吸收了各國最優秀的人才 |
| 卷三·〈虞師晉師滅下陽〉 | 春秋時期「假道伐虢」的故事中，虞國因拒絕接納賢才而滅亡——與〈諫逐客書〉探討的「封閉導致的後果」可對照閱讀 |

### 7.2 經典延伸

- 《史記·李斯列傳》（司馬遷）——李斯完整傳記，從「廁中鼠」到秦相到被腰斬的傳奇一生
- 《韓非子·顯學》——李斯的老同學韓非的思想。李斯與韓非的關係是中國思想史上最複雜的糾葛之一——兩人同為荀子學生，但一個助秦統一（李斯），一個抗秦而被殺（韓非）
- 《資治通鑑》卷六（司馬光）——對秦王政朝初年逐客事件的歷史編年記述
- 賈誼〈過秦論〉（古文觀止卷五·漢文）——對秦國由興到亡的全面分析，可與〈諫逐客書〉對照：秦因開放而興，因封閉而亡

### 7.3 現代延伸

- 黃仁宇《中國大歷史》——從宏觀歷史角度分析秦帝國的制度設計，其中對李斯的評價尤為精彩
- 法蘭西斯·福山《政治秩序的起源》——從比較政治學視角分析中國古代官僚制度的形成，李斯的郡縣制設計是其中的關鍵章節
- 凱文·凱利《必然》——關於「開放vs.封閉」在網絡時代的戰略意義，與李斯的核心論證遙相呼應
