# 〈駒支不屈於晉〉背景資料

## ▎座標定位

| 項目 | 內容 |
|------|------|
| **典籍** | 《左傳》·襄公十四年（卷三十二） |
| **時間** | 魯襄公十四年，西元前 559 年 |
| **地點** | 晉國霸權核心——向地（今河南尉氏一帶），晉國主盟諸侯之會 |
| **事件** | 晉國大會諸侯，范宣子（士匄）當眾責難戎子駒支「洩密於楚」，意圖將戎人逐出盟會 |
| **結果** | 駒支從容辯駁，賦《青蠅》詩退下，范宣子「拜，使就席」——賠禮道歉，收場 |
| **出處** | 《古文觀止》卷二·左傳，第 92 篇 |

## ▎時代壓力：霸主秩序下的少數民族死局

### 晉楚爭霸的「夾縫邏輯」

西元前 559 年，正值**晉悼公復霸**後期。晉國在「三駕之役」後壓制楚國，但楚國仍不斷藉助南方少數民族（群蠻、百濮）及西方戎人騷擾晉的盟國。此時華夏諸侯的共識是：戎狄不可信，戎狄是「禽獸」，戎狄是楚國的天然盟友。

### 戎人在春秋話語體系中的位置

- **話語標籤**：「戎狄豺狼，不可厭也」（《左傳》閔公元年）；「戎狄無親而貪」（《左傳》襄公四年）
- **法律地位**：不是「諸侯」，而是「不與於會盟」的邊緣族群。即便駒支的部落（姜戎氏）從晉惠公時代就歸附晉國、世代替晉國守西疆，身份上仍是「非我族類」
- **現實處境**：姜戎氏替晉國抵禦秦國，立有戰功（最著名的是崤之戰，西元前 627 年，姜戎助晉大敗秦軍），但功勞歸功勞，政治聚會上照樣被當成外人踢出去

### 范宣子發難的真面目

范宣子指控駒支洩密——「來！姜戎氏！昔秦人迫逐乃祖吾離于瓜州……今諸侯之事我寡君不如昔者，蓋言語漏泄，則職女之由。」翻譯：以前是晉國收留了你們戎人，現在諸侯聯盟不團結，都是你們洩密害的。

**這不是情報指控，這是驅逐令。** 范宣子要用一個「洩密」的罪名，將戎人從國際權力體系中合法排除。本質是：霸主需要一個內部異己來承擔聯盟鬆散的責任。戎人就是最好的祭品。

## ▎關鍵人物

### 戎子駒支（反歧視的戰略家）

姜戎氏首領。他的反駁完全不是哀求，而是一場**以中原規則打敗中原人**的經典戰例：

1. **先立事實**：戎人「飲食衣服不與華同，贄幣不通，言語不達」，連基本外交溝通工具都沒有，怎麼洩密？
2. **再算舊賬**：列舉晉國對戎人的壓迫——從追逐祖先、侵占草場、壓榨賦役，到現在還要背黑鍋。反過來質問：戎人沒背叛晉國，是誰在背叛誰？
3. **最後以詩收局**：賦《青蠅》——「營營青蠅，止於樊。豈弟君子，無信讒言。」用周代最正統的外交禮儀（賦詩言志），當面諷刺范宣子聽信讒言。

最致命的一擊在於：**一個被認為不會說人話的戎人，用中原最高級的修辭擊敗了中原的執政。** 這讓范宣子完全無法反駁，因為反駁等於承認「戎人會用詩經」——而這本身就推翻了「戎人野蠻」的前提。

### 范宣子士匄（被拉下神壇的霸主）

晉國中軍將、執政大夫。他是晉悼公死後晉國最強勢的政治人物。他的錯誤在於誤判了駒支的水平——以為戎人首領只會磕頭認罪。當駒支行雲流水地賦詩後，范宣子陷入了雙重困境：
- 如果繼續發怒，等於承認自己聽不懂詩經的諷諫；
- 如果當場反駁詩經解釋，等於和一個戎人進行經典辯論——這本身就是自降身分。

所以他只能「拜，使就席」——**鞠躬道歉，請戎人坐回席上。** 這是春秋霸權史上最尷尬的一幕。

## ▎寫作意圖：左傳最精彩的「反歧視」篇章

### 文本的顛覆性

《左傳》的作者（傳統認為左丘明）在這裡做了一件極大膽的事：**讓「華夷之辨」的受害者親自上場辯護，而且讓辯護者完勝。**

在春秋筆法下，戎狄通常是配角或反派——要麼被討伐，要麼被教化。但在這一篇中，駒支是主角，而且是用對手的文化打敗對手。

### 修辭戰術分析

駒支的辯論結構完全符合先秦論辯的最高規格：
- **正名**（先說明戎人身分與溝通限制）
- **訴諸歷史**（列舉戎人對晉的功勞與晉的背信）
- **反客為主**（反問「誰在洩密？」——暗示洩密的可能是晉國大夫自己）
- **詩經背書**（用《詩經》賦詩，這是春秋外交的最高仲裁手段）

### 為什麼這篇千古流傳？

因為它寫出了一個最樸素、最強而有力的真理：**歧視的邏輯，在被歧視者學會用你的語言之後，就不再成立了。** 駒支之所以贏，不是因為他比范宣子更有權力，而是因為他用晉國人自認為最文明的工具，證明了晉國人的不文明。

## ▎一句話總結

**一個被晉國視為「野蠻人」的戎族首領，在中原霸主面前引《詩經》為自己辯護，反而讓熟悉經典的晉國執政啞口無言、賠禮道歉——這是《左傳》全書中最漂亮的一場「以文明反噬文明」的反歧視戰役。**
