## 二、時代背景——司馬遷的「天道懷疑論」

### 2.1 司馬遷與漢武帝時代的知識氛圍

西漢武帝時期，儒學雖然被定為一尊，但思想的多元性並未完全消失。董仲舒的「天人感應」學說強調天道與人事之間存在對應關係——善有善報，惡有惡報——這是當時官方意識形態的核心命題。然而司馬遷的〈伯夷列傳〉，從根本上挑戰了這個命題：

| 面向 | 官方意識形態（董仲舒） | 司馬遷的質疑 |
|------|----------------------|--------------|
| **天道觀** | 天人感應——天會獎善罰惡 | 歷史上善人往往不得善終，惡人往往壽終正寢——「余甚惑焉」 |
| **歷史觀** | 歷史服從於道德秩序——善惡到頭終有報 | 歷史是混亂、不公義的——道德秩序只是人的一廂情願 |
| **寫作態度** | 歌頌聖王、譴責暴君——維護政治正確 | 記錄真相——即使真相令人不安 |

> **軍師按**：司馬遷寫〈伯夷列傳〉的時候，他自己已經經歷了「李陵之禍」——一個忠誠的人被冤枉判刑，受宮刑之辱。這不是書齋裡的哲學思辨，這是他用身體和尊嚴換來的血淚之問。伯夷、叔齊的命運與司馬遷自己的遭遇交織在一起——〈伯夷列傳〉中的「天道懷疑」，是司馬遷對自己人生的回應。

### 2.2 「天道無親，常與善人」——信條還是騙局？

「天道無親，常與善人」出自《老子》第七十九章，後來成為中國傳統社會中被反覆引用的「道德樂觀主義」命題——做好事有好報，天是不會辜負好人的。但司馬遷在〈伯夷列傳〉中，用歷史事實和邏輯推理，對這個命題進行了徹底的拆解：

| 論證層次 | 內容 |
|----------|------|
| **第一層：舉證——伯夷、叔齊的善行與結局** | 伯夷、叔齊是孤竹國的兩位公子——父親想要立叔齊為國君，但叔齊認為伯夷是長兄，應該繼位。伯夷說「這是父親的命令」，不肯違背父命。兩兄弟互相推讓，最後一起逃走了。他們後來聽說西伯昌（周文王）善待老人，便去投奔。但到了那裡，周武王已經載著文王的靈牌去討伐商紂。伯夷、叔齊攔住馬頭進諫：「父親死了不安葬，卻動干戈，這能叫孝嗎？以臣子的身份去殺君王，這能叫仁嗎？」周武王的手下想要殺他們，姜太公說：「這是義人啊！」扶他們離去。武王平定殷商之後，天下歸順周朝。伯夷、叔齊以做周朝的臣民為恥，為了堅守節義，不吃周朝的糧食，隱居在首陽山，採摘薇菜充飢。快要餓死的時候，他們作了一首歌：「登彼西山兮，採其薇矣。以暴易暴兮，不知其非矣。神農、虞、夏忽焉沒兮，我安適歸矣？于嗟徂兮，命之衰矣！」——就這樣餓死在首陽山上。 |
| **第二層：類比——盜跖的惡行與結局** | 與伯夷、叔齊形成鮮明對比的是盜跖——傳說中的大盜，率領數千人橫行天下，「暴戾恣睢」，殺人無數，食人肝肉——然而這樣一個惡貫滿盈的人，卻「竟以壽終」——安享天年。 |
| **第三層：歸納——歷史上的善惡報應是否成立？** | 司馬遷進一步擴大考察範圍——他列舉了歷史上的一系列現象：顏回（孔子最優秀的學生）好學守善，卻「亞聖」早死；而盜跖這樣的惡人卻壽終正寢。如果天道真的獎善罰惡，這該如何解釋？ |
| **第四層：結論——「余甚惑焉」** | 司馬遷坦承：「余甚惑焉，儻所謂天道，是邪非邪？」——我非常困惑啊，所謂的天道，到底是對的還是錯的？他沒有給出答案——因為他根本不知道答案。但他用這句話表明了一個態度：作為歷史學家，他寧可困惑，也不撒謊。 |

> **軍師按**：司馬遷的「余甚惑焉」是中國思想史上最誠實的一句話之一。在一個要求「天道」成為政治正確的時代，他公開承認自己對「天道」的懷疑——這不僅是哲學上的勇氣，更是政治上的危險。〈伯夷列傳〉之所以被放在七十列傳之首，正是因為司馬遷要用這篇「懷疑論」的宣言，為整部《史記》定調：這是一部記錄真相的書，不是一部歌功頌德的書。

## 三、原文結構與核心論點

### 3.1 文章結構

〈伯夷列傳〉是《史記》七十列傳的第一篇，全文以議論為主，敘事為輔。其結構如下：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：開宗明義——引出伯夷、叔齊的典故** | 司馬遷開篇引用了孔子對伯夷、叔齊的評價：「伯夷、叔齊，不念舊惡，怨是用希」「求仁而得仁，又何怨乎？」——孔子對伯夷、叔齊評價極高。但司馬遷接著寫道，他到箕山（傳說中許由隱居的地方）時，看到當地人還在紀念許由——然而許由的事蹟在經典中幾乎找不到記載。由此引出一個問題：什麼樣的人值得被歷史銘記？ | 以「求仁得仁，又何怨」為引子，為後文的「天道懷疑」鋪路 |
| **第二層：核心論證——善惡報應的質疑** | 這就是前述的核心段落——伯夷、叔齊的善行與餓死結局，對比盜跖的惡行與壽終正寢。司馬遷由此對「天道無親，常與善人」提出根本性質疑。這是全文的高潮。 | 提出全文核心問題——天道真的公平嗎？ |
| **第三層：進一步思辨——「怨」與「不怨」** | 司馬遷回到孔子的「又何怨乎？」這個命題。他沒有直接反駁孔子，而是用了一個微妙的寫法——他引用了伯夷、叔齊餓死前作的那首歌：「登彼西山兮，採其薇矣。以暴易暴兮，不知其非矣。神農、虞、夏忽焉沒兮，我安適歸矣？于嗟徂兮，命之衰矣！」——這首歌中充滿了對命運的感嘆和不甘。司馬遷用這種方式，委婉地表達了「他們的確有怨」的態度——從而與孔子的「何怨」形成了微妙的張力。 | 以「歌」代「論」——用伯夷、叔齊自己的詩來回答「何怨」的問題——有怨，當然有怨 |
| **第四層：總結——「君子疾沒世而名不稱焉」** | 司馬遷引用了孔子的「君子疾沒世而名不稱焉」——君子擔心的是死後名字不被後人記住。然後他對比了伯夷、叔齊與盜跖——「伯夷、叔齊雖賢，得夫子而名益彰……盜跖日殺不辜，肝人之肉，暴戾恣睢，聚黨數千人橫行天下，竟以壽終，是遵何德哉？」——伯夷、叔齊的「名」之所以流傳，是因為得到了孔子的讚揚。但那些所謂的「天道報應」，誰能說得清楚呢？ | 回歸到史學家的使命——記錄那些值得記錄的人，不問天道是否公平 |

> **軍師按**：〈伯夷列傳〉的結構非常特別——它不是傳統意義上的「傳記」，而是一篇「論證散文」。司馬遷用伯夷、叔齊的故事作為證據，展開了一場關於「天道」的法庭辯論。他自己扮演了辯護律師——為那些被天道辜負的善人辯護。這種寫法在中國史學中是開天闢地的——史記不再只是記錄發生了什麼，而是追問「為什麼」以及「這公平嗎」。

## 四、關鍵人物

### 4.1 伯夷與叔齊——讓國與餓死

| 項目 | 內容 |
|------|------|
| **身份** | 孤竹國（殷商時期諸侯國，在今河北盧龍一帶）國君的兩位公子 |
| **讓國故事** | 孤竹國君想要立叔齊為太子。國君去世後，叔齊說：「該由長兄伯夷繼位。」伯夷說：「這是父親的命令，我不能違背。」於是伯夷逃走。叔齊也不肯繼位，也逃走了。孤竹國人只好立了中間的兒子為國君 |
| **諫伐紂** | 伯夷、叔齊聽說西伯昌（周文王）善待老人，便去投奔。到了那裡，周武王正載著文王的靈牌討伐商紂。兩兄弟攔住馬頭進諫：「父死不葬，爰及干戈，可謂孝乎？以臣弒君，可謂仁乎？」——周武王的手下要殺他們，姜太公說：「這是義人啊！」扶他們離去 |
| **餓死首陽山** | 周武王平定殷商後，天下歸順。伯夷、叔齊以做周朝的臣民為恥，不吃周朝的糧食，隱居首陽山採薇而食。餓死前作歌：「以暴易暴兮，不知其非矣」——諷刺周武王用暴力取代商紂的暴力，卻不覺得自己錯了 |
| **歷史定位** | 被儒家尊為「聖之清者」（孟子語）——具有堅定的道德原則、不與世俗妥協、寧可餓死也不違背自己的信念。他們的「讓國」被視為謙讓美德的極致體現；他們的「不食周粟」被視為忠於故國（殷商）的象徵；他們的「餓死首陽」被視為「捨生取義」的典範 |

**軍師評**：伯夷、叔齊的歷史形象，其實是孔子和司馬遷共同塑造的。在孔子之前，這兩兄弟不過是邊遠小國的兩位公子，名不見經傳。孔子對他們的讚揚（「求仁而得仁，又何怨乎？」）讓他們進入了中國思想的核心版圖。而司馬遷的〈伯夷列傳〉則將他們從道德楷模提升為「天道質疑」的關鍵證據——如果他們這樣的好人都餓死了，天道何在？伯夷、叔齊在司馬遷筆下，不再是單純的道德標本，而是一個尖銳的質問。

### 4.2 司馬遷——天道的「原告」

| 項目 | 內容 |
|------|------|
| **身份** | 西漢史學家、文學家，太史令司馬談之子 |
| **核心貢獻** | 以一人之力完成《史記》——中國第一部紀傳體通史 |
| **《史記》結構** | 十二本紀、三十世家、七十列傳、十表、八書——〈伯夷列傳〉位列七十列傳之首，具有全書「序論」的性質 |
| **李陵之禍** | 司馬遷為投降匈奴的李陵辯護，觸怒漢武帝，被判死刑——後改為宮刑。這是他人生中的重大轉折——從一個信奉「天道」的樂觀主義者，變成了一個對「天道」充滿懷疑的悲觀主義者 |
| **本文與個人的關聯** | 〈伯夷列傳〉中的「天道懷疑」與司馬遷自己的經歷密不可分——他自己就是一個「善人不得善終」的活證據。他在〈報任安書〉中痛陳「太上不辱先，其次不辱身……最下腐刑極矣」——他受盡屈辱，卻仍然活著寫完了《史記》。〈伯夷列傳〉中的「余甚惑焉」，不僅是對伯夷、叔齊命運的困惑，更是對自己命運的困惑 |

**軍師評**：理解〈伯夷列傳〉的關鍵，在於理解司馬遷寫這篇文章時的心境。一個受過宮刑的人，一個被皇帝羞辱的人，一個眼睜睜看著好人受苦、惡人逍遙的人——他要如何面對「天道無親，常與善人」這句話？他不能公開罵皇帝（那是找死），也不能撒謊說天道真的公平（那違背良心）。於是他用了一種極為巧妙的方式——他引用了古代的伯夷、叔齊，說「余甚惑焉」。這個「惑」字，既是對天道的質疑，也是對時代的控訴，更是對自己命運的嘆息。

## 五、核心分析——「天道失靈」的史學意義

### 5.1 為什麼〈伯夷列傳〉是七十列傳之首？

〈伯夷列傳〉不是《史記》中篇幅最長的列傳，不是文采最華美的列傳，甚至不是傳統意義上最「典型」的列傳——但司馬遷偏偏把它放在七十列傳的第一篇。這不是巧合，而是一個刻意的安排：

| 原因 | 分析 |
|------|------|
| **全書的價值宣言** | 七十列傳是《史記》的主體部分，記錄了從上古到漢代各類人物的生平。司馬遷在第一篇就告訴讀者：這不是一本歌功頌德的書，而是一本記錄真相的書——即使真相令人不安。他用「余甚惑焉」為整部《史記》定下了基調：歷史學家的責任不是證明天道的公平，而是記錄天道的失靈 |
| **對官方意識形態的挑戰** | 漢武帝時代的官方意識形態強調「天人感應」——善有善報，惡有惡報。司馬遷在《史記》的第一篇列傳中就公開質疑這個命題，這本身就是一種政治態度的表態：他拒絕用「天道」來粉飾現實中的不公 |
| **回應孔子的權威** | 孔子說伯夷、叔齊「求仁而得仁，又何怨乎」——孔子認為他們是滿足的，沒有怨恨。但司馬遷用他們餓死前所作的詩（充滿了不甘和哀嘆）來證明：他們有怨恨。這是對孔子權威的溫和挑戰——司馬遷不敢也不願直接否定孔子，但他用史料中的具體證據，展示了比孔子更接近真相的判斷 |
| **為全書的「悲劇人物」建立框架** | 《史記》中充滿了悲劇人物——屈原、賈誼、李廣、韓信——這些有功有德的人，結局都是悲慘的。〈伯夷列傳〉從一開始就告訴讀者：不要期待一個「善有善報」的歷史敘事——歷史不是童話，好人常常沒有好報。這為後續的所有悲劇人物提供了一個理解框架 |

> **軍師按**：把〈伯夷列傳〉放在七十列傳之首，是司馬遷在告訴每一個讀者——如果你期待一部「天道公平」的歷史書，請合上這本書，去看別人的作品。這是我司馬遷的書，我記錄的是真相——真相從來不溫柔。

### 5.2 「余甚惑焉」——中國思想史上最誠實的感嘆

「余甚惑焉，儻所謂天道，是邪非邪？」——這十六個字，是中國思想史上最震撼人心的一句話之一：

| 維度 | 分析 |
|------|------|
| **哲學層面** | 這是對「道德宇宙論」的徹底質疑——如果宇宙中存在一個道德秩序（天道），為什麼善人不得好報？要麼天道不存在，要麼天道不公平——無論哪一種結論，都是對傳統信仰的顛覆。司馬遷沒有選擇任何一個結論，而是選擇了「困惑」——這是最誠實的態度 |
| **史學層面** | 司馬遷的「困惑」不是消極的懷疑主義，而是積極的求真精神——正因為天道不可測，所以歷史學家更應該忠實記錄人事。既然無法證明天道公平，那就證明人事的真實——這是司馬遷史學的核心方法論 |
| **政治層面** | 在漢武帝時代，質疑「天道」是一件極其危險的事情——因為皇帝自稱是「天子」，天道的代言人。質疑天道就是在質疑皇帝的合法性。司馬遷用「余甚惑焉」這種看似謙虛的語言，包裹了一個極其尖銳的政治質問——這是他從李陵之禍中學到的生存智慧 |
| **人生層面** | 司馬遷的「惑」也是對自己命運的困惑——他忠於職守、為李陵說公道話，卻換來了宮刑之辱。他做錯了什麼嗎？沒有。那為什麼他要承受這樣的命運？他不知道答案——但他選擇了活下來，把問題寫進歷史，讓後人去思考 |

**軍師評**：兩千年後，我們仍然無法回答司馬遷的問題。天災人禍仍然在發生，好人依然在受苦，惡人依然在享福——「天道無親，常與善人」這句話，在今天和兩千年前一樣，既無法被證實，也無法被否定。但司馬遷留給我們的不是一個答案，而是一個態度——困惑是可以的，質疑是正當的，歷史學家的任務不是替天行道，而是記錄那些被天道辜負的人。這種態度，比任何答案都更有價值。

## 六、軍師核心分析——「史記的靈魂在於困惑」

### 6.1 〈伯夷列傳〉在《古文觀止》卷五中的位置

《古文觀止》卷五收錄了多篇《史記》中的重要篇章，而〈伯夷列傳〉在其中處於一個特殊的位置：

| 篇目 | 與〈伯夷列傳〉的關聯 |
|------|---------------------|
| 卷五·〈孔子世家贊〉 | 孔子世家贊是對孔子一生的總結——孔子在生前不得志，死後卻被尊為至聖——這是否也算一種「天道」的補償？但伯夷、叔齊呢？他們沒有孔子的運氣——他們的「名」固然流傳了下來，但他們的「命」卻沒有得到補償 |
| 卷五·〈屈原列傳贊〉 | 屈原忠君愛國，卻在汨羅江沉江而死——和伯夷、叔齊一樣，是好人的悲劇。司馬遷把伯夷列傳放在七十列傳之首，把屈原列傳放在中間——這兩篇形成了深刻的呼應：歷史中的好人，大多沒有好下場 |
| 卷五·〈外戚世家序〉 | 外戚世家講的是后妃對國家的影響——但從更高的層面看，它和伯夷列傳共享同一個命題：權力和道德之間沒有必然的關係。后妃的賢德不一定帶來國家的興盛（因為還有其他因素），善人的善行也不一定帶來善終——世界是複雜的，不是道德童話 |
| 卷五·〈刺客列傳〉、〈遊俠列傳〉等 | 這些列傳中的「邊緣人物」——刺客、遊俠——在某種意義上也是「天道失靈」的產物。在一個正常的社會中，正義應該由制度和官府來維護；但在一個天道失靈的世界中，人們只能依靠個人的義氣和膽識來追求公道 |

> **軍師按**：〈伯夷列傳〉是整部《古文觀止》卷五的「精神起點」——從這裡開始，司馬遷的一系列篇章都在追問同一個問題：在一個沒有天道的世界裡，人應該如何活著？伯夷、叔齊選擇了堅守原則而死；屈原選擇了忠誠而死；刺客和遊俠選擇了用自己的方式追求公道——他們都是不同形式的回答。而司馬遷本人選擇了——寫下來。讓後人知道，這個世界上曾經有這麼一群人，在困惑中堅持了自己的信念。

### 6.2 司馬遷的「不確定性史學」

〈伯夷列傳〉最了不起的地方，是司馬遷建立了一種「不確定性史學」——他坦白地告訴讀者：我不知道天道是否公平，我不知道好人為什麼受苦，我不知道惡人為什麼享福——但我知道一件事：這些事情真實發生過，我把它記下來了。

| 傳統史學 | 司馬遷的史學 |
|----------|-------------|
| 歷史是道德教訓的載體——「以史為鑑」——歷史的意義在於提供榜樣和警示 | 歷史是真相的載體——即使真相沒有「教育意義」，也應該被記錄——因為真相本身就是意義 |
| 天道是歷史敘事的前提——善有善報，惡有惡報——歷史的結局必須符合道德秩序 | 天道是歷史敘事的問題——不要假設天道存在——先記錄事實，再提出問題——讓後人去思考答案 |
| 歷史學家的角色是「裁判」——斷定是非善惡 | 歷史學家的角色是「證人」——記錄所見所聞——把判斷留給後人 |

**軍師評**：司馬遷的「不確定性史學」，在今天看來仍然極具現代性。現代歷史學的核心任務——不是提供「歷史的教訓」，而是記錄「歷史的真相」——司馬遷在兩千年前就已經實踐了這個原則。他沒有把《史記》寫成一本道德教科書，而是寫成了一本「人類行為的記錄」——好人不好報、惡人不惡報——這些事實讓後世每一個讀者都感到不安，但這也是歷史學的尊嚴所在：它不討好任何人的期待。

## 七、一句話總結（再錄）

> **〈伯夷列傳〉是整部《史記》的價值宣言——歷史學家的責任不是歌頌天道，而是記錄天道失靈的時刻。司馬遷在七十列傳的開篇就向讀者宣告：這是一部敢於困惑的書，一部敢於質疑的書，一部寧可承認「我不知道」也不願撒謊的書。伯夷、叔齊的悲劇，不僅是兩千年前的故事——它提醒每一個時代的人：天道不會自動站在善人這邊，但記錄歷史的人，必須站在真相這邊。兩千年過去了，我們仍然無法回答司馬遷的問題——但正因為無法回答，才更需要像他那樣，誠實地記錄、誠實地困惑、誠實地發問。**

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈孔子世家贊〉（司馬遷） | 同為《史記》中的議論性篇章——孔子世家贊肯定孔子精神的不朽，而伯夷列傳則質疑天道的公平——兩篇合看，可以看到司馬遷對「什麼值得被銘記」的完整思考 |
| 卷五·〈屈原列傳贊〉（司馬遷） | 屈原的悲劇與伯夷、叔齊的悲劇如出一轍——好人不得好報——兩篇共同構成了《史記》的「悲劇敘事」主線 |
| 卷五·〈外戚世家序〉（司馬遷） | 外戚世家的「制度分析」與伯夷列傳的「天道質疑」——一個看制度，一個看天命——共同展示了司馬遷對世界運作邏輯的全面思考 |
| 卷四·〈逍遙遊〉（莊子） | 莊子的「逍遙」與伯夷、叔齊的「堅守」——兩種面對命運的態度：一個選擇超越，一個選擇對抗——都是對不公平世界的回應 |
| 卷三·〈報任安書〉（司馬遷） | 這是理解〈伯夷列傳〉最重要的輔助文本——司馬遷在〈報任安書〉中詳細闡述了自己受宮刑後的心路歷程，以及他為什麼選擇活下來完成《史記》。「人固有一死，死有重於泰山，或輕於鴻毛」——這段話直接回答了〈伯夷列傳〉中的「天道質疑」：既然天道不可靠，那就用自己的筆來創造意義 |

### 8.2 經典延伸

- 《史記·伯夷列傳》（司馬遷）——全文閱讀——體會司馬遷的議論筆法，特別是最後一段的「天道之問」
- 《史記·太史公自序》（司馬遷）——司馬遷對自己撰寫《史記》動機和體例的說明——其中提到〈伯夷列傳〉作為七十列傳之首的原因
- 《報任安書》（司馬遷）——司馬遷受刑後寫給朋友的信——全文記錄了他的屈辱、痛苦、困惑和堅持——是理解〈伯夷列傳〉背後情感的第一手資料
- 《孟子·萬章下》（孟子）——孟子論伯夷為「聖之清者」——這是儒家對伯夷、叔齊最早的經典定位
- 《論語·公冶長》（孔子）——孔子論伯夷、叔齊「不念舊惡，怨是用希」——司馬遷在〈伯夷列傳〉中反覆引用的孔子之言

### 8.3 現代延伸

- 卡繆《薛西弗斯的神話》——關於「荒謬」的哲學——在一個沒有意義的宇宙中，人如何活下去？——與伯夷、叔齊在一個不公義的世界中堅守原則，形成了跨越時空的對話
- 漢娜·鄂蘭《平凡的邪惡》——關於「邪惡的平庸性」——為什麼好人受苦，而普通人可以參與邪惡？——司馬遷對盜跖的困惑，在現代世界找到了新的表達形式
- 維克多·弗蘭克《活出意義來》——集中營倖存者對生命的思考——在極端苦難中如何找到意義——與司馬遷在宮刑後堅持完成《史記》的心境有著深刻的共鳴
