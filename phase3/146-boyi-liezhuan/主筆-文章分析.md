## 1. 文章結構分析

〈伯夷列傳〉是《史記》七十列傳之首，但它的寫法與傳統列傳截然不同——全文以議論為主、以敘事為輔。司馬遷不是為了記錄伯夷、叔齊的生平才寫這篇文章，而是借他們的故事，提出一個對「天道」的終極質問。

### 分段大意

**第一段：開篇——從許由引出「什麼人值得被銘記」**

> 「夫學者載籍極博，猶考信於六藝……余以所聞由、光義至高，其文辭不少概見，何哉？」

司馬遷開篇不直接寫伯夷、叔齊，而是先談許由、務光——這些堯舜時代的隱士，義行極高，但在經典中幾乎沒有記載。這引出一個問題：為什麼有些人被歷史記住了，有些人沒有？孔子說伯夷、叔齊「求仁而得仁，又何怨乎」——他們被孔子記住了。但那些沒有孔子讚揚的賢人呢？誰來記得他們？

**第二段：伯夷、叔齊的事蹟——讓國與餓死**

> 「伯夷、叔齊，孤竹君之二子也……遂餓死於首陽山。」

這是全文中唯一一段純粹的敘事。司馬遷用極其精煉的筆法，講述了伯夷、叔齊的一生：
- **讓國**：孤竹君欲立叔齊，叔齊讓伯夷，伯夷逃，叔齊亦逃——國人立其中子
- **諫伐紂**：二人叩馬諫武王——「父死不葬，爰及干戈，可謂孝乎？以臣弒君，可謂仁乎？」
- **不食周粟**：武王平殷後，天下宗周，伯夷、叔齊恥之，義不食周粟，隱於首陽山，採薇而食
- **餓死**：作歌後餓死——「以暴易暴兮，不知其非矣」

這一段敘事看似平實，但每一句話都在為後文的「天道質疑」積蓄力量——如此仁義的兩個人，結局是餓死。

**第三段：核心論證——天道真的公平嗎？**

> 「或曰：『天道無親，常與善人。』若伯夷、叔齊，可謂善人者非邪？積仁潔行如此而餓死！……盜跖日殺不辜，肝人之肉，暴戾恣睢，聚黨數千人橫行天下，竟以壽終。是遵何德哉？」

這是全文最核心的段落，也是中國思想史上最震撼的「天道之問」。司馬遷的論證層層推進：

| 層次 | 內容 | 效果 |
|:----:|------|------|
| **引用成說** | 「天道無親，常與善人」——老子之說 | 先樹一個靶子 |
| **正面舉證** | 伯夷、叔齊——「積仁潔行如此而餓死」 | 善人最悲慘的下場 |
| **反面舉證** | 盜跖——「日殺不辜，竟以壽終」 | 惡人最得意的結局 |
| **擴大考證** | 歷史上「操行不軌，專犯忌諱，而終身逸樂，富厚累世不絕」的人比比皆是 | 不是個案，是普遍現象 |
| **最終結論** | 「余甚惑焉，儻所謂天道，是邪非邪？」 | 不給答案——困惑本身就是答案 |

**第四段：回歸孔子——「怨」與「不怨」的微妙張力**

> 「子曰：『道不同不相為謀』，亦各從其志也……『君子疾沒世而名不稱焉。』」

司馬遷回到開篇孔子對伯夷、叔齊的評價——「求仁而得仁，又何怨乎？」但他用了一個極其巧妙的手法：他引用了伯夷、叔齊餓死前所作的歌：「登彼西山兮，採其薇矣。以暴易暴兮，不知其非矣。神農、虞、夏忽焉沒兮，我安適歸矣？于嗟徂兮，命之衰矣！」這首歌充滿了對命運的哀嘆與不甘——「我安適歸矣？」（我該去哪裡？）「命之衰矣！」（命運太衰了！）——司馬遷不直接反駁孔子，而是讓伯夷、叔齊自己的詩來回答：他們有怨，當然有怨。

**第五段：總結——史家的使命**

> 「賈子曰：『貪夫徇財，烈士徇名，夸者死權，眾庶馮生。』……伯夷、叔齊雖賢，得夫子而名益彰……閭巷之人，欲砥行立名者，非附青雲之士，惡能施於後世哉？」

最後一段，司馬遷回到了史學家的使命。他說了一句話：「君子疾沒世而名不稱焉」——君子擔心的是死後名字不被後人記住。伯夷、叔齊之所以被記住，是因為得到了孔子的讚揚。而那些沒有「青雲之士」為他們發聲的普通人，即使比伯夷、叔齊更賢德，也淹沒在歷史的塵埃中。這其實是司馬遷對自己使命的隱喻——他就是那個「青雲之士」，《史記》就是那個讓被天道辜負的人「名彰於後世」的工具。

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| **起** | 從許由引出問題——什麼樣的人值得被銘記？孔子說伯夷、叔齊「求仁得仁，又何怨乎」 |
| **承** | 敘述伯夷、叔齊的事蹟——讓國、諫伐、餓死——積仁潔行而不得善終 |
| **轉** | 核心轉折——「天道無親，常與善人」是真的嗎？以伯夷、叔齊對比盜跖——「余甚惑焉」 |
| **合** | 回歸「名」的命題——君子疾沒世而名不稱——伯夷、叔齊因孔子而名彰，史家為被天道辜負的人留下見證 |

## 2. 寫作技巧分析

### 「以論代傳」——非典型列傳的寫法

〈伯夷列傳〉是《史記》七十列傳中非常特殊的一篇。傳統列傳以人物事蹟為主，而這篇文章中，純粹的敘事只佔了不到三分之一的篇幅，其餘全是議論和引證。司馬遷不是在為伯夷、叔齊立傳，而是在用他們的命運作證據，打一場關於「天道是否公平」的官司。

| 傳統列傳寫法 | 伯夷列傳的寫法 |
|-------------|---------------|
| 以人物生平為主線 | 以論證「天道質疑」為主線 |
| 敘事為主、議論為輔 | 議論為主、敘事為輔 |
| 目的是讓後人了解這個人物 | 目的是讓後人思考這個問題 |
| 結尾通常有「太史公曰」的評論 | 「太史公曰」散布於全文，本身就是正文 |

### 「引而不發」的質疑策略——如何讓困惑比答案更有力量

司馬遷面對的是一個極其敏感的命題——質疑天道，在某種程度上就是在質疑漢武帝時代「天人感應」的官方意識形態。他採取了一種極其高明的策略：

**第一步：引用孔子**——「求仁而得仁，又何怨乎？」——先以孔子的權威定調。

**第二步：用事實說話**——伯夷、叔齊餓死了，盜跖壽終了——這些是事實，孔子也無法改變的事實。

**第三步：用伯夷、叔齊自己的歌回答孔子**——「命之衰矣！」——你的學生說他們不怨，但他們自己明明在怨。

**第四步：坦承困惑**——「余甚惑焉，儻所謂天道，是邪非邪？」——我不下結論，我只是說我不明白。

這個策略的厲害之處在於：司馬遷從頭到尾沒有說「天道不存在」或「天道不公平」——他只是說「我很困惑」。但這個困惑，建立在無可辯駁的事實之上——伯夷、叔齊餓死了，盜跖壽終了。讀者看完之後會自己得出那個結論，不需要司馬遷說出口。

### 「對比法」的極致運用——伯夷、叔齊 vs 盜跖

全文最核心的寫作技巧，是伯夷、叔齊與盜跖的驚人對比：

| 維度 | 伯夷、叔齊 | 盜跖 |
|:----:|-----------|------|
| **行為** | 積仁潔行、讓國、諫伐、不食周粟 | 日殺不辜、肝人之肉、暴戾恣睢 |
| **結局** | 餓死於首陽山 | 竟以壽終 |
| **道德評價** | 聖之清者（孟子語） | 天下大盜 |
| **後世名聲** | 因孔子而名彰 | 惡名昭彰 |
| **天道報應** | 無——善人不得善報 | 無——惡人不受惡報 |

這個對比本身就已經說明了問題——不需要任何修辭，事實就是最有力的論證。

### 「余甚惑焉」——困惑作為最高級的修辭

「余甚惑焉，儻所謂天道，是邪非邪？」——這十六個字是全篇文章的靈魂。它的力量來自三個層面：

1. **誠實的力量**：在一個要求「天道」成為政治正確的時代，司馬遷公開承認自己「惑」——這比任何雄辯都更有震撼力。
2. **開放的力量**：他不給出答案——不是因為他不敢，而是因為他真的不知道。這種開放性，讓這篇文章在兩千年後的今天仍然具有生命力。
3. **反諷的力量**：一個寫了五十二萬字、記錄了三千年歷史的偉大史學家，在面對終極問題時說「我不知道」——這種反諷本身，就是對那些自稱知道天道奧秘的人的最大諷刺。

### 修辭手法一覽

| 手法 | 原文 | 效果 |
|:----|------|------|
| **引用經典** | 「天道無親，常與善人」——老子 | 先樹立一個被社會普遍接受的命題，然後進行質疑 |
| **反問** | 「若伯夷、叔齊，可謂善人者非邪？」 | 用問句引出質疑，而不是直接否定 |
| **對比** | 伯夷、叔齊之餓死 vs 盜跖之壽終 | 道德與命運的斷裂——最直觀的證據 |
| **排比** | 「或擇地而蹈之，時然後出言……然回也屢空，糟糠不厭，而卒蚤夭」 | 列舉一系列「好人不得好報」的現象，累積說服力 |
| **引用詩歌** | 伯夷、叔齊採薇之歌 | 用當事人自己的話反駁孔子「又何怨乎」的說法 |
| **謙虛語氣** | 「余甚惑焉」 | 看似謙卑，實則以退為進——困惑比肯定更有批判力 |
| **格言** | 「君子疾沒世而名不稱焉」 | 用孔子的話收束全文，回歸到史學家的使命 |
| **隱喻** | 「非附青雲之士，惡能施於後世哉？」 | 暗示自己就是那個「青雲之士」——《史記》就是那個讓賢者名垂後世的工具 |

## 3. 佳句賞析

### 「或曰：『天道無親，常與善人。』若伯夷、叔齊，可謂善人者非邪？積仁潔行如此而餓死！」

有人說：「天道沒有偏私，總是幫助好人。」——那麼像伯夷、叔齊這樣的人，算不算好人呢？他們積累仁德、品行高潔到這種地步，卻活活餓死！

這幾句話是全文最鋒利的一刀。司馬遷先用「或曰」引出一個社會公認的命題——「天道無親，常與善人」——然後用一個簡單至極的邏輯將其徹底擊碎：如果天道真的幫助好人，那伯夷、叔齊為什麼會餓死？要麼天道不存在，要麼天道不公平，要麼他們不是好人——但連孔子都說他們是「求仁得仁」的聖人。所以結論只有一個：天道無親這句話是錯的——至少不值得相信。

司馬遷沒有說出這個結論，他只是問了兩個問題——但答案已經清清楚楚。

### 「盜跖日殺不辜，肝人之肉，暴戾恣睢，聚黨數千人橫行天下，竟以壽終。是遵何德哉？」

盜跖每天殺害無辜的人，吃人的肝肉，兇暴殘忍，聚集了幾千人橫行天下——竟然得以壽終正寢。這遵循的是什麼德行？

這句話的殺傷力在於「是遵何德哉？」——如果天道獎善罰惡，盜跖憑什麼「德」得以壽終？司馬遷用了一個極其辛辣的反諷：盜跖確實在「遵德」——只不過他遵循的不是善人之德，而是惡人之德。而更諷刺的是，惡人之德在世間似乎比善人之德更有「回報」。

「竟以壽終」四個字，是對「天道無親，常與善人」最殘酷的嘲諷。

### 「余甚惑焉，儻所謂天道，是邪非邪？」

我非常困惑啊——所謂的天道，到底是對的還是錯的？

這是全文的靈魂，也是中國思想史上最誠實的一句話之一。一個受宮刑之辱的史學家，一個見證了無數善惡報應不公的人，面對「天道」這個終極問題時，坦承自己「惑」。

這個「惑」字蘊含了三層意思：
- **對經典的惑**：老子說天道無親常與善人——但事實擺在眼前，我該相信經典還是相信事實？
- **對命運的惑**：伯夷、叔齊為什麼餓死？盜跖為什麼壽終？我自己為什麼受宮刑？——我不知道。
- **對天道的惑**：如果天道存在，它為什麼不公？如果天道不存在，那世界還有意義嗎？——我無法回答。

司馬遷的偉大之處在於：他寧可困惑，也不撒謊。他寧可承認自己不知道，也不願意用一個虛假的「天道」來安慰自己或讀者。

### 「伯夷、叔齊雖賢，得夫子而名益彰……閭巷之人，欲砥行立名者，非附青雲之士，惡能施於後世哉？」

伯夷、叔齊雖然賢德，但因為得到了孔子的讚揚，名聲才更加顯赫……鄉野之間的普通人，想要磨礪品行、樹立名聲，如果不依附於「青雲之士」，怎麼能流傳到後世呢？

這一段話是司馬遷對自己使命最含蓄也最自信的宣言。「青雲之士」表面上指孔子——伯夷、叔齊因孔子而名彰。但「青雲之士」也可以指司馬遷自己——那些被歷史遺忘的賢人，因為《史記》而得以名垂後世。司馬遷在告訴讀者：天道可能不公平，但歷史學家可以做到一件事——讓那些被天道辜負的人，不被歷史辜負。

### 「登彼西山兮，採其薇矣。以暴易暴兮，不知其非矣。神農、虞、夏忽焉沒兮，我安適歸矣？于嗟徂兮，命之衰矣！」

登上那西山啊，採摘薇菜充飢。用暴力取代暴力啊，卻不覺得自己錯了。神農、虞舜、夏禹的時代忽然過去了啊，我該去哪裡？唉，要走了啊，命運太衰了！

這是伯夷、叔齊餓死前所作的歌，也是全文中除了司馬遷自己的話之外最有力的「證據」。「以暴易暴兮，不知其非矣」——這句直接批評了周武王：你用暴力取代商紂的暴力，卻不認為自己做錯了。「神農、虞、夏忽焉沒兮，我安適歸矣」——他們懷念上古的黃金時代，卻發現自己無處可去。最後一句「命之衰矣」——是對命運最絕望的感嘆。

司馬遷引用這首歌的真正用意，是回應孔子說的「又何怨乎」——伯夷、叔齊自己已經說得很清楚了，他們有怨，怨命運、怨時代、怨暴力。孔子說他們不怨，但他們的歌證明了他們在怨。司馬遷不直接反駁孔子，而是讓事實說話——這是最溫柔也最堅定的反駁。

## 4. 白話改編

學者們讀的書雖然極其廣博，但仍然要以六藝（《詩》《書》《禮》《樂》《易》《春秋》）作為考信的依據。《尚書》雖然從堯舜時代開始記載，但其中關於許由、務光的文字卻極少見到，這是為什麼呢？

孔子曾說伯夷、叔齊「不念舊惡，怨是用希」「求仁而得仁，又何怨乎？」——我對伯夷、叔齊的志向深感感慨。但當我來到箕山（傳說中許由隱居的地方），看到當地人還在紀念許由時，不禁又對那些「神龍見首不見尾」的聖賢產生了嚮往。那些像許由這樣義行極高的人，在經典中卻幾乎找不到記載——這難道不令人感慨嗎？

伯夷、叔齊是孤竹國君的兩個兒子。父親想要立叔齊為太子。父親去世後，叔齊讓給伯夷。伯夷說：「這是父親的命令啊。」於是伯夷逃走了。叔齊也不肯即位，也逃走了。孤竹國人只好立中間的兒子為國君。

伯夷、叔齊聽說西伯昌（周文王）善待老人，便去投奔。但到了那裡，周武王已經載著文王的靈牌，號稱「奉文王之命」去討伐商紂。伯夷、叔齊攔住武王的馬頭進諫說：「父親死了不安葬，卻大動干戈，這能叫孝嗎？以臣子的身份去殺君王，這能叫仁嗎？」武王的手下想要殺他們。太公姜尚說：「這是義人啊！」扶著他們離去。

武王平定殷商之後，天下都歸順了周朝。伯夷、叔齊以做周朝的臣民為恥，為了堅守節義，不吃周朝的糧食，隱居在首陽山，採摘薇菜充飢。快要餓死的時候，他們作了一首歌：

> 登上那西山啊，採摘薇菜充飢。
> 用暴力取代暴力啊，卻不覺得自己錯了。
> 神農、虞舜、夏禹的時代忽然過去了啊，

盜跖每天殺害無辜的人，吃人的肝肉，兇暴殘忍，聚集了幾千人橫行天下——竟然得以壽終正寢。這遵循的是什麼德行？

這是特別重大、特別明顯的例子。到了近代，那些行為不端、專門違法犯禁的人，卻終身安逸享樂、財富積累世代不絕；而那些選擇好路才走、看準時機才說話的人，不走捷徑、不發不義之財、小心謹慎地做事，卻遭遇災禍的人——簡直數也數不清。我對此深感困惑——所謂的天道，到底是對的還是錯的？
