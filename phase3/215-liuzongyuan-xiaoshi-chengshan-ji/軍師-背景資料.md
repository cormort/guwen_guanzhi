# 〈小石城山記〉背景資料

> **軍師分析 · 古文觀止卷九 · 唐文 · 柳宗元**

---

## 一、核心骨架

| 項目 | 內容 |
|------|------|
| **出處** | 《柳河東集》 |
| **時代** | 中唐（約西元 809–810 年，柳宗元在永州時） |
| **核心命題** | 永州八記之終章——「造物者之有無」的終極追問 |
| **關鍵人物** | 柳宗元 |
| **文本重點** | 柳宗元發現小石城山——這座山的石頭形狀奇特，像城牆和堡壘。他在讚嘆自然之美的同時，忽然提出一個問題：這樣的奇景為什麼不在繁華的都市附近，卻在偏遠的永州？如果說有造物主，為什麼把美景放在沒人能看到的地方？如果說沒有造物主，這鬼斧神工又是誰創造的？ |
| **一句話總結** | 這篇遊記的最後不是在寫山，而是在寫一個被遺忘的人的孤獨——柳宗元不是在問造物主為什麼創造這座山，而是在問為什麼把他放在這裡。 |

## 二、創作背景

### 政治風暴：永貞革新與貶謫

- **805 年（永貞元年）**：唐順宗即位，王叔文集團推行「永貞革新」。柳宗元（時任禮部員外郎）是核心成員之一，與劉禹錫等人積極參與。
- **同年 8 月**：順宗被迫內禪，憲宗即位。革新僅維持 146 天即告失敗。王叔文被殺，柳宗元、劉禹錫等八人被貶為遠州司馬——史稱「二王八司馬事件」。
- **柳宗元貶所**：永州（今湖南永州）司馬。名為司馬，實為「編外」閒員，無實權、無衙署，寄居龍興寺，形同軟禁。

### 永州十年：從政治犯到山水知己

- 永州地處湘南，在唐代屬於「蠻荒瘴癘之地」。柳宗元初到時母親病逝，自己亦水土不服，身心備受煎熬。
- 為排遣苦悶，他開始縱情山水，寫下了中國文學史上最著名的山水遊記系列——**〈永州八記〉**。
- 八記順序為：〈始得西山宴遊記〉→〈鈷鉧潭記〉→〈鈷鉧潭西小丘記〉→〈至小丘西小石潭記〉→〈袁家渴記〉→〈石渠記〉→〈石澗記〉→**〈小石城山記〉**。

### 〈小石城山記〉的特殊位置

- 八記中最晚寫成（約 809–810 年），是永州八記的壓卷之作。
- 前七記多在寫景抒情中流露淡淡的孤寂，到了〈小石城山記〉，積蓄多年的憤懣終於化作石破天驚的哲學質問。
- 山名「小石城」——石頭排列如城牆堡壘，暗合柳宗元心中那座困住自己的無形牢城。

## 三、文本細讀重點

### 發現與描寫

- 從西山（八記起點）一路往黃茅嶺（八記終點），柳宗元「自西山道口徑北，踰黃茅嶺而下」，發現小石城山。
- 山的石頭「有若城郭狀」——像城牆、像堡壘、像房屋。這種對日常政治建築的聯想，暴露了他心頭從未放下的政治創傷。
- 整體描寫筆法冷靜、節制，與前七記一脈相承，但景物背後已湧動著暗流。

### 轉折：從寫景到問天

- 「類智者所施設也」——如此精巧的安排，似乎出自一位智慧設計者之手。
- 但隨即筆鋒一轉：「噫！吾疑造物者之有無久矣。」——我早就懷疑造物主是否存在了。
- 如果真的有造物主，為什麼把這座鬼斧神工的山放在偏遠的永州，不讓它出現在中原地區、供更多人欣賞？「勞而無用」——這是造物主的浪費，還是造物主根本不存在？

### 終極質問

- 柳宗元的兩個選項：
  1. **有造物主** → 但祂把美景放在人跡罕至之處，等於白費功夫——這不合理。
  2. **無造物主** → 但石頭的鬼斧神工又像出自有意義的設計——這也說不通。
- 最後他用一個疲倦的回答收尾：「神者儻不宜如是，則其果無乎？」——神明或許不該是這樣的，那麼終究是沒有的吧？
- 這不是哲學論證的結論，而是一個被拋棄的人的自我安慰。

## 四、歷史定位與文學價值

- **永州八記的完結篇**：從〈始得西山宴遊記〉的欣喜發現，到〈小石城山記〉的哲學絕望，完整記錄了柳宗元十年貶謫的心靈軌跡。
- **山水遊記的新典範**：柳宗元之前，山水在文章中多為背景或比興工具；柳宗元讓山水成為獨立的美學對象，同時又是心靈的鏡像。
- **造物者的追問**：這種對「造物者有無」的直接質疑，在中國散文史上極其罕見。其思路後來影響了蘇軾〈赤壁賦〉中「惟江上之清風，與山間之明月……是造物者之無盡藏也」——但蘇軾是曠達的接受，柳宗元是不甘的質問。

---

> **軍師按：** 〈小石城山記〉的偉大不在於它回答了什麼，而在於它敢於提問。柳宗元用一篇不到三百字的遊記，把政治迫害、個人孤獨、宇宙意義三個層次的困境同時壓縮進一座石山之中。後人讀〈小石城山記〉，讀到的不只是永州的一座小山，而是一個知識分子在被世界拋棄後，還能保持質問的勇氣。
