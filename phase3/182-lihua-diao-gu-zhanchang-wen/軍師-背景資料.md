# 〈弔古戰場文〉背景資料

> **軍師分析 · 古文觀止卷七 · 唐文 · 李華**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《李遐叔文集》——李華（字遐叔）文集，原集十卷，今存四卷，多為散佚後輯佚重編 |
| **選集** | 《古文觀止》卷七·唐文類——繼〈滕王閣序〉的華麗之後，卷七以〈弔古戰場文〉的蒼涼作結，一華一悲，對比鮮明 |
| **體裁** | 賦體悼文——以「古戰場」為題，借弔古之名，行反戰之實——形式上採用騷體賦與四言韻文的混合體，氣勢磅礴又沉痛哀切 |
| **時代** | 盛唐轉中唐時期（約西元 8 世紀中葉，安史之亂前夕）——帝國由盛而衰的轉捩點，邊疆戰爭頻繁——「邊庭流血成海水，武皇開邊意未已」的時代 |
| **核心命題** | 戰爭的傷痛——「蒼蒼蒸民，誰無父母」——對統治者的反戰控訴，對無名士兵的人道關懷 |
| **關鍵人物** | 李華（唐代散文家，字遐叔，開元二十三年進士，曾為安祿山所俘，被迫受偽職，晚年引咎辭官，以文章自娛） |
| **文本重點** | 李華在古戰場遺址上，想像當年戰爭的慘烈——「河水縈帶，群山糾紛。黯兮慘悴，風悲日曛。」他描述了士兵們離鄉背井、戰死他鄉的悲劇——「無貴無賤，同為枯骨。」最後點出全篇的主題：戰爭的起因往往不在於百姓，而在於統治者的好大喜功——「文教失宣，武臣用奇。……蒼蒼蒸民，誰無父母？提攜捧負，畏其不壽。誰無兄弟？如足如手。誰無夫婦？如賓如友。生也何恩，殺之何咎？」 |
| **一句話總結** | **李華站在白骨累累的古戰場上，對統治者提出了最溫柔也最嚴厲的質問——這些死去的人，他們也是某人的父母、兄弟、夫妻——你憑什麼讓他們去死？〈弔古戰場文〉不是一篇哭哭啼啼的傷感文學——它是一篇以蒼生為名的討伐檄文——用最華美的文字——說最殘酷的事實——然後——把最後的審判——留給了千古的讀者。** |

## 二、時代背景——盛唐的最後一抹血色

### 2.1 從「開天盛世」到「邊功禍患」

〈弔古戰場文〉寫於盛唐轉中唐的關鍵節點——大約在唐玄宗天寶年間（742–756 年）——也就是安史之亂（755 年）爆發前夕。這個時代的唐朝，呈現出極度矛盾的兩面：

| 面向 | 說明 |
|------|------|
| **政治** | 玄宗李隆基在位前期開創了「開元盛世」（713–741 年）——政治清明——國力鼎盛——然而天寶年間玄宗怠政——寵信楊國忠——縱容邊將——朝廷日趨腐敗 |
| **軍事** | 唐朝在西北、東北、西南三線同時用兵——尤其是與吐蕃的戰爭曠日持久——哥舒翰、高仙芝、安祿山等邊將手握重兵——「窮兵黷武」成為朝廷的主流政策——無數士卒在邊疆白白送命 |
| **社會** | 杜甫同時期寫下「車轔轔，馬蕭蕭，行人弓箭各在腰。爺孃妻子走相送，塵埃不見咸陽橋」——徵兵的慘狀——離別的淚水——是這個時代最真實的民間寫照 |
| **精神氣候** | 盛唐的樂觀主義正在褪色——天寶年間的戰爭已經不再是太宗、高宗時期的「擴張正義」——而是統治者的好大喜功——「邊庭流血成海水，武皇開邊意未已」——李華和杜甫一樣——是這個時代清醒的聲音 |

> **軍師按**：李華的偉大——在於他在「盛唐」的最高潮——就看出了這個帝國的內部創傷。當時人人都在歌頌國力的強盛——邊將們用士卒的頭顱換取戰功和封賞——朝廷裡沒有人關心那些死在沙漠裡、埋在荒草下的年輕人——李華站了出來——用一篇〈弔古戰場文〉——為那些「沒有名字的人」——寫了一篇墓誌銘。

### 2.2 李華的個人命運——寫反戰文的人被戰爭吞噬

李華寫〈弔古戰場文〉時——他還沒有親身經歷戰爭——他憑藉的是對歷史的反省和對現實的觀察——但命運跟他開了一個殘酷的玩笑：

| 面向 | 說明 |
|------|------|
| **早年仕途** | 李華開元二十三年（735 年）進士及第——天寶年間官至監察御史——以文章著稱——與蕭穎士齊名——世稱「蕭李」——是唐代古文運動的先驅 |
| **安史之亂的考驗** | 天寶十四載（755 年）——安祿山叛軍攻陷長安——李華未能逃出——被叛軍俘虜——被迫接受了鳳閣舍人的偽職——這成為他一生最大的汙點 |
| **晚年引咎** | 亂平之後——李華因「失節」被貶官——雖然朝廷最終寬恕了他——但他始終無法原諒自己——晚年以病辭官——隱居山陽——潛心著述——以「自咎」之心度過餘生 |
| **創作的辯證** | 李華寫〈弔古戰場文〉時——他是在「想像」戰爭——安史之亂後——他「親歷」了戰爭——他筆下的「蒼蒼蒸民」——從文學的修辭——變成了他親眼所見的現實——這讓他的後期作品——更加沉痛——更加真實 |

> **軍師按**：李華的人生——與他寫的〈弔古戰場文〉——形成了一個黑色的諷刺——他寫了一篇反對戰爭的文章——結果戰爭親自找上了他——他用文字為無名的士兵哀悼——結果自己也成了一個「失節者」——需要用餘生來為自己的選擇贖罪——也許正是因為他經歷了這一切——他筆下的「蒼蒼蒸民」——才如此真實——如此刺骨。

## 三、文本結構——從「蒼茫」到「蒼涼」的悲劇三部曲

### 3.1 全文三段式結構

〈弔古戰場文〉全篇層層遞進——從景到事——從事到情——從情到理——像一支樂曲——由低沉的序曲——到激昂的高潮——再到歸於沉靜的尾聲：

| 段落 | 起句 | 情感基調 | 核心內容 |
|------|------|---------|---------|
| **第一段：古戰場的蒼茫** | 「浩浩乎，平沙無垠，夐不見人」 | 蒼茫、寂寥、壓抑 | 以「亭長告余」的對話引入——亭長指著這片荒原說「此古戰場也，常覆三軍」——然後展開對古戰場景色的描寫——「河水縈帶，群山糾紛。黯兮慘悴，風悲日曛。蓬斷草枯，凜若霜晨。鳥飛不下，獸鋌亡群」——用無聲的自然——呈現戰爭留下的創傷 |
| **第二段：戰爭的殘酷** | 「吾聞夫齊魏徭戍，荊韓召募」 | 悲憤、沉痛、激昂 | 縱橫古今——歷數從春秋戰國到秦漢的戰爭——「萬里奔走，連年暴露。沙草晨牧，河水夜渡。地闊天長，不知歸路」——士兵們在絕望中的掙扎——「積屍暴骸，泣血飲泣」——最終——所有的交戰方都輸了——「無貴無賤，同為枯骨」 |
| **第三段：反戰的吶喊** | 「蒼蒼蒸民，誰無父母」 | 溫柔、嚴厲、昇華 | 此處是全篇的最高潮——也是李華最有力的控訴——用三個排比——「誰無父母」「誰無兄弟」「誰無夫婦」——將戰爭的受害者——從抽象的「士兵」還原為具體的「人」——然後——提出那個千古質問——「生也何恩，殺之何咎？」——最後給出解決方案——「守在四夷」——以德懷遠——而不是以殺止殺 |

全文的情感曲線：**蒼茫（第一段）→ 悲憤（第二段）→ 沉靜的控訴（第三段）**——不是從悲到喜——而是從「眼前的荒涼」到「歷史的殘酷」再到「永恆的質問」——這是一場無法和解的悲劇。

### 3.2 第一段：古戰場的蒼茫——「浩浩乎，平沙無垠」

> 「浩浩乎，平沙無垠，夐不見人。河水縈帶，群山糾紛。黯兮慘悴，風悲日曛。蓬斷草枯，凜若霜晨。鳥飛不下，獸鋌亡群。亭長告余曰：『此古戰場也，常覆三軍。往往鬼哭，天陰則聞。』」

| 層次 | 內容 | 分析 |
|------|------|------|
| **空間的極簡** | 浩浩乎——平沙無垠——夐不見人 | 開篇三個詞——「浩浩乎」（廣闊）、「平沙無垠」（無邊的沙漠）、「夐不見人」（空無一人）——將空間推到了極致的荒涼——不是美麗的風景——是讓人窒息的空無 |
| **自然的不祥** | 黯兮慘悴——風悲日曛——蓬斷草枯——凜若霜晨 | 六個自然意象——沒有一個是溫暖的——「黯」（昏暗）、「慘悴」（悽慘憔悴）、「風悲」（風在哭）、「日曛」（太陽暗淡）——連自然都在為這場戰爭流淚 |
| **亭長的見證** | 亭長告余曰——此古戰場也——常覆三軍——往往鬼哭——天陰則聞 | 從直接描寫轉入「口述歷史」——亭長是這個地方的守護者——他用最樸素的語言——說出了最恐怖的事實——「常常有軍隊在這裡全軍覆沒」——「陰天的時候——還能聽到鬼哭」——從客觀描寫——轉入主體感受——讀者也跟著毛骨悚然 |

> **軍師按**：開篇的「浩浩乎，平沙無垠」——是中國文學史上最蒼涼的開篇之一——與〈滕王閣序〉的「豫章故郡，洪都新府」形成極致對比——一個是人間的繁華——一個是死亡的荒原——王勃寫的是「落霞與孤鶩齊飛」——李華寫的是「鳥飛不下，獸鋌亡群」——連鳥獸都不願意停留的地方——那是人與自然都已經放棄的領域。

### 3.3 第三段核心：反戰的終極質問——「蒼蒼蒸民，誰無父母」

> 「蒼蒼蒸民，誰無父母？提攜捧負，畏其不壽。誰無兄弟？如足如手。誰無夫婦？如賓如友。生也何恩，殺之何咎？其存其歿，家莫聞知。人或有言，將信將疑。悁悁心目，寤寐見之。布奠傾觴，哭望天涯。天地為愁，草木悽悲。弔祭不至，精魂何依？必有凶年，人其流離。嗚呼噫嘻！時耶命耶？從古如斯！為之奈何？守在四夷。」

| 層次 | 內容 | 分析 |
|------|------|------|
| **三個排比** | 誰無父母——誰無兄弟——誰無夫婦 | 李華用最簡單的語言——把士兵還原為「社會的人」——他是一個母親捧在手心的孩子——是一個兄長呵護的弟弟——是一個妻子倚靠的丈夫——李華不說戰爭「死傷多少人」——他說的是「這些死去的人——他們也有愛他們的人」——這比任何數字都更有力量 |
| **終極質問** | 生也何恩——殺之何咎 | 全篇最鋒利的一句話——六個字——問出了反戰思想的核心——「他們活著的時候——你（統治者）給了他們什麼恩惠？——他們犯了什麼錯——你就要他們去死？」——不是說「戰爭不好」——而是說「你——作為統治者——沒有資格讓他們去死」 |
| **家屬的視角** | 其存其歿——家莫聞知——人或有言——將信將疑——悁悁心目——寤寐見之——布奠傾觴——哭望天涯 | 李華把鏡頭從戰場——轉向了家鄉——那些在遠方等待的親人——不知道自己的兒子/丈夫/父親是死是活——聽到傳言——不敢相信——日夜憂心——最後在絕望中——擺下祭品——對著天涯痛哭——連魂魄都無法回到故鄉——這是戰爭最殘酷的側面——生者比死者更痛苦 |
| **解決方案** | 守在四夷 | 李華的最終結論——不是否認所有戰爭——而是反對「好戰」——真正的安全——不在於征服——而在於以德懷遠——用文化的力量——讓四夷歸心——這是儒家「修文德以來之」思想的體現——也是對玄宗朝「窮兵黷武」政策的直接批判 |

> **軍師按**：「生也何恩，殺之何咎？」——這六個字——是李華對中國歷史上所有好大喜功的統治者的審判——你不是上帝——你沒有賜予他們生命——所以你沒有權力奪走他們的生命——在中國文學史上——幾乎沒有人——敢用如此鋒利的語言——質問權力——而且——是以一個「文臣」的身份——在一篇「賦體」文章裡——〈弔古戰場文〉之所以不朽——不是因為它寫得華麗——而是因為它——敢說真話。

## 四、歷史影響——從〈弔古戰場文〉到反戰文學傳統

〈弔古戰場文〉對後世產生了深遠的影響：

| 面向 | 說明 |
|------|------|
| **文學典範** | 李華的「賦體反戰」傳統——直接影響了中唐、晚唐乃至宋代的反戰文學——白居易的〈新豐折臂翁〉、杜牧的〈阿房宮賦〉——都可以看到〈弔古戰場文〉的影子——「用文學質問權力」——成為中國知識分子的精神傳統 |
| **古文運動的先聲** | 李華與蕭穎士——被認為是唐代古文運動的先驅——韓愈、柳宗元倡導「文以載道」——而李華在一百年前——已經用〈弔古戰場文〉證明——文章——可以承載最嚴肅的社會關懷——它是「文以載道」最早的實踐之一 |
| **思想價值** | 「蒼蒼蒸民」——成為後世反戰文學的核心母題——明代歸有光的〈戰士〉——清代顧炎武的〈天下興亡匹夫有責〉——都延續了李華「以民為本」的戰爭觀——戰爭不是英雄的史詩——而是母親失去兒子的哭聲 |
| **跨時代共鳴** | 在中國文學史上——反戰的主題一直存在——《詩經》的〈采薇〉——杜甫的〈兵車行〉——李華的〈弔古戰場文〉——形成了一條完整的反戰文學脈絡——每一篇都是對權力的質問——每一篇都是對死者的哀悼——每一篇都是對生者的告誡 |

## 五、終章——誰在為無名者寫史？

〈弔古戰場文〉最深的悲哀——在於它提出的問題——在一千三百年後——依然沒有答案。

李華在文章裡問：「蒼蒼蒸民，誰無父母？」——這個問題——在中國歷史上——每一個朝代都在問——每一個朝代都沒有解決——戰爭——永遠是少數人發動——多數人承受——而承受的人——永遠是那些「無名者」——他們沒有留下名字——他們的屍骨埋在無名的荒原上——他們的故事——被歷史的風沙淹沒。

李華為他們寫了一篇碑文——不是為某個將軍——不是為某場戰役——而是為「所有在戰爭中死去的普通人」。

這就是〈弔古戰場文〉之所以不朽——不是因為它寫得多好——而是因為它——替沒有聲音的人——發出了聲音。

> **軍師按**：讀完〈弔古戰場文〉——再來看看我們今天的世界——戰爭從未停止——而李華的質問——依然迴盪在每一個戰場的上空——「生也何恩，殺之何咎？」——這篇文章——不是寫給唐朝的——是寫給所有人的——在任何時代——當權力開始輕視生命——李華就會站出來——用他的文字——替那些死去的人說一句話。
