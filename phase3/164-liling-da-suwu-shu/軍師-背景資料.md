# 〈李陵答蘇武書〉背景資料

> **軍師分析 · 古文觀止卷六 · 西漢文 · 李陵**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《文選》卷四十一——一說為後人偽托（六朝人擬作），傳統上歸名李陵 |
| **選集** | 《古文觀止》卷六·西漢文類——西漢書信體散文的壓卷之作 |
| **體裁** | 書信體——以「子卿足下」開篇的私人信件，實為一篇情感飽滿的自我辯白書 |
| **時代** | 西漢武帝時期（約西元前 1 世紀），李陵降匈奴後（前 99 年兵敗，蘇武歸漢為前 81 年） |
| **核心命題** | 一個投降者的辯白——「身負國恩，為世所悲」——當忠誠與生存矛盾時，該如何選擇？ |
| **關鍵人物** | 李陵（漢將李廣之孫，以五千步兵對戰十萬匈奴騎兵，兵敗投降）、蘇武（漢使出使匈奴被扣十九年，牧羊北海，節旄盡落而不降） |
| **文本重點** | 李陵投降匈奴後，漢朝派蘇武出使匈奴被扣。蘇武堅守氣節十九年，李陵多次受命勸降無果。後來蘇武歸漢，李陵寫了這封信為自己辯解——以五千步兵對抗十萬騎兵，苦戰八日，彈盡糧絕，卻被武帝懷疑而族滅全家。信中充滿了痛苦、委屈和對命運的控訴——「身出禮義之鄉，而入無知之俗」——他不求原諒，只求理解 |
| **思想譜系** | 出自《文選》「書」類。李陵這封信在中國文學史上引發了長達千年的真偽之辯——唐宋以降，學者多認為此文為六朝人擬作（因文風與漢代樸質之風有異）。但無論真偽，這封信所表達的「降將之苦」已成為中國文學中一個永恆的母題——從〈報任安書〉到〈與陳伯之書〉，都能看到這封信的影子 |
| **一句話總結** | **李陵的辯白之所以動人，不是因為他為投降找到了理由——而是因為他提出了一個沒有答案的問題：你到底該為一個不信任你的國家去死，還是該為自己的生存投降？** |

## 二、時代背景——大將軍的末路與一個使節的堅守

### 2.1 李陵之敗——五千步兵 vs 十萬騎兵

李陵是「飛將軍」李廣的孫子，李廣第三子李當戶的遺腹子。他繼承了祖父的騎射天賦——「善騎射，愛人下士」——武帝曾稱他有「李廣之風」。天漢二年（西元前 99 年），李陵主動請纓率五千步兵出擊匈奴——目標是牽制匈奴主力，配合李廣利的西路大軍：

| 面向 | 說明 |
|------|------|
| **兵力懸殊** | 李陵率五千步兵北上——行軍三十日至浚稽山，遭遇且鞮侯單于的主力——三萬騎兵。李陵以車陣迎戰——殺數千人——單于大驚，又調來八萬騎兵——總計十餘萬騎兵圍攻五千步兵。李陵且戰且退——南行至距漢塞僅百餘里的地方——連戰八日——殺敵萬餘 |
| **彈盡糧絕** | 八日苦戰後——箭矢用盡——士兵只剩下「尺刀」——糧食也已耗盡。李陵嘆曰：「復得數十矢，足以脫矣。今無兵復戰，天明坐受縛矣！」——深夜下令突圍——只有十餘人跟隨——匈奴騎兵數千追擊——最後李陵被俘投降 |
| **武帝的反應** | 武帝最初聽聞李陵投降，大怒。後聽司馬遷為李陵辯護（說李陵是以少擊眾、力竭而降，其功足以補過）——反而將司馬遷處以宮刑。隨後——武帝聽信傳言說李陵在匈奴為練兵——下令族滅李陵全家——母、弟、妻、子皆被殺 |
| **李陵的絕望** | 當李陵得知全家被族滅——他徹底斷了歸漢之心。單于將女兒嫁給他——立他為右校王——李陵從此在匈奴生活了二十餘年——至死未歸 |

> **軍師按**：李陵的敗亡——從純軍事角度看——是一場可歌可泣的悲壯之戰。五千步兵對抗十萬騎兵——還能殺敵萬餘、且戰且退八日——已經是人類軍事史上的奇蹟。但李陵敗就敗在：他遇到的是一個不信任他的皇帝——武帝在開戰之初就不信任李陵（因為他是李廣的孫子——李廣一生未封侯——武帝對李氏家族有偏見）——所以當戰敗的消息傳來——武帝的第一反應不是「李陵盡力了」——而是「他投降了」。李陵的悲劇——不是戰敗的悲劇——而是不被信任的悲劇。

### 2.2 蘇武的十九年——北海牧羊的氣節

蘇武以中郎將的身份出使匈奴——原為修復漢匈關係——卻被匈奴扣留。單于多次勸降——先以高官厚祿誘惑——後以囚禁飢餓逼迫——蘇武始終不屈：

| 面向 | 說明 |
|------|------|
| **被扣的起因** | 蘇武與副使張勝出使匈奴——適逢匈奴內部有人謀反（虞常等人）——張勝暗中參與——事洩後蘇武受牽連——單于派衛律勸降——蘇武不從 |
| **北海牧羊** | 單于將蘇武流放到北海（今貝加爾湖）牧羊——說「公羊生仔才能歸漢」——實際上是永不釋放。蘇武在北海「掘野鼠去草實而食之」（挖野鼠洞裡儲藏的草籽吃）——手持漢朝使節——十九年——節旄盡落 |
| **李陵的勸降** | 單于數次派李陵去北海探望蘇武——李陵告訴蘇武自己的遭遇——希望說服蘇武降匈奴。李陵說：你的兄弟在漢朝因小過自殺——你的母親已死——你的妻子改嫁——你回去還有什麼？蘇武回答：「臣事君，猶子事父也。子為父死，無所恨。」——李陵感嘆：「嗟乎，義士！陵與衛律之罪，上通於天！」——泣下沾襟 |
| **歸漢** | 漢昭帝即位後——漢匈和親——漢使以「天子在上林苑射得一雁，足繫帛書，言蘇武在某澤中」為由——迫使匈奴交出蘇武——蘇武在十九年後終於歸漢——鬚髮盡白 |

> **軍師按**：李陵與蘇武的會面——是中國文學史上最戲劇化的場景之一——兩個截然不同的人：一個投降了，一個沒有；一個在權力與生存面前低下了頭，一個用十九年的時間證明了什麼是不屈。李陵去勸降蘇武——反而被蘇武的氣節所震撼——他說「陵與衛律之罪，上通於天」——這是他對自己命運最深刻的懺悔——他知道自己錯了——但他已經無法回頭。

## 三、原文結構與核心情感

### 3.1 〈李陵答蘇武書〉的結構

這封信的核心邏輯是「解釋」——不是「求饒」，也不是「辯解」——而是向蘇武（以及所有讀這封信的人）解釋「我為什麼會投降」：

| 層次 | 內容 | 用意 |
|------|------|------|
| **開場——故人重逢的哀傷** | 李陵開篇便說：「子卿足下：勤宣令德，策名清時，榮問休暢，幸甚幸甚。陵不遠萬里，故為足下道此。」——他先讚美蘇武的氣節得到回報——然後說自己「遠託異國，昔人所悲」——對比兩人的命運——一個榮歸故國，一個遠寄異鄉 | 開場即以強烈的對比定下全文的基調——「你成功了，我失敗了」——李陵毫不掩飾自己的痛苦——他直面命運的不公 |
| **中段——兵敗的詳細描述** | 這是最長、最關鍵的部分。李陵詳細描述了浚稽山之戰的全過程——五千步兵如何被十萬騎兵圍困——如何以車陣應戰——如何殺敵萬餘——如何且戰且退——如何在最後一刻箭矢用盡、糧食斷絕——他如何突圍——最後被俘。他告訴蘇武：我不是沒有努力——我盡了全力——但命運不站在我這邊 | 這是全文最重要的「證據」——李陵用近乎戰報的詳實筆法，復原了那場戰爭的全部細節。他告訴讀者：我的投降不是在戰場上——而是在彈盡糧絕之後——我已經盡了一個軍人最大的努力——如果武帝願意相信我——我本可以回到漢朝 |
| **後段——對武帝的控訴** | 李陵說：「然陵不死，罪也。子卿視陵，豈偷生之士而惜死之人哉？寧有背君親、捐妻子、而反為利者乎？」——然後他話鋒一轉——控訴武帝的不信任：「而執事者云云，苟怨陵以不死。然陵不死，有所為也。」——「且漢厚誅陵不顧，殺陵之母、弟、妻子。」——他投降後——漢朝沒有給他任何辯白的機會——直接殺了他全家 | 這一段是全信最憤怒的部分。李陵不是為自己投降辯解——他是質問：一個為國家出生入死的將領——為什麼在戰敗之後——連被理解的可能性都沒有？武帝對他——從來就沒有信任過 |
| **結尾——對命運的屈服** | 信的結尾極其悲涼：「嗟乎子卿，人之相知，貴相知心。前書倉卒，未盡所懐，故復略而言之。方圖遠略，為世所悲。遠託異國，昔人之所悲也。陵雖孤恩，漢亦負德。」——最後說：「願足下勿復望陵。」——你不要再指望我回去了 | 結尾是全信的靈魂——李陵最終承認了自己的失敗——但他提出了一個讓蘇武無法反駁的命題：「陵雖孤恩，漢亦負德」——我辜負了漢朝——但漢朝也辜負了我——我們扯平了 |

> **軍師按**：這封信最精彩的地方——不是它的文采——而是它的坦誠。李陵完全不迴避自己的投降事實——他直面它——然後他用整整一封信來解釋「為什麼」。他不是說「我沒有錯」——他是說「我有錯——但漢朝也有錯——命運對我實在太不公平了」。這種坦誠——比任何辯解都更有力量。

### 3.2 核心段落節錄

> 陵不遠萬里，故為足下道此。遠託異國，昔人所悲，望風懷想，能不依依？

——開篇的悲涼基調。李陵告訴蘇武：我之所以寫這封信，是因為我太想念故國了——但我回不去了。

> 然陵不死，罪也。子卿視陵，豈偷生之士而惜死之人哉？

——李陵承認投降是「罪」——但他反問：你覺得我是那種怕死的人嗎？

> 陵孤恩負義，為世所悲。身出禮義之鄉，而入無知之俗，違棄君親之恩，長為蠻夷之域，傷已！

——全信最感人的句子之一。李陵說：我出身文明之邦——卻流落蠻夷之地——背棄了君王的恩情——永遠留在了異鄉——太可悲了！

> 陵雖孤恩，漢亦負德。

——全文的結論句。八個字，道盡了李陵的全部委屈：我辜負了你（漢朝），但你也辜負了我——我們誰也不欠誰。

> ——《文選》卷四十一

## 四、關鍵人物

### 4.1 李陵——戰敗的將軍，永恆的罪人

| 項目 | 內容 |
|------|------|
| **身份** | 西漢名將，李廣之孫。字少卿，隴西成紀（今甘肅靜寧）人。曾任騎都尉——率五千步兵出擊匈奴——戰敗投降——被封為匈奴右校王——二十餘年後死於匈奴 |
| **生平核心** | 李陵是中國軍事史上最悲劇的人物之一——他的能力毋庸置疑（五千對十萬，殺敵萬餘），他的忠誠也毋庸置疑（如果不是武帝族滅其全家，他很可能會設法歸漢）。但他命運的轉折點——不是戰敗的時刻——而是武帝下令族滅他全家的那一刻——那一刻，他從「戰俘」變成了「叛徒」——再也無法回頭 |
| **歷史評價的分裂** | 對李陵的評價——在中國歷史上分裂了兩千年。司馬遷認為他是「國士」——以少擊眾、力竭而降——不應被貶斥。後世的正統儒者則認為投降就是投降——無論有多少理由——都不能改變「背叛」的事實。李陵自己——在〈答蘇武書〉中——也承認了自己的「罪」——他沒有為自己開脫——他只是希望被理解 |

**軍師評**：李陵的問題——本質上是一個「信任」的問題。武帝從來沒有真正信任過李家——從李廣到李陵——李氏三代將門——卻沒有一個得到應有的信任和封賞。李陵之敗——敗的不只是匈奴的十萬騎兵——更敗在武帝的猜忌之心。如果武帝信任他——在他戰敗之後給他一個解釋的機會——歷史可能會完全不同。但武帝沒有——他選擇了滅族。

### 4.2 蘇武——十九年不降的使節

| 項目 | 內容 |
|------|------|
| **身份** | 西漢使節，中郎將。字子卿——杜陵（今陝西西安）人。以出使匈奴被扣十九年、不辱使命而聞名——是中國歷史上「氣節」的化身 |
| **與李陵的關係** | 李陵與蘇武在漢朝時即相識——兩人在匈奴重逢的場景——是中國文學史上最令人動容的時刻之一。蘇武的堅守——讓李陵感嘆自己的投降是「上通於天」的罪過。而李陵寫給蘇武的這封信——既是告別，也是懺悔——他把蘇武當作唯一能夠理解他的人——因為蘇武親眼見證了他的掙扎與痛苦 |
| **歸漢後的結局** | 蘇武歸漢後——被漢昭帝拜為典屬國（管理歸附民族事務）——後封關內侯——年八十餘卒。他歸漢時——李陵設宴送別——說「今足下還歸，揚名於匈奴，功顯於漢室——雖古竹帛所載，丹青所畫，何以過子卿！」——李陵目送蘇武離去——泣下數行 |

**軍師評**：蘇武與李陵——代表了面對命運的兩種極端選擇。蘇武選擇了「堅守」——即使沒有任何回報的希望（誰知道十九年後會有人來接他？）——他也選擇了不投降。李陵選擇了「妥協」——在彈盡糧絕、全家被殺之後——他選擇了活下去。沒有人有權利評判這兩種選擇——因為沒有人經歷過他們經歷的一切。但歷史給了蘇武最高的讚譽——而李陵——則成為了一個永遠的「問題」——一個關於「忠誠與生存」的沒有答案的問題。

### 4.3 漢武帝劉徹——這封信真正的收信人

| 項目 | 內容 |
|------|------|
| **身份** | 西漢第七位皇帝——在位五十四年——將漢帝國推向了空前的高峰——但也因好大喜功、猜忌大臣而留下了許多悲劇 |
| **與李陵的關係** | 武帝是李陵悲劇的「幕後推手」——他不信任李陵——他聽信謠言族滅了李陵全家——他處死了為李陵辯護的司馬遷（宮刑）。李陵的〈答蘇武書〉雖然是寫給蘇武的——但它的真正讀者——是武帝、是漢朝、是所有的後人——李陵在信中控訴的——就是武帝的「不信任」 |
| **軍師評** | 武帝是一個偉大的皇帝——但也是一個危險的皇帝。他的偉大——來自於他的開拓與雄才大略——他的危險——來自於他的猜忌與殘忍。李陵和司馬遷——都是武帝猜忌的犧牲品。而蘇武——是武帝盛世的一個「意外」——武帝在世時沒有管過蘇武——是昭帝即位後才把他接回來的。蘇武的十九年——在某種意義上——也是武帝時代的一個縮影——一個人在絕望中堅持——等待一個不知道何時會到來的轉機 |

## 五、軍師核心分析——投降者的辯白：為什麼〈李陵答蘇武書〉如此動人？

### 5.1 李陵的「辯護策略」——不辯護，只懺悔

〈李陵答蘇武書〉最令人震撼的地方——是李陵完全放棄了傳統的辯護策略。他不說自己沒有投降——他不說投降是對的——他直接承認了自己的「罪」：

| 面向 | 分析 |
|------|------|
| **承認失敗** | 李陵從不否認自己的投降——他一開始就說「陵不死，罪也」——我沒有死——這是我的罪過。這種坦誠——反而讓他的控訴更有力量——因為他沒有試圖隱瞞什麼——他把所有的牌都攤在桌子上 |
| **把問題從「忠誠」轉移到「理解」** | 李陵不問「我投降對不對」——他問的是「你（武帝、蘇武、歷史）能不能理解我為什麼投降？」他把道德問題——轉化為理解問題——這是一個極其高明的策略——因為在道德層面——投降是無法辯解的——但在理解層面——每一個人都可能在某種處境下做出同樣的選擇 |
| **用詳細的戰報證明自己「盡力了」** | 李陵用了大量篇幅描述浚稽山之戰——不是為了炫耀戰功——而是為了證明一件事：我不是沒有努力——我努力到了最後一刻——我只是遇到了我無法克服的困難。當他描述箭矢用盡、士兵只能使用「尺刀」作戰、糧食斷絕、突圍失敗——讀者幾乎無法不同情他——因為他確實已經盡了全力 |
| **最後一擊：陵雖孤恩，漢亦負德** | 全信最高潮的八個字——李陵把問題從「你對不起漢朝嗎？」變成了「漢朝對得起你嗎？」——他提出了一個無法否認的事實：漢朝（武帝）殺了他的全家——而他在投降之前——為漢朝殺了上萬敵人——這個帳——怎麼算？ |

> **軍師按**：李陵的辯護之所以高明——是因為他沒有把問題簡單化。他不是在說「投降是對的」——他在說的是：「我跟蘇武一樣——都是有尊嚴的人——只是命運對我們開了不同的玩笑。」他承認蘇武比他偉大——但他也堅持——他的選擇——是有原因的——而那些原因——是值得被理解的。

### 5.2 〈李陵答蘇武書〉與〈報任安書〉的對比

李陵的這封信——與司馬遷的〈報任安書〉——是中國文學史上最著名的兩篇「絕境之書」——而且兩者之間有著千絲萬縷的聯繫（司馬遷正是因為為李陵辯護而遭受宮刑）：

| 面向 | 〈李陵答蘇武書〉 | 〈報任安書〉 |
|------|------|------|
| **寫作處境** | 投降匈奴之後——寫給故人蘇武——為自己的行為辯解 | 遭受宮刑之後——寫給朋友任安——解釋自己為什麼選擇活著 |
| **核心問題** | 我為什麼投降？ | 我為什麼不死？ |
| **情感基調** | 悲涼、痛苦、控訴——但也有對命運的深刻認命 | 憤怒、屈辱、堅忍——帶著對未完成的使命的執著 |
| **論證方式** | 用詳細的戰報證明自己力竭而降——用全家被殺來控訴漢朝的無情 | 用屈原、左丘明、孫臏等聖賢的苦難——證明「隱忍苟活」也可以成就偉大事業 |
| **結局態度** | 認命——不回來了——願勿復望陵 | 發誓——一定要完成《史記》——「通古今之變，成一家之言」 |
| **兩者的關係** | 司馬遷因李陵而受刑——李陵的這封信——在某種意義上——是對司馬遷所受苦難的一種「回應」——看，你說的不錯——我就是一個被命運壓垮的人——但我沒有你想像的那麼不堪——我也有我的苦衷 | 司馬遷選擇活著是為了完成《史記》——李陵選擇活著是因為已經沒有回漢的可能。兩個人都選擇了「活著」——但理由截然不同 |

### 5.3 真偽之辯——這封信真的是李陵寫的嗎？

從唐宋開始——學者們就對〈李陵答蘇武書〉的真實性提出了質疑——主要理由包括：

| 面向 | 分析 |
|------|------|
| **文風問題** | 這封信的駢儷風格（對仗、用典、修辭華麗）——與西漢樸質的文風不符——更像六朝駢文的產物。尤其是信中大量使用的駢句和對偶——如「身出禮義之鄉，而入無知之俗」——這種整齊的對仗是東漢末年至魏晉才流行的 |
| **史料問題** | 《漢書·李廣蘇建傳》中收錄了李陵與蘇武的幾次對話——但沒有收錄這封信。《漢書》是李陵時代最近的史書——如果李陵寫過這樣一封信——班固不可能不收入——這封信首次出現在蕭統的《文選》中——已經是六世紀的事了 |
| **情感問題** | 也有學者為其辯護——認為正因為這封信過於真實、過於痛苦——所以才不可能出自後人之手——因為沒有人能比李陵自己更了解那種「全家被殺、身陷異域」的痛苦——「擬作」能做到形似——但做不到神似 |

> **軍師按**：無論真偽——〈李陵答蘇武書〉在中華文學史上的地位已經不可動搖。如果是真的——它就是一個戰敗將軍最真摯的懺悔——如果是假的——它就是一個六朝文人對李陵命運最深刻的理解——比真實還真實。歸根結底——這封信之所以流傳千古——不是因為它出自誰手——而是因為它說出了一句所有人都不敢說、但所有人都想說的話：**有時候——好人——真的會遇到壞的命運——而那個時候——你該怎麼辦？**

## 六、對後世的影響與啟示

### 6.1 在中國文學史上的意義

| 時期 | 影響 |
|------|------|
| **魏晉南北朝** | 這封信在六朝時期被收入《文選》——成為「書」類中的典範——對六朝駢文的發展產生了重要影響。信中「身出禮義之鄉，而入無知之俗」等駢句——直接影響了後世的書信體散文 |
| **唐代** | 李陵與蘇武的故事——成為唐詩中最常見的典故之一。李白、杜甫、王維等人都寫過詠李陵或詠蘇武的詩——李陵的悲劇成了中國詩人反覆吟詠的主題。李白說「李陵沒胡沙，蘇武還漢家」——短短十個字——道盡了兩人的命運對比 |
| **宋代** | 蘇軾等人對這封信的真偽提出質疑——開啟了持續千年的「李陵答蘇武書真偽之辯」——但這反而讓這封信更加出名——因為爭論本身就是一種關注 |
| **明清** | 《古文觀止》編者吳楚材、吳調侯將此篇選入卷六——使其成為清代以後士人必讀的經典——進一步確立了其在中國散文史上的地位 |

### 6.2 對當代讀者的啟示

| 面向 | 啟示 |
|------|------|
| **理解不等於原諒** | 李陵告訴我們一件重要的事：理解一個人的選擇——不等於認同那個選擇。你可以理解李陵為什麼投降——但你也可以堅持認為投降是錯的。理解——不是為了讓道德變得模糊——而是為了讓我們的判斷不那麼草率 |
| **信任的雙向性** | 李陵的悲劇——本質上是一場「信任危機」——武帝不相信他——他也不相信武帝（在被滅族之後）——最終雙方都輸了。這告訴我們：信任——不是單方面的要求——而是雙向的給予。如果你要求別人忠誠——你首先要給予信任——否則——忠誠是無法被要求的 |
| **命運的殘酷與人的尊嚴** | 李陵和蘇武——面對的是同樣的困境——命運把他們都拋到了匈奴——但他們做出了截然不同的選擇。這證明了：命運可以決定你的處境——但它不能決定你的選擇——至少在選擇的那一刻——你還是自由的 |
| **「陵雖孤恩，漢亦負德」的現代意義** | 這句話在今天——仍然適用於許多場合：員工與公司的關係、個人與國家的關係、付出與回報的關係——當一方辜負了另一方——問題就變成了誰先辜負誰——而這個問題——往往沒有答案 |

## 七、一句話總結（再錄）

> **〈李陵答蘇武書〉是一個戰敗者寫給勝利者的告別信。李陵知道自己錯了——但他也不認為漢朝是對的。他寫了這場戰爭的全部經過——五千步兵如何對抗十萬騎兵——如何殺敵萬餘——如何在彈盡糧絕之後被俘——然後——他全家被殺了。他說：「陵雖孤恩，漢亦負德。」——八個字，把所有的道德正義都打平了。他不求蘇武原諒他——也不會原諒自己——他只是想讓蘇武知道——那個在北海牧羊十九年的你——和你面前這個在匈奴做了二十年駙馬的我——我們之間——不是好人與壞人的差別——而是運氣與命運的差別。李陵的辯白之所以動人，不是因為他為投降找到了理由——而是因為他提出了一個沒有答案的問題：你到底該為一個不信任你的國家去死，還是該為自己的生存投降？兩千年過去了——沒有人能回答這個問題。而這——恰恰是這封信之所以偉大的原因。** |

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈報任安書〉（司馬遷） | 司馬遷就是因為為李陵辯護而遭受宮刑——〈報任安書〉與〈李陵答蘇武書〉——一篇是為「不死」而辯護，一篇是為「投降」而辯護——兩篇合起來——構成了中國文學史上最震撼的「苦難二重奏」 |
| 卷五·〈伯夷列傳〉（司馬遷） | 司馬遷在〈伯夷列傳〉中提出了「天道無親，常與善人」的質疑——李陵的命運——就是這個質疑的最好例證——好人（李陵）未必有好報——壞人（衛律）反而活得很好 |
| 卷四·〈卜居〉（屈原） | 屈原在困惑中問卜——「我該正直還是圓滑？」——李陵不需要問——他已經做出了選擇——一個讓自己痛苦終生的選擇 |
| 卷三·〈漁父〉（屈原） | 漁父勸屈原「與世推移」——李陵就是那個「與世推移」的人——但他沒有得到快樂——只有痛苦——這說明「妥協」不一定比「堅守」更容易——只是不同的痛苦方式 |
| 卷六·〈武帝求茂材異等詔〉（漢武帝） | 武帝在詔書中說要尋找「非常之人」——李陵就是那個「非常之人」——但武帝並沒有真正重用他——這篇詔書讀起來——就像一個諷刺 |
| 卷六·〈上書諫獵〉（司馬相如） | 司馬相如勸武帝不要冒險打獵——武帝聽進去了——但李陵為武帝出生入死——武帝卻不信任他——同樣是武帝——對不同的勸諫者——態度截然不同 |

### 8.2 經典延伸

- 《漢書·李廣蘇建傳》（班固）——李陵與蘇武生平的第一手史料——特別是李陵與蘇武在匈奴會面的場景——是中國史學中最動人的段落之一
- 《史記·李將軍列傳》（司馬遷）——司馬遷為李廣祖孫作的傳——可以從中看到李氏家族的悲劇傳承
- 《文選·李陵答蘇武書》（蕭統編）——〈李陵答蘇武書〉的原始出處——《文選》將此篇歸為「書」類
- 《文選·蘇武詩四首》（蕭統編）——傳說中蘇武與李陵的贈答詩——與這封信可以對照閱讀——雖然同樣有真偽之爭——但已成為中國文學的重要組成部分
- 杜甫〈詠懷古蹟五首·其三〉——「千載琵琶作胡語，分明怨恨曲中論」——杜甫寫王昭君——但也隱含了對李陵命運的感慨
- 李白〈經下邳圯橋懷張子房〉——李白對李陵的感慨——「李陵沒胡沙，蘇武還漢家。一別長萬里，相逢天一涯。」

### 8.3 現代延伸

- 錢穆《國史大綱》——對漢武帝時代的整體分析——可以幫助理解李陵悲劇的宏觀背景
- 李長之《司馬遷之人格與風格》——對司馬遷與李陵關係的深入分析——特別是司馬遷為何甘願為李陵辯護
- 王國維《人間詞話》——王國維對「真感情」的論述——李陵的這封信是「以血書者」的典範
- 黃仁宇《萬曆十五年》——雖然寫的是明代——但書中關於「道德評價」與「現實困境」之間的張力——與李陵的處境高度相似——一個人在體制中——到底應該服從道德規範——還是服從生存本能？
