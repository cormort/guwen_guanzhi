## 1. 文章結構分析

### 第一層：敘事引入——李賀考進士被攻擊

> 「愈與李賀書，勸賀舉進士。賀舉進士，有名。與賀爭名者毀之，曰：『賀父名晉肅，賀不舉進士為是，勸之舉者為非。』聽者不察也，和而倡之，同然一辭。皇甫湜曰：『若不明白，子與賀且得罪。』愈曰：『然。』」

開門見山，交代事件背景：韓愈勸李賀考進士——李賀考了——有人攻擊他不該考，因為父親名叫「晉肅」——「晉」與「進士」的「進」同音。攻擊者不只針對李賀，連勸考的韓愈也一起告。皇甫湜提醒韓愈——不說清楚，你跟李賀都要倒霉。於是韓愈寫了這篇文章。

### 第二層：經典與法律的雙重反擊

> 「律曰：『二名不偏諱。』釋之者曰：『謂若言「徵」不稱「在」，言「在」不稱「徵」是也。』律曰：『不諱嫌名。』釋之者曰：『謂若「禹」與「雨」、「丘」與「菆」之類是也。』今賀父名晉肅，賀舉進士，為犯嫌名律乎？為犯二名律乎？父名晉肅，子不得舉進士；若父名仁，子不得為人乎？」

韓愈先引經典——《禮記·曲禮上》明明白白寫著「不諱嫌名」（同音字不必避諱）——再引法律——唐代律法也明文規定「不諱嫌名」——然後一記重拳：按照你們的邏輯，如果父親名叫「仁」，兒子是不是就不能當人了？這是全文最凌厲的一擊——歸謬法，把對手的論點推到極致，荒謬立即顯現。

### 第三層：歷史例證——聖賢不避諱

> 「夫諱始於何時？作法制以教天下者，非周公、孔子歟？周公作詩不諱，孔子不偏諱二名，《春秋》不譏不諱嫌名。曾子之父名皙，曾子不諱『昔』。周之時有騏期，漢之時有杜度，此其子宜如何諱？將諱其嫌，遂諱其姓乎？將不諱其嫌者乎？漢諱武帝名『徹』為『通』，不聞又諱『車轍』之『轍』為某字也。諱呂后名『雉』為『野雞』，不聞又諱『治天下』之『治』為某字也。」

韓愈從歷史中拉出一連串聖賢——周公作詩不避諱，孔子不避諱二名，《春秋》不批評不避嫌名的人，曾子父親名皙但他不避諱「昔」這個同音字。然後用了一個極妙的例子：「周之時有騏期，漢之時有杜度」——如果按照嫌名之諱的邏輯，「騏期」的兒子要不要避諱「期」和「騏」這兩個同音字？「杜度」的兒子要不要避諱「杜」和「度」？難道要連姓一起改掉？一連串反問，讓對手的論點在歷史事實面前土崩瓦解。

### 第四層：釜底抽薪——最後的道德羞辱

> 「今上章及詔，不聞諱『滸』『勢』『秉』『機』也。惟宦官宮妾，乃不敢言『諭』及『機』，以為觸犯。士君子立言行事，宜何所法守也？今考之於經，質之於律，稽之以國家之典，賀舉進士為可邪？為不可邪？凡事父母，得如曾子，可以無譏矣。作人得如周公、孔子，亦可以止矣。今世之士，不務行曾子、周公、孔子之行，而諱親之名，則務勝於曾子、周公、孔子，亦見其惑也。夫周公、孔子、曾子卒不可勝，勝周公、孔子、曾子，乃比於宦者宮妾，則是宦者宮妾之孝於其親，賢於周公、孔子、曾子者邪？」

最後一層是全文最高潮——連皇帝自己上朝下詔都不避嫌名（「滸」「勢」「秉」「機」分別觸犯唐太祖、太宗、代宗、玄宗的嫌名，但詔書中照樣使用）——只有宦官宮妾才在這些細枝末節上斤斤計較。你們這些「守禮君子」做的事，跟宦官宮妾有什麼區別？你們的「孝」，連周公、孔子、曾子都不如，卻比宦官宮妾還講究——這叫什麼孝？這是最辛辣的諷刺——不是罵你不懂禮，而是說——你的禮，跟太監的禮，是同一個水平。

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| 起 | 李賀因父名被攻擊不得考進士——韓愈為其辯護 |
| 承 | 引經典與法律——「不諱嫌名」——父名仁，子不得為人乎？ |
| 轉 | 列舉歷史聖賢——周公、孔子、曾子皆不避嫌名——周有騏期、漢有杜度 |
| 合 | 皇帝不避嫌名，唯宦官宮妾避之——你們的孝比周公孔子還高？ |

## 2. 寫作技巧分析

### 論辯武器——七種手段層層推進

韓愈在〈諱辯〉中用了七種論辯武器，由淺入深、步步緊逼：

| 武器 | 原文例 | 效果 |
|:----|--------|------|
| **引經** | 律曰「不諱嫌名」 | 以經典權威壓制對手 |
| **據律** | 唐代律法明文「二名不偏諱」 | 連法律都不站在對手那邊 |
| **稽典** | 周公、孔子、曾子、漢代避諱實例 | 歷史事實碾壓 |
| **設問** | 為犯嫌名律乎？為犯二名律乎？ | 逼對手正視問題 |
| **歸謬** | 父名仁，子不得為人乎？ | 把對手推到荒謬立場 |
| **類比** | 周有騏期、漢有杜度——其子如何諱？ | 用相似案例反擊 |
| **反諷** | 乃比於宦者宮妾——賢於周公孔子曾子者邪？ | 道德羞辱，一擊致命 |

### 最有力量的修辭——層層反問

〈諱辯〉全篇的節奏就是「提問—反駁—再提問—再反駁」。韓愈每設一問，對手就矮一截。全文幾乎沒有平鋪直敘的論述——全部以反問和設問推進：

| 反問 | 攻擊目標 |
|:----|---------|
| 為犯嫌名律乎？為犯二名律乎？ | 規則依據——你們的規則從哪裡來？ |
| 父名仁，子不得為人乎？ | 邏輯漏洞——推到極致荒謬立現 |
| 周之時有騏期——漢之時有杜度——此其子宜如何諱？ | 歷史實踐——你們比古人還嚴格？ |
| 士君子立言行事，宜何所法守也？ | 道德標準——你們跟誰學的？ |
| 宦者宮妾之孝於其親——賢於周公孔子曾子者邪？ | 人格羞辱——你們跟太監一個水平？ |

### 修辭手法一覽

| 手法 | 原文 | 效果 |
|:----|------|------|
| 歸謬 | 「父名仁，子不得為人乎？」 | 把對方推到荒謬絕倫的立場 |
| 反問串聯 | 連續多處「為……邪？為……邪？」 | 推理節奏如浪，一浪高過一浪 |
| 對比 | 皇帝不避嫌名 vs 宦官宮妾避之 | 高下立判 |
| 層遞 | 經典→法律→歷史→道德 | 四層夾擊，無路可退 |
| 反諷 | 「賢於周公、孔子、曾子者邪？」 | 最致命的一刀——不罵人勝過罵人 |

### 核心論證結構——「設問—反駁」的問答戰

〈諱辯〉最獨特的結構，是模擬法庭辯論的節奏——韓愈假設對手會怎麼說（「某曰」），然後用更凌厲的證據一一反駁。全篇就是一場「問答戰」：

1. **對手說：李賀不該考，因為父名晉肅**
2. **韓愈反駁：經典和法律都說「不諱嫌名」**——你的規則哪裡來的？
3. **對手說：避諱自古就是禮**
4. **韓愈反駁：周公孔子都不避諱**——你比聖人還懂禮？
5. **對手說：避諱是孝的表現**
6. **韓愈反駁：真正在乎這些的是宦官宮妾**——你的孝比宦官宮妾還講究？

每一層，韓愈都把對手推到一個更尷尬的位置——到最後，對手已經完全沒有說話的餘地。

## 3. 佳句賞析

### 「父名仁，子不得為人乎？」

如果父親名叫「仁」，兒子是不是就不能當人了？這是全文最鋒利的一刀——歸謬法的極致運用。韓愈不說你要講道理，而是直接把對方的邏輯推到荒謬絕倫的地步——讓你親眼看到，你們堅持的規則，推到極致會荒謬到什麼程度。十個字，說盡了所有虛偽禮教的荒謬本質。後世反禮教思潮——魯迅的「禮教吃人」——都可以追溯到這一句。

### 「周之時有騏期，漢之時有杜度，此其子宜如何諱？」

周朝有個叫騏期的人，漢朝有個叫杜度的人——按照你們的邏輯，他們的兒子該怎麼避諱？「騏期」——「騏」和「期」同音——如果避嫌名，兒子連父親的名字和姓都不能說？「杜度」——「杜」和「度」同音——難道兒子要改姓？這是一個極其機智的歷史類比——用具體人名，把抽象規則的荒謬暴露無遺。

### 「惟宦官宮妾，乃不敢言『諭』及『機』，以為觸犯。士君子立言行事，宜何所法守也？」

只有宦官宮妾才不敢說「諭」和「機」這些字——你們這些士君子，到底應該效法誰？這是全文最狠的一段——韓愈用「宦官宮妾」作為參照系，徹底剝奪了對手的道德高地。你們自以為是「守禮君子」——但你們做的事，跟那些最卑微、最沒見識的宮裡人一模一樣。

### 「夫周公、孔子、曾子卒不可勝，勝周公、孔子、曾子，乃比於宦者宮妾，則是宦者宮妾之孝於其親，賢於周公、孔子、曾子者邪？」

周公、孔子、曾子你們勝不了——但你們在避諱這件事上比他們還講究——這說明什麼？這說明你們的「孝」——跟宦官宮妾的「孝」是同一個路數——難道宦官宮妾的孝比周公、孔子、曾子還厲害？全文最後一句反問——把對手徹底釘在道德的最低處。不罵人，卻比任何罵人的話都狠。

## 4. 白話改編

我寫信勸李賀去考進士。李賀考了，也考出了名聲。那些跟李賀爭名的人就攻擊他說：「李賀的父親叫晉肅，他不考進士才是對的——勸他去考的人才是錯的。」旁聽的人也不細想，就跟著起哄，大家都這麼說。皇甫湜對我說：「你如果不把話說清楚，你跟李賀都要被定罪。」我說：「對。」

於是就有了這篇文章。

法律說：「兩個字的名字，不必每個字都避諱。」解釋的人說：「比如孔子母親叫『徵在』——說『徵』的時候不說『在』，說『在』的時候不說『徵』——就是這個道理。」法律又說：「不必避諱同音字。」解釋的人說：「就像『禹』和『雨』、『丘』和『菆』這類情況。」現在李賀的父親名叫晉肅——李賀考進士——是觸犯了「避嫌名」的法律？還是觸犯了「二名不偏諱」的法律？父親名叫晉肅，兒子就不能考進士——那如果父親名叫「仁」，兒子是不是就不能當人了？

再說，避諱這件事，是從什麼時候開始的？制定禮法來教化天下的，不就是周公和孔子嗎？周公寫詩不避諱，孔子不避諱雙名中的單一字，《春秋》也不批評不避嫌名的人。曾子的父親名叫「皙」，曾子從來不避諱說「昔」這個字。周朝有個叫「騏期」的，漢朝有個叫「杜度」的——他們的兒子該怎麼避諱？是為了避同音就連姓一起改掉？還是乾脆不管？漢朝因為避武帝劉徹的名諱，把「徹」字改成「通」——但沒聽說把「車轍」的「轍」也改成什麼別的字。漢朝避呂后名「雉」，把「雉」叫做「野雞」——但沒聽說把「治天下」的「治」也改成什麼別的字。

現在皇帝上朝下詔，沒聽說避諱「滸」「勢」「秉」「機」這些字——只有宦官宮妾才不敢說「諭」和「機」，覺得會觸犯忌諱。士君子立身行事，到底應該效法誰呢？

現在我查過經典——對照過法律——考察過國家的典章制度——李賀考進士，到底是行？還是不行？

侍奉父母，能做到曾子那樣——就沒有人能挑剔了。做人能做到周公、孔子那樣——也就到頂了。現在的人——不努力學習曾子、周公、孔子的德行——卻在避諱父母名字這件事上——非要超過曾子、周公、孔子——這也太糊塗了吧。周公、孔子、曾子——終究是沒辦法超過的——如果非要超過他們——那就只能跟宦官宮妾比了——難道說——宦官宮妾對父母的孝順——比周公、孔子、曾子還要高明嗎？
