# 〈諱辯〉背景資料

> **軍師分析 · 古文觀止卷八 · 唐文 · 韓愈**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《韓昌黎集》——韓愈詩文合集——卷十二〈雜著〉類——〈諱辯〉為韓愈為友人李賀辯護而作的論辯文 |
| **選集** | 《古文觀止》卷八·唐文——韓愈入選《古文觀止》共六篇（〈原道〉〈師說〉〈進學解〉〈雜說四〉〈送李愿歸盤谷序〉及本篇）——其中〈諱辯〉是最具「論戰」色彩的一篇——全文以層層反問——逼退禮教虛偽 |
| **體裁** | 辯——唐代古文運動的「論辯類」創作——不同於一般議論文——「辯」體以反駁對方論點為核心——常用設問、類比、歷史例證等手法——韓愈此文——堪稱「辯體」典範——問到你啞口無言 |
| **時代** | 中唐（約西元 9 世紀初）——避諱制度在唐代達到極盛——科舉考試中——考生若因父祖名諱而觸犯考題——輕則落第——重則終身不得應試——禮教之縛——前所未有 |
| **核心命題** | 避諱制度合理嗎？——「父名仁——子不得為人乎？」——韓愈用極致的邏輯推演——證明避諱的本質——是一種不經思考的盲從——真正的孝——不在於避諱——而在於行得正 |
| **關鍵人物** | **李賀**（唐代詩人——才華橫溢——因父名「晉肅」——「晉」與「進士」之「進」同音——被攻擊不得考進士——鬱鬱而終）、**韓愈**（為李賀辯護——寫下此篇千古雄文）、**皇甫湜**（支持李賀的友人之一——與韓愈一起為李賀奔走） |
| **文本重點** | 韓愈的朋友李賀——才華橫溢——卻因為父親名叫「晉肅」——「晉」與「進士」的「進」同音——被人攻擊說不能參加進士考試。韓愈寫了這篇〈諱辯〉為李賀辯護——指出避諱制度的荒謬——按照這種邏輯——如果父親叫「仁」——兒子是不是就不能當人了？他用大量歷史事實證明——聖賢之人——從不因為避諱而放棄正當的事——周公、孔子都不避諱——你們這些「守禮」的人——比周公、孔子還懂禮嗎？全文以排山倒海的提問——層層推進——每段都以「某曰」開頭——模擬對手發問——再一一反駁——讓人讀完——啞口無言 |
| **一句話總結** | **韓愈用一篇雄文撕開了禮教虛偽的面紗——當規則荒謬到阻礙正常社會運作時——真正的德行不是遵守規則——而是打破規則。李賀的悲劇不在於他父親的名字——而在於一群「守禮」的人——用一條荒謬的規則——毀掉了一個天才。韓愈問得好——「父名仁——子不得為人乎？」——道理沒那麼複雜——複雜的是——那些人的心。** |

## 二、時代背景——唐代避諱制度與李賀的悲劇

### 2.1 唐代避諱——從「禮」到「枷鎖」

避諱制度在唐代——已經從儒家的「敬祖」——演變為一種嚴苛的社會規範：

| 面向 | 說明 |
|------|------|
| **避諱範圍擴大** | 唐代不僅避「正諱」（父祖本名）——還避「嫌名」（同音字）——李賀的問題——「晉」與「進」同音——這在嚴格意義上屬於「嫌名」之諱——按禮經——「嫌名」本不當避——《禮記·曲禮上》云「不諱嫌名」——但到了唐代——執行力度遠超經典本意 |
| **科舉制度中的避諱** | 如果考生的父祖名諱——與考試科目、考題、甚至考官姓名衝突——都可能影響考試資格——李賀的狀況——父親名「晉肅」——「晉」與「進士」之「進」同音——攻擊者便以此為由——說他「不當舉進士」 |
| **避諱的荒謬化** | 唐代甚至出現——有人因為父名「嶽」——就不去終南山（因為終南山屬五嶽之一）——有人因為父名「仁」——就不敢跟人說「仁義」——避諱已經從敬祖——變成了生存障礙 |
| **韓愈的切入角度** | 韓愈的策略非常高明——他不是從「應不應該避諱」這個大問題入手——而是直接針對攻擊者的邏輯漏洞——「你們說李賀不該考進士——是因為嫌名之諱——但經典明明白白寫著『不諱嫌名』——你們是比經典還懂經典嗎？」 |

> **軍師按**：唐代的避諱制度——堪稱「禮教吃人」的經典案例——用一條連經典本身都不承認的規則——去否定一個天才的入仕資格——李賀的問題——不是他的父親叫什麼——而是他活在一個——規則可以凌駕於常識的時代。

### 2.2 李賀其人——天才的命運

李賀（約西元 790–816 年）——字長吉——唐代中期的天才詩人——人稱「詩鬼」——與李白（詩仙）、杜甫（詩聖）齊名——「太白仙才——長吉鬼才」——但——他的命運——極為悲慘：

| 面向 | 說明 |
|------|------|
| **才華** | 七歲能辭章——韓愈、皇甫湜初聞其名——不信——親至其家——命李賀賦詩——李賀揮筆立就〈高軒過〉——二人大驚——從此結為忘年交 |
| **科舉阻礙** | 李賀本該參加進士考試——卻因父名「晉肅」——「晉」與「進」同音——被人攻擊「不當舉進士」——韓愈為他寫〈諱辯〉——但終究沒能改變結果——李賀終身不得考進士 |
| **仕途坎坷** | 最後只做了一個從九品的小官（奉禮郎）——鬱鬱不得志——年僅二十七歲便去世——臨終前——將詩稿託付給友人 |
| **身後之名** | 李賀死後的詩名——遠遠超過他在世時的際遇——他那些瑰麗奇崛的詩句——「黑雲壓城城欲摧」「大漠沙如雪——燕山月似鉤」「天若有情天亦老」——成為唐詩中最獨特的存在——而他的早逝——與科舉之挫——有直接的因果關係 |

> **軍師按**：李賀的悲劇——是「才華敵不過規則」的典型案例。韓愈寫〈諱辯〉的時候——一定沒有想到——這篇他以為可以改變結果的文章——最終——什麼也沒改變。李賀還是考不了進士——還是二十七歲就死了——但〈諱辯〉——卻留了下來——成為對那個荒謬時代——最響亮的回擊。

## 三、文本結構——排山倒海的「問」

### 3.1 全文結構一覽

〈諱辯〉最獨特的結構——是以「設問—反駁」的節奏——層層推進——每一層——先設想對手會怎麼說——然後——用更凌厲的證據——把對手的論點——一個一個打碎：

| 層次 | 核心內容 | 論辯策略 |
|------|---------|---------|
| **第一層：釐清事實——李賀該不該考？** | 「愈與李賀書——勸賀舉進士——賀舉進士——有名——與賀爭名者毀之——曰：『賀父名晉肅——賀不舉進士為是——勸之舉者為非』——聽者不察也——和而倡之——同然一辭——皇甫湜曰：『若不明白——子與賀且得罪——』愈曰：『然。』」——開門見山——交代事件背景——韓愈勸李賀考進士——李賀考了——有人攻擊他不該考——因為父名晉肅——而韓愈身為勸考者——同樣有罪——皇甫湜提醒韓愈——你如果不出來說清楚——你也要被牽連——韓愈於是寫這篇文章 | 敘事引入——先立靶——再反擊——對手是誰——他們的論點是什麼——交代清楚——不讓讀者有模糊空間 |
| **第二層：經典權威——「不諱嫌名」** | 「律曰——『二名不偏諱』——釋之者曰：『謂若言「徵」不稱「在」——言「在」不稱「徵」是也』——律曰：『不諱嫌名』——釋之者曰：『謂若「禹」與「雨」——「丘」與「菆」之類是也』——今賀父名晉肅——賀舉進士——為犯嫌名律乎——為犯二名律乎——父名晉肅——子不得舉進士——若父名仁——子不得為人乎？」 | 引經典（《禮記》「不諱嫌名」）——引法律（唐代律法也明文「不諱嫌名」）——然後——用最凌厲的提問收尾——「父名仁——子不得為人乎？」——這一問——是全文的刀鋒所在 |
| **第三層：歷史例證——聖賢不避諱** | 「夫諱——始於何時——作法制以教天下者——非周公、孔子歟——周公作詩不諱——孔子不偏諱二名——《春秋》不譏不諱嫌名——曾子之父名皙——曾子不諱『昔』——周之時有騏期——漢之時有杜度——此其子——宜如何諱——將諱其嫌——遂諱其姓乎——將不諱其嫌者乎——漢諱武帝名『徹』為『通』——不聞又諱『車轍』之『轍』為某字也——諱呂后名『雉』為『野雞』——不聞又諱『治天下』之『治』為某字也」 | 列出一連串歷史事實——周公不避諱——孔子不避諱——曾子不避諱——漢代避諱——只避正諱——不避嫌名——你們這些攻擊李賀的人——比周公、孔子還懂得禮嗎？ |
| **第四層：釜底抽薪——真正的孝不在於避諱** | 「今上章及詔——不聞諱『滸』『勢』『秉』『機』也——（此四字分別觸犯唐太祖、太宗、代宗、玄宗的嫌名）——惟宦官宮妾——乃不敢言『諭』及『機』——以為觸犯——士君子立言行事——宜何所法守也——今考之於經——質之於律——稽之以國家之典——賀舉進士——為可邪——為不可邪——凡事父母——得如曾子——可以無譏矣——作人——得如周公、孔子——亦可以止矣——今世之士——不務行曾子、周公、孔子之行——而諱親之名——則務勝於曾子、周公、孔子——亦見其惑也——夫周公、孔子、曾子——卒不可勝——勝周公、孔子、曾子——乃比於宦者宮妾——則是宦者宮妾之孝於其親——賢於周公、孔子、曾子者邪？」 | 最終的致命一擊——連皇帝自己上朝下詔——都不避嫌名——只有宦官宮妾才在這上面斤斤計較——你們這些「守禮君子」——做的事——跟宦官宮妾有什麼區別？——你們的「孝」——連周公、孔子都不如——卻比宦官宮妾還講究——這叫什麼孝？——這是全文最辛辣的諷刺 |

> **軍師按**：〈諱辯〉的結構——就是一場「問答戰」——韓愈每設一問——對手就矮一截——到最後——對手已經完全沒有說話的餘地——因為韓愈用經典、法律、歷史——三層夾擊——最後再用「宦官宮妾」這個對比——把對手的道德位置——拉到最低。高手辯論——不是讓對方啞口無言——而是讓對方——自己都覺得自己——荒謬可笑。

## 四、核心命題——「父名仁，子不得為人乎？」

### 4.1 韓愈的邏輯武器

〈諱辯〉最精彩的地方——不是它的結論——而是它的邏輯推演方式：

| 邏輯步驟 | 內容 | 效果 |
|---------|------|------|
| **第一步：釐清規則** | 經典說「不諱嫌名」——法律說「不諱嫌名」——你們拿什麼規則來攻擊李賀？ | 釜底抽薪——斬斷對手的規則依據 |
| **第二步：還原事實** | 周公不諱——孔子不諱——漢代不諱嫌名——唐代皇帝也不諱嫌名——你們比誰都嚴格？ | 歷史證據碾壓——對手的論點在史實面前不堪一擊 |
| **第三步：極限推演** | 「父名仁——子不得為人乎？」——按照你們的邏輯——推到極致——荒謬立現 | 歸謬法——讓對手自己發現自己的荒謬 |
| **第四步：道德剝離** | 真正在乎這些細枝末節的是誰？宦官宮妾——你們跟他們一樣——還自以為比周公孔子懂禮？ | 道德羞辱——你們的「禮」——跟太監的「禮」——是一個級別的 |

> **軍師按**：韓愈這篇文章——不僅贏了論點——還贏了姿態。他從頭到尾——沒有罵人——但每一句話——都像耳光一樣——打在那群攻擊者的臉上。最後那句「宦者宮妾之孝於其親——賢於周公、孔子、曾子者邪？」——是全文最狠的一刀——不是罵你不懂禮——而是說——你的禮——跟太監的禮——一個水平。

### 4.2 避諱的本質——「禮」的異化

韓愈的文章——雖然是為李賀辯護——但它的意義——遠遠超出了這個具體事件：

| 面向 | 分析 |
|------|------|
| **禮的初衷** | 避諱——原本是為了表達對祖先的尊敬——是一種情感的表現——「敬其所尊——愛其所親」——出發點是好的 |
| **禮的異化** | 但當「避諱」從一種自願的情感——變成一種強制的規則——從「避本名」——擴大到「避嫌名」——甚至擴大到「同音字都不能說」——它就從「禮」——變成了「枷鎖」——從「敬祖」——變成了「害人」 |
| **規則與常識的衝突** | 韓愈最深刻的洞見是——當規則與常識衝突時——錯的是規則——不是常識——「父名仁——子不得為人乎？」——這個問題的荒謬——不需要任何學問——只要一個正常的大腦——就能判斷 |
| **誰在維護荒謬的規則** | 韓愈特別點出——攻擊李賀的人——不是真的「懂禮」——而是「與賀爭名者」——他們打著禮教的旗號——做的是打壓競爭對手的勾當——這——才是〈諱辯〉真正的刀鋒所向——禮教——從來不是他們的信仰——只是他們的武器 |

> **軍師按**：韓愈〈諱辯〉之所以千古傳誦——不是因為它解決了李賀的問題（它沒有）——而是因為它提出了一個永恆的問題——「當規則荒謬到傷害正常人的正常生活時——你是選擇守規則——還是選擇做正確的事？」韓愈的選擇——從來都只有一個——「做正確的事——哪怕得罪全世界。」

## 五、歷史定位——古文運動中的論辯典範

### 5.1 〈諱辯〉在韓愈創作中的位置

| 面向 | 分析 |
|------|------|
| **類型定位** | 這是韓愈「論辯文」中最鋒利的一篇——與〈原道〉（講儒家道統）〈師說〉（講從師之道）不同——〈諱辯〉的風格——更加直接、尖銳——帶有強烈的「訟辯」色彩——韓愈像是法庭上的律師——為李賀——也為真理——做了一場精彩的辯護 |
| **與韓愈其他作品的關係** | 〈諱辯〉與〈原毀〉（批評毀謗之風）精神相通——韓愈一生——都在與「虛偽的禮教」和「不經思考的盲從」作戰——〈原道〉打的是佛老——〈師說〉打的是「位卑則足羞」的門第觀念——〈諱辯〉打的是「避諱」這個荒謬的制度——每一篇——都在挑戰一個「大家都覺得理所當然」的社會規則 |
| **論辯技巧的巔峰** | 韓愈的論辯技巧——在〈諱辯〉中達到巔峰——他用了「引經」「據律」「稽典」「設問」「歸謬」「類比」「反諷」——七種論辯武器——層層遞進——最後一擊——把對手打入道德谷底——後世論辯文——無出其右 |
| **思想史意義** | 〈諱辯〉的核心——是對「盲從」的批判——韓愈告訴我們——「守禮」不等於「有德」——真正有德的人——是知道什麼時候該守禮——什麼時候——該打破禮——這種「規則之上的判斷力」——是韓愈思想中最寶貴的部分 |

### 5.2 對後世的影響

| 面向 | 說明 |
|------|------|
| **對科舉制度的影響** | 〈諱辯〉雖然沒能改變李賀的命運——但它讓後世的人——開始反思避諱制度的荒謬——宋代以後——「不諱嫌名」的原則——逐漸被重新重視 |
| **對論辯文的影響** | 〈諱辯〉的論辯結構——成為後世論辯文的典範——尤其是「設問—反駁」的節奏——被蘇軾、王安石等大家繼承發展 |
| **對思想的影響** | 「父名仁——子不得為人乎？」——這句話——成為中國思想史上——對「禮教吃人」——最早也最響亮的控訴——之後千年的反禮教思潮——都可以在〈諱辯〉中找到源頭 |

> **軍師評**：韓愈寫〈諱辯〉——不是為了改變李賀的命運——他知道他改變不了——他是為了——讓後人知道——曾經有一個天才——因為一條荒謬的規則——毀掉了——「當你看到不公不義的事——不要沉默——說出來——寫下來——哪怕什麼也改變不了——至少——這個世界——會因為你的聲音——而不再那麼——理所當然地荒謬。」
