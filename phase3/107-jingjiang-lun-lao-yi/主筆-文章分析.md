## 1. 文章結構分析

**第一段：兒子勸母——您別紡織了**

> 「公父文伯退朝，朝其母，其母方績。文伯曰：『以歜之家，而主猶績，懼忤季孫之怒也。其以歜為不能事主乎？』」

公父文伯退朝回家，看見母親敬姜正在紡織，便說：「以我們這樣的家世，您還在紡織——我怕季孫氏會生氣，以為我沒有好好奉養您。」

**第二段：敬姜的教訓——勞動是道德的根本**

> 「敬姜嘆曰：『魯其亡乎！使僮子備官而未之聞耶？居，吾語女！夫民勞則思，思則善心生；逸則淫，淫則忘善，忘善則惡心生。沃土之民不材，淫也；瘠土之民莫不向義，勞也。』」

敬姜長嘆一聲：魯國大概要亡了吧！讓孩子當官卻沒有告訴他最基本的道理——勞動使人思考，思考使人產生善心；安逸使人放縱，放縱使人忘記善，忘善則產生惡心。

**第三段：從天子到庶人——全員勞動制度**

> 從天子到庶人，她列出每一個階層每日必須完成的勞動：天子有朝日、考政、夕月、監九御；諸侯有修天子之命、考國職、省典刑；卿大夫有考職、講庶政、序業；士有受業、講貫、習復；庶人從日出勞動到日落。

**第四段：結論——「君子勞心，小人勞力」**

> 「君子勞心，小人勞力，先王之訓也。自上以下，誰敢淫心舍力？」

君子用腦力勞動、小人用體力勞動——這是先王的教訓。從上到下，誰敢放縱心智、捨棄勞力？

## 2. 寫作技巧分析

### 「勞則思，思則善心生」——道德養成的因果鏈

敬姜的核心論證是一條因果鏈：

**勞動 → 思考 → 善心生 → 品德堅固**
**安逸 → 放縱 → 忘善 → 惡心生 → 墮落犯罪**

這個因果鏈的革命性在於：她把勞動從「生存手段」提升到了「道德養成」的層次。勞動不是為了賺錢，而是為了維持心靈的健康。

### 全員勞動模型

敬姜提出了一個從天子到庶人的「全員勞動模型」——每個階層都有自己必須完成的勞動，沒有人可以豁免。這在貴族階層普遍以逸樂為榮的時代是極為前衛的觀點：

| 階層 | 勞動內容 | 
|:----|:--------|
| 天子 | 朝日、考政、夕月、監九御 |
| 諸侯 | 修天子之命、考國職、省典刑 |
| 卿大夫 | 考職、講庶政、序業 |
| 士 | 受業、講貫、習復、計過 |
| 庶人 | 明而動、晦而休 |

## 3. 佳句賞析

### 「夫民勞則思，思則善心生；逸則淫，淫則忘善，忘善則惡心生。」

人勞動就會思考，思考就會產生善心；安逸就會放縱，放縱就會忘記善，忘記善就會產生惡心。這是一套完整的人格養成理論——道德不是天生的，而是在勞動中鍛鍊出來的。

### 「沃土之民不材，淫也；瘠土之民莫不向義，勞也。」

肥沃土地上的人民不成材——因為安逸放縱；貧瘠土地上的人民都很講道義——因為勞動磨礪了他們。這是一個大膽的命題：富足反而有害，貧窮反而有益。敬姜把環境決定的命題與道德養成結合，形成了極具衝擊力的論證。
