# 〈諫太宗十思疏〉— 文章分析

> 魏徵
>
> 本檔案由主筆分析

---

## 1. 文章結構分析

### 全文三層結構：提出問題 → 歷史教訓 → 十思解方

〈諫太宗十思疏〉全文約六百餘字，以「三段遞進」的結構展開——先設喻立論、再借史為鑑、最後提出具體解方。三段之間環環相扣，構成一個完整的論證閉環。

---

#### 第一段：設喻立論——「思國之安者，必積其德義」

> 「臣聞求木之長者，必固其根本；欲流之遠者，必浚其泉源；思國之安者，必積其德義。」

魏徵開篇連用三個比喻——樹木（根本）、水流（泉源）、國家（德義）——層層推進，從自然現象推到治國道理。這是典型的「由物及人」的論證手法：木要長高必須先固根，水要流遠必須先疏源，國家要安定必須先積德——三個「必」字，斬釘截鐵，毫無商量餘地。

然後一轉——「源不深而望流之遠，根不固而求木之長，德不厚而思國之安」——從正面論述翻為反面假設，指出當今的問題：「不念居安思危，戒奢以儉，德不處其厚，情不勝其欲。」這段話等於直接點名批評——皇帝你正在走向驕奢，你的德義不夠厚、你的慾望已經壓過了理智。

> **結構功能**：第一段的核心任務是「建立論證前提」——用無可反駁的比喻，讓讀者(唐太宗)不得不接受「德義是治國的根本」這個前提。前提一旦成立，後面的論證就有了立足點。

---

#### 第二段：借史為鑑——「載舟覆舟，所宜深慎」

> 「凡百元首，承天景命，善始者實繁，克終者蓋寡。」

第二段從比喻轉向歷史。魏徵提出一個千古難題：為什麼每個皇帝開頭都很好，但很少有人能善終？答案——「取之易而守之難」。創業時戰戰兢兢、如履薄冰，所以什麼都做對了；成功之後開始鬆懈、開始驕傲、開始覺得一切都是理所當然——然後就垮了。

這段最關鍵的一句話：「怨不在大，可畏惟人；載舟覆舟，所宜深慎。」——百姓對你的怨恨不在於大小，可怕的是人心的向背。水可以載船，也可以翻船——這不是抽象的道理，而是李世民親眼所見的隋朝教訓。

> **結構功能**：第二段的核心任務是「製造危機感」——用歷史規律告訴李世民：你現在正處在「善始」與「克終」之間的危險地帶。不是你要不要警惕——而是你如果不警惕，歷史規律就會重演。這一段把第一段的「德義必要論」轉化為「不德義必亡論」——從應該做，變成了必須做。

---

#### 第三段：提出十思——十個方向的自我提醒

> 「誠能見可欲則思知足以自戒，將有作則思知止以安人……」

在論證了「為什麼必須警惕」之後，魏徵提出「十思」——不是十條禁令，而是十個方向性的提醒。每一思都以一個觸發情境開頭（見可欲、將有作、念高危……），以「思」字引導皇帝進入反省，以「以」字導出具體行動。這個句型結構本身就是一種方法論——它不是告訴你「不要做什麼」，而是教你在每個關鍵時刻「停下來想一想」。

第三段的結尾是全文最樂觀的部分——「文武爭馳，君臣無事，可以盡豫游之樂，可以養松喬之壽。」——只要做到十思，皇帝不但可以安享太平，還可以像仙人赤松子、王子喬一樣長壽。這是在前面兩段沉重的壓力之後，給予一個積極的遠景——做對了，什麼都好。

> **結構功能**：第三段是「給解方」。前面兩段說「你有問題」「問題很嚴重」，第三段說「但你可以解決——用這個方法」。這個結構讓整篇文章從批評變成了建設——從指責變成了幫助。

---

### 十思的內部結構——三層遞進的自我管理系統

「十思」不是隨意排列的十條建議——它們有一個內在的邏輯層次，由內而外、由己及人、由修心到行政：

| 層次 | 編號 | 原文關鍵句 | 面向 | 核心邏輯 |
|:----:|:----:|-----------|:----:|---------|
| **第一層：修身克己**<br>（對待自己的慾望） | 第一思 | 見可欲則思知足以自戒 | 貪慾 | 慾望來了——先問「夠了嗎」 |
| | 第二思 | 將有作則思知止以安人 | 興作 | 想搞大工程——先問「該停嗎」 |
| | 第三思 | 念高危則思謙沖而自牧 | 驕傲 | 位高權重——更需低姿態 |
| | 第四思 | 懼滿溢則思江海下百川 | 自滿 | 怕自滿——想想海納百川 |
| **第二層：勤政納諫**<br>（對待工作的態度） | 第五思 | 樂盤遊則思三驅以為度 | 享樂 | 喜歡玩——要有節制 |
| | 第六思 | 憂懈怠則思慎始而敬終 | 懈怠 | 怕懶散——要善始善終 |
| | 第七思 | 慮壅蔽則思虛心以納下 | 納諫 | 怕被蒙蔽——要虛心聽意見 |
| | 第八思 | 想讒邪則思正身以黜惡 | 小人 | 怕小人——先端正自己 |
| **第三層：公平執法**<br>（對待他人的賞罰） | 第九思 | 恩所加則思無因喜以謬賞 | 獎賞 | 高興時——別亂賞 |
| | 第十思 | 罰所及則思無以怒而濫刑 | 處罰 | 生氣時——別亂罰 |

> **主筆按**：「十思」的排列順序本身就是一種心理洞察——先從最私人的慾望開始（見可欲、將有作），然後擴展到對待權力的態度（念高危、懼滿溢），再到對待工作的紀律（樂盤遊、憂懈怠），最後才是對待他人的公正（慮壅蔽、想讒邪、恩所加、罰所及）。這不是一份隨手列出的清單——而是一個從內到外的自我修養系統：你先管好自己的慾望，才能管好權力；你先管好權力，才能管好工作；你先管好工作，才能公正地對待他人。

---

### 十思的句型結構——「情境 → 思考 → 行動」的標準模板

每一思都遵循一個固定的三拍節奏，這個句型本身就是魏徵的方法論：

```
情境觸發  →  思考引導  →  行動導向
（見/將/念/懼/樂/憂/慮/想/恩/罰）  →  （則思）  →  （以……）
```

以第一思為例：
- **情境**：「見可欲」——看到想要的東西（這是每個人都會遇到的時刻）
- **思考**：「則思知足」——停下來想想「夠了沒有」
- **行動**：「以自戒」——用這個思考來告誡自己

魏徵沒有說「你不要貪心」——而是說「當你看到想要的東西時，停下來想一想『夠了沒有』」。這不是一個禁令，而是一個**提醒自己的方法**。這個區別至關重要——禁令是外來的約束，方法是內在的修養。

---

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| **起** | 求木之長者必固根本——治國的根本在於德義——「不念居安思危」是當前問題 |
| **承** | 善始者實繁，克終者蓋寡——歷史規律證明創業易守成難——「載舟覆舟，所宜深慎」 |
| **轉** | 從「為什麼要警惕」轉到「如何警惕」——提出十思作為解方 |

## 2. 寫作技巧分析

### 喻證法——用自然規律建立無可反駁的前提

魏徵開篇用「木之長者必固根本」「流之遠者必浚泉源」兩個比喻——這兩個比喻的力量在於它們是自然規律，沒有人能夠否認。樹要長高必須根深，水要流遠必須源長——這是常識，不需要論證。魏徵巧妙地把這個常識推到治國上：「思國之安者，必積其德義」——治國和種樹、疏水一樣，都有不可違反的根本原則。這個類比一旦成立，唐太宗就無法反駁——因為他無法否認樹要根深、水要源長——所以他也就無法否認治國要積德義。

### 正反對比——從肯定到否定，不留退路

魏徵在同一段中先正面論證（「求木之長者，必固其根本」），再反面推導（「源不深而望流之遠……不勝其欲」），形成完整的邏輯閉環。這種「正面說應該怎麼做 + 反面說不這樣會怎樣」的結構，讓論證無懈可擊——要麼你接受我的建議，要麼你接受失敗的必然性。

### 歷史論證——用集體記憶製造壓迫感

「凡百元首，承天景命，善始者實繁，克終者蓋寡」——這不是理論推導，而是歷史統計。魏徵把所有朝代皇帝都拉出來作證——絕大多數人開頭做得好，但很少有人能堅持到底。然後用隋朝這個最近的例子——隋煬帝的教訓就發生在幾十年前，李世民親眼見證——「載舟覆舟」不是比喻，是真實發生過的事情。歷史的重量壓下來，讓唐太宗無法輕視。

### 「十思」的語言節奏——整飭中有變化

十思在句式上是整齊的排比（每句都是「X則思Y以Z」），但魏徵在整齊中製造了變化：

- 前兩字（情境觸發詞）變換多樣：見、將、念、懼、樂、憂、慮、想、恩、罰——動詞涵蓋了視覺、行為、心理、情感各個層面
- 中間的「思」對象各不相同：知足、知止、謙沖、江海下百川、三驅為度、慎始敬終、虛心納下、正身黜惡、無因喜謬賞、無以怒濫刑——從個人修養到行政原則
- 結尾的「以」字引導的行動：自戒、安人、自牧、下百川、為度、敬終、納下、黜惡、謬賞、濫刑——每個行動都精準對應開頭的情境

這種「同中有異」的排比結構，讓十思讀起來規整而靈動——規整是因為魏徵在傳遞一個統一的思考方法，靈動是因為每一條都針對不同的具體情境。

### 修辭手法一覽

| 手法 | 原文 | 效果 |
|:----|------|------|
| **比喻** | 「求木之長者，必固其根本；欲流之遠者，必浚其泉源」 | 用自然規律建立不可反駁的前提 |
| **排比** | 十思的完整句式（X則思Y以Z） | 節奏規整，氣勢連貫，強化方法論的系統性 |
| **對比** | 「善始者實繁」vs「克終者蓋寡」 | 突出創業易守成難的矛盾 |
| **引用/用典** | 「載舟覆舟」（出自《孔子家語》/《荀子》） | 借古人的權威強化論證 |
| **反襯** | 前兩段寫壓力（必須警惕），第三段寫遠景（可以養壽） | 以苦口良藥始、以美好承諾終——既有說服力又不讓人絕望 |
| **層遞** | 第一層修己 → 第二層勤政 → 第三層執法 | 由內而外，層層擴大——先解決自己的問題，再解決治國的問題 |
| **設問** | 「善始者實繁，克終者蓋寡……豈取之易而守之難乎？」 | 自問自答，引導讀者思考 |
| **警句** | 「載舟覆舟，所宜深慎」「思國之安者，必積其德義」 | 精煉有力，易記易誦——便於傳播 |

### 結構上的匠心——「三段遞進」的論證閉環

〈諫太宗十思疏〉的結構之所以經典，是因為三段之間構成了一個完整的「問題→原因→解方」的論證閉環：

```
第一段 ───→ 提出命題：德義是治國的根本
   │
   │  （用比喻建立前提，讓皇帝無法否認）
   │
   ▼
第二段 ───→ 驗證命題：不積德義的後果是「載舟覆舟」
   │
   │  （用歷史規律製造危機感）
   │
   ▼
第三段 ───→ 實踐命題：十思——如何積德義的具體方法
   │
   │  （用系統性的自我提醒方法給出解方）
   │
   ▼
   閉環：從「應該做」到「必須做」到「如何做」
```

這個結構的巧妙在於：第一、二段解決的是「為什麼要做」的動機問題——用比喻和歷史讓皇帝心甘情願接受建議；第三段解決的是「怎麼做」的方法問題——用十思給出可執行的行動指南。如果魏徵直接說「陛下你應該十思」，沒有前面的鋪墊，這就是說教；有了前面的鋪墊，這就是幫助。

## 3. 佳句賞析

### 「求木之長者，必固其根本；欲流之遠者，必浚其泉源；思國之安者，必積其德義。」

全文的開篇三句，也是全文的理論基石。三個「必」字，一氣呵成——沒有商量、沒有猶豫。木要長高必須根深，水要流遠必須源長——這是沒有人能反駁的自然常識。魏徵把這個常識推到治國上，等於在告訴李世民：治國的常識和種樹、疏水一樣簡單——你做不到不是因為你不知道，而是因為你不想做。這三句的修辭力量在於：它不是論證——它是提醒。你本來就知道這個道理，只是你忘了。

### 「不念居安思危，戒奢以儉，德不處其厚，情不勝其欲。」

這是全文最直接、最不留情面的批評——而且是寫在奏疏裡，要給皇帝看的。魏徵沒有一句客套話，沒有「陛下或許可以考慮」這樣的委婉修辭——他直接說：你現在的問題是——你沒有居安思危、你沒有戒奢以儉、你的德義不夠厚、你的慾望壓過了理智。在中國政治史上，敢在奏疏裡這樣對皇帝說話的人不多——魏徵是其中最著名的一個。

### 「怨不在大，可畏惟人；載舟覆舟，所宜深慎。」

全文的核心警句。「怨不在大」——不是說小怨恨沒關係，而是說「怨恨的大小不重要，重要的是人心向背」。一個小怨累積起來，足以顛覆整個王朝。「可畏惟人」——唯一真正可怕的，是人民。「載舟覆舟」——水可以載船、也可以翻船——你以為船穩了，但水從來沒有給過任何船永遠不沉的承諾。「所宜深慎」——這是應該深深警惕的事情。十六個字，包含了完整的論證：什麼可怕（怨）→為什麼可怕（惟人）→有多可怕（載舟覆舟）→應該怎麼辦（深慎）。

### 「見可欲則思知足以自戒，將有作則思知止以安人……」

這是「十思」的開頭兩句，也是最能代表十思句型的句子。注意魏徵的思維方式——他不是說「不要貪心」「不要大興土木」——他是說「當你看到想要的東西時，停下來想一想夠不夠」「當你想搞大工程時，停下來想一想該不該停」。這是「過程指導」而非「結果告知」——他在教皇帝一個思考習慣，而不只是告訴他結論。整句話的結構——「情境 → 思考 → 行動」——就是一個可以反覆使用的自我管理工具。

### 「文武爭馳，君臣無事，可以盡豫游之樂，可以養松喬之壽。」

全文的收尾——也是唯一輕鬆的句子。在前面兩段的壓力和第三段的層層提醒之後，魏徵終於給出了一個美好的遠景：只要做到十思，文武百官各盡其職、君臣之間無事可憂——你就可以盡情享受遊玩的快樂、像仙人一樣長壽。這個結尾的作用不是真的在許諾長壽——而是在前面說了那麼多讓人不舒服的話之後，給皇帝一個台階下、一個美好的盼頭。諫官的智慧——不只在於敢說難聽的話，更在於知道什麼時候該說好聽的話。

## 4. 白話改編

我聽說：要使樹木長得高大，一定要穩固它的根本；要使水流得長遠，一定要疏通它的源頭；要使國家安定，一定要累積道德仁義。源頭不深卻希望水流長遠，根本不穩固卻希望樹木長大，道德不深厚卻希望國家安定——我雖然愚笨，也知道這是不可能的，更何況是明智的人呢！

君主掌握著國家的重大權力，處於天地間最崇高的地位——如果不想到居安思危、用節儉來戒除奢侈——就像砍樹的根卻希望枝葉茂盛、堵塞源頭卻想要水流長遠一樣，是做不到的啊！

歷代所有的君主，承受上天的大命——開始做得好的人非常多，能夠堅持到最後的人卻很少。難道是奪取天下容易、守成天下困難嗎？大概因為創業的時候憂患重重，所以能夠竭盡全力對待百姓；成功之後就開始縱情享受、驕傲自滿——所以誰也看不起。用盡全力對待百姓，那麼就算像吳國越國那樣敵對的雙方也會團結一心（江南的老百姓都會擁護你）；傲慢地對待百姓，那麼就算是骨肉至親也會變成路人。雖然用嚴刑來威嚇人民、用怒氣來壓制人民——人們最終只會表面順從、內心不服——怨恨不在於大小，可怕的是人心。水可以載船，也可以翻船——這是應該深深警惕的事情！

如果真的能夠做到——看到想要的東西就想到知足，用來告誡自己；將要大興土木就想到適可而止，用來安定百姓；擔心地位高權力大就想到謙虛低調，修養自己的品德；害怕驕傲自滿就想到江海容納百川的胸懷；喜歡遊獵享樂就想到以一年三次圍獵為限度；擔心意志鬆懈就想到開始謹慎、結尾更要慎重；擔心被矇蔽就想到虛心聽取下面的意見；害怕讒佞小人就想到端正自身來斥退邪惡；施加恩惠的時候就想到不要因為高興就錯誤地獎賞；執行刑罰的時候就想到不要因為生氣就濫施刑罰——

完全做到這十個方面，選拔賢能的人來任用他，挑選好的意見來聽從它——那麼聰明的人就會貢獻所有智謀、勇敢的人就會用盡全身力氣、仁德的人就會傳播所有恩惠、誠信的人就會獻出所有忠心。文武百官爭先恐後地做事，君臣之間沒有煩惱——您可以盡情享受遊玩的快樂、可以像赤松子王子喬一樣長壽。彈琴奏瑟、垂衣拱手——不用說什麼話，天下就已經治理得很好了了。為什麼還要在勞神費思、代替下屬管理職務、勞累自己耳聰目明的聽覺視覺——虧損了無為而治的最高治國之道呢！

### 十思句型模板——可反覆使用的思考工具

每一思都是一個完整的「自我提醒流程」，可以拆解為三個步驟：

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  ① 情境觸發  │ ──→ │  ② 思考引導  │ ──→ │  ③ 行動導向  │
│             │     │             │     │             │
│  當你遇到    │     │  停下來想想   │     │  用這個思考   │
│  這個情況時…  │     │  這件事…     │     │  來約束自己…  │
│             │     │             │     │             │
│  見可欲      │     │  則思知足    │     │  以自戒      │
│  將有作      │     │  則思知止    │     │  以安人      │
│  念高危      │     │  則思謙沖    │     │  以自牧      │
│  ……        │     │  ……        │     │  ……        │
└─────────────┘     └─────────────┘     └─────────────┘
```

> **主筆按**：魏徵最了不起的貢獻不是提出了十條具體的要求——而是發明了一個「如何提醒自己」的方法論。他把「思」變成了一個可以隨時隨地使用的心智工具：遇到任何可能犯錯的時刻，都先用「思」字叫自己停下來、想一想——然後再決定怎麼做。這個方法比任何具體要求都重要——因為具體要求會過時，但「停下來想一想」這個習慣——在任何時代、任何位置——都永遠有效。

### 為什麼「思」比「規矩」更高明？

| 面向 | 規矩（禁令） | 思（方法） |
|:----:|:-----------|:---------|
| **本質** | 告訴你「不能做什麼」 | 告訴你「該怎麼想」 |
| **適用範圍** | 只針對特定情境 | 可以應對任何新情境 |
| **持續時間** | 規矩會過時 | 方法永遠有效 |
| **執行方式** | 被動遵守 | 主動思考 |
| **權力關係** | 上對下的命令 | 自我的內在修養 |
| **核心問題** | 「我違反了哪條規定？」 | 「我是不是需要停下來想一想？」 |

魏徵選擇用「思」而不是用「禁」——是因為他知道，對於一個掌握絕對權力的皇帝來說，沒有人能夠真正「禁止」他做任何事。如果你給他一千條禁令——他只要心情不好，一條都不會遵守。但如果你教他一個「停下來想一想」的習慣——那麼在他每一次即將犯錯的時刻，這個習慣都會跳出來拉他一把。十思不是規矩——十思是訓練李世民養成一個終身受用的好習慣。

---

### 一句話總結

> **〈諫太宗十思疏〉不是十條規矩——而是一個「在每一個可能犯錯的時刻停下來想一想」的方法。魏徵用三段遞進的結構完成了從「為什麼要警惕」（比喻+歷史）到「如何警惕」（十思）的完整論證。十思的內部排列由內而外——先修身（第一至四思）、再勤政（第五至八思）、後執法（第九、十思）——構成了一個完整的領導者自我管理系統。而十思的核心，不是「什麼不能做」——而是「在關鍵時刻，你願不願意叫自己停下來，問一句：『我這樣做，對嗎？』」——這個問題——才是魏徵真正留給後世所有掌權者的——最珍貴的遺產。**
