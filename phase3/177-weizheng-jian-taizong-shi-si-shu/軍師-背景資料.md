## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《舊唐書·魏徵傳》卷七十一；另見《貞觀政要》卷一〈君道〉——兩處均載魏徵此疏全文 |
| **選集** | 《古文觀止》卷七·唐文——卷七唐文類的首篇名作，也是魏徵唯一入選《古文觀止》的篇章 |
| **體裁** | 疏——古代臣子向皇帝上奏的一種文體（又稱「奏疏」「上疏」），用於陳述意見、提出諫議 |
| **時代** | 唐太宗貞觀十一年前後（約西元 637 年）——此時距「貞觀之治」的開端已約十年，唐朝國力強盛，社會安定 |
| **核心命題** | 創業容易守成難——「載舟覆舟，所宜深慎」——統治者不能因為天下已定就放鬆警惕 |
| **關鍵人物** | 魏徵（唐太宗最敢言的諫臣）、唐太宗李世民 |
| **文本重點** | 魏徵在唐太宗逐漸驕奢之際，上疏提出「十思」——從十個方面提醒皇帝要自我克制：知足、知止、謙沖、節制、勤政、納諫、賞罰公平等。魏徵的核心論證是——統治者不能因為天下已定就放鬆警惕，「載舟覆舟」的道理永遠不變——「思」不是知識問題，而是態度問題——「不思」才是最大的災難 |
| **一段話總結** | **〈諫太宗十思疏〉不是十條規矩，而是一個清醒的提醒——當一個領導者開始覺得一切理所當然的時候，就是他最危險的時候。魏徵說的不只是治國——他說的是所有權力擁有者共同的盲點：以為過去的成功會自動延續到未來——卻忘了「水能載舟，亦能覆舟」——你以為船已經穩了——但水從來沒有給過任何船——永遠不沉的承諾。** |

## 二、時代背景——貞觀盛世與君臣共治

### 2.1 從玄武門到貞觀之治

西元 626 年，李世民發動玄武門之變，殺兄（太子李建成）、弟（齊王李元吉），逼迫父親唐高祖李淵退位，登基為帝，次年改元貞觀。

| 面向 | 說明 |
|------|------|
| **貞觀之治的形成** | 李世民即位後，以「亡隋為鑑」——親眼目睹了隋煬帝因驕奢淫逸、拒諫飾非而導致隋朝迅速滅亡的教訓——下定決心改革政治。他廣開言路、尊重諫官、輕徭薄賦、完善三省六部制——使唐朝在短短十年內——從內戰的廢墟中崛起——成為東亞最強大的帝國 |
| **魏徵的角色** | 魏徵原是太子李建成手下的幕僚——玄武門之變後被李世民重用——成為貞觀朝最著名的諫官——以「敢言直諫」聞名——史書記載他前後上諫二百餘事——李世民對他又敬又怕——曾說「人言魏徵舉止疏慢，我視之更覺嫵媚」 |
| **貞觀十一年的轉折點** | 到貞觀十一年——唐朝已經穩定繁榮——李世民也開始出現驕傲自滿的傾向——逐漸放鬆了對自己的要求——開始大修宮殿、追求享樂。魏徵觀察到這種變化——在數年內多次上疏規勸——〈諫太宗十思疏〉就是其中最著名的一篇 |

> **軍師按**：〈諫太宗十思疏〉的寫作時機——不是唐朝最困難的時候——而是唐朝最風光的時候。魏徵的偉大——不在於他在亂世中保持清醒——而在於他在盛世中——所有人都覺得一切都很好的時候——站出來說——「陛下——你需要注意了。」

### 2.2 以隋為鑑——「載舟覆舟」的真實歷史教訓

魏徵在〈諫太宗十思疏〉中反覆強調「載舟覆舟」的道理——這不是抽象的說教——而是李世民親眼所見的歷史教訓。隋煬帝楊廣——即位之初國力強盛——卻因好大喜功、窮兵黷武、不聽勸諫——最終身死國滅——隋朝從統一到滅亡——僅三十八年。

| 面向 | 隋煬帝（反面教材） | 唐太宗（應有的方向） |
|------|-------------------|---------------------|
| **對待諫言** | 殺諫官——如殺張衡、殺薛道衡——「我性不喜人諫」 | 重諫官——魏徵死後說「以銅為鏡，可以正衣冠——以史為鏡，可以知興替——以人為鏡，可以明得失——朕常保此三鏡——以防己過——今魏徵殂逝——遂亡一鏡矣」 |
| **治理風格** | 驕奢淫逸——大興土木——三徵高麗——耗盡國力 | 節制——與民休息——克制個人慾望——謹慎處理邊境戰爭 |
| **最終結局** | 被縊殺於江都——隋朝滅亡 | 開創貞觀之治——為唐朝三百年盛世奠基 |

> **軍師按**：魏徵的論證之所以有力——不是因為他的道理多麼深奧——而是因為他說的事情——就發生在李世民這一代人的記憶裡。隋朝是什麼時候滅亡的？不過是幾十年前的事。隋煬帝的瘋狂——李世民親眼見過——甚至親身參與過推翻它。魏徵說「載舟覆舟」——李世民知道——這是真的。

## 三、文本結構——「十思」不是十條規矩——而是一種態度

### 3.1 全文三段式結構

〈諫太宗十思疏〉全文約 600 餘字——以駢散結合的唐文風格寫成——結構分明：

| 層次 | 核心內容 | 情感基調 |
|------|---------|---------|
| **第一段：提出問題** | 開篇即提出核心命題——「臣聞求木之長者，必固其根本——欲流之遠者，必浚其泉源——思國之安者，必積其德義」——用樹根和水源做比喻——說明治國的根本在於「德義」。然後指出——當今的問題是——「不念居安思危，戒奢以儉」——「德不處其厚，情不勝其欲」——這就是在批評李世民的驕奢傾向 | 懇切、有力——比喻生動——開門見山 |
| **第二段：歷史教訓** | 歷數歷代興亡——「凡百元首，承天景命——善始者實繁——克終者蓋寡」——為什麼？因為「取之易而守之難」——創業的時候——大家都很努力——但成功之後——就開始鬆懈了。然後引出「載舟覆舟」的經典論斷——這是全文的核心論證——「怨不在大，可畏惟人——載舟覆舟，所宜深慎」 | 沉痛、嚴肅——用歷史的重量壓下來 |
| **第三段：提出十思** | 在論證了「為什麼必須警惕」之後——魏徵提出了「十思」——十個提醒自己克制的事項——最後以「文武爭馳，君臣無事——可以盡豫游之樂——可以養松喬之壽」收尾——只要做到這十思——皇帝不但可以安享太平——還可以長壽成仙 | 具體、實用——每一思都是一個具體的行動指南——結尾帶有激勵和許諾 |

### 3.2 「十思」的具體內容

| 編號 | 原文關鍵句 | 白話解讀 |
|------|-----------|---------|
| 第一思 | 見可欲——則思知足以自戒 | 看到想要的東西——想想「夠了没有」 |
| 第二思 | 將有作——則思知止以安人 | 想要大興土木——想想「該停了」——别擾民 |
| 第三思 | 念高危——則思謙沖而自牧 | 想到位高權重——想想「放低姿態」 |
| 第四思 | 懼滿溢——則思江海下百川 | 擔心驕傲自滿——想想「大海在百川之下」 |
| 第五思 | 樂盤遊——則思三驅以為度 | 喜歡遊獵玩樂——想想「有限度」 |
| 第六思 | 憂懈怠——則思慎始而敬終 | 擔心自己懶散——想想「善始善終」 |
| 第七思 | 慮壅蔽——則思虛心以納下 | 怕被蒙蔽——想想「虛心聽取意見」 |
| 第八思 | 想讒邪——則思正身以黜惡 | 怕小人在身邊——想想「先端正自己」 |
| 第九思 | 恩所加——則思無因喜以謬賞 | 獎賞的時候——想想「别因為高興就亂賞」 |
| 第十思 | 罰所及——則思無以怒而濫刑 | 處罰的時候——想想「别因為生氣就亂罰」 |

> **軍師按**：仔細看這「十思」——它沒有一條是「怎麼管理天下」——全部都是「怎麼管理自己」。魏徵的邏輯是——只要你管好了自己——天下自然就管好了。十思的核心——不是知識——而是態度——不是「你不知道什麼是對的」——而是「你明知道什麼是對的——但你做不到——因為你不願意約束自己」。

## 四、關鍵人物

### 4.1 魏徵——貞觀第一諫臣

| 項目 | 內容 |
|------|------|
| **身份** | 唐初名相、諫官領袖。字玄成，巨鹿下曲陽（今河北晉州）人。生於西元 580 年——卒於西元 643 年 |
| **生平** | 早年貧困——出家為道士。隋末天下大亂——先後投奔李密、竇建德——最後成為太子李建成的幕僚。玄武門之變後——被李世民重用——先後任諫議大夫、尚書右丞、秘書監、侍中等職——封鄭國公 |
| **諫臣風格** | 以「直言敢諫」著稱——不迎合、不逢迎——前後上諫二百餘事——當面批評李世民——常常讓皇帝下不了台。李世民曾說「魏徵每廷爭辱我——使我常不自得」——但又說「人言魏徵舉止疏慢——我視之更覺嫵媚」——又恨又愛——是李世民對魏徵最真實的態度 |
| **歷史地位** | 在中國政治史上——魏徵是「諫臣」的最高典範——他的名字——就是「敢說真話」的代名詞。他死後——李世民悲痛地說出了那段千古名言——「以銅為鏡——可以正衣冠——以史為鏡——可以知興替——以人為鏡——可以明得失」 |

**軍師評**：魏徵的特殊——不在於他敢說——而在於他「說的時候——自己沒有任何私心」。他不是在挑戰李世民的權威——他是在保護李世民的江山——以及李世民本人。因為他沒有私心——所以李世民即使生氣——也無法反駁他。這是「諫」的最高境界——不是因為你勇敢——而是因為你正確。

### 4.2 唐太宗李世民——兼聽則明的帝王

| 項目 | 內容 |
|------|------|
| **身份** | 唐朝第二位皇帝。生於西元 598 年——卒於西元 649 年。在位二十三年——年號貞觀 |
| **與魏徵的關係** | 中國歷史上最著名的君臣關係之一——不是單純的「君明臣直」——而是一種充滿張力的關係——李世民多次被魏徵氣得想殺他——但又離不開他——因為他知道——魏徵是對的 |
| **對諫言的態度** | 李世民是中國歷史上極少數——真正願意聽取不同意見的皇帝——他說「兼聽則明，偏信則闇」——這是他的信念——也是他能夠成就貞觀之治的原因 |

**軍師評**：李世民和魏徵——是「正確的君臣關係」的完美範例——不是因為李世民脾氣好——而是因為他知道——權力最可怕的陷阱——不是敵人的攻擊——而是沒有人敢對你說真話。魏徵是他的「鏡子」——鏡子不會說好聽的話——但鏡子永遠說真話——這就是李世民需要魏徵的原因——也是〈諫太宗十思疏〉之所以能夠被寫出來、被接受、被流傳的原因。

## 五、軍師核心分析——「十思」的本質：領導者的自我管理

### 5.1 「思」——為什麼是「思考」而不是「規矩」？

魏徵提出的「十思」——每一條都以「思」開頭——「見可欲則思知足以自戒」——「將有作則思知止以安人」——這不是巧合。**「思」的對立面——不是「不知」——而是「不思」——是麻木——是理所當然——是「我已經成功了——我可以放鬆了」——這才是魏徵最想提醒李世民的。**

魏徵的核心論證邏輯：

| 層次 | 內容 |
|------|------|
| **問題** | 善始者實繁——克終者蓋寡——每個皇帝開頭都很好——但很少有人能善終 |
| **原因** | 不是因為他們變笨了——而是因為他們「不思」了——他們覺得一切都理所當然了——就停止了自我反省 |
| **解藥** | 十思——每一次面對誘惑、權力、享樂、懈怠的時刻——都停下來——想一想——「我這樣做對嗎？」 |

> **軍師按**：〈諫太宗十思疏〉最深刻的地方——不在於它說了什麼——而在於它要求你做什麼——它要求你——在每一次可能犯錯的時刻——停下來——想一想。這不是一份「禁止清單」——而是一個「提醒自己自律的方法」。魏徵不是在告訴李世民「什麼不能做」——他是在教李世民——「在每做一件事之前——先問自己一句——我是不是又被權力慣壞了？」

### 5.2 〈諫太宗十思疏〉的歷史影響

| 面向 | 影響 |
|------|------|
| **對李世民** | 李世民讀後——不但沒有生氣——反而深以為然——將疏文置於案頭——時時提醒自己。據《貞觀政要》記載——此後李世民在生活上確實有所節制——雖然不是完美做到——但「十思」成為他晚年治國的重要參考 |
| **對唐代政治** | 「十思」理念影響了唐代後續帝王——特別是「以人為鑑」的傳統——成為唐代諫官制度的文化基礎 |
| **對後世** | 〈諫太宗十思疏〉成為中國政治文獻中最著名的「勸諫文」之一——與諸葛亮的〈出師表〉、李密的〈陳情表〉並列——被稱為「臣子三表」——每一篇都是一種臣子對君主說話的典範——諸葛亮是「忠」——李密是「孝」——魏徵是「直」 |

### 5.3 一句話總結

> **〈諫太宗十思疏〉不是十條規矩——而是一個清醒的提醒——當一個領導者開始覺得一切理所當然的時候——就是他最危險的時候。魏徵用「樹根和水源」的比喻告訴李世民——治國的根本不在於權力有多大——而在於德義有多厚。用「載舟覆舟」的歷史告訴李世民——你以為船已經穩了——但水從來沒有給過任何船永遠不沉的承諾。用「十思」的方法告訴李世民——每一次面對誘惑和懈怠——停下來——想一想——你是不是又忘了——你曾經也是一個——戰戰兢兢——如履薄冰的人。因為——一個成功的領導者——最需要的不是更多的權力——而是——對權力的恐懼。**——**而〈諫太宗十思疏〉——就是魏徵寫給李世民——也寫給所有掌權者——的一封——「別忘了你曾經害怕過」的信。**

## 六、關聯篇目與延伸閱讀

### 6.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷六·〈前出師表〉（諸葛亮） | 諸葛亮「鞠躬盡瘁」與魏徵「直言敢諫」——都是臣子對君主的真誠告誡——一忠一直——都是中國政治文書的巔峰 |
| 卷七·〈陳情表〉（李密） | 李密用孝拒絕皇帝——魏徵用直諫告誡皇帝——一個是「我不來」——一個是「你要改」——兩種不同的臣子姿態 |
| 卷九·〈送李愿歸盤谷序〉（韓愈） | 韓愈寫歸隱者的自由——魏徵寫在位者的責任——從不同的方向——討論了權力與個人的關係 |

### 6.2 經典延伸

- 《舊唐書·魏徵傳》（劉昫等）——正史中關於魏徵最完整的傳記——記載了他與李世民之間多次著名的諫爭——是理解〈諫太宗十思疏〉的第一手文獻
- 《貞觀政要·君道》（吳兢）——唐代史學家吳兢所編——系統記錄了貞觀年間的君臣對話——其中記載了〈諫太宗十思疏〉的完整內容和背景
- 《資治通鑑·唐紀》（司馬光）——司馬光從「以史為鑑」的角度——記錄了貞觀年間的政治——特別是魏徵諫言的歷史影響
