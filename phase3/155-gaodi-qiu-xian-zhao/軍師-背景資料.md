# 〈高帝求賢詔〉背景資料

> **軍師分析 · 古文觀止卷六 · 西漢文 · 劉邦**

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《漢書·高帝紀》——漢高祖劉邦晚年頒布的一道求賢令 |
| **選集** | 《古文觀止》卷六·漢文類 |
| **體裁** | 詔令文——皇帝頒布的命令，但口吻更像創業者的誠摯邀請 |
| **時代** | 西漢初年，漢高祖十一年（約西元前 196 年）——劉邦稱帝後第十一年，離去世僅一年 |
| **核心命題** | 開國皇帝的用人格局——打天下靠人才，守天下更靠人才——「安得猛士兮守四方」的具體實踐 |
| **關鍵人物** | 漢高祖劉邦（發布詔書者）、周文王／齊桓公（詔書中引用的榜樣）、漢代各級官員（執行人） |
| **文本重點** | 這是一道非常簡短的詔書，全文不足四百字，卻展現了劉邦晚年對人才的深刻渴求。開篇以周室大封與齊桓之霸為例，說明「得人者興」的道理。核心在於兩點：一是承認「賢士大夫」在國家治理中的關鍵作用；二是以皇帝的尊嚴承諾——「賢士大夫有肯從我游者，吾能尊顯之」——這不是命令，而是邀請 |
| **思想譜系** | 延續先秦「尚賢」思想——從孔子「舉賢才」、墨子「尚賢」到劉邦這道詔書，用人唯賢是中國政治思想的傳統。但劉邦的特殊之處在於：他不是以道德理想發表議論，而是以一個開國皇帝的實際經驗告訴後人——我之所以能得天下，就是因為我會用人 |
| **一句話總結** | **一個流氓出身的皇帝，在晚年最擔心的事情不是江山不穩，而是沒有人才可用——這是創業者與世襲者最大的區別。** |

## 二、時代背景——漢高祖晚年的憂慮

### 2.1 劉邦晚年的處境——天下已定，但天下並不安

| 面向 | 說明 |
|------|------|
| **開國形勢** | 西元前 202 年，劉邦打敗項羽，即皇帝位，建立漢朝。但「漢承秦弊」——秦末大亂加上楚漢相爭，天下殘破——「民失作業，而大饑饉」——人口銳減，經濟崩潰，社會秩序蕩然無存 |
| **外部威脅** | 匈奴在北方崛起——冒頓單于統一草原，屢次犯邊。西元前 200 年，劉邦親征匈奴，在平城白登山被圍七天七夜——險些全軍覆沒。此後漢朝被迫採取「和親」政策——將宗室女嫁給單于——以換取邊境安寧。劉邦對這個屈辱的妥協耿耿於懷 |
| **內部叛亂** | 漢初的異姓諸侯王相繼反叛——韓信、彭越、英布……劉邦在晚年幾乎年年征討自己親手分封的功臣。西元前 196 年——就是頒布這道求賢詔的同一年——淮南王英布反，劉邦親自帶兵征討，中箭受傷，一年後便因箭傷去世 |
| **繼承人問題** | 太子劉盈（後來的漢惠帝）性格柔弱，劉邦曾數度想改立戚夫人之子趙王如意——因而引發了朝中元老（特別是呂后）與劉邦之間的張力。劉邦去世後，呂后專權——幾乎毀了劉家的江山——反而印證了劉邦晚年對「人才不足」的憂慮：不是沒有官員，而是沒有能夠輔佐幼主、穩定大局的賢士大夫 |

> **軍師按**：理解〈高帝求賢詔〉的關鍵，在於理解劉邦寫這道詔書時的處境——他已經老了（時年六十二，在漢代已算高齡），身體也不好（征英布時中箭），天下看似統一，實則內外交困。他自己是從亂世中殺出來的開國皇帝，知道江山是怎麼打下來的——但他也知道，他死後，這片江山將交給一個軟弱的太子。這道求賢詔，與其說是在為自己找人才——不如說是在為兒子劉盈找人才。一個即將離世的父親，用一紙詔書為兒子安排後路——這就是〈高帝求賢詔〉最深層的情感底色。

### 2.2 「安得猛士兮守四方」——〈大風歌〉中的同一心聲

劉邦晚年還有一首著名的作品——〈大風歌〉：

> 大風起兮雲飛揚，威加海內兮歸故鄉，安得猛士兮守四方！

劉邦平定英布之亂後，路過沛縣（他的家鄉），與故鄉父老飲酒，在酒酣之際擊筑高歌——唱出了這三句詩。這首詩與〈高帝求賢詔〉幾乎寫於同一時期（前 196 年至前 195 年之間）：

| 面向 | 〈大風歌〉 | 〈高帝求賢詔〉 |
|------|-----------|--------------|
| **形式** | 即興詩歌——情感的直接流露 | 正式詔書——理性的政策宣示 |
| **情感** | 悲涼——「安得猛士兮守四方」——帶著無可奈何的感嘆 | 懇切——「賢士大夫有肯從我游者，吾能尊顯之」——帶著務實的邀約 |
| **對象** | 自己——自問自答 | 天下——向所有人喊話 |
| **共同主題** | 打天下的人已經老了——守天下的人在哪裡？ |

> **軍師按**：〈大風歌〉和〈高帝求賢詔〉是同一顆心的兩種表達。前者是酒後的悲歌——劉邦在故鄉回憶起自己的輝煌，但面對未來充滿了不確定；後者是清醒的詔書——他用皇帝的權力為這個不確定尋找答案。兩者加在一起，才構成劉邦晚年完整的精神圖譜——他既是一個英雄（威加海內），也是一個父親（為兒子憂慮）。

## 三、原文結構與核心論點

### 3.1 〈高帝求賢詔〉的結構

這道詔書雖然簡短，但結構完整、邏輯清晰：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：引古立論** | 蓋聞王者莫高於周文，伯者莫高於齊桓，皆待賢人而成名。今天下賢者智能，豈特古之人乎？患在人主不交故也，士奚由進！ | 開篇以周文王、齊桓公為例——「得賢者興」——然後一個反問：難道今天的賢人不如古代嗎？不——問題在於君主不願意去結交他們。劉邦把責任攬到自己（以及所有君主）身上——不是賢人不夠好，是君主不夠主動 |
| **第二層：自我反思** | 今吾以天之靈、賢士大夫定有天下，以為一家。欲其長久，世世奉宗廟亡絕也。賢人已與我共平之矣，而不與吾共安利之，可乎？ | 劉邦坦率承認——我的天下是靠「天之靈」和「賢士大夫」打下的。現在你們幫我打完了天下，卻不和我一起治理——這合理嗎？這裡的「可乎」是一個帶有真誠愧疚的追問——不是責備，而是自省 |
| **第三層：具體措施** | 賢士大夫有肯從我游者，吾能尊顯之。布告天下，使明知朕意。御史大夫昌下相國，相國酇侯下諸侯王，御史中執法下郡守。其有意稱明德者，必身勸，為之駕，遣詣相國府，署行、義、年。有而弗言，覺，免。年老癃病，勿遣。 | 這是詔書的核心——具體的執行方案。各級官員必須主動尋訪賢才——「身勸」（親自去勸說）、「為之駕」（為他準備車馬）、「遣詣相國府」（送到中央政府）。不推薦的官員——被發現後——免職。這不是道德呼籲——這是帶有懲罰機制的行政命令 |

> **軍師按**：這道詔書最打動人的地方，不是它的文采——而是它的真誠。劉邦沒有說「聯要徵召賢才」——他說的是「肯從我游者」——「游」這個字，不是君臣關係，而是夥伴關係。他說「吾能尊顯之」——不是用皇帝的權威來命令——而是用一個創業者的承諾來邀請。你來跟我一起幹——我一定讓你顯貴。這口吻，像極了一個創業團隊的 CEO 在招募合夥人——而不是一個皇帝在頒布聖旨。

### 3.2 核心段落節錄

> 蓋聞王者莫高於周文，伯者莫高於齊桓，皆待賢人而成名。今天下賢者智能，豈特古之人乎？患在人主不交故也，士奚由進！

——劉邦開篇就把問題點破：不是時代沒有賢人，而是君主不願意主動去找。這句話是對所有統治者的當頭棒喝——「患在人主不交」——問題在你，不在別人。

> 今吾以天之靈、賢士大夫定有天下，以為一家。欲其長久，世世奉宗廟亡絕也。賢人已與我共平之矣，而不與吾共安利之，可乎？

——劉邦的坦白令人動容。他承認：天下不是我一個人打下來的——是賢士大夫幫我打下來的。現在天下太平了——我卻不讓你們和我一起享受這太平——這合理嗎？一個皇帝能夠說出這樣的話——說明他真正理解了「分享」的意義。

> 賢士大夫有肯從我游者，吾能尊顯之。

——全詔書中最經典的一句話。劉邦不說「召」——他說「游」；不說「官」——他說「尊顯」。這是一個創業者的語言——不是皇帝的語言。你們來跟我做朋友——我一定讓他們風光。

> ——《漢書·高帝紀》

## 四、關鍵人物

### 4.1 漢高祖劉邦——發布詔書的皇帝

| 項目 | 內容 |
|------|------|
| **身份** | 西漢開國皇帝（在位：西元前 202–195 年）。字季，沛縣（今江蘇沛縣）人。中國歷史上第一個平民出身的皇帝 |
| **生平核心** | 劉邦出身布衣——年少時不事生產，好酒色，曾為亭長（秦朝最基層的小吏）。秦末大亂中，他起兵反秦——先投項梁，後與項羽爭天下——最終在垓下之戰中打敗項羽，建立漢朝。劉邦的一生，是「得人者得天下」的最好案例——他的軍事能力遠不如項羽，但他用了韓信、張良、蕭何、陳平——這四位中國歷史上最頂尖的人才——最終打敗了不可一世的項羽 |
| **這道詔書中的劉邦** | 65 歲的劉邦——征討英布時中箭，傷勢沉重——知道自己時日無多。他在這道詔書中展現的不是皇帝的威嚴——而是一個即將離世的創業者的憂慮：我打下了天下，但我死之後，誰來守天下？這道詔書是他對「接班人問題」的回答——為兒子劉盈儲備人才 |
| **歷史定位** | 劉邦是中國歷史上最偉大的開國皇帝之一——他的偉大不在於他多麼能征善戰——而在於他「知人善任」——他知道自己的能力邊界——知道什麼時候該用什麼人。他曾說：「夫運籌策帷帳之中，決勝於千里之外，吾不如子房。鎮國家，撫百姓，給饋饟，不絕糧道，吾不如蕭何。連百萬之軍，戰必勝，攻必取，吾不如韓信。此三者，皆人傑也，吾能用之，此吾所以取天下也。」——這段話與〈高帝求賢詔〉的精神完全一致：劉邦的成功之道，就是「用人之道」 |

**軍師評**：劉邦是一個非常特別的皇帝——他不裝。他不像後世的皇帝那樣把自己包裝成「聖明天子」——他就是一個從社會底層爬上來的人——粗魯、好酒色、自私——但他有一個極大的優點：他服氣。遇到比自己強的人——比如韓信——他可以忍。他願意承認自己的不足——「吾不如子房」「吾不如蕭何」「吾不如韓信」——這三個「不如」——是中國歷史上最了不起的自我認知。〈高帝求賢詔〉之所以讀起來不像皇帝的聖旨——就是因為說這話的人本來就不像一個皇帝。

### 4.2 周文王與齊桓公——詔書中的榜樣

| 項目 | 內容 |
|------|------|
| **周文王** | 周朝的奠基者——「王者莫高於周文」——劉邦把周文王作為「王」的最高榜樣。周文王以仁德聞名，廣納賢才——姜太公就是在渭水邊被周文王請出來的。劉邦引用周文王，意在說明：即使是最高明的王，也需要賢人輔佐 |
| **齊桓公** | 春秋五霸之首——「伯者莫高於齊桓」——劉邦把齊桓公作為「霸」的最高榜樣。齊桓公依靠管仲，九合諸侯、一匡天下。劉邦引用齊桓公，意在說明：即使是稱霸天下的人，也需要賢人輔佐 |
| **引用意義** | 劉邦用這兩個歷史典範告訴天下人：不論你是想做王（統一天下）還是做霸（稱雄一方）——都需要賢人。我劉邦今天——既不是要做王也不是要做霸——我只想讓這個國家「長久，世世奉宗廟亡絕」——所以更需要賢人 |

**軍師評**：劉邦引用周文王和齊桓公，有一個微妙之處——這兩位都是「得賢而興」的典範，但也都離劉邦自己的時代很遠。劉邦在暗示：我的時代——也需要這樣的賢人。他把自己放在周文王和齊桓公的位置上——不是驕傲——而是謙虛：我需要你們，就像他們需要姜太公和管仲一樣。

## 五、軍師總論——創業者的遺囑

### 5.1 〈高帝求賢詔〉的獨特之處——「皇帝不像皇帝」

〈高帝求賢詔〉最令人印象深刻的，是它的口吻——它完全不像一道詔書：

| 面向 | 傳統詔書 | 〈高帝求賢詔〉 |
|------|---------|--------------|
| **口吻** | 命令式——「聯令」「欽此」 | 邀請式——「肯從我游者」「吾能尊顯之」 |
| **立場** | 高高在上——皇帝對臣民的訓誡 | 平起平坐——創業者找合夥人 |
| **自稱** | 「聯」——強調權威 | 「吾」——拉近距離 |
| **態度** | 你必須服從——否則嚴懲 | 你來跟我一起幹——我讓你風光 |
| **情感底色** | 威嚴、不容置疑 | 懇切、甚至有些悲涼 |

劉邦的詔書之所以有這樣的特質——因為他從未真正忘記自己「布衣」的身份。他不是在皇宮中長大的皇帝——他是在沛縣街頭長大的流氓。他不太會用「聯」這個字來說話——他更習慣用「吾」——用老百姓的語言。這不是修辭技巧——這是本色。

### 5.2 詔書背後的政治智慧——選拔機制的設計

〈高帝求賢詔〉不只是道德呼籲——它有一套完整的執行機制：

1. **分層負責**：詔書明確了各級官員的職責——御史大夫下相國、相國下諸侯王、御史中執法下郡守——層層傳達、層層負責
2. **主動推薦**：「必身勸，為之駕」——官員必須親自去勸說賢才，為他們準備車馬——不是等他們自己送上門來
3. **建檔管理**：「署行、義、年」——記錄賢才的行為、儀表、年齡——建立人才資料庫
4. **連坐懲罰**：「有而弗言，覺，免」——如果轄區內有賢才而你沒有推薦——被發現後——直接免職。這是帶有強制力的制度

這套制度設計顯示了劉邦作為創業者的務實——他不只是「呼籲」——他建立了一套讓人才可以被發現、被輸送、被運用的機制。

### 5.3 與〈大風歌〉的互文——同一首歌的兩種唱法

| 面向 | 〈大風歌〉 | 〈高帝求賢詔〉 |
|------|-----------|--------------|
| **創作時間** | 前 195 年——征英布歸途 | 前 196 年——征英布前夕 |
| **形式** | 三句即興詩——隨口唱出 | 正式公文——層層下達 |
| **情感** | 悲壯——「安得猛士兮」——無可奈何 | 務實——「吾能尊顯之」——給出方案 |
| **共同主題** | 我已經老了——我可以把江山交給你們——但你們在哪裡？ |

> **軍師按**：如果我們把〈大風歌〉和〈高帝求賢詔〉放在一起讀——就會看到一個完整的劉邦。〈大風歌〉是他的心——那三句詩裡有他對故鄉的深情、對自己一生功業的自豪、對未來的憂慮。〈高帝求賢詔〉是他的行動——他不是只會感嘆——他拿出了具體的方案。他對天下人說：我需要你們——你們來——我給你們尊貴。這是創業者特有的思維方式——有感嘆——但不只是感嘆——感嘆完了就付諸行動。

### 5.4 對後世的影響

| 時期 | 影響 |
|------|------|
| **漢代** | 這道詔書確立了漢代「察舉制」的雛形——地方官員有責任發現和推薦人才。後來漢武帝推行「察舉孝廉」——正是從劉邦這道詔書發展而來 |
| **後世** | 劉邦「賢士大夫有肯從我游者，吾能尊顯之」——成為後世眾多求賢詔的範本。曹操的〈求賢令〉——「唯才是舉」——直接繼承了劉邦的用人精神。唐太宗的「天下英雄，入吾彀中」——也是同一種心態 |
| **現代啟示** | 這道詔書對現代管理者仍然有意義：一個好的領導者不是什麼事都自己做——而是知道如何發現和用好人才。劉邦的三個「不如」——「吾不如子房」「吾不如蕭何」「吾不如韓信」——可以作為每一個管理者的座右銘 |

## 六、一句話總結（再錄）

> **〈高帝求賢詔〉是一個流氓出身的開國皇帝，在生命最後一年給兒子留下的政治遺囑。他清楚地知道：江山不是我一個人打下來的——是賢士大夫幫我打下來的。如果沒有人才來守——這片江山遲早會易手。所以他放下了皇帝的架子——用一個創業者的真誠——對天下人說：「賢士大夫有肯從我游者，吾能尊顯之。」這道詔書之所以動人，不是因為它寫得好——而是因為說這話的人，用他的一生證明了這句話的真實性。他用三個「不如」解釋了他的成功——用這道詔書宣告了他對未來的憂慮——用〈大風歌〉中的那句「安得猛士兮守四方」——為這兩千年的創業者與守成者之間，劃下了一道永遠無法逾越的鴻溝：創業者擔心的是沒有人才可用——守成者擔心的是權力不穩。而〈高帝求賢詔〉，就是這道鴻溝最好的證明。**

## 七、關聯篇目與延伸閱讀

### 7.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷六·〈求賢詔〉（漢武帝） | 漢武帝同樣頒布過求賢詔——但兩者的風格截然不同。劉邦的口吻像創業者招募夥伴——漢武帝的口吻已經是帝國皇帝在徵召官員。從劉邦到漢武帝——可以看到「帝國」從創業到制度化過程中的語言變化 |
| 卷六·〈報任安書〉（司馬遷） | 〈報任安書〉中司馬遷提到「推賢進士」——這是漢代士大夫對人才選拔的核心責任。劉邦的這道詔書——正是「推賢進士」制度化的起點 |
| 卷四·〈馮諼客孟嘗君〉（《戰國策》） | 孟嘗君養士——與劉邦求賢——都是「人才為本」的展現。但孟嘗君養的是「門客」——劉邦求的是「守天下」的賢大夫——格局不同 |
| 卷四·〈觸讋說趙太后〉（《戰國策》） | 觸讋的核心論點——「位尊而無功，奉厚而無勞」——與劉邦「賢人已與我共平之矣，而不與吾共安利之，可乎？」——關注的是同一個問題：有功的人應該得到回報 |

### 7.2 經典延伸

- 《史記·高祖本紀》（司馬遷）——劉邦的全傳——是理解這道詔書最重要的背景。從劉邦「不事生產」的青年到「吾不如」的晚年——可以看到一個人如何在歲月中成長
- 《史記·蕭相國世家》（司馬遷）——劉邦最重要的合夥人——蕭何。劉邦用蕭何、疑蕭何、最終與蕭何和解——是理解漢初君臣關係的關鍵
- 《史記·淮陰侯列傳》（司馬遷）——劉邦用韓信、疑韓信、最終殺韓信——是理解漢初「分封功臣」制度的悲劇。劉邦求賢是認真的——但他也在晚年親手殺了不少賢人——這是創業者作為政治人物的另一面
- 《漢書·高帝紀》（班固）——〈高帝求賢詔〉的原始出處——班固對劉邦的評價比司馬遷更加正面——但〈高帝紀〉中保留了這道詔書的完整文本
- 曹操〈求賢令〉（三國）——曹操「唯才是舉」的口號——是劉邦求賢精神在亂世中的延續。曹操與劉邦一樣——都是從亂世中打出來的創業者——他們對人才的理解有驚人的相似
- 唐太宗與魏徵的故事（《貞觀政要》）——唐太宗說：「以銅為鏡，可以正衣冠；以古為鏡，可以知興替；以人為鏡，可以明得失。」——這與劉邦「吾不如」的精神一脈相承

### 7.3 現代延伸

- 彼得·德魯克《管理的實踐》——德魯克說：「管理者的任務不是管理人才——而是發現人才、用好人才。」——這與劉邦「吾不如」的精神完全一致
- 吉姆·柯林斯《從優秀到卓越》——柯林斯提出「先把對的人請上車，再決定車往哪裡開」——與劉邦「先找賢人，再打天下」的思路不謀而合
- 李開復《做最好的自己》——李開復的「用人之道」——強調「知人善任」——與劉邦的領導哲學有相通之處
