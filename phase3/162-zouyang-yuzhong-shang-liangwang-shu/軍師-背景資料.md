# 〈獄中上梁王書〉背景資料

> **軍師分析 · 古文觀止卷六 · 西漢文 · 鄒陽**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《漢書·鄒陽傳》——鄒陽在梁孝王獄中所寫的書信 |
| **選集** | 《古文觀止》卷六·西漢文類——卷六中的一篇著名書信體論辯文 |
| **體裁** | 書信體／辯難文——以歷史典故為論證核心的自我辯護書 |
| **時代** | 西漢景帝時期（約西元前 154–144 年間），梁孝王劉武入朝後期 |
| **核心命題** | 忠而被謗，信而見疑——自古忠臣往往因小人的讒言而被害——「白頭如新，傾蓋如故」 |
| **關鍵人物** | 鄒陽（齊國遊士，被讒下獄後寫此信）、梁孝王劉武（西漢諸侯王，景帝同母弟）、羊勝、公孫詭（梁王身邊的讒臣） |
| **文本重點** | 鄒陽在梁孝王門下為客，被羊勝、公孫詭等人讒害，下獄將被處死。他在獄中寫了這封信給梁王——不是求饒，而是一連串的歷史典故，論證「忠臣往往被讒言所害」。他列舉了荊軻、卞和、百里奚、司馬喜、范雎等無數例子，證明自古以來忠誠的人總是因為小人的讒言而受害。這篇「以典為證」的雄文，最終打動了梁孝王，使他得以獲釋 |
| **思想譜系** | 出自《漢書·鄒陽傳》。鄒陽是西漢初期著名的縱橫策士，以善辯著稱。他的這篇文章代表了一種特殊的辯護策略——不為自己辯解，而是用歷史的普遍規律來證明自己的清白：「不是我背叛了你，而是自古以來忠臣都是這樣被冤枉的。」這種「以古證今」的論證方式，直接影響了後世無數的獄中書信與辯難文章 |
| **一句話總結** | **〈獄中上梁王書〉是一篇在死囚牢中寫出的「歷史證明題」——鄒陽不說「我冤枉」，而是把整個中國歷史搬出來，證明「自古以來的忠臣都和我一樣冤枉」。他不是在求饒，他是在上一堂歷史課——告訴梁王：如果你殺了我，你就是在重複歷史上的所有悲劇。** |

## 二、時代背景——梁孝王與西漢諸侯王政治

### 2.1 梁孝王劉武——太后寵兒，皇位候選人

梁孝王劉武是漢文帝的次子、漢景帝的同母弟（竇太后所生）。在漢朝諸侯王中，他是權力最大、地位最特殊的一個：

| 面向 | 說明 |
|------|------|
| **地位特殊** | 梁孝王是竇太后最寵愛的兒子——太后曾多次希望景帝傳位給梁孝王。景帝曾在酒後對梁孝王說：「千秋萬歲後傳於王。」雖是戲言，但梁孝王從此懷有覬覦皇位之心 |
| **封國廣大** | 梁國地處中原要衝——「居天下膏腴之地」——從今天的河南開封到山東一帶，是西漢最大的封國之一。梁國「府庫金錢且百巨萬，珠玉寶器多於京師」——比朝廷還有錢 |
| **招致賓客** | 梁孝王大興宮室，招攬天下名士——鄒陽、枚乘、嚴忌、司馬相如等人皆為其門客。梁國成為西漢初期文士匯聚的文化中心——「梁園」在文學史上成為文人雅集的代名詞 |
| **權力膨脹** | 梁孝王入朝時，「入則與上同輦，出則同車」——與景帝平起平坐。他的梁國「多作兵器、弩弓、矛戈」——擁有超過諸侯王規格的軍事力量。這使他在景帝時期成為一個極具政治敏感性的存在 |

> **軍師按**：鄒陽就是在這樣一個「潛在皇位繼承人」的宮廷中做門客。梁國既是文人的天堂——金錢、榮譽、禮遇——也是政治鬥爭的溫床——因為梁孝王的特殊地位，他身邊的每一個人都想爭寵奪權。鄒陽被讒下獄，正是這種複雜政治局面的縮影。

### 2.2 羊勝、公孫詭——讒言的來源

羊勝和公孫詭是梁孝王身邊最受寵信的謀士。史載兩人「多奇邪計」——善於出謀劃策。他們極力慫恿梁孝王爭取皇位繼承權——並在梁孝王面前不斷讒害其他門客，以鞏固自己的地位：

| 面向 | 說明 |
|------|------|
| **羊勝** | 梁孝王身邊的謀士，以善於陰謀著稱。他與公孫詭聯手排擠鄒陽，使鄒陽下獄。後來梁孝王在謀求皇位繼承的過程中——暗殺朝廷大臣袁盎——的主謀之一就是羊勝 |
| **公孫詭** | 同樣是梁孝王的謀士，以「多奇計」聞名。與羊勝同為梁孝王的核心幕僚，是鄒陽的直接競爭對手 |
| **讒害手法** | 羊勝、公孫詭向梁孝王進讒——說鄒陽是「齊人」（來自齊國的外鄉人），不可能忠於梁國——挑撥梁孝王對鄒陽的信任——說鄒陽在外宣揚梁國的機密——導致梁孝王大怒，將鄒陽下獄 |

> **軍師按**：這是中國政治史上最經典的讒言套路——「他是外地人，他不值得信任」。鄒陽的〈獄中上梁王書〉之所以偉大，正是因為他意識到：如果直接為自己辯解——說自己是忠誠的——根本沒有用。因為讒言的邏輯不是「你做了什麼」，而是「你是誰」。所以鄒陽選擇了另一條路——他用歷史告訴梁王：自古以來被讒言害死的忠臣，沒有一個不是忠誠的——你殺了我，你就會成為和那些昏君一樣的人。

### 2.3 「七國之亂」的餘波

鄒陽寫作這封信的時代背景，正值七國之亂（西元前 154 年）平定後的數年間。七國之亂是西漢諸侯王叛亂的最高潮——吳王劉濞聯合楚、趙、膠西、膠東、菑川、濟南六國起兵反漢，最後被周亞夫平定。此後——景帝對諸侯王的猜忌加深——諸侯王之間的政治鬥爭也更加激烈。梁孝王在七國之亂中是站在朝廷一邊的（梁國的都城睢陽是抵抗吳楚聯軍的關鍵防線），但他戰後的膨脹——反而引起了景帝的警惕。鄒陽就是在這個風口浪尖上被讒下獄的——他面對的不僅是梁孝王的怒氣，還有一個政治極度敏感的時代。

## 三、文本結構與核心論證

### 3.1 〈獄中上梁王書〉的結構

鄒陽的這封信是一篇層層遞進的歷史論證。他不用「我冤枉」這樣的直白表達——而是用歷史來「圍攻」梁孝王的判斷：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：開場——提出「忠而見疑」的普遍命題** | 鄒陽開篇就說：「臣聞忠無不報，信不見疑，臣常以為然，徒虛語耳。」——我曾經以為忠誠一定會有回報、誠信一定不會被懷疑——但現在我知道了，這只是空話。然後他提出全篇的核心論斷：「白頭如新，傾蓋如故。」——有些人認識一輩子，到白頭還像陌生人一樣互不信任；有些人在路上相遇，停車交談幾句，就像老朋友一樣推心置腹。為什麼？不是因為認識的時間長短——而是因為是否真正了解一個人的內心 | 一開場就把信從「個人求饒」的層次提升到「普遍人性」的層次。鄒陽告訴梁王：你之所以懷疑我——不是因為我做錯了什麼——而是因為你從來沒有真正了解我。而你不知道怎麼了解我——這不是你的錯——這是人性 |
| **第二層：用歷史證明「忠臣被讒」是普遍現象** | 這是最長、最重要的部分。鄒陽列舉了荊軻、卞和、百里奚、司馬喜、范雎等歷史人物——每一個人的故事都是一樣的：他們忠於自己的君主，但被讒言害得生不如死。荊軻為燕太子丹刺殺秦王——但在此之前，太子丹也懷疑過他。卞和獻和氏璧給楚王——但被當作騙子，先後被砍去了雙腳。百里奚在秦國幫助秦穆公稱霸——但在此之前，他在虞國不被重用，甚至被迫賣身為奴。司馬喜被挖去膝蓋骨——范雎被打斷了肋骨和牙齒——「這些人哪一個不是忠誠的？但他們的忠誠，並沒有保護他們不受讒言的傷害」 | 這是一個極其巧妙的論證策略——鄒陽不直接說「我是忠誠的」，而是說「歷史上有那麼多忠誠的人都被冤枉了——我只是這個長長的名單上最新的那個」。梁王無法反駁——因為他不能否認歷史。如果他堅持說鄒陽有罪——他就必須同時否認荊軻、卞和、百里奚、司馬喜、范雎——這是不可能的 |
| **第三層：分析「為什麼讒言如此有效？」** | 鄒陽在論證的過程中——穿插了對讒言心理學的分析。他指出：讒言之所以有效——不是因為說謊——而是因為「人主有私心」。君主總是對權力有天然的戒備心理——讒臣就是利用這種心理——把忠臣的正常行為解釋為「不忠」的證據。他說：「偏聽生姦，獨任成亂。」——只聽一面之詞就會產生奸邪，只信任一個人就會釀成禍亂 | 這一層是全文的「戰略升維」——從歷史舉例上升到理論分析。鄒陽告訴梁王：你不是第一個被讒言左右的君王——幾乎所有的昏君都是這樣被蒙蔽的。如果你不想重蹈覆轍——你就應該聽聽不同的聲音——尤其是那些被冤枉的人的聲音 |
| **第四層：尾聲——以「士為知己者死」收束** | 信的結尾，鄒陽回歸到一個古老的命題：「士為知己者用，女為說己者容。」——一個士人只為真正了解他的人效力，一個女子只為真正欣賞她的人妝扮。他告訴梁王：我之所以在梁國——是因為我認為你是我的「知己」——如果你殺了我——你不僅殺了一個人——你殺了一個把你當作知己的人 | 這是一個極為高明的結尾——鄒陽不是在恐嚇梁王——他是在懇求梁王給自己一個「被了解」的機會。同時——他也暗示梁王：如果你殺了我——你以後就不會有真正忠誠的人願意為你效力了——因為所有人都會看到你的「不能容人」 |

> **軍師按**：〈獄中上梁王書〉的論證結構從根本上解決了一個難題——如何讓一個已經對你產生偏見的人重新信任你？鄒陽的答案是：不要直接證明自己——而是證明「懷疑忠臣」這件事本身就是錯的。如果你能說服對方——「懷疑忠臣」是人類歷史上所有悲劇的共同原因——那麼你作為「被懷疑的忠臣」——自然就是對的。這是一場「降維打擊」式的辯護。

### 3.2 核心典故節錄

> 「白頭如新，傾蓋如故。」

——全篇最著名的句子。認識一輩子，到白頭還像陌生人；路上相遇，停車一談就像老朋友。鄒陽把信任的本質歸結為「知心」——而不是時間。

> 「偏聽生姦，獨任成亂。」

——只聽一面之詞就會產生奸邪，只信任一個人就會釀成禍亂。這是全文的理論核心，直接點出讒言運作的機制。

> 「昔者荊軻慕燕丹之義，白虹貫日，太子畏之。」

——荊軻為了燕太子丹去刺殺秦王，他的忠誠感動了天地（「白虹貫日」），但太子丹仍然懷疑他。鄒陽用這個例子告訴梁王：連荊軻這樣的烈士都會被懷疑——何況是我？

> 「士為知己者用，女為說己者容。」

——一個士人只為真正了解他的人效力。這是全文的情感收束——鄒陽用這句古老的名言，把梁王從「審判者」的位置拉到了「知己」的位置上。

> ——《漢書·鄒陽傳》

## 四、關鍵人物

### 4.1 鄒陽——獄中上書的遊士

| 項目 | 內容 |
|------|------|
| **身份** | 西漢初年齊國（今山東東部）人。縱橫策士，梁孝王的門客。生卒年不詳——約活動於西元前 2 世紀中期（漢文帝至景帝年間） |
| **生平核心** | 鄒陽早年以文辯聞名於齊國。他遊歷各地——最後成為梁孝王的門客。在梁國，他與羊勝、公孫詭等人共事——但他以正直著稱，不參與陰謀。羊勝等人嫉妒他的才能，在梁王面前進讒，致其下獄。他在獄中寫了這封信——梁王讀後大為感動，立即釋放了他，並將他尊為上客。後來羊勝、公孫詭因暗殺袁盎之事敗露而自殺——鄒陽則得以善終 |
| **歷史定位** | 鄒陽在文學史上以〈獄中上梁王書〉一文聞名。他的文章以「善用典故」著稱——是西漢散文從縱橫家的辯論風格向漢賦的鋪陳風格過渡的代表人物之一。他的文章對後世——尤其是六朝駢文——產生了深遠的影響 |

**軍師評**：鄒陽的故事告訴我們一個殘酷的事實——在權力場上，才華本身就是一種危險。你不需要做錯任何事——你只需要比別人優秀——就會有人想要毀掉你。鄒陽在獄中意識到了這一點——所以他沒有說「我錯了」——而是說「自古以來優秀的人都和我一樣被冤枉」。這不是傲慢——這是他對自己命運最深刻的洞察。

### 4.2 梁孝王劉武——權力核心中的諸侯王

| 項目 | 內容 |
|------|------|
| **身份** | 漢文帝之子、漢景帝同母弟——梁國國君。竇太后最寵愛的兒子——曾一度被認為是皇位繼承人 |
| **與鄒陽的關係** | 梁孝王最初賞識鄒陽的才華，將他招為門客。但羊勝、公孫詭的讒言動搖了他對鄒陽的信任——直至將鄒陽下獄。讀了〈獄中上梁王書〉後——梁孝王深受感動——立即釋放鄒陽——並從此更加信任他 |
| **梁孝王的悲劇** | 梁孝王雖然釋放了鄒陽——但他最終還是沒有擺脫權力的誘惑。羊勝、公孫詭後來策劃暗殺朝廷大臣袁盎（反對立梁孝王為太子）——事情敗露後，景帝大怒——梁孝王被迫令羊勝、公孫詭自殺——他自己也從此失去了景帝的信任，鬱鬱而終 |

**軍師評**：梁孝王是一個有矛盾的人——他既能欣賞鄒陽的才華，又能被羊勝的讒言說動；既能因一封書信而釋放鄒陽，又能縱容羊勝去暗殺大臣。他是西漢諸侯王的縮影——既有文化修養，又有權力野心。鄒陽的〈獄中上梁王書〉之所以能打動梁孝王——正是因為梁孝王的本質不壞——他只是被權力和讒言矇蔽了雙眼。<br>但這也說明：讒言的可怕之處——在於它對任何人都有效——即使是像梁孝王這樣懂得欣賞才華的人。

### 4.3 羊勝與公孫詭——讒臣的典型

| 項目 | 內容 |
|------|------|
| **羊勝** | 梁孝王的寵臣，以「多奇邪計」聞名。他與公孫詭聯手排擠鄒陽，使其下獄。後來——他主導暗殺袁盎——事敗後被梁孝王賜死 |
| **公孫詭** | 梁孝王的謀士，與羊勝齊名。兩人同為梁孝王的核心幕僚——後因暗殺案被梁孝王下令自殺 |
| **在本文中的角色** | 他們是推動情節發展的「反派」——沒有他們的讒言，就沒有鄒陽的入獄——也就沒有〈獄中上梁王書〉這篇文章。他們的讒言「策略」極為經典：第一，利用鄒陽的「外地人」身份挑起地域偏見；第二，誇大鄒陽的才華——暗示他「功高震主」；第三，歪曲鄒陽的言行——把正常的社交活動說成是「洩密」。這套詭計——幾乎所有中國歷史上的讒臣都用過 |

**軍師評**：羊勝和公孫詭是中國歷史上最常見的「小人」類型——他們不是特別邪惡——他們只是想要權力。而要排除競爭對手——最有效的方法不是證明自己有多優秀——而是證明別人有問題。這就是讒言的邏輯：它不是用事實說話——而是用猜疑說話。你永遠無法用事實去反駁一個猜疑——因為猜疑不需要證據。鄒陽的高明之處——就是他完全避開了這個陷阱——他根本不去反駁讒言——而是用整個歷史告訴梁王：讒言是不可信的。

## 五、核心分析——「白頭如新，傾蓋如故」

### 5.1 「知心」——鄒陽對信任的本質理解

鄒陽對信任的理解，達到了非常深刻的層次。他開篇提出的「白頭如新，傾蓋如故」——不僅僅是一句名言——它是對「信任」本質的終極洞察：

| 面向 | 分析 |
|------|------|
| **時間 ≠ 信任** | 鄒陽指出：認識時間的長短與信任的程度沒有必然關係。有些人認識一輩子——仍然互不信任（「白頭如新」）。有些人在路上偶然相遇——卻能立刻推心置腹（「傾蓋如故」）。信任的本質——不是時間的累積——而是心靈的相通 |
| **什麼是「傾蓋如故」？** | 「傾蓋」——指兩輛車在路上相遇，停下來交談時車蓋傾斜在一起——形容一見如故。鄒陽暗示：真正的信任——是「知心」——即一個人能夠理解另一個人的內心。如果你不能「知心」——你就會像那些懷疑荊軻、卞和、百里奚的君主一樣——錯過真正忠誠的人 |
| **對梁王的提醒** | 鄒陽實際上是在委婉地批評梁王：你與我相處了這麼久——卻仍然聽信讒言——這是「白頭如新」。你沒有真正了解我的內心——所以你才會被讒言左右。但如果你能「傾蓋如故」——你就能跨越表面的判斷——直達我內心的忠誠 |

> **軍師按**：「白頭如新，傾蓋如故」這八個字——是整封信的靈魂。它道出了一個人際關係的永恆真理：認識一個人多久不重要——重要的是你是否真的了解他。在現代社會——我們認識幾千個人——但真正信任的，也不過是那幾個「傾蓋如故」的人。鄒陽兩千年前就說破了這件事。

### 5.2 「以典為證」——鄒陽的辯護策略

〈獄中上梁王書〉最獨特的地方——是它完全不使用傳統的辯護方式。鄒陽既不證明自己的清白——也不反駁讒言的內容——他選擇的是一條「降維打擊」的路徑：

| 面向 | 分析 |
|------|------|
| **為什麼不直接為自己辯護？** | 鄒陽知道——讒言的邏輯不是基於事實——而是基於猜疑。如果你試圖用事實反駁猜疑——你會陷入「越解釋越可疑」的陷阱。所以——乾脆不解釋——直接跳到更高的層次 |
| **為什麼選擇用歷史典故？** | 因為歷史是「不可否認的事實」——荊軻被懷疑、卞和被砍腳、范雎被打斷肋骨——這些事已經發生了，沒有人能否認。鄒陽把這些事實「打包」成一個論點：忠臣被讒害是歷史的普遍規律。然後——他把自己的命運鑲嵌進這個規律中——成為這個規律的最新例證 |
| **這種策略的效果** | 當梁王讀到這封信時——他會發現自己不是第一個被讒言左右的君主——如果殺掉鄒陽——他會成為歷史上下一個「因讒殺忠」的昏君。這不是恐嚇——這是一個理性的選擇：你是想做一個英明的君主——像那些最終醒悟的君王——還是想做一個被讒臣控制的昏君？ |

**軍師評**：鄒陽的策略可以總結為一句話——「不要反駁敵人——直接消滅戰場。」他不和羊勝在「鄒陽是否忠誠」這個問題上糾纏——他把討論的層次直接拉升到「如何判斷一個人的忠誠」。在這個層次上——羊勝那些低級的讒言——就顯得微不足道了。這是一種「將軍」式的論辯——不和你下棋——直接掀翻棋盤。

### 5.3 〈獄中上梁王書〉與〈報任安書〉的對比

鄒陽的〈獄中上梁王書〉與司馬遷的〈報任安書〉——都是獄中或絕境中寫給權力者的書信——但兩者有著本質的不同：

| 面向 | 〈獄中上梁王書〉 | 〈報任安書〉 |
|------|------|------|
| **處境** | 死囚牢中——寫給要殺自己的梁王 | 宮刑之後——寫給朋友的私人回信 |
| **目的** | 求活——說服梁王釋放自己 | 解釋——說明白己為什麼選擇活著 |
| **策略** | 用歷史典故證明「忠臣被讒」是普遍規律 | 用個人痛苦證明「隱忍苟活」是為了完成《史記》 |
| **情感** | 理性、雄辯——情緒藏在典故背後 | 感性、直接——淚水與憤怒都在紙上 |
| **語氣** | 對君主的恭敬與策略性的「升維打擊」 | 對朋友的坦誠與毫不掩飾的剖白 |

## 六、軍師核心分析——讒言的邏輯與破解之道

### 6.1 為什麼讒言如此有效？

鄒陽的〈獄中上梁王書〉——除了是一篇偉大的文章——還是一篇對「讒言心理學」的精妙分析。他告訴我們——讒言有效的秘密不在於說謊——而在於利用人性的弱點：

| 弱點 | 說明 |
|------|------|
| **權力的焦慮** | 每一個統治者都對權力有天然的戒備——他們永遠在擔心：這個人是不是太強了？他會不會威脅到我的權力？讒臣就是利用這種焦慮——把忠臣的一切正常行為——都解讀為「不忠」的證據 |
| **偏聽偏信** | 鄒陽說：「偏聽生姦，獨任成亂。」——如果一個人只聽一種聲音——他就會被這種聲音控制。讒臣的策略非常簡單：孤立君主——讓君主只聽到讒言——聽不到其他聲音 |
| **外鄉人偏見** | 羊勝攻擊鄒陽的第一個理由——就是他是「齊人」——不是梁國本土人。這種利用地域偏見的手法——從古至今從未改變。因為「外來者」永遠是最容易被攻擊的目標——他們沒有本土的社會網絡——更容易被孤立 |

### 6.2 鄒陽的破解之道——「以史為鑑」

面對讒言的邏輯——鄒陽給出的破解方案是「以史為鑑」：

- **不與讒言在同一個層次上爭論**：讒言說你不忠——你不要說「我忠」——你要說「歷史上所有忠臣都被說過不忠」。
- **把個案上升到普遍規律**：你不是第一個被冤枉的忠臣——你也不會是最後一個——但你可以讓當下的君主選擇不做那個被讒言左右的昏君。
- **用文化的力量壓制權力**：梁王可以殺一個人——但他無法否定整個中國歷史。當鄒陽把荊軻、卞和、百里奚這些名字一個接一個擺出來時——梁王面對的不是一個囚犯——而是幾千年的歷史壓力。

> **軍師按**：鄒陽的〈獄中上梁王書〉——是中國歷史上最早、最成功的「以知識反抗權力」的案例之一。在一個囚犯面對一個諸侯王的懸殊權力對比中——知識（歷史知識）成了唯一的武器。而且——它成功了。這證明了：當權力想要消滅一個人的時候——如果這個人的知識足夠豐富——他可以用歷史來對抗權力。因為權力可以殺人——但權力不能改寫已經發生的事實。

## 七、一句話總結（再錄）

> **〈獄中上梁王書〉是一個人在死囚牢中寫出的最優雅的「自救操作手冊」。鄒陽不說「我冤枉」，不說「饒命」——他什麼也不為自己說——他只是在講歷史故事。他講荊軻被懷疑、卞和被砍腳、百里奚賣身為奴、范雎被打斷肋骨——他講了那麼多故事——梁王終於明白了：鄒陽講的不是歷史——鄒陽講的是此刻正在發生的事。而梁王如果殺了他——就會成為這個故事中的又一個昏君。鄒陽輸了嗎？沒有——他用自己的學問，從死牢裡走了出來。他用歷史證明了一件事：當一個人站在真理一邊的時候——即使他被關在監獄裡——他也是自由的。而那個站在監獄外面、手握生殺大權的人——如果他在做錯的事——他才真正被關在監獄裡。那間監獄——叫做偏見。** |

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈報任安書〉（司馬遷） | 同為絕境中的申訴——但司馬遷是向朋友解釋「為什麼活著」——鄒陽是向君王證明「為什麼不該殺我」。兩封獄中書信，兩種不同的自救策略——但同樣展現了文字改變命運的力量 |
| 卷四·〈卜居〉（屈原） | 屈原在〈卜居〉中質問天命——為什麼正直的人沒有好下場？鄒陽的〈獄中上梁王書〉在某種意義上是對這個問題的回答：因為讒言永遠存在——所以你要學會對抗讒言 |
| 卷三·〈漁父〉（屈原） | 漁父勸屈原「與世推移」——隨波逐流。鄒陽沒有隨波逐流——他用知識和策略對抗權力——他選擇了用歷史來「說服」而非「順從」 |
| 卷一·〈曹劌論戰〉（左傳） | 都是用小人物對大權力的說服——曹劌用軍事常識說服魯莊公——鄒陽用歷史常識說服梁孝王 |
| 卷六·〈過秦論上〉（賈誼） | 賈誼用歷史分析秦朝興亡——鄒陽用歷史分析讒言的本質——兩者都是「以史論政」的典範 |

### 8.2 經典延伸

- 《漢書·鄒陽傳》（班固）——〈獄中上梁王書〉的原始出處——班固在傳中完整收錄了這封信，並記載了鄒陽的完整生平
- 《漢書·文三王傳》（班固）——梁孝王劉武的傳記——是理解鄒陽寫作此信時代背景的第一手資料
- 《史記·梁孝王世家》（司馬遷）——司馬遷對梁孝王的記述——可與班固的記載互相補充
- 《戰國策》——鄒陽的論辯風格深受戰國縱橫家的影響——讀《戰國策》可以理解鄒陽的「歷史典故辯論法」的源頭
- 《文選》（蕭統編）——〈獄中上梁王書〉收入《昭明文選》卷三十九——蕭統將此篇歸為「書」類
- 劉勰《文心雕龍》——劉勰在〈論說〉篇中討論了古代辯論文的藝術——鄒陽的這封信是「以典論理」的典範案例

### 8.3 現代延伸

- 史蒂芬·柯維《高效能人士的七個習慣》——柯維提出的「知彼解己」（先理解別人，再被別人理解）——與鄒陽的溝通策略高度一致——先理解歷史的普遍規律，再讓梁王理解自己的處境
- 羅伯特·西奧迪尼《影響力》——書中分析了社會影響力的六大原則——鄒陽的核心策略是「權威原則」（以歷史為權威）和「社會認同原則」（歷史上有那麼多忠臣都被冤枉了——我也一樣）
- 克里斯多福·沃格勒《作家之路》——關於「英雄之旅」的結構——鄒陽的故事本身也是一部「英雄之旅」：從被誣陷（召喚）到入獄（磨難）到用知識反擊（考驗）到最終獲釋（回歸）
