# 〈周鄭交質〉背景資料

> 古文觀止卷一·左傳·隱公三年
> 撰寫風格：軍師策論

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **時間** | 魯隱公三年（西元前720年），東周初期 |
| **空間** | 洛邑（東周王都）與鄭國（今河南新鄭一帶） |
| **核心事件** | 周平王與鄭莊公交換人質——王子狐質於鄭，公子忽質於周 |
| **引爆點** | 周平王死，周桓王即位，欲以虢公忌父分鄭伯之政 |
| **經典文眼** | 「信不由中，質無益也。」 |

此為春秋時代第一場天子與諸侯的正面權力攤牌。地點在天下之中——洛邑與鄭之間，距離不過數百里，卻丈量出周禮崩潰的深度。

---

## 二、時代壓力：禮崩樂壞的開局

### 2.1 東周王室的軟骨症

西元前771年，犬戎破鎬京，周幽王死於驪山。周平王在晉、鄭、秦、衛等諸侯護送下東遷洛邑——名義上是遷都，實質上是逃跑。

東遷後的周王室，面臨三大致命傷：

1. **領土縮水**：王畿從關中千里縮為洛邑周邊數百里，實力不如大諸侯。
2. **軍事依賴**：東遷全程靠諸侯護送，此後軍隊征討也需要諸侯出兵。
3. **威信破產**：幽王戲諸侯（烽火戲諸侯）已使天子信譽掃地，平王靠弒父（一說）上位更添汙名。

### 2.2 鄭國的特殊地位

鄭國是西周最晚分封的諸侯（宣王封其弟友於鄭），但鄭桓公、鄭武公、鄭莊公三代在王朝擔任**卿士**（執政官），身分既是諸侯，又是天子近臣。鄭國挾此雙重身分，在東周初期迅速擴張：

- **鄭武公**：護送平王東遷有功，取得大量東虢、鄶國土地
- **鄭莊公**：在位期間平定共叔段內亂、滅許、敗宋，儼然小霸

莊公以卿士身分留在洛邑執政，實際上掌握了天子朝政。他既是諸侯，又是宰相——這本身就是對周禮的挑戰：諸侯不應干預天子內政，但天子又離不開這個強有力的「下屬」。

### 2.3 人質外交的荒謬

周平王不堪鄭莊公專權，想讓虢公分權制衡。鄭莊公質問，平王矢口否認，最後用**交換人質**來自清。

這在周禮中是無法想像的：

> 天子與諸侯關係，本質上是君臣父子。父親需要交出兒子做擔保、才能證明自己沒有背叛兒子——荒謬至極。

**周鄭交質**象徵的是：天子與諸侯已經站在同一水平線上交涉。君臣之間的權力關係，變成了對等國家之間的外交契約。周禮的「天子無敵體」原則，至此徹底破產。

### 2.4 惡化的必然

人質交換並未解決問題。西元前720年，周平王駕崩。周人想趁新君即位，將鄭國土地正式轉給虢公。鄭莊公大怒，做了一件史無前例的事：

**派祭足率軍割周室溫地之麥，又取成周之禾。**

這是春秋時代諸侯第一次武力搶劫天子領地。從交換人質到武裝收割，不到一年。人質沒有換來信任，反而暴露了雙方連表面功夫都懶得維持了。

周桓王即位後，乾脆正式剝奪鄭莊公的卿士職位。雙方決裂。後續的**繻葛之戰**（前707年，桓王親征鄭國，被鄭軍擊敗，桓王中箭）將是這一衝突的終局——但〈周鄭交質〉只寫到收割麥禾為止，留下一個「周鄭關係已經不可逆轉」的結論。

## 三、關鍵人物

### 周平王（？—前720年）

| 面向 | 分析 |
|------|------|
| **處境** | 東周開國之君，依靠諸侯上位，從即位第一天就是弱主 |
| **性格** | 軟弱、求和、不果斷——既不敢得罪鄭國，又想制衡鄭國 |
| **失策** | 想要分權卻不敢明說；被質問後全盤否認；用交換人質這種毫無約束力的手段來維持和平。每一步都在退讓，退到無路可退時，什麼也沒解決。 |
| **軍師評語** | 平王之病，不在無謀，而在無膽。既用鄭，又疑鄭；既疑鄭，又不敢去鄭。人質之約，不過是懦夫的自欺欺人。 |

### 周桓王（？—前697年）

| 面向 | 分析 |
|------|------|
| **處境** | 平王之孫（太子泄父早死），少年即位，血氣方剛 |
| **性格** | 剛愎、急進、不懂權謀——見不得鄭國囂張，但沒有實力壓制 |
| **行動** | 一即位就分鄭伯之政予虢公，直接引爆衝突。後來親征鄭國，中箭而敗，更使王室威信掃地。 |
| **軍師評語** | 桓王有恢復王室之心，無恢復王室之才。少年意氣，不知實力懸殊。引而不發是最痛苦的，但貿然出手是最愚蠢的。桓王兩者都犯了。 |

### 鄭莊公（前757—前701年）

| 面向 | 分析 |
|------|------|
| **身分** | 鄭國第三代君主，王朝卿士，春秋第一位小霸主 |
| **性格** | 深沉、剛毅、精於算計——史料中最能忍、最會抓時機的諸侯之一 |
| **策略** | 以卿士身分控制朝政，用「奉王命討不臣」的名義擴張實力，實質上是在挾天子以令諸侯（這比齊桓公早了近百年） |
| **行動** | 平王在世時施壓換人質；桓王翻臉後派兵收割天子糧食；四年後在繻葛戰場上射傷天子 |
| **軍師評語** | 莊公是春秋第一位懂得「實力即正義」的諸侯。他從不主動挑戰天子名義——他只挑戰天子實際權力。人質是他要的，麥禾是他割的，天子是他射的，但他從未口出狂言說不要天子。這才是真正可怕的政治人物。 |

## 四、寫作意圖

左丘明寫〈周鄭交質〉，用意不在記錄一次外交事件，而在揭示一個時代轉折：

### 4.1 破題——信不由中

文章開頭八個字：「信不由中，質無益也。」這不是旁白，而是一句審判。左丘明在告訴讀者：**人質只是表象，信任才是本質。沒有信任，一百個王子當人質也沒用。**

### 4.2 借古諷今

左丘明引用〈召公·君奭〉（「明德慎罰」）、〈風〉〈雅〉（「不遐有佐」），對比周鄭交質的墮落。古代聖王靠明德維繫君臣關係，現在的周天子卻要靠交換人質來保住表面和平。這是左丘明對時代的哀嘆：禮樂已經被功利主義全面取代。

### 4.3 為春秋大義立基

〈周鄭交質〉是《左傳》前幾篇之一。左丘明要在這裡告訴讀者：**春秋之所以亂，根源在於天子自棄天下**。當天子把自己降到諸侯的層次去做交易，天下就再也不會有人把天子當天子了。

### 4.4 名句賞析

> **「信不由中，質無益也。明恕而行，要之以禮，雖無有質，誰能間之？」**

翻譯：信任不是發自內心，交換人質也沒用。以明理寬恕行事、以禮儀為約束，就算沒有質子，誰又能離間呢？

這是全文的命脈所在。左丘明認為：人質交換是技術手段，禮義信任才是根本。周王室和鄭國的問題從來不是人質太少，而是心已經不在一起了。

## 五、戰略貫通——此事件的歷史意義

| 層面 | 意義 |
|------|------|
| **政治象徵** | 天子與諸侯交換人質，打破了「天子至尊」的禮制底線 |
| **權力轉移** | 鄭莊公以卿士挾天子，開「挾天子以令諸侯」之先河 |
| **外交型態** | 人質外交（質子制度）從此成為春秋戰國的常態，但〈周鄭交質〉證明了其無效性 |
| **戰爭導火線** | 此事直接導致繻葛之戰（周桓王伐鄭，王師敗績） |
| **史書價值** | 左丘明以此篇表明：春秋之亂，始於天子自輕 |

---

## 六、一句話總結

> **當天子必須與諸侯交換人質才能維持關係時，周禮已經死了——人質換不來信任，信不由中，雖質何益？**

---

*軍師筆記：後世讀者讀到此篇，當知所謂「質子制度」從未真正維繫過和平。無論是周鄭交質、戰國時期的各國質子，乃至後世的和親、納質，本質上都是實力不對等下的妥協方案。真正能長治久安的，從來不是人質，而是「明恕而行，要之以禮」。左丘明兩千六百年前的結論，至今未過時。*
