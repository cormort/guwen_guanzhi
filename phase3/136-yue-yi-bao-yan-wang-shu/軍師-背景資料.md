# 〈樂毅報燕王書〉背景資料

> **軍師分析 · 古文觀止卷四 · 戰國策·燕策**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《戰國策·燕策》（《史記·樂毅列傳》亦載） |
| **選集** | 《古文觀止》卷四·周文類（戰國策文／史傳文） |
| **體裁** | 書信體／自白辯護文 |
| **時代** | 戰國中晚期（約西元前 279 年，燕惠王時期） |
| **核心命題** | 忠臣的自白——「善作者不必善成，善始者不必善終」 |
| **關鍵人物** | 樂毅（燕國名將，率五國聯軍破齊，連下七十餘城）、燕昭王（知人善任的明君，築黃金臺招賢）、燕惠王（繼位後猜忌樂毅，中反間計逼走樂毅） |
| **文本重點** | 樂毅為燕昭王攻齊，幾乎滅齊。昭王死，惠王即位，中齊國反間計，以騎劫代樂毅。樂毅逃往趙國，騎劫被齊軍擊敗，燕國丟失全部攻佔的齊地。惠王後悔，派人責備樂毅為何不忠。樂毅回信——解釋自己當初的忠誠不被新君信任，被迫離開；並表明自己不會率趙軍反攻燕國，因為他曾受昭王知遇之恩。「臣聞古之君子，交絕不出惡聲；忠臣去國，不潔其名。」 |
| **思想譜系** | 出自《戰國策·燕策》，亦見於《史記·樂毅列傳》，司馬遷將其收錄於列傳中，視為戰國最動人的書信之一。樂毅這封信開創了「去國書信」的文體典範——被冤屈的大臣在離開故國後，寫信向舊主解釋自己為何離開、為何不會報復。後世諸葛亮〈出師表〉、李斯〈諫逐客書〉皆受其影響。信中「交絕不出惡聲」成為中國士大夫「君子絕交」的最高道德標準 |
| **一句話總結** | **樂毅的回信是中國史上最高貴的離職信——他用一封信解決了政治上的敵意，也用一封信保全了君子的體面。他沒有罵新君一句，沒有替自己辯解一句，但每一個字都在說：我沒有對不起燕國，是燕國對不起我。** |

## 二、時代背景——樂毅破齊與燕國興衰

### 2.1 燕昭王復仇與樂毅伐齊

燕國在戰國七雄中實力最弱。西元前 314 年，齊宣王趁燕國內亂（子之之亂）攻破燕國，幾乎滅燕。燕昭王（姬職）即位後，立志報齊國滅國之仇，「卑身厚幣以招賢者」——築黃金臺，拜郭隗為師，吸引各國人才。樂毅正是在此時從魏國來到燕國，被昭王任命為亞卿。

| 面向 | 說明 |
|------|------|
| **燕昭王復仇** | 昭王在位二十八年（前 311—前 279 年），勵精圖治，整軍經武，養精蓄銳，等待伐齊時機 |
| **黃金臺招賢** | 昭王築黃金臺（今河北易縣），置千金於臺上以招天下賢士——樂毅、鄒衍、劇辛等各國人才紛紛來投 |
| **樂毅入燕** | 樂毅本是魏國將領之後，出使燕國時被昭王禮遇，遂留在燕國，官至亞卿 |
| **五國聯軍伐齊** | 西元前 284 年，燕昭王以樂毅為上將軍，聯合趙、楚、韓、魏四國，大舉攻齊。樂毅在濟西之戰大破齊軍主力 |
| **連下七十餘城** | 樂毅攻入齊國後，分兵五路，連下七十餘城，僅剩莒和即墨兩城未克。齊湣王出逃，被楚將淖齒所殺——燕國幾乎滅亡齊國 |

> **軍師按**：樂毅伐齊是戰國史上最驚人的以弱勝強戰例之一。燕國本是七雄中最弱的國家，但在燕昭王的經營和樂毅的指揮下，幾乎滅掉了東方最強的齊國。這背後是兩代人的積累——燕昭王用了二十八年來準備這一戰，而樂毅只用了不到一年就把齊國打到了亡國的邊緣。這是一場完美的「長期戰略」的勝利：先用時間累積實力，再用一個關鍵人物實現突破。

### 2.2 燕惠王即位與樂毅奔趙

西元前 279 年，燕昭王去世，其子燕惠王（姬樂資）即位。惠王早在為太子時就與樂毅不和。齊國大將田單抓住這個機會，使用反間計：派人到燕國散布謠言，說樂毅之所以不攻下莒和即墨，是因為他想在齊國稱王。惠王中計，派騎劫取代樂毅。樂毅知道惠王不信任自己，回到燕國必被殺害，便逃往趙國。趙國封他為望諸君。

| 事件 | 後果 |
|------|------|
| **燕昭王去世** | 樂毅失去最信任他的君主——這是整件事的轉折點 |
| **齊國反間計** | 田單散布謠言：「樂毅與燕新王有隙，畏誅欲連兵留齊南面而王」 |
| **燕惠王中計** | 惠王派騎劫取代樂毅——下了一步足以毀掉燕國全部戰果的昏棋 |
| **樂毅奔趙** | 樂毅逃往趙國——他不是背叛燕國，而是為了自保。趙王封他為望諸君 |
| **田單復國** | 騎劫不懂軍事，被田單以火牛陣大敗。燕軍潰敗，齊國收復全部失地 |
| **燕國功虧一簣** | 樂毅用五年打下來的七十餘城，騎劫在幾個月內全部丟光——燕國從此一蹶不振 |

> **軍師按**：燕惠王的「換將」是戰國史上最愚蠢的決策之一。樂毅的戰略是「先圍城，後攻心」——他留下莒和即墨不攻，不是因為攻不下來，而是因為他要讓齊國人慢慢放棄抵抗意志。騎劫一到，改用強攻，結果被田單一舉擊敗。這是一個經典的「外行干預內行」的案例——一個不懂軍事的新君，因為對部下的猜忌，親手毀掉了父親用二十八年積累的全部成果。而樂毅的回信，就是在這個背景下寫成的——他沒有罵惠王愚蠢，但他告訴惠王：你換掉我，不是因為我不行，而是因為你不相信我。而你不相信我，失去的卻是你自己的七十座城池。

## 三、原文結構與關鍵情節

### 3.1 書信的結構

〈樂毅報燕王書〉是一封結構極為完整的辯護信，可以分為五個層次：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：自述忠誠** | 樂毅先說明自己當初為什麼接受燕昭王的任命——因為昭王賞識他，他願意為知己者死。他率軍伐齊，不是為了自己的功名，而是為了報答昭王的知遇之恩 | 建立「我對燕國有功、無過」的基本敘事 |
| **第二層：解釋離開** | 樂毅說明自己為什麼逃往趙國——不是因為不忠，而是因為惠王派騎劫取代他之後，他回到燕國必被殺害。他不想因為自己被殺而讓燕國背上「殺功臣」的惡名 | 把責任推給惠王的「猜忌」，而非自己的「背叛」 |
| **第三層：表明不報復** | 樂毅明確表示：我不會像伍子胥那樣率領敵國軍隊回來報復燕國。因為昭王對我有知遇之恩，我絕不會做對不起昭王的事 | 消除惠王最大的恐懼——擔心樂毅率趙軍反攻燕國 |
| **第四層：君子之道** | 樂毅提出著名的「交絕不出惡聲，忠臣去國不潔其名」——君子即使絕交，也不會說對方的壞話；忠臣離開了國家，也不會為了證明自己的清白而揭發舊主的過錯 | 這是全信最崇高的段落——樂毅用君子的標準要求自己，不罵惠王、不揭短、不羞辱。他不是在為自己辯解，而是在教惠王什麼叫做君子 |
| **第五層：結尾** | 樂毅以謙卑的語氣作結——請惠王不要再責備他，他會安分守己地在趙國生活，不會對燕國造成任何威脅 | 給惠王一個台階下，也為自己爭取一個安全的結局 |

> **軍師按**：這封信最了不起的地方，不在於它有多雄辯，而在於它「沒有說的話」。樂毅沒有說「你這個昏君，換掉我讓你失去了七十座城池」——他完全沒有提這一點。為什麼？因為當一個人已經用事實懲罰了你（惠王因為換將而失去全部戰果），你再去說「你看，我早就告訴你了」，不僅沒有任何意義，反而會讓對方惱羞成怒。樂毅選擇了最高貴的沉默——他讓事實替自己說話。

### 3.2 核心段落節錄

> 臣聞賢聖之君，不以祿私其親，功多者授之；不以官隨其愛，能當者處之。故察能而授官者，成功之君也；論行而結交者，立名之士也。

> ——樂毅闡述「用人唯賢」的治國原則——這也是對燕惠王的間接批評

> 臣聞古之君子，交絕不出惡聲；忠臣去國，不潔其名。

> ——全文最著名的兩句話——「君子即使絕交，也不口出惡言；忠臣離開故國，也不為求清白而揭短。」這是樂毅為這封信定下的基調：我不會罵你，但我也不會為你背黑鍋。

> 臣雖不佞，數奉教於君子矣。恐侍御者之親左右之說，而不察疏遠之行也。故敢以書報，唯君之留意焉。

> ——結尾：謙卑、克制、不卑不亢

> ——《戰國策·燕策》

## 四、關鍵人物

### 4.1 樂毅——被冤屈的名將，高貴的離去者

| 項目 | 內容 |
|------|------|
| **身份** | 燕國上將軍，率五國聯軍伐齊，連下七十餘城 |
| **奔趙後** | 趙國封為望諸君，與趙國保持良好關係 |
| **核心貢獻** | 為燕昭王幾乎滅亡齊國——這是戰國史上最驚人的軍事成就之一 |
| **人格特質** | 忠誠但理性——他忠於燕昭王，但不愚忠於燕惠王。他知道什麼時候該走、該怎麼走、走了之後該怎麼說 |
| **歷史定位** | 戰國四大名將之一（與白起、王翦、李牧齊名），但樂毅與其他三人最大的不同是：他靠的不是殺戮，而是戰略與攻心——「樂毅攻齊，不屠城、不殺降，以德服人」——這在戰國是極罕見的 |

**軍師評**：樂毅的偉大不全在他能打仗，更在他能「去國」。他在燕惠王猜忌他、逼他離開的時候，沒有選擇玉石俱焚（如後來的白起之於秦昭王），也沒有選擇以怨報德（如伍子胥之於楚平王），而是選擇了「走」，然後用一封充滿君子風度的信為自己畫上句點。他的尊嚴不來自於他的戰功，而來自於他在受委屈時的選擇——而他選擇了最高貴的那條路。

> **軍師按**：樂毅的智慧在於他明白一個道理：**有些仗可以打贏，但有些仗根本就不該打。** 他可以在趙國率軍反攻燕國，他也許能打贏，但打贏了又能怎樣？他會成為一個「叛將」，會對不起燕昭王的知遇之恩，會讓自己在歷史上蒙上污點。所以他選擇了不打——這不是懦弱，這是一種超越了勝負的格局。一個打贏了七十場仗的人，在他最有理由報復的時候選擇了不報復——這才是真正的強者。

### 4.2 燕昭王——知人善任的理想君主

| 項目 | 內容 |
|------|------|
| **身份** | 燕國國君，在位 28 年（前 311—前 279 年） |
| **核心事蹟** | 築黃金臺招賢，使弱小的燕國一躍成為能夠滅齊的強國 |
| **與樂毅的關係** | 君臣相知——昭王完全信任樂毅，給他最大的權力和時間去實現伐齊的戰略 |
| **歷史定位** | 戰國最善於用人的君主之一——他懂得「用人不疑」這個最簡單也最難做到的道理 |

**軍師評**：燕昭王的故事和燕惠王的故事，是中國歷史上最強烈的對比之一：同一個父親，同一個兒子；同一個國家，同一個將軍。父親用信任換來了七十座城池，兒子用猜忌丟掉了七十座城池。昭王的偉大不在於他有多聰明，而在於他願意信任一個比自己聰明的人——這才是最難得的領導者品質。

### 4.3 燕惠王——猜忌的代價

| 項目 | 內容 |
|------|------|
| **身份** | 燕昭王之子，燕國國君 |
| **核心錯誤** | 中齊國反間計，以騎劫代樂毅，導致燕國失去全部戰果 |
| **性格缺陷** | 猜忌、短視、急功近利——他不理解樂毅「圍城攻心」的戰略，只想快點看到結果 |
| **尷尬處境** | 本來想責備樂毅「為何不忠」，結果樂毅的回信讓他無言以對——因為事實證明，不忠的不是樂毅，而是他自己（他親手毀了父親的遺產） |

**軍師評**：燕惠王是中國歷史上最典型的「敗家子」——父親用二十八年的心血累積的國力、用五年的時間打下的天下，他在幾個月內全部丟光。而他最愚蠢的地方是：他不但不反省自己的錯誤，還派人去責備樂毅。這就像一個賭徒輸光了家產，然後去責備那個提醒他「不要賭」的人一樣荒謬。

## 五、核心分析——「交絕不出惡聲」的君子哲學

### 5.1 一封「不罵人」的控訴信

〈樂毅報燕王書〉最獨特的地方，就在於它是一封「沒有罵人的控訴信」。樂毅被冤屈、被逼走、被奪走了一切——但他沒有一個字是在責罵惠王。他用了更高明的方式：

| 策略 | 樂毅的做法 | 效果 |
|------|-----------|------|
| **讓事實說話** | 絕口不提騎劫的失敗和七十城的丟失 | 惠王自己知道發生了什麼——不需要樂毅來說 |
| **用讚美來批評** | 反覆提及燕昭王的知人善任 | 每一句讚美昭王，都是在批評惠王——如果你父親能做到，為什麼你做不到？ |
| **表明不報復** | 我不會像伍子胥那樣 | 這是最聰明的一步——消除了惠王最大的恐懼，讓惠王沒有理由再追殺他 |
| **用君子的標準要求自己** | 交絕不出惡聲，去國不潔其名 | 將自己置於道德高地——惠王如果繼續指責他，反而顯得自己心胸狹隘 |

> **軍師按**：最高明的辯護，不是證明「我是對的」，而是讓對方自己發現「原來是我不對」。樂毅沒有說惠王錯了，但惠王看完這封信，不可能不感到慚愧——因為信裡的每一個字都在說：我對得起燕國，對得起昭王，對得起我的良心。而你呢？

### 5.2 「善作者不必善成，善始者不必善終」

這是樂毅信中最深刻的一句話——它不僅是在說自己的遭遇，更是在說一句關於人類命運的永恆真理：

- **善作者不必善成**：擅長開始的人，不一定能夠完成。樂毅開始了伐齊的事業並幾乎完成，但接替他的騎劫卻毀掉了這一切。
- **善始者不必善終**：有好的開始，不一定有好的結束。燕昭王有好的開始（用人得當），但燕惠王卻讓一切都結束在失敗中。

這句話的深層含義是：**一個人能控制的，只有自己是否努力去做；至於結果如何、是否能夠善終，往往取決於那些你無法控制的因素——比如繼任者的素質、命運的變化、他人的選擇。** 樂毅用這句話告訴惠王：我已經盡了我的全力，我給了燕國一個「善作」和「善始」；至於「善成」和「善終」，那不是我能決定的——是你決定的。

## 六、軍師核心分析——最高貴的離職信

### 6.1 樂毅的選擇：為什麼他不報復？

樂毅完全有理由、有能力報復燕國：
- 他有軍事天才（連下七十餘城的戰績足以證明）
- 他有趙國的支持（趙國封他為望諸君，願意給他軍隊）
- 燕國的主力軍剛在即墨被田單擊敗，防備空虛

但他選擇了不報復。為什麼？

| 原因 | 說明 |
|------|------|
| **感恩昭王** | 燕昭王對他的知遇之恩，是他這一生最大的榮耀。他寧可受委屈，也不願意讓這份榮耀蒙上污點 |
| **君子的自我要求** | 樂毅對自己有極高的道德要求——他不是那種「你對不起我，我就報復你」的人。他選擇了「交絕不出惡聲」 |
| **戰略眼光** | 如果他報復了，他會贏得一場戰爭，但會輸掉歷史上的名聲——他會從「名將」變成「叛將」。而他不報復，歷史會記住他的高貴 |
| **真正的自信** | 一個真正自信的人，不需要用報復來證明自己的強大。樂毅的沉默不是軟弱，而是自信到不需要向任何人證明 |

> **軍師按**：樂毅不報復的決定，讓他在中國歷史上得到了一個極其罕見的地位——他是一個「被冤屈後仍然被尊重的名將」。回顧歷史，被冤屈的名將幾乎沒有一個有好下場：白起被賜死，李牧被殺，岳飛被冤殺，袁崇煥被凌遲——只有樂毅，在被冤屈之後，不但活得好好的，而且仍然贏得了所有人的尊重。為什麼？因為他在最該憤怒的時候選擇了寬容，在最該報復的時候選擇了沉默。**他的尊嚴來自於他沒有做的事，而不是他做了的事。**

## 七、一句話總結（再錄）

> **樂毅的回信是中國史上最高貴的離職信——他用一封信解決了政治上的敵意，也用一封信保全了君子的體面。他沒有罵新君一句，沒有替自己辯解一句，但每一個字都在說：我沒有對不起燕國，是燕國對不起我。而當他寫下「交絕不出惡聲，忠臣去國不潔其名」的時候，他不只是在為自己辯護，更是在為後世所有被冤屈的士人立下了一個標準：你的尊嚴不在於你如何對待敵人，而在於你如何對待曾經信任過你的人。樂毅沒有報復燕國，不是因為他沒有能力，而是因為他不願讓燕昭王的名聲受損。這份超越個人恩怨的格局，才是這封兩千年前的信到今天仍然能打動人的真正原因。**
