# 〈齊桓下拜受胙〉背景資料

## 一、座標定位

| 軸線 | 定位 |
|------|------|
| **時間** | 魯僖公九年（公元前 651 年）夏，周襄王即位元年 |
| **空間** | 宋國葵丘（今河南商丘民權縣一帶），諸侯會盟之地 |
| **坐標意義** | 齊桓公霸業巔峰的最後一塊拼圖——以天子之禮，坐實盟主之位 |
| **原文出處** | 《左傳·僖公九年》，收錄於《古文觀止》卷一 |

## 二、時代壓力

### 2.1 天下大勢：霸業至此登頂

- 葵丘之會是齊桓公**第九次**召集諸侯會盟，也是規模最大的一次——魯、宋、衛、鄭、許、曹等國君皆至，周天子亦遣使與會。
- 在此之前，齊桓公已先後伐山戎、救邢存衛、遏楚北進（召陵之盟），「尊王攘夷」的口號已從宣傳戰術變成戰略成果。
- 若說召陵之盟（前 656 年）是齊桓公逼退楚國的**軍事高點**，葵丘之會就是他把軍事優勢轉化為**政治合法性**的**制度高點**。

### 2.2 內部暗流：霸主也有黃昏

- 齊桓公此時已年過六十（在位已逾三十年），管仲亦垂垂老矣。霸主團隊正在老化。
- 諸侯表面順服，實則各有盤算——鄭國反覆無常，楚國虎視眈眈，周王室內部剛經歷王子帶之亂（惠王欲廢太子鄭，立王子帶）。
- 周襄王（太子鄭）即位未久，地位尚不穩固，需要齊桓公這個「諸侯伯」來撐場面。
- **齊桓公的真正困境**：霸業愈高，人心愈疑。諸侯怕你變成另一個「挾天子以令諸侯」的獨夫，天子怕你功高震主。一個動作失誤，十年累積的「尊王」形象就會崩盤。

## 三、關鍵人物

### 齊桓公（小白）

- **身份**：春秋五霸之首，姜姓呂氏，名小白。在位時間約 43 年。
- **此時狀態**：年過六旬，霸主生涯進入收割期，也是**最危險的轉折期**——功業頂點往往就是下墜起點。
- **核心需求**：不是更多土地或軍隊，而是**天下人對他的信服**。他要的不只是諸侯表面稱臣，而是諸侯心理上承認：齊桓公之所以是霸主，不是因為力大，而是因為義正。
- **性格側寫**：管仲稱其「淫樂不恤國政」，前期甚至有公子糾之爭時的逃亡紀錄。但他最厲害的本事是**能聽能忍**——聽管仲的話，忍自己的慾望，在關鍵時刻做出正確的政治決策。下拜受胙，正是這種「精確的自我克制」的頂級展示。

### 宰孔

- **身份**：周王室卿士，周襄王特派使者，爵位為「宰」，名「孔」。
- **任務**：代表天子賜胙（祭肉）給齊桓公，並傳達特命——「無下拜」。
- **角色定位**：他既是天子的傳聲筒，也是這場政治戲劇的**必要配角**。沒有宰孔「無下拜」的宣讀，齊桓公就沒有表演下拜的舞台。宰孔的存在，讓齊桓公的謙遜不是自發的，而是**拒絕了天子的恩典之後才發生的**——這層遞進是關鍵。

### 周襄王（姬鄭）

- **身份**：周惠王之子，前 652 年即位。
- **此時狀態**：即位第二年，地位尚未站穩。其父惠王晚年曾想廢他而立王子帶，王室内鬥餘波未平。
- **政治盤算**：賜胙只是表面，真正的政治訊號是——**我周襄王需要你齊桓公，你比我強，我給你特權。** 這是一個弱勢天子向強勢霸主示好的手法。但齊桓公不接這個示好——不是因為忠誠，是因為他算得更深。

## 四、寫作意圖：一場精密的權力表演

### 4.1 表象層

- 周天子說：「伯舅（齊桓公）年紀大了，而且有大功於天下，不用下拜了。」
- 齊桓公回答：「小白豈敢貪圖天子的特命而不下拜？只怕因此失禮墮落，給天子帶來羞辱。」
- 於是「下、拜、登、受」——整套禮儀，一個動作都沒少。

### 4.2 真相層：這不是謙虛，是精算

齊桓公堅持下拜，不是因為他尊敬周天子到這個地步。他算的是三筆帳：

| 帳本 | 內容 |
|------|------|
| **名義帳** | 接受「無下拜」，等於宣告自己凌駕於周禮之上。諸侯會想：「你齊桓公今天可以不受天子之禮，明天是不是就可以取天子之位？」——十年尊王形象，一夕瓦解。 |
| **人心帳** | 拒絕天子的特權，反過來對天子行禮，這在諸侯面前是**最昂貴的廣告**——「你們看，連天子都說我不用拜，我還是拜了。」這比打十場勝仗更能收買人心。 |
| **長遠帳** | 自己老了，管仲也老了。死後齊國能否保住霸業？不知道。但如果在天下人心中留下「齊桓公是真正守禮尊王的人」這個印象，後代諸侯就不敢明目張膽地反齊——因為反齊就是反尊王。 |

**一句話總結：齊桓公一個下拜的動作，同時完成了三件事——鞏固了自己的政治正當性、為周天子的權威做了擔保、並在諸侯心中種下了「齊國霸業有理」的心理種子。**

### 4.3 史筆之妙

《左傳》寫這段，只用了一小段對話和一個動作序列。沒有大篇幅分析，沒有讚美。但正是因為**節制**，才讓這個動作的力量完全顯現出來。左丘明懂政治：真正厲害的政治表演，不需要旁白——讀者看到齊桓公下拜的那一瞬間，就已經知道他贏了。

## 五、一句話總結

**在葵丘之會的權力巔峰上，齊桓公用一個下拜的動作，拒絕了天子給他的面子，卻賺盡了天下的人心；尊王不是他的信仰，是他最昂貴也最划算的政治投資。**
