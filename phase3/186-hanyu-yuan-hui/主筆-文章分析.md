## 1. 文章結構分析

**第一段：古之君子——責己重以周，待人輕以約**

> 「古之君子，其責己也重以周，其待人也輕以約。重以周，故不怠；輕以約，故人樂為善。」

韓愈先樹立一個理想的標準：古代的君子，對自己要求嚴格而全面，對別人要求寬鬆而簡略。因為對自己嚴格，所以不敢懈怠；因為對別人寬容，所以別人都樂意做好事。這是「不毀謗」的正面典範——一個人若忙著提升自己，哪有時間去說別人閒話？

**第二段：今之君子——責人也詳，待己也廉**

> 「今之君子則不然。其責人也詳，其待己也廉。詳，故人難於為善；廉，故自取也少。」

接著韓愈用強烈的對比，展現時代的墮落：現在的君子，對別人的缺點看得清清楚楚、吹毛求疵，對自己的缺點卻輕描淡寫、百般開脫。因為對別人苛刻，所以別人不願意做好事；因為對自己寬鬆，所以自己得到的進步很少。因果鏈完全反轉——從雙贏變成了雙輸。

**第三段：根源診斷——怠者不能修，忌者畏人修**

> 「雖然，為是者有本有原，怠與忌之謂也。怠者不能修，而忌者畏人修。」

韓愈追問：為什麼今之君子會如此？答案是「怠」與「忌」——懶惰的人自己無法進步，嫉妒的人害怕別人進步。毀謗，就是他們維持自尊的唯一武器。

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| 起 | 古之君子——責己重以周，待人輕以約（正面典範） |
| 承 | 今之君子——責人也詳，待己也廉（反面批判） |
| 轉 | 追問根源——怠者不能修，忌者畏人修（心理剖析） |
| 合 | 事修而謗興，德高而毀來——毀謗是對優秀者的自動回應 |

## 2. 寫作技巧分析

### 核心對比：責己 vs 責人

韓愈全篇的論證引擎，建立在一個精妙的對比結構上：

| 面向 | 古之君子 | 今之君子 |
|:----|---------|---------|
| **對自己** | 責己重以周——嚴格而全面 | 待己也廉——寬鬆而簡略 |
| **對別人** | 待人輕以約——寬鬆而簡略 | 責人也詳——苛刻而詳細 |
| **因果** | 不怠 → 人樂為善（雙贏） | 人難於為善 → 自取也少（雙輸） |
| **心態** | 專注修身，無暇毀人 | 忙於批評，從不反省 |
| **結果** | 己日進，人和諧 | 己不進，人遠離 |

這個對比暗藏了一個驚人的反諷：古之君子「重己輕人」，今之君子「重人輕己」——表面上只調換了兩個字的位置，卻決定了完全相反的人生格局。

### 修辭手法

| 手法 | 原文 | 效果 |
|:----|------|------|
| 對比 | 古之君子 vs 今之君子 | 善惡分明，無須多言 |
| 重複 | 「重以周……輕以約」兩次出現（正面與反面） | 鏡像結構，強化印象 |
| 因果遞進 | 重以周→不怠→不修；輕以約→人樂為善→畏人修 | 層層深入，無可反駁 |
| 設問自答 | 為是者有本有原，怠與忌之謂也 | 追根究底，直指人心 |
| 對偶 | 「事修而謗興，德高而毀來」 | 金句濃縮，傳誦千古 |

### 最精彩的論證設計——因果鏈的雙面鏡

韓愈用同一套因果邏輯，分別套在古之君子和今之君子身上：

- **正面**：責己重以周 → 不怠 → 己之修養日進
- **正面**：待人輕以約 → 人樂為善 → 人際和諧
- **反面**：待己廉 → 自取也少 → 己之修養停滯
- **反面**：責人詳 → 人難於為善 → 人際緊張

同一套公式，因為方向的改變，得出完全相反的結果。讀者無法否認這個邏輯，只能承認：選擇「重己輕人」還是「重人輕己」，決定了我們成為什麼樣的人。

## 3. 佳句賞析

### 「古之君子，其責己也重以周，其待人也輕以約。」

古代的君子，要求自己嚴格而全面，要求別人寬鬆而簡略。這是全文的總綱，也是韓愈心目中君子修身的最高境界。十二個字，講完了為人處世的全部智慧——對自己狠，對別人寬。

### 「今之君子則不然。其責人也詳，其待己也廉。」

現在的君子卻不是這樣——他們要求別人苛刻詳細，要求自己寬鬆簡略。韓愈只換了「己」和「人」的主客位置，就完成了全文最有力的一擊。讀到這裡，沒有人能不心虛。

### 「怠者不能修，而忌者畏人修。」

懶惰的人自己無法進步，嫉妒的人害怕別人進步。十個字，講完了人類毀謗行為的全部心理根源。懶惰是消極的不作為，嫉妒是積極的攻擊——先有怠，後有忌；怠是根，忌是果。

### 「事修而謗興，德高而毀來。」

事情做好了，毀謗就跟著來了；品德高尚了，攻擊也就跟著來了。這是全文最殘酷的真理——你的優秀，本身就是對他人的無言指責。毀謗，是弱者對強者最省力的攻擊。

## 4. 白話改編

古代的君子，要求自己嚴格而全面，要求別人寬鬆而簡略。因為對自己嚴格，所以不敢懈怠；因為對別人寬容，所以別人都樂意做好事。

現在的君子卻不是這樣。他們要求別人苛刻詳細，要求自己寬鬆簡單。因為對別人苛刻，所以別人不願意做好事；因為對自己寬鬆，所以自己得到的進步很少。

雖然如此，這樣做的根源在於「怠」和「忌」——懶惰的人，自己無法進步；嫉妒的人，害怕別人進步。懶惰的人不想努力，於是貶低努力的人來維持自尊；嫉妒的人看到別人比自己好，就受不了，於是用毀謗來拉低別人。

如果你不懶惰，不嫉妒，專注於提升自己——那你也許做不到不被人毀謗，但你至少不會成為那個毀謗別人的人。
