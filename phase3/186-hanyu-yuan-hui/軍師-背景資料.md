# 〈原毀〉背景資料

> **軍師分析 · 古文觀止卷七 · 唐文 · 韓愈**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《韓昌黎集》——韓愈詩文合集，由其門人李漢編纂，共四十卷；〈原毀〉為韓愈論說文的代表作之一，與〈原道〉〈原性〉〈原人〉〈原鬼〉合稱「五原」 |
| **選集** | 《古文觀止》卷七·唐文——韓愈入選《古文觀止》共四篇（〈原道〉〈師說〉〈進學解〉〈送李愿歸盤谷序〉），〈原毀〉位列其中，是其「原」體系列的重要篇章 |
| **體裁** | 論說文（「原」體）——「原」為推究本源之意——韓愈以「原毀」為題——推究「毀謗」這種行為的根源——分析人性中的「怠」與「忌」——並提出「責己重以周，待人輕以約」的君子修養之道 |
| **時代** | 中唐（約西元 9 世紀初）——安史之亂後——士風敗壞——朋黨之爭日趨激烈——文人相輕——互相詆毀之風盛行——韓愈正是在這樣的環境中——寫下這篇針砭時弊的文章 |
| **核心命題** | 為什麼人總是喜歡貶低別人？——「事修而謗興，德高而毀來」——韓愈診斷出毀謗的根源在於「怠」與「忌」——懶惰的人不希望看到別人勤奮——平庸的人嫉妒別人的才華——於是——毀謗——成了人類最普遍的惡習 |
| **關鍵人物** | 韓愈（唐代古文運動領袖）、古代的君子（「責己重以周，待人輕以約」的典範——韓愈以此對比當代的風氣墮落） |
| **文本重點** | 韓愈在本文中——以「古之君子」與「今之君子」的對比——展開論證——古代的君子——對自己要求嚴格而全面——對別人要求寬鬆而簡略——所以能不斷進步——並且與人和睦相處——現在的君子——恰恰相反——對別人的缺點看得清清楚楚——對自己的缺點輕描淡寫——所以——整天忙著批評別人——而從不反省自己——韓愈進一步追問——為什麼會這樣？——答案是——「怠者不能修——而忌者畏人修」——懶惰的人——自己不想進步——也怕別人進步——嫉妒的人——看到別人比自己好——就受不了——於是——「毀」——就成了他們——唯一的武器 |
| **一段話總結** | **韓愈在一千兩百年前——就診斷出了人類社交媒體時代的病灶——我們之所以熱衷於批評別人——不是因為我們真的比別人好——而是因為我們無法面對自己的不足。〈原毀〉——是一面鏡子——照出了每個人心中——那個「怠」和「忌」的影子——它告訴我們——每一次——當我們想要貶低一個人的時候——先看看自己——是不是——那個懶惰的人——那個——嫉妒的人。** |

## 二、時代背景——中唐的士風問題

### 2.1 朋黨之爭與文人相輕

〈原毀〉的寫作背景——與中唐的政治生態——密不可分：

| 面向 | 說明 |
|------|------|
| **政治** | 安史之亂後——唐帝國中央權力衰弱——朝中派系林立——文人集團之間的鬥爭——愈演愈烈——「牛李黨爭」——是唐代後期最著名的朋黨之爭——持續近四十年——文人們為了政治利益——互相攻訐——毀謗之風——蔚為潮流 |
| **科舉** | 唐代科舉制度——是文人主要的進身之階——但科舉競爭激烈——錄取率極低——「五十少進士」——五十歲考中進士——都算年輕——在這種高壓競爭下——文人互相傾軋、貶低對手——成為常態——「同行是冤家」——在唐代——已經不是一句話——而是活生生的現實 |
| **士風** | 韓愈所處的中唐——士風敗壞——知識分子——不再以修身為本——而是以「求名」為務——為了名聲——可以互相吹捧——也可以互相詆毀——一切——都取決於——利益——韓愈對此——深感憂慮 |

> **軍師按**：〈原毀〉不是一篇「純理論」文章——它是韓愈——在官場中被貶、被罵、被攻擊之後——寫下的「切膚之痛」。韓愈一生——屢遭毀謗——他的古文運動——他的排佛主張——都讓他成為眾矢之的——所以——他比誰都清楚——「毀謗」——是怎麼發生的——以及——為什麼會發生。

### 2.2 「原」體系列——韓愈的思想體系

〈原毀〉是韓愈「五原」系列中的一篇——其他四篇分別是：

| 篇目 | 核心論述 |
|------|---------|
| 〈原道〉 | 推究儒家之道的本源——批判佛老——建立「道統」說——是韓愈的思想總綱 |
| 〈原性〉 | 探討人性的本質——提出「性三品說」——上品善、中品可導、下品惡——繼承並發展了董仲舒的人性論 |
| 〈原人〉 | 討論「人」的定義與人倫關係——強調人與動物的區別——在於人懂得仁義禮智 |
| 〈原鬼〉 | 辨析鬼神有無——韓愈認為——鬼神確實存在——但——「不明」——人不應為了祈福而事鬼——儒家應以「修身」為本 |
| 〈原毀〉 | 推究毀謗的根源——診斷人性中的「怠」與「忌」——提出君子修身之道——「責己重以周，待人輕以約」 |

> **軍師按**：「五原」——是韓愈思想體系的五根支柱——〈原道〉——講「道」是什麼——〈原性〉——講「人性」是什麼——〈原人〉——講「人」是什麼——〈原鬼〉——講「鬼神」是什麼——〈原毀〉——講「人的惡」是怎麼來的——五篇合在一起——就是韓愈對世界——最完整的——解釋。其中——〈原毀〉——最接地氣——最貼近日常生活——因為——毀謗——是人人都能感受到的——惡。

## 三、文本結構——「原」毀的三層剖析

### 3.1 全文結構一覽

〈原毀〉篇幅不長——約五百餘字——但結構極其緊湊——可分為三大層次：

| 層次 | 核心內容 | 論證策略 |
|------|---------|---------|
| **第一層：古之君子的榜樣** | 古代的君子——「責己重以周——待人輕以約」——對自己——嚴格——全面——對別人——寬容——簡略——用這種態度——不斷提升自己——與人和諧相處 | 正面典範——先樹立一個理想的標準——作為對比參照 |
| **第二層：今之君子的墮落** | 現在的君子——「責人也詳——待己也廉」——對別人——吹毛求疵——對自己——輕描淡寫——整天忙著批評別人的缺點——從不反省自己的不足 | 反面批判——與古代的君子——形成強烈對比——凸顯時代的墮落 |
| **第三層：根源診斷——「怠」與「忌」** | 韓愈追問——為什麼會這樣？——答案是——「怠者不能修——而忌者畏人修」——懶惰的人——自己不想進步——還怕別人進步——嫉妒的人——看到別人好——心裡就不舒服——於是——毀謗——就成了他們——唯一的「武器」 | 心理分析——從人性的根源——解釋毀謗行為——為何如此普遍——如此難以根除 |

### 3.2 開篇立論——「責己重以周，待人輕以約」

> 「古之君子，其責己也重以周，其待人也輕以約。重以周，故不怠；輕以約，故人樂為善。」

— 翻譯：古代的君子——他們要求自己——嚴格而全面——他們要求別人——寬鬆而簡略——因為對自己要求嚴格——所以不敢懈怠——因為對別人要求寬鬆——所以別人都樂意做好事。

| 層次 | 分析 |
|------|------|
| **「責己重以周」——對自己狠** | 韓愈說——古代君子的第一條原則——是「嚴於律己」——不是偶爾嚴格——而是「重以周」——既嚴格——又全面——涵蓋生活的各個方面——睡覺前——要反省今天有沒有做錯——起床後——要想今天該做什麼——這種態度——是「修己」的基礎 |
| **「待人輕以約」——對別人寬** | 古之君子的第二條原則——是「寬以待人」——不是假裝寬容——而是「輕以約」——既寬鬆——又簡略——別人的小缺點——不計較——別人的小錯誤——不追究——這樣——別人才願意——與你交往——才願意——做好事 |
| **因果鏈** | 韓愈用因果關係——把這兩條原則——串了起來——「重以周——故不怠」——因為對自己嚴格——所以不敢懈怠——「輕以約——故人樂為善」——因為對別人寬容——所以別人樂意做好事——這是一個——雙贏的——修身之道 |

> **軍師按**：韓愈這八個字——「責己重以周，待人輕以約」——是整篇文章的核心——也是他所認為的——君子修身的——最高境界。一個人能做到這八個字——就不會——去毀謗別人——因為——你忙著提升自己——哪有時間——去說別人閒話？——而別人——因為感受到你的寬容——也會用同樣的態度——對待你——這是一個——正向循環。

### 3.3 反面批判——「責人也詳，待己也廉」

> 「今之君子則不然。其責人也詳，其待己也廉。詳，故人難於為善；廉，故自取也少。」

— 翻譯：現在的君子——恰恰相反——他們要求別人——苛刻而詳細——他們要求自己——寬鬆而簡單——因為對別人苛刻——所以別人不願意做好事——因為對自己寬鬆——所以自己得到的進步——很少。

| 層次 | 分析 |
|------|------|
| **「責人也詳」——對別人吹毛求疵** | 韓愈說——今之君子的第一條問題——是「對別人太苛刻」——別人做了一百件好事——沒有表揚——別人不小心犯了一個小錯——立刻揪出來——大肆批評——這種態度——讓別人——不敢做事——不願做事——因為——做得越多——錯得越多——不如什麼都不做 |
| **「待己也廉」——對自己太寬容** | 今之君子的第二條問題——是「對自己太寬容」——自己做錯了事——找各種理由為自己開脫——「我不是故意的」「環境造成的」「大家都這麼做」——從來不認真反省——不認真改過——所以——「自取也少」——自己從中得到的進步——非常有限 |
| **因果鏈的反轉** | 韓愈用同樣的因果關係——反過來證明——今之君子的行為——導致了——「人難於為善」——「自取也少」——對別人苛刻——別人就不願意做好事——對自己寬容——自己就沒有進步——這是一個——雙輸的——墮落之道 |

> **軍師按**：韓愈用「古之君子」和「今之君子」的對比——把文章推向了一個——無法迴避的命題——「我們——到底想當哪一種人？」——古之君子——嚴以律己——寬以待人——人際關係好——自己也不斷進步——今之君子——寬以律己——嚴以待人——人際關係差——自己也原地踏步——選擇——其實很簡單——但大多數人——還是選擇了——後者——為什麼？——因為——「責己」太痛苦了——而「責人」——太爽了。

### 3.4 根源診斷——「怠者不能修，忌者畏人修」

> 「雖然，為是者有本有原，怠與忌之謂也。怠者不能修，而忌者畏人修。」

— 翻譯：雖然如此——但人之所以這樣做——是有根源的——那就是——「怠」和「忌」——懶惰的人——自己無法進步——嫉妒的人——害怕別人進步。

| 層次 | 分析 |
|------|------|
| **「怠」——懶惰** | 「怠」——就是懶惰——懶惰的人——自己不想努力——不想進步——但——他又無法面對——自己「不如別人」的現實——於是——他選擇了一種——最省力的方式——來維持自己的自尊——「毀謗」——「他之所以比我好——不是因為他努力——而是因為他運氣好——或者——他用了不正當的手段」——這種心理——讓懶惰的人——可以心安理得地——繼續懶惰 |
| **「忌」——嫉妒** | 「忌」——就是嫉妒——嫉妒的人——比懶惰的人——更危險——懶惰的人——只是「不修」——嫉妒的人——是「畏人修」——他不僅自己不想進步——而且——害怕——害怕別人進步——因為——別人的進步——會襯托出——他的不足——所以——他必須——用毀謗——來阻止別人前進——把別人拉下來——他就安全了——這就是——「木秀於林——風必摧之」——的——心理根源 |
| **「怠」與「忌」的關係** | 韓愈把「怠」和「忌」——放在一起——不是偶然的——它們——是一體兩面——「怠」——是消極的——不作為——「忌」——是積極的——攻擊別人——先有「怠」——自己不想進步——然後——為了掩蓋自己的「怠」——就產生了「忌」——去攻擊那些進步的人——所以——「怠」是根——「忌」是果——解決「忌」——要先解決「怠」——要讓自己——不懶惰——不躺平——不斷進步——這樣——你就不需要——用毀謗——來證明自己的價值了 |

> **軍師按**：韓愈對「怠」與「忌」的診斷——是整篇文章——最精彩的部分——他不是在「說教」——他是在「解剖」——他把人性中最醜陋的部分——拿到陽光下——告訴你——「看——這就是你——這就是為什麼——你總是忍不住——要去說別人的壞話」——這種直面人性的勇氣——讓〈原毀〉——超越了一篇普通的「勸善」文章——成為了一篇——「人性診斷書」——每一個讀到它的人——都會在某一瞬間——感到——「他說的就是我」——這——才是真正的——文學。

## 四、關鍵人物

### 4.1 韓愈——被毀謗了一輩子的思想家

| 項目 | 內容 |
|------|------|
| **身份** | 唐代文學家、思想家、政治家——字退之——自稱「昌黎韓愈」——世稱「韓昌黎」——生於唐代宗大曆三年（768 年）——卒於唐穆宗長慶四年（824 年）——五十七歲——諡號「文」——故又稱「韓文公」 |
| **與毀謗的淵源** | 韓愈一生——都在被毀謗——他推行古文運動——被嘲笑為「好為人師」——他上〈諫迎佛骨表〉——被貶潮州——朝中同僚落井下石——他提倡儒學、排佛排老——被視為「迂闊」「不識時務」——可以說——〈原毀〉——不是韓愈在書齋中——憑空想出來的——而是他在——被毀謗的風暴中——寫下的——「自白書」——他比誰都清楚——「怠」和「忌」——是什麼滋味——因為——他自己——就是那個——被「怠」者和「忌」者——圍攻的人 |
| **〈原毀〉的獨特地位** | 與〈原道〉的「戰鬥姿態」不同——〈原毀〉——是韓愈——較為「平和」的一篇文章——他沒有罵人——沒有憤怒——他只是——冷靜地——分析了——一種普遍的人性弱點——然後——提出了一個解決方案——「責己重以周——待人輕以約」——這種語氣——與〈原道〉中的「人其人——火其書——廬其居」——截然不同——展現了——韓愈——另一個——更為——「成熟」——和——「圓融」——的面貌 |

**軍師評**：韓愈寫〈原毀〉的時候——想的不是「我要報復那些罵我的人」——他想的是——「為什麼人會罵人？——能不能——讓大家——不要這樣？」——這是一個——被傷害過的人——寫下的——最溫柔的文章——他用理性——來對抗——最不理性的攻擊——用內省——來回應——最惡毒的毀謗——這是韓愈——比他的論敵們——高出一個層次的地方。

### 4.2 古之君子——韓愈的理想人格

〈原毀〉中的「古之君子」——不是具體的某個人——而是韓愈筆下的「理想人格」——綜合了堯、舜、禹、湯、文、武、周公、孔子、孟子的修身精神：

| 項目 | 內容 |
|------|------|
| **特徵一：嚴以律己** | 古之君子——「責己重以周」——對自己的缺點——不回避——不放過——時刻反省——時刻改過——「吾日三省吾身」（曾子）——「見賢思齊焉——見不賢而內自省也」（孔子）——這是儒家修身傳統的核心 |
| **特徵二：寬以待人** | 古之君子——「待人輕以約」——對別人的缺點——不過分追究——不吹毛求疵——「君子和而不同」（孔子）——「躬自厚而薄責於人」（孔子）——這是儒家處世哲學的精髓 |
| **特徵三：不怠不忌** | 古之君子——既不懶惰——也不嫉妒——他們專注於自己的修養——「苟日新——日日新——又日新」（湯之盤銘）——他們為別人的進步感到高興——「君子成人之美——不成人之惡」（孔子）——這種心態——是毀謗——永遠無法侵蝕的 |

**軍師評**：韓愈筆下的「古之君子」——本質上——是他自己——想要成為的樣子——也是他——希望所有讀書人——都能成為的樣子。但韓愈沒有天真到——以為所有人都能做到——他寫〈原毀〉——不是為了——讓所有人——都變成聖人——他只是希望——當你想毀謗一個人的時候——至少——可以先想一想——「我這樣做——是因為——我真的比他好——還是——因為——我懶惰——我嫉妒？」——有這一問——就已經——勝利了一半。

## 五、核心分析——為什麼〈原毀〉在今天仍然讓人臉紅？

### 5.1 「事修而謗興，德高而毀來」——跨越千年的真理

〈原毀〉的結論——用八個字——就可以概括——「事修而謗興——德高而毀來」——事情做好了——毀謗就跟著來了——品德高尚了——攻擊也就跟著來了——為什麼？——因為——你的進步——就是對他人的——「無言指責」——當你勤奮學習的時候——那些懶惰的人——就會感到不安——當你品德高尚的時候——那些平庸的人——就會感到羞愧——為了消除這種不安和羞愧——他們唯一的辦法——就是——把你拉下來——「毀謗」——是弱者對強者——最省力的攻擊。

| 面向 | 分析 |
|------|------|
| **古代的例證** | 屈原被放逐——是因為「眾人皆醉我獨醒」——司馬遷遭受宮刑——是因為他為李陵說了公道話——蘇軾被貶黃州——是因為烏臺詩案——每一個時代——最傑出的人——往往——也是被毀謗最多的人——這不是巧合——這是——人性 |
| **現代的翻版** | 社交媒體時代——「事修而謗興」——比任何時代——都更加明顯——你發了一篇好文章——下面一定有人——挑你的語病——你取得了一個成就——背後一定有人——說你是靠關係——你過得比別人好——就有人——說你「裝」——嫉妒——在今天——不需要寫匿名信——只需要——滑鼠一點——鍵盤一敲——就能完成——一次完美的——「毀謗」 |
| **不變的人性** | 韓愈說——毀謗的根源——在於「怠」與「忌」——這個診斷——跨越了一千兩百年——依然準確——為什麼？——因為——人性——沒有改變——一千兩百年前——懶惰的人——嫉妒的人——會說別人的壞話——一千兩百年後——懶惰的人——嫉妒的人——依舊會說別人的壞話——技術變了——媒體變了——但——人性——沒有變——這就是——〈原毀〉——到今天——仍然有震撼力的原因 |

> **軍師按**：〈原毀〉告訴我們一個殘酷的真相——當你被毀謗的時候——不要急著——去解釋——去反駁——去證明「我不是那樣的人」——因為——那些毀謗你的人——根本不關心——你是什麼樣的人——他們關心的——只有一件事——「為什麼你比我好？」——所以——最好的反擊——不是——「我沒有」——而是——繼續——變得更好——因為——只有——當你遠遠地——把他們甩在身後的時候——毀謗——才會消失——取而代之的——是——仰望。

### 5.2 「怠者不能修，忌者畏人修」——韓愈的人性診斷學

韓愈對人性——有著極其深刻的——洞察——他不是一個——只會說「你要善良」的——道德家——他是一個——懂得——「人為什麼不善良」的——心理學家：

| 層次 | 分析 |
|------|------|
| **「怠」的心理機制** | 懶惰——不是「不想做事」——而是——「不想面對——自己不如別人」的現實——懶惰的人——需要的——不是努力——而是——一個「為什麼我不需要努力」的——藉口——「他比我好——但他用了不正當的手段」——這個藉口——讓懶惰的人——可以在不努力的情況下——保持自尊——所以——他必須——不斷地說——別人的壞話——來維持——這個藉口 |
| **「忌」的心理機制** | 嫉妒——比懶惰——更複雜——嫉妒的人——不是不努力——而是——「努力了——卻追不上」——當一個人——付出了很多——卻發現——別人輕鬆地——超過了他——他的內心——就會產生——一種扭曲的——恨意——「憑什麼是他？——不是我？」——這種恨意——需要一個出口——而——毀謗——就是最方便的出口 |
| **「怠」與「忌」的現代解讀** | 在現代社會——「怠」和「忌」——常常交織在一起——「內卷」——讓每個人都很焦慮——焦慮的人——更容易——去嫉妒——已經成功的人——「躺平」——讓很多人——選擇了放棄——放棄的人——更容易——去毀謗——那些還在努力的人——韓愈在一千兩百年前——就預見了——「內卷」和「躺平」——這兩個現代現象——的心理根源——只不過——他用的是——「怠」和「忌」——這兩個字 |

> **軍師按**：韓愈的偉大——不在於他「解決了」毀謗的問題——而在於他「說明白了」毀謗是怎麼發生的——他告訴你——「他不是恨你——他是恨——他自己」——一旦你理解了這一點——你就——不再需要——為毀謗——感到憤怒——和——受傷——因為——那些毀謗你的人——真正恨的——不是你——是——站在你面前——卻——不如你的——那個——自己。

### 5.3 〈原毀〉的文學成就——簡潔中的力量

〈原毀〉——是韓愈「古文」理論的——完美實踐：

| 面向 | 分析 |
|------|------|
| **語言風格** | 〈原毀〉全文約五百餘字——極其簡潔——沒有一個多餘的字——每一個字——都精準到位——「責己重以周——待人輕以約」——十二個字——講完了整篇文章的主旨——「怠者不能修——忌者畏人修」——十個字——講完了毀謗的心理根源——這種「力透紙背」的簡潔——正是韓愈古文運動——所追求的——「文以載道」——的最高境界 |
| **結構技巧** | 〈原毀〉的結構——極其工整——先古之君子（正面）——再今之君子（反面）——再追問根源（深入）——層層遞進——邏輯嚴密——這種「正—反—合」的結構——後來——成為中國論說文的——經典範式 |
| **情感克制** | 與〈原道〉的慷慨激昂不同——〈原毀〉的語氣——非常克制——韓愈沒有罵人——沒有發怒——他只是——冷靜地——分析——解剖——這讓文章——顯得——更加——有力量——因為——你不是被「罵醒」的——你是被「說服」的——而說服——比罵——持久得多 |

> **軍師按**：〈原毀〉——是韓愈「五原」中——最短的一篇——但——也是最有力的一篇——有人說——〈原道〉——是韓愈的「大砲」——轟向佛老——聲音大——威力大——而〈原毀〉——是韓愈的「手術刀」——劃開人性——精準——細緻——不流血——但——痛——一針見血——讓你——無處可逃。

## 六、軍師核心分析——「毀謗」是人類最古老的社交媒體

### 6.1 〈原毀〉與社交媒體時代——為什麼韓愈比任何時代的人都懂我們？

如果把〈原毀〉放在社交媒體時代來讀——你會發現——韓愈才是——真正——懂「互聯網」的人：

| 面向 | 〈原毀〉的洞察 | 社交媒體時代的對應 |
|------|---------------|------------------|
| **「怠」的表現** | 自己不想進步——所以貶低進步的人 | 鍵盤俠——自己不讀書——嘲笑讀書的人「裝文藝」——自己不努力——嘲笑努力的人「內卷」——自己沒成就——嘲笑有成就的人「靠運氣」 |
| **「忌」的表現** | 害怕別人進步——所以攻擊進步的人 | 網絡暴力——一個人因為表現出色——被曝光——然後——無數人——因為嫉妒——加入「圍攻」的行列——「扒歷史」「挖黑料」「人肉搜索」——目標就一個——「把他拉下來」 |
| **「毀」的傳播** | 「事修而謗興——德高而毀來」——毀謗——是對優秀者的「自動回應」 | 在社交媒體上——任何一個——表現突出的人——很快就會——迎來「反轉」「塌房」「被扒」——這不是——因為——他們真的做了什麼壞事——而是因為——「他太好了——這讓很多人——不舒服」 |
| **韓愈的解決方案** | 不要理會毀謗——專注於自己的修養——「責己重以周——待人輕以約」 | 不要和鍵盤俠爭論——繼續做你該做的事——真正優秀的人——不會因為網絡暴力——就停止進步——而——最終——時間——會證明一切 |

> **軍師按**：如果韓愈活在今天——他會是一個——社交媒體的——「終極批判者」——他會告訴你——每一次——你想要在網上——罵一個人的時候——先問自己兩個問題——「我罵他——是因為——我真的比他好——還是因為——我嫉妒他？」——「我花時間罵他——對我自己的進步——有什麼幫助？」——如果你誠實地回答——你就會發現——十次裡面——有八次——你罵人——不是因為別人錯了——而是因為——你自己——「怠」了——或者——「忌」了。這就是——〈原毀〉——留給我們——最寶貴的——禮物。

### 6.2 從「責己重以周」到「成長型思維」——〈原毀〉的現代詮釋

韓愈的「責己重以周，待人輕以約」——其實——就是——現代心理學中的——「成長型思維」：

| 面向 | 韓愈的版本 | 現代心理學的版本（卡羅爾·德韋克） |
|------|----------|--------------------------------|
| **對自己的態度** | 「責己重以周」——對自己嚴格——不放過每一個可以進步的機會——「故不怠」 | 「成長型思維」——相信能力可以通過努力提升——所以——不斷學習——不斷挑戰自己——不怕失敗 |
| **對別人的態度** | 「待人輕以約」——對別人寬容——不吹毛求疵——「故人樂為善」 | 對別人的成功——不嫉妒——而是學習——「他做到了——我也終將做到」——對別人的失敗——不嘲笑——而是理解 |
| **對比較的態度** | 不與別人比——只與自己比——「古之君子——其責己也重以周」 | 不與別人比——只與過去的自己比——「今天的我——是否比昨天的我——更好？」 |
| **對批評的態度** | 不毀謗別人——也不畏懼別人的毀謗——「事修而謗興」——這是正常的——不必在意 | 對批評——保持開放——吸收有益的——忽略惡意的——專注於自己的成長 |

> **軍師按**：韓愈在一千兩百年前——就提出了——與「成長型思維」——完全一致的——修身哲學——這告訴我們——真正偉大的思想——不需要——現代心理學的包裝——因為——它們本身——就是——真理——我們今天——用「成長型思維」——這個詞——來解釋韓愈的「責己重以周——待人輕以約」——不是為了讓韓愈——顯得更「現代」——而是為了讓我們自己——更容易——記住——和——實踐——這個——古老的——智慧。

### 6.3 〈原毀〉與〈原道〉——韓愈的「一體兩面」

把〈原毀〉和〈原道〉放在一起讀——可以看到——韓愈——這個人——的——一體兩面：

| 面向 | 〈原道〉 | 〈原毀〉 |

## 九、背景補遺

### 9.1 「毀」的雙重含義

中文的「毀」——有兩個層面的意思——一是「毀壞」——把東西弄壞——二是「毀謗」——說別人的壞話——韓愈在〈原毀〉中——用的——是第二個意思——但他同時也暗示——「毀謗」——本質上——就是——「毀壞」——只不過——毀壞的不是物質——而是——名譽——和——關係。一個人——用毀謗——「毀」了別人的名聲——也——「毀」了自己的人品——所以——「毀」——到最後——都是——雙輸的。

### 9.2 〈原毀〉的寫作時間

〈原毀〉的具體寫作時間——學術界——沒有確切定論——一般認為——寫於韓愈——擔任監察御史——或被貶陽山之後（約西元 800–805 年）——這個時期——韓愈——在政治上——遭受打擊——在官場上——屢遭誹謗——他的「古文運動」——也受到——很多人的——嘲笑——和——攻擊——正是在這種——內外交困的——情境下——他寫下了〈原毀〉——為自己——也為所有——被毀謗的人——寫下了一篇——「心理自癒」的——文章。

### 9.3 〈原毀〉對後世的影響

〈原毀〉的影響——雖然不如〈原道〉——那樣巨大——但在中國知識分子的——修身傳統中——佔據了——非常重要的位置：

| 面向 | 說明 |
|------|------|
| **對宋代理學的影響** | 宋代的理學家——如程頤、朱熹——在談論「修身」時——經常引用韓愈的「責己重以周——待人輕以約」——這個原則——成為宋明理學——「克己復禮」——的重要內涵 |
| **對士人修養的影響** | 在後代——「責己重以周——待人輕以約」——成為——中國知識分子的——座右銘——它告訴讀書人——不要把時間——浪費在——批評別人上——而應該——專注於——提升自己——「靜坐常思己過——閒談莫論人非」——這句流傳更廣的箴言——其精神內核——正是來自——韓愈的〈原毀〉 |
| **對現代人的啟示** | 在社交媒體時代——〈原毀〉——有了——全新的——現實意義——它告訴我們——不要做——那個「怠」和「忌」的人——也不要在意——那些「怠」和「忌」的人——對自己——嚴格一點——對別人——寬容一點——然後——繼續——走自己的路——讓別人——說去吧——這就是——韓愈——留給我們——最簡單——也最不容易做到的——人生智慧 |
