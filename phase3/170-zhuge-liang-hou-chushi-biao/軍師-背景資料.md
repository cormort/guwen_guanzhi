> **軍師分析 · 古文觀止卷六 · 後漢文 · 諸葛亮**

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《三國志·蜀書·諸葛亮傳》裴松之注引《漢晉春秋》——陳壽《三國志》未收錄本文，係裴松之作注時從《漢晉春秋》輯出 |
| **選集** | 《古文觀止》卷六·後漢文類——卷六壓軸名篇之一，與〈前出師表〉並稱「前後出師表」 |
| **體裁** | 表文——臣子呈給君主的奏章，但本質上是戰後檢討與再動員的辯護書 |
| **時代** | 三國時期蜀漢建興六年（西元 228 年），第一次北伐因馬謖失街亭而失敗後 |
| **核心命題** | 知其不可而為之——「鞠躬盡瘁，死而後已」——一個深知自己時日無多的老人，用最後的力氣說服一個小孩繼續支持他的戰爭 |
| **關鍵人物** | 諸葛亮（蜀漢丞相，時年 48 歲）、劉禪（後主，時年 22 歲） |
| **文本重點** | 第一次北伐因馬謖失街亭而全盤失敗，朝中反對北伐的聲浪高漲。諸葛亮上〈後出師表〉為自己辯護。全文以「六不解」為核心論證框架——列出六個「為什麼必須繼續北伐」的理由。風格與〈前出師表〉截然不同：前者是臨行囑咐，冷靜克制；後者是戰敗後的檢討與再出發，急切悲壯。「鞠躬盡瘁，死而後已」即出於此篇 |
| **真偽爭議** | 學術界對〈後出師表〉的真偽有重大爭議。陳壽《三國志》未收錄此篇，裴松之注引《漢晉春秋》時特別註明「此表亮集所無，出張儼《默記》」——即諸葛亮的個人文集《諸葛亮集》中並無此篇，係張儼《默記》所載。反對者認為文中列舉的軍力數字與《三國志》記載不符，且語氣與〈前出師表〉差異過大；支持者則認為其內容與當時政治局勢相符，且裴松之慎重收錄自有道理。無論真偽，此篇的文化影響力已超越考據爭議——「鞠躬盡瘁，死而後已」八個字已成為中國文化中忠誠的最高境界 |
| **一句話總結** | **〈後出師表〉是一個深知自己時日無多的老人，用最後的力氣說服一個小孩繼續支持他的戰爭——他知道自己可能看不到勝利，但他必須讓劉禪相信這條路是對的。** |

## 二、時代背景——街亭之敗與蜀漢的轉捩點

### 2.1 第一次北伐：從輝煌開局到全面潰敗

西元 228 年春，諸葛亮率軍從漢中出發，開始了第一次北伐。這是一次精心策劃的軍事行動——諸葛亮用聲東擊西之計，派趙雲、鄧芝率疑兵出斜谷（今陝西眉縣），吸引曹魏主力。自己則率主力出祁山（今甘肅禮縣），直取隴右。

戰役初期，蜀軍取得驚人進展——天水、南安、安定三郡相繼叛魏降蜀，關中震動，魏明帝曹叡親征長安，派大將軍曹真、張郃迎戰。然而——關鍵時刻——諸葛亮選擇了一個錯誤的將領守衛戰略要地街亭。

| 面向 | 說明 |
|------|------|
| **馬謖其人** | 馬謖是蜀漢名士馬良之弟，以才學謀略著稱。諸葛亮對他極為器重，經常與他通宵談論軍國大事。劉備臨終前曾告誡諸葛亮：「馬謖言過其實，不可大用。」但諸葛亮沒有聽——他認為馬謖是難得的軍事人才 |
| **街亭之戰** | 諸葛亮派馬謖為前鋒，率兩萬餘人守衛街亭（今甘肅秦安東北），阻擋魏將張郃。馬謖違反諸葛亮的部署——不在當道紮營，反而上山據守，理由是「居高臨下，勢如破竹」。副將王平多次勸諫，馬謖不聽。張郃到達後，切斷山上水源，蜀軍缺水大亂。馬謖棄軍逃亡，全軍潰敗 |
| **戰敗後果** | 街亭一失，天水、南安、安定三郡重新被魏軍收復。趙雲、鄧芝的疑兵也被曹真擊敗。蜀軍全面撤退，第一次北伐以失敗告終。諸葛亮退兵漢中後，揮淚斬馬謖——並上疏劉禪自貶三級（從丞相降為右將軍，行丞相事） |

> **軍師按**：第一次北伐的失敗——不僅是一次軍事失敗——更是一次心理打擊。諸葛亮用了五年時間準備北伐——他選了最好的時機（曹魏內部不穩、曹叡剛即位不久）——他用了最精妙的計策（聲東擊西）——他幾乎成功了——天水三郡投降的消息傳來時——諸葛亮可能真的以為——復興漢室是有希望的。然後——所有這一切——都被馬謖一個人毀了。〈後出師表〉就是在這種背景下寫的——諸葛亮不僅要面對外部的失敗——還要面對內部的質疑——「你到底行不行？」

### 2.2 朝中反對北伐的聲浪

街亭戰敗後，蜀漢朝中出現了強烈的反對北伐的聲音。這些反對者的理由很充分：

| 反對理由 | 具體內容 |
|-----------|----------|
| **國力不支** | 蜀漢僅有益州一州之地——人口約百萬——而曹魏擁有整個北方中國——人口約四千萬。連續戰爭必然耗盡國力 |
| **戰略不可行** | 從蜀中北伐曹魏——需要穿越秦嶺——運糧極其困難——歷史上幾乎沒有從四川成功北定中原的先例 |
| **時機已失** | 第一次北伐是最好的機會——曹魏沒有防備——但我們失敗了——現在曹魏已經加強了防禦——更難成功了 |
| **將領不足** | 關羽、張飛、趙雲等一批老將已陸續去世——蜀漢缺乏能夠獨當一面的大將——馬謖的失敗就是證明 |

這些反對意見——從軍事角度——都是有道理的。但諸葛亮面臨的問題——不僅僅是軍事問題——更是政治問題和心理問題。〈後出師表〉就是諸葛亮對這些反對意見的——全面反擊。

> **軍師按**：〈後出師表〉最特別的地方——是它與〈前出師表〉的語氣完全不同。〈前出師表〉是諸葛亮對劉禪說「聽話」。〈後出師表〉是諸葛亮對所有人說「聽我解釋」。前者是父親在教孩子——後者是將領在為自己辯護。前者從容——後者急切。前者冷靜——後者悲壯。這是因為——〈前出師表〉寫在出發前——諸葛亮還有信心。而〈後出師表〉寫在失敗後——諸葛亮已經不確定——自己還能撐多久。

## 三、文本結構與核心論證——「六不解」

### 3.1 何謂「六不解」？

〈後出師表〉的核心論證框架——是所謂的「六不解」——諸葛亮列出六個「為什麼必須繼續北伐」的理由。之所以稱為「不解」——意思是「我想不通為什麼有人反對北伐」——其實諸葛亮完全想得通——他只是用這種方式來反駁朝中的反對意見。

**第一不解：為何坐以待斃？**

>「今賊適疲於西，又務於東，兵法乘勞，此進趨之時也。」

諸葛亮說：曹魏在西邊（關中）剛剛被我們打了一仗——雖然我們輸了——但他們也被消耗了——同時他們在東邊（合肥）也要對付東吳——現在正是我們趁他們疲憊的時候出擊的時機。如果錯過這個時機——等他們休息好了——我們就更沒有機會了。

**第二不解：為何高估敵人？**

>「高帝明並日月，謀臣淵深，然涉險被創，危然後安。今陛下未及高帝，謀臣不如良、平，而欲以長計取勝，坐定天下，此臣之未解二也。」

諸葛亮用劉邦的例子來反駁。他說：漢高祖劉邦——英明如日月——手下有張良、陳平這樣的謀臣——仍然多次經歷危險、多次被打敗——最後才安定天下。而我們——陛下不如劉邦——謀臣不如張良、陳平——卻想安安穩穩地坐著就能打敗曹魏？這不可能。

**第三不解：為何懼怕失敗？**

>「劉繇、王朗各據州郡，論安言計，動引聖人，群疑滿腹，眾難塞胸，今歲不戰，明年不征，使孫策坐大，遂并江東。」

諸葛亮舉了劉繇和王朗的例子——這兩個人在江東割據——天天開會討論——引經據典——就是不打仗——結果眼睜睜看著孫策一天天坐大——最後被孫策消滅。諸葛亮的意思是：如果我們現在不動手——等曹魏消化了力量——我們的下場就和劉繇、王朗一樣。

**第四不解：為何不承認實力差距？**

>「曹操之智計，殊絕於人，其用兵也，仿佛孫、吳，然困於南陽，險於烏巢，危於祁連，偪於黎陽，幾敗北山，殆死潼關，然後偽定一時耳。況臣駑下，何能必勝？」

這是全篇最震撼的一段話。諸葛亮說：曹操——用兵如孫武、吳起——但他還是多次打敗仗——在南陽被困——在烏巢冒險——在祁連遇險——在黎陽受壓——在北山幾乎全軍覆沒——在潼關差點被殺死——他打了無數敗仗——最後才勉強成功。而曹操都這樣——我諸葛亮——一個「駑下」之人——一個平庸的人——怎麼可能保證每次都贏？——連曹操都會失敗，何況是我？

> **軍師按**：這一段是全篇最悲壯的論證。諸葛亮不是在說「我們會贏」——他是在說「我不保證能贏，但我必須打」。他拿曹操來做對比——不是在貶低自己——而是在告訴所有人：失敗是正常的——勝利才是僥倖。連曹操都輸了那麼多次——你為什麼要求我諸葛亮——一個遠不如曹操的人——一次都不能輸？

**第五不解：為何只求偏安？**

>「今民窮兵疲，而事不可息。事不可息，則住與行，勞費正等。而不及早圖之，欲以一州之地，與賊持久，此臣之未解五也。」

諸葛亮說：我們現在是「民窮兵疲」——百姓窮、士兵累。但戰爭是不可避免的——只要我們還想復興漢室——就必須打仗。既然一定要打——那防守和進攻——消耗的資源是一樣的。與其被動防守——不如主動出擊。這個論證非常巧妙——它反駁了「偏安論」：偏安不是選項——因為曹魏最終會來消滅我們——與其等著被打——不如主動去打。

**第六不解：為何忽視不戰之危？**

>「夫難平者，事也。昔先帝敗軍於楚，當此時，曹操拊手，謂天下已定。然後先帝東連吳越，西取巴蜀，舉兵北征，夏侯授首，此操之失計，而漢事將成也。然後吳更違盟，關羽毀敗，秭歸蹉跌，曹丕稱帝。凡事如是，難可逆見。臣鞠躬盡瘁，死而後已。至於成敗利鈍，非臣所能逆睹。」

最後一段——諸葛亮不再辯論——他說了整篇文章最重要的一句話：「鞠躬盡瘁，死而後已」——我願意付出一切——直到我死的那一天——至於成敗——那不是我能預見的。然後他用了劉備的經歷來證明「世事難料」：劉備曾經幾乎一無所有——曹操以為天下已定——但劉備後來聯吳抗曹——奪取了荊州和益州——打敗了夏侯淵——形勢一片大好。然後呢？東吳背盟——關羽被殺——劉備慘敗——曹丕稱帝。天下事——誰說得準？

> **軍師按**：「鞠躬盡瘁，死而後已」——這八個字——是〈後出師表〉最偉大的貢獻——也是諸葛亮對自己一生的最終定義。他不是在說一個口號——他是在說一個事實：我已經決定了——無論結果如何——我都會堅持到最後一口氣。這種「不問成敗，只問本心」的態度——正是儒家的「義命分立」——義（我該做什麼）由我決定——命（結果如何）由天決定。這是一個真正成熟的人的選擇——他不再被結果捆綁——他只看自己的本心。

## 四、關鍵人物

### 4.1 諸葛亮——蜀漢的靈魂

| 項目 | 內容 |
|------|------|
| **身份** | 蜀漢丞相，武鄉侯。字孔明，號臥龍。琅琊陽都（今山東臨沂）人。生於西元 181 年——卒於西元 234 年（五丈原） |
| **此時狀態** | 寫〈後出師表〉時諸葛亮 48 歲——距離他去世還有六年。此時的他——剛剛經歷了人生中最大的軍事失敗——揮淚斬了最信任的將領——自貶官職——承受著朝內外的巨大壓力。〈後出師表〉中的他——不再是〈前出師表〉中那個從容的父親——而是一個疲憊的老人——正在用最後的氣力說服所有人相信自己 |
| **歷史定位** | 諸葛亮是中國歷史上最完美的政治人物形象——「鞠躬盡瘁，死而後已」的忠臣典範。他不僅是一個政治家、軍事家——還是一個發明家（木牛流馬、諸葛連弩）——一個文學家（〈出師表〉二篇）——一個法家實踐者（治蜀以嚴著稱）。他的形象在後世被不斷神化——從《三國演義》中的「多智而近妖」——到民間信仰中的「諸葛神算」——但〈後出師表〉中的諸葛亮——是那個最真實的人——一個正在經歷失敗、「不知明天會怎樣」的疲憊中年 |

**軍師評**：〈後出師表〉中的諸葛亮——與〈前出師表〉中的諸葛亮——是兩個不同的人。前者是「相父」——權威自信——對劉禪說「你應該這樣做」。後者是「辯護者」——謹慎急切——對所有人說「請聽我解釋」。這種變化——不是因為諸葛亮變了——是因為現實變了。六年前——他還有五年時間準備——他還有信心。六年後——他只剩六年壽命——他已經打了敗仗——他必須在死前——完成先帝的遺願。〈後出師表〉——就是一個知道自己時日無多的人的——最後告白。

### 4.2 劉禪——那個被說服的皇帝

| 項目 | 內容 |
|------|------|
| **身份** | 蜀漢後主，劉備之子。字公嗣，小名阿斗。此時 22 歲——正值年輕——但已當了五年皇帝 |
| **此時態度** | 第一次北伐失敗後——劉禪面臨朝中要求停止北伐的巨大壓力。作為皇帝——他必須在「聽相父的話繼續北伐」和「聽大臣的話休養生息」之間做出選擇。〈後出師表〉就是諸葛亮寫給他的「說服信」——目的是讓他在朝議時繼續支持北伐 |
| **與諸葛亮的關係** | 劉禪對諸葛亮非常尊敬——稱之為「相父」。但這種關係——在第一次北伐失敗後——面臨著考驗。劉禪必須決定：繼續相信這個打過敗仗的丞相——還是聽從其他大臣的意見？諸葛亮在〈後出師表〉中——用「六不解」——用「鞠躬盡瘁」——實際上是在懇求劉禪：再相信我一次 |

**軍師評**：劉禪在〈後出師表〉中的角色——比在〈前出師表〉中更加被動。在〈前出師表〉中——諸葛亮把他當成一個需要教育的孩子。在〈後出師表〉中——諸葛亮把他當成一個需要說服的君主。這是一個微妙的轉變——因為戰敗之後——諸葛亮不再是那個「不會犯錯的相父」了——他必須用論證——而不是用權威——來獲得劉禪的支持。從這個角度來看——〈後出師表〉不僅是一次戰略辯護——它也是一次政治求生——諸葛亮在用自己的名譽——賭最後一次。

### 4.3 隱藏的關鍵人物——反對北伐的人

〈後出師表〉的「對手」——不是曹魏——而是蜀漢朝中的反對派。雖然文中沒有指名道姓——但從「六不解」的論證框架來看——這些人應該是蜀漢朝廷中——主張休養生息、反對繼續用兵的大臣：

| 類型 | 代表人物（推測） | 主張 |
|------|-----------------|------|
| **保守派** | 可能包括譙周（後來的《仇國論》作者）、費禕（後期反對諸葛亮頻繁北伐）等 | 主張休養生息——認為蜀漢國力不足以支撐長期戰爭——應該先發展內政——等待時機 |
| **務實派** | 可能包括一些掌管財政和民政的官員 | 認為連續戰爭會耗盡國力——應該優先恢復生產——而不是繼續擴張 |
| **求和派** | 可能包括一些親東吳或親曹魏的勢力 | 認為蜀漢不具備統一中國的實力——應該維持三國鼎立的現狀——甚至與曹魏和解 |

**軍師評**：這些反對者——從今天的角度來看——他們的意見可能比諸葛亮的更理性。以蜀漢的國力——要消滅曹魏——確實是天方夜譚。但諸葛亮之所以堅持北伐——不是因為他真的相信自己會贏——而是因為他別無選擇。如果不北伐——蜀漢最終會被曹魏吞併——只是時間問題。如果北伐——至少還有一線生機。這就是「知其不可而為之」的真諦——不是因為有希望才堅持——是因為堅持了——才會有希望。

## 五、核心分析——「鞠躬盡瘁，死而後已」

### 5.1 一句改變中國文化的話

「鞠躬盡瘁，死而後已」——這八個字——是〈後出師表〉對中國文化最大的貢獻。在它出現之前——中國文化中最高級的「忠誠」——是「比干諫而死」——是「忠臣不事二主」——是一種消極的、守節的忠誠。在它出現之後——忠誠的定義被改寫了——它變成了「竭盡全力，直到死亡」。

| 面向 | 分析 |
|------|------|
| **字面意思** | 「鞠躬」——小心謹慎、不敢怠慢的樣子。「瘁」——勞累、疲倦。「死而後已」——直到死亡才停止。整句話的意思是：我會小心謹慎地操勞——直到我死的那一天——我才會停下來 |
| **文化影響** | 這句話成為中國文化中「忠誠」的最高境界。從岳飛到文天祥——從曾國藩到孫中山——無數中國精英在人生的關鍵時刻——用這八個字來激勵自己——也用這八個字來定義自己的一生 |
| **與〈前出師表〉的關係** | 雖然這句話出於〈後出師表〉——但它的精神貫穿〈前出師表〉全文。可以說——〈前出師表〉是「為什麼要鞠躬盡瘁」的說明文——〈後出師表〉是「如何鞠躬盡瘁」的實踐報告——兩篇合在一起——才是完整的諸葛亮 |

> **軍師按**：這句話之所以偉大——不是因為它華麗——是因為它真實。諸葛亮說這句話的時候——他已經打了一場敗仗——他已經斬了自己最喜歡的將領——他已經自貶官職——朝中已經有人開始質疑他——他知道自己可能永遠無法完成先帝的遺願。但他在這個時候說——「鞠躬盡瘁，死而後已」——不是一個勝利者的宣言——這是一個失敗者的誓言——而失敗者的誓言——永遠比勝利者的宣言——更有力量。

### 5.2 〈前出師表〉與〈後出師表〉的對比

兩篇〈出師表〉——雖然出自同一個作者——寫於同一個時代——但風格和內容截然不同。它們的差異——恰恰反映了諸葛亮在六年間的變化：

| 面向 | 〈前出師表〉 | 〈後出師表〉 |
|------|-------------|-------------|
| **寫作時間** | 建興五年（227 年）——北伐之前 | 建興六年（228 年）——第一次北伐失敗後 |
| **寫作動機** | 臨行前的政治交代——告訴劉禪如何治國 | 戰敗後的辯護——反擊朝中反對北伐的意見 |
| **情感基調** | 冷靜、慈愛、謹慎——像父親教導兒子 | 急切、悲壯、激烈——像將領為自己辯護 |
| **核心論證** | 「親賢臣，遠小人」——一個簡單樸素的道理 | 「六不解」——六個反駁反對意見的論證 |
| **最著名的句子** | 「先帝創業未半而中道崩殂」「臣本布衣，躬耕於南陽」 | 「鞠躬盡瘁，死而後已」「連曹操都會失敗，何況是我？」 |
| **語言風格** | 平實、流暢、真誠——以情感人 | 急切、重複、強烈——以理服人 |
| **讀者的感受** | 感動——被諸葛亮的忠誠和真誠打動 | 震撼——被諸葛亮的執著和悲壯震撼 |

### 5.3 〈後出師表〉為什麼沒有被《古文觀止》收錄？

這是一個值得注意的事實：《古文觀止》收錄了〈前出師表〉——但沒有收錄〈後出師表〉。這個選擇——反映了清初文人對兩篇表文的判斷：

| 可能的原因 | 分析 |
|------------|------|
| **真偽問題** | 如前所述——〈後出師表〉的真偽學術界有爭議。《古文觀止》的編者可能考慮到這個爭議——選擇不收錄 |
| **文學價值** | 〈前出師表〉被認為是「公文文學化」的極致——從容、真誠、結構完美。〈後出師表〉則偏重論證——語言急切——文學性不如前者 |
| **思想價值** | 〈前出師表〉提出了「親賢臣，遠小人」的普遍政治原則——具有超越時代的價值。〈後出師表〉則主要針對當時的具體政治爭論——時效性更強 |
| **情感價值** | 〈前出師表〉以「真情」取勝——可以跨越時代打動任何人。〈後出師表〉以「執著」取勝——但需要讀者理解三國時期的具體背景才能充分體會 |

**軍師評**：雖然《古文觀止》沒有收錄〈後出師表〉——但它在中國文化中的地位——不亞於〈前出師表〉。如果說〈前出師表〉是諸葛亮「生前」的交代——那麼〈後出師表〉就是諸葛亮「死前」的遺言。前者教人如何活——後者教人如何死——而後者在中國文化中的影響——如果只論一句話——可能比〈前出師表〉的整篇加起來還要大。

## 六、軍師核心分析——「知其不可而為之」的政治哲學

### 6.1 什麼是「知其不可而為之」？

「知其不可而為之」——這句話出自《論語·憲問》——是孔子說的。有一次——孔子在衛國碰上了「晨門」（一個守門人）——守門人說：「你是那個明知做不到卻還要去做的人嗎？」（「是知其不可而為之者與？」）。孔子沒有否認——這確實是他的人生態度。

諸葛亮——就是中國歷史中最完美的「知其不可而為之」的實踐者：

| 面向 | 分析 |
|------|------|
| **理性上知道不可能** | 諸葛亮不是傻子——他知道蜀漢和曹魏的實力懸殊——他知道北伐成功的機率極低——他知道自己的身體已經不行了——〈後出師表〉中他自己說：「以先帝之明，量臣之才，固知臣伐賊，才弱敵強也」——連先帝都知道——我的才能不足以對付強敵 |
| **情感上不得不為** | 但他還是要打——因為他欠先帝一條命。〈後出師表〉的「六不解」——表面上是六個理性的論證——但本質上——它們是一個理由的反覆表達：我必須打——因為我不打——我就對不起先帝——我活著就沒有意義了 |
| **哲學上的「義命分立」** | 儒家的「義命分立」——「義」是我該做什麼——由我決定——「命」是結果如何——由天決定。諸葛亮在〈後出師表〉中的最後一句話——「至於成敗利鈍，非臣所能逆睹」——就是「義命分立」最完美的表達：我能決定的——只有我的行動——至於成敗——那不是我能預見的 |

> **軍師按**：「知其不可而為之」——不是愚蠢——也不是魯莽——而是一種更高層次的智慧。它承認了人的有限性——我們永遠無法完全掌控結果——但我們可以掌控自己的選擇。諸葛亮選擇了「鞠躬盡瘁」——不是因為他相信自己會贏——而是因為他相信：對的事情——就應該去做——不管結果如何。這——就是〈後出師表〉留給後世的最重要的遺產。

### 6.2 「連曹操都會失敗」——諸葛亮論失敗的合法性

〈後出師表〉中最震撼人心的論證——不是「鞠躬盡瘁」——而是諸葛亮對曹操失敗史的詳細列舉：

> 「曹操之智計，殊絕於人，其用兵也，仿佛孫、吳，然困於南陽，險於烏巢，危於祁連，偪於黎陽，幾敗北山，殆死潼關，然後偽定一時耳。況臣駑下，何能必勝？」

這段話的深層意涵非常豐富：

| 層次 | 分析 |
|------|------|
| **第一層** | 表面上是為自己的失敗辯護：曹操都輸了那麼多次——我輸一次有什麼大不了的？ |
| **第二層** | 更深層是在告訴所有人：失敗是正常的——成功才是例外。連曹操——「用兵仿佛孫吳」的超級天才——都經歷了無數次失敗——才勉強成功。你為什麼要求我——一個「駑下」之人——一次都不能輸？ |
| **第三層** | 最深的層次——是在告訴劉禪：不要因為一次失敗就放棄整個計劃。曹操輸了那麼多次——他沒有放棄——所以他最後成功了。我們輸了一次——為什麼就要放棄？ |

**軍師評**：這是中國古代政治論辯中——最真誠的一段自我批評。諸葛亮沒有誇大自己的能力——他把自己說成「駑下」——他說連曹操都會失敗——意思是——我的失敗不是個人能力的問題——是戰爭本身的問題。然後他轉了一個彎——既然失敗是正常的——那我們為什麼還要繼續？因為——如果不繼續——之前的失敗就白費了。這是一種「沉沒成本」的論證——但用在諸葛亮身上——它不顯得算計——只顯得悲壯。

### 6.3 〈後出師表〉的文化符號

雖然《古文觀止》沒有收錄〈後出師表〉——但它在中國文化中的影響力——絲毫不亞於〈前出師表〉：

| 人物 | 與〈後出師表〉的關係 |
|------|----------------------|
| **岳飛** | 岳飛曾手書〈前出師表〉和〈後出師表〉——以諸葛亮自比——激勵自己北伐金朝。他的結局與諸葛亮相似——「出師未捷身先死」——都是「鞠躬盡瘁，死而後已」的實踐者 |
| **文天祥** | 文天祥在《正氣歌》中寫道：「或為出師表，鬼神泣壯烈」——他的〈後出師表〉不是指具體哪一篇——而是指出師表的精神——一種「知其不可而為之」的悲壯 |
| **陸游** | 陸游在〈書憤〉中寫道：「出師一表真名世，千載誰堪伯仲間」——他所說的「出師一表」——通常被認為是〈前出師表〉——但「真名世」的評價——同樣適用於〈後出師表〉 |
| **近代中國** | 「鞠躬盡瘁，死而後已」——成為中國政治文化中最常被引用的句子之一——從孫中山到周恩來——無數政治人物用這句話來表達自己的忠誠和奉獻 |

## 七、一句話總結（再錄）

> **〈後出師表〉是一個深知自己時日無多的老人，用最後的力氣說服一個小孩繼續支持他的戰爭——他知道自己可能看不到勝利，但他必須讓劉禪相信這條路是對的。全篇以「六不解」為框架，每一「不解」都在回答同一個問題：為什麼我輸了一次——還要繼續打？答案是——不是因為我確定會贏——是因為我不打——就對不起那個已經死了的人。「鞠躬盡瘁，死而後已」——八個字——說盡了諸葛亮的一生——也說盡了所有「知其不可而為之」者的心聲。連曹操都會失敗——何況是我？但正因為連曹操都失敗了那麼多次——卻依然成功了——所以我——也不能放棄。這就是〈後出師表〉——不是一個勝利者的凱歌——而是一個失敗者的誓言——而失敗者的誓言——永遠比勝利者的凱歌——更有力量。**

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷六·〈前出師表〉（諸葛亮） | 與〈後出師表〉是直接的姊妹篇——前者寫於出征前——後者寫於戰敗後——前後對照可見諸葛亮的心理變化——從「親賢臣遠小人」的冷靜交代——到「鞠躬盡瘁死而後已」的悲壯宣言——一個是正在遠行的人——一個是知道自己可能回不來的人 |
| 卷六·〈報任安書〉（司馬遷） | 兩者都是在「失敗」之後寫的內心獨白——司馬遷被宮刑——諸葛亮戰敗——兩封信都在回答同一個問題：我為什麼要活著（或繼續）？司馬遷為了《史記》——諸葛亮為了先帝——都是「為了某個比生命更重要的東西而活著」 |
| 卷五·〈屈原列傳〉（司馬遷） | 屈原與諸葛亮——「忠而被謗」——都遭到了不公正的對待——但屈原選擇投江——諸葛亮選擇「鞠躬盡瘁」——兩種截然不同的回應——兩種不同的人生哲學——但都展現了中國知識分子對「忠誠」的極致追求 |
| 卷四·〈觸讋說趙太后〉（《戰國策》） | 觸讋說服趙太后——與諸葛亮說服劉禪——都是「以老勸幼」的經典案例——觸讋用「愛子則為之計深遠」打動趙太后——諸葛亮用「鞠躬盡瘁死而後已」打動劉禪——兩個老人——兩種說服方式——但都成功了 |

### 8.2 經典延伸

- 《三國志·蜀書·諸葛亮傳》（陳壽）——諸葛亮傳記——陳壽未收錄〈後出師表〉——但記載了第一次北伐的全過程——包括馬謖失街亭、揮淚斬馬謖等事件
- 《三國志·裴松之注》（裴松之）——裴松之注引《漢晉春秋》全文收錄〈後出師表〉——並特別注明「此表亮集所無，出張儼《默記》」——是〈後出師表〉真偽爭議的源頭文獻
- 《漢晉春秋》（習鑿齒）——〈後出師表〉的原始出處——習鑿齒是東晉史學家——他從何處獲得這份文獻——無從考證
- 《三國演義》（羅貫中）——第九十七回「討魏國武侯再上表」——將〈後出師表〉的寫作過程小說化——並在其中加入了大量的戲劇衝突
- 《諸葛亮集》（諸葛亮著，張澍編）——諸葛亮作品的完整合集——張澍在此書中收錄了〈後出師表〉——但標註了真偽爭議的考證
- 《資治通鑑·魏紀》（司馬光）——司馬光對三國時期的記載——提供了街亭之戰前後的完整歷史背景
- 杜甫〈蜀相〉——「出師未捷身先死，長使英雄淚滿襟」——雖然寫的是諸葛亮的整體形象——但與〈後出師表〉的「鞠躬盡瘁」精神完全一致
- 陸游〈書憤〉——「出師一表真名世，千載誰堪伯仲間」——陸游以諸葛亮自比——表達北伐中原的志向

### 8.3 現代延伸

- 維克多·弗蘭克爾《活出生命的意義》——弗蘭克爾從集中營倖存後提出——人活著的意義——在於他選擇了什麼樣的態度來面對苦難。諸葛亮選擇了「鞠躬盡瘁」——他不問成敗——只問是否盡力——這與弗蘭克爾的「意義療法」精神一致——都是在絕望中找到繼續前進的意義
- 西蒙·斯涅克《先問為什麼》——黃金圈理論（Why → How → What）——諸葛亮的「Why」是「報先帝而忠陛下」——他的「How」是「鞠躬盡瘁」——他的「What」是北伐。〈後出師表〉就是一篇「為什麼我們必須繼續」的宣言——一個偉大的領袖——總是在「為什麼」的層次上說服人
- 納西姆·塔勒布《反脆弱》——書中指出：有些系統——反而在混亂和壓力中變得更強。諸葛亮的北伐——雖然屢次失敗——但蜀漢的國力——並沒有因為北伐而崩潰——反而在抵抗壓力中維持了內部團結。這是一種另類的「反脆弱」
