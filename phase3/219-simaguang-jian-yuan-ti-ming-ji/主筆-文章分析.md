# 諫院題名記 — 文章分析

> 司馬光
>
> 本檔案由 @Hermes Agent 主筆

## 1. 文章結構分析

**第一段：追述諫官制度的由來——古者諫無官，自公卿大夫至於工商**

> 「古者諫無官，自公卿大夫至於工商，無不得諫者。漢興以來，始置官。」

開篇以古今對比破題——古代沒有專職諫官，人人都可以進諫；漢代以後才設官專司其職。一古一今，看似客觀的歷史回溯，其實已暗含褒貶：古代人人皆可諫——那是理想的狀態；後世設官專責——雖有制度但責任也更重了。

**第二段：揭出諫官的責任——當志其大，舍其細；先其急，後其緩**

> 「夫以天下之政，四海之眾，得失利病，萃於一官，使言之，其為任亦重矣。居是官者，當志其大，舍其細；先其急，後其緩；專利國家，而不為身謀。」

這是全文的核心段落。司馬光一口氣提出諫官的四個行事原則——志大舍細、先急後緩、專利國家、不為身謀。這四句緊湊有力，層層遞進，把諫官的責任從「做什麼」講到「怎麼做」，再講到「為誰做」。最後「專利國家而不為身謀」九字，是全篇的綱領所在。

**第三段：批判追求名聲的心態——彼汲汲於名者，猶汲汲於利也**

> 「彼汲汲於名者，猶汲汲於利也，其間相去何遠哉？」

一轉——從「該怎麼做」轉到「不該怎麼做」。司馬光敏銳地指出：追逐名聲和追逐利益，在本質上沒有區別。諫官如果只是為了博取直名、留下美譽，那就和貪官污吏沒有兩樣。這是一記當頭棒喝，讓所有以「諫官」之名行「沽名」之實的人無所遁形。

**第四段：交代題名來歷——天禧初詔置諫官，慶曆中書名於版，嘉祐八年刻於石**

> 「天禧初，真宗詔置諫官六員，責其職事。慶曆中，錢君始書其名於版；光恐久而漫滅，嘉祐八年，刻於石。」

司馬光用極簡的筆法交代了題名的沿革——從天禧設官、慶曆書版、到嘉祐刻石。一個「恐」字道出玄機：怕的是名字漫滅——但真正怕的，恐怕是諫官們的責任也跟著漫滅了。刻石，是為了讓名字永遠留在那裡。

**第五段：點出題名深意——後之人將歷指其名而議之曰：某也忠，某也詐**

> 「後之人將歷指其名而議之曰：『某也忠，某也詐，某也直，某也曲。』嗚呼，可不懼哉！」

全文最震撼的一句。名字刻在石頭上，不是榮耀——而是審判。後人會一個一個指著名字問：這個人是忠是奸？是直是曲？「嗚呼，可不懼哉」五字收束全文——這是對所有在職諫官的最後警醒，也是對後世所有讀者的一記警鐘。

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| 起 | 古者諫無官——從歷史溯源諫官制度的由來 |
| 承 | 居是官者當志其大舍其細——揭出諫官的責任與原則 |
| 轉 | 彼汲汲於名者猶汲汲於利——批判追逐名聲的心態 |
| 合 | 後之人將歷指其名而議之——題名刻石，留待歷史審判 |

## 2. 寫作技巧分析

### 步步緊逼的論說節奏

司馬光這篇短文僅百餘字，卻完成了從歷史到責任、從批判到警醒的完整論述。每一段都比上一段更緊迫：從「制度由來」到「行事原則」到「心態批判」再到「歷史審判」——讀者沒有喘息的空間，只能跟著司馬光的筆一路走到最後那聲「可不懼哉」。

### 短句的力量

全文多用短句，節奏鏗鏘有力。尤其是第二段「當志其大，舍其細；先其急，後其緩；專利國家，而不為身謀」——三個分句、六個動作，一氣呵成，沒有一個多餘的字。這種極簡的筆法，本身就是一種道德的力量。

### 修辭手法

| 手法 | 原文 | 效果 |
|:----|------|------|
| 古今對比 | 「古者諫無官」 vs 「漢興以來，始置官」 | 從歷史對比中凸顯制度的演變 |
| 排比 | 「志其大，舍其細；先其急，後其緩」 | 兩組對仗，節奏鮮明，原則清晰 |
| 類比 | 「汲汲於名者，猶汲汲於利也」 | 把求名和求利劃上等號，一針見血 |
| 反問 | 「其間相去何遠哉？」 | 反問句加強批判力度，無需回答 |
| 設景 | 「後之人將歷指其名而議之」 | 虛擬後世場景，讓人如臨其境 |
| 感嘆 | 「嗚呼，可不懼哉！」 | 五字收束，如暮鼓晨鐘，餘音不絕 |

## 3. 佳句賞析

### 「居是官者，當志其大，舍其細；先其急，後其緩；專利國家，而不為身謀。」

全文最重要的句子。三組對仗構成整齊的節奏——第一組講視野（志大舍細），第二組講順序（先急後緩），第三組講立場（為國不為身）。前兩句是方法論，最後一句是價值觀——如果沒有最後的「不為身謀」，前面的所有原則都只是技術問題。司馬光的高明在於：他把道德放在最後，卻是最重的一擊。

### 「彼汲汲於名者，猶汲汲於利也，其間相去何遠哉？」

這是一個極具洞見的觀察。司馬光沒有停留在表面——他看到了「求名」和「求利」的同一性。對諫官來說，以「直言進諫」之名博取清譽，和貪官以權謀私沒有本質區別。這句話拆穿了所有以道德為幌子的虛偽，是全文最鋒利的一刀。

### 「後之人將歷指其名而議之曰：某也忠，某也詐，某也直，某也曲。」

全文最有畫面感的句子。司馬光沒有說「名字很重要」，而是把一個虛構的後世場景推到讀者面前——未來的人會一個一個指著名碑上的名字，做出評判。「歷指其名」四字極為生動，彷彿那些手指正指著碑上每一個名字，也包括讀者自己的名字。歷史的審判不在遠方——就在此刻。

### 「嗚呼，可不懼哉！」

五個字收束全篇。從冷靜的論述到最後的感嘆——司馬光終於不再克制，發出了最直接的警告。這聲「嗚呼」不是文學修辭，而是一個正直士大夫面對朝廷諫官失職現象的真心憂慮。全文到此戛然而止，沒有再多說一個字——因為已經夠了。

## 4. 白話改編

古代沒有專職的諫官——從公卿大夫到工匠商人，沒有人不可以進諫。漢朝建立以後才開始設置諫官。

拿全天下的政事、四海的民眾、所有的得失利害，全部集中在一個官員身上，讓他來發表意見——這個責任也太重了。

擔任這個官職的人，應該關注大事、捨棄小事；先處理緊急的、後處理不急的；專門為國家謀利益，不為自己打算。

那些急著追求名聲的人，就像急著追求利益的人一樣——兩者之間又相差多少呢？

天禧初年，真宗皇帝下詔設置了六名諫官，明確了他們的職責。慶曆年間，錢君開始把諫官的名字寫在木版上；我擔心時間久了會模糊不清，所以在嘉祐八年，把名字刻在了石頭上。

後代的人會一個一個指著名碑上的名字，議論說：「這個人忠誠，這個人奸詐，這個人正直，這個人邪曲。」

唉——能不戒懼嗎！
