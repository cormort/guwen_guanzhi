# 〈子革對靈王〉背景資料

> 古文觀止卷二·左傳·昭公十二年
> 撰寫風格：軍師策論

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **史載出處** | 《左傳·昭公十二年》 |
| **時間** | 魯昭公十二年（西元前530年），春秋晚期 |
| **地理經緯** | 州來（今安徽鳳台一帶），楚靈王狩獵於此；談話內容涉及周都洛邑（今洛陽）與鄭國國都 |
| **核心事件** | 楚靈王在州來狩獵，右尹子革隨侍在側。靈王得意洋洋地問：從前諸侯朝貢周天子，現在都來朝貢我楚國，是不是很厲害？繼而問：若派兵去向周天子要九鼎、向鄭國搶土地，別人會怎麼說？子革先以「君王一定會做到的」回應，而後以歷史故事（周穆王、祭公謀父作〈祈招〉之詩）委婉勸諫靈王不可驕奢無度 |
| **經典文眼** | 「祈招之愔愔，式昭德音。思我王度，式如玉，式如金。形民之力，而無醉飽之心。」——祭公謀父所作〈祈招〉之詩，告誡為君者節制慾望、珍惜民力 |
| **敗亡伏筆** | 子革退朝後感嘆：「君王的慾望沒有滿足的一天，我怕他不會有好下場。」——此話在一年後（前529年）應驗：楚靈王眾叛親離，自縊於乾谿 |

## 二、時代壓力：春秋末期的楚國霸業與暴政循環

### 2.1 楚國的百年崛起

楚國自春秋以來，歷代君主持續向中原擴張：

- **楚武王（前740—前690年在位）**：開始稱王，與周天子分庭抗禮
- **楚莊王（前613—前591年在位）**：問鼎中原（「問鼎輕重」典故出此），在邲之戰擊敗晉國，成為春秋五霸之一
- **楚共王（前590—前560年在位）**：維持霸業，鄢陵之戰敗於晉國，國力開始下滑
- **楚靈王（前540—前529年在位）**：殺侄自立，窮兵黷武，大肆擴張，表面上達到楚國疆域的極盛

### 2.2 楚靈王其人——權力慾望的化身

楚靈王（熊虔，原名圍）是春秋晚期最令人矚目也最令人膽寒的君主之一。

#### 得位不正——弒君篡位的開局

西元前541年，公子圍趁楚王郟敖（其侄）生病，入宮問疾時以冠纓勒死国君，自立為王。這在宗法倫理上是極其嚴重的罪行——不僅是弒君，還是弒侄。從他即位的第一天起，合法性就是他最大的心結。

#### 對外戰爭——以征服掩蓋內部矛盾

楚靈王在位十二年（前540—前529年），幾乎年年用兵：

| 年份 | 征伐對象 | 結果 |
|------|----------|------|
| 前539年 | 伐吳 | 奪取州來（本次談話地點） |
| 前538年 | 會諸侯於申，伐吳、滅賴 | 楚國稱霸中原達到頂點 |
| 前537年 | 伐吳 | 敗吳於鵲岸 |
| 前534年 | 滅陳 | 將陳國變為楚縣 |
| 前531年 | 滅蔡 | 殺蔡靈侯，將蔡國變為楚縣 |

**滅陳、滅蔡是楚靈王對外擴張的高峰**。陳、蔡雖是小國，但都是周天子冊封的諸侯國，楚國公然滅之，等於向整個中原諸侯體系宣戰。這在周禮框架下，是霸權行為的極限——超出了楚莊王「問鼎輕重」的試探，直接動手改寫地圖。

#### 內部統治——以恐怖維持秩序

楚靈王在國內的統治同樣高壓：

- **大興土木**：修建章華宮，極盡奢華，號稱「天下第一台」，耗費民力無數
- **重用寵臣**：信任薳啟疆、薳洩等近臣，疏遠公族子弟
- **壓制親族**：畏懼宗族反撲，對親兄弟、遠房宗室嚴加監視
- **刑罰過重**：以嚴刑峻法鎮壓反對者，國內怨聲載道

### 2.3 風雨欲來——乾谿之難的倒計時

當子革與靈王談話時（前530年），距離靈王的垮台僅剩一年。這是暴風雨前的寧靜——外表的強大掩蓋了內部的潰爛。

楚靈王的困境：

1. **合法性赤字**：弒君篡位的原罪始終無法抹去，他是以恐懼維持統治的君主
2. **擴張極限**：滅陳、滅蔡之後，楚國疆域雖廣，但統治成本急遽上升，邊境烽火四起
3. **內部離心**：公族、貴族對靈王的驕奢與高壓早已不滿，只是畏於威勢不敢發作
4. **吳國的牽制**：楚國的宿敵吳國在闔閭即位前雖未成大器，但已在與楚的邊境摩擦中積累戰力

### 2.4 歷史的諷刺——「問鼎」的兩種結局

楚莊王在前606年「問鼎輕重」，王孫滿以「在德不在鼎」回應，莊王知難而退，保全了楚國的大國形象。八十餘年後，楚靈王重提「問鼎」——但這一次他不是試探，而是真的想要。他沒有聽到「在德不在鼎」的教訓，因為子革告訴他的，是一個更嚴厲的警告：**即使是周穆王那樣的天子，也會因為驕奢而身敗名裂，何況是你？**

楚莊王問鼎後成就了霸業；楚靈王問鼎後一年即自縊身亡。同一種野心，同一個國家，同一種提問——不同的歷史評價，差別只在於一個有節制、一個沒有。

## 三、關鍵人物

### 楚靈王（熊虔，？—前529年）

| 面向 | 分析 |
|------|------|
| **身分** | 楚國第二十六代君主，殺侄篡位，在位十二年 |
| **性格** | 傲慢、貪婪、虛榮——一個被「政治合法性焦慮」驅動的暴君。他需要不斷用對外征服來證明自己配得上王位 |
| **優勢** | 軍事才能卓越，楚國在他手中疆域達到極盛；有威儀、能震懾諸侯 |
| **致命弱點** | 不知節制。他對子革說「我想要的東西還沒有得不到的」，這句話本身就是亡國之音——**一個認為自己無所不能的君主，離滅亡只剩一步** |
| **關鍵一句** | **「昔諸侯遠我而畏晉，今我大城陳、蔡、不羹，賦皆千乘，子與有勞焉。諸侯其畏我乎？」**——他關心的只有諸侯是否畏懼他，而不是國家的長治久安 |
| **最終下場** | 前529年，楚國內亂爆發，公子棄疾（後來的楚平王）起兵攻入郢都。靈王的親兵潰散、兒子被殺。他流亡荒野，飢寒交迫，最終自縊於申亥氏家中。死前感嘆：「我不是個好君主，我沒有早點聽取勸諫」——但已經太遲了 |
| **軍師評語** | 楚靈王是春秋晚期最典型的「強人政治」案例。他以恐懼治國，也死於恐懼。歷史從不同情那些「雖然有才能但沒有節制」的君主——因為沒有節制的才能，恰恰是最危險的東西。 |

### 子革（鄭丹／然丹，生卒年不詳）

| 面向 | 分析 |
|------|------|
| **身分** | 楚國右尹（相當於副宰相），本是鄭國大夫，後逃奔楚國，成為靈王近臣 |
| **性格** | 深諳權術、極度內斂、善於在虎狼之側保全自己的「生存型賢臣」 |
| **處境** | 他是外國投奔而來的大夫，在楚國的根基並不深厚。他無法像中國本土大臣那樣直言進諫——因為靈王隨時可以殺他而沒有任何政治後果 |
| **進諫策略** | 子革的勸諫方式極其高明，是春秋諫術的教科書級案例：<br><br>**第一步：順從（建立對話）**<br>靈王問：「諸侯是不是都怕我？」「我去跟周朝要九鼎，他們會給我嗎？」——子革全部回答：「一定會的。」<br><br>**第二步：設伏（引導反思）**<br>靈王得意之餘，問：「我聽說從前鄭國大夫子產說鄭國就像楚國的小弟弟，是真的嗎？」——子革回答：「君王問什麼我就答什麼，豈敢隱瞞？」<br><br>**第三步：致命一擊（說故事）**<br>當靈王心情正好時，子革突然講了周穆王的故事：周穆王想周遊天下，祭公謀父作〈祈招〉之詩勸諫，穆王聽從，得以善終。這首詩的核心思想是：「形民之力，而無醉飽之心」——**役使民力要有限度，不能像喝酒吃肉那樣不知飽足。**<br><br>**關鍵**：子革從頭到尾沒有說一句「君王你錯了」。他只是講了一個周穆王聽從勸諫的故事。靈王聽懂了——但他選擇不聽。 |
| **軍師評語** | 子革是「對不對的人說對的話」的典範。他知道靈王不是能聽勸的人，但他仍然完成了一個臣子該做的事——把正確的道理放在正確的時機遞到君主面前，即使知道對方不會採納。這是一種深沉的智慧：**盡人事，聽天命，同時保護好自己不被陪葬。** 子革沒有在靈王死的時候殉葬——他活了下來，後來還在楚平王（公子棄疾）朝中繼續任職。這不是苟且偷生，而是在亂世中保全有用之身。 |

### 祭公謀父（引用人物，故事中的關鍵配角）

| 面向 | 分析 |
|------|------|
| **身分** | 周穆王的卿士（執政大臣） |
| **地位** | 從子革口中所引用的歷史人物，事蹟出自周代史書〈祈招〉篇 |
| **作用** | 祭公謀父是子革用來對照楚靈王的一面鏡子。周穆王是周朝最著名的漫遊天子，傳說他曾西巡至崑崙山，見西王母——如果他不是被祭公謀父勸住，恐怕早就因為勞民傷財而死於途中。祭公謀父做了該做的事，周穆王聽進了該聽的話——而靈王正好相反：子革做了該做的事，但靈王沒有聽進該聽的話 |

## 四、寫作意圖——左丘明為什麼要寫這件事？

### 4.1 春秋筆法：以「對答」寫「命運」

〈子革對靈王〉是《左傳》中極具特色的篇章。它不像傳統的諫諍文章那樣記錄大臣如何說服君主、君主如何幡然醒悟——它記錄的是一個**失敗的諫諍**。

左丘明的寫法非常高明：

1. **先寫靈王的得意**（長篇對白，展現其驕傲自滿）
2. **再寫子革的順從**（看似阿諛奉承，實則引君入甕）
3. **再寫子革的勸諫**（引用歷史故事，十分委婉）
4. **最後寫靈王的反應**（「王揖而入」——靈王只是拱手行禮，然後進去了。既沒有發怒，也沒有悔改）
5. **以子革退朝後對人說的話作結**：「君王不會有好下場的」

左丘明用不到三百字，刻畫了一個完整的悲劇：大臣盡了力，君主聽懂了，但什麼都沒有改變。

### 4.2 「不諫之諫」的最高境界

子革的進諫方式是《左傳》中最值得玩味的一篇：

- 他沒有直接批評靈王的野心
- 他沒有說「不可以要九鼎」
- 他沒有說「你這樣做會亡國」
- 他只是講了一個故事——周穆王的故事
- 故事的寓意很清楚：你像周穆王一樣有野心，但周穆王最後克制了自己
- 你比周穆王更危險，因為你連克制自己的念頭都沒有

這是一種「不諫之諫」——**真正高明的勸諫，不是告訴君主你錯了，而是讓君主自己發現自己錯了。**

### 4.3 為什麼要寫一個失敗的勸諫？

左丘明選擇記錄這個故事，原因有三：

1. **對比與諷刺**：〈子革對靈王〉與〈王孫滿對楚子〉（楚莊王問鼎）、〈子產不毀鄉校〉（子產的開明統治）形成連鎖對照。楚莊王問鼎而知退，楚靈王問鼎而亡身——**同一種行為，不同的結果，關鍵在於當事人有沒有「節制」這個品質。** 左丘明讓讀者自己比較。

2. **歷史的紀錄價值**：子革的預言在一年後成真——靈王死於乾谿。左丘明記錄這一番對話，是要後人看到：**一個國家的滅亡，往往在君主說出某些話的時候就已經註定了。** 靈王說「諸侯其畏我乎」的時候，離死只差一步。

3. **對「什麼是忠臣」的重新定義**：忠臣不一定是觸柱而死、直言犯上的那種。子革式的「委婉、迂迴、保全自己、但一定把話送到」的進諫方式，同樣是忠——甚至在某些時代是更可貴的忠，因為這種忠臣能活下來繼續做事。

### 4.4 左丘明的結構藝術——從驕傲到死亡的伏線

全文走勢：

```
靈王發問（驕傲）  →  子革附和（順從）  →  子革說故事（轉折）
→  靈王聽懂了（沉默）  →  子革預言（判決）
```

左丘明沒有寫靈王的死，但子革退朝後對人說的那句話，已經等於宣判了靈王的死刑。這是一種極高明的「留白」敘事——不需要寫後續的乾谿之難，因為讀者已經知道了結局。

## 五、戰略貫通——此事件的歷史意義

| 層面 | 意義 |
|------|------|
| **政治哲學** | 「形民之力，而無醉飽之心」——這是中國政治思想史上對「節制統治」最經典的表述之一。統治者役使民力，就像飲食一樣，必須有節制。不知節制的人，會把自己撐死 |
| **諫諍藝術** | 子革的進諫方式是「不諫之諫」的典範——不直接說「不」，而是用故事讓對方自己領悟。這是中國政治史上「諫術」的高級階段 |
| **霸權邏輯** | 楚靈王的滅亡揭示了春秋晚期霸權政治的內在矛盾：靠武力擴張可以快速稱霸，但沒有以德服人的基礎，擴張越快，崩盤越猛。乾谿之難前後不過數月，靈王的龐大帝國就在一夜之間土崩瓦解——因為沒有一個人是真心服從他的 |
| **歷史預言** | 子革的「王沒有好下場」是人臣對君主的終極判決。在中國史書傳統中，臣子預言君主的死亡，本身就是一種極具象徵意義的行為——代表君臣之間的信任已經斷裂，剩下的只是時間問題 |
| **人事智慧** | 子革在靈王死後繼續服務於楚國政權（平王時期），並沒有殉葬或隱退。這說明了古代政治中最務實的生存法則：**忠於國家，而不是忠於某個具體的君主。當君主走上了毀滅之路，有智慧的臣子會拉住他；但如果拉不住，也不該跟著他一起跳下去** |

## 六、一句話總結

> **楚靈王問鼎中原、滅陳滅蔡，自以為天下無敵；子革以一首〈祈招〉之詩、一個周穆王的故事，委婉告誡他「役民有限」。靈王聽懂了——但他選擇不聽。一年後，他在乾谿自縊身亡。這個故事不是關於諫諍的成功，而是關於諫諍的失敗——以及失敗之後，歷史對失敗者的精準審判。**

---

*軍師筆記：後世讀〈子革對靈王〉，當知這段對話的核心不在「子革說了什麼」，而在「靈王聽懂了什麼卻沒有做」。楚靈王是中國古代史上最典型的「聰明而無節制」的君主——他不是不聰明，他是不知道什麼時候該停下來。子革說〈祈招〉之詩，靈王「揖而入」——這一個動作，勝過千言萬語。他懂，他全都懂，但他做不到。「知而不能行」，是為驕奢淫逸者最深的悲哀。*

*對比閱讀：建議與〈王孫滿對楚子〉（楚莊王問鼎）並讀。莊王聽人勸，成為五霸之一；靈王不聽勸，成為亡國之君。同一本《左傳》，同一種問鼎，兩個結局——左丘明的春秋筆法，於此最見功力。*
