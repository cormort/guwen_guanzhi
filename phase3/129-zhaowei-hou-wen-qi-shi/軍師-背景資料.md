## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《戰國策·齊策四》 |
| **選集** | 《古文觀止》卷四·周文類（戰國策文） |
| **體裁** | 對話體論辯／書信問對 |
| **時代** | 戰國中期（約西元前 265 年，趙孝成王初即位，趙威后攝政時期） |
| **核心命題** | 以民為本——「苟無歲，何以有民？苟無民，何以有君？」——執政者究竟應該先問國君還是先問百姓？ |
| **關鍵人物** | 趙威后（趙惠文王之妻，趙孝成王之母，趙國實際執政者）、齊使（齊王派往趙國的使臣，代表傳統禮儀觀念）、鍾離子（齊國賢人，未被重用）、葉陽子（齊國賢人，未被重用）、嬰兒（北宮之女，未受表彰）、於陵子仲（齊國隱士，不事君王） |
| **文本重點** | 齊王派使者到趙國問候趙威后，威后尚未打開問候信，就先問使者：「你們齊國年景好嗎？百姓好嗎？齊王安好嗎？」使者不高興地說：「我先代表齊王來問候您，您卻先問年景和百姓，最後才問齊王——這是先賤後貴啊！」趙威后以「苟無歲，何以有民？苟無民，何以有君？」反駁——沒有收成就沒有百姓，沒有百姓就沒有國君，怎麼能捨本逐末、先問君後問民？接著她進一步追問齊國的賢人政治：鍾離子為何不重用？葉陽子為何不表彰？北宮之女為何不撫卹？為什麼卻供養於陵子仲這種無益於社會的人？——她把「問候」變成了一場對齊國政治的檢驗 |
| **思想譜系** | 出自《戰國策·齊策四》，為戰國策士言論彙編。本文雖以「趙威后」而非「策士」為主角，但其核心論辯方式——以問代答、層層逼進——完全是戰國策士的修辭風格。趙威后本人可以被視為一位「以統治者身份行策士之智」的罕見人物。這篇是中國政治思想史上最早將「民本」從道德呼籲提升為執政邏輯的文獻之一，比孟子「民為貴，社稷次之，君為輕」的宣言更具體、更直接——因為它不是一個理論陳述，而是一個政策質詢 |
| **一句話總結** | **趙威后是中國歷史上第一個將「以民為本」從口號變成執政邏輯的政治家——她不是不知道禮儀的重要性，而是她知道什麼才是真正重要的。當所有人都覺得「先問君、後問民」是天經地義的時候，她用一句「苟無歲，何以有民？苟無民，何以有君？」就把整個權力邏輯翻了過來：不是國君養活百姓，而是百姓養活國君——所以問候的順序，應該從根源問起。** |

## 二、時代背景——戰國中期的趙國與趙威后攝政

### 2.1 趙國在戰國中期的處境

戰國中期，趙國經歷了從趙武靈王「胡服騎射」（西元前 307 年）以來的軍事改革，一躍成為北方強國。但趙國的強盛並不穩定——它地處四戰之地，西有強秦、南有魏韓、東有齊國、北有燕國與匈奴。沒有哪一個鄰居是真正可靠的盟友。

| 面向 | 說明 |
|------|------|
| **軍事優勢** | 趙武靈王的胡服騎射改革使趙國擁有戰國最強的騎兵部隊，在對抗匈奴和中原列國時佔有機動優勢 |
| **地緣劣勢** | 趙國都城邯鄲（今河北邯鄲）距離秦國函谷關僅數百里，秦國東出中原的第一個打擊目標往往就是趙國 |
| **外交困境** | 趙國與齊國有聯盟關係（趙威后本人就是齊國嫁過來的公主），但齊國同樣是趙國的潛在競爭者——兩國之間既有婚姻紐帶，也有領土矛盾 |
| **趙孝成王即位** | 趙惠文王去世後，其子趙孝成王（西元前 265—前 245 年在位）即位時年紀尚幼（一說十餘歲），由母親趙威后攝政——這是本文發生的時間背景 |

| > **軍師按**：趙威后攝政的時期，正好是趙國從強盛走向危機的轉折點。不久之後（西元前 260 年），趙國就將在長平之戰中遭受毀滅性的打擊——四十萬趙軍被白起坑殺。趙威后問齊使這番話，表面上是對齊國政治的議論，但實際上反映了她對趙國自身治理的憂慮——她在齊使身上看到的，不只是齊國的毛病，而是所有當權者共同的毛病：重君輕民。她在提醒齊使，也是在提醒自己。|

### 2.2 趙威后——中國最早的「攝政王太后」

趙威后是趙惠文王的王后、趙孝成王的母親。丈夫去世後，因新君年幼，她以太后身份攝政，實際執掌趙國大權。在戰國時期，女性攝政並非絕無僅有——秦國的宣太后（羋八子）就是更早的例子——但趙威后與宣太后形成了鮮明的對比。

| 面向 | 說明 |
|------|------|
| **丈夫** | 趙惠文王（趙何，西元前 298—前 266 年在位），趙武靈王之子，在位期間趙國國勢強盛 |
| **兒子** | 趙孝成王（趙丹，西元前 265—前 245 年在位），即位時年幼，由趙威后攝政 |
| **攝政時間** | 約西元前 265 年起，為時數年 |
| **執政風格** | 重視民生、尊重賢能、敢於挑戰傳統禮儀（本文即為明證） |
| **與宣太后的對比** | 宣太后以權謀和情慾聞名（曾與義渠王私通，後設計殺之），趙威后則以賢明和民本思想著稱——兩者代表了戰國女性統治者的兩種極端形象 |

| > **軍師按**：趙威后與宣太后，是戰國時期兩位最著名的攝政女性。但她們的執政風格完全不同——宣太后的權力來自於她在後宮和朝堂之間操作的權謀，而趙威后的權力來自於她對「什麼是正確的事」的清晰判斷。〈趙威后問齊使〉這篇文章，讓我們看到一個不靠美色、不靠陰謀、純粹靠智慧和原則來行使權力的女性統治者——這在戰國史料中極其罕見。|

### 2.3 齊國的困境與齊王建

本文的另一方——齊國——在戰國中期同樣處於關鍵轉折點。齊王建（西元前 264—前 221 年在位）是齊襄王之子、齊國最後一位國君。他在位的四十四年間，齊國基本上採取「不戰」政策——既不得罪秦國，也不積極援助其他五國——最終在秦始皇統一戰爭中不戰而降。

| 面向 | 說明 |
|------|------|
| **齊王建** | 齊國末代國君，在位 44 年（西元前 264—前 221 年），以消極避戰著稱 |
| **齊國的沒落** | 自齊湣王（西元前 300—284 年）被五國聯軍擊敗、齊國幾乎滅亡之後，齊國國力大衰，從東方霸主淪為二流強國 |
| **齊國的政治問題** | 據《戰國策》記載，齊王建時期「君王后」（王后）擅政，賢人不用、小人當道，國家治理日益敗壞——這正是趙威后在質問中批評的內容 |
| **齊使的角色** | 齊使代表的不僅是齊王，更是傳統的「貴族禮儀」觀點——先問君、後問民，這是春秋以來諸侯之間的外交慣例。趙威后顛覆這個慣例，本身就是一種政治宣言 |

| > **軍師按**：趙威后對齊使的質問，準確地指出了齊國的核心問題——不是沒有賢人，而是賢人不得重用。鍾離子、葉陽子、北宮之女——這些都是齊國的「國寶」，但齊國當權者視而不見；反而供養於陵子仲這種「不事君王、不治產業」的人。趙威后只問了幾個問題，就給齊國的政治治理做了一次「體檢」——而且診斷結果非常精準。後來齊國在秦國統一天下的過程中不戰而降，某種程度上印證了趙威后的憂慮：一個不重用賢人的國家，是沒有抵抗力的。|

## 三、原文與結構

### 3.1 原文（節選核心段落）

> 齊王使使者問趙威后。書未發，威后問使者曰：「歲亦無恙耶？民亦無恙耶？王亦無恙耶？」
>
> 使者不說，曰：「臣奉使使威后，今不問王而先問歲與民，豈先賤而後尊貴者乎？」
>
> 威后曰：「不然！苟無歲，何以有民？苟無民，何以有君？故有問，舍本而問末者耶？」

——第一回合：問候的順序

> 乃進而問之曰：「齊有處士曰鍾離子，無恙耶？是其為人也，有糧者亦食，無糧者亦食；有衣者亦衣，無衣者亦衣。是助王養其民者也，何以至今不業也？
>
> 葉陽子無恙乎？是其為人，哀鰥寡，卹孤獨，振困窮，補不足。是助王息其民者也，何以至今不業也？
>
> 北宮之女嬰兒子無恙耶？徹其環瑱，至老不嫁，以養父母。是皆率民而出於孝情者也，胡為至今不朝也？
>
> 此二士弗業，一女不朝，何以王齊國、子萬民乎？」

——第二回合：賢人政治的質詢

> 「於陵子仲尚存乎？是其為人也，上不臣於王，下不治其家，中不索交諸侯。此率民而出於無用者，何為至今不殺乎？」

——第三回合：對無用之人的質問

> ——《戰國策·齊策四》

### 3.2 論辯結構分析

本文的結構可以用「三個回合」來概括，層層深入：

| 階段 | 內容 | 分析 |
|------|------|------|
| **第一回合：顛覆禮儀** | 趙威后未拆信，先問「歲→民→王」 | 用一個動作（先問歲與民）直接挑戰了傳統的外交禮儀——這不是無心的失禮，而是有意為之的政治宣言 |
| **第二回合：質疑賢政** | 趙威后追問鍾離子、葉陽子、北宮之女為何不被重用 | 從「問候的順序」推向「政治治理的實質」——你不重用賢人，你怎麼治理國家？ |
| **第三回合：痛斥無用** | 趙威后質問於陵子仲為何不被懲治 | 從「不用賢人」推向「縱容無用之輩」——如果你既不用好人、又不罰壞人，你的國家還有希望嗎？ |

| > **軍師按**：這個三層結構非常精妙。第一層是「禮儀問題」——讓人覺得「這個太后怎麼不懂禮數」。但當你跟著她的思路走，你會發現：第二層上升到「政治問題」——她在質問齊國的用人政策。第三層更進一步——上升到「道德與社會問題」——她在質問齊國的價值標準。從一個看似簡單的問候，一路推演到對整個國家治理體系的批判——這就是戰國策士「以小見大」的經典修辭手法。而這一次用這個手法的人，不是策士，而是一個太后。|

### 3.3 對比閱讀：兩種「問候」

趙威后「先問歲與民，後問王」的順序，本身是一個強烈的象徵——它代表了兩種治理邏輯的對立：

| 邏輯 | 提問順序 | 代表什麼 | 潛在假設 |
|------|---------|---------|---------|
| **傳統禮儀** | 先問王 → 再問民 | 權力至上——國君是國家的代表，問候國君就是問候國家 | 國君 = 國家 |
| **民本政治** | 先問歲 → 再問民 → 最後問王 | 民生至上——年景好才有百姓好，百姓好才有國君好 | 百姓 = 國家的根基 |

| > **軍師按**：趙威后這裡做了一件極具顛覆性的事情——她不是在「談論」民本思想，而是在「實踐」它。她不是先說「我認為民比君重要」，而是直接用行動展示——她先問民，後問君。在政治哲學中，這叫做「performativity」（以言行事的效力）——你的行動本身，比你的言辭更有說服力。趙威后沒有解釋為什麼民比君重要，她直接用問候的順序證明了這一點。|

## 四、關鍵人物

### 4.1 趙威后——以「問」代「教」的哲學家太后

| 項目 | 內容 |
|------|------|
| **身份** | 趙惠文王王后，趙孝成王之母，趙國攝政太后 |
| **時代定位** | 戰國中期最著名的女性統治者之一，與秦宣太后並稱 |
| **核心思想** | 以民為本——統治者的正當性來自於對百姓生活的保障 |
| **性格特質** | 果斷、犀利、深諳論辯之道——以問代答、以退為進、用問題來回答問題 |
| **歷史定位** | 中國政治思想史上最早的「民本實踐者」——她不只是說了「民貴君輕」，她還用外交場合的行動來展現了這個理念 |

**軍師評**：趙威后是《古文觀止》中最被低估的人物之一。她既不是縱橫家（如蘇秦、張儀），也不是謀士（如范雎），更不是文學家——她是一個「用行動寫作」的政治家。她的「問」，本身就是一篇論文。她的三問——「歲無恙耶？民無恙耶？王無恙耶？」——幾乎可以說是中國政治思想史上最濃縮、最有力的民本宣言。而她隨後對齊國賢人政治的質詢，更展現了她不只是有理念，還有對具體政務的深入了解——她知道齊國有哪些賢人、做了哪些好事、應該被如何任用。這不是一個在深宮中養尊處優的太后，這是一個對國家治理有著深入觀察和獨立判斷的政治領袖。

### 4.2 齊使——傳統價值的代言人

| 項目 | 內容 |
|------|------|
| **身份** | 齊王派往趙國的使臣 |
| **出場時間** | 僅在和趙威后的對話中短暫出現 |
| **角色功能** | 代表「傳統禮儀」的立場——他對趙威后先問歲與民、後問王的順序感到不悅，認為這是「先賤後尊貴」 |
| **關鍵台詞** | 「臣奉使使威后，今不問王而先問歲與民，豈先賤而後尊貴者乎？」 |
| **作用** | 他是趙威后論證的「靶子」——沒有他的反駁，趙威后的「苟無歲，何以有民？苟無民，何以有君？」這句話就不會出現 |

**軍師評**：齊使的角色雖然只是「出場被罵」，但他非常重要——他是整個「傳統」的化身。他所說的「先賤而後尊貴」，代表了春秋戰國時期主流的社會價值觀：君貴民賤，天經地義。趙威后要打破的就是這個「天經地義」。齊使的存在，讓趙威后的反駁有了一個具體的對象——她不是在對空氣說話，而是在對一個真實的、來自傳統陣營的人說話。如果沒有這個人的反對，她的民本宣言就沒有那麼大的震撼力。

### 4.3 鍾離子、葉陽子、北宮嬰兒子——被埋沒的齊國賢人

| 人物 | 身份 | 善行 | 趙威后的質問 |
|------|------|------|-------------|
| **鍾離子** | 齊國處士（隱居的賢人） | 「有糧者亦食，無糧者亦食；有衣者亦衣，無衣者亦衣」——不分貧富，一律救濟——「助王養其民者也」 | 此等賢人，為何至今不授予官職？ |
| **葉陽子** | 齊國處士 | 「哀鰥寡，卹孤獨，振困窮，補不足」——關懷社會弱勢——「助王息其民者也」 | 此等賢人，為何至今不授予官職？ |
| **北宮嬰兒子** | 齊國女子（北宮之女） | 「徹其環瑱，至老不嫁，以養父母」——放棄嫁妝，終身不嫁以奉養雙親——「率民而出於孝情者也」 | 此等孝女，為何至今不受朝廷表彰？ |

**軍師評**：趙威后挑選這三個人的眼光非常精準——她不是在批評齊國沒有賢人，而是在批評齊國「有賢人而不用」。鍾離子和葉陽子的善行，本質上是幫齊王在做的「社會福利」工作——他們在幫國君養百姓、安百姓。齊國不用他們，等於白白放著兩個「免費的公務員」不用。而北宮嬰兒子的行為，是「孝」的極致——齊國不表彰她，等於是在告訴天下：孝道不重要。趙威后的質問背後的邏輯是：**治理國家說到底就是兩件事——用好人來幫助百姓，用表彰來引導風氣。這兩件事你都沒做好，你還算什麼國君？**

### 4.4 於陵子仲——社會的寄生者

| 項目 | 內容 |
|------|------|
| **身份** | 齊國隱士（於陵地方人士） |
| **問題行為** | 「上不臣於王，下不治其家，中不索交諸侯」——既不服侍國君，也不管理家業，也不與諸侯往來——完全是一個「與社會無關」的人 |
| **趙威后的態度** | 「此率民而出於無用者，何為至今不殺乎？」——這種帶頭教人做無用之事的人，為什麼還不處置他？ |
| **與鍾離子等人的對比** | 賢人不用，無用之人卻被縱容——齊國的價值標準完全顛倒了 |

**軍師評**：趙威后對鍾離子等人是「為何不重用？」的惋惜語氣，對於陵子仲卻是「為何不殺？」的嚴厲語氣——這一正一反，構成了完整的政治批判。她不是反對隱士——顏斶也是隱士，但趙威后對顏斶這種「不慕權勢、獨善其身」的隱士是尊重的。於陵子仲的問題在於：他什麼都不做——不事君、不治家、不交友——對社會沒有任何貢獻，卻還能活得好好的。這在趙威后看來，就是「帶壞社會風氣」的源頭。**一個社會如果容忍不貢獻的人，那麼努力貢獻的人就會失去動力。** 趙威后對齊國的批評，歸根結底就是一句話：你們的價值標準顛倒了——該賞的不賞，該罰的不罰。這在任何時代、任何社會，都是政府失能的根源。

## 五、核心分析——「苟無歲，何以有民？苟無民，何以有君？」

### 5.1 論證的三層邏輯

趙威后看似只問了幾個問題，但她的論證邏輯其實非常嚴密，可以拆解為三個層次：

| 層次 | 內容 | 論證邏輯 |
|------|------|---------|
| **第一層：根基論** | 歲→民→王的因果鏈 | 收成是百姓生存的基礎 → 百姓是國家存在的基礎 → 問候的順序應該從最根基的開始。趙威后不是在「失禮」，她是在用問候的順序來展示「什麼才是真正重要的」 |
| **第二層：賢人論** | 賢人不用，等於放棄治理的工具 | 鍾離子、葉陽子是在幫國君養民安民——不用他們，等於國君自己不做這些事。北宮嬰兒子的孝行應該被表彰——不表彰她，等於否定孝道的社會價值。這對齊國社會風氣的影響是深遠的 |
| **第三層：懲戒論** | 無用之人不懲罰，等於鼓勵無用 | 於陵子仲的存在本身就是一個負面教材——他什麼都不做，卻可以安然無恙。這在向社會傳遞一個訊息：你可以不事君、不治家、不交友，什麼貢獻都不做——照樣可以活得好好的。這對社會秩序的破壞力，遠大於一個強盜 |

| > **軍師按**：這三層論證，從「哲學命題」（什麼是根本）到「政策問題」（為什麼重要）到「社會影響」（不管會怎樣）——層層遞進，無懈可擊。最精彩的是，趙威后始終沒有用「我認為應該如何」的語氣，而是用「請問為什麼」的語氣——她把問題丟回給齊使，讓齊使自己去思考答案。這比直接教訓人要高明一千倍。|

### 5.2 「苟無歲，何以有民？苟無民，何以有君？」——一句話的顛覆力

這句話是全文的靈魂。它的力量在於：它把傳統的權力邏輯完全翻了過來。

| 邏輯 | 傳統觀點 | 趙威后的觀點 |
|------|---------|-------------|
| **權力的來源** | 國君受命於天，權力由上而下 | 國君的權力來自於百姓的存在——沒有百姓，國君什麼都不是 |
| **施政的重心** | 維護國君的權威和利益 | 保障百姓的生存和福祉——因為這是國君權力的前提 |
| **問候的順序** | 先問君，後問民（君為貴，民為賤） | 先問民，後問君（民為本，君為末） |

趙威后的邏輯鏈條非常簡單且無懈可擊：**歲（收成）→ 民（百姓）→ 君（國君）**。這是一條「存在論」的因果鏈——沒有收成，百姓活不下去；沒有百姓，國君就不成其為國君。所以，問候的順序應該從上游問到下游：先問歲，再問民，最後問君——而不是反過來。

| > **軍師按**：趙威后的這句話，在某種意義上比孟子的「民為貴，社稷次之，君為輕」更深刻——因為孟子說的是一個「道德主張」（你應該這樣想），而趙威后說的是一個「事實陳述」（實際上是這樣）。前者是「你應該把百姓放在最高位置」，後者是「沒有百姓就沒有國君——這是事實，不是價值判斷」。事實陳述比道德主張更難反駁——你可以不同意一個道德主張，但你無法否認一個事實。這就是趙威后的厲害之處——她不是用道德來要求齊使，而是用邏輯來說服他。|

### 5.3 趙威后的「用人觀」——賞罰分明的治理哲學

趙威后對齊國政治的批評，歸根結底是「賞罰不分」四個字：

| 應該做的 | 齊國做了嗎？ | 結果 |
|---------|------------|------|
| 獎勵鍾離子、葉陽子這種幫國君養民的賢人 | 沒有授予官職 | 賢人寒心，社會失去正面榜樣 |
| 表彰北宮嬰兒子這種孝女 | 沒有朝廷表彰 | 孝道不受重視，社會風氣敗壞 |
| 懲罰於陵子仲這種無益於社會的人 | 沒有懲罰，反而任其逍遙 | 無用之人不受制裁，社會失去底線 |

| > **軍師按**：趙威后的治理哲學可以用一句話概括：**賞善罰惡，各得其所。** 鍾離子和葉陽子是「善」——應該被賞（授予官職，讓他們發揮作用）；北宮嬰兒子是「孝」——應該被表彰（列入朝廷禮儀，引導社會風氣）；於陵子仲是「惡」——雖然他沒有違法，但他的存在本身就是對「積極貢獻社會」這一價值觀的侵蝕——應該被懲罰（至少不能讓他繼續逍遙）。趙威后對齊國的批評，說到底就是在說：**你們的價值標準亂了——好人沒有好報，壞人沒有惡報。一個這樣的國家，還能維持多久？** 而歷史的答案是：在趙威后問出這個問題之後不到五十年，齊國就滅亡了。|

## 六、軍師核心分析——趙威后的「問政」智慧

### 6.1 以「問」為「教」——最高明的說服術

趙威后整篇文章沒有說一句「你應該如何」——她全部用問題來表達觀點。這是一種極其高明的說服策略：

| 提問 | 表面意思 | 深層含義 |
|------|---------|---------|
| 「歲亦無恙耶？民亦無恙耶？王亦無恙耶？」 | 我在關心齊國 | 我在告訴你，什麼東西才是真正重要的 |
| 「何以至今不業也？」（鍾離子） | 我在問一個人事問題 | 我在批評你們不用賢人 |
| 「何以至今不業也？」（葉陽子） | 同上 | 同上 |
| 「胡為至今不朝也？」（北宮嬰兒子） | 我在問一個禮儀問題 | 我在批評你們不重視孝道 |
| 「何為至今不殺乎？」（於陵子仲） | 我在問一個法律問題 | 我在批評你們縱容無用之人 |

| > **軍師按**：趙威后使用的，是蘇格拉底式的「產婆術」——用問題引導對方自己得出結論，而不是直接把結論塞給對方。她問齊使「歲無恙耶？民無恙耶？王亦無恙耶？」——齊使如果回答「歲好、民好、王好」，那就等於承認了「歲和民應該被先問到」。如果齊使回答「歲不好、民不好」，那就更糟——連你自己都承認國家治理有問題。不管齊使怎麼回答，他都會掉進趙威后的邏輯陷阱裡。這就是「問話」的力量——**掌控問題的人，就是掌控對話的人。** |

### 6.2 「先問歲與民」——一個姿勢背後的整個哲學

趙威后「未發書而先問歲與民」，這個動作本身就是整篇文章最精華的部分。它不僅僅是一個「問候順序」的問題，而是一個關於「執政者應該如何看待自己的權力」的哲學問題。

| 面向 | 傳統君主的邏輯 | 趙威后的邏輯 |
|------|---------------|-------------|
| **權力的基礎** | 我的權力來自我的地位 | 我的權力來自百姓的承認 |
| **施政的優先順序** | 先鞏固我的權力 | 先保障百姓的生活 |
| **與百姓的關係** | 百姓應該服從我 | 我應該服務百姓 |
| **問候的順序** | 先君後民 | 先民後君 |

趙威后的「先問歲與民」，是她整個執政哲學的「濃縮版」——她不需要寫一本書來論述她的政治理念，她只需要調整一個問候的順序，就把一切說清楚了。

| > **軍師按**：在政治傳播中，有一個核心原則叫做「symbolic action」（象徵性行動）——用一個小而具體的行動來表達一個大而抽象的理念。趙威后「先問歲與民」的動作，就是一個完美的象徵性行動——它不需要解釋，不需要補充，所有人都看得懂：這位太后認為，百姓比國王更重要。而在外交場合做這個動作，等於在國際社會面前公開宣示這個立場——這比在國內頒布一百道詔令都更有影響力。|

### 6.3 趙威后與其他「問政」名篇的對比

《古文觀止》中有好幾篇以「問」為核心的文章，可以互相對照：

| 篇目 | 提問者 | 提問對象 | 提問目的 |
|------|--------|---------|---------|
| **〈趙威后問齊使〉** | 趙威后（執政者） | 齊使（外國使者） | 批評齊國政治，宣示民本理念 |
| **〈鄒忌諷齊王納諫〉** | 鄒忌（臣子） | 齊威王（國君） | 引導國君廣開言路 |
| **〈顏斶說齊王〉** | 顏斶（隱士） | 齊宣王（國君） | 宣示士人的尊嚴 |
| **〈曹劌論戰〉** | 曹劌（平民） | 魯莊公（國君） | 檢驗國君的執政準備 |

趙威后的獨特之處在於：她是唯一一個以「執政者」身份提問的人，也是唯一一個「對外國使者」提問的人。她不是在為自己的利益說話，而是在用一個外國的案例來宣示一種普遍的治理哲學。

| > **軍師按**：如果把這四篇「問政」文章放在一起讀，你會發現一個有趣的模式——所有的提問者都是「挑戰者」：他們在用提問來質疑既有的權力結構和思維方式。鄒忌挑戰的是「國君不聽諫言」的習慣；顏斶挑戰的是「君貴臣賤」的觀念；曹劌挑戰的是「肉食者鄙」的階級偏見；趙威后挑戰的是「先君後民」的禮儀傳統。四個人用不同的方式，在做同一件事：用「問」來打破「理所當然」。|

### 6.4 趙威后的歷史定位——被低估的女性政治思想家

趙威后在中國政治思想史上的地位，長期被低估。原因可能有兩個：

1. **性別因素**——中國傳統史料對女性政治人物的記錄本來就稀少，而且往往給予負面評價（如呂后、武則天）
2. **文本的篇幅**——〈趙威后問齊使〉全文不到三百字，比〈顏斶說齊王〉還要短——篇幅的簡短可能讓後人低估了它的思想深度

但事實上，趙威后提出的「苟無歲，何以有民？苟無民，何以有君？」——是中國政治思想史上最簡潔、最精準、最無可辯駁的「民本論證」。它比《孟子》的「民貴君輕」更具操作性（因為它直接指向了政策問題），比《尚書》的「民惟邦本」更具邏輯性（因為它建立了一個因果鏈條），比《左傳》的「國將興，聽於民」更具體（因為它點名了具體的人與事）。

| > **軍師按**：在中國政治思想史上，「民本」是一個被反覆提起的概念，但大多數時候它停留在道德呼籲的層次——「你應該愛民如子」。趙威后把這個概念往前推了一步：她不是在說「你應該愛民」，而是在說「你沒有民就什麼都不是」——這是一個事實，不是一個道德要求。當你把一個道德問題轉換為一個事實問題，你就從「說服別人做好事」進化到了「說服別人理解真相」。前者需要情感，後者只需要邏輯。趙威后用的是後者——這讓她超越了所有同時代的「民本論述者」。|

## 七、寫作藝術與修辭特色

### 7.1 本文的修辭手法

| 手法 | 內容 | 效果 |
|------|------|------|
| **設問（以問代答）** | 趙威后整篇文章都是用問題來表達觀點 | 不是把結論塞給對方，而是引導對方自己思考——比直接批評更有說服力 |
| **層層遞進** | 從問候順序 → 賢人不用 → 無用之人不罰 | 從禮儀問題逐步深入到政治治理的核心問題——每一層都比前一層更尖銳 |
| **對比** | 鍾離子、葉陽子（賢人不用）vs. 於陵子仲（無用之人不罰） | 正反對比凸顯齊國價值標準的顛倒——賞罰不分，是非不明 |
| **排比** | 「何以至今不業也？」（對鍾離子）→「何以至今不業也？」（對葉陽子）→「胡為至今不朝也？」（對北宮嬰兒子）→「何為至今不殺乎？」（對於陵子仲） | 四連問，一次比一次嚴厲——從「為什麼不任用」到「為什麼不殺」，語氣逐步升級 |
| **歸謬法** | 「豈先賤而後尊貴者乎？」——把對方的邏輯推到荒謬 | 你不是說「先問民是失禮」嗎？那我告訴你：如果你覺得先問民是錯的，那你的邏輯就是「沒有百姓也無所謂」——你自己覺得這個邏輯站得住嗎？ |

### 7.2 一個動作的張力——「書未發」

「齊王使使者問趙威后。書未發，威后問使者曰」——趙威后連使者帶來的問候信都還沒有打開，就開口問了。

| 要素 | 效果 |
|------|------|
| **「書未發」** | 三個字製造了極大的懸念——為什麼她連信都不看就開始問？ |
| **象徵意義** | 信還沒有打開、禮節還沒有開始，趙威后就直接問了「歲與民」——這說明對她來說，這些問題比任何禮儀更重要 |
| **戲劇張力** | 「書未發」代表著「還沒按照劇本來」——趙威后拒絕按照傳統的外交劇本行事，她要創造自己的劇本 |

| > **軍師按**：全文最精采的三個字，可能就是「書未發」。它不僅僅是一個時間描述，更是一句潛台詞：**我不需要看齊王寫了什麼，因為我知道齊王寫的一定是一些客套話。而我要問的，不是客套話能回答的。** 這是多麼高傲、多麼自信的一個開場——她連看都不看對方的來信，就開始用自己的問題主導整個對話。|

### 7.3 語氣的三次轉折

趙威后的語氣在文章中經歷了三次明顯的轉折：

| 階段 | 語氣 | 台詞 |
|------|------|------|
| **第一問：平和關心** | 關心式的問話，聽起來像是普通的外交寒暄 | 「歲亦無恙耶？民亦無恙耶？王亦無恙耶？」 |
| **第二問：質問批評** | 語氣轉為嚴厲，開始正面批評齊國的政治 | 「此二士弗業，一女不朝，何以王齊國、子萬民乎？」 |
| **第三問：憤怒痛斥** | 語氣達到最嚴厲的程度——「何為至今不殺乎？」 | 從質疑用人政策升級到質疑整個國家的價值標準 |

| > **軍師按**：語氣的三次轉折，讓整篇文章形成了一個「漸進式壓迫」的效果——從溫柔的關心開始，逐漸加大力度，到最後一句「何為至今不殺乎？」已經是當面打臉了。齊使從一開始的「不說」（不高興），到後來應該已經完全說不出話來——因為趙威后的問題一個比一個難回答，一個比一個致命。這就是文章結構的力量——**不是一開始就出重拳，而是先出一個輕拳，讓對方覺得你不過如此；然後再加一點力道；等到對方發現你已經打到要害的時候，已經來不及反擊了。** |

## 八、軍師總論

### 8.1 本文的四層意義

| 層次 | 內容 | 啟示 |
|------|------|------|
| **第一層：民本思想** | 趙威后以「先問歲與民」的動作，將民本思想從理論變為實踐 | 理念不是用來說的，是用來做的——一個行動勝過一千句口號 |
| **第二層：賢人政治** | 批評齊國不用賢人、不表彰孝行、不懲罰無用之人 | 政治治理的核心是「價值標準」——你獎勵什麼、懲罰什麼，決定了你的國家會變成什麼樣子 |
| **第三層：問政藝術** | 以問題代替答案，引導對方自己得出結論 | 最高明的說服，不是告訴別人答案，而是讓別人自己發現答案 |
| **第四層：女性領導力** | 在戰國時代以女性身份公開發表政治批評，並展現出超越同時代男性的政治智慧 | 領導力與性別無關——趙威后用行動證明了女性統治者在政治智慧和判斷力上絲毫不遜於男性 |

### 8.2 與其他篇目的互文

- **〈觸讋說趙太后〉（卷四）**——同為趙國太后的故事。觸讋說服趙威后（注意：此趙太后是趙孝成王之妻、趙悼襄王之母——不是同一個人，而是另一位趙太后）送長安君為質。兩篇合看，可以完整理解趙國太后攝政時期的政治生態——〈趙威后問齊使〉展現了一位強勢、有為的太后形象；〈觸讋說趙太后〉則展現了一位一開始不夠清醒、但最終願意聽取諫言的太后形象。一強一弱，但都是「女性統治者」的真實寫照

- **〈顏斶說齊王〉（卷四）**——同出《戰國策·齊策四》。顏斶說「士貴耳，王者不貴」，強調士人的尊嚴；趙威后說「苟無歲，何以有民？苟無民，何以有君？」強調百姓的優先地位。兩篇合讀，可以看到《戰國策·齊策四》的一條潛在脈絡：**對權力進行反思**——可以是從「士」的角度（顏斶），也可以是從「民」的角度（趙威后），但核心都是對傳統權力結構的質疑

- **〈孟子見梁惠王〉（卷三）**——孟子對梁惠王說「何必曰利？亦有仁義而已矣」——同樣是在質問國君的政治立場。但孟子的語氣是「教訓」（我告訴你什麼是對的），趙威后的語氣是「問話」（你覺得這樣對嗎？）。兩相比較，趙威后的說服策略其實更高明——因為被質問的人往往比被教訓的人更容易接受你的觀點

- **〈曹劌論戰〉（卷一）**——曹劌問魯莊公「何以戰」——通過三個問題來檢驗魯國的戰備（是否得民心）。趙威后問齊使「歲、民、王」——也是用問題來檢驗齊國的治理。曹劌和趙威后用的都是「以問代答」的策略——不同的是，曹劌是平民問國君，趙威后是太后問外國使者

- **〈鄒忌諷齊王納諫〉（卷四）**——鄒忌以自身經歷為比喻，引導齊威王廣開言路。兩篇文章的相似之處在於：都是「引導式說服」（不是直接告訴對方答案，而是讓對方自己悟出來）。差別在於：鄒忌用的是日常生活的比喻（妻之美我者，私我也），趙威后用的是政治邏輯的推演（歲→民→君的因果鏈）

### 8.3 現代啟示

1. **先問根本問題**：在任何工作或決策中，先問「最根本的條件是否存在」——就像趙威后先問年景一樣。不要一上來就討論高層次問題，先確認地基是否穩固

2. **用問題引導思考**：當你需要批評或說服別人時，不要直接說「你錯了」——用問題引導對方自己發現錯誤。被自己發現的錯誤，比被別人指出的錯誤更容易被接受

3. **價值標準決定一切**：一個組織的興衰，往往取決於它獎勵什麼、懲罰什麼。如果好人得不到獎勵、壞人不受懲罰——這個組織遲早會崩潰。趙威后對齊國的批評，放到現代企業管理中依然完全適用

4. **不要被「禮儀」綁架**：齊使之所以生氣，是因為趙威后「失禮」了。但趙威后用行動證明：有時候，正確的「失禮」遠比錯誤的「禮貌」更有價值。不要為了遵守形式而犧牲實質

5. **女性領導者的獨特優勢**：趙威后展現的「問政」風格——細膩、精準、不正面衝突但直擊要害——是女性領導者獨特的溝通優勢。在現代職場中，這種「以柔克剛」的溝通方式往往比男性的強勢對抗更有效

| > **軍師總評**：〈趙威后問齊使〉是《古文觀止》中最短、最精悍、最被低估的文章之一。全文不到三百字，卻涵蓋了政治哲學（民本思想）、政策分析（賢人政治）、修辭藝術（以問代答）、外交智慧（象徵性行動）、和人性洞察（賞罰分明的治理邏輯）——五個層次濃縮在一篇簡短的對話中。趙威后這個人物，在某種意義上是《古文觀止》中最特別的「軍師」——她不是軍師，她是「以統治者之身行軍師之智」的人。她的問題比別人的答案更有力量，她的行動比別人的宣言更有說服力。當她在兩千多年前打開齊使的問候信之前，先問「歲亦無恙耶？民亦無恙耶？」的時候，她已經用一個簡單的動作，完成了對戰國時代所有「君權至上」論者的最致命一擊——因為她證明了：真正的權力，不是來自於你命令了多少人，而是來自於你養活了多少人。|

## 九、一句話總結（再錄）

> **趙威后是中國歷史上第一個將「以民為本」從口號變成執政邏輯的政治家——她不是不知道禮儀的重要性，而是她知道什麼才是真正重要的。當所有人都覺得「先問君、後問民」是天經地義的時候，她用一句「苟無歲，何以有民？苟無民，何以有君？」就把整個權力邏輯翻了過來：不是國君養活百姓，而是百姓養活國君——所以問候的順序，應該從根源問起。兩千多年後，當我們重讀這篇短短三百字的對話時，仍然能感受到那種震撼——一個女人，一句問話，就把兩千年來的「天經地義」打得粉碎。而更令人敬畏的是，她不是用憤怒來打碎它的，而是用邏輯——最冷靜、最簡單、最不可否認的邏輯。這不是一個太后的驕傲，這是一個思想家的自信。**

### 10.2 經典延伸

- 《戰國策·齊策四》——〈趙威后問齊使〉的完整原文，以及齊策中其他關於齊國政治的篇章
- 《戰國策·趙策》——趙國相關史料，包括趙惠文王、趙孝成王時期的政治記錄

- 《孟子·盡心下》（孟子）——「民為貴，社稷次之，君為輕」——可與趙威后的民本思想對照閱讀
- 《資治通鑑》卷五（司馬光）——對趙國長平之戰前後的系統記載
- 楊寬《戰國史》——對戰國各國政治制度與治理思想的學術研究

### 10.3 現代延伸

- 漢娜·鄂蘭《人的條件》——關於「公共領域」與「行動」的哲學：趙威后的「問政」本身就是一種「公共行動」——她在用行動創造一個討論「什麼是好的治理」的公共空間
- 詹姆斯·斯科特《國家的視角》——關於國家治理如何「看見」百姓的生活——趙威后「先問歲與民」的動作，本質上就是在告訴統治者：不要只看到國君的權力，要看到百姓的生存
- 史蒂芬·柯維《與成功有約》——「以終為始」（Begin with the end in mind）——趙威后「先問歲與民」就是一種「以終為始」的思維：先確認最根本的目標（百姓安居樂業），再倒推治理策略
- 羅伯特·席爾《影響力》——關於「社會認同」與「說服策略」——趙威后用問題引導齊使，是「影響力」的最高形式：讓對方自己得出結論，而不是強迫對方接受
