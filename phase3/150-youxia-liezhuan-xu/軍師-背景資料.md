## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《史記·游俠列傳》序——〈太史公自序〉中亦提及作傳動機 |
| **選集** | 《古文觀止》卷五·史記文類 |
| **體裁** | 史傳體序文／人物列傳序——兼有議論與敘事，實為一篇尖銳的社會批評 |
| **時代** | 西漢武帝時期（約西元前 91 年，《史記》成書前後），所記為漢興以來游俠的興衰 |
| **核心命題** | 當法律失靈時，誰來主持正義？——「俠」的社會功能及其倫理正當性 |
| **關鍵人物** | 司馬遷（太史公，為游俠立傳者）、郭解（漢代游俠代表，被漢武帝誅殺）、朱家（魯國游俠，救季布）、劇孟（洛陽游俠，周亞夫之客） |
| **文本重點** | 這是《史記》中最大膽的一篇文章之一。司馬遷為「游俠」正名——這些人「言必信，行必果，諾必誠」，在體制無法解決的問題面前挺身而出。他批評儒生墨守成規而無補於實際，讚美游俠雖然「不軌於正義」（行為不合官方正義），卻能解決官場和法律無法解決的冤屈。他在開篇引用了韓非子的矛盾——「儒以文亂法，而俠以武犯禁」——但他翻轉了這個判斷：儒者之亂法是虛偽，俠者之犯禁是正義 |
| **思想譜系** | 出自《史記》七十「列傳」中的〈游俠列傳〉——與〈刺客列傳〉、〈貨殖列傳〉、〈滑稽列傳〉等同為司馬遷「邊緣人物」系列的傑作。這些傳記的共同特點是：司馬遷將目光投向了正統史學不屑一顧的群體——游俠、刺客、商人、優伶——為他們立傳，賦予他們在歷史中的位置。相較於〈伯夷列傳〉中對「天道無親」的困惑，〈游俠列傳序〉更進一步——它直接向「制度正義」發起了挑戰：如果法律本身就是不公的，違法是否反而是正義？這是司馬遷思想中最激進、最大膽的一面 |
| **一句話總結** | **司馬遷為游俠辯護是在冒一個極大的風險——他等於在說：當制度失靈時，制度外的手段不僅合理，而且必要。這不是對秩序的挑戰，而是對虛偽的秩序的不屑。** |

## 二、時代背景——西漢武帝朝的法律、社會與司馬遷的「李陵之禍」

### 2.1 漢武帝的「酷吏政治」與游俠的對立面

漢武帝在位五十四年（西元前 141–前 87 年），是西漢從「文景之治」的休養生息轉向「積極有為」的關鍵時期。武帝一方面對外用兵——征伐匈奴、開拓西域、征服南越——另一方面對內加強中央集權。後者塑造了游俠所處的直接政治環境：

| 面向 | 說明 |
|------|------|
| **酷吏的興起** | 漢武帝為了打壓地方豪強與貴族勢力，大量啟用「酷吏」——如張湯、杜周、王溫舒等。這些人以嚴刑峻法聞名，濫用權力、羅織罪名，社會上層的冤獄層出不窮。酷吏的存在，使得「法律」不再是一種公正的工具，而成了權力打壓異己的手段 |
| **地方豪強的困境** | 戰國至漢初，地方上存在著大量「豪強」——他們或為舊貴族後裔，或為地方大族，憑藉財力、武力、人脈在地方上擁有極大的影響力。漢武帝的中央集權政策，就是要削弱這股力量。游俠往往與這些豪強有密切的關係——他們本身就是地方社會的「非正式權力中心」 |
| **法律的不公** | 在一個酷吏橫行的政治環境中，「打官司」常常不是為了正義，而是為了毀滅敵人。誰有權力、誰有錢、誰認識酷吏——誰就能贏得官司。在這種情況下，法律失去了它最基本的社會功能——保護弱者、懲罰不義 |
| **游俠的社會角色** | 正是在這樣的背景下，游俠承擔了一個重要的社會功能：在法律無法主持正義的時候，他們用個人的力量——名聲、人脈、勇氣、武力——來幫助那些被冤枉、被欺凌、走投無路的人。游俠不是要顛覆秩序，而是要填補秩序崩壞後的空缺 |

> **軍師按**：理解〈游俠列傳序〉的關鍵，在於理解「酷吏政治」與「游俠」之間的共生關係。酷吏越猖獗，法律越腐敗，游俠的社會角色就越重要、越正當。司馬遷之所以敢於為游俠辯護，不是因為他不重視法律——恰恰相反，正是因為他太重視法律的公正性，所以當法律變成權力的工具時，他才會站出來說：當法律不再公正時，那些用個人力量維護公道的人——即使「不軌於正義」——也值得尊敬。

### 2.2 司馬遷寫游俠的個人動機——「李陵之禍」的傷痕

司馬遷為游俠立傳，不是一個冷靜的史學選擇——它是一次極其個人化的、充滿風險的社會批評。司馬遷本人對「法律不公」的感受，比任何人都深刻：

| 面向 | 說明 |
|------|------|
| **司馬遷的遭遇** | 西元前 99 年，李陵率五千步兵出征匈奴，以少戰多、殺敵過萬，最終因孤立無援而降。司馬遷為李陵辯護，觸怒漢武帝，被處以宮刑——這是漢朝法律對他的「懲罰」。但這個懲罰公正嗎？司馬遷認為不公正——他只是說了一句公道話 |
| **法律作為權力的工具** | 司馬遷親身經歷了「法律被權力扭曲」的過程——漢武帝用法律的名義懲罰了一個說實話的人。這讓司馬遷深刻地認識到：法律本身並不天然地代表正義——法律只是權力的工具，當權力腐敗時，法律就是腐敗的 |
| **游俠的對照組** | 司馬遷在游俠身上看到了與自己命運的對照：那些被法律拋棄的人——無論是李陵還是他自己——都沒有得到制度的保護。但游俠用自己的方式保護了那些被拋棄的人。司馬遷寫游俠，在某種程度上也是在為自己尋找答案——當制度遺棄了你，誰會來救你？ |
| **最大的風險** | 在漢武帝朝為游俠辯護，是一個極其危險的行為——因為漢武帝本人就是打壓游俠最嚴厲的人（郭解就是被漢武帝親自下令誅殺的）。司馬遷等於在說：皇權所打壓的人，其實是社會需要的正義捍衛者。這幾乎是對皇權的直接挑戰 |

> **軍師按**：〈游俠列傳序〉之所以是《史記》中最「大膽」的文章之一，不是因為它的文采——雖然文采確實極好——而是因為它觸及了一個權力絕對無法容忍的命題：**如果法律本身就不公正，那麼反抗法律是否反而是正義的？** 這個問題在任何時代都是禁忌——在漢武帝的時代更是如此。司馬遷敢於寫出這個問題，本身就是一種「俠氣」。

### 2.3 漢代游俠的歷史脈絡——從戰國到西漢的演變

游俠不是漢代才有的現象。戰國時代的「士」階層——包括刺客、門客、遊說之士——就是游俠的前身：

| 時期 | 游俠的形態 | 代表性人物 |
|------|-----------|-----------|
| **戰國時代** | 列國爭霸，諸侯養士——如戰國四公子（孟嘗君、平原君、信陵君、春申君）門下數千食客。這些食客中就有大量游俠——他們為主人效力，以「士為知己者死」為信條。此時的游俠依附於貴族，尚未成為獨立的社會力量 | 豫讓、聶政、荊軻——〈刺客列傳〉中的主人公，都是戰國游俠的極致——他們用生命實踐了「諾必誠、死必赴」的信條 |
| **秦末漢初** | 天下大亂，群雄並起——游俠從貴族門下走向民間。劉邦本人年輕時就有「俠氣」——他「仁而愛人，喜施」，但「不事家人生產作業」——某種程度上也是一個游俠式的角色。漢初的游俠開始在地方社會中發揮獨立作用 | 劉邦（早年）、張耳、陳餘——都是「好俠」之人，後來成為漢朝的開國功臣 |
| **文景之治** | 天下安定，中央權力尚未全面滲透地方——游俠在地方社會中扮演著類似「民間法官」的角色。他們調解糾紛、救濟窮困、保護弱者——在百姓眼中，游俠比官府更值得信任 | 朱家（魯國游俠，救濟數百人）、劇孟（洛陽游俠，周亞夫平定七國之亂時說「吳楚舉大事而不求劇孟，吾知其無能為矣」） |
| **漢武帝時期** | 中央集權大大增強，游俠成為皇權的打壓對象——因為游俠的地方影響力威脅了國家的權力壟斷。漢武帝對游俠進行了大規模的清洗——其中最著名的事件就是誅殺郭解 | 郭解——漢代游俠的最後高峰——他的被殺，象徵著游俠時代的終結 |

> **軍師按**：游俠的興衰，與中央權力的強弱成反比。中央權力越弱，地方社會越需要游俠來填補權力真空——這就是戰國到秦末游俠興盛的原因。中央權力越強，游俠的生存空間就越小——這就是漢武帝時期游俠被系統性打壓的原因。司馬遷寫〈游俠列傳〉的時候，游俠已經處於消亡的邊緣——他的這篇序，既是對游俠的頌歌，也是對這個群體的悼亡。

## 三、原文結構與核心段落

### 3.1 〈游俠列傳序〉的結構

〈游俠列傳序〉作為《史記·游俠列傳》的開篇，其結構極為嚴謹——從引述批評、到層層翻轉、到確立立場，形成了一個完美的辯論結構：

| 層次 | 內容 | 用意 |
|------|------|------|
| **第一層：引用韓非子的批評** | 司馬遷一開篇就引用了韓非子的名言——「儒以文亂法，而俠以武犯禁」——儒者用學術擾亂法律，游俠用武力觸犯禁令。這兩種人，在韓非子看來都是法治的破壞者，都是國家應該消滅的對象。司馬遷先說「二者皆譏」——這兩種人都被批評——但隨即話鋒一轉：儒者雖然亂法，但從孔子到今天，儒者一直受到尊重——而那些批評儒者的人，反而被罵。那麼游俠呢？ | 先揚後抑——表面上是引用批評，實際上是在為翻轉做準備。他先承認「俠以武犯禁」這個事實，但接著要問的是：犯禁是否等於邪惡？ |
| **第二層：翻轉韓非子的判斷** | 司馬遷說：那些真正的儒者——如孔子的弟子——雖然沒有官職，但他們的言行足以影響社會。而游俠呢？——「今游俠，其行雖不軌於正義，然其言必信，其行必果，已諾必誠，不愛其軀，赴士之厄困」——他們的行為雖然不合官方的「正義」，但他們說話一定守信用，做事一定有結果，答應的事一定誠心做到，不惜犧牲自己的生命去幫助陷入困境的人。司馬遷說：單就這一點，他們就值得尊敬 | 關鍵的翻轉——「不軌於正義」是最重的一句話。司馬遷承認游俠「不合官方正義」，但他暗示：官方正義不等於真正的正義。游俠雖然不守法，但他們守信、重諾、救人——這些品質比法律條文更重要 |
| **第三層：對比——儒生的虛偽 vs. 游俠的真實** | 司馬遷指出：那些自稱君子的人——穿著儒服、說著仁義——但在別人真正遇到困難時卻袖手旁觀。而游俠呢？他們不說漂亮話，但他們真的救人。司馬遷說：「要取足當世之務，以取重於世」——游俠的意義在於他們真正地解決了現實中的問題 | 儒俠對比——這是司馬遷最尖銳的批評。他不是在罵所有的儒生——他罵的是那些「有君子之名而無君子之實」的人。游俠雖然沒有君子的名號，但有君子的實際行為 |
| **第四層：舉例——朱家、劇孟、郭解** | 司馬遷舉出了漢代游俠的代表人物——朱家救了季布（劉邦的通緝犯），劇孟在七國之亂中發揮了關鍵作用，郭解更是「天下無賢與不肖，知與不知，皆慕其名」——無論是否認識他，都敬佩他的為人。郭解雖然最終被殺，但他的名聲永遠不會消失 | 用實例來證明游俠的價值——這些不是傳說中的人物，而是當代的真實存在。他們的生命證明了游俠的價值——即使官方要消滅他們，百姓仍然記得他們 |
| **第五層：結論——游俠的不朽** | 司馬遷總結：真正的社會價值，不在於是否符合官方標準，而在於是否真正地幫助了那些需要幫助的人。游俠雖然被官方打壓、被儒生批評、被歷史遺忘——但他們在民間的名聲，是任何權力都無法抹去的。他以「委曲取容」與游俠的「言必信、行必果」對比——那些在官場中委曲求全的人獲得了權力，但他們在歷史上沒有留下任何痕跡 | 全文的結論——也是司馬遷對游俠的終極評價：權力可以殺死一個人，但不能殺死他的名聲。郭解死了，但他的精神還活著 |

> **軍師按**：這篇序的辯證結構極為高明——它不是一篇簡單的「為游俠叫屈」的文章，而是一篇重新定義「正義」的文章。司馬遷的邏輯是這樣的：（1）官方正義（法律）不一定是真正的正義；（2）當官方正義失靈的時候，那些用個人力量維護公道的人——即使不合法——也是正義的；（3）游俠就是這樣的人。這個邏輯鏈條，每一環都是對官方意識形態的挑戰——而司馬遷把它們一層層地拆開，讓讀者不得不跟著他的思路走。

### 3.2 核心段落節錄

> 韓子曰：「儒以文亂法，而俠以武犯禁。」二者皆譏，而學士多稱於世云。……今游俠，其行雖不軌於正義，然其言必信，其行必果，已諾必誠，不愛其軀，赴士之厄困。既已存亡死生矣，而不矜其能，羞伐其德，蓋亦有足多者焉。

> ——全文的核心論點——司馬遷完全承認游俠「不軌於正義」（行為不合官方規範），但他隨即列舉了游俠的品格：守信、果斷、誠實、自我犧牲、救人於危難、不炫耀功勞——他問：這樣的人，難道不值得尊敬嗎？「亦足多者」——也足以被人讚美了——這是一個極為克制的表達，但背後是一個極為大膽的判斷

> 且緩急，人之所時有也。……及若季布、朱家、劇孟、郭解之徒，雖時扞當世之文罔，然其私義，廉潔退讓，有足稱者。

> ——司馬遷從人性的角度出發——每個人都有急難的時候。當你走投無路的時候，那些「扞當世之文罔」（觸犯當代法律）的人——如果他們幫助了你——你還會在意他們有沒有犯法嗎？這裡的「私義」是司馬遷發明的一個概念——與「公義」（官方正義）相對——他認為在某些情況下，「私義」比「公義」更重要

> 至如閭巷之俠，修行砥名，聲施於天下，莫不稱賢，是為難耳。然儒、墨皆排擯不載。自秦以前，匹夫之俠，湮滅不見，余甚恨之。

> ——司馬遷說出了他為游俠立傳的原因：民間游俠——那些從市井閭巷中走出來的人——他們修養自己的品德、磨練自己的名聲，影響力遍及天下，沒有人不稱讚他們的賢德。但儒者、墨者都不願意記錄他們。在秦朝之前，那些匹夫之俠——全部湮沒在歷史中，連名字都沒有留下。司馬遷說：「余甚恨之」——我為此感到非常遺憾。這句話，就是司馬遷為游俠立傳的直接動機

> ——《史記·游俠列傳》

## 四、關鍵人物

### 4.1 郭解——漢代游俠的最後高峰

| 項目 | 內容 |
|------|------|
| **身份** | 字翁伯，河內軹人（今河南濟源）。漢代最著名的游俠，也是漢武帝打壓游俠行動中被犧牲的最重要人物 |
| **生平核心** | 郭解年輕時「陰賊」（內心陰險殘忍）——因小事就殺人，甚至為朋友報仇而多次犯法。但他中年之後「更折節為儉」——改變了作風，變得謙遜節儉——「以德報怨，厚施而薄望」——以善報惡，多施予而少要求回報。他「救人不使其知」——救了人卻不讓對方知道——「既已振人之命，不矜其功」——救了人的命卻不炫耀功勞。他的名聲越來越大，天下人都敬慕他，但他仍然低調。最終因為他的名望太大——「天下無賢與不肖，知與不知，皆慕其名」——漢武帝認為他威脅了國家的權威，下令將其滅族 |
| **與司馬遷的關係** | 郭解被殺的時間（西元前 127 年）與司馬遷出生時間接近——司馬遷應該是聽聞過郭解事蹟的當代人。郭解的命運——因為名聲太大而被權力消滅——與司馬遷的命運（因為說了真話而被權力懲罰）形成了某種對照 |
| **歷史爭議** | 郭解是否真的是一個「好人」？司馬遷在傳記中也老實地記錄了他年輕時的暴行——他確實殺過人、確實犯過法。但司馬遷要問的是：一個人年輕時的錯，是否可以否定他後來的善？更重要的是：即使郭解有錯，他的被殺——因為名聲太大而被滅族——是公正的嗎？ |

**軍師評**：郭解是司馬遷筆下最複雜的人物之一。他不是一個完美的英雄——他年輕時殺人犯法，中年後才改過自新。但正是因為他的複雜性，他才是最真實的「俠」——游俠的「俠」不是天生的道德完人，而是在社會的夾縫中，用自己的方式做了一些好事的人。司馬遷沒有把郭解塑造成一個道德楷模——他只是真實地記錄了一個人，然後問：這樣一個人，是否因為他不完美，就不值得被記住？這個問題，在今天仍然有意義。

### 4.2 朱家——魯國的「匿名俠」

| 項目 | 內容 |
|------|------|
| **身份** | 魯國人（今山東曲阜一帶），與孔子同鄉。漢初最著名的游俠之一，以「匿名行善」著稱 |
| **生平核心** | 朱家是游俠中最「低調」的一類——他救了幾百個被追捕的人，但從不讓人知道是自己救的。他最著名的事蹟是救了季布——季布原是項羽的部將，劉邦稱帝後懸賞追捕他。朱家通過夏侯嬰向劉邦進言，最終使季布獲得赦免並被重用。季布後來成為漢朝的重臣，但朱家從未向季布提過這件事——「終身不見」——從此再也沒有見季布。他終身過著簡樸的生活，「衣不完采，食不重味」——衣服沒有華麗的顏色，吃飯只有一個菜——卻把所有的財富都用來幫助別人 |
| **道德境界** | 朱家代表了游俠的最高境界——「行善而不欲人知」。他救人不是為了名聲，不是為了回報——甚至不是為了「俠」這個稱號。他就是單純地覺得：看到有人需要幫助，就去幫助。這種「不為名、不為利、不求回報」的善行，即使在兩千年後，仍然令人肅然起敬 |

**軍師評**：朱家是〈游俠列傳〉中最「乾淨」的人物——他沒有任何爭議。郭解有年輕時的黑歷史，劇孟的生活方式被批評為「奢侈」，但朱家幾乎就是一個道德楷模——他的一生就是「行善」兩個字的完美詮釋。司馬遷把朱家放在游俠傳的首位，是有深意的：他要用朱家這樣一個無可挑剔的人物，來證明游俠這個群體的價值——如果連朱家都應該被歷史遺忘，那這個世界還有什麼公正可言？

### 4.3 劇孟——洛陽的「無冕之王」

| 項目 | 內容 |
|------|------|
| **身份** | 洛陽人，漢初著名的游俠——他的影響力大到讓周亞夫這樣的國家重臣都極為重視 |
| **生平核心** | 劇孟最有名的事蹟，發生在七國之亂（西元前 154 年）。太尉周亞夫率軍平叛，路過洛陽時，聽說劇孟還在洛陽——沒有加入吳楚聯軍。周亞夫大喜，說：「吳楚舉大事而不求劇孟，吾知其無能為矣。」——吳楚聯軍起兵造反，卻沒有爭取劇孟的支持——周亞夫由此判斷他們成不了大事。一個民間游俠的態度，竟然成為一位大將軍判斷敵我實力的依據——這說明了劇孟在地方社會中擁有的巨大影響力 |
| **象徵意義** | 劇孟的存在證明了游俠在漢初社會中的真實地位——他們不是「地下勢力」，而是公開的、被社會各階層承認的「民間力量」。官府對他們客氣，將領對他們尊重，老百姓對他們愛戴。這種地位，在漢武帝的中央集權運動中被徹底摧毀——游俠的黃金時代，在劇孟之後就開始落幕了 |

**軍師評**：劇孟的故事告訴我們一個重要的歷史事實——在漢武帝之前，游俠是社會結構中的一個正常組成部分。官府和民間都接受他們的存在——他們不是反抗者，而是「秩序的補充者」。周亞夫之所以在乎劇孟的態度，不是因為劇孟有軍隊——他沒有——而是因為劇孟代表了洛陽乃至整個東方地區的「民間態度」。吳楚聯軍沒有爭取劇孟，說明他們在老百姓中不得人心。這種「民間態度」的力量，是任何政權都無法忽視的——這也是為什麼漢武帝最終決定消滅游俠：因為這種力量太大了，大到威脅了國家的權力壟斷。

### 4.4 司馬遷——為游俠辯護的孤獨史家

| 項目 | 內容 |
|------|------|
| **身份** | 西漢太史令，《史記》作者——中國最偉大的歷史學家 |
| **與游俠的關係** | 司馬遷與游俠的關係，是所有《史記》人物中最值得玩味的之一。他自身經歷了「李陵之禍」——因說公道話而被法律懲罰——使他在情感上與那些「被法律拋棄的人」產生了深刻的共鳴。他寫游俠，不只是記錄一個歷史現象——他是在為一群「不被官方承認的英雄」爭取歷史中的位置。而他自己，在某種程度上也是這樣一個「不被承認的人」——他受宮刑，被社會視為「閹人」，但他仍然用自己的方式——寫作——來對抗這個世界的不公 |
| **寫作風險** | 在漢武帝朝為游俠辯護——尤其是在漢武帝親自下令誅殺郭解之後——是非常危險的。司馬遷在〈游俠列傳序〉中雖然措辭謹慎（如「不軌於正義」這樣的保留用語），但整體立場極為鮮明：游俠的行為雖然「不合法」，但他們的品格值得尊敬。這在某種程度上是對皇權的挑戰——因為漢武帝就是那個決定「什麼是正義」的人。司馬遷等於在說：你定的「正義」，不一定是真正的正義 |
| **歷史定位** | 〈游俠列傳〉是《史記》中最具「邊緣視角」的篇章之一。司馬遷沒有把目光投向帝王將相——而是投向了那些被歷史遺忘的「邊緣人」：游俠、刺客、滑稽、貨殖（商人）……他認為這些人同樣值得被記錄。這種「為邊緣人立傳」的史學態度，在中國正史中是獨一無二的——也是司馬遷最偉大的貢獻之一 |

**軍師評**：司馬遷寫游俠，其實是在寫自己的理想——一個不需要依附權力、可以憑藉個人的品格和行為來獲得尊重的人。游俠不需要皇帝的任命、不需要官方的認證——他們的權威來自民間的口碑。而司馬遷自己，在被漢武帝拋棄之後，也成為了一個「不需要官方認證」的人——他的《史記》不是奉皇帝的命令寫的，也不是為了讓皇帝高興而寫的——他是為歷史本身而寫的。從這個意義上說，司馬遷自己就是一個「史家之俠」。

## 五、核心分析——「俠」的社會功能及其倫理正當性

### 5.1 「俠」的定義——司馬遷的三個層次

司馬遷對「俠」的定義，不是法律定義——而是一個倫理定義。在〈游俠列傳序〉中，他從三個層次界定了什麼是「俠」：

| 層次 | 內容 | 分析 |
|------|------|------|
| **行為層次：做什麼？** | 「言必信，行必果，已諾必誠，不愛其軀，赴士之厄困」——說話算話、做事有結果、承諾必誠心踐行、不惜犧牲自己去幫助困境中的人 | 這是游俠的行為標準——它與儒家的「君子」標準幾乎完全一致。司馬遷的用意很明顯：你們儒生天天講「信」「義」「勇」——但真正做到的，反而是這些不讀書的游俠 |
| **態度層次：做完之後的態度** | 「不矜其能，羞伐其德」——不炫耀自己的才能，不誇耀自己的功勞——甚至「救人不使其知」——救了人卻不讓對方知道 | 這是游俠最令人敬佩的地方——他們不是為了名聲才去做好事。朱家救了季布後「終身不見」，郭解「既已振人之命，不矜其功」——他們做好事就像呼吸一樣自然，做完就忘 |
| **社會層次：在社會中的角色** | 游俠填補了「制度真空」——在法律無法伸張正義的地方，他們用個人的力量——名聲、關係、勇氣——來幫助那些被制度遺棄的人。他們不是要取代法律，而是要在法律失靈時提供最後的救濟 | 這是游俠的社會功能。司馬遷用「緩急，人之所時有也」來論證——每個人都有急難的時候——當你有急難時，如果法律幫不了你，如果有人願意不顧一切地幫你——你還在意他是否「合法」嗎？ |

> **軍師按**：司馬遷對「俠」的定義，其實是一種「反制度的人格理想」——它不需要任何制度授權，也不需要任何官方認證。一個人的價值，不取決於他的身份、地位、財富——而取決於他的行為。這種「以德不以位」的價值觀，是司馬遷整個史學思想的基礎——也是他為游俠立傳的最深層動機：他要告訴世界，有些人雖然沒有官位、沒有權力、被官方打壓——但他們的品格，比那些位高權重的人更高貴。

### 5.2 「儒俠之辯」——司馬遷對儒家士大夫的批評

〈游俠列傳序〉中最尖銳的部分，是司馬遷對儒生的批評。他不是反對儒家學說——他反對的是那些「只會說、不會做」的偽君子：

| 比較維度 | 真正的儒者（理想） | 偽儒者（現實） | 游俠 |
|---------|------------------|--------------|------|
| **言行** | 言行一致——如孔子弟子 | 說一套做一套——「有君子之名而無君子之實」 | 不說漂亮話，直接行動 |
| **遇到他人困境時** | 「己所不欲，勿施於人」——會伸出援手 | 「持尺書、抱尺寸之義」——用書本上的道理來推脫 | 「赴士之厄困，不愛其軀」——不惜犧牲自己去救人 |
| **對名聲的態度** | 不追求名聲，但名聲自然來 | 「以君子之名取重於世」——利用君子的名號獲取利益 | 做了好事卻不讓人知道 |
| **司馬遷的態度** | 尊敬 | 批評——並且認為這些人還不如游俠 | 讚美——「亦足多者」 |

> **軍師按**：司馬遷對儒生的批評，不是出於對儒學的敵意——而是出於對「知行不一」的極度厭惡。他自己就是一個知行合一的人——為了說一句公道話，付出了宮刑的代價。所以當他看到那些儒生穿著儒服、說著仁義道德、但在別人真正需要幫助時卻袖手旁觀時，他的憤怒是可想而知的。某種程度上，〈游俠列傳序〉中的儒俠之辯，就是司馬遷對那些「暗室虧心」的偽君子的公開斥責：你們連一個流氓都不如——因為流氓至少說到做到。

### 5.3 游俠的內部矛盾——「不軌於正義」的兩面性

司馬遷在讚美游俠的同時，也誠實地記錄了游俠的黑暗面。郭解年輕時的暴行、某些游俠的「私鬥」行為——這些都是游俠群體的真實面目：

| 正面 | 負面 |
|------|------|
| 「言必信，行必果」——嚴格的道德自律 | 某些游俠「私睚眥必報」——睚眥之怨必報——報復心極強 |
| 「赴士之厄困」——幫助弱者 | 某些游俠「借交報仇」——為朋友去殺人——成了私人恩怨的工具 |
| 「不矜其能，羞伐其德」——低調行善 | 某些游俠「橫行鄉里」——成了地方惡霸 |
| 填補制度真空——在法律失靈時提供正義 | 「以武犯禁」——破壞社會秩序——即使是不公正的秩序 |

**軍師評**：司馬遷的高明之處在於：他沒有把游俠完美化。他知道游俠有黑暗面——有些人殺了人，有些人成了地方惡霸。但他問了一個更深刻的問題：當一個不公正的制度與一個不完美的游俠對立時，我們應該站在哪一邊？司馬遷的態度是：一個不公正的制度，比一個不完美的個人更可怕——因為制度的惡是系統性的，而個人的惡是可以被遏制的。這種判斷，即使在今天——當我們面對「程序正義」與「實質正義」的兩難時——仍然極具啟發性。

## 六、軍師核心分析——當制度失靈時，誰來主持正義？

### 6.1 為什麼〈游俠列傳序〉是《史記》中最「大膽」的文章？

〈游俠列傳序〉的大膽，不在於它的文字——文字雖然犀利，但還算節制——而在於它提出的問題：**當法律本身成為不義的工具時，違反法律是不是反而成為了一種正義？**

這個問題的危險性在於：它動搖了權力的基礎——權力的基礎就是「法律」和「正義」的同一性。如果權力說「法律=正義」，但事實是「法律≠正義」——那麼權力賴以統治的正當性就崩潰了。司馬遷在〈游俠列傳序〉中，恰恰就指出了這個矛盾：

| 層面 | 官方的說法 | 司馬遷的觀察 |
|------|-----------|-------------|
| **法律的功能** | 法律是公正的，保護所有的人 | 法律是權力的工具——酷吏（張湯、杜周）可以隨意羅織罪名 |
| **正義的標準** | 合法的就是正義的 | 有些合法的行為（如酷吏的濫權）是不義的，有些不合法的行為（如游俠救人）是正義的 |
| **誰有權定義正義** | 皇帝和官府 | 每個人都有權用自己的良知判斷 |
| **游俠的角色** | 違法者、社會秩序的破壞者 | 填補制度真空的「民間正義使者」 |

> **軍師按**：司馬遷沒有直接挑戰皇帝的權威——那在漢武帝時代是自尋死路——但他用了一個極其巧妙的策略：他讚美游俠的品質（言必信、行必果、諾必誠），然後讓讀者自己去判斷。他從來沒有說「游俠是對的，法律是錯的」——他只是說「游俠雖然不合法，但他們有這些美好的品質——你們說，這樣的人值不值得尊敬？」這個問題，每個讀者心裡都有自己的答案——而司馬遷知道，任何有良知的人都會回答「值得」。

### 6.2 「俠」的中國文化意義——從游俠到江湖

〈游俠列傳序〉不僅是一篇史論——它還深刻地影響了中國文化中的「俠」概念。司馬遷之後，「俠」從一個歷史現象變成了一個文化符號：

| 時期 | 俠的形態 | 代表作／代表人物 |
|------|---------|----------------|
| **司馬遷的游俠** | 真實的歷史人物——朱家、劇孟、郭解——他們是生活在漢代的真人真事 | 《史記·游俠列傳》 |
| **唐傳奇中的俠** | 文學虛構——紅線女、聶隱娘、虯髯客——從歷史人物變成小說角色 | 〈紅線傳〉、〈聶隱娘〉、〈虯髯客傳〉 |
| **宋明話本與水滸** | 游俠與「江湖」概念的結合——梁山好漢——「替天行道」——游俠從個人行為變成了集體反抗 | 《水滸傳》——宋江等人被逼上梁山，以「俠義」之名對抗腐敗的官府 |
| **金庸武俠小說** | 俠的現代巔峰——「俠之大者，為國為民」——從個人義氣上升到家國情懷 | 金庸——郭靖、喬峰、楊過——「為國為民」的俠 |
| **現代社會中的俠** | 沒有了武力，但保留了「俠」的精神——見義勇為、路見不平、扶危濟困 | 平民英雄、公益律師、調查記者 |

**軍師評**：司馬遷的〈游俠列傳序〉，是中國「俠文化」的真正起點。在他之前，有刺客（豫讓、聶政、荊軻），有勇士（孟賁、夏育），但沒有「俠」這個概念。是司馬遷第一次明確地定義了「俠」——「言必信，行必果，已諾必誠，不愛其軀，赴士之厄困」——這個定義，成為中國兩千年來所有關於「俠」的想像的基礎。從唐傳奇到《水滸傳》到金庸武俠，每一個「俠」的形象，都可以在〈游俠列傳序〉中找到它的影子。司馬遷不僅記錄了一個歷史群體——他創造了一個文化原型。

### 6.3 與其他篇目的互文

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈刺客列傳〉（司馬遷） | 〈刺客列傳〉與〈游俠列傳〉是《史記》中「邊緣英雄」的雙璧。刺客（豫讓、聶政、荊軻）是「為知己死」的極致——他們為一個人的知遇之恩而付出生命；游俠則是「為眾生助」——他們幫助那些素不相識的弱者。兩者合在一起，構成了一幅完整的「民間英雄」圖譜 |
| 卷五·〈伯夷列傳〉（司馬遷） | 〈伯夷列傳〉中司馬遷對「天道無親」的困惑——好人為什麼沒有好報？——與〈游俠列傳序〉中對「法律不公」的質疑——當制度不保護好人時，誰來保護他們？——形成了深刻的呼應。兩篇都是司馬遷對「正義」的追問 |
| 卷五·〈貨殖列傳序〉（司馬遷） | 〈貨殖列傳序〉是司馬遷為商人辯護的文章——與〈游俠列傳序〉為游俠辯護如出一轍。兩篇的共同點是：司馬遷挑戰了「士農工商」的傳統排序——他認為商人和游俠一樣，為社會做出了重要貢獻，理應得到歷史的尊重。這種「為邊緣群體正名」的寫作，是司馬遷史學中最具現代性的部分 |
| 卷五·〈屈原列傳〉（司馬遷） | 屈原也是一個「不被制度保護的人」——他被楚懷王拋棄，被讒臣陷害，被放逐——和游俠一樣，他們都在制度之外尋找正義。不同的是：屈原選擇了死亡，游俠選擇了反抗——但兩者都拒絕了「妥協」 |
| 卷四·〈宋玉對楚王問〉（宋玉） | 宋玉面對楚王的質問時，選擇了「曲高和寡」的姿態——拒絕向世俗解釋自己。游俠面對官方的打壓時，也選擇了「不屑解釋」——他們用行動證明自己的價值。兩者都是「被權力拋棄後，用自己的方式活下來的人」 |

### 6.4 對後世的深遠影響

| 領域 | 影響 |
|------|------|
| **史學傳統** | 司馬遷開創了「為邊緣人物立傳」的傳統——後世正史雖然主要記錄帝王將相，但始終保留了「游俠」「隱逸」「獨行」等類傳。這是中國史學中一條隱秘的「民間史」脈絡——即使在最正統的史書中，仍然給邊緣人留下了一席之地 |
| **俠文化** | 〈游俠列傳序〉奠定了中國「俠文化」的思想基礎。司馬遷定義的「言必信、行必果、諾必誠」——成為後世所有俠義人物的道德標準。從唐傳奇到《水滸傳》到金庸武俠——每一個「大俠」身上，都有司馬遷筆下游俠的影子 |
| **知識分子精神** | 「俠」在後世成為中國知識分子的一種精神寄託——當知識分子對現實政治失望時，他們往往會轉向「俠」的理想——一種「脫離體制、憑藉個人品格行事」的生活方式。李白「十步殺一人，千里不留行」——這不是真實的李白，這是李白心中的「俠客夢」 |
| **民間社會** | 在中國傳統社會中，「江湖」始終是一個與「官府」相對的概念——江湖有自己的規矩（行規、幫規、義氣），有自己的制裁方式（比武、評理、復仇），這些都源於游俠的傳統。即使到了現代社會——黑社會、幫派、地下秩序——也可以在某種程度上看作是游俠傳統的陰暗變體 |

## 七、一句話總結（再錄）

> **〈游俠列傳序〉是《史記》中最鋒利的一篇文章——它用「言必信，行必果，已諾必誠，不愛其軀，赴士之厄困」這十六個字，為一群被歷史遺忘的民間英雄重塑了尊嚴。司馬遷在這篇文章中做了一件極為大膽的事：他告訴這個世界——當法律成為權力的工具時，那些「不軌於正義」的人，可能比那些「自稱正義」的人更接近正義。這不是對秩序的挑戰——這是對虛偽的秩序的不屑。兩千年來，每一個在制度中受過委屈的人——從被貶的官員到被欺負的百姓——都能在游俠的身上看到自己渴望的模樣：一個不需要權力授權、不需要官方認證、僅憑個人品格就能主持公道的人。而這種形象之所以如此動人，是因為它寄託了人類最樸素的願望：無論制度多麼腐敗，總有人願意站出來——不是為了名聲，不是為了回報，只是因為他覺得這樣做是對的。這就是司馬遷為我們留下的「俠」——一個關於「做一個好人」的最簡單、也最難的標準。** |

## 八、關聯篇目與延伸閱讀

### 8.1 古文觀止關聯篇目

| 篇目 | 關聯點 |
|------|--------|
| 卷五·〈刺客列傳〉（司馬遷） | 〈刺客列傳〉與〈游俠列傳〉是司馬遷「民間英雄」系列的雙璧——刺客為知己死，游俠為眾生助——兩篇合讀，可以看到司馬遷對「邊緣人物」的完整態度 |
| 卷五·〈伯夷列傳〉（司馬遷） | 同為司馬遷對「天道」與「正義」的追問——伯夷叔齊「不食周粟」而死，郭解「以名太大」而被殺——兩者的共同點是：好人不一定有好報，但歷史應該記住他們 |
| 卷五·〈貨殖列傳序〉（司馬遷） | 同為司馬遷「為邊緣群體正名」之作——商人與游俠一樣，都是被傳統鄙視卻對社會有實際貢獻的人——兩篇是司馬遷經濟觀與社會觀的姊妹篇 |
| 卷五·〈屈原列傳〉（司馬遷） | 屈原與游俠都是「不被制度保護的人」——屈原選擇了以死抗爭，游俠選擇了以生助人——兩者都是對不公世界的不同回應 |
| 卷四·〈宋玉對楚王問〉（宋玉） | 游俠的「不屑解釋」與宋玉的「曲高和寡」——兩者都是被誤解者的自我保護——只是一個用行動證明，一個用比喻開脫 |

### 8.2 經典延伸

- 《史記·游俠列傳》（司馬遷）——全文閱讀——〈游俠列傳序〉只是開頭，後面還有朱家、劇孟、郭解等人的詳細傳記。讀完序文之後，再讀正文，可以看到司馬遷的理論如何在具體人物身上得到體現
- 《史記·刺客列傳》（司馬遷）——游俠的「兄弟篇」——豫讓、聶政、荊軻等人的故事——與游俠互為補充，共同構成戰國秦漢時期的「民間英雄」畫卷
- 《史記·貨殖列傳》（司馬遷）——司馬遷對商人與經濟活動的看法——與〈游俠列傳〉共同體現了司馬遷「包容多元」的史學態度
- 《史記·太史公自序》（司馬遷）——司馬遷對自己寫作《史記》的完整說明——其中提到他為游俠立傳的動機
- 《報任安書》（司馬遷）——司馬遷自己的生命宣言——「人固有一死，死有重於泰山，或輕於鴻毛」——這封信是理解司馬遷（以及他筆下的游俠）精神世界的關鍵文獻
- 《韓非子·五蠹》（韓非）——〈游俠列傳序〉開篇引用的「儒以文亂法，俠以武犯禁」即出自此篇。讀韓非子的原文，可以理解司馬遷所要翻轉的「官方立場」是什麼
- 《水滸傳》（施耐庵）——游俠傳統在小說中的巔峰體現——梁山好漢「替天行道」——與司馬遷筆下的游俠一脈相承

### 8.3 現代延伸

- 余英時《士與中國文化》——關於中國知識分子與「俠」的關係——余英時認為中國知識分子在精神上與游俠有深刻的共鳴：兩者都堅持「獨立於權力之外」的人格理想
- 陳平原《千古文人俠客夢》——陳平原對「俠文化」在中國文學中的演變的系統性研究——從司馬遷到金庸——全面梳理了「俠」從歷史到文學的演變軌跡
- 龔鵬程《俠的精神文化史論》——龔鵬程對「俠」這一概念的文化哲學分析——他認為俠的核心不是武力，而是「一種超越制度的人格力量」——與司馬遷在〈游俠列傳序〉中的論述遙相呼應
- 韓兆琦《史記箋證》——對《史記》最詳細的現代注釋——其中對〈游俠列傳〉的箋證是理解本文的首選參考
- 金庸《天龍八部》——金庸武俠小說中對「俠」的最高演繹——喬峰「為國為民」的俠義精神——與司馬遷「言必信、行必果」的游俠定義一脈相承——金庸自己也承認：他的武俠世界，根源在《史記》
- 漢娜·阿倫特《極權主義的起源》——關於「制度性邪惡」——當一個制度本身成為邪惡的工具時，個人的反抗是否具有正當性？——司馬遷在兩千年前提出的問題，在阿倫特的政治哲學中得到了現代的回響
