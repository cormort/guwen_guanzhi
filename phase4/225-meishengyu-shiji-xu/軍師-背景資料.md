## 二、時代背景

### 2.1 北宋慶曆年間的政治氣候

慶曆六年（1046）正值北宋政治史上一個關鍵轉折期。此前數年（1043–1045），范仲淹、歐陽修、韓琦、富弼等人推動「慶曆新政」，針對北宋積弊（冗官、冗兵、冗費）進行一系列改革——整頓吏治、改革科舉、加強軍備、發展經濟。然而新政觸動了大量既得利益者（尤其是守舊官僚集團），反對聲浪日益洶湧。

慶曆五年（1045），范仲淹、富弼等改革派核心人物相繼被貶出京，慶曆新政宣告失敗。歐陽修作為改革派的堅定支持者與核心筆桿，同樣遭到政治清算：先被貶知滁州（寫下〈醉翁亭記〉），後又輾轉遷徙。

正是在這樣的背景下，歐陽修為摯友梅堯臣（梅聖俞）的詩集作序。兩人同為改革失敗後的失意者，寫作此文既是在文學層面為梅堯臣的詩歌成就立論，也是藉題抒發自己對仕途困頓的深刻體悟。

### 2.2 北宋詩文革新運動

北宋初期，文壇承襲晚唐五代浮豔之風（「西崑體」盛行，以楊億、劉筠為代表，講究辭藻華麗、用典繁密，內容卻空疏貧乏）。歐陽修、梅堯臣、蘇舜欽等人正是反對此風的主力軍，推動詩文革新運動：

- **歐陽修**：主張「文以載道」「明道致用」，領導古文運動，改革科舉文風
- **梅堯臣**：針對西崑體提出詩歌應「意新語工」「狀難寫之景如在目前，含不盡之意見於言外」，提倡平淡自然、反映社會現實的詩風

〈梅聖俞詩集序〉不僅是一篇詩集序文，更是歐陽修為好友詩歌成就背書、為詩文革新運動張目之作。文中提出的「窮而後工」論，從理論上為「苦難—創作」之間的因果關係提供了經典詮釋，成為宋代詩學最重要的命題之一。

### 2.3 「窮而後工」的歷史淵源

歐陽修「窮而後工」的命題並非憑空產生，而是有深遠的思想淵源：

| 時代 | 代表人物 | 觀點 |
|------|----------|------|
| 戰國 | 屈原 | 「發憤以抒情」——《九章·惜誦》 |
| 西漢 | 司馬遷 | 「《詩》三百篇，大抵賢聖發憤之所為作也」——〈報任安書〉，提出「發憤著書」說 |
| 唐代 | 韓愈 | 「大凡物不得其平則鳴」——〈送孟東野序〉，提出「不平則鳴」論 |
| 北宋 | 歐陽修 | 「非詩之能窮人，殆窮者而後工也」——〈梅聖俞詩集序〉 |
| 南宋 | 陸游、辛棄疾 | 國破家亡的體驗催生愛國詩詞的巔峰 |
| 明清 | 李贄、金聖歎 | 「發憤著書」說的延續與深化 |

歐陽修的貢獻在於：他將司馬遷的「發憤著書」與韓愈的「不平則鳴」具體化為針對詩歌創作的命題，並且反轉了傳統因果——不是「寫詩使人窮困」，而是「困頓使人寫出好詩」。

## 三、關鍵人物

### 3.1 歐陽修（1007–1072）

- **字號**：字永叔，號醉翁，晚號六一居士
- **籍貫**：吉州廬陵（今江西吉安）
- **身份**：北宋文壇領袖、「唐宋八大家」之一，與韓愈、柳宗元、蘇軾並稱
- **仕途**：天聖八年（1030）進士，歷任館閣校勘、知諫院、滁州知州、翰林學士、樞密副使、參知政事等職。一生三起三落，因支持慶曆新政兩度遭貶
- **文學貢獻**：領導北宋古文運動，主編《新唐書》《新五代史》，詩文革新領袖
- **與梅堯臣關係**：兩人於天聖九年（1031）在洛陽結識，從此結為終身摯友，詩文酬唱不絕。歐陽修對梅堯臣的詩才極其推崇，自謂「吾於聖俞詩，論之詳矣」

**歐陽修寫作此文的心理狀態**：慶曆六年，歐陽修身在滁州貶所，政治理想受挫。為同樣困頓的梅堯臣作序，既是對友人的安慰與肯定，也是對自身處境的投射——〈醉翁亭記〉中的豁達與此文的深沉，恰好構成同一時期歐陽修心境的兩面。

### 3.2 梅堯臣（1002–1060）

- **字號**：字聖俞，世稱宛陵先生
- **籍貫**：宣州宣城（今安徽宣城）
- **身份**：北宋著名現實主義詩人，宋詩開山祖師之一
- **仕途**：生平最坎坷之處——**終身未能考中進士**。以叔父梅詢的恩蔭補為太廟齋郎，後歷任桐城主簿、河南主簿、德興知縣等職，長期沉淪下僚。直到皇祐三年（1051），因大臣推薦才獲賜同進士出身，時已年近五十。官至尚書都官員外郎（從六品），在北宋官場中屬於邊緣人物
- **詩歌成就**：與歐陽修、蘇舜欽齊名，人稱「梅歐」「蘇梅」。詩風平淡含蓄、意在言外，開創宋詩風格的先河。代表作〈汝墳貧女〉、〈田家語〉等反映民間疾苦
- **與歐陽修關係**：兩人交誼長達三十年。歐陽修稱讚梅詩「清麗閒肆，平淡而精工」，並在此序中系統總結其成就

**梅堯臣的「窮」具體表現在**：
1. **科舉之窮**：累試進士不第，這是宋代文人最大的挫折
2. **官職之窮**：長期擔任縣主簿、知縣等地方小官，位卑權輕
3. **經濟之窮**：俸祿微薄，生活清貧
4. **政治之窮**：因為詩風不合流俗、為人剛正，始終未被朝廷重用

### 3.3 謝景初（與序文相關的人物）

歐陽修在文中提到「謝景初」，此人即謝絳之子、梅堯臣的妻弟。他最早為梅堯臣整理編次詩集，歐陽修得見此編本後欣喜不已，遂為之作序。後謝景初所編之本已佚，現存梅堯臣詩集為後人所輯。

## 四、核心分析

### 4.1 「窮而後工」的完整邏輯鏈

歐陽修在序文中構築了一條精密的三段論證：

**第一層：反駁命題（破題）**
> 「予聞世謂詩人少達而多窮，夫豈然哉？」

世俗認為詩人多困窘、少騰達。歐陽修的第一反應是「真的是這樣嗎？」——看似質疑，實則為後文鋪墊。

**第二層：證據鋪排（舉證）**
歐陽修一口氣舉出文學史上經典案例：
- 屈原放逐而《離騷》作
- 杜甫流離而詩史成
- 李白漂泊而詩篇雄
- 唐代窮苦詩人（孟郊、賈島等）愈窮愈工

他論證：但凡傳世的好詩，幾乎都出自失意者之手。政治上得意的人（如台閣重臣），大多寫不出動人心魄的作品。

**第三層：因果翻轉（結論）**
> 「然則非詩之能窮人，殆窮者而後工也。」

這是最關鍵的一句話。歐陽修翻轉了傳統認知——不是因為你寫詩才導致貧窮困頓，而是因為你經歷了窮困磨難，才能寫出真正的好詩。因果關係從「詩→窮」變成了「窮→工」。

### 4.2 為什麼「窮」能「工」？

歐陽修雖未在文中詳細展開，但其邏輯可歸納為以下幾點：

| 原因 | 說明 |
|------|------|
| **情感驅動** | 窮困使人有真切的痛苦、憤懣、憂患，這些強烈情感是創作的內在動力 |
| **視野轉向** | 富貴者目光局限於宮廷台閣，困窮者方能深入民間，見證真實的人間百態 |
| **專注沉潛** | 仕途得意者忙於政務應酬，窮困者無所事事，反而有大量時間專注於詩藝打磨 |
| **超越功利** | 困頓使詩人不受世俗功利束縛，敢於寫真話、抒真情，不必迎合權貴 |
| **語言淬煉** | 處境艱難促使詩人尋找更精準、更有力的語言來表達內心 |

### 4.3 對梅堯臣的具體應用

歐陽修將上述理論完全落實到梅堯臣身上：

1. **「少以蔭補為吏，累舉進士，輒抑於有司，困於州縣，凡十餘年」**——梅堯臣科舉失敗、長期沉淪下僚
2. **「年今五十，猶從辟書，為人之佐，鬱其所蓄，不得奮見於事業」**——年近五十仍是幕僚，抱負無法施展
3. **「其家宛陵，幼習於詩，自為童子，出語已驚其長老」**——天賦早已展現
4. **「既長，學乎六經仁義之說，其為文章，簡古純粹，不求苟說於世」**——品格高尚、不趨炎附勢

結論：正因為梅堯臣經歷了這些困頓，他的詩才達到了「工」的極致——「世之人徒知其詩而已」（世人只知道他的詩好，卻不知道他的詩為什麼好）。

### 4.4 文章的修辭藝術

歐陽修作為古文大家，此文在藝術上有鮮明特點：

- **層層遞進**：先泛論詩人「窮」的普遍現象，再舉證歷史人物，最後聚焦到梅堯臣身上
- **夾敘夾議**：敘事與議論交織，既有理論高度，又有具體實例
- **情理交融**：既是文學評論，又是朋友間的深厚情誼流露
- **以問代答**：「夫豈然哉？」以反問開篇，引人深思
- **反轉收束**：最後一句「非詩之能窮人，殆窮者而後工也」，擲地有聲，成為千古名言

### 4.5 歷史影響與評價

「窮而後工」的命題影響深遠：

1. **宋代詩學核心命題**：被黃庭堅、陳師道、陸游等宋代詩人廣泛接受與引用
2. **中國文學批評的經典話語**：與司馬遷「發憤著書」、韓愈「不平則鳴」並列為中國文學批評三大動力論
3. **超越文學的普世價值**：成為對一切苦難與創造之間關係的經典概括——困境催生偉大
4. **現代美學的迴響**：王國維《人間詞話》「詩人對宇宙人生，須入乎其內，又須出乎其外」，錢鍾書《詩可以怨》等，皆在此脈絡中展開

> 錢鍾書在《詩可以怨》中精闢總結：中國文學傳統中，「痛苦比快樂更能產生詩歌，好詩主要是不愉快、煩惱或『窮愁』的表現或發洩。」

## 五、原文全文

### 〈梅聖俞詩集序〉

**歐陽修**（1007–1072）

予聞世謂詩人少達而多窮，夫豈然哉？蓋世所傳詩者，多出於古窮人之辭也。凡士之蘊其所有而不得施於世者，多喜自放於山巔水涯之外，見蟲魚草木、風雲鳥獸之狀類，往往探其奇怪。內有憂思感憤之鬱積，其興於怨刺，以道羈臣寡婦之所嘆，而寫人情之難言。蓋愈窮則愈工。然則非詩之能窮人，殆窮者而後工也。

予友梅聖俞，少以蔭補為吏，累舉進士，輒抑於有司，困於州縣，凡十餘年。年今五十，猶從辟書，為人之佐，鬱其所蓄，不得奮見於事業。其家宛陵，幼習於詩，自為童子，出語已驚其長老。既長，學乎六經仁義之說，其為文章，簡古純粹，不求苟說於世。世之人徒知其詩而已。

然時無賢愚，語詩者必求之聖俞。聖俞亦自以其不得志者，樂於詩而發之。故其平生所作，於詩尤多。世既知之矣，而未有薦於上者。昔王文康公嘗見而嘆曰：「二百年無此作矣！」雖知之深，亦不果薦也。若使其幸得用於朝廷，作為雅、頌，以歌詠大宋之功德，薦之清廟，而追商、周、魯頌之作者，豈不偉歟！奈何使其老不得志，而為窮者之詩，乃徒發於蟲魚物類、羈愁感嘆之言？世徒喜其工，不知其窮之久而將老也！可不惜哉！

聖俞詩既多，不自收拾。其妻之兄子謝景初，懼其多而易失也，取其自洛陽至於吳興以來所作，次為十卷。予嘗嗜聖俞詩，而患不能盡得之，遽喜謝氏之能類次也，輒序而藏之。

其後十五年，聖俞以疾卒於京師。余既哭而銘之，因索於其家，得其遺稿千餘篇，並舊所藏，掇其尤者六百七十七篇，為一十五卷。嗚呼！吾於聖俞詩，論之詳矣，故不復云。

**廬陵歐陽修序。**

---

> **軍師按語**：本文最動人之處，在於歐陽修以理論為表、以友情為裡，既為梅堯臣的詩歌成就立下千古定評，又為千百年來所有失意文人找到了尊嚴——你們的苦難不是白費的，它成就了不朽的詩篇。而當我們讀到「奈何使其老不得志，而為窮者之詩……可不惜哉！」這一句時，能深切感受到歐陽修筆端那份難以掩抑的悲憫——理論說得再漂亮，也無法掩蓋好友一生困頓的事實。這是文學批評，更是一封寫給時代的控訴信。
