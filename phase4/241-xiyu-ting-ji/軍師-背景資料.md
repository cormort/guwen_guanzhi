## 二、時代背景——嘉祐之治與青年蘇軾的初仕歲月

### （一）宋仁宗的「嘉祐之治」——守成時代下的文人盛世

宋仁宗趙禎在位四十二年（1022–1063），是宋代在位時間最長的皇帝。仁宗朝雖然邊患不斷（西夏元昊稱帝、宋夏戰爭），但文化上卻達到了北宋的巔峰——范仲淹、歐陽修、王安石、司馬光、蘇軾兄弟，皆在此時期登上歷史舞台。

嘉祐年間（1056–1063）是仁宗朝的最後七年，政局相對穩定，科舉改革逐步推行，文官制度臻於成熟。嘉祐二年（1057），歐陽修知貢舉，嚴厲打擊「太學體」怪僻文風，提拔蘇軾、蘇轍、曾鞏等一批年輕才俊，標誌著宋代古文運動的全面勝利。蘇軾正是在此氛圍下，以二十二歲之齡高中進士，聲名鵲起。

| 年份 | 事件 | 對蘇軾的意義 |
|------|------|-------------|
| 嘉祐元年（1056） | 蘇洵攜二子赴京應試 | 三蘇名動京師 |
| 嘉祐二年（1057） | 蘇軾、蘇轍同中進士 | 歐陽修讚曰：「讀軾書，不覺汗出，快哉快哉！」 |
| 嘉祐五年（1060） | 蘇軾授河南府福昌縣主簿 | 未就任 |
| 嘉祐六年（1061） | 應制科試，入三等（百年來第二人） | 授大理評事、簽書鳳翔府判官 |
| 嘉祐七年（1062） | 在鳳翔修建官舍，築亭得雨 | 作〈喜雨亭記〉 |

### （二）鳳翔歲月——蘇軾仕途的第一站

嘉祐六年（1061）十二月，蘇軾赴鳳翔府（今陝西省寶雞市鳳翔區）任籤書判官廳公事（簡稱簽判），這是他的第一份實職。其弟蘇轍送他到鄭州，兄弟分別，蘇軾寫下著名的「人生到處知何似，應似飛鴻踏雪泥」（〈和子由澠池懷舊〉）。

鳳翔位於關中平原西部，自古為軍事重鎮，也是農業要地。蘇軾在此任職約三年（1061–1064），雖只是副職佐官，卻是他第一次接觸地方行政、體察民間疾苦。他在鳳翔期間寫了不少關心民瘼的詩文，除了〈喜雨亭記〉，還有〈凌虛臺記〉、〈鳳翔八觀〉等名作。

> **軍師按**：〈喜雨亭記〉寫於嘉祐七年（1062）春，蘇軾年僅二十六歲。此時的他，尚未經歷王安石變法、烏臺詩案、黃州貶謫等人生風浪，筆下充滿了初入仕途的樂觀與幽默。讀者不妨將此文與十五年後〈前赤壁賦〉的曠達、晚年〈潮州韓文公廟碑〉的深沉對照——同一支筆，三種心境，恰是蘇軾一生的縮影。

### （三）雨澤與民本——宋代士大夫的天人觀

宋代儒學深受「天人感應」思想的影響——天象（雨、旱、災異）被視為對君主德行的回應。久旱逢雨，不僅是農業上的好消息，更被解讀為「天不遺斯民」的政治象徵。

蘇軾在本文中巧妙地處理了這個敏感話題。他先寫春雨「其占為有年」，再寫「彌月不雨，民方以為憂」，最後「丁卯大雨，三日乃止」。整個敘事節奏從憂到喜，營造出一種「天意眷顧」的氛圍。但他並未將功勞簡單歸於天子或太守，而是層層推辭——太守推給天子，天子推給造物，造物推給太空，最終落在「吾以名吾亭」的幽默收尾。這既是謙遜，也是對儒家「天功不可貪」理念的巧妙體現。

## 三、關鍵人物

### ① 蘇軾（1037–1101）—— 文的絕對天才

| 維度 | 內容 |
|------|------|
| **字號** | 字子瞻，號東坡居士 |
| **本篇角色** | 作者、亭子的建造者與命名者 |
| **年齡** | 二十六歲（嘉祐七年），仕途起步階段 |
| **風格特徵** | 輕快、幽默、自信。全文一氣呵成，毫不滯澀 |
| **後續命運** | 日後歷經王安石變法、烏臺詩案、三度貶謫，晚年流放海南儋州 |
| **與本文的對比** | 本文的歡快風格，與後期作品的深沉曠達形成鮮明對照，見證了蘇軾從「青年才俊」到「歷盡滄桑」的精神蛻變 |

蘇軾在鳳翔期間，實際上與頂頭上司陳希亮（字公弼）關係並不融洽。陳希亮性格剛嚴，對這位名滿天下的年輕下屬多有約束。蘇軾後來在〈凌虛臺記〉中甚至以文章暗諷陳希亮。但〈喜雨亭記〉的基調卻是一派天真純粹，絲毫看不出職場摩擦的痕跡——這正是年輕蘇軾的可貴之處：他能將公務與創作分離，在亭中舉酒屬客時，忘記所有不愉快。

### ② 宋仁宗趙禎（1010–1063）—— 盛世之君

| 維度 | 內容 |
|------|------|
| **在位** | 1022–1063 |
| **廟號** | 仁宗 |
| **與本文的關係** | 文末「歸之天子」中的天子 |
| **歷史定位** | 宋代最寬仁的君主，善養士大夫，文化黃金時代的締造者 |
| **與蘇軾的關係** | 嘉祐二年親自主持殿試，欽點蘇軾為進士。據說仁宗讀到蘇軾兄弟的制科對策後大喜，回宮對曹皇后說：「吾為子孫得兩宰相。」可惜仁宗駕崩時（1063），蘇軾尚在鳳翔任上，未及大用 |

### ③ 陳公弼（陳希亮，?–1065）—— 嚴厲的上司

| 維度 | 內容 |
|------|------|
| **時任官職** | 鳳翔府知府（蘇軾的直屬長官） |
| **性格** | 剛直嚴峻，不假辭色 |
| **與蘇軾的互動** | 對年少成名的蘇軾多有督責，蘇軾一度不滿，後來二人和解。蘇軾為其作〈陳公弼傳〉，稱其「平生不好名，而人莫能欺」 |
| **與本文的關係** | 文中「民曰太守」的太守即指陳希亮。蘇軾在此以幽默筆法將功勞歸於太守，再代為推辭，既是禮節也是巧妙的化解 |

> **軍師按**：本文雖題為〈喜雨亭記〉，實則全篇未著力描寫亭子本身的建築、方位或景觀，而是以雨為經、以喜為緯，通篇扣住一個「喜」字。這種「避實就虛」的寫法，正是蘇軾年輕時就已爐火純青的筆法。

## 四、核心分析——以雨為名的三層敘事結構

### （一）破題奇兵——開篇即盡全篇之意

「亭以雨名，誌喜也。」開門見山，六個字就將亭、雨、喜三者關係說盡。古文觀止的評點家贊曰：「起筆便將『喜雨亭』三字拆開，倒點出，已盡一篇之意。」

接著蘇軾連用三個歷史典故來證明「以喜名物」的傳統：

1. **周公得禾** —— 周成王時，同禾異畝（不同田壟長出合穗的禾），周公作〈嘉禾〉篇
2. **漢武得鼎** —— 漢武帝在汾陰得一寶鼎，改年號為元鼎
3. **叔孫勝敵** —— 魯國大夫叔孫得臣擊敗狄人，以其君之名「僑如」為兒子命名

這三段典故，從周代到漢代再到春秋，跨越千年，蘇軾卻信手拈來，毫不費力。其用意在於：**古今聖賢都以喜名物，我今日以「喜雨」名亭，有何不可？** 這既是文章學養的展現，也是青年蘇軾自信從容的表現。

> **軍師按**：這三段典故中，「叔孫勝敵，以名其子」最妙。將兒子的名字取為敵酋之名，在今天看來近乎荒誕——但蘇軾正是用這種略帶幽默的典故，為下文的輕鬆基調鋪路。讀者至此，已知此文不是板著臉的政論，而是帶著微笑的隨筆。

### （二）場景鋪陳——從憂到喜的戲劇性轉折

第一部分是說理（古今皆以喜名物），第二部分轉入敘事，交代建亭過程與氣候變化：

「予至扶風之明年，始治官舍。為亭於堂之北，而鑿池其南，引流種樹，以為休息之所。」

語氣平淡如日記。然後筆鋒一轉——

「是歲之春，雨麥於岐山之陽，其占為有年。既而彌月不雨，民方以為憂。」

一個「方」字，將全民的焦慮感烘托而出。接著是連串精確的時間記錄：

- **乙卯乃雨**（四月初二下雨）
- **甲子又雨**（四月十一又下雨）
- **丁卯大雨，三日乃止**（四月十四至十六連下三天）

三個日子由疏到密，雨勢由小到大，節奏越來越快，讀者的心情也隨之鬆弛。然後是全篇最精彩的一段排比：

> 「官吏相與慶於庭，商賈相與歌於市，農夫相與忭於野，憂者以喜，病者以愈，而吾亭適成。」

六種人、六種反應，層層推進，最後落到「而吾亭適成」——一個「適」字，彷彿天意安排，雨來得正是時候，亭也建得正是時候。這種巧合帶來的愉悅感，正是全篇的核心情感。

### （三）主客問答——從「五日不雨」到「誰之力」

第三部分設為酒宴上的問答。蘇軾以設問層層遞進：

1. 五日不雨 → 無麥
2. 十日不雨 → 無禾
3. 無麥無禾 → 歲荐饑 → 獄訟繁興 → 盜賊滋熾

這個遞推，將旱災的影響逐步放大：從農業到社會秩序，最終影響到「吾與二三子雖欲優遊以樂於此亭，其可得耶？」——把自己也放進去，拉近了作者與讀者的距離。這種「以切身利害說理」的手法，比抽象的仁政說教更有說服力。

然後是全篇最著名的名句——以「使天而雨珠」與「使天而雨玉」兩個對仗極工巧的假設，引出「一雨三日」的可貴。最後四層推功（民→太守→天子→造物→太空），越推越高，越高越虛，最終落到「太空冥冥，不可得而名。吾以名吾亭」——宇宙無名，我便以吾亭之名記之，何等瀟灑！

### （四）寫作特色總評

| 層面 | 特色 |
|------|------|
| **結構** | 起（破題）—承（歷史典故）—轉（情況逆轉，雨至）—合（亭上對答、歌舞收束） |
| **修辭** | 排比（三古事、六種人、四層遞推）、對仗（雨珠雨玉）、頂真（層層推功） |
| **風格** | 歡快明暢，不故作深沉。與范仲淹〈岳陽樓記〉的凝重、歐陽修〈醉翁亭記〉的灑脫、曾鞏〈墨池記〉的嚴謹皆不同，純是少年氣盛 |
| **思想** | 儒家的民本思想 + 天人感應的雨澤觀 + 蘇軾特有的樂觀幽默 |
| **地位** | 蘇軾「亭記四篇」（喜雨亭記、凌虛臺記、超然臺記、放鶴亭記）之首，是研究蘇軾青年時期思想與文風的最佳文本 |

## 五、原文全文

**喜雨亭記** · 蘇軾

亭以雨名，誌喜也。古者有喜，則以名物，示不忘也。周公得禾，以名其書；漢武得鼎，以名其年；叔孫勝敵，以名其子。其喜之大小不齊，其示不忘一也。

予至扶風之明年，始治官舍。為亭於堂之北，而鑿池其南，引流種樹，以為休息之所。是歲之春，雨麥於岐山之陽，其占為有年。既而彌月不雨，民方以為憂。越三月，乙卯乃雨，甲子又雨，民以為未足。丁卯大雨，三日乃止。官吏相與慶於庭，商賈相與歌於市，農夫相與忭於野，憂者以喜，病者以愈，而吾亭適成。

於是舉酒於亭上，以屬客而告之，曰：「五日不雨可乎？」曰：「五日不雨則無麥。」「十日不雨可乎？」曰：「十日不雨則無禾。」「無麥無禾，歲且荐饑，獄訟繁興，而盜賊滋熾。則吾與二三子，雖欲優遊以樂於此亭，其可得耶？今天不遺斯民，始旱而賜之以雨，使吾與二三子得相與優遊而樂於此亭者，皆雨之賜也。其又可忘耶？」

既以名亭，又從而歌之，曰：「使天而雨珠，寒者不得以為襦；使天而雨玉，飢者不得以為粟。一雨三日，伊誰之力？民曰太守。太守不有，歸之天子。天子曰不然，歸之造物。造物不自以為功，歸之太空。太空冥冥，不可得而名。吾以名吾亭。」
