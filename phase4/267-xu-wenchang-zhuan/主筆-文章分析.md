# 〈徐文長傳〉文章分析

> **主筆分析 · 古文觀止卷十二·明文·袁宏道**

---

## 一、文章結構與起承轉合

### 1.1 全文結構總覽

〈徐文長傳〉全文約八百字，是《古文觀止》中最具革命性的傳記文。袁宏道完全拋棄了司馬遷以來的史傳傳統——不按時間順序列生平，不以「太史公曰」作結，而是以「奇」字為綱領，從自己的「發現」經驗寫起，層層推進，最終以一聲「悲夫」收束。全文分為四大層次：

| 段落 | 核心內容 | 功能 |
|:----:|----------|:----:|
| **發現（開頭）** | 余少時過里肆中……兩人躍起，燈影下，讀復叫，叫復讀，僮僕睡者皆驚起 | **開篇造勢**——以親身發現徐渭詩稿的震撼經歷開篇，製造強烈懸念：什麼樣的詩能讓人讀到大叫、驚醒僮僕？ |
| **生平（主體一）** | 徐渭，字文長……議者方之劉真長、杜少陵云 | **勾勒生平**——從身世寫到胡宗憲幕府經歷，用「葛衣烏巾，縱談天下事」與「介冑之士，膝語蛇行」的強烈對比，凸顯徐渭的風骨之奇 |
| **藝術（主體二）** | 文長既已不得志於有司……皆超逸有致 | **論證才華**——從詩、文、書、畫四個方面逐層評析，用七個比喻（如嗔如笑、如水鳴峽等）集中展示徐渭的藝術成就之奇 |
| **悲劇（結尾）** | 卒以疑殺其後妻……無之而不奇，斯無之而不奇也。悲夫！ | **收束昇華**——從「數奇」到「狂疾」到「囹圄」的悲劇鏈條，最終以「無之而不奇，斯無之而不奇也」收束，同字異義，令人震撼 |

### 1.2 起承轉合分析

〈徐文長傳〉不以傳統的「起承轉合」為框架，而是以「奇」字為軸心，形成了一種螺旋推進的結構：

| 結構 | 對應段落 | 核心內容 | 寫作策略 |
|:----:|:--------:|----------|:---------|
| **起** | 開頭：「余少時過里肆中……如魘得鹿，如久渴而得瀑」 | 以第一人稱講述自己「發現」徐渭的過程。少年在里肆見《四聲猿》——驚疑；後見字畫署「田水月」——更驚；終於在陶編修樓上翻到《闕編》——徹底震撼。三層遞進，懸念層層疊加。 | **以「發現」替代「介紹」** 。傳統傳記開頭是「韓信者，淮陰人也」，袁宏道卻從自己的閱讀經驗寫起——讀者跟著作者的視角，一步步走近徐渭，彷彿也在經歷一次「考古式」的發現。 |
| **承** | 主體一：「文長之為人……議者方之劉真長、杜少陵云」 | 正式介紹徐渭的生平：山陰諸生——薛蕙賞識——屢試輒蹶——入胡宗憲幕——葛衣烏巾傲視總督——代作白鹿表。重點是布衣對王侯的「傲」。 | **以「奇」字統攝生平選材**。不寫徐渭的生卒年月，不列舉科舉年表，只選擇最能體現「奇」的細節——葛衣烏巾、膝語蛇行、密議兵計、樓上斬健兒。每一個細節都是「奇」的具體化。 |
| **轉** | 主體二：「文長既已不得志於有司……皆超逸有致」 | 從生平轉入藝術評價。先交代「不得志於有司」的命運背景，再逐一評析詩、文、書、畫四項成就。這是全篇藝術成就最高的部分。 | **以「不得志」解釋「藝術之奇」** 。袁宏道提出一個精彩的因果鏈：因為科舉失意→所以放浪山水→所以見天地奇觀→所以胸中鬱結「不可磨滅之氣」→所以詩有「王者氣」。苦難→創作，這是袁宏道對徐渭藝術的根本解釋。 |
| **合** | 結尾：「卒以疑殺其後妻……悲夫！」 | 收束於「數奇」——從殺妻下獄到自殺九次到晚年潦倒，最終以「無之而不奇，斯無之而不奇也。悲夫！」作結。 | **以「悲夫」替代「太史公曰」** 。傳統傳記結尾是史家的蓋棺論定，袁宏道卻以一聲長嘆收束——不是冷靜的評判，而是無法自已的悲憫。 |

### 1.4 結構特點

1. **以「奇」字統攝全篇**：全文沒有一條清晰的時間線，而是以「奇」為經，以生平、藝術、命運為緯，編織成網。這在《古文觀止》全書的傳記文中是獨一無二的——袁宏道關心的是「徐渭有多奇、有多悲」，而不是「徐渭這一生做了什麼」。

2. **作者作為「發現者」直接登場**：傳統傳記的作者隱身於敘述之後（「太史公曰」也只出現在結尾），但袁宏道在第一段就以「余」的身份出場，講述自己發現徐渭的經歷。這不是史家筆法，而是知音筆法——袁宏道不是徐渭的歷史記錄者，而是徐渭的精神知音。

3. **以比喻替代評判**：傳統傳記評價人物，多用判斷句（「某，能人也」）。袁宏道評價徐渭的詩，用了七個比喻（如嗔如笑、如水鳴峽、如種出土、如寡婦之夜哭、羈人之寒起）——他不是在「評」，而是在「感」。這是公安派「獨抒性靈」的具體實踐。

4. **「同字異義」的修辭收束**：結尾「無之而不奇，斯無之而不奇也」——前一個「奇」是「奇才」，後一個「奇」是「奇數」（命運乖戾）。同一個字，前後含義截然不同，卻精確地概括了徐渭的全部悲劇：才華有多奇絕，命運就有多坎坷。

> 💡 **主筆按**：〈徐文長傳〉的結構革命，本質上是公安派文學主張在傳記文體上的實踐。袁宏道認為「文章新奇，無定格式」——寫文章不該被傳統的框架綁死。他寫徐渭這樣的「奇人」，如果用司馬遷的史傳筆法，就像用八股文的格子寫狂草，形式與內容完全不相容。所以他創造了一種全新的傳記寫法：以情感為線索、以比喻為工具、以知音為立場。這種寫法在當時是驚世駭俗的，但正因為它真誠、它有力，它最終成為了經典。

## 二、修辭與寫作手法

### 2.1 修辭手法分析

| 修辭手法 | 原文例證 | 作用與效果 |
|:---------|----------|:-----------|
| **比喻（博喻）** | 「其為詩，如嗔如笑，如水鳴峽，如種出土，如寡婦之夜哭，羈人之寒起」 | **全篇最精彩的修辭段落**。袁宏道一口氣用了七個比喻來描述徐渭的詩——不是理性的分析，而是感性的共鳴。七個意象各自代表一種極端的情感狀態：憤怒（嗔）、狂放（笑）、激盪（水鳴峽）、頑強（種出土）、悲愴（寡婦夜哭）、孤獨（羈人寒起）。這種「博喻」手法（用多個比喻共同描寫一個對象），在中國古典散文中極為罕見。 |
| **對比** | 「介冑之士，膝語蛇行，不敢舉頭；而文長以部下一諸生傲之」 | **視覺衝擊力極強的畫面對比**。一邊是全副武裝的將軍們跪著說話（「膝語蛇行」），一邊是一個布衣秀才昂首挺立（「傲之」）。地位與姿態的雙重反差，讓徐渭的風骨躍然紙上。 |
| **層遞** | 「先生數奇不已，遂為狂疾；狂疾不已，遂為囹圄」 | **因果鏈條式層遞**。三個短句，環環相扣——因為命運不斷打擊→所以發狂；因為發狂→所以入獄。每一環都無法避免，每一次打擊都推向下一次更深的深淵。這是全文最令人心碎的句子。 |
| **同字異義** | 「無之而不奇，斯無之而不奇也」 | **全文的文眼，也是最精彩的語言遊戲**。前一個「奇」= 奇才、奇絕；後一個「奇」= 奇數、命運乖戾。同一個字，前後兩種截然不同的含義，形成強烈的反諷——才華越奇，命運越奇。最後「悲夫」二字，將所有力量凝聚在結尾。 |
| **排比** | 「其所見山奔海立，沙起雲行，風鳴樹偃，幽谷大都，人物魚鳥，一切可驚可愕之狀」 | 一連串四字短語排比而出，節奏急促，讀者彷彿也跟著徐渭的足跡，經歷了那山奔海立的壯闊景象。排比造成的速度感，模擬了「放浪山水」的奔放節奏。 |
| **感嘆** | 「悲夫！」 | 全文最後一個詞，也是唯一一個直接抒發情感的感嘆詞。以一聲長嘆收束全篇，既避免了傳統「太史公曰」的冷靜，又以極簡的方式完成了情感的昇華。 |
| **反襯** | 「惡楮毛書，煙煤敗黑，微有字形」 | 用最差的紙（惡楮）、最模糊的字跡（微有字形）來反襯詩歌內容的偉大。外表越破爛，內在越驚人——這是徐渭一生的隱喻。 |
| **反覆** | 「讀復叫，叫復讀」 | 通過「讀」與「叫」兩個動作的反覆交替，表現袁宏道和陶望齡讀詩時的狂喜狀態。五個字寫出兩個人的癡迷——讀兩句就叫好，叫完繼續讀，讀了又叫。 |
| **引用** | 「歐陽公所謂『妖韶女老自有餘態』者也」 | 用歐陽修評論書法的名言來評價徐渭的書法——「蒼勁中姿媚躍出」，像是年老的美人仍有餘韻。借歐陽修的話，既提高了評價的權威性，又以形象化的語言傳達了徐渭書法的特質。 |

### 2.2 寫作策略

**策略一：反傳統的傳記寫法——從「發現」開始，不以時間為綱**

這是〈徐文長傳〉最核心的寫作策略，也是它與《古文觀止》全書所有傳記最大的不同。

傳統傳記（從〈管晏列傳〉到〈五人墓碑記〉）遵循的基本範式是：
```
某某者，某地人也 → 少年 → 中年事蹟 → 晚年 → 卒 → 史家評語
```

袁宏道完全打破了這個範式。他從自己發現徐渭的經歷開始寫——這是一個極具創意的開場：讀者跟著作者的目光，先看到《四聲猿》（不知作者）→ 後看到「田水月」字畫（仍不知何人）→ 最後在燈下讀到《闕編》（終於知道是徐渭）。這個過程本身就是一個懸念的疊加——袁宏道用考古學家發現寶藏的方式，來寫一個天才的傳記。

這種寫法的效果是：讀者在還不知道徐渭是誰的時候，已經被袁宏道的狂喜所感染——「讀未數首，不覺驚躍」「如魘得鹿，如久渴而得瀑」——讀者會想，什麼樣的詩能讓人這樣？於是帶著強烈的好奇心繼續讀下去。

**策略二：以「不得志」解釋「藝術之奇」——苦難創作論**

袁宏道對徐渭藝術成就的解釋，不是簡單的「天才論」，而是一套完整的因果鏈：

```
科舉失意（不得志於有司）
    │
    ▼
放浪山水（走齊魯燕趙之地，窮覽朔漠）
    │
    ▼
見天地奇觀（山奔海立，沙起雲行，風鳴樹偃）
    │
    ▼
胸中積鬱不可磨滅之氣（英雄失路、托足無門之悲）
    │
    ▼
發而為詩（如嗔如笑，如水鳴峽，如種出土…）
    │
    ▼
詩有王者氣（匠心獨出，非巾幗而事人者所敢望）
```

這個因果鏈條是理解全文的關鍵。袁宏道認為，徐渭的藝術成就不是憑空而來的——它的根源是「不得志」。因為不被主流社會接納，所以徐渭只能把自己的全部生命力注入山水與創作之中；因為胸中有一團「不可磨滅之氣」，所以他的詩才有那樣震撼人心的力量。

這實際上是袁宏道對中國文學批評史上「發憤著書」傳統的繼承與發展。司馬遷說「詩三百篇，大抵賢聖發憤之所為作也」——但司馬遷是從普遍意義上說的。袁宏道則具體地、細緻地描述了從「不得志」到「創作」的轉化過程——這已經接近現代心理學對創造力的理解了。

**策略三：以「同字異義」完成最高級的語言藝術**

全文最精彩的語言手法，是結尾的「無之而不奇，斯無之而不奇也」。袁宏道用同一個「奇」字，在短短十三個字中實現了含義的翻轉：

- 前兩個「奇」：奇才、奇絕 → 徐渭在詩文書畫等所有方面都奇
- 後兩個「奇」：奇數、命運乖戾 → 正因為處處奇，所以處處坎坷

這種修辭手法在古文中稱為「同字異義」——用同一個字在不同語境中的不同含義，形成強烈的對比與反諷。袁宏道不是在玩文字遊戲，他是在用一個字來承載徐渭全部的生命悲劇：天才與苦難，是同一個硬幣的兩面。

### 2.3 節奏控制

〈徐文長傳〉的節奏極具辨識度，呈現出「先慢後快、快中帶緩、終歸於靜」的節奏曲線：

- **開篇（懸疑節奏）**：從「余少時過里肆中」到「得《闕編》詩一帙」——節奏較慢，細節豐富，像偵探小說的開場，一步步接近真相。直到「讀未數首，不覺驚躍」，節奏驟然加速。

- **中段生平（敘事節奏）**：「文長每見，則葛衣烏巾，縱談天下事」——簡潔的敘事句，節奏從容。但到「一軍股栗」「其信任多此類」等短句時，節奏突然緊湊起來，模擬了胡宗憲對徐渭的雷厲風行的信任。

- **藝術評價（激越節奏）**：從「其胸中又有一段不可磨滅之氣」開始，節奏明顯加速。七個比喻連續轟擊——「如嗔如笑，如水鳴峽，如種出土，如寡婦之夜哭，羈人之寒起」——比喻之間沒有停頓，一口氣讀下來，就像連珠炮一樣，讀者根本沒有喘息的機會。這是全篇節奏最激烈的部分，也是情感濃度的最高點。

- **悲劇收束（沉緩節奏）**：從「卒以疑殺其後妻」開始，節奏驟然放緩。句子變長、語氣變沉重——「晚年憤益深，佯狂益甚」——最後落在「無之而不奇，斯無之而不奇也。悲夫！」，節奏降至最慢，情緒卻收束在最濃烈處。

這種節奏變化可以用一條曲線來表示：

```
節奏快慢
    ↑
    │         ╱╲
    │        ╱  ╲        (藝術評價：七喻連珠)
    │       ╱    ╲
    │  ╱──╱      ╲──╲  (生平敘事)
    │ ╱                ╲
    │╱                  ╲ (悲劇收束)
    └─────────────────────────→ 時間
      開頭（懸疑）    結尾（沉緩）
```

## 三、名句深度解析

### 3.1 「兩人躍起，燈影下，讀復叫，叫復讀，僮僕睡者皆驚起。」

**白話：** 兩個人在燈光下跳起來，讀了又叫好，叫好了又繼續讀，連已經睡下的僕人都被驚醒了。

**賞析：**

這是中國傳記文學中最生動的開頭段落，也是一個極具畫面感的「發現場景」。袁宏道沒有說徐渭的詩有多好——沒有引用具體的詩句、沒有做理性的分析——他只用一個場景，讓讀者自己去感受。

「兩人躍起」——不是站起來，是「躍起」，跳起來。什麼樣的書能讓人讀得跳起來？「燈影下」——夜深了，燈光昏暗，兩個人圍著一盞燈。這是一個極其私密的閱讀場景。「讀復叫，叫復讀」——五個字寫出兩個人的狂喜狀態：讀兩句就忍不住叫好，叫完繼續讀，讀了又叫。這種「讀」與「叫」的反覆交替，不是冷靜的閱讀，而是忍不住的喝采。

最後一句「僮僕睡者皆驚起」——連僕人都被吵醒了。這是側面描寫：不需要正面說「這詩多好」，只需要告訴讀者「兩個人讀詩讀到把僕人都吵醒了」——讀者自然會想像，那是怎樣震撼的詩歌。

這一段的寫法也是公安派「性靈說」的絕佳體現：袁宏道不隱藏自己的情感，不克制自己的反應——他把自己最真實的狂喜寫出來，讓這份狂喜本身成為評價徐渭詩歌的最好方式。

### 3.2 「文長每見，則葛衣烏巾，縱談天下事。」

**白話：** 徐文長每次去見（胡宗憲），穿的是粗布衣、戴的是黑頭巾，放開胸懷議論天下大事。

**賞析：**

十一個字，畫出一個人一生的風骨。

「葛衣烏巾」——粗布衣服、黑色頭巾，這是明代布衣（沒有功名的讀書人）的標準裝扮。但在胡宗憲的總督府中，四周都是錦衣華服的將軍們，徐渭這身打扮格外扎眼。他不是沒有好衣服穿，他是故意的——用最樸素的裝束來表明自己的身份：我不是你的屬下，我是你的客人。

「縱談天下事」——一個「縱」字，寫盡了徐渭的狂放。他不是小心翼翼地匯報工作，而是「縱談」——放開來說，無所顧忌地說。「天下事」三個字，暗示了談話的內容不是瑣碎小事，而是軍國大事——徐渭雖然是布衣，但他談的是天下興亡。

這十一個字與後文「介冑之士，膝語蛇行，不敢舉頭」形成極具張力的對比。將軍們跪著說話（膝語）、像蛇一樣在地上爬行（蛇行），而一個布衣卻昂首挺胸、高談闊論。這種「上下顛倒」的畫面，是袁宏道最擅長的筆法——他不需要評論，只需要呈現。讀者自會感受到：徐渭的「傲」不是裝出來的，是真有傲的資本——因為胡宗憲需要他的才華，因為他談的是「天下事」。

### 3.3 「其胸中又有一段不可磨滅之氣，英雄失路、托足無門之悲。」

**白話：** 他的胸中有一股無法磨滅的浩然之氣，是英雄找不到出路、無處安身的悲愴。

**賞析：**

這是袁宏道對徐渭藝術成就的終極解釋，也是全文從「生平敘述」轉入「藝術評價」的關鍵轉折句。

「不可磨滅之氣」——這是全文最有力量的詞組之一。袁宏道認為，徐渭的詩文書畫之所以有那樣的力量，不是因為技巧高超，而是因為胸中有這股氣。這股氣是什麼？是憤怒，是不平，是對命運的反抗——它不會因為打擊而消失，不會因為困頓而熄滅，它「不可磨滅」。

「英雄失路」——英雄找不到路。徐渭有經天緯地之才，但科舉制度不給他出路，社會不給他位置。他就像一頭被困在牢籠中的猛獸，有力無處使。

「托足無門」——連落腳的地方都找不到。這比「失路」更加絕望——失路是找不到方向，托足無門是連站的地方都沒有。徐渭在胡宗憲幕府時還有立足之地，胡宗憲死後，他連這個位置都失去了。

袁宏道把這兩句放在一起，構成了一種雙重絕望：不僅是「失路」（沒有出路），而且是「托足無門」（沒有位置）。這解釋了徐渭為什麼會發狂——當一個人的才華和能量沒有被安置的空間時，它只能向內坍縮，成為毀滅自己的力量。

### 3.4 「先生數奇不已，遂為狂疾；狂疾不已，遂為囹圄。」

**白話：** 先生因為命運一直不順，於是發了狂疾；因為狂疾一直不好，於是進了監獄。

**賞析：**

這是全文最令人心痛的句子，也是袁宏道對徐渭悲劇的核心診斷。

「數奇不已」——「數奇」是徐渭命運的總綱。袁宏道在前面已經兩次提到「數奇」——第一次是「然數奇，屢試輒蹶」，第二次是隱含在「不得志於有司」中。到這裡，「數奇不已」是第三次出現，但加了「不已」二字——不是一次兩次的不順，而是持續不斷的打擊。

「遂為狂疾」——「遂」字是關鍵。它傳達的是一種「必然性」——不是偶然失心，而是命運層層壓迫下的必然結果。袁宏道認為徐渭的發狂不是病理意義上的，而是命運意義上的——一個人被逼到極限，發狂是唯一可以理解的出路。

「狂疾不已，遂為囹圄」——第二個「不已」再次強調了不斷疊加的惡性循環。狂疾不只是徐渭的個人悲劇，它還導致了殺妻的慘劇（「卒以疑殺其後妻」——注意「以疑」兩個字，袁宏道暗示殺妻是源於精神疾病的妄想，而不是蓄意謀殺）。

三個短句，兩個「不已」，兩個「遂為」，構成了一種不可阻擋的命運鏈條。每一個環節都無法避免，每一次打擊都推向下一次更深的深淵。這不是徐渭的錯，這是一個天才在一個不接納天才的社會中的必然結局。

### 3.5 「文長無之而不奇者也。無之而不奇，斯無之而不奇也。悲夫！」

**白話：** 文長沒有一處不奇絕。正因為沒有一處不奇絕，所以沒有一處不坎坷。可悲啊！

**賞析：**

這是全文的結尾，也是中國文學史上最精煉、最震撼的人物評語之一。短短十三個字（加「悲夫」二字共十五字），總結了徐渭的全部人生。

語言技巧上，這是「同字異義」修辭的典範。前兩個「奇」是奇才、奇絕（形容詞），後兩個「奇」是奇數、命運乖戾（名詞）。袁宏道用同一個字，在兩句話之間完成了含義的翻轉——事情做得太奇了，命運就會太奇了。

句式結構上，三個「無之而不……」，構成了一個層層推進的節奏：
- 「無之而不奇」：肯定——徐渭處處奇絕
- 「無之而不奇」：翻轉——同一個「奇」，含義變成了不幸
- 「無之而不奇也」：總結——所以處處不幸

最後的「悲夫」是全文唯一直接抒發情感的詞。袁宏道在整篇文章中一直保持著理性的克制（即使在最激動的發現場景中，他也只是在敘述，而不是直接抒情），但到這裡，他再也忍不住了。一聲「悲夫」，包含了全部的同情、惋惜、憤怒和無奈。

> 💡 **主筆按**：這十三個字之所以震撼，不僅僅是因為它的語言技巧，更是因為它觸及了一個永恆的命題：天才與痛苦的關係。中國文化中始終存在一種「天才詛咒」的觀念——才華越高，命運越悲。從屈原到賈誼到李賀到徐渭，這個模式反覆出現。袁宏道用十三個字將這個命題濃縮到極致——他不是在為徐渭一個人嘆息，他是在為所有被時代辜負的天才嘆息。

### 4.3 全文的「三奇」結構

袁宏道的〈徐文長傳〉以「奇」字統攝全文，但這個「奇」可以細分為三個層面：

| 層面 | 內容 | 原文證據 | 功能 |
|:----:|:----:|:---------|:----:|
| **才華之奇** | 徐渭在六個領域（詩、文、書、畫、戲曲、軍事）都達到頂尖水準 | 「匠心獨出，有王者氣」「韓曾之流亞」「蒼勁中姿媚躍出」「超逸有致」 | 建立徐渭作為「曠世奇才」的形象——他不是一般的文人，是百年一遇的天才 |
| **風骨之奇** | 布衣傲王侯的氣節、不與時調合的態度 | 「葛衣烏巾，縱談天下事」「介冑之士膝語蛇行，而文長以部下一諸生傲之」「文長皆叱而奴之」 | 凸顯徐渭的人格魅力——他不只是才華出眾，更是人格獨立 |
| **命運之奇** | 八次落第、精神崩潰、自殺九次、殺妻入獄、貧困而死 | 「然數奇，屢試輒蹶」「自持斧擊破其頭」「以疑殺其後妻」「古今文人牢騷困苦未有若先生者」 | 完成悲劇主題——才華越奇，命運越奇，形成強烈的反諷 |

這三個層面的「奇」，最終在結尾的「無之而不奇，斯無之而不奇也」中匯聚：才華之奇讓徐渭成為徐渭，命運之奇讓徐渭付出代價——而風骨之奇，是這兩者之間的橋樑。正因為他不肯妥協（風骨之奇），他才會保持才華的純粹（才華之奇），也正因為他不肯妥協，他才被時代拋棄（命運之奇）。

### 4.4 文章技巧總結

〈徐文長傳〉之所以成為千古名篇，得益於以下幾個層面的技巧：

1. **立意突破**：打破司馬遷以來的史傳傳統，以「奇」字為綱，以個人情感為線索——傳記不再是史學的附屬，而是獨立的文學創作

2. **結構創新**：不按時間順序列生平，而以主題為綱——發現→生平→藝術→悲劇，四段螺旋推進，層層深入

3. **修辭精湛**：七喻評詩（博喻）、葛衣烏巾（對比）、數奇不已（層遞）、無之而不奇（同字異義）——每一種修辭手法都精準地服務於內容

4. **節奏控制**：從懸疑到激越到沉緩的節奏曲線，讓讀者在不知不覺中被帶入情感的高潮與低谷

5. **以情動人**：袁宏道不是以史家的冷靜來評判徐渭，而是以知音的熱情來為他鳴不平。這種「獨抒性靈」的寫法，讓這篇傳記充滿了超越一般傳記的情感力量

> 💡 **主筆終按**：〈徐文長傳〉最了不起的地方，在於它完美地實現了「形式即內容」的最高境界。袁宏道用一篇「不按牌理出牌」的傳記，來寫一個「不按牌理出牌」的天才——傳記的奇，與傳主的奇，形成了完美的共振。如果袁宏道用傳統的史傳筆法來寫徐渭——時間順序、生平事蹟、太史公曰——那這篇文章不過是又一篇普通的傳記，徐渭也不過是一個命運悲慘的文人。但正因為袁宏道敢於打破傳統，敢於用自己的方式來寫，徐渭的「奇」才真正被讀者感受到。這就是公安派「獨抒性靈，不拘格套」的最高成就——不是理論上的宣導，而是在創作中的實現。
