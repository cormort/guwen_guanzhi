# 〈賈誼論〉文章分析

> **主筆分析 · 古文觀止卷十一·宋文·蘇軾**

---

## 一、總論

〈賈誼論〉是蘇軾早年（約嘉祐年間，1057–1060）史論翻案文的代表作，收於《經進東坡文集事略》卷八。全文不過五六百字，卻完成了一次對千年歷史定論的徹底翻轉——歷代讀賈誼，多怪漢文帝不用賢才、權臣排擠英才；蘇軾偏說問題出在賈誼自己身上。核心命題是一句驚人之語：「非漢文之不能用生，生之不能用漢文也。」

蘇軾不是在寫歷史考證，他是在借賈誼說一個關於「忍耐與等待」的普遍命題。從北宋士人精神自省的大背景看，這篇文章表面是史論，骨子裡是一篇修身勸世文——寫給所有「覺得自己被世界虧待了」的人。以下從起承轉合、修辭手法、論證邏輯、名句解析四個層面深入分析。

---

## 二、起承轉合分析

〈賈誼論〉全文結構可分為五個段落，對應起承轉合中的四個功能層次：

| 段落 | 起承轉合 | 對應原文 | 核心功能 | 論證策略 |
|:----:|:--------:|----------|----------|----------|
| **第1段**（「非才之難」至「或者其自取也」） | **起** | 「非才之難，所以自用者實難。惜乎！賈生，王者之佐，而不能自用其才也。夫君子之所取者遠，則必有所待；所就者大，則必有所忍。古之賢人，皆負可致之才，而卒不能行其萬一者，未必皆其時君之罪，或者其自取也。」 | 開門見山，亮出核心論點——賈誼的問題不在朝廷，在他自己。同時提出全篇理論框架：「有待有忍」的君子哲學。 | **先立論、後設框**：第一句即翻轉定論，緊接著提出「待與忍」的普遍原則作為分析工具，使讀者在進入具體討論前已接受了蘇軾的評價基準。 |
| **第2段**（「愚觀賈生之論」至「生之不能用漢文也」） | **承** | 「愚觀賈生之論……雖三代何以遠過？得君如漢文，猶且以不用死。然則是天下無堯舜，終不可有所為耶？……仲尼聖人，歷試於天下……孟子去齊，三宿而後出晝……若賈生者，非漢文之不能用生，生之不能用漢文也。」 | 深化論證：以孔子、孟子為正面案例，建立「君子如何對待君主」的標準——勤於尋求、厚於不忍、至於愛身。再以這個標準衡量賈誼，得出結論。 | **以聖賢為尺**：用孔子周遊列國、孟子三宿出晝的典故，建立一個「事君應有的態度」的標杆，再將賈誼的行為與之對照，落差立現。 |
| **第3段**（「夫絳侯親握天子璽」至「是亦不善處窮者也」） | **轉** | 「夫絳侯親握天子璽而授之文帝，灌嬰連兵數十萬……賈生，洛陽之少年，欲使其一朝之間，盡棄其舊而謀其新，亦已難矣。為賈生者……優遊浸漬而深交之……安有立談之間，而遽為人『痛哭』哉？……觀其過湘，為賦以弔屈原，紆鬱憤悶，趯然有遠舉之志。其後卒以自傷哭泣，至於夭絕。是亦不善處窮者也。」 | 筆鋒從「應該怎樣做」轉向「賈誼實際做了什麼」——具體分析賈誼被貶的全過程，從政治現實角度指出其天真：22歲的外來年輕人想一朝取代開國元勳，任何系統都會反彈。更致命的是他面對挫折的態度——「自傷哭泣，至於夭絕」。 | **設身處地 → 反事實推演**：先還原賈誼處境（功臣共治的政治現實），再以「為賈生者」開頭進行反事實假設——如果他懂得「優遊浸漬而深交之」，十年後就能得志。這種推演極具說服力。 |
| **第4段**（「夫謀之一不見用」至「才有餘而識不足也」） | **轉**（轉入結論性批判） | 「夫謀之一不見用，則安知終不復用也？不知默默以待其變，而自殘至此。嗚呼！賈生志大而量小，才有餘而識不足也。」 | 從三個方向完成對賈誼的批判：(1) 一次不被採納不等於永遠不被採納——邏輯謬誤；(2) 不該「自殘」——心理素質問題；(3) 直接下判語——「志大而量小，才有餘而識不足」。 | **邏輯反駁 → 心理分析 → 終極判語**：三層遞進，從理性到情感，層層收緊。 |
| **第5段**（「古之人」至「亦謹其所發哉」） | **合** | 「古之人，有高世之才，必有遺俗之累……愚深悲生之志，故備論之。亦使人君得如賈生之臣，則知其有狷介之操，一不見用，則憂傷病沮，不能復振。而為賈生者，亦謹其所發哉！」 | 收束全文，包含兩個層次：(1) 為君主發言——如果遇到賈誼這樣的大臣，要知道他性子剛烈，一次不用就會一蹶不振；(2) 為賈誼發言——你也要謹慎自己的言行啊！最後一句「亦謹其所發哉」既是告誡也是惋惜。 | **雙向勸誡**：一面告誡君主（要體諒賈誼這類大臣的狷介之操），一面告誡士人（要謹慎自己的表現與心態）。雙向收束，格局宏大。 |

### 起承轉合結構圖解

```
起（第1段）：亮出核心論點 + 提出「有待有忍」的理論框架
     │
     ↓
承（第2段）：以孔孟為尺，建立君子事君的規範標準
     │
     ↓
轉（第3段）：回到賈誼具體處境——政治現實分析與反事實推演
     │
     ↓
轉（第4段）：邏輯反駁 + 心理分析 → 終極判語「志大而量小，才有餘而識不足」
     │
     ↓
合（第5段）：雙向勸誡——告誡君主，也告誡士人，以「謹其所發」收束
```

### 結構特點

1. **先理論後案例**：不直接討論賈誼，而是先提出「有待有忍」的普遍道理，讓讀者先接受這個評價尺規，再套用到賈誼身上——使翻案論證建立在穩固的哲學基礎上。
2. **以聖賢壓才子**：用孔子、孟子這兩個道德權威來壓制賈誼，使讀者難以反駁——「連孔子都這麼有耐心，你賈誼有什麼資格不耐煩？」
3. **雙重轉折設計**：不像一般文章只有一個「轉」，蘇軾在第3段（具體分析）和第4段（終極判語）安排了兩個轉折層次，使論證更加豐滿有力。
4. **雙向收束**：結尾不是單方面的批判，而是同時指向君主和士人——這種雙向視角使文章格局超出一般的翻案文，具有了普遍的勸世價值。

## 三、修辭手法分析

| 修辭手法 | 原文例證 | 作用與效果 |
|:---------|----------|:-----------|
| **翻案（反常理立論）** | 「非漢文之不能用生，生之不能用漢文也。」 | 全篇的立論基石。翻轉千年定論，以「生之不能用漢文」這一詭譎命題製造強烈的認知衝擊，迫使讀者放下成見、重新思考。這是蘇軾史論最鋒利的武器——反眾人之見而獨出新論。 |
| **設問（懸問）** | 「然則是天下無堯舜，終不可有所為耶？」 | 設下一個看似無法反駁的陷阱問題——如果只有堯舜才能用你，那天下豈不是永遠不能有所作為？隨之以孔子、孟子為例破除這個謬論。設問在此處的功能是「預設錯誤前提，再加以推翻」。 |
| **設問（懸問）** | 「夫謀之一不見用，則安知終不復用也？」 | 以反問形式提出一個邏輯問題——一次不被採納，為什麼就認定永遠不會再被採納？這個設問直接戳破了賈誼行為的邏輯漏洞，論證極為犀利。 |
| **設喻（暗喻）** | 「又皆高帝之舊將，此其君臣相得之分，豈特父子骨肉手足哉？」 | 將周勃、灌嬰與文帝的關係比喻為「父子骨肉手足」——暗示這份君臣情分遠非普通關係可比，賈誼一個外來少年想一朝取代，勢比登天。 |
| **設喻（暗喻）** | 「安有立談之間，而遽為人『痛哭』哉？」 | 以「立談之間」喻時間之短、「痛哭」喻賈誼在《治安策》中的激烈措辭——諷刺賈誼在沒有任何鋪墊的情況下就要求君主全面變革，政治智慧上的幼稚一目了然。 |
| **引用（用典）** | 「仲尼聖人，歷試於天下……將之荊，先之以冉有，申之以子夏。」 | 以孔子周遊列國、先後派遣弟子打前站的典故，建立「君子勤於事君」的標準。孔子對待不認識的國君都如此耐心，賈誼對待賞識他的漢文帝卻如此暴躁——對比強烈。 |
| **引用（用典）** | 「孟子去齊，三宿而後出晝，猶曰：『王其庶幾召我。』」 | 用孟子離開齊國時在晝地停留三天的典故，說明君子不忍輕易拋棄君主。這個典故的巧妙之處在於：孟子在齊國其實也不被重用，但他選擇等待，而不是像賈誼一樣「紆鬱憤悶」。 |
| **引用（用典）** | 「公孫丑問曰：『夫子何為不豫？』孟子曰：『方今天下，舍我其誰哉？而吾何為不豫？』」 | 最精彩的典故運用。孟子說「方今天下，舍我其誰哉」——這是極大的自信，但蘇軾將它解釋為「君子之愛其身，如此其至也」。意思是：正因為孟子珍惜自己的才華，他才不輕易放棄、不輕易沮喪。這反過來諷刺賈誼——你如果真愛惜自己的才華，就不該33歲把自己氣死。 |
| **引用（用典）** | 「古今稱苻堅得王猛於草茅之中，一朝盡斥其舊臣而與之謀。」 | 用苻堅與王猛的案例作為「明君能用賢才」的對比參照——說明如果君主真正鐵了心，還是可以做到的。但問題在於：(1) 苻堅是苻堅，文帝是文帝；(2) 這樣的案例本就「古今稱」——極為稀有，不能當作常態。蘇軾用這個案例既設置了一個理想標準，又暗示了它的稀有性。 |
| **對比** | 孔孟（耐心等待、積極進取） vs. 賈誼（自傷哭泣、消極放棄） | 全文最核心的對比——一個是「君子愛其身」、「勤於事君」，一個是「不善處窮」、「自殘至此」。正面案例越崇高，賈誼的行為越顯得不合理。 |
| **對比** | 賈誼在朝（一年升太中大夫） vs. 賈誼被貶後（「卒以自傷哭泣，至於夭絕」） | 蘇軾暗示：如果賈誼真的懷才不遇，為什麼一開始能火箭般升遷？這說明文帝確實賞識他。問題在於他一遇挫折就徹底崩潰——從極度自信到極度自棄，反差之大恰恰證明了他「志大而量小」。 |
| **層遞（遞進）** | 「志大而量小，才有餘而識不足」 | 兩個並列短語構成遞進關係：「志大而量小」是性格判斷——有宏大志向但格局狹窄；「才有餘而識不足」是能力判斷——才華橫溢但缺乏見識和智慧。兩句疊加，完成了對賈誼的完整人格批判。 |
| **層遞（遞進）** | 「默默以待其變」→「自殘至此」→「嗚呼！」 | 從應該怎麼做（默默等待）到做了什麼（自殘），再到感嘆（嗚呼），三層遞進，情感從理性分析逐步升溫到痛惜。 |
| **呼告** | 「惜乎！賈生，王者之佐，而不能自用其才也。」 | 開篇即以「惜乎」發聲，相當於對著歷史人物當面喊話——既表明了蘇軾的態度（惋惜），也迅速拉近了讀者與文本的情感距離。 |
| **呼告** | 「嗚呼！賈生志大而量小，才有餘而識不足也。」 | 第二次也是最後一次呼告，緊接在對賈誼行為的心理分析之後——「嗚呼」二字承載了惋惜、批判、感慨的多重情感，是全文的情感高潮。 |
| **感嘆** | 「亦已難矣。」 | 在分析了周勃、灌嬰的政治地位後插入三個字——「也太難了」。既是對賈誼處境的同情，也是對其政治天真的一種無可奈何的嘆息。 |
| **諷刺（反語）** | 「安有立談之間，而遽為人『痛哭』哉？」 | 「痛哭」二字加引號——賈誼在《治安策》中使用了「可為痛哭者一，可為流涕者二，可為長太息者六」的措辭。蘇軾引用這個「痛哭」，但語氣中帶著諷刺——你一個剛到朝廷的年輕人，第一次見皇帝就哭天喊地，這不是政治家該有的姿態。 |
| **諷刺（反語）** | 「其後卒以自傷哭泣，至於夭絕。是亦不善處窮者也。」 | 「不善處窮」四個字是極為節制但極為辛辣的諷刺——賈誼不是被殺的，不是被貶死的，而是「不善於面對逆境」，自己把自己哭死了。 |
| **映襯（反襯）** | 孔子「雖大無道之國，皆欲勉強扶持」 vs. 賈誼「一不見用即自傷哭泣」 | 孔子對待最糟糕的國家都要勉強維持，賈誼對待「得君如漢文」卻因一次不順心就放棄——以聖人的堅忍反襯才子的脆弱。 |
| **借代** | 「絳侯」（周勃封號）、「灌嬰」（直呼其名） | 用封號稱周勃、用姓名稱灌嬰，形成一種莊重而具體的歷史感，同時暗示這兩位不是抽象的「權臣」，而是有血有肉、有歷史功勳的具體人物。 |
| **婉曲（迂迴表達）** | 「亦謹其所發哉！」 | 結尾五個字極為含蓄——「謹其所發」直譯是「謹慎自己的表現」，但結合全文，意思是：賈誼啊，你太不會表現自己了，你把自己的人生毀在了不該有的激烈上。蘇軾沒有直接說「賈誼你錯了」，而是以一種告誡的口吻輕輕帶過，留給讀者無限回味。 |

## 四、論證邏輯圖解

### 4.1 核心論證鏈

```
前提一：「非才之難，所以自用者實難」（真正的難題不是擁有才華，而是如何運用才華）
    ↓
前提二：「君子之所取者遠，則必有所待；所就者大，則必有所忍」
（遠大目標必須匹配相應的耐心）
    ↓
建立標準：以孔子、孟子為榜樣——他們如何對待不被重用的處境
    ├─ 孔子：「歷試於天下」→ 勤於尋求
    ├─ 孟子：「三宿而後出晝」→ 不忍棄君
    └─ 孟子：「方今天下，舍我其誰哉？」→ 至愛其身
    ↓
對照案例：賈誼的表現
    ├─ 被貶後：「為賦以弔屈原，紆鬱憤悶」→ 消極抱怨
    ├─ 面對挫折：「卒以自傷哭泣，至於夭絕」→ 自殘放棄
    └─ 根本問題：「不善處窮」→ 缺乏逆境承受力
    ↓
政治現實分析：周勃、灌嬰與文帝的關係「豈特父子骨肉手足哉？」
    ↓
反事實推演：如果賈誼「優遊浸漬而深交之……不過十年，可以得志」
    ↓
邏輯反駁：「夫謀之一不見用，則安知終不復用也？」
    ↓
最終判語：「賈生志大而量小，才有餘而識不足也」
    ↓
雙向收束：告誡君主（體諒狷介之臣）＋ 告誡士人（謹其所發）
```

### 4.2 三層論證結構

蘇軾的論證不是單線推進，而是從三個不同層面同時夾擊：

| 論證層次 | 論證內容 | 攻擊方向 |
|:---------|----------|:---------|
| **倫理層**（第1–2段） | 以孔孟為標準，證明賈誼「不善事君」——真正的君子應該像孔孟一樣勤於尋求、不忍棄君、愛惜自身。賈誼一遇挫折就放棄，在道德層面無法與聖賢比肩。 | 從「應該怎樣」的規範倫理入手，建立理想標準 |
| **政治層**（第3段） | 分析西漢初年功臣共治的政治現實——周勃、灌嬰是擁立文帝的人，情分如同父子；賈誼一個22歲的洛陽少年，想「一朝之間盡棄其舊而謀其新」，在政治現實上不可能。 | 從「現實如何」的客觀分析入手，說明賈誼的要求不切實際 |
| **邏輯層**（第4段） | 「謀之一不見用，則安知終不復用也？」——一次不被採納就認定永遠不被採納，這在邏輯上是站不住腳的。賈誼的悲劇源於邏輯謬誤，而非政治迫害。 | 從「推理是否有效」的邏輯分析入手，指出賈誼的思維錯誤 |

三層論證互相獨立又互相加強——你可以在一個層面反駁（比如說「孔孟不是政治家，不能拿他們類比」），但另外兩個層面仍然成立。這種多角度的論證設計使蘇軾的翻案幾乎無懈可擊。

### 4.3 反事實推演的運用

蘇軾在文中運用了一個極為精妙的論證技巧——**反事實推演**（counterfactual reasoning）：

> 「為賈生者，上得其君，下得其大臣，如絳、灌之屬，優遊浸漬而深交之，使天子不疑，大臣不忌，然後舉天下而唯吾之所欲為，不過十年，可以得志。」

這段話的邏輯結構是：
1. **前提設定**：假設賈誼換一種做法（「優遊浸漬而深交之」）
2. **中間推演**：結果會是什麼？天子不疑 + 大臣不忌 → 可以放手做事
3. **時間預測**：「不過十年，可以得志」
4. **隱含結論**：賈誼的悲劇不是命運使然，而是選擇使然

這種推演的巧妙之處在於：蘇軾不需要證明「如果A則B」的真實性，因為讀者自己會去想像這個反事實場景——而一旦讀者開始想像「如果賈誼活到43歲會怎樣」，他們就不知不覺接受了蘇軾的論證前提：賈誼本來有另一條路可走，是他自己選錯了。

### 4.4 對比系統

| 對比維度 | 正面（值得效法） | 反面（應當警惕） |
|:---------|:----------------|:----------------|
| **事君態度** | 孔子——「歷試於天下，苟非大無道之國，皆欲勉強扶持」 | 賈誼——被貶一次就「紆鬱憤悶，趯然有遠舉之志」 |
| **面對挫折** | 孟子——「三宿而後出晝，猶曰『王其庶幾召我』」 | 賈誼——「卒以自傷哭泣，至於夭絕」 |
| **自我認知** | 孟子——「方今天下，舍我其誰哉？」（自信但不自棄） | 賈誼——從極度自信（一年求公卿）到極度自卑（自傷夭絕） |
| **君臣關係** | 周勃、灌嬰——「親握天子璽而授之文帝」 | 賈誼——「洛陽之少年，欲使其一朝之間，盡棄其舊而謀其新」 |
| **用君之道** | 理想做法——「優遊浸漬而深交之，不過十年，可以得志」 | 賈誼的做法——「立談之間而遽為人痛哭」 |

## 五、名句深度解析

### 「非才之難，所以自用者實難。」

**白話翻譯：** 擁有才華並不難，真正難的是如何正確地運用自己的才華。

這是全文的開篇之句，也是全篇的理論基石。蘇軾在一開頭就區分了兩個層次：擁有才華（才）和運用才華（自用）。世人只羨慕賈誼的才華，卻忽略了他根本不知道如何運用這份才華。這個區分的深意在於：

- **擁有才華是被動的**——天賦、運氣，非人力可完全掌控
- **運用才華是主動的**——需要策略、耐心、格局、自我管理

蘇軾將焦點從「才」轉向「自用」，本質上是從「天才論」轉向「修養論」——這是他與歷代評論者最根本的不同。歷代評論者看到的是「一個天才被辜負了」，蘇軾看到的是「一個天才辜負了自己的天才」。

### 「夫君子之所取者遠，則必有所待；所就者大，則必有所忍。」

**白話翻譯：** 君子想要達到的目標長遠，就一定要有所等待；想要成就的事業宏大，就一定要有所忍耐。

這是全篇最著名的兩句話，也是蘇軾「忍耐哲學」的濃縮表達。這句話的邏輯結構是一個三段論：

```
大前提：任何遠大目標都需要時間來實現（取遠必待）
小前提：賈誼的目標是遠大的（王者之佐）
結  論：因此賈誼必須接受長時間的等待和忍耐
```

賈誼的問題在於：他只接受了「取遠」的目標，卻拒絕了「有待有忍」的代價——這在邏輯上是不自洽的。蘇軾沒有說賈誼不夠好，而是說他的思想不自洽——這是比道德批判更高明的論證。

### 「非漢文之不能用生，生之不能用漢文也。」

**白話翻譯：** 不是漢文帝不能夠重用賈誼，而是賈誼不懂得如何讓漢文帝重用自己。

這句是全文的翻案核心，也是中國古代散文中最精彩的翻案句之一。表面上看，這句話極為弔詭——臣子怎麼可能「用」君主？但蘇軾的深意是：

- **君臣關係是雙向的**，不是單向的君用臣——臣也需要引導、等待、說服君主，把君主爭取到自己的立場上來
- **「用」是一種政治能力**——賈誼有政見，但他沒有能力讓文帝接受這些政見，這說明他在政治操作層面有硬傷
- **這不是道德判斷，而是能力判斷**——蘇軾沒有說賈誼「不忠」或「不賢」，而是說他「不會用」——這是一個政治技術層面的批評

### 「賈生志大而量小，才有餘而識不足。」

**白話翻譯：** 賈誼志向遠大但格局狹小，才華橫溢但見識不足。

這是全文的終極判語，由兩個並列的判斷句構成：

| 前半句 | 後半句 | 關係 |
|:------|:-------|:----|
| **志大而量小**（志向宏大，但格局狹窄） | **才有餘而識不足**（才華橫溢，但缺乏見識） | 前半句是性格判斷（heart），後半句是能力判斷（mind） |

「志大而量小」是一個極具張力的對比——志向是向外的（想成就什麼），格局是向內的（能承受什麼）。這兩者之間的落差，正是賈誼悲劇的根源。

「才有餘而識不足」則揭示了蘇軾對「才」與「識」的區分——才華是提出好方案的能力，見識是知道什麼時候該說什麼、什麼時候該等什麼的能力。賈誼有前者，缺後者。

### 「亦謹其所發哉！」

**白話翻譯：** 也要謹慎自己的言行表現啊！

這是全文的最後五個字，極其含蓄、極其克制。直譯是「也要謹慎自己的表現啊」，但深層含義遠為豐富：

- **「所發」**——發出的言行、表現出來的情緒、做出的反應
- **「謹」**——謹慎、節制、有策略意識
- **「哉」**——感嘆語氣，帶有惋惜和告誡

蘇軾沒有直接說「賈誼你錯了」，而是以一種老朋友的口吻輕輕告誡：「以後要謹慎啊。」但這個「以後」是不可能的——賈誼已經死了。正是這種「對死人說話」的弔詭感，使結尾充滿了無盡的蒼涼和惋惜。

## 六、佳句賞析

### 佳句一：「惜乎！賈生，王者之佐，而不能自用其才也。」

> **原句：**「惜乎！賈生，王者之佐，而不能自用其才也。」

**白話翻譯：** 可惜啊！賈誼是能夠輔佐帝王的人才，卻不能好好地運用自己的才華。

**賞析：** 開篇的這一聲嘆息，奠定了全文的情感基調——蘇軾不是要對賈誼進行冷冰冰的批判，而是帶著深深的惋惜來重新審視他的悲劇。首句將賈誼定位為「王者之佐」（帝王之佐），說明蘇軾完全肯定他的才華與潛力；但緊接著一個轉折——「而不能自用其才」——直接點出問題的核心不在外部環境，而在賈誼自身。這個「能」字是全句的關鍵：賈誼不是沒有才華，而是「不會用」。「惜乎」二字中的情感尤其值得玩味——既有對一個夭折天才的同情，也有對他不爭氣的無奈。蘇軾用這一句話，完成了從傳統同情敘事到自身反省敘事的轉換。

### 佳句二：「安有立談之間，而遽為人『痛哭』哉？」

> **原句：**「安有立談之間，而遽為人『痛哭』哉？」

**白話翻譯：** 哪裡有在交談片刻之間，就突然為人「痛哭」起來的道理呢？

**賞析：** 這是全篇最辛辣的反問句。蘇軾以一個反問，徹底瓦解了賈誼在政治上所謂的「正當性」。「立談之間」四個字寫出了賈誼的急切——他與漢文帝的關係才剛剛開始建立，連基本的信任基礎都沒有；「遽為人痛哭」則直指賈誼在《治安策》中「可為痛哭者一，可為流涕者二，可為長太息者六」的激烈措辭。蘇軾故意將「痛哭」二字加上引號，既是點明典故出處，也是居高臨下地評價——你一個初來乍到的年輕人，第一次見皇帝就哭天喊地，這不是政治家該有的姿態。反問句的妙處在於不直接下結論，而是讓讀者自己得出「賈誼的做法確實不妥」的判斷。這句反問背後，是蘇軾對政治分寸感的深刻理解：再好的建議，也要等待合適的時機才能提出。

## 七、結語

〈賈誼論〉的卓越之處，在於它同時做到了三件事：

**第一，翻案而不失公允。** 蘇軾雖然翻轉了千年定論，但他沒有妖魔化賈誼。開頭的「惜乎」、結尾的「愚深悲生之志」，始終保持著對一個天才早逝的同情。翻案不是為了標新立異，而是為了求得更深的理解。

**第二，史論即是勸世文。** 蘇軾不是在跟兩千年前的賈誼對話，他是在跟北宋那些心懷抱負卻又焦慮不安的年輕士大夫對話。「不要動不動就覺得朝廷辜負了你——先問問你自己有沒有耐心等到那一天。」這是這篇文章真正的當代關懷。

**第三，用一生為文章作注。** 蘇軾寫〈賈誼論〉時不過二十出頭，意氣風發。此後他的一生——三次被貶（黃州、惠州、儋州），一次比一次遠——恰恰是這篇文章最好的實踐證明。他從未像賈誼那樣「自傷哭泣」，而是每到一地都認真生活、認真寫作、認真做事。他用自己的生命證明了「有待有忍」不是空談，而是可以做到的。

從這個意義上說，〈賈誼論〉不僅是一篇優秀的史論文章，更是一劑給所有「志大者」的清醒藥——當你覺得全世界都對不起你的時候，先問問自己：你對得起你的才華嗎？你對得起你的等待嗎？
