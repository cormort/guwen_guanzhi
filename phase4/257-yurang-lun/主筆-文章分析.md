# 〈豫讓論〉文章分析

> **主筆分析 · 古文觀止卷十二·明文·方孝孺**

---

## 1. 文章結構分析

〈豫讓論〉是明初理學家方孝孺（1357–1402）翻案史論的代表作，收於《遜志齋集》。全文約四百字，篇幅短小卻涵蓋了提出論點、反駁、深化、結論的全過程。方孝孺挑戰了千年來對「士為知己者死」的道德讚美——他不否定豫讓的忠，而是將「忠」的標準從「死後報仇」重新定義為「生前諫諍」。核心命題是：真正的國士不在於為知己而死，而在於在知己未亡之前竭盡智謀、忠告善道。

本文之精要在於「翻案而不翻功」——承認豫讓的忠與死，但否定其「處死之道」。方孝孺以嚴密的邏輯層層剝開豫讓行為的內在矛盾：他對智伯的報仇表面上是「忠」，實則是「沽烈士之名」。以下從「起承轉合」「修辭手法」「論證邏輯」及「名句解析」四個角度深入分析。

---

## 二、起承轉合分析

| 段落 | 起承轉合 | 對應原文 | 核心功能 | 論證策略 |
|:----:|:--------:|----------|----------|----------|
| **第1段** | **起** | 「士君子立身事主，既名知己，則當竭盡智謀，忠告善道，銷患於未形，保治於未然，俾身全而主安。生為名臣，死為上鬼，垂光百世，照耀簡策，斯為美也。苟遇知己，不能扶其未危，而反為之報仇於既亡之後，其於忠也不亦遠乎？讓之謂也。」 | 開篇提出「真正的忠臣」標準——先不談豫讓，先建立一個超越世俗的價值尺度。真正的忠臣應該是「銷患於未形，保治於未然」的事前預防者，而非事後報仇者，為後文的批判奠定理論基礎 | **定義入題、以退為進**：先確立理想標準，再以「讓之謂也」四字將豫讓放入這個框架——既完成了從一般到具體的過渡，又暗示豫讓恰恰是反例 |
| **第2段** | **承** | 「當其弗事智伯之時，曾無一言以動其心；及智伯既亡，乃為之報仇，以泄其憤。讓之於智伯，未為知己之深也。讓有以自誓，不過為智伯報仇而已。其所以報仇者，非欲動智伯也，實欲成己之名。……讓於此時，曾無一語開悟主心，而顧視其亡，以沽烈士之名。」 | 承接開篇標準，開始剖析豫讓的具體行為：①指出豫讓在智伯生前從未進諫 → ②指控他「顧視其亡」——眼看著智伯滅亡不作挽救 → ③將「報仇」重新定性為「成己之名」而非真忠 → ④得出「雖有致死之節，未足為忠也」的過渡性結論 | **翻案式重釋**：不否定豫讓刺殺行為的真實性，而是重新解釋其動機——從「為知己死」翻轉為「為己名死」，用同一套史料得出截然相反的道德判斷 |
| **第3段** | **轉** | 「蓋之死生、去就，惟其所欲。伸於知己，而屈於不知己。使讓能於未敗之前，先事而諫，則智伯雖頑，亦未必不為之動。諫而不聽，則去之可也；諫而不聽，讓亦可以死矣。謂之死於知己，可也。何至漆身吞炭，以區區之軀，為仇讎之所笑哉？」 | 筆鋒一轉，從批判豫讓轉入建構「最好方案」：①提出國士的選擇自由——可死、可去、可留 → ②假設性論證——如果豫讓在智伯未敗之前先事而諫，即使不成功也是一種真正的「死於知己」→ ③反問——何必非要走「漆身吞炭」的極端道路，讓仇人嘲笑？ | **假設情境論證**：用「如使……則……」的假設語氣構建一個理想情境，讓讀者自行比較豫讓的實際行為與理想路徑之間的差距——差距有多大，批判就有多深 |
| **第4段** | **合** | 「或曰：『讓之於智伯，報仇之義也。子何抑之深也？』曰：讓之死固忠矣，惜乎處死之道有未忠者存焉。古之君子，愛其君之身，甚於愛其己之身；愛其君之國，甚於愛其己之國。……若以忠臣義士論之，則讓之死，不足為忠也。不足為義也。何也？讓之於智伯，未嘗有君臣之分也。……是故，讓之死，忠乎？非忠也。義乎？非義也。直匹夫之諒，婦人之仁而已。」 | 收束全文，以先肯定後否定的方式完成最終判決：①讓虛擬的反對者先發問——「子何抑之深也？」→ ②先承認「讓之死固忠矣」以示公正 → ③再用「惜乎」轉折，指出其「處死之道」有未忠之處 → ④層層推進，最終否定其忠與義的性質，定調為「匹夫之諒，婦人之仁」 | **先揚後抑、反駁論證**：先舉出反對意見再一一駁倒，讓辯論走向不可逆轉的終點——從一開始的「讓之死固忠矣」到最終的「忠乎？非忠也。義乎？非義也」，讀者已經被方孝孺的邏輯完全說服 |

### 起承轉合結構圖解

```
起（第1段）：定義「真忠」標準 ──→ 承（第2段）：翻案重釋豫讓動機
                                        │
                                        ↓
合（第4段）：最終判決・先揚後抑 ←── 轉（第3段）：建構「最好方案」
```

### 結構特點

1. **先立標準，後論人物**：不同於一般史論直接評點古人，方孝孺先建立「真忠」的定義——「銷患於未形，保治於未然」——讓讀者帶著這個標準去審視豫讓。這是「論從定義出」的高級寫法。

2. **由破到立，先抑後揚**：前兩段是「破」——破壞豫讓的忠義形象；後兩段是「立」——建立真忠的標準與可行方案。破立結合，論證完整。

3. **反駁收束，出奇制勝**：第四段以虛擬反問「或曰」開場，讓反對者先發言，再一一駁斥。這種手法既展示了方孝孺對反對意見的充分考慮，又在反駁中將論證推向更深層次——從「忠與不忠」的討論下沉為「匹夫之諒 vs 君臣大義」的本質之辯。

4. **篇幅短小，層層深入**：全文約四百字，卻涵蓋了定義標準、提出質疑、翻案重釋、假設論證、反駁反對意見、最終判決六個層次——無一字浪費。

## 三、修辭手法分析

| 修辭手法 | 原文例證 | 作用與效果 |
|:---------|----------|:-----------|
| **對比** | 「銷患於未形，保治於未然」vs.「不能扶其未危，而反為之報仇於既亡之後」 | 開篇即建立「事前預防」與「事後報仇」的對立——前者是真忠，後者「於忠也不亦遠乎」。這是全文的核心對比框架 |
| **對比** | 「曾無一語開悟主心」vs.「顧視其亡，以沽烈士之名」 | 將豫讓的行為對比為兩個階段——生前「沉默」與死後「行動」——沉默愈徹底，死後行動的動機就愈可疑 |
| **對比** | 「讓之死固忠矣」vs.「處死之道有未忠者存焉」 | 全文最重要的轉折對比——先肯定（讓之死固忠），再否定（處死之道未忠）。表面矛盾，實則精妙：承認其死的「忠」之表象，否定其「忠」的實質 |
| **對比** | 「愛其君之身，甚於愛其己之身；愛其君之國，甚於愛其己之國」vs.「讓雖有以自誓，不過為智伯報仇而已」 | 真忠者的標準（愛君勝於愛己）與豫讓的行為（為己成名）之間的鴻溝——對比愈強烈，批判愈有力 |
| **反問（激問）** | 「苟遇知己，不能扶其未危，而反為之報仇於既亡之後，其於忠也不亦遠乎？」 | 開篇即以反問收束定義——「不亦遠乎」用「不亦……乎」的句式，語氣強烈但不過於尖刻，引導讀者自問自答 |
| **反問（激問）** | 「何至漆身吞炭，以區區之軀，為仇讎之所笑哉？」 | 對豫讓極端行為的終極反問——「何至」二字充滿了遺憾與不以為然，而「為仇讎之所笑」則是從旁觀者的視角消解了豫讓行為的悲壯感 |
| **反問（激問）** | 「讓何以異於是？」 | 在比較「古之君子」與豫讓之後，用這個反問直接逼問——「豫讓有什麼不同呢？」答案不言而喻：本質完全不同 |
| **反問（激問）** | 「忠乎？非忠也。義乎？非義也。」 | 結尾一連串自問自答式的反問，節奏短促、斬釘截鐵——全文的最終判決，沒有絲毫含糊 |
| **設問** | 「或曰：『讓之於智伯，報仇之義也。子何抑之深也？』」 | 先代反對者提問，再作答——這是中國論說文中常見的「設難」手法，讓讀者心中可能的疑問先被提出來，再被一一駁倒 |
| **層遞** | 「不諫」→「顧視其亡」→「沽烈士之名」 | 三層遞進：從消極的不作為（不諫）→ 到中性的旁觀（顧視其亡）→ 再到積極的逐名（沽烈士之名）——豫讓的道德形象被層層剝落 |
| **層遞** | 「諫而不聽，則去之可也；諫而不聽，讓亦可以死矣」 | 三種選擇由輕到重：先勸諫→若不聽則離開→若不聽甚至可死諫——層層遞進，展示了國士真正可以選擇的廣闊空間，反襯出豫讓的選擇是多麼狹隘 |
| **層遞** | 「忠乎？非忠也。義乎？非義也。直匹夫之諒，婦人之仁而已。」 | 三個否定層層加碼：從「非忠」到「非義」，再從「非忠義」定性為「匹夫之諒、婦人之仁」——用最低的道德層次來定位豫讓的行為 |
| **引用（用典）** | 「古之君子，愛其君之身，甚於愛其己之身；愛其君之國，甚於愛其己之國」 | 引用儒家經典中的理想君子形象，作為衡量豫讓的道德標尺——這不是方孝孺個人的標準，而是儒家道統的標準，增加了權威性 |
| **比喻（設喻）** | 「匹夫之諒，婦人之仁」 | 結尾將豫讓的忠義定性為「匹夫之諒」（普通人的守信）和「婦人之仁」（瑣碎的仁慈）——這兩個比喻將豫讓的行為從「大忠大義」降格為「小信小善」，極具殺傷力 |
| **諷刺** | 「顧視其亡，以沽烈士之名」 | 「顧視」二字（期待地注視著）與「沽」（買賣、獵取）搭配——豫讓的報仇不是出於對智伯的忠誠，而是一種交易：用智伯的死亡換取自己的名聲。諷刺之意溢於言表 |
| **諷刺** | 「為仇讎之所笑哉」 | 豫讓費盡心機漆身吞炭，最終的結果卻是被仇人嘲笑——方孝孺用一個「笑」字，將豫讓行為的荒誕性徹底暴露 |
| **先揚後抑** | 「讓之死固忠矣，惜乎處死之道有未忠者存焉」 | 全文最精妙的修辭手法。先肯定（讓之死固忠矣）——順應傳統觀點，不讓讀者感到被冒犯；再用「惜乎」轉折——提出更高標準，讓讀者在不知不覺中接受新的判斷 |
| **排比** | 「銷患於未形，保治於未然」 | 兩個動賓結構排比，節奏整齊，增強了定義的權威感 |
| **排比** | 「愛其君之身，甚於愛其己之身；愛其君之國，甚於愛其己之國」 | 兩個「愛其君之……甚於愛其己之……」的並列排比，層層遞進，強調真忠者應有的境界 |
| **映襯** | 智伯以國士待豫讓（厚遇）vs. 豫讓生前不諫（薄報） | 厚遇與薄報之間的反差映襯——「以國士遇我者，我以國士報之」只是口號，豫讓實際上並沒有做到國士應盡的責任 |
| **映襯** | 豫讓的「漆身吞炭」的極端痛苦 vs. 「為仇讎之所笑」的可笑結局 | 過程的悲壯與結局的荒誕形成強烈映襯——讀者讀完不禁捫心自問：這一切值得嗎？這就是方孝孺想要的效果 |
| **映襯** | 方孝孺本人對建文帝的諫諍（真忠典範）vs. 豫讓對智伯的沉默（不忠） | 雖然文中沒有直接出現方孝孺的自況，但了解方孝孺生平的讀者都能感受到——他用自己的命運，完成了對豫讓最無聲、最有力的批判 |

## 四、論證邏輯圖解

### 4.1 核心論證鏈

```
前提：士君子立身事主，當「銷患於未形，保治於未然」
    ↓
定義：真正的「忠」= 事前預防，而非事後報仇
    ↓
     ┌─ 歷史事實：豫讓在智伯生前「曾無一語開悟主心」
     │   （完全的沉默）
     │
     ├─ 方孝孺的指控：「顧視其亡，以沽烈士之名」
     │    ├─ 消極層面：沒有盡到國士的責任（不進諫）
     │    └─ 積極層面：甚至期待智伯滅亡（以成就自己的名聲）
     │
     ├─ 本質定性：豫讓的報仇「非欲動智伯也，實欲成己之名」
     │    └─ 從忠義降格為自私
     │
     ├─ 對比論證：古之君子愛君勝於愛己 → 豫讓不及
     │
     ├─ 假設論證：如果豫讓……→ 有三種更好的選擇
     │    ├─ 方案一：先事而諫（最佳）—可能改變智伯
     │    ├─ 方案二：諫而不聽則去（次佳）—保有尊嚴
     │    └─ 方案三：諫而不聽則死（死諫）—真正的死於知己
     │
     └─ 反駁反對意見：「或曰」→ 以更深刻的標準否定豫讓
              ↓
最終判決：
     ├─ 「讓之死固忠矣」→ 表象上的肯定
     ├─ 「惜乎處死之道有未忠者存焉」→ 本質上的否定
     ├─ 「不足為忠」→ 「不足為義」→ 完全否定
     └─ 「直匹夫之諒，婦人之仁而已」→ 最低定性
```

### 4.2 對比系統

| 對比維度 | 正面（真忠） | 反面（豫讓） |
|:---------|:------------|:--------------|
| **時間維度** | 事前預防——「銷患於未形，保治於未然」 | 事後報仇——「不能扶其未危，而反為之報仇於既亡之後」 |
| **行為表現** | 竭盡智謀、忠告善道——生前不斷勸諫 | 沉默旁觀——「曾無一語開悟主心」 |
| **內心動機** | 為君、為國——「愛其君之身，甚於愛其己之身」 | 為己、為名——「顧視其亡，以沽烈士之名」 |
| **對君主態度** | 救君於未危——在危險來臨前保護主君 | 期待主君滅亡——「非欲動智伯也，實欲成己之名」 |
| **處死之道** | 死得其所——可以死於勸諫，可以死於守護 | 死得不明——「漆身吞炭，為仇讎之所笑」 |
| **道德層次** | 君臣大義——基於責任與道義的忠誠 | 匹夫之諒、婦人之仁——個人層面的小信小義 |
| **君臣關係** | 有君臣之分——「君臣之義，以死繼之」 | 無君臣之分——「未嘗有君臣之分」的私人知遇 |
| **最終評價** | 真忠、真義——名垂青史 | 「忠乎？非忠也。義乎？非義也」——被徹底否定 |

### 4.3 層層翻剝的論證技巧

方孝孺此文最精彩之處在於**層層翻剝**——像剝洋蔥一樣，每翻一層，豫讓行為的真相就更加赤裸：

1. **第一層（現象）**：豫讓為智伯報仇而死 → 世人讚為忠義 → 方孝孺說：等等

2. **第二層（標準）**：真正的忠不是死後報仇，而是生前諫諍 → 「銷患於未形，保治於未然」

3. **第三層（事實）**：豫讓生前一句話都沒說 → 「曾無一語開悟主心」

4. **第四層（動機）**：他不是為了智伯，而是為了自己的名聲 → 「顧視其亡，以沽烈士之名」

5. **第五層（方案）**：真正的國士有三種更好的選擇 → 勸諫、離開、死諫——每一種都比豫讓的選擇更高尚

6. **第六層（反駁）**：即使是反對者的觀點也站不住腳 → 答「或曰」之問

7. **第七層（定案）**：從「讓之死固忠矣」到「非忠也」「非義也」「匹夫之諒，婦人之仁」——層層降格，最終徹底否定

## 4. 白話改編

以下為〈豫讓論〉全文分段的現代白話翻譯，便於對照原文理解文意。

### 原文：第一段（定義「真忠」標準）

> 士君子立身事主，既名知己，則當竭盡智謀，忠告善道，銷患於未形，保治於未然，俾身全而主安。生為名臣，死為上鬼，垂光百世，照耀簡策，斯為美也。苟遇知己，不能扶其未危，而反為之報仇於既亡之後，其於忠也不亦遠乎？

**白話：** 士君子立身處世、事奉主君，既然被稱為知己，就應當竭盡智慧謀略，忠實勸告、善加引導，在禍患尚未形成之前就消除它，在動亂尚未發生之前就安定它，使自己安全、主君平安。活著的時候是名臣，死後是高尚的鬼魂，光輝流傳百世，照耀史冊，這才是美好的。如果遇到了知己，卻不能在主君還沒有陷入危險時扶持他，反而在他已經死亡之後去報仇，這離「忠」也太遠了吧？

### 原文：第二段（翻案——豫讓生前沉默）

> 讓之謂也。當其弗事智伯之時，曾無一言以動其心；及智伯既亡，乃為之報仇，以泄其憤。讓之於智伯，未為知己之深也。讓有以自誓，不過為智伯報仇而已。其所以報仇者，非欲動智伯也，實欲成己之名。……讓於此時，曾無一語開悟主心，而顧視其亡，以沽烈士之名。雖有致死之節，未足為忠也。

**白話：** 說的就是豫讓這樣的人啊。當他在事奉智伯的時候，沒有一句話去打動智伯的心；等到智伯死了，才為他報仇，來發泄自己的憤恨。豫讓對於智伯，稱不上是知己深厚。豫讓雖然自我發誓，不過是為智伯報仇罷了。他之所以報仇，不是為了感動智伯，實際上是為了成就自己的名聲。……豫讓在那個時候，沒有一句話去開導警醒主君的心，反而眼睜睜看著他滅亡，藉此獵取烈士的名聲。雖然有犧牲生命的節操，卻稱不上是忠。

### 原文：第三段（建構「最好方案」）

> 蓋之死生、去就，惟其所欲。伸於知己，而屈於不知己。使讓能於未敗之前，先事而諫，則智伯雖頑，亦未必不為之動。諫而不聽，則去之可也；諫而不聽，讓亦可以死矣。謂之死於知己，可也。何至漆身吞炭，以區區之軀，為仇讎之所笑哉？

**白話：** 一個人的死生、去留，全憑他自己的選擇。為知己者可以犧牲，為不知己者可以屈身活著。假如豫讓能夠在智伯還沒有失敗之前，預先進行勸諫，那麼智伯即使再頑固，也不一定不會被觸動。勸諫了不聽，那麼離開就可以了；勸諫了不聽，豫讓也可以為此而死——這才能稱得上是為知己而死。何必要用漆塗身、吞炭變聲，用這小小的身軀，去被仇人所嘲笑呢？

### 原文：第四段（反駁與最終判決）

> 或曰：「讓之於智伯，報仇之義也。子何抑之深也？」曰：讓之死固忠矣，惜乎處死之道有未忠者存焉。……是故，讓之死，忠乎？非忠也。義乎？非義也。直匹夫之諒，婦人之仁而已。

**白話：** 有人說：「豫讓對於智伯，那是報仇的正義之舉，你為什麼貶抑得這麼厲害呢？」回答是：豫讓的死固然是忠的，可惜他面對死亡的方式，還有不忠的地方存在。……所以，豫讓的死，忠嗎？不忠。義嗎？不義。只不過是普通人的守信、婦道人家的仁愛罷了。

## 五、佳句賞析

### 佳句一

> **「苟遇知己，不能扶其未危，而反為之報仇於既亡之後，其於忠也不亦遠乎？」**

**現代漢語：** 如果遇到了知己的人，卻不能在他還沒有危險的時候扶持他，反而在他已經死亡之後去報仇，這離「忠」也太遠了吧？

**賞析：** 這是全文的第一個核心論斷，也是方孝孺翻案史論的邏輯起點。其論證層次極其精密：

- **設定情境**：「苟遇知己」——先承認豫讓與智伯之間的知遇關係，表示方孝孺不否定這個前提
- **否定行為**：「不能扶其未危」——否定豫讓在智伯生前的作為（或者說「不作為」）
- **指出偏差**：「反為之報仇於既亡之後」——「反」字是轉折關鍵，將「事前不救」與「事後報仇」並置，凸顯其矛盾
- **終極反問**：「其於忠也不亦遠乎？」——「不亦……乎」是文言中經典的反問句式，語氣溫和但立場堅定，相當於「不是太……了嗎？」

這段話的修辭力量在於：方孝孺沒有直接說「豫讓不忠」，而是用一個反問讓讀者自己去得出結論。他先確立了「真忠」的標準——在未危之前扶助——然後將豫讓放入這個標準中，讓事實自己說話。

### 佳句二

> **「讓於此時，曾無一語開悟主心，而顧視其亡，以沽烈士之名。」**

**現代漢語：** 豫讓在那個時候，沒有一句話去開導警醒主君的心，反而眼睜睜看著他滅亡，藉以獵取烈士的名聲。

**賞析：** 這是全文最著名、最具殺傷力的一句，也是方孝孺翻案的核心判詞。其修辭藝術體現在以下幾個層面：

1. **「曾無一語」的絕對否定**：「曾」字表示「竟然」「連……都」——不是「很少有話」，而是「連一句話都沒有」。這種絕對化的表述讓讀者沒有迴旋的餘地。
2. **「開悟主心」的關鍵定位**：方孝孺巧妙地將豫讓的責任定位為「開悟」——不是治理國家、不是衝鋒陷陣，而是最基礎的「讓主君覺悟」。連這個最基本的責任都沒有盡到，豫讓的國士身分從根本上被否定。
3. **「顧視其亡」的致命細節**：「顧視」二字是全文最精妙的用詞——它不是「看著」（被動的觀察），而是「注視著」「期待地看著」（帶有主動的意味）。方孝孺用這兩個字暗示：豫讓不是被動地旁觀智伯走向滅亡，而是主動地期待這一天——因為只有智伯死了，他才能以「報仇的刺客」身分登上歷史舞台。
4. **「沽」字的交易意象**：「沽」是買賣、獵取的意思——將烈士之名比喻為可以交易的商品。豫讓的報仇不是出於真誠的忠義，而是一樁交易：用智伯的滅亡換取自己的千古名聲。

### 佳句三

> **「讓之死固忠矣，惜乎處死之道有未忠者存焉。」**

**現代漢語：** 豫讓的死固然是忠的，可惜他面對死亡的方式，還有不忠的地方存在。

**賞析：** 這是全文的總結句，也是方孝孺最為精妙的修辭。與宋濂在〈送天台陳庭學序〉中的翻轉結構如出一轍：

- **先揚**：「讓之死固忠矣」——先肯定傳統觀點，讓讀者放鬆警惕。方孝孺非常聰明：他不與主流價值正面碰撞，而是先承認豫讓的忠，以示客觀公正。
- **後抑**：「惜乎處死之道有未忠者存焉」——「惜乎」二字是全文最重要的轉折詞，帶有「可惜」「遺憾」的惋惜語氣。方孝孺不是粗暴地否定，而是用一種「更高標準下的惋惜」來表達批評——就像一個老師對學生的成績說：「你考了85分，很不錯了，可惜你本來可以考100分。」
- **「處死之道」的四字創新**：方孝孺將「死」分為兩個層面——「死」的本身（事實層面）和「處死之道」（方法層面）。他承認豫讓的「死」是忠的，但指出他的「處死之道」還不夠忠。這種區分極其精巧：既不得罪傳統觀點，又開闢了全新的批評空間。

### 佳句四

> **「何至漆身吞炭，以區區之軀，為仇讎之所笑哉？」**

**現代漢語：** 何必要用漆塗身、吞炭變聲，用這小小的身軀，去被仇人所嘲笑呢？

**賞析：** 這是全文最具情緒感染力的一句話。方孝孺在此前三段都在進行嚴肅的邏輯論證，至此突然用一個反問直擊人心：

- **「漆身吞炭」的濃縮意象**：四個字概括了豫讓最極端的自殘行為——為了復仇，他不惜毀容、變聲，連妻子都認不出他。這些行為在傳統敘述中是「悲壯」的，但在方孝孺的反問中，它們變成了「荒謬」的。
- **「區區之軀」的輕蔑**：用「區區」來形容豫讓的犧牲——不是「偉大的身軀」，而是「區區的小身軀」。這種輕描淡寫的用詞消解了犧牲的神聖感。
- **「為仇讎之所笑」的致命一擊**：方孝孺在結尾引入了一個旁觀者的視角——仇人在嘲笑。「笑」字是全文最殘酷的用詞：豫讓付出了一切，得到的卻是敵人的嘲笑。這種諷刺將豫讓的行為從「悲壯」徹底翻轉為「可笑」。

### 佳句五

> **「是故，讓之死，忠乎？非忠也。義乎？非義也。直匹夫之諒，婦人之仁而已。」**

**現代漢語：** 所以，豫讓的死，忠嗎？不忠。義嗎？不義。只不過是普通人的守信、婦道人家的仁愛罷了。

**賞析：** 這是全文的最終判決，字字千鈞：

- **短促的節奏**：「忠乎？非忠也。義乎？非義也。」——兩個三字反問與兩個四字否定交替，節奏短促、斬釘截鐵，猶如法官落槌。
- **「匹夫之諒」與「婦人之仁」的典籍淵源**：這兩個典故都來自儒家經典。「匹夫之諒」出自《論語·憲問》——「豈若匹夫匹婦之為諒也，自經於溝瀆而莫之知也」，孔子批評管仲沒有像普通人那樣為公子糾殉死，而是選擇活下來成就更大的事業。方孝孺借用孔子對管仲的寬容標準來批評豫讓的狹隘——孔子連管仲都可以寬恕，你豫讓又有什麼值得讚美的？「婦人之仁」出自《史記·項羽本紀》——項羽「婦人之仁」的缺點恰恰是婦人之仁。方孝孺將這兩個負面典故疊加在豫讓身上，將他的「忠義」徹底降格為最低層次的道德行為。

## 七、結語

〈豫讓論〉的成就建立在**論證邏輯與修辭藝術的高度統一**之上。方孝孺不是為了炫技而使用修辭手法，而是每一種手法都精準服務於論證目的：

- **開篇定義**（「銷患於未形，保治於未然」）建立全文的話語框架
- **翻案重釋**（「曾無一語開悟主心，而顧視其亡，以沽烈士之名」）顛覆傳統認知
- **假設論證**（「使讓能於未敗之前，先事而諫……」）提供理想路徑
- **反駁收束**（「或曰……讓之死固忠矣，惜乎……」）以先揚後抑完成最終判決
- **結尾定案**（「忠乎？非忠也。義乎？非義也。直匹夫之諒，婦人之仁而已」）層層降格，一字定讞

與蘇軾的〈留侯論〉相比，方孝孺此文呈現出截然不同的風格：蘇軾是「從容引導」——他用對話式的筆觸讓讀者不知不覺被說服；方孝孺則是「剛直凌厲」——他的邏輯像刀鋒一樣逐層剝開對手的防禦，毫不留情。這種風格差異不僅源於兩人的個性不同（蘇軾灑脫、方孝孺剛直），更源於兩人的學術根基不同——蘇軾是雜家（儒釋道兼修），方孝孺是純粹的理學家（嚴格的儒家道義）。

〈豫讓論〉的最深層價值，在於方孝孺用這篇四百字的短文提出了一個超越時代的命題：**真正的忠誠不是犧牲的姿態，而是責任的擔當；不是事後的壯烈，而是事前的努力。** 一個人真正的價值，不在於他願意為誰去死，而在於他願意為誰而活——為了他人的利益去思考、去勸諫、去承擔風險。豫讓為智伯而死，但他在智伯生前連一句規勸的話都不曾說過——這種「死後才忠」的忠誠，不過是對生前失職的補償，其本質是一種自我安慰。

更令人震撼的是，方孝孺本人後來用生命踐行了他對豫讓的批判。當燕王朱棣攻破南京、逼迫他起草即位詔書時，他擲筆大罵，最終被處以滅十族的極刑。然而他在建文帝生前已經盡了全部的努力——反覆勸諫、參與決策、直到最後一刻。他的死不是「顧視其亡」的沉默，而是「生前已盡忠」之後的從容殉道。**他的命運恰好證明了他的理論：真正的殉道者，必然在生前已經盡過一切努力。**

〈豫讓論〉不僅是方孝孺對千年歷史的一聲質問，也是他對自己內心的一次宣誓——而他用生命兌現了這份誓言。
