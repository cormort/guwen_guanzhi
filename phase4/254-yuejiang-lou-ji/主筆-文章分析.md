# 〈閱江樓記〉— 文章分析

> 宋濂（1310–1381）
>
> 本檔案由 @Hermes Agent 主筆

---

## 1. 文章結構分析

〈閱江樓記〉全文約五百餘字，是宋濂奉命為朱元璋計劃在南京獅子山修建的「閱江樓」所作的記。但閱江樓最終從未建成——這是一篇為「不存在」的建築所寫的「樓記」。宋濂的處理方式極為穩妥：他避開對樓閣本身的描寫（因為無樓可寫），轉而以長江天險與南京形勝為切入點，將「樓記」寫成了「江山記」，在頌聖的外殼下完成了一次委婉的規諫。

全文結構呈現為標準的「頌聖→議論→規諫」遞進模式：

| 段落 | 功能 | 核心內容 |
|:----:|------|----------|
| 第一段 | 開篇立論 | 讚頌朱元璋為「雄才大略之主，應天順人」，為閱江樓的建造賦予合法性 |
| 第二段 | 地理形勢 | 描寫長江天險與南京形勝——「長江天塹，所以限南北也」 |
| 第三段 | 歷史對比 | 回顧六朝以來定都南京的王朝，雖有長江天險仍不免滅亡 |
| 第四段 | 委婉規諫 | 點出核心命題——天險不如仁政，「在德不在險」 |
| 第五段 | 結語頌聖 | 回到對朱元璋太平盛世的讚美，與開篇呼應，完成閉環 |

### 起承轉合

| 結構 | 段落 | 內容 |
|:----:|------|------|
| **起** | 第一段：為樓正名 | 開篇以「臣聞昔者，帝王之興，必有雄才大略之主，以應天順人」破題，將朱元璋定位為開國聖君。用「應天順人」四字確立了朱元璋政權的合法性——既是天意，也是民心。此段為全篇的歌頌基調定調，讓朱元璋放下戒心。 |
| **承** | 第二段：寫江山形勝 | 描寫長江的壯闊與南京的險要地勢。「長江天塹，所以限南北也」——這句既是在讚美天險，也在暗中埋下伏筆：長江只能「限南北」，不能「保江山」。所謂「承」，是先順著朱元璋的期待，把天險的好處說足說夠。 |
| **轉** | 第三至四段：歷史教訓 → 核心命題 | 這是全文最重要的轉折。宋濂筆鋒一轉，從「天險之利」轉入「天險之限」——六朝皆據長江天險，卻無一例外全部滅亡。由此引出全文核心命題：「帝王之興，在德不在險」。這個轉折極其溫柔：宋濂沒有說一句批評朱元璋的話，只是分析了歷史事實——但讀者（朱元璋）自然會明白，這句話是對他說的。 |
| **合** | 第五段：回歸頌聖 | 宋濂在完成規諫之後，立即回歸頌聖：今天下已定、四海昇平，朱元璋以仁德治天下，已經做到了六朝君主做不到的事。這個「合」既是結構上的收束，也是策略上的安全網——讓整篇文章以歌頌結尾，不會讓皇帝感到被冒犯。 |

### 分段解析

**第一段：開篇立論——「雄才大略之主」**

> 「臣聞昔者，帝王之興，必有雄才大略之主，以應天順人。」

開篇以「臣聞」起筆，第一時間標明此文是臣子對皇帝進言的身份定位。宋濂將朱元璋比作歷史上開創王朝的「雄才大略之主」，用「應天順人」為其權力提供儒家政治理論的正當性——天意與民心的雙重授權。這一句的功能不僅是歌頌，更是為後文設置了一個前提：既然您是如此英明的君主，自然懂得天險不足恃、仁政方為本的道理。

**第二段：江山形勝——「長江天塹，所以限南北也」**

此段是全文最經典的寫景段落。宋濂以長江為核心，描寫其作為天險的地理特徵。但他沒有停留在單純的寫景——「所以限南北也」五個字，將長江的功能精準鎖定在「分隔南北」之上。這個限定詞極其講究：它可以分隔南北，但不能確保統一；它是地理上的屏障，但不是政治上的保障。這為後文的歷史論證埋下了伏筆。

**第三段：歷史回顧——六朝往事**

> 「昔者六朝之君……雖有長江之險，終不免於滅亡。」

這一段是全文的轉折樞紐。宋濂以極簡的筆墨回顧了定都南京的六個王朝——三國吳、東晉、宋、齊、梁、陳——這些王朝無一例外擁有長江天險，但無一例外走向滅亡。理由不是長江不夠險，而是這些君主「不能以仁德守之」。

這段歷史回顧的高明之處在於：宋濂說的每一句話都是歷史事實，朱元璋無法反駁；但這些歷史事實組合起來，構成了一個讓朱元璋無法迴避的問題——「您的王朝會不會也重蹈六朝的覆轍？」

**第四段：核心命題——「在德不在險」**

這是全文的結論段落。宋濂從歷史回顧中提煉出核心命題：天險不如仁政，江山之固在德不在險。他沒有使用直接說教的語氣，而是以「臣聞古人有言」的方式，將這個命題包裝為古人的智慧，而非自己的創見。這樣既傳達了規諫之意，又不會讓皇帝感到被教訓。

**第五段：結語頌聖——太平盛世**

> 「今則四海一家，萬物咸亨。」

最後一段回歸頌聖。宋濂以「四海一家，萬物咸亨」八字描寫朱元璋治下的太平盛世，既是對明初現實的肯定，也是對前文規諫的「緩衝」——他已經說完了該說的話，現在需要讓皇帝帶著愉快的心情結束閱讀。最後以對朱元璋功業的讚美收束，與開篇的「雄才大略之主」遙相呼應，結構圓滿閉合。

## 2. 寫作技巧分析

### 2.1 核心策略——「頌聖中藏規諫」的糖衣炮彈

〈閱江樓記〉最核心的寫作技巧，是一種極其精巧的「雙層文本」：表層是歌功頌德的頌聖文章，深層是委婉規諫的政治議論。宋濂採用了「先揚後抑再揚」的三段式修辭策略：

| 階段 | 內容 | 效果 |
|:----:|:----|:----:|
| **先揚**（第1–2段） | 讚朱元璋為雄才大略之主，讚長江為天下天險 | 讓朱元璋愉悅、放鬆戒心——這是一篇正常的頌聖文章 |
| **後抑**（第3–4段） | 回顧六朝滅亡，指出天險不足恃 | 在朱元璋愉悅的閱讀中冷不防插入「歷史教訓」——但他無法反駁，因為說的都是事實 |
| **再揚**（第5段） | 讚美當下太平盛世，歸功於朱元璋的仁德 | 讓文章以歌頌結束，確保皇帝不會帶著不悅的心情合上書卷 |

這種「糖衣炮彈」結構，使宋濂完成了表面上看起來不可能完成的任務：在皇帝面前既說了真話，又讓皇帝無法發作。

### 2.2 歷史對比論證——不批評當下，只回顧過去

宋濂規諫朱元璋的核心手段是**歷史對比法**：他不說「您應該如何如何」，而是說「歷史上那些王朝因為什麼而滅亡」。這種間接批評的手法極其安全，因為：

1. **批評的是歷史人物**（六朝君主），不是當今皇帝
2. **引用的是歷史事實**（六朝滅亡），不是個人觀點
3. **推論留給讀者**：宋濂只給出歷史案例，讓朱元璋自己推導出「如果不施行仁政，我的王朝也可能步六朝後塵」的結論

這種「讓皇帝自己教育自己」的寫法，比任何直諫都更有效，也更安全。

### 2.3 虛實結合——為不存在的建築寫記

閱江樓從未建成，這是一個極其特殊的寫作處境。宋濂的應對策略是：**避虛就實，以江山代樓閣**。

| 避開（虛） | 轉向（實） |
|:-----------|:-----------|
| 樓閣本身的建築描寫（無樓可寫） | 長江的壯闊景色（真實存在） |
| 登樓所見的具體景觀（不存在） | 南京的歷史滄桑（真實存在） |
| 閱江樓的內部陳設（不存在） | 六朝興亡的歷史教訓（真實存在） |

這種處理方式讓〈閱江樓記〉在「樓記」體裁中獨樹一幟：它不寫樓而寫江山，不寫建築而寫歷史。名義上是為一座樓作記，實際上是為整片江山作論。

### 2.4 章法開合——「今→古→今」的時空結構

全文的章法是一個標準的「今→古→今」結構：

```
今（第1–2段）：當下的太平盛世 + 長江天險
        │
        ↓
古（第3段）：六朝興亡的歷史教訓
        │
        ↓
今（第4–5段）：朱元璋的當下與未來——以仁政守江山
```

這種結構的好處是：

- **縱深感**：從當下回溯歷史，再回到當下，賦予文章歷史的厚度
- **針對性**：歷史教訓直接指向當下的現實問題，使規諫有的放矢
- **可接受性**：以當下的讚美開始和結束，讓規諫隱藏在中間，降低皇帝的抗拒心理

### 2.5 語言的「溫柔敦厚」風格

宋濂的語言風格被評為「典重婉曲」，特別體現在以下幾個方面：

1. **用語委婉**：不說「你不要只靠長江」，而說「長江天塹，所以限南北也」——用一個「限」字暗示局限性
2. **引古為據**：核心論點以「臣聞古人有言」的方式呈現，把自己的觀點包裝為古人的教訓
3. **節奏平穩**：全文沒有激烈的措辭、沒有尖銳的批評，語氣始終溫和平實
4. **分寸精準**：每說完一段「可能讓皇帝不高興的話」，立即補上一段讓皇帝高興的話——節奏控制精準到字

這種語言風格是宋濂在朱元璋身邊生存的必備技能——在一個文字獄頻發、猜忌心極重的皇帝面前，每一句話都必須經過反覆掂量。

### 2.6 修辭手法總覽

| 手法 | 原文 | 效果 |
|:----|:-----|:-----|
| **引用/用典** | 「應天順人」（出自《周易·革卦》） | 用儒家經典為朱元璋的統治提供合法性，也為後文規諫鋪墊 |
| **引用/用典** | 「在德不在險」（出自《左傳·昭公四年》司馬侯諫楚靈王） | 將核心命題包裝為古人的智慧，降低規諫的攻擊性 |
| **對比** | 六朝據長江天險而亡 vs. 朱元璋以仁政治天下而興 | 以歷史的失敗襯托當下的成功，歌頌中暗含提醒 |
| **象徵** | 長江 = 天險的象徵；閱江樓 = 帝王氣象的象徵 | 使具體的地理事物承載抽象的政治哲學意義 |
| **層遞** | 讚美天險 → 指出天險局限性 → 歷史證明天險不足恃 → 結論：仁政為本 | 層層推進，邏輯嚴密，讓讀者無法反駁 |
| **設問（隱含）** | 通篇不言問，但每一段都在回答一個隱含的問題：「天險可靠嗎？」 | 引導讀者自己思考，而不是被動接受結論 |
| **呼應** | 開篇「雄才大略之主」→ 結尾「四海一家，萬物咸亨」 | 首尾圓合，頌聖基調貫穿全篇 |
| **省略/避實就虛** | 全篇不寫樓閣本身 | 巧妙地繞過了「無樓可寫」的困境，將文章從建築描寫轉向歷史議論 |
| **含蓄/婉曲** | 不直接批評任何人 | 所有規諫都以歷史分析的形式呈現，字面上沒有批評當今皇帝的任何內容 |

## 3. 佳句賞析

### 佳句一

> **「臣聞昔者，帝王之興，必有雄才大略之主，以應天順人。」**

**現代漢語：** 臣聽說自古以來，帝王的興起，必定有雄才大略的君主，來順應天意、合乎民心。

**賞析：** 這是全篇開篇第一句。宋濂以「臣聞」開頭，第一時間表明此文是臣子向皇帝進言的身份定位——這不是自由創作，而是奉命進呈的文字。「雄才大略之主，以應天順人」——十一個字完成了三重功能：一是將朱元璋合法化（應天順人是儒家對開國之君的最高評價），二是將朱元璋神聖化（天意授權），三是為後文規諫設置前提（既然是雄才大略之主，自然應該明白「在德不在險」的道理）。此句可謂「一句定調」，全篇的頌聖基調與議論空間都在這一句中確立。

### 佳句二

> **「長江天塹，所以限南北也。」**

**現代漢語：** 長江這道天然的壕溝，是用來分隔南北的。

**賞析：** 這是全文最著名的名句，也是理解整篇文章的關鍵鑰匙。字面上，宋濂在讚美長江作為天險的險要——「天塹」二字將長江比作天然的護城河，極言其不可逾越。但關鍵在於「所以限南北也」五個字——「限」是限制、分隔之意。長江的作用只是「分隔南北」，它不能確保王朝的長治久安。這句話表面上是在說長江的重要性，實際上是在暗示長江的局限性。在中國歷史上，以長江為天險的王朝（六朝、南唐）全部偏安一隅，沒有統一中國的能力；而統一王朝（秦、漢、唐）的長江天險都在自己的控制之下。宋濂這句話，將長江從「國家的屏障」降格為「南北的分界線」——看似讚美，實為鋪墊。

### 佳句三

> **「今則四海一家，萬物咸亨。」**

**現代漢語：** 如今天下統一、四海成為一家，萬物都順利亨通。

**賞析：** 這是全文的關鍵轉折句，出現在歷史回顧與核心命題之後。宋濂在論證了「天險不足恃」、回顧了六朝滅亡的歷史教訓之後，筆鋒一轉，以「今則」二字將讀者從歷史的廢墟中拉回當下的太平盛世。「四海一家」是對朱元璋統一的讚美（與六朝偏安形成對比），「萬物咸亨」是對天下安定的描寫（與六朝動盪形成對比）。這一句的作用至關重要：它讓規諫之後的文章以頌聖收尾，確保朱元璋不會帶著被批評的感覺結束閱讀。同時，它也在暗示朱元璋：您已經做到了六朝君主做不到的事——統一了天下——那麼接下來要做的事，就是守住這份統一，而守住的關鍵不在長江，而在仁政。

### 佳句四

> **「在德不在險」——全文的核心命題**

**現代漢語：** 國家的長治久安在於德行，不在於地勢的險要。

**賞析：** 這四個字雖非原文中的獨立句，卻是全文的核心論點。宋濂化用了《左傳·昭公四年》中司馬侯對楚靈王的勸諫：「國之興也，視民如傷……其亡也，以民為土芥……（山河之險）非所恃也。」宋濂將這個古老的命題與閱江樓的主題緊密結合——朱元璋在建閱江樓，是因為長江天險的戰略價值；但宋濂提醒他：天險再險，也不如仁政可靠。

這個命題的巧妙之處在於：它不是一個針對朱元璋的批評，而是一個普遍的政治哲學命題。宋濂沒有說「您做錯了什麼」，而是說「歷史上那些做錯了的王朝，都是因為不施行仁政」。朱元璋如果認同這個命題，他就必須在自己的治理中體現仁政；如果他不認同，他就等於在否定儒家基本的政治理念——而他作為一個剛剛建立王朝的皇帝，不可能公開否定儒家理念。所以，無論朱元璋內心是否高興，他都不能反對這個命題。

### 佳句五

> **「昔者六朝之君……雖有長江之險，終不免於滅亡。」**

**現代漢語：** 從前六朝的君主……雖然有長江這樣的天險，最終還是免不了滅亡。

**賞析：** 這是全文最具衝擊力的一段論證。宋濂以「六朝」為案例，舉出了定都南京的六個短命王朝——三國吳（59年）、東晉（104年）、宋（60年）、齊（24年）、梁（56年）、陳（33年）。這些王朝無一例外擁有長江天險，但平均國祚不到六十年。宋濂用一個「雖……終……」的轉折句式，將長江天險與王朝滅亡之間的因果關係清晰呈現：長江天險不是王朝存續的充分條件。

這段論證對朱元璋的特別意義在於：朱元璋定都南京——正是六朝定都的同一座城市。宋濂的言外之意不言自明：如果您只靠長江，您的王朝不會比六朝更長久。這是最溫柔的規諫，也是最嚴厲的警告。

## 4. 白話改編

以下是〈閱江樓記〉的原文與現代漢語翻譯對照：

### 原文

```
金陵為帝王之州。自六朝迄於南唐，類皆偏據一方，無以應山川之王氣。逮我皇帝，定鼎於茲，始足以當之。由是聲教所暨，罔閒朔南，存神穆清，與道同體。雖一豫一遊，亦可為天下後世法。京城之西北有獅子山，自盧龍蜿蜒而來。長江如虹貫，蟠遶其下。上以其地雄勝，詔建樓於巔，與民同遊觀之樂，遂錫嘉名為閱江云。
```

### 白話翻譯

金陵（南京）自古以來就是帝王建都的地方。然而從六朝一直到南唐，這些朝代都不過是偏安一隅，沒有人能夠真正匹配這片山川所匯聚的帝王之氣。直到我朝皇帝（朱元璋）在此定都，才足以擔當這份王者氣象。從此以後，朝廷的聲威教化所到之處，無論南北都沒有阻隔；皇上涵養精神、清靜平和，與天道合而為一。即便是一次休閒、一次遊覽，也可以成為天下後世效法的榜樣。京城的西北方有一座獅子山，山脈從盧龍蜿蜒伸展而來。長江如同一道長虹橫貫大地，盤繞在山腳之下。皇上認為此地地勢雄偉壯麗，於是下詔在山頂建造一座樓閣，讓百姓同享遊覽觀賞的樂趣，並賜予它美好的名字——「閱江樓」。

### 原文

```
登覽之頃，萬象森列，千載之祕，一旦軒露。豈非天造地設，以俟大一統之君，而開千萬世之偉觀者歟？當風日清美，法駕幸臨，升其崇椒，凭闌遙矚，必悠然而動遐思。見江漢之朝宗，諸侯之述職，城池之高深，關阨之嚴固，必曰：「此朕櫛風沐雨、戰勝攻取之所致也。」中夏之廣，益思有以保之。見波濤之浩蕩，風帆之下上，番舶接跡而來庭，蠻琛聯肩而入貢，必曰：「此朕德綏威服，覃及內外之所及也。」四陲之遠，益思有以柔之。見兩岸之閒，四郊之上，耕人有炙膚皸足之煩，農女有捋桑行饁之勤，必曰：「此朕拔諸水火，而登於袵席者也。」萬方之民，益思有以安之。觸類而思，不一而足。臣知斯樓之建，皇上所以發舒精神，因物興感，無不寓其致治之思，悉止閱夫長江而已哉。彼臨春、結綺，非不華矣；齊雲、落星，非不高矣。不過樂管絃之淫響，藏燕趙之豔姬，一旋踵間而感慨係之，臣不知其為何說也。
```

### 白話翻譯

當登樓觀覽的時候，萬千景象羅列眼前，千年以來隱藏的秘密，一下子豁然開朗。這難道不是上天創造、大地設置，等待著大一統的君主，來開創千秋萬世的宏偉景觀嗎？每當風和日麗、天清氣爽，皇上的車駕親臨此地，登上高高的山頂，憑欄遠眺，必然會悠然生出深遠的思緒。看到長江、漢水像諸侯朝見天子一樣奔流匯聚，各地官員前來述職，城池的深固、關隘的嚴密，他一定會說：「這是我風吹雨打、浴血奮戰、戰勝攻取所換來的啊。」想到中原的廣闊，就更要思考如何保有它。看到波濤浩蕩、船帆上下往來，外國的船隻接連前來朝貢，遠方的珍寶源源不斷進獻，他一定會說：「這是我以恩德安撫、以威勢降服，使德化遍及內外所達到的啊。」想到四方邊境的遙遠，就更要思考如何懷柔遠方。看到兩岸之間、四郊之上，農夫有烈日炙烤皮膚、雙腳凍裂的辛勞，農婦有採桑送飯的勤苦，他一定會說：「這是我把他們從水深火熱中拯救出來，使他們能夠安居樂業的啊。」想到天下萬民，就更要思考如何讓他們安寧。觸類旁通，聯想不止。小臣我明白，這座閱江樓的建造，是皇上用來舒展精神、因物感懷的方式，每一處都寄寓著他治理天下的深遠思慮，哪裡僅僅是觀賞長江的景色而已呢？那些前朝的臨春閣、結綺閣，不是不華麗；齊雲樓、落星樓，不是不高聳。但它們不過是用來享受靡靡之音、收藏美豔歌姬的場所，一轉眼之間就煙消雲散，令人感慨繫之，我不知道那樣有什麼好說的。

### 原文

```
雖然，長江發源岷山，委蛇七千餘里而入海，白涌碧翻。六朝之時，往往倚之為天塹。今則南北一家，視為安流，無所事乎戰爭矣。然則果誰之力歟？逢掖之士，有登斯樓而閱斯江者，當思聖德如天，蕩蕩難名，與神禹疏鑿之功，同一罔極，忠君報上之心，其有不油然而興耶？臣不敏，奉旨撰記，故上推宵旰圖治之功者，勒諸貞珉。他若留連光景之辭，皆略而不陳，懼褻也。
```

### 白話翻譯

話說回來，長江發源於岷山，蜿蜒七千多里流入大海，白浪翻湧、碧波蕩漾。六朝的時代，往往倚仗長江作為天然壕溝。而現在南北已經統一，長江被視為平穩的水流，不再需要依靠它來從事戰爭了。那麼，這究竟是誰的力量呢？那些身穿儒服的讀書人，如果有機會登上這座樓、觀賞這條江，應當想到皇上的聖德如天一般廣大，浩瀚得難以形容，與大禹疏導開鑿的功績一樣無窮無盡，那麼忠君報國的心情，怎麼會不油然而生呢？小臣我不才，奉聖旨撰寫這篇記，所以在此推述皇上宵衣旰食、勵精圖治的功績，刻在堅硬的美玉石碑上。至於那些流連光景、吟風弄月的辭句，都一概省略而不寫，因為怕褻瀆了這篇文章的莊重意義。

## 5. 綜合心得

〈閱江樓記〉的偉大，不在於辭藻的華麗，也不在於結構的工整，而在於它完成了一個幾乎不可能完成的任務——在一個猜忌心極重、動輒誅殺功臣的皇帝面前，說出了「天險不如仁政」這樣的話，卻沒有讓皇帝感到被冒犯。

### 寫在鋼絲上的文字

理解〈閱江樓記〉，必須先理解宋濂所處的政治環境。朱元璋是中國歷史上出身最低微的開國皇帝——乞丐、和尚出身——卻也因此對權力有著極度的敏感與戒備。洪武年間，文字獄頻發，朝臣因一句詩、一個用字不當而被處死的事時有發生。宋濂身為朱元璋的「御用筆桿子」，每天寫作時都如履薄冰。

在這種環境下，〈閱江樓記〉的成就就不僅是文學層面的了。它更是一次政治上的高難度動作——宋濂要在歌頌中說真話，在讚美中規諫，在安全中表達立場。全文沒有一句刺耳的話，沒有一個可能觸怒皇帝的詞——但該說的道理一個字也沒少說。這不是投機取巧，而是極高政治智慧的體現。

### 從「未建成」到「不朽」

閱江樓從未建成，這個事實賦予了〈閱江樓記〉一種特殊的詩意張力。讀者在閱讀全文時，知道這篇文章所描述的樓閣從來沒有真實存在過——宋濂筆下的一切都是想像中的景象。但正因為樓沒有建成，文章反而獲得了超越建築的生命力。

這就形成了一個深具諷刺意味的悖論：

- 那些真實存在的建築——滕王閣、黃鶴樓、岳陽樓——經歷了無數次毀滅與重建
- 而從未存在的閱江樓，因為一篇文章而在文學史中永遠「存在」
- 直到2001年，南京真的在獅子山上建起了閱江樓——距離宋濂寫作此文的1374年，整整627年

建築會倒塌，權力會消失，但好的文章可以跨越時間的長河。宋濂的文字不僅讓一座不存在的樓在文學中屹立了六個多世紀，最終還催生出了真實的建築——這或許是對「蓋世有足恃者，而不在乎臺之存亡也」最完美的註腳，雖然這句話出自蘇軾而非宋濂。

### 「在德不在險」——一個跨越時代的命題

宋濂在〈閱江樓記〉中提出的核心命題——「在德不在險」——不僅適用於明初，也適用於任何時代、任何政權。這個命題的深刻之處在於：它揭示了「合法性」的真正來源——不是武力、不是地利、不是天險，而是民心。

朱元璋以「驅逐胡虜、恢復中華」的口號起兵，依靠的是民心；他最終統一全國，依靠的也是民心。宋濂在〈閱江樓記〉中不過是委婉地提醒他：你能得到天下是因為民心，你能否守住天下也取決於民心。長江天險可以幫你擋住一時的敵軍，但擋不住失去民心後的天下大亂。

這個命題在今天仍然具有現實意義。任何一個政權、一個組織、一個領導者，如果只依賴外在的優勢（資源、地位、權力）而忽略了內在的德行（誠信、公正、仁愛），遲早會面臨「六朝式的困境」——外在優勢似乎很強大，但內在的腐敗已經開始動搖根基。

### 溫柔敦厚的力量

〈閱江樓記〉是中國古代「溫柔敦厚」詩教的典範之作。宋濂沒有用尖銳的語言、沒有用激昂的語氣、沒有用咄咄逼人的邏輯——他始終保持著儒臣的溫和語調。但正是這種溫柔的語氣，讓他的話語更有分量。

暴怒的指責可以被反駁、可以被無視，但溫柔的提醒卻會在心中久久迴盪。宋濂的〈閱江樓記〉就像一粒種子，悄悄地種在朱元璋的心裡——它不會當場發作，但會在日後的某個時刻悄然發芽。這或許是規諫的最高境界：不是要皇帝當場認錯，而是讓他在夜深人靜時獨自思考。
