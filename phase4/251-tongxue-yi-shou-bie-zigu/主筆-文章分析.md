# 〈同學一首別子固〉文章分析

> **主筆分析 · 古文觀止卷十一·宋文·王安石**

---

## 一、文章結構與起承轉合

### 1.1 全文結構總覽

〈同學一首別子固〉全文不到三百字，是《古文觀止》卷十一中最短的文章之一。然篇幅雖短，結構卻極其精巧——以「同學」為骨，以「陪襯法」為脈，三段遞進，從立論到印證到點題，一氣呵成。

| 段落 | 核心內容 | 功能 |
|:----:|----------|:----:|
| **第一段** | 並舉二賢：江南有子固，淮南有正之。二人不相識，但言行極度相似。原因：學聖人而已矣。 | **立論**——提出「同」的哲學根據，以「學聖人」解釋為何不相識之人如此相似 |
| **第二段** | 王安石自述為二人互傳消息：在淮南對正之提子固，正之不疑；回江南對子固提正之，子固亦信。曾鞏作〈懷友〉，正之亦有類似之言。 | **印證**——用具體事例證明「同學」之實，將哲學論證落實為可驗證的人際交往 |
| **第三段** | 感慨：「官有守，私有繫，會合不可以常也。」作此篇以相警且相慰。 | **點題**——從「同」轉回「別」，完成贈別文的實際功能，收束全篇 |

### 1.2 起承轉合分析

| 結構 | 對應段落 | 核心內容 | 寫作策略 |
|:----:|:--------:|----------|:---------|
| **起** | 第一段前半：「江之南有賢人焉……其不相似者何其少也！」 | 先設立兩個賢人形象，再以「考其言行」發現其驚人相似，製造懸念——為什麼兩個素未謀面的人會如此相似？ | **設懸念，引人入彀**。王安石不直接說「我和曾鞏是好朋友」，而是從一個普遍的現象入手：兩個不相干的人，言行卻高度相似。讀者被這個現象吸引，自然想追問原因。 |
| **承** | 第一段後半：「曰：學聖人而已矣……其相似也，適然。」 | 給出答案——原因在於「學聖人」。聖人之道唯一，學聖人者自然殊途同歸。 | **揭謎底，立論據**。用一個簡潔的判斷句「學聖人而已矣」回答前文的懸念，再用反問「聖人之言行，豈有二哉」加以強化，最後以「適然」（順理成章）收束，滴水不漏。 |
| **轉** | 第二段：自述為二人傳話，二人相互信任，且各自有相近的言論與志向。 | 從哲學論證轉入具體事例——王安石用自己的親身經歷，證明曾鞏與孫正之不但「相似」，而且「相信不疑」。 | **以實證虛，以事證理**。前一段是純粹的邏輯論證，這一段用王安石作為「中介人」的具體經驗來驗證——你不信他們相似？我親口告訴他們彼此的存在，他們都毫不懷疑。 |
| **合** | 第三段：「噫！官有守，私有繫，會合不可以常也。作《同學》一首別子固，以相警，且相慰云。」 | 點出「別」的主題，交代寫作目的——以「同學」互相提醒與安慰。 | **收束點題，淡而有味**。全篇最動人的一句感慨——沒有「我想你」「勿忘我」，只有淡淡的「公務在身，家事牽絆，無法常見」。正因如此，才需要這篇「同學」來維繫精神上的連結。 |

### 1.3 起承轉合結構圖解

```
起（第一段前半）：設立二賢（子固、正之）→ 發現二人相似 → 製造懸念：為何相似？
    │
    ▼
承（第一段後半）：解答懸念 →「學聖人而已矣」→ 聖人之道唯一 → 相似是必然
    │
    ▼
轉（第二段）：個人經驗印證 → 為二人互傳信息 → 皆「相信不疑」→ 曾鞏與正之有相近的志向
    │
    ▼
合（第三段）：「官有守，私有繫」→ 不能常見 → 作此篇以相警相慰 → 完成贈別功能
```

### 1.4 結構特點

1. **三段遞進，層層深入**：立論（哲學）→ 印證（事實）→ 點題（情感），從抽象到具體再到情感回歸，邏輯鏈條完整
2. **陪襯法（以賓顯主）**：引入孫正之作為「賓」，以襯托曾鞏這個「主」——因為正之與鞏不相識卻相似，更證明了「學聖人」的普遍性
3. **先立後破不破而立**：王安石不批評任何與他觀點不同的立場，而是直接建立一個新的概念（「同學」作為精神共同體），讓舊的觀念自然失效
4. **淡筆收束，餘味無窮**：結尾不煽情、不誇張，以一個「噫」字輕嘆，以「相警且相慰」作結，平實中見真情

> 💡 **主筆按**：〈同學一首別子固〉的結構之所以為人稱道，關鍵在於「以賓顯主」的手法。一般贈別文只寫兩個人的友誼，但王安石引入一個第三人（孫正之），並且是曾鞏不認識的人——這就把個人化的友誼提升到了「道」的層次。你說你和曾鞏是好朋友？這沒什麼稀奇。但你看，連曾鞏不認識的孫正之，因為學聖人，言行都和曾鞏相似——那你說，聖人之道是不是真的讓所有追隨者都殊途同歸？這個結構設計，是文章從個人情誼走向普遍命題的關鍵。

## 二、修辭與寫作手法

### 2.1 修辭手法分析

| 修辭手法 | 原文例證 | 作用與效果 |
|:---------|----------|:-----------|
| **並舉（對舉）** | 「江之南有賢人焉，字子固……淮之南有賢人焉，字正之」 | **全文最核心的結構手法**。以空間對稱（江南/淮南）引出兩個人物，讀者立刻感受到一種工整的平衡感。這種並舉不是隨意的，而是為後文的「對照」做準備——二人在空間上相隔千里，卻在精神上高度一致。 |
| **層遞（否定遞進）** | 「足未嘗相過也，口未嘗相語也，辭幣未嘗相接也」 | 三個否定句層層遞進——沒見過面、沒說過話、沒通過信。從空間接觸（足）→ 語言交流（口）→ 物質往來（辭幣），由表及裡，徹底排除了一切「外部聯繫」的可能性，凸顯後文「相似」的純粹性。 |
| **設問** | 「其師若友，豈盡同哉？」 | 表面是問句，實則是引導讀者自己得出結論：如果師友都不相同，但言行卻如此相似——這只能說明不是師友的因素，而是他們自身的原因（學聖人）。 |
| **反問** | 「聖人之言行，豈有二哉？」 | 以反問肯定聖人之道的唯一性。語氣堅定，不容置疑——聖人的言行只有一個標準，所以學聖人的人自然也只有一個方向。 |
| **斷言（判斷句）** | 「學聖人而已矣。」 | 全文的「文眼」，也是邏輯鏈條的關鍵節點。以一個極簡的判斷句給出答案，「而已矣」三字帶有「不過如此」的輕描淡寫——看似謙虛，實則自信。 |
| **條件推論** | 「學聖人，則其師若友，必學聖人者。」 | 用一個「則」字構成邏輯推演：前提→結論。因為自己學聖人，所以選擇的師友也必然是學聖人的——這就解釋了「其師若友豈盡同哉」的疑問。 |
| **遞進（從結論到反響）** | 「予又知所謂賢人者，既相似又相信不疑也。」 | 從「相似」進一步推進到「相信不疑」。相似是客觀的觀察結果，相信是主觀的態度——二者相輔相成，讓「同學」的概念更加立體。 |
| **比喻（隱喻）** | 「安驅徐行，轥中庸之庭，而造於其室」 | 以「駕車進入中庸的庭院、登堂入室」比喻修養聖賢之道的過程——從入門到深入，形象而生動。這種節制的比喻風格，正是王安石「淡筆」的特色。 |
| **感嘆** | 開篇的「何其少也！」＋ 結尾的「噫！」 | 全篇僅有的兩處感嘆。前者表達發現二人相似時的驚喜，後者表達不能常聚的無奈——一喜一嘆，遙相呼應。 |
| **互文** | 「以相警，且相慰云」 | 「相警」（互相提醒）與「相慰」（互相安慰）互文見義——提醒是為了安慰，安慰中也包含著提醒。兩個動作其實是同一件事的兩面。 |

### 2.2 寫作策略

**策略一：陪襯法——以第三人證普遍之理**

這是本文最核心的寫作策略，也是王安石最具創造性的結構設計。一般贈別文寫友誼，不外乎直抒胸臆或回憶往事。但王安石選擇了一個極為高明的方法：引入一個雙方的共同朋友（孫正之）作為「證人」——你不需要認識他，但你們的相似已經說明了所有的問題。

這個策略的好處是：它把「我們是好朋友」這個主觀感受，變成了「你們（曾鞏與孫正之）因為學聖人而相似」這個客觀事實。讀者無法否認曾鞏與孫正之兩人的相似性（因為王安石已經在文中以自己為中介證明了），因此只能接受他的前提——學聖人的人確實會殊途同歸。

《古文觀止》編者吳楚材、吳調侯評曰：「別子固而以正之陪說，交互映發，錯落參差。」——「陪說」二字，精確點出了陪襯法的核心。

**策略二：不言別而別意自出——虛實相生的贈別藝術**

全文只有最後一段直接提到「別」字，但「別」的主題貫穿始終。第一段寫二人「足未嘗相過，口未嘗相語」——這是「隔」；第二段寫王安石為兩地傳話——這是「通」；第三段說「官有守，私有繫，會合不可以常」——這又是「隔」。全文就在「隔」與「通」、「同」與「別」之間反覆搖擺，最終落在「以相警且相慰」——用精神上的「同」來彌補空間上的「別」。

這種寫法的精妙在於：它不直接說「我會想你」，而是通過「我們都是學聖人的人」這個前提，推導出「即使不見面，我們的心也在同一條路上」。情感不是直接表達的，而是被讀者自己「推導」出來的——這就是「不說別而別意自出」的境界。

**策略三：以淡筆寫深情——宋代散文「平淡自然」美學的典範**

王安石的散文以議論雄健著稱（如〈答司馬諫議書〉、〈讀孟嘗君傳〉），但這篇文章卻寫得極淡。沒有華麗的修辭、沒有激昂的情感、沒有精妙的比喻——有的只是一個二十三歲的年輕人，用最平靜的語氣，告訴他的朋友：「我們雖然不能常見，但我們是同一條路上的人。」

這種「淡」不是無力，而是一種更高級的審美追求。宋代理學影響下的散文追求「平淡自然」——最好的文章不是讓人拍案叫絕，而是讓人讀完之後「尋味無窮」。〈同學一首別子固〉正是這種美學的完美體現。

### 2.3 節奏控制

〈同學一首別子固〉的節奏極具辨識度：

- **開篇平穩**：以「江之南有賢人焉」「淮之南有賢人焉」兩個平行句式開篇，節奏從容不迫，讀者彷彿跟隨王安石的目光在江南淮南之間來回掃視
- **中段漸緊**：從「足未嘗相過」的三個否定句開始，節奏逐漸加快；到「學聖人而已矣」戛然而止，形成一個節奏上的「急剎車」——前面所有的鋪墊都指向這六個字
- **結尾從容**：「噫」字一嘆之後，節奏再次放緩——「官有守，私有繫，會合不可以常也」——三句一氣讀來，語氣平和而略帶感傷，最終落在「以相警，且相慰云」，餘音裊裊

這種「平穩→漸緊→急停→放緩」的節奏變化，讓不到三百字的短文擁有了一種內在的音樂性。

## 三、名句深度解析

### 3.1 「學聖人而已矣。」

**白話：** 只不過是在學習聖人罷了。

**賞析：**

這是全文的「文眼」，也是邏輯鏈條的關鍵一環。前文層層鋪墊——兩個不相識的人、不說話的人、不通信的人，言行卻驚人地相似——讀者已經被懸念吊足了胃口，等待一個答案。王安石以最簡潔的六個字給出回答。

「而已矣」這個語氣詞極為講究。它不是「就是這樣」（太肯定），也不是「大概是這樣吧」（太不肯定），而是「不過如此罷了」——一種輕描淡寫的確定。王安石說出這四個字時，彷彿在說：這個道理如此簡單，你早就應該想到了。這種自信，是一個二十三歲的年輕人在精神上已經成熟的標誌。

更深一層看，這六個字其實隱含了王安石對知識與修養的根本理解：聖人之道是唯一的、客觀的、超越時空的。你不需要創造什麼新的東西，你只需要「學」——而學習的結果，自然是與所有同樣在學聖人的人走向一致。這種「道唯一」的信念，既是宋代理學的基石，也是王安石後來變法的思想根基。

### 3.2 「聖人之言行，豈有二哉？其相似也，適然。」

**白話：** 聖人的言行，難道會有兩種標準嗎？所以他們的相似，是順理成章的事情。

**賞析：**

這是全文的邏輯核心——王安石用一句反問（豈有二哉）和一個判斷（適然），完成了從「現象」到「規律」的飛躍。

「豈有二哉」——這是一個不容辯駁的反問。它建立在一個基本預設之上：聖人之道是唯一的。如果你接受這個預設，那麼結論就自然而然地推導出來了。

「適然」二字是全篇最精彩的用詞之一。「適然」不是「偶然」，而是「順理成章」——不是碰巧，而是理所當然。王安石的邏輯是：因為聖人之言行只有一個標準，所以越是深入地學習，人的言行就越接近這個標準——而當兩個人都達到了相當深度的時候，他們之間的相似就是必然的結果。

這個詞的選擇極為精妙：它既消解了「相似」的神秘感（不是巧合，是必然），又保持了對聖人之道的敬畏（不是刻意模仿，是自然結果）。文章從此不再只是談兩個人的友誼，而是上升到了對整個「聖人之學」的論證。

### 3.3 「官有守，私有繫，會合不可以常也。」

**白話：** 公職有職責的約束，私事有家庭的牽絆，所以我們不能經常見面相聚。

**賞析：**

全文情感濃度最高的一句話，卻以最平淡的方式寫出。

十三個字，三層意思：「官有守」——我在淮南任判官，你有你的學業與生活，我們都有各自的事業要承擔；「私有繫」——我們還有家庭的責任、個人的牽絆；「會合不可以常也」——所以，我們註定不能經常見面。

這句話之所以動人，在於它的「節制」。王安石明明可以寫「我很想你」「我們什麼時候才能再見」，但他選擇了最克制的方式——不是抒發感情，而是陳述事實。這種「事實陳述式」的情感表達，比任何直白的抒情都更加有力量。因為它告訴讀者：這不是一個人的情感衝動，而是兩個人必須共同面對的生命現實。

更要緊的是，這句話放在「學聖人」的大背景下，有了另一層深意：雖然我們不能常見，但我們在「學聖人」這條路上是常見的——因為聖人之道不會因為距離而改變。

### 3.4 「作《同學》一首別子固，以相警，且相慰云。」

**白話：** 我寫了這篇〈同學〉來為子固送別，用來自警，也用來自慰。

**賞析：**

全文的結尾句，也是王安石對這篇文章的「功能說明」。

「相警」——互相提醒。提醒什麼？提醒對方不要忘記「學聖人」的初心。在科舉社會中，人很容易被功名利祿所迷失——王安石自己剛剛考上進士，曾鞏剛剛落第，兩人都面臨著世俗評價的考驗。王安石用這篇文章提醒自己和曾鞏：「學聖人」才是我們真正的目標，中不中舉、做不做官，都不重要。

「相慰」——互相安慰。安慰什麼？安慰對方不能常見的遺憾。因為「官有守，私有繫」，我們不能像普通朋友那樣朝夕相處，但這篇文章就是我們的「精神契約」——只要你讀它，你就知道我們在同一條路上。

這兩個詞放在一起，精確地概括了全文的雙重功能：既是對外的「勵志」（相警），也是對內的「抒情」（相慰）。最後一個「云」字，帶有輕微的感嘆語氣，讓整篇文章在一聲輕嘆中收束，餘韻無窮。

## 四、文章邏輯圖解

### 4.1 核心論證鏈

```
前提 A：江南有賢人曾鞏，淮南有賢人孫正之
前提 B：二人從未見面、從未說話、從未通信
前提 C：但二人的言行高度相似
────────────────────────
問題：為什麼兩個不相識的人會如此相似？
     │
     ▼
答案：學聖人而已矣。
     │
     ├─ 推理一：「學聖人，則其師若友，必學聖人者」
     │    └─ 因為學聖人，所以選擇的師友也是學聖人的人→ 相似是有原因的
     │
```

**結構原理：**

王安石用孫正之這個「第三人」來實現「以賓顯主」的效果：

1. **直接效果**：如果只寫「我和曾鞏是好朋友」，文章局限於個人情誼
2. **引入正之後**：曾鞏與正之素未謀面卻言行相似 → 證明「學聖人」的普遍力量
3. **最終效果**：文章從「王曾之交」上升到「一切以聖賢為師者皆為同道」的普遍命題

> 💡 **主筆按**：這就是《古文觀止》編者所謂「交互映發，錯落參差」的真諦。賓主互相映照，兩條線索交織推進——第一段先分寫（子固、正之各一段），再合寫（考其言行、同歸於聖人）；第二段以王安石為中介，將兩條線索交織在一起；第三段收束於「相警相慰」，將賓主匯於一體。

### 4.3 對比系統

| 對比維度 | 正面（文章主張） | 反面（文章要消解的） |
|:---------|:----------------|:-------------------|
| **「賢人」的定義** | 「學聖人」的賢——內在修養為標準 | 「今所謂賢人」——世俗認可（官位、名聲）為標準 |
| **友誼的基礎** | 精神共鳴——「同道」 | 空間接近——「常見」 |
| **見面的意義** | 次要——即使不見也能有共同的精神方向 | 主要——不見面就等於疏遠 |
| **「同學」的定義** | 名詞——同一學派、同一精神陣營的人 | 動詞——一起讀書的同學 |
| **個人與道的關係** | 個體差異不影響本質一致（「其異也亦不遠矣」） | 個體差異導致無法溝通 |

### 4.4 文章技巧總結

〈同學一首別子固〉之所以成為千古名篇，得益於以下幾個層面的技巧：

1. **立意高遠**：將個人化的贈別，上升到「聖人之道唯一」的哲學高度
2. **結構精巧**：以陪襯法實現「以賓顯主」，三段遞進層層深入
3. **淡而有味**：不煽情、不議論，以最平淡的語氣寫最深的情誼
4. **虛實相生**：不直接寫「別」而同中見別、別中有同
5. **簡潔有力**：不到三百字的篇幅，完成立論→印證→點題的完整閉環

> 💡 **主筆終按**：〈同學一首別子固〉最了不起的地方，不是它的文章技巧，而是它背後的精神氣度。一個二十三歲的年輕人，剛剛考上進士，卻繞道幾百里去拜訪一個落第的朋友——他寫這篇文章，不是炫耀自己的成功，而是告訴對方：「我們的友誼不取決於科舉的成敗，不取決於見面的頻率，只取決於我們是否都在學聖人。」這種超越功名的友誼觀，這種「以道相交」的士人精神，才是這篇文章能夠穿越千年而依然動人的根本原因。王安石後來成為中國歷史上最具爭議的改革家，但在這篇文章中，我們看到的只是一個溫柔、真誠、相信人可以靠「同學」而連結的年輕人——這份真誠，比任何文章技巧都更加珍貴。
