# 〈藺相如完璧歸趙論〉文章分析

> **主筆分析 · 古文觀止卷十二·明文·王世貞**

---

## 1. 文章結構分析

### 1.1 總論

〈藺相如完璧歸趙論〉是明代文壇盟主王世貞（1526–1590）翻案史論的代表作，收於《弇州山人四部稿》。全文約三百五十字，篇幅極短，卻以嚴密的邏輯論證挑戰了千年來對藺相如「智勇雙全」的讚美定論。王世貞開門見山推翻傳統敘事——他不否定藺相如的個人勇氣，而是從「情」與「理」兩方面指出：藺相如的成功不是智謀，而是運氣。核心命題是：藺相如的每一步都是在賭秦國不翻臉，而真正使他全身而退的，是「秦意未欲與趙絕」的客觀局勢，而非其計謀的高明。

本文之精要在於「翻案而不全盤否定」——王世貞在結尾肯定了藺相如後來在澠池會上的強硬（勁澠池）和對廉頗的謙和（柔廉頗），認為這些才是「愈出而愈妙於用」的真正智慧。這種「有破有立」的寫法，使翻案免於偏激，讓批評更有說服力。

### 1.2 起承轉合分析

| 段落 | 起承轉合 | 對應原文 | 核心功能 | 論證策略 |
|:----:|:--------:|----------|----------|----------|
| **第1段** | **起** | 「藺相如之完璧，人皆稱之，予未敢以為信也。」 | 開門見山，直接推翻千年定論——一句「予未敢以為信也」，將所有人對藺相如的讚美全部懸置。這是翻案文章的標準開局：在最短時間內建立「我與所有人看法不同」的立場 | **開門見山、先聲奪人**：不鋪陳、不迂迴，直接亮出翻案立場。十個字就完成了翻案的宣示——這種自信和霸氣，只有文壇盟主才寫得出來 |
| **第2段** | **承** | 「夫秦以十五城之空名，詐趙而脅其璧。是時言取璧者情也，非欲以窺趙也。趙得其情則弗予，不得其情則予；得其情而畏之則予，得其情而弗畏之則弗予。此兩言決耳，奈之何既畏而復挑其怒也？」 | 承接翻案立場，開始「論情」——重新定義秦國的動機（秦的本意是取璧，而非攻趙），然後用二分法將趙國的選擇簡化為「予或弗予」，最後批評藺相如選了最糟的中間方案 | **二分法推理、反問收束**：先以「情」字重新定位秦國動機，再用「此兩言決耳」將複雜問題極簡化——最後用反問「奈之何既畏而復挑其怒也」點出核心矛盾 |
| **第3段** | **轉** | 「且夫秦欲璧，趙弗予璧，兩無所曲直也。……今奈何使舍人懷而逃之，而歸直於秦？」 | 文章最主要的轉折部分。從「論情」轉入「析理」：先設定理論框架（誰給誰不給的曲直歸屬），再代藺相如設想更好的策略，最後批評藺相如「歸直於秦」——把道理讓給了秦國 | **假設論證、二難推理、代擬對策**：通過假設「如果藺相如何如……」，建構一個更好的替代方案，讓讀者自行比較——替代方案愈合理，對藺相如的批評愈深刻 |
| **第4段** | **合** | 「是時秦意未欲與趙絕耳。令秦王怒而僇相如於市，武安君十萬眾壓邯鄲，而責璧與信，一勝而相如族，再勝而璧終入秦矣。吾故曰：藺相如之獲全於璧也，天也。」 | 全文的「結穴」——以經典的假設論證收束：如果秦王翻臉，後果不堪設想。得出核心結論：藺相如的成功是「天也」（運氣），而非智謀 | **反事實推理、假設推演**：用「令……則……」的假設語氣構建一個恐怖場景，讓讀者意識到藺相如的成功是如何脆弱——假設場景愈可怕，結論愈令人心服 |
| **第5段** | **尾（續合）** | 「若其勁澠池，柔廉頗，則愈出而愈妙於用。所以能完趙者，天固曲全之哉！」 | 補充收束，做重要區分：完璧歸趙是運氣（天），但「勁澠池、柔廉頗」是真智慧。以對藺相如的局部肯定，強化前面翻案批評的客觀性 | **對比肯定、公允收尾**：用「若其……」的轉折，從批評轉向肯定——先否定完璧歸趙的智謀，再肯定澠池與負荊的智慧，讓翻案不流於偏激 |

#### 起承轉合結構圖解

```
起（第1段）：推翻千年定論「予未敢以為信也」
    │
    ↓
承（第2段）：論情——秦的本意是取璧，非攻趙；趙的決策只需「予或弗予」
    │
    ↓
轉（第3段）：析理——藺相如「歸直於秦」；代擬更好的策略
    │
    ↓
合（第4段）：假設——令秦王怒而僇相如、白起攻邯鄲；結論——「天也」
    │
    ↓
尾（第5段）：補筆——肯定「勁澠池、柔廉頑」為真智慧
```

#### 結構特點

1. **先破後立，破立分明**：前四段是「破」——破壞藺相如完璧歸趙的「智謀」形象；第五段是「立」——建立「勁澠池、柔廉頗」為真智慧的正面標準。破立之間層次清晰。

2. **從情到理再到假設，層層遞進**：第一步論「情」（秦的動機），第二步論「理」（誰曲誰直），第三步做「假設」（如果翻臉）——從客觀事實分析，到抽象邏輯推演，再到虛擬情境設想，邏輯層次由淺入深。

3. **二段式收束，先定論後補充**：第四段用「吾故曰」做核心結論（天也），第五段再用「若其……」補充肯定。這種「結論＋補充」的結構，讓翻案文章既有結論的力道，又不失客觀公允。

4. **篇幅短小，論證緊密**：全文約三百五十字，卻涵蓋了翻案宣示、重新定義動機、二分法分析、曲直理論、代擬對策、假設論證、補筆肯定七個層次——論證密度極高，無一字浪費。

#### 對比系統

| 對比維度 | 正面（理想方案） | 反面（藺相如實際做法） |
|:---------|:----------------|:----------------------|
| **決策邏輯** | 清晰——要麼予、要麼弗予，「此兩言決耳」 | 矛盾——「既畏而復挑其怒」，既給了又偷回 |
| **對秦策略** | 用民怨和失信的道德壓力迫使秦王歸還璧玉 | 讓舍人懷璧而逃，把道理讓給了秦國 |
| **曲直歸屬** | 讓曲在秦——入璧後秦不給城，秦理虧 | 讓曲在趙——秦準備給城，趙偷回璧，趙理虧 |
| **風險評估** | 控制風險——當面說理，秦國若不從再從長計議 | 無視風險——賭秦王不翻臉，沒有任何備案 |
| **最終結果的性質** | 靠智慧——透過外交手段和道德壓力實現目標 | 靠運氣——因為秦意未欲與趙絕，僥倖成功 |
| **個人的成熟度** | 後期的「勁澠池、柔廉頗」——愈出愈妙 | 早期的「完璧歸趙」——賭博式的冒險 |
| **戰略視野** | 全局——考慮國家之間的外交利害關係 | 局部——專注於一璧之得失 |

#### 層層翻剝的論證技巧

王世貞此文最精彩之處在於**層層翻剝**——從外層的「人皆稱之」開始，每翻一層，藺相如行為的邏輯漏洞就更加赤裸：

1. **第一層（現象）**：藺相如完璧歸趙 → 世人讚為智勇 → 王世貞說：等等，「予未敢以為信也」

2. **第二層（動機）**：重新定義秦國的意圖 → 秦不是要攻趙，只是想要璧 → 趙國的選擇其實很簡單——「此兩言決耳」

3. **第三層（矛盾）**：藺相如選了不存在的中間方案 → 「既畏而復挑其怒」——既要屈服又要挑釁，邏輯上自相矛盾

4. **第四層（曲直）**：理論上的歸屬分析 → 三種情況的曲直 → 藺相如選擇了讓趙國理虧的那一種——「歸直於秦」

5. **第五層（替代方案）**：王世貞代擬一段「理想台詞」 → 如果藺相如當面說理，用「民怨＋失信」兩點說服秦王，「秦王未必不返璧也」——對比之下，藺相如的實際做法更顯不智

6. **第六層（假設）**：如果秦王翻臉 → 「僇相如於市」「武安君十萬眾壓邯鄲」→ 「一勝而相如族，再勝而璧終入秦」——成功假設的脆弱性被徹底暴露

7. **第七層（定案）**：不是智謀，不是勇氣，而是「天也」（運氣）——但補充肯定「勁澠池、柔廉頗」為真智慧，避免全盤否定

### 1.4 結語

〈藺相如完璧歸趙論〉的成就建立在**翻案立意與邏輯論證的高度統一**之上。王世貞作為文壇盟主，不僅有推翻千年定論的自信，更有以嚴謹邏輯支撐翻案的功力。全文三百五十字，從「論情」到「析理」再到「假設」，層層推進，步步緊逼，最終以「天也」二字一字定讞。

與宋代翻案文章（如蘇洵〈六國論〉、王安石〈讀孟嘗君傳〉）相比，王世貞的翻案展現出截然不同的風格。蘇洵的翻案帶有強烈的道德義憤和現實諷諭色彩（諫北宋對西夏的妥協）；王安石以極簡的篇幅完成對歷史人物的道德否定。而王世貞的翻案則是冷靜的戰略分析——他不從道德角度評價藺相如（忠不忠、勇不勇），也不從情感角度評價事件（悲壯或喜劇），而是像一個軍事分析師一樣，從利害關係、風險評估、決策邏輯三個角度檢視藺相如的行為。這種「策士思維」使他的翻案文章顯得獨樹一幟。

王世貞之所以選擇翻藺相如的案，與他的個人經歷密切相關。他的父親王忬被嚴嵩陷害致死，他在哀求嚴嵩的過程中，深刻體會到「假設錯誤」的致命後果——你假設對手會講道理，但對手根本不講道理。當他用這個目光重新審視藺相如的故事時，他看到的不是智勇雙全的成功典範，而是一個危險的邏輯漏洞：藺相如的所有計劃都建立在「秦國會講道理」這個假設上——但萬一秦國不講道理呢？

然而，「不服他議論，服他文章」——這句《古文觀止》評點者的話，恰恰說明了這篇文章的獨特魅力。你可以不同意王世貞對藺相如的批評（畢竟歷史上的秦王沒有翻臉，藺相如賭贏了），但你無法否認他的邏輯力量和文采。翻案文章的最高境界，不是讓讀者覺得「原來我以前的看法全錯了」，而是讓讀者覺得「我原來以為這件事很簡單，現在發現它比我想像的複雜」。王世貞的文章做到了這一點——讀完後你可能仍然不同意他，但你再也無法以原來的眼光看藺相如的故事了。

〈藺相如完璧歸趙論〉的最深層價值，在於它提醒我們：在任何成功的背後，都有我們看不到的偶然因素。藺相如的成功中有運氣，王世貞的成功中也有運氣——意識到這一點，本身就是一種謙卑的智慧。而那些真正值得讚美的事，不是賭博式的冒險成功，而是在艱難處境中表現出來的戰略眼光與人格力量——正如藺相如後來在澠池會上的「勁」與對廉頗的「柔」，那才是「愈出而愈妙於用」的真智慧。

## 2. 寫作技巧分析

### 2.1 修辭手法分析

| 修辭手法 | 原文例證 | 作用與效果 |
|:---------|----------|:-----------|
| **開門見山** | 「藺相如之完璧，人皆稱之，予未敢以為信也。」 | 十個字完成翻案宣示——「人皆稱之」先順應傳統，「予未敢以為信也」再推翻所有讚美。這種「先順後逆」的開局極具戲劇性 |
| **對比** | 「趙得其情則弗予，不得其情則予；得其情而畏之則予，得其情而弗畏之則弗予」 | 一組對稱的二分法推演——「得」與「不得」→「畏」與「弗畏」→「予」與「弗予」，形成四個可能選項，展示作者面對複雜問題時的清晰思路 |
| **對比** | 「兩無所曲直也」vs.「曲在秦」vs.「曲在趙」 | 三種情況的曲直歸屬對比——趙不給→雙方無曲直；秦不給城→秦理虧；趙偷回璧→趙理虧。通過對比凸顯藺相如選擇了最糟的方案 |
| **對比** | 「完璧」（運氣）vs.「勁澠池、柔廉頗」（真智慧） | 全文最重要的對比：同一人物、兩件事，一個是「天也」（運氣），另一個是「愈出而愈妙於用」（智慧）。對比愈強烈，翻案的批評就愈有分寸感 |
| **反問（激問）** | 「奈之何既畏而復挑其怒也？」 | 對藺相如策略矛盾的核心反問——「既畏」（給了璧＝表示怕秦）又「挑其怒」（偷回璧＝挑釁秦國），一個「奈之何」道盡了策略的荒謬 |
| **反問（激問）** | 「今奈何使舍人懷而逃之，而歸直於秦？」 | 在代擬對策之後的致命反問——「奈何」二字是對藺相如決策的終極質疑：明明有更好的方法，為什麼選擇最糟的？ |
| **設問（代擬對策）** | 「相如則前請曰：『臣固知大王之弗予城也。……大王弗予城而紿趙璧，以一璧故而失信於天下，臣請就死於國，以明大王之失信。』」 | 這是全文最精彩的修辭創新——王世貞代替藺相如構思了一段「理想台詞」，讓他當著秦王的面說。如果藺相如真的這樣說，說不定秦王會歸還璧玉。這種「代擬」手法既展示了作者的思路，又讓讀者自行比較，比直接批判更有說服力 |
| **假設論證（反事實推理）** | 「令秦王怒而僇相如於市，武安君十萬眾壓邯鄲，而責璧與信，一勝而相如族，再勝而璧終入秦矣。」 | 全文最震撼的修辭手法——用「令……則……」構建一個恐怖假設場景，讓讀者親眼看到藺相如失敗時的血腥後果。「僇於市」「十萬眾」「壓邯鄲」「族」「璧終入秦」——每一個詞都是鋒利的刀 |
| **層遞** | 「一勝而相如族，再勝而璧終入秦矣」 | 從「一勝」到「再勝」，從「族」到「璧終入秦」——秦軍勝利一次就滅藺相如全族，勝利兩次就連璧也保不住了。層層推進，步步緊逼 |
| **層遞** | 「勁澠池，柔廉頗，則愈出而愈妙於用」 | 從「勁」到「柔」，從「澠池」到「廉頗」——時間上由遠到近，策略上由剛到柔，最後用「愈出而愈妙於用」作總結，形成一個層進序列 |
| **引用（用典）** | 「武安君十萬眾壓邯鄲」 | 「武安君」即白起，長平之戰坑殺趙軍四十萬人。引用這個名字本身就具有極強的恐怖暗示——讀者一看到「武安君」，立刻想到白起的戰績，恐懼感油然而生 |
| **比喻** | 「此兩言決耳」 | 將複雜的外交難題比喻為「兩句話可以解決的事」——極簡的意象，凸顯藺相如把簡單事情複雜化的荒謬 |
| **映襯** | 藺相如「完璧歸趙」的成功形象 vs. 王世貞假設中「僇於市」「族滅」的失敗結局 | 成功的光輝與失敗的恐怖形成強烈映襯——成功愈光輝，失敗的恐怖就愈令人心驚。映襯的效果是：讀者在讀完後再也無法以原來的眼光看藺相如的故事 |
| **映襯** | 「完璧」的風險冒進 vs. 「勁澠池、柔廉頗」的從容智慧 | 藺相如同一人、不同時間的兩次表現形成映襯——早期的冒險賭博與後期的老練成熟，映襯出完璧歸趙事件的不成熟 |
| **諷刺** | 「既畏而復挑其怒」 | 六個字精準概括了藺相如策略的內在矛盾——「既畏」和「復挑其怒」在邏輯上是完全對立的，王世貞將這兩個對立面並置，諷刺意味十足 |
| **諷刺** | 「歸直於秦」 | 三字濃縮了翻案的核心指控——藺相如本應讓秦國理虧，結果反而讓秦國佔了道理。「歸」字帶有「白送」的意味 |
| **一字定論** | 「天也」 | 全文最有力的兩個字——不是「智也」「勇也」，而是「天也」。用一個「天」字將藺相如的成功降格為不可控的命運，而非個人的能力。一字定讞，無可辯駁 |
| **先揚後抑** | 「若其勁澠池，柔廉頗，則愈出而愈妙於用」 | 在全文「天也」的判決之後，用「若其」轉折，對藺相如後來表現做出公允的肯定。這種「先抑後揚」或「先批判後肯定」的策略，讓前面的批評不致顯得刻薄偏激 |

## 3. 佳句賞析

### 「藺相如之完璧，人皆稱之，予未敢以為信也。」

**白話翻譯：** 藺相如完璧歸趙這件事，人人都稱讚他，我卻不敢認為這是真的（正確的）。

**賞析：** 這是中國翻案文章最著名的開篇之一，十個字完成了翻案的全部宣示。其結構可分為兩層：

- **「人皆稱之」**——先承認傳統觀點，表示作者知道主流意見是什麼。這既是對讀者期待的順應，也是為後面的翻案做鋪墊。
- **「予未敢以為信也」**——「予」字帶出作者的主體性，「未敢」二字不是「不信」，而是「不敢信」——語氣上留了餘地，不顯得傲慢武斷，但立場極其堅定。

這十個字的精妙在於：王世貞沒有說「人皆稱之，我獨不然」（那會顯得對立），而是用「未敢以為信」——這種語氣更像是在表達一種審慎的懷疑，而非激烈的反對。但實際上，「未敢以為信」的否定力量是巨大的——因為「人皆稱之」的歷史定論如果連「信」都不能信，那它還有什麼價值？

### 「夫秦以十五城之空名，詐趙而脅其璧。是時言取璧者情也，非欲以窺趙也。」

**白話翻譯：** 秦國用十五座城池的空頭名義，欺騙趙國、脅迫趙國交出和氏璧。那個時候，秦國說要取璧是它的真實意圖，並不是想要藉機窺伺趙國。

**賞析：** 這兩句是翻案的奠基之作——王世貞做的第一件事不是批評藺相如，而是**重新定義秦國的動機**。這個定義看似輕描淡寫，實則是全文論證的基石：

- **「空名」二字**是全句的關鍵——王世貞承認秦國用十五城交換璧玉是假的（空名），但立刻轉向：「是時言取璧者情也」（秦國當時確實只是想要璧玉）。這是一個極其精妙的切割：秦國欺騙趙國是真的（用空名詐趙），但秦國發動這場外交攻勢的目的只是取璧、不是攻趙（非欲以窺趙）。

- 為什麼這個切割重要？因為傳統觀點認為藺相如的勇氣在於「不畏強秦」，但如果秦國本來就沒有要攻打趙國，那藺相如的「不畏」就毫無風險可言——他不是在虎口拔牙，而是在一個安全的環境中表演勇敢。王世貞從動機層面徹底顛覆了整個故事的風險評估。

- 這個句子的位置也極具策略性：它位於「起」之後的第一個論證段落，直接為後面的二分法推理（予或弗予）鋪路——既然秦國只是想要璧，那麼趙國的選擇就很簡單：給或不給。這是全文論證的地基，地基一旦奠定，後面的全部推理都無可撼動。

### 「此兩言決耳，奈之何既畏而復挑其怒也？」

**白話翻譯：** 這不過是兩句話就能決定的事，為什麼既害怕秦國，卻又要挑釁它的怒火呢？

**賞析：** 這是全文批評的關鍵句，包含了兩個層次的修辭設計：

1. **「此兩言決耳」的極簡化處理**：王世貞將一個看似複雜的外交難題簡化為「一句話可以解決」——要麼給、要麼不給。這種極簡化或許不完全符合歷史現實，但作為翻案修辭，它的效果是震撼的：讀者會不自覺地接受這個框架，然後發現藺相如的做法確實超出了這個框架。

2. **「既畏而復挑其怒」的內在矛盾揭示**：「既畏」和「挑其怒」在邏輯上是完全矛盾的——如果你害怕一個人，你會避免挑釁他；如果你挑釁他，說明你不害怕他。藺相如的選擇是「既給璧（表示畏）又把璧偷回來（表示挑釁）」——這就像你對一個人又鞠躬又吐口水，邏輯上完全說不通。

王世貞用一個「奈之何」（為什麼要這樣做）的反問，將這個矛盾赤裸裸地擺在讀者面前。這個反問是全文最鋒利的一刀——它不給藺相如任何辯解的餘地。

### 「欲使曲在秦，則莫如棄璧；畏棄璧，則莫如弗予。」

**白話翻譯：** 想要讓秦國理虧，不如乾脆拋棄璧玉；如果捨不得拋棄璧玉，不如一開始就不要給。

**賞析：** 這是全文論證中最精煉的戰略推理句，濃縮了王世貞對藺相如困局的深刻洞察：

- **「欲使曲在秦，則莫如棄璧」**——這是上策。王世貞的意思是：如果你真的在乎「道理」歸誰，最簡單的辦法就是：你把璧玉交給秦國，然後看秦國給不給城。秦國不給城，天下人都知道秦國理虧。但你捨不得璧玉？那就退而求其次。

- **「畏棄璧，則莫如弗予」**——這是中策。捨不得璧玉，那就乾脆不要給，也不要去秦國。雙方都不給，誰也不理虧。雖然拿不到城，但至少保留了尊嚴。

- 這兩句話背後隱藏著王世貞對藺相如的核心批評：**你既想要璧玉（不捨棄璧），又想要道德優勢（讓秦理虧），又想要城（完成使命）——你什麼都想要，最後什麼都得不到。** 真正的戰略家懂得取捨，而藺相如的錯誤就在於他不願意做取捨。

- 句子本身的修辭也極為精妙：十四個字，一組對稱的假言推理（「欲……則……；畏……則……」），節奏明快，邏輯清晰，像一把鋒利的手術刀，一刀切開問題的本質。

### 「今奈何使舍人懷而逃之，而歸直於秦？」

**白話翻譯：** 現在為什麼卻讓隨從懷揣璧玉從小路逃走，反而把道理歸給了秦國呢？

**賞析：** 這是王世貞在代擬對策之後的致命反問。在論證的結構上，它處於「代擬對策」之後——王世貞先展示了一條更好的路徑（如果藺相如當面說理……），然後反問「你為什麼不走這條路？」

- **「歸直於秦」** 三字是全文的關鍵判詞之一——「歸」是「歸類、歸屬」，帶有「白送」「拱手讓給」的意味。王世貞的意思是：藺相如明明可以讓秦國理虧，卻反過來讓秦國佔了理。這是一個策略上的巨大失誤。

- 「使舍人懷而逃之」——用一個具體的動作細節（懷揣璧玉、走小路）來描述藺相如的「偷回」行為，讓讀者看到這一決定的荒謬性。

整句話的結構是：先提出「更好方案」（代擬對策），再用「今奈何」反問對比實際做法的不足——這種先立標準、再批評的寫法，比直接批判更有說服力。

### 「令秦王怒而僇相如於市，武安君十萬眾壓邯鄲，而責璧與信，一勝而相如族，再勝而璧終入秦矣。」

**白話翻譯：** 假如秦王發怒，在街市上殺死藺相如，派武安君白起帶領十萬大軍壓向邯鄲，責問璧玉和信義的事——打一次勝仗，藺相如全族被滅；打兩次勝仗，璧玉最終還是歸了秦國。

**賞析：** 這是全文最高潮的部分，也是中國翻案文章中最經典的「反事實推理」案例。其論證力量來自三個方面：

1. **「令」字建立的虛擬場景**：用「令」（假如）開頭，將讀者帶入一個假設世界——這個假設在歷史上是完全可能發生的。王世貞不是在做空想，而是在提醒讀者：當時的秦國完全有能力、也有理由這樣做。

2. **人物的恐怖暗示**：「武安君」三字本身就攜帶著極強的情感力量——白起在長平之戰坑殺四十萬趙軍，是趙國人心中的惡魔。當王世貞寫出「武安君十萬眾壓邯鄲」時，明代讀者（以及任何知道這段歷史的讀者）都會立刻感受到那種恐懼。

3. **「一勝……再勝……」的層層推進**：從「一勝」到「再勝」，從「相如族」到「璧終入秦」——失敗的後果被層層剝開。最恐怖的是「璧終入秦」：你保護璧玉的初衷，到頭來是最終的失敗——你努力保護的東西，最終還是落入敵人之手，而且付出了全族被滅的代價。

這段話的修辭力量在於：它不是空洞的假設，而是建立在歷史真實性之上的「可能性分析」。讀者讀完這段話，會不由自主地倒吸一口涼氣——原來藺相如離災難只有「秦王發怒」這一步之遙。

### 「吾故曰：藺相如之獲全於璧也，天也。」

**白話翻譯：** 所以我說：藺相如能夠保全璧玉，是上天保佑啊。

**賞析：** 這是全文的核心結論，也是王世貞對藺相如完璧歸趙事件的最終判決。兩個字——「天也」——定下了全文的基調：

- **「天」字的雙重涵義**：在這裡，「天」不是「天意」「天道」的抽象概念，而是一種「不可控的偶然性」——即「運氣」。王世貞不是在說上天的安排，而是在說藺相如的成功是一種僥倖。這個「天」字與他的文章最核心的假設論證（如果秦王翻臉）形成呼應：因為秦王沒有翻臉，所以是「天」——如果秦王翻臉了，那就是「禍」了。

- 用「天也」來取代傳統的「智也」「勇也」，是王世貞翻案文章的最終落點。全文論證的層層推進，最終都指向這兩個字——它不是一個空喊的口號，而是建立在「論情」「析理」「假設」三層論證之上的必然結論。

### 「若其勁澠池，柔廉頗，則愈出而愈妙於用。所以能完趙者，天固曲全之哉！」

**白話翻譯：** 至於他在澠池會上的強硬態度、對廉頗的謙和退讓，那是越往後越運用得巧妙。他之所以能保全趙國，實在是上天在曲折地成全他啊！

**賞析：** 這是全文的收尾，也是王世貞翻案文章能夠超越一般翻案文章的高明之處：

- **「若其」轉折，從批判到肯定**：用「若其」二字，從前文的嚴厲批評轉向公允的肯定——這是翻案文章中最難做到的一點：如何在推翻一個定論之後，不顯得偏激？王世貞用「若其」這個輕巧的轉折完成了這個任務。

- **「勁」與「柔」的對照**：用「勁」來描述澠池會上的強硬（正面對抗秦王），用「柔」來描述對廉頗的態度（負荊請罪的退讓）——一剛一柔，展示了藺相如策略上的成熟。「愈出而愈妙於用」五字，既是對藺相如的肯定，也是對完璧歸趙事件的間接批評：後來的表現愈精彩，之前的做法就愈顯得是運氣。

- **「天固曲全之哉」的收束**：最後一句回到「天」的主題，但語氣與前面的「天也」不同：「固曲全之哉」——上天真是曲折地成全他啊！「曲」字非常精妙：天的成全不是直接給予，而是通過一系列曲折的事件（完璧歸趙的僥倖成功、澠池會上的出色表現、與廉頗的和好）最終保全了趙國。這既是對藺相如最終成就的肯定，也是對「天」這個主題的加深。

## 4. 白話改編

### 原文
> 藺相如之完璧，人皆稱之，予未敢以為信也。

**白話：** 藺相如完璧歸趙這件事，人人都稱讚他，我卻不敢苟同。

### 原文
> 夫秦以十五城之空名，詐趙而脅其璧。是時言取璧者情也，非欲以窺趙也。趙得其情則弗予，不得其情則予；得其情而畏之則予，得其情而弗畏之則弗予。此兩言決耳，奈之何既畏而復挑其怒也？

**白話：** 秦國打著十五座城池的幌子，欺騙趙國、脅迫趙國交出和氏璧。那個時候，秦國的真正意圖不過是想要璧玉罷了，並不是想藉機攻打趙國。趙國如果看穿了秦國的意圖，就可以不給璧；如果沒看穿，就會給璧。看穿了卻害怕秦國，那就給璧；看穿了但不怕秦國，那就不給璧。說來說去，就這兩種選擇而已——兩句話就能決定的事。為什麼藺相如既表現得害怕秦國（把璧送了過去），卻又去挑釁秦國的怒火（把璧偷回來）呢？

### 原文
> 且夫秦欲璧，趙弗予璧，兩無所曲直也。入璧而秦弗予城，曲在秦；秦出城而璧歸，曲在趙。欲使曲在秦，則莫如棄璧；畏棄璧，則莫如弗予。夫秦王既按圖以予城，又設九賓，齋而受璧，其勢不得不予城。璧入而城弗予，相如則前請曰：「臣固知大王之弗予城也。夫璧非趙璧乎？而十五城秦寶也。今使大王以璧故而亡其十五城，十五城之子弟，皆厚怨大王以棄我如草芥也。大王弗予城而紿趙璧，以一璧故而失信於天下，臣請就死於國，以明大王之失信。」秦王未必不返璧也。今奈何使舍人懷而逃之，而歸直於秦？

**白話：** 再說，秦國想要璧，趙國不給璧，雙方都沒有道理虧欠的問題。但如果趙國把璧送進秦國而秦國不給城，那就是秦國理虧；如果秦國已經表示要給城了，趙國卻把璧偷回去，那就是趙國理虧了。要想讓秦國理虧，最好的辦法就是乾脆把璧丟給秦國（然後看秦國給不給城）；如果捨不得把璧丟掉，那就一開始就不要給。

秦王既然已經指著地圖說要把城給趙國，又安排了隆重的九賓之禮，齋戒沐浴後接受璧玉，在這種情況下他已經騎虎難下，不能不給城了。如果璧玉進獻後秦王不給城，藺相如完全可以走上前去說：「臣早就知道大王不會給城。這塊璧難道不是趙國的璧嗎？那十五座城可是秦國的寶地啊。如今大王為了這塊璧而丟掉十五座城，十五座城的百姓都會深深地怨恨大王把他們當作草芥一樣拋棄。大王不給城卻騙走了趙國的璧，為了一塊璧而失信於天下，臣請求就在這裡就死，好讓天下人都知道大王是失信之人。」——這樣一說，秦王未必就不歸還璧玉。

可藺相如做了什麼呢？他竟然讓隨從把璧玉藏在懷裡從小路偷偷逃走，反而讓秦國佔了道理！這不是荒唐嗎？

### 原文
> 是時秦意未欲與趙絕耳。令秦王怒而僇相如於市，武安君十萬眾壓邯鄲，而責璧與信，一勝而相如族，再勝而璧終入秦矣。吾故曰：藺相如之獲全於璧也，天也。

**白話：** 說到底，當時只是秦國還不想跟趙國徹底翻臉罷了。假如秦王當真發怒，把藺相如在街市上處死，再派武安君白起率領十萬大軍直逼邯鄲，質問趙國為什麼不守信用——秦軍打一次勝仗，藺相如全族被滅；打兩次勝仗，那塊璧終究還是會落入秦國之手。所以我說：藺相如能夠保全璧玉平安歸來，不是因為他有多高明，純粹是運氣好——老天保佑啊！

### 原文
> 若其勁澠池，柔廉頗，則愈出而愈妙於用。所以能完趙者，天固曲全之哉！

**白話：** 至於他後來在澠池會上對秦王的強硬，以及對廉頗的謙和退讓，那才真的是越往後越運用得巧妙、越成熟老練。他之所以能夠最終保全趙國，實在是老天爺用這樣曲折的方式在成全他啊！
