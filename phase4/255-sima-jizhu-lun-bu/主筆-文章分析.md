# 〈司馬季主論卜〉— 文章分析

> 劉基（1311–1375），字伯溫，號郁離子
>
> 本檔案由 @Hermes Agent 主筆

## 一、文章結構分析

〈司馬季主論卜〉全文約四百字，採用對話體形式，結構可分為三個層次：東陵侯發問 → 司馬季主初答 → 司馬季主深入論述。文章以問答推進，從具體的遭際困惑，到哲學層次的覺悟，層層深入，篇幅極短卻意蘊極豐。

### 起承轉合

| 結構 | 對話者 | 內容 | 功能 |
|:----:|:------:|------|------|
| **起** | 東陵侯發問 | 「久臥者思起，久蟄者思啟……蓄極則泄，閟極則達，熱極則風，壅極則通。一冬一春，靡屈不伸；一起一伏，無往不復。」 | 提出問題——東陵侯以物極必反的道理自我安慰，但他仍然無法安於現狀，於是想問卜求一個明確的答案 |
| **承** | 司馬季主初答 | 「若是，則君侯已喻之矣，又何卜為？」 | 點醒對方——你已經懂了物極必反的道理，為什麼還要問卜？這是全文的「關鍵一問」，直接將東陵侯的矛盾暴露出來 |
| **轉** | 司馬季主深入論述 | 長篇對比論證——碎瓦頹垣 vs 歌樓舞館、荒榛斷梗 vs 瓊蕤玉樹……六組對比鋪陳盛衰之理 → 「天道何親？惟德之親；鬼神何靈？因人而靈」→ 「夫蓍，枯草也；龜，枯骨也——物也。人，靈於物者也，何不自聽而聽於物乎？」 | 全文核心——司馬季主不再回答占卜的問題，而是直接否定占卜本身：你不應該問卜，你應該問自己 |
| **合** | 歸結於人事自修 | 「君侯亦知之矣」——東陵侯的困惑被消解了，但答案不是來自占卜，而是來自他內心本來就懂的道理 | 全文收束——真正的答案不在卜卦中，而在每個人的自覺中 |

### 分段解析

**第一層：東陵侯發問——從「物極必反」到「問卜求解」**

> 「久臥者思起，久蟄者思啟，久懣者思嚏。吾聞之：蓄極則泄，閟極則達，熱極則風，壅極則通。一冬一春，靡屈不伸；一起一伏，無往不復。僕竊有疑，願受教焉。」

東陵侯的開場白本身就是一段精彩的議論。他一口氣用了七組「極則反」的排比句式——久臥思起、久蟄思啟、久懣思嚏、蓄極則泄、閟極則達、熱極則風、壅極則通——全部指向同一個道理：事物發展到了極點，必然向對立面轉化。

這七組排比涵蓋了人體感受（臥→起、蟄→啟、懣→嚏）、物理現象（蓄→泄、閟→達、熱→風、壅→通），再配合「一冬一春，靡屈不伸；一起一伏，無往不復」的季節與宇宙週期，說明了東陵侯對「物極必反」這個道理的理解非常透徹。

但他說「僕竊有疑，願受教焉」——他已經懂了，卻仍然有疑、仍然要問卜。這是一個微妙的心理矛盾：**一個人不可能不知道答案的情況下，仍然無法接受答案。** 這是全文的起點，也是全篇最根本的張力所在。

**第二層：司馬季主的精準回擊——「君侯已喻之矣」**

> 「若是，則君侯已喻之矣，又何卜為？」

司馬季主的回答只有一句話，卻是一記精準的棒喝。東陵侯長篇大論講了七個「極則反」的道理，司馬季主只用了十一個字就點破了東陵侯的矛盾：「你已經懂了，為什麼還要來問？」——這句話的潛臺詞是：你不是不知道答案，你是不願意接受這個答案應用在自己身上。

這一句既完成了「承」的功能（承接上文），又完成了「轉」的準備（為後文的長篇論述打開缺口）。東陵侯被點破之後，自然會追問：「既然如此，那我該怎麼辦？」——於是司馬季主便有了進行長篇論述的空間。

**第三層：司馬季主的長篇論述——全文核心**

這一段是全篇文章最精采的部分，也是劉基寫作功力的集中體現。司馬季主的論述可分為三個層次：

**（一）以盛衰對比，點破時間的真相**

> 「君侯亦知之矣，而猶有疑焉，何也？今夫碎瓦頹垣，昔日之歌樓舞館也；荒榛斷梗，昔日之瓊蕤玉樹也；露蠶風蟬，昔日之鳳笙龍笛也；鬼燐螢火，昔日之金缸華燭也；秋荼春薺，昔日之象白駝峰也；丹楓白荻，昔日之蜀錦齊紈也。」

六組對比排比，每一組都是一個強烈的視覺衝擊。「今」與「昔」並列——在時間的河流中，今日的廢墟就是昨日的輝煌。這六組對比讓抽象的「盛衰無常」化為具體可感的畫面：讀者彷彿看到了一座曾經歌舞昇平的樓閣，如今只剩下碎瓦頹垣；曾經高聳入雲的瓊樹玉枝，如今成了荒榛斷梗。

這段排比之所以震撼，不僅因為意象的豐富，更因為**不需要論證，只需要看見**。劉基沒有說「這個世界是變化的」，而是直接把變化的結果展現在讀者面前——讓讀者自己得出結論。

**（二）從物轉到人，揭開占卜的荒謬**

> 「夫蓍，枯草也；龜，枯骨也——物也。人，靈於物者也，何不自聽而聽於物乎？」

這一段是全文最具「理性主義」色彩的論證。劉基將占卜所用的蓍草和龜甲拉下神壇：蓍草不過是枯草，龜甲不過是枯骨——它們只是「物」。而「人，靈於物者也」——人比物更有靈性。那麼問題來了：為什麼一個比物更有靈性的人，要去聽一個「物」的指示？

這段話的邏輯非常簡單，卻有著無可辯駁的力量——它不需要複雜的哲學論證，只需要一個最樸素的常識判斷：人比物高級，所以人不應該聽命於物。

**（三）給出正面答案——天道與鬼神的真相**

> 「天道何親？惟德之親。鬼神何靈？因人而靈。」

這是全文最著名的句子，也是核心的哲學命題。劉基以兩個反問和兩個回答，完成了對「天人感應」的翻轉：天道並不偏愛誰，它只親近有德之人；鬼神的「靈」不是鬼神本身的屬性，而是人賦予的。結論是——**人是一切價值的來源，占卜只是人自我逃避的藉口。**

最後司馬季主說：「君侯亦知之矣」——回到開頭的對話，形成一個完整閉環。東陵侯其實什麼都知道，他所需要的不是占卜的答案，而是有人告訴他「你不需要占卜」。

## 二、寫作技巧分析

### 2.1 對話體的「層次落差」

全文以對話展開，但兩人的話語形成了鮮明的智慧層次落差。這種落差本身就是一種文學效果：

| 層次 | 東陵侯 | 司馬季主 |
|:----:|:-------|:---------|
| 認知的起點 | 知道物極必反，但無法接受 | 知道物極必反，且能坦然面對 |
| 對占卜的態度 | 依賴占卜求解 | 否定占卜——不需要問，只需要信自己 |
| 最終的狀態 | 從「不信自己的判斷」到「被點醒」 | 從開始就知道答案 |

東陵侯代表的是「常識層次」的道理——物極必反，這是人人都知道的常識。司馬季主代表的是「哲學層次」的覺悟——不只知道物極必反，而且知道不需要占卜來確認這件事。讀者跟隨東陵侯一路追問，最後一起接受司馬季主的當頭棒喝——這種「層次落差」本身就是最強的論證：司馬季主不一定比東陵侯知道的更多，但他比東陵侯更願意接受現實。

### 2.2 六組「今昔對比」的鋪陳藝術

這是全文最為人稱道的段落。六組對比如下：

| 今日（衰敗之景） | 昔日（繁華之景） | 感官層面 |
|:----------------|:----------------|:--------:|
| 碎瓦頹垣 | 歌樓舞館 | 視覺（建築） |
| 荒榛斷梗 | 瓊蕤玉樹 | 視覺（植物） |
| 露蠶風蟬 | 鳳笙龍笛 | 聽覺（聲音） |
| 鬼燐螢火 | 金缸華燭 | 視覺（光亮） |
| 秋荼春薺 | 象白駝峰 | 味覺（飲食） |
| 丹楓白荻 | 蜀錦齊紈 | 視覺（衣飾） |

這六組對比涵蓋了建築、植物、音樂、光亮、飲食、衣飾六個層面——人生在世所能享有的所有美好，無一不在覆滅之列。鋪陳的手法學自《楚辭·卜居》，但劉基將這種技法推向了極致：他不是簡單的排比，而是讓每一組對比都產生一個獨立的畫面——讀者讀完這六組，頭腦中已經浮現了從宮殿到荒草、從音樂到鬼火、從盛宴到野菜的全景式衰亡畫面。

**最精妙的是：這段排比中沒有一個「變」字或「衰」字**——劉基只是把兩個畫面並列在一起，讓讀者自己感受到時間的殘酷。這種「不著一字，盡得風流」的寫法，是中國古典散文的最高境界之一。

### 2.3 「破」與「立」的論證結構

全文的論證可以理解為一個「破→立」的雙層結構：

```
破（否定占卜）：
  蓍，枯草也；龜，枯骨也 → 物也
  人，靈於物者也 → 何不自聽而聽於物？
  → 占卜是荒謬的

立（建立人的自信）：
  天道何親？惟德之親 → 天道不神秘
  鬼神何靈？因人而靈 → 鬼神不權威
  人，靈於物者也 → 人才是根本
  → 人應該相信自己
```

值得注意的是，「破」的部分非常短（三句話），「立」的部分也很節制（兩句話）。劉基沒有花費大量篇幅去論證占卜如何荒謬，也沒有長篇大論地建立一套新的哲學——他只是點到為止，因為他相信真正的道理不需要長篇大論。你懂了就是懂了，不懂再多說也無用。

### 2.4 語言特色

| 特色 | 表現 | 分析 |
|:----|:-----|:-----|
| **極簡極精** | 全文不足四百字，對話僅三輪 | 劉基的筆法極其乾淨：東陵侯的開場白雖然排比多，但每一句都不冗長；司馬季主的回答更是簡潔到極致——初答僅十一字，長篇論述也不過二三百字 |
| **以排比代論證** | 六組今昔對比 | 最精彩的論證不是邏輯推導，而是意象疊加。六幅畫面排開，不需要說「盛衰無常」，讀者已經感受到了 |
| **以反問代直述** | 「何不自聽而聽於物乎？」「天道何親？」「鬼神何靈？」 | 連續的反問比直述更有力量——不是告訴你答案，而是逼你自己去想 |
| **近於口語的對話感** | 「若是，則君侯已喻之矣」；「君侯亦知之矣」 | 對話體的優勢在於自然流暢，劉基的對話非常接近真實的談話語氣，毫無造作之感 |
| **楚辭風韻** | 六組對比的句法模仿《卜居》 | 句式整齊中有變化，前面用「今夫……」領起，後面以「……也」收束，古樸而有韻律感 |

### 2.5 修辭手法一覽

| 手法 | 原文 | 效果 |
|:----|:-----|:-----|
| **排比** | 「久臥者思起，久蟄者思啟，久懣者思嚏」 | 一氣呵成，氣勢充沛；以三種身體感受遞進，讓抽象道理變得可感 |
| **排比** | 「蓄極則泄，閟極則達，熱極則風，壅極則通」 | 四組「極則反」的並列，從物理到人事，涵蓋萬物 |
| **排比+對比** | 六組今昔對比的排比句 | 格式統一（「今夫……，昔日之……也」），每組內含強烈對比（衰 vs 盛），六組並列形成排比——排比中有對比，對比中有排比 |
| **反問** | 「何不自聽而聽於物乎？」 | 以反問代替直述，語氣更強烈，讓讀者自己得出結論 |
| **設問+自答** | 「天道何親？惟德之親。鬼神何靈？因人而靈。」 | 自問自答的形式簡潔有力，每一問一答都是一個完整的論證單元 |
| **對比（宏觀）** | 今日之衰敗 vs 昔日之繁華 | 全文最大的對比框架，以時間為軸，以六組意象為具體內容 |
| **對比（微觀）** | 物（蓍草、龜甲）vs 人（靈於物者） | 從物的低賤反襯人的高貴，論證占卜的荒謬 |
| **比喻** | 蓍草比喻為「枯草」，龜甲比喻為「枯骨」 | 將神聖的占卜工具拉到最低層次——不過是枯草枯骨，破除迷信 |
| **層遞** | 東陵侯說物極必反 → 司馬季主說你懂了 → 司馬季主否定占卜 → 歸結於人的自信 | 層層推進，從具體的困惑到抽象的覺悟，邏輯鏈條完整 |
| **首尾呼應** | 開頭「僕竊有疑，願受教焉」→ 結尾「君侯亦知之矣」 | 東陵侯的「疑」被司馬季主消解了——但不是給了他新的答案，而是告訴他答案他本來就有 |
| **引用（化用）** | 六組對比模仿《楚辭·卜居》句式，司馬季主事蹟取自《史記·日者列傳》 | 讓文章有經典的重量，但又自出機杼 |

## 三、佳句賞析

### 佳句一

> **「天道何親？惟德之親。鬼神何靈？因人而靈。」**

**現代漢語：** 天道親近誰呢？只親近有德行的人。鬼神憑什麼靈驗呢？因為人才能使他們靈驗。

**賞析：** 這是全文最著名的句子，也是中國思想史上關於「天人關係」最精煉的表述之一。劉基用兩個自問自答，完成了對傳統天人感應理論的根本翻轉。

先看句式：兩個「何……？惟……」的平行結構，節奏極為整齊，一問一答之間沒有多餘的字。這種簡潔本身就在傳達一個訊息——道理很簡單，不需要複雜的論證。

再看思想內涵。「天道何親？惟德之親」——這句話直接呼應《尚書·蔡仲之命》的「皇天無親，惟德是輔」。劉基不是否定天道，而是重新定義天道：天道不是一個有意志的、可以透過占卜來討好的存在，它只「親近」有德行的人。換句話說，天道的運作是客觀的，你只能用自己的德行去「感應」它。

「鬼神何靈？因人而靈」——這一句更加激進。鬼神的「靈」不是鬼神本身的屬性，而是人賦予的。這相當於說：你覺得占卜靈驗，不是因為龜甲和蓍草真的通靈，而是因為你相信它靈驗。人是一切靈驗的來源。

兩句話合在一起，構成了一個完整的命題：天與鬼神的權威，都建立在人的德行與信念之上。**失去了人的主體性，天與鬼神不過是虛設。** 這在元末的宗教氛圍中，可謂一記石破天驚的覺醒之語。

### 佳句二

> **「夫蓍，枯草也；龜，枯骨也——物也。人，靈於物者也，何不自聽而聽於物乎？」**

**現代漢語：** 蓍草，不過是枯草；龜甲，不過是枯骨——都是物罷了。人，比物更有靈性，為什麼不聽從自己卻去聽從物呢？

**賞析：** 這是全文最具「理性主義」色彩的句子，被後世反覆引用的名句。劉基的論證邏輯極其簡單卻無可辯駁——蓍草和龜甲都是物理的存在（「物」），而人比物更高級（「靈於物者」），那麼一個更高級的存在為什麼要去聽從一個更低級的存在的指示？

這句話的修辭技巧體現在三個層面：

1. **降級處理**：將占卜工具從「神聖之物」降級為「枯草」「枯骨」——「枯」字的使用至關重要，它暗示了這些東西是沒有生命力的、是死的。死物怎麼可能指引活人？

2. **重複強調**：「蓍，枯草也；龜，枯骨也——物也」——從具體（枯草、枯骨）到抽象（物），層層推進，最後用一個「物也」收束，斬釘截鐵。

3. **反問收束**：「何不自聽而聽於物乎？」——不直接說「你不該聽從物」，而是用反問讓讀者自己得出結論。當讀者自己想到「對啊，我為什麼要聽從枯草枯骨」的時候，占卜的權威就已經在讀者的心中瓦解了。

### 佳句三

> **「今夫碎瓦頹垣，昔日之歌樓舞館也；荒榛斷梗，昔日之瓊蕤玉樹也；露蠶風蟬，昔日之鳳笙龍笛也；鬼燐螢火，昔日之金缸華燭也；秋荼春薺，昔日之象白駝峰也；丹楓白荻，昔日之蜀錦齊紈也。」**

**現代漢語：** 如今這些破碎的瓦片、倒塌的牆壁，是昔日的歌樓舞館啊；荒蕪的荊棘、折斷的枝梗，是昔日的玉樹瓊花啊；沾著露水的蠶蟲、在風中鳴叫的蟬，是昔日鳳笙龍笛的聲音啊；鬼火和螢光，是昔日金燈華燭的光亮啊；秋天的苦菜、春天的薺菜，是昔日象白駝峰的美味啊；紅色的楓葉、白色的蘆荻，是昔日的蜀錦齊紈啊。

**賞析：** 這是全文最華麗、最具視覺衝擊力的段落。六組「今夫……，昔日之……也」的平行句式，每一組都是一個獨立的畫面，合起來構成了一幅從極盛到極衰的全景圖。

這段文字的力量在於「不需要論證，只需要看見」。劉基沒有說「盛衰無常」、「世事變遷」——他直接把兩個畫面並列在一起，讓讀者自己去感受時間的殘酷。當你看到「碎瓦頹垣」就想到它曾經是「歌樓舞館」，當你看到「鬼燐螢火」就想到它曾經是「金缸華燭」——那種滄海桑田的衝擊力，遠勝於任何理論論證。

從修辭角度來看，這六組對比涵蓋了人類生活的六大層面：建築（居住）、植物（觀賞）、音樂（娛樂）、光亮（照明）、飲食（口腹）、衣飾（穿戴）。換句話說，人生在世所能享受的一切，都在時間的流逝中化為了灰燼。這種全面性讓論證變得無懈可擊——不是某一方面的變化，而是全方位的覆滅。

### 佳句四

> **「蓄極則泄，閟極則達，熱極則風，壅極則通。」**

**現代漢語：** 水蓄積太多就會洩出，東西封閉太久就會通達，空氣太熱就會生風，通道堵塞太久就會暢通。

**賞析：** 這是東陵侯開場排比句中最核心的一組。他用了四個「極則反」的物理現象來說明同一個道理：事物發展到了極點，必然向對立面轉化。

這四組對比的特點是：每一組都是一個「因果關係」的壓縮——不是簡單的並列，而是「因為A到了極點，所以B會發生」。這種因果鏈條讓「物極必反」從一個抽象的哲學命題變成了具體可感的物理常識：水太多自然會溢出來，空氣太熱自然會颳風——不需要任何神秘力量介入，這是宇宙內在的規律。

東陵侯用這一組排比證明了自己對自然規律的了解，但他仍然要問卜——這就形成了一種諷刺：他懂得萬物的變化規律，卻不能將這個規律應用在自己身上。這就是人性的矛盾：**知道不等於接受。**

### 佳句五

> **「久臥者思起，久蟄者思啟，久懣者思嚏。」**

**現代漢語：** 躺臥久了就想起來，蟄伏久了就想開啟，鬱悶久了就想打噴嚏。

**賞析：** 這是東陵侯開場的第一組排比句，也是全文的起點。三組「久→思」的句式，從身體感受入手，說明了壓抑久了自然會尋求釋放。這三句沒有物理現象的「因果」邏輯（不像後面的「蓄極則泄」），而是更接近心理層面的描述——「思」是一種欲望，一種本能。

特別值得注意的是「久懣者思嚏」——「懣」是胸中憋悶，「嚏」是打噴嚏。把胸中憋悶與打噴嚏聯繫在一起，是一種極具生活化、甚至有些幽默的表達。劉基用這種接地氣的比喻，讓整篇文章的開場不顯得嚴肅沉重，反而帶著一種輕鬆的機鋒。

「久臥思起」對應東陵侯從侯爵淪為布衣的處境——他已經「久臥」了，他「思起」，他想要重新站起來。但他不知道該怎麼站起來，所以他來問卜。這三句話既是物理描述，也是東陵侯內心狀態的隱喻。

## 四、綜合心得

〈司馬季主論卜〉是《古文觀止》卷十二中最短的文章之一，卻也是思想密度最高的一篇。它表面上是一篇關於占卜的對話，實際上是一篇徹底否定占卜的文章——借占卜者司馬季主的口，說出了一個最不該由占卜者說出的結論：「你不需要占卜。」

### 占卜者的自我顛覆

這篇文章最精彩的地方在於一個悖論：**為占卜辯護的人，卻在否定占卜。** 司馬季主是一個職業占卜者，他的生計來自於為人占卜。但他卻告訴來占卜的東陵侯：「你已經懂了，為什麼還要問卜？」「你不應該聽從枯草枯骨，你應該聽從自己。」

這不是職業的自殺，而是哲學的超脫。劉基筆下的司馬季主不是一個普通的算命先生，而是一個「賢者隱於卜肆」的哲人。他的功能不是預測未來，而是幫助當事人看清楚自己的內心——幫助人回到自身的判斷力，而不是把判斷權交給外物。這才是「占卜」的真正意義：不是告訴你吉凶，而是幫你認識自己。

### 人貴於物的理性精神

「人，靈於物者也，何不自聽而聽於物乎？」——這句話可以作為全文的總綱。劉基的核心命題是「人貴於物」，這個命題在中國思想史上有著清晰的譜系：

- **先秦儒家**：孟子「萬物皆備於我」——人的內在本性是價值的源頭
- **宋代道學**：張載「為天地立心」——人參與宇宙秩序的建立
- **劉基此文**：直截了當地說「人，靈於物者也」——這是最直接、最樸素的表述

值得注意的是，劉基說「何不自聽而聽於物」，這不是現代科學意義上的「理性主義」。他反對的不是天人感應本身，而是**放棄人的主體性、盲目依賴外物的態度**。他仍然相信天道與人事之間存在感應（「天道何親？惟德之親」），只是強調這種感應需要以人的德行作為前提。也就是說，天人感應不是「天說了算」，而是「人做了算」。

### 文學手法與思想力量的統一

〈司馬季主論卜〉之所以成為千古名篇，在於它的文學手法與思想力量達到了完美的統一：六組今昔對比不僅是文學上的排比修辭，更是思想上的「時間教育」——讓讀者直接「看到」盛衰無常。簡潔的對話不僅是文體上的精煉，更是思想上的「棒喝」——司馬季主只用了十一字就點醒了東陵侯。

如果沒有那六組排比，文章的思想可能顯得乾澀；如果沒有「天道何親？惟德之親」的哲學高度，那些排比也只不過是華麗的辭藻。劉基的偉大之處在於，他將最高深的哲學命題，用最樸素的語言、最生動的意象表達了出來——讀者在感受到美的同時，也完成了一次思想的覺醒。

與〈楚辭·卜居〉的對話

劉基此文明顯模仿屈原的〈卜居〉。〈卜居〉寫屈原被放逐後，往見太卜鄭詹尹，請他占卜自己的疑惑。鄭詹尹的回答是「龜策誠不能知此事」——占卜不能回答人生的大問題。劉基將這個故事搬到漢初，用司馬季主代替鄭詹尹，用東陵侯代替屈原，核心命題完全相同：真正重要的問題，占卜回答不了，只能由你自己回答。

但劉基比屈原更進了一步：屈原在〈卜居〉中仍然保留了對占卜的某種敬意——他只是說「這件事占卜回答不了」，但沒有否定占卜本身的價值。而劉基直接說占卜的工具不過是枯草枯骨——他不僅否定了占卜對特定問題的適用性，更否定了占卜本身的權威。這種思想上的激進性，可能與劉基身處元末亂世的經驗有關——在一個「天道」已經失序的時代，唯一可信的只剩下人的判斷力。

### 歷史的回響

〈司馬季主論卜〉選入《古文觀止》後，成為明清以來讀書人最熟悉的文章之一。它的影響不僅在文學層面——為後世對話體論說文提供了典範——更在思想層面。在一個充斥著算命、風水、占卜的文化氛圍中，這篇文章始終是一個清醒的聲音：它提醒每一個人，真正的答案不在於求助於外物，而在於審視自己的內心。

人貴於物——這四個字，是劉基留給後世最珍貴的遺產。

## 五、白話改編

> 以下為〈司馬季主論卜〉全文的白話文改寫，以現代口語完整重述劉基的原作。

---

東陵侯（召平）被廢為平民之後，去拜訪司馬季主，想要請他占卜。

司馬季主問：「君侯想占卜什麼呢？」

東陵侯說：「躺臥久了就想起來，冬眠久了就想出來活動，胸中憋悶久了就想打噴嚏。我聽說：水蓄積太滿就會洩出，東西封閉太久就會通達，空氣太熱就會生風，通道堵塞太久就會暢通。一年之中有冬天有春天，沒有什麼屈曲是不會再伸直的；事物有起有落，沒有去了就不再回來的。我心中有些疑惑，希望能得到您的指教。」

司馬季主說：「如果是這樣，那麼君侯已經明白這個道理了，又何必還要占卜呢？」

東陵侯說：「我還沒有徹底領悟其中的奧妙，希望先生能教導我。」

司馬季主於是說：「唉！天道親近誰呢？只親近有德行的人。鬼神憑什麼靈驗？因人而靈。蓍草不過是枯草，龜甲不過是枯骨——都是物罷了。人比物更有靈性，為什麼不聽從自己，卻去聽從物呢？況且君侯為什麼不想想過去呢？有過去，就一定會有今天。所以現在這些破碎的瓦片、倒塌的牆壁，從前是歌舞昇平的樓閣館舍啊；荒蕪的荊棘、折斷的枝梗，從前是美玉般的花樹啊；沾著露水的蠶、在風中鳴叫的蟬，從前是鳳笙龍笛的聲音啊；鬼火和螢光，從前是金燈華燭的光芒啊；秋天的苦菜、春天的薺菜，從前是象白駝峰之類的美味啊；紅色的楓葉、白色的蘆荻，從前是蜀錦齊紈之類的華服啊。過去之所以不再是今天，這個道理由來已久了。君侯也已經明白這個道理，卻仍然心存疑惑，這是為什麼呢？」

東陵侯沉默不語，於是豁然領悟了。

> **— 白話改寫完 —**
