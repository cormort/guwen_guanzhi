# 〈管仲論〉文章分析

> **主筆分析 · 古文觀止卷十·宋文·蘇洵**

---

〈管仲論〉是北宋蘇洵（1009–1066）翻案史論的代表作，收於《嘉祐集》卷九《權書》。全文約五百餘字，結構極其緊密。蘇洵翻轉了千古以來「管仲為賢相」的歷史定論——他並非否定管仲的功績，而是以「身後安排」為更高標準，指出管仲作為宰相的終極失職：未能在死前為齊國培養足以託國的接班人，導致齊國在他死後陷入長期內亂。

本文之精要在於「翻案而不翻功」——承認管仲的能力與成就，但提出一個更嚴格的評價基準，使讀者無法反駁。以下從「起承轉合」與「修辭手法」兩個角度深入分析。

---

## 1. 文章結構分析

### 起承轉合

| 段落 | 起承轉合 | 對應原文 | 核心功能 | 論證策略 |
|:----:|:--------:|----------|----------|----------|
| **第1段** | **起** | 「管仲相威公，霸諸侯，攘戎狄，終其身，齊國富強，諸侯不敢叛。管仲死，豎刁、易牙、開方用，威公薨於亂，五公子爭立，其禍蔓延，訖簡公，齊無寧歲。」 | 開門見山，製造矛盾懸念——同一個國家，管仲在世時極盛，死後極亂，引發讀者追問：「為什麼？」 | **對比入題**：不發議論，僅陳述事實，用「生前/死後」的極端反差製造閱讀張力 |
| **第2段** | **承** | 「夫功之成，非成於成之日，蓋必有所由起；禍之作，不作於作之日，亦必有所由兆……仲可謂不知本者矣。」 | 層層推進論證：①提出因果律（成功必有遠因，禍亂必有先兆）→ ②將齊亂的責任追溯至管仲 → ③還原管仲臨終對話，逐條反駁其「只言三奸而不薦賢」的失誤 → ④結論：「仲可謂不知本者矣」 | **因果追溯**：從結果（齊亂）回推原因（管仲失職），邏輯鏈條完整。**設身處地**：還原管仲臨終場景，以假設推演（「當是時也，吾意以仲且舉天下之賢者以對……」）強化批判力度 |
| **第3段** | **轉** | 「五伯莫盛於威、文……仲之書，有記其將死論鮑叔、賓胥無之為人，且各疏其短。是其心以為數子者，皆不足以托國。而又逆知其將死，則其書誕謾不足信也。」 | 筆鋒一轉，從兩個方向深化論證：①以晉文公為對比——晉文公死後晉國仍存續百餘年，證明國家不應繫於一人；②引用《管子·戒》中管仲臨死評論鮑叔牙等人的段落，反駁「管仲自己知道該怎麼做」——他知道誰不合適卻不培養合適的人，反而說明他的書不可信 | **正反夾擊**：先用晉國案例從「國家制度面」轉折，再用管仲自己的話從「個人責任面」轉折，形成雙重反駁 |
| **第4段** | **合** | 「吾觀史鰍，以不能進蘧伯玉而退彌子瑕，故有身後之諫；蕭何且死，舉曹參以自代。大臣之用心，固宜如此也。一國以一人興，以一人亡。賢者不悲其身之死，而憂其國之衰。故必復有賢者，而後有以死。彼管仲者，何以死哉？」 | 收束全文，以三個層次完成論證：①正面案例（史鰌死後進諫、蕭何臨死薦曹參）建立「大臣之用心」的標準 → ②提煉核心命題「賢者不悲其身之死，而憂其國之衰」 → ③終極追問「彼管仲者，何以死哉？」 | **例證歸納 → 提煉原則 → 凌厲追問**：最後七字是全篇最強的一擊，將千年的歷史評價一筆翻轉 |

### 起承轉合結構圖解

```
起（第1段）：對比入題 ──→ 承（第2段）：因果追溯、逐層反駁
                                      │
                                      ↓
合（第4段）：歸納原則、凌厲追問 ←── 轉（第3段）：晉國對比、反駁管仲自述
```

### 結構特點

1. **節奏極快**：不到六百字完成「提出矛盾→分析原因→反駁辯解→得出結論」的完整閉環，沒有一句廢話。
2. **層層加壓**：每一段都在前一段的基礎上追加新的論證角度，而非簡單重複。
3. **收尾凌厲**：結尾不拖泥帶水，「彼管仲者，何以死哉？」七字收束全篇，餘音繞梁。

## 2. 寫作技巧分析

### 修辭手法

| 修辭手法 | 原文例證 | 作用與效果 |
|:---------|:----------|:-----------|
| **對比** | 「管仲相威公……終其身，齊國富強」vs.「管仲死……齊無寧歲」
| **對比** | 「齊之治也，吾不曰管仲，而曰鮑叔；及其亂也，吾不曰豎刁、易牙、開方，而曰管仲」 | 將治亂的因果關係重新分配——治歸鮑叔（薦仲），亂歸管仲（不薦人），翻轉了傳統的功過歸屬 |
| **對比** | 管仲（不薦人）vs. 蕭何（薦曹參） | 全文最核心的對比論證。兩位同為名相，臨終表現卻天差地別，使「管仲失職」的結論無可辯駁 |
| **對比** | 齊桓公（死後一亂塗地）vs. 晉文公（死後晉國存續百餘年） | 以晉國為參照系，證明「國家不應繫於一人」——齊國之亡非天命，而是管仲未建立制度性延續 |
| **反問（激問）** | 「彼威公何人也？」 | 以反問貶低齊桓公的識人能力，暗示其昏庸——一個昏君沒有三奸也會有別的小人，問題在於管仲沒有為這樣的昏君準備好「護欄」 |
| **反問（激問）** | 「仲以為威公果能不用三子矣乎？」 | 以反問戳破管仲臨終遺言的蒼白——一句「非人情不可近」根本無法阻止一個昏君重新召回小人 |
| **反問（激問）** | 「仲以為將死之言，可以縶威公之手足邪？」 | 用比喻（「縶威公之手足」）加強反問的力度——臨終遺言像繩子一樣綁住齊桓公的手腳？不可能 |
| **反問（激問）** | 「彼管仲者，何以死哉？」 | **全文最強一擊**。七字終極反問，將全部論證收束為一個無法迴避的問題：你沒有安排好後事，憑什麼安心去死？ |
| **設問** | 「何則？豎刁、易牙、開方三子，彼固亂人國者，顧其用之者，威公也。夫有舜而後知放四凶，有仲尼而後知去少正卯。」 | 自問自答，引導讀者接受蘇洵的因果歸屬——用三奸的不是三奸本人，而是齊桓公；而讓齊桓公得以用三奸的，正是管仲 |
| **設問** | 「夫何患三子者？不言可也。」 | 設問後給出驚人結論：如果管仲推薦了賢人，三奸根本不足為慮，連提都不必提——這反過來證明管仲「只言三奸不薦賢」是本末倒置 |
| **排比** | 「豎刁、易牙、開方用」「豎刁、易牙、開方三子」「豎刁、易牙、開方三子，非人情不可近」 | 三奸之名在文中反覆出現，形成節奏感，加深讀者對三奸的印象，同時強化其與管仲的因果聯繫 |
| **排比** | 「聲不絕於耳，色不絕於目」 | 兩個「不絕於」形成排比，刻畫齊桓公沉溺聲色的形象，為其必然寵信小人的論斷鋪墊 |
| **引用（用典）** | 史鰌「死後進諫」典故 | 以衛國大夫史鰌死後仍以「屍諫」方式盡忠，襯托管仲生前連舉薦賢人都做不到——連死人都比管仲盡責 |
| **引用（用典）** | 蕭何「臨死薦曹參」典故 | 最重要的正面案例。蕭何與曹參原有矛盾，但蕭何不計前嫌、以國為先——相形之下，管仲的格局差距立現 |
| **引用（用典）** | 舜放四凶、仲尼去少正卯 | 借聖人之舉說明：只有聖君才知道如何識別並清除奸佞，齊桓公不是聖君，所以管仲不能指望齊桓公自己處理三奸 |
| **層遞** | 全文四段：對比（現象）→ 因果（分析）→ 反駁（深化）→ 追問（結論） | 論證不斷加壓，說服力逐層增強。每一層都建立在前一層的基礎上，讀者無法在中途反駁 |
| **層遞** | 「一國以一人興，以一人亡。賢者不悲其身之死，而憂其國之衰。故必復有賢者，而後有以死。」 | 從國家現象（一興一亡）→ 賢者心態（不悲身死，憂國之衰）→ 賢者責任（必復有賢者而後可死），三層推進至最終結論 |
| **諷刺（反語）** | 「是其心以為數子者，皆不足以托國。而又逆知其將死，則其書誕謾不足信也。」 | 蘇洵說《管子》書中管仲的言論「誕謾不足信」——這是極具諷刺性的指控，意思是：「如果管仲真像書裡說的那樣思考，那他就是在胡說八道。」這種諷刺直接瓦解了管仲自我辯護的可能性 |
| **諷刺（反語）** | 「仲可謂不知本者矣」 | 表面溫和的批評（「不知本」），實則是極嚴厲的指責——作為宰相卻不知「本」（根本要務在於培養接班人），那他的所有功績都失去了根基 |
| **設喻** | 「仲以為將死之言，可以縶威公之手足邪？」 | 將「臨終遺言」比喻為「捆綁手腳的繩子」——形象地揭示了一句話無法束縛活人的行動，特別是無法束縛一個昏君 |
| **設喻** | 「有仲，則三子者，三匹夫耳。不然，天下豈少三子之徒哉？」 | 將管仲比喻為一道堤壩——有他在，三奸不過是三個普通人（「三匹夫」）；沒有他，天下到處都是這種人。這個比喻將管仲的重要性與其失職的後果直接掛鉤 |
| **誇飾** | 「齊無寧歲」 | 簡潔而強烈的誇飾——齊國從桓公死到簡公被殺（約前643–前481，一百六十餘年間）始終動盪不安，用四字濃縮了百年的混亂 |
| **誇飾** | 「一國以一人興，以一人亡」 | 對齊國命運的高度概括——「一興一亡」皆繫於管仲一人，既是事實陳述也是誇飾，凸顯管仲的關鍵地位與其失職的嚴重性 |
| **頂針** | 「功之成，非成於成之日……禍之作，不作於作之日」 | 前後句以「成之日/作之日」重複呼應，形成迴環往復的節奏感，增強了因果律的哲學意味 |
| **映襯** | 史鰌「死後進諫」vs. 管仲「生前不薦人」 | 以死人的盡忠映襯生人的失職——連已經死去的人都在盡力為國謀劃，而活著的管仲卻沒有做到，反差極為強烈 |
| **映襯** | 「威公聲不絕於耳，色不絕於目」vs.「非三子者，則無以遂其欲」 | 以齊桓公的享樂映襯其對三奸的依賴——越是沉溺聲色，越是離不開迎合他慾望的小人，管仲的話就越不可能被聽從 |

### 論證邏輯圖解

```
前提 A：管仲在世 → 齊國強盛
前提 B：管仲死後 → 齊國大亂
────────────────
問題：為什麼？（懸念建立）

因果律：功成有由起，禍作有由兆
    ↓
推論：齊亂的禍根在管仲生前就已埋下
    ↓
還原現場：管仲臨終只言三奸「非人情不可近」，未薦賢人
    ↓
反駁一：臨終遺言無法束縛昏君（「將死之言，可以縶威公之手足邪？」）
反駁二：有管仲在時三奸不足為慮（「有仲，則三子者，三匹夫耳」）
反駁三：管仲自評鮑叔等人的文字不可信（「其書誕謾不足信也」）
    ↓
正面對比：史鰌死後進諫、蕭何臨死薦曹參
    ↓
結論：管仲失職——「彼管仲者，何以死哉？」
```

### 對比系統

| 對比維度 | 正面（值得效法） | 反面（應當警惕） |
|:---------|:----------------|:----------------|
| **宰相** | 蕭何——臨死薦曹參 | 管仲——只言三奸不薦賢 |
| **諸侯國** | 晉國——文公死後存續百餘年 | 齊國——桓公死後一亂塗地 |
| **標準** | 大臣之用心——憂國之衰，復有賢者 | 管仲之用心——知人不薦人，死而不安 |
| **人物** | 史鰌——死後猶以屍諫盡忠 | 管仲——生前不為身後計 |
| **君主** | 舜（放四凶）、仲尼（去少正卯） | 齊桓公（昏庸，必用小人） |

### 層層翻剝的論證技巧

蘇洵此文最精彩之處不在於提出一個新觀點，而在於**層層翻剝**——像剝洋蔥一樣，每翻一層，管仲的「罪證」就更加清晰：

1. **第一層（現象）**：齊國治亂對比 → 管仲背後有問題
2. **第二層（因果）**：追溯禍根 → 管仲未盡責
3. **第三層（現場）**：還原管仲臨終對話 → 他只說不該用誰，沒說該用誰
4. **第四層（反駁）**：證明「只說三奸」無用 → 必須有人接班
5. **第五層（對比）**：蕭何做得到，管仲做不到 → 不是能力問題，是態度問題
6. **第六層（追問）**：「何以死哉？」→ 全篇最高潮

## 3. 佳句賞析

### 「夫功之成，非成於成之日，蓋必有所由起；禍之作，不作於作之日，亦必有所由兆。」

這是全文的哲學基石。蘇洵提出了一種「歷史因果論」——任何結果都不是突發的，而是長期積累的必然。這為後文將齊亂歸咎於管仲提供了理論依據：齊國的禍亂看似在管仲死後才發生，但實際上在管仲生前就已埋下伏筆（即未培養接班人）。這句話同時也是蘇洵史論方法論的精華——他從不看結果評斷人物，而是追問結果背後的因果鏈條。

### 「彼管仲者，何以死哉？」

七字終極追問，是中國古代散文中最著名的結尾之一。這句話的力量在於：
- **語法上**：以問句收束，強迫讀者自行思考答案——而答案已在文中給出
- **倫理上**：提出了一個道德命題——賢者不應只為自己活著，更要為自己死後的國家負責
- **修辭上**：「何以死哉」四字斬釘截鐵，無可迴避

### 「仲可謂不知本者矣。」

「不知本」三字是蘇洵對管仲最嚴厲的批評——作為宰相，卻不知道治理國家的根本（人才培養）。這句話的巧妙在於：蘇洵沒有說管仲「奸」「惡」「無能」，而是說他「不知本」——這是一種居高臨下的評判，暗示蘇洵自己掌握了更高的政治智慧。

### 「賢者不悲其身之死，而憂其國之衰。故必復有賢者，而後有以死。」

**白話：** 賢德的人不為自己的死亡而悲傷，而是為國家的衰敗而憂慮。所以一定要找到賢能的接班人，然後才可以安心死去。

**賞析：**
這是全文最核心的道德命題，也是蘇洵為「大臣之用心」設定的終極標準。這句話將賢者的責任從個人層面（生死）提升到國家層面（興衰），重新定義了什麼是真正的盡忠——不是在任時做出成績，而是確保自己死後國家仍然能夠運行。蘇洵用了一個三段式的邏輯推進：（1）賢者不悲身死（拋棄個人生死觀）→（2）憂國之衰（以國家為重）→（3）必復有賢者而後可死（以行動落實憂國）。第三句是全文論證的關鍵跳躍——僅有「憂國」還不夠，必須將憂國轉化為實際行動（培養接班人），這正是管仲做不到的。這句話也是〈管仲論〉對儒家「死而後已」精神的批判性補充——真正的賢者，死而不已，死後仍要為國家留下希望。

### 「齊之治也，吾不曰管仲，而曰鮑叔；及其亂也，吾不曰豎刁、易牙、開方，而曰管仲。」

**白話：** 齊國的強盛，我不歸功於管仲，而歸功於鮑叔牙（因為鮑叔牙推薦了管仲）；等到齊國大亂，我不歸咎於豎刁、易牙、開方，而歸咎於管仲（因為管仲沒有推薦賢人）。

**賞析：**
這是蘇洵翻案史論最具顛覆性的一句話，也是全文的「鑰匙句」。他用了一個極其大膽的因果重新分配——把齊國強盛的功勞從管仲轉給鮑叔牙（「吾不曰管仲，而曰鮑叔」），把齊國大亂的責任從三奸轉給管仲（「吾不曰豎刁、易牙、開方，而曰管仲」）。這個對仗句的修辭力量來自兩個層面：一是結構上的對稱——「治」對「亂」、「管仲」對「鮑叔／三奸」；二是邏輯上的翻轉——讀者習慣了「管仲=賢相」的定式，蘇洵卻說齊國的強盛不該歸功於管仲（因為管仲自己也是被推薦的），而齊國的衰敗反而應該歸咎於管仲（因為他沒有複製鮑叔牙的做法）。一句話同時完成兩個翻轉，可謂極簡而凌厲。

## 4. 白話改編

> 管仲輔佐齊桓公，稱霸諸侯，驅逐戎狄，終其一生，齊國富強，諸侯不敢背叛。管仲死後，豎刁、易牙、開方三人被重用，齊桓公死於內亂，五公子爭奪君位，禍亂蔓延，一直到齊簡公，齊國沒有安寧的日子。
>
> 大凡功業的完成，不是完成於成功的那一天，一定有其深遠的起因；禍亂的發生，也不是發生於爆發的那一天，也一定有它的先兆。所以齊國的治理，我不說是管仲的功勞，而說是鮑叔牙的功勞（因為他推薦了管仲）；至於齊國的亂局，我不責怪豎刁、易牙、開方，而要歸咎於管仲（因為他沒有推薦賢人來接班）。
>
> 當年管仲病重的時候，齊桓公問他誰可以接任宰相。管仲說：「了解臣下的，沒有人比得上國君。」齊桓公說：「易牙怎麼樣？」（易牙把自己的兒子煮給桓公吃）管仲說：「殺了自己的兒子來討好國君，這不合人情，不可重用。」齊桓公又問：「開方怎麼樣？」（開方是衛國公子，放棄繼承權來侍奉桓公）管仲說：「背叛自己的父母來討好國君，這不合人情，難以親近。」齊桓公再問：「豎刁怎麼樣？」（豎刁自宮來服侍桓公）管仲說：「閹割自己來討好國君，這不合人情，難以親近。」管仲死後，齊桓公沒有聽從他的話，重用了這三個人，三人於是專權，最終導致了齊國的動亂。
>
> 蘇洵認為：如果管仲在臨終時，不是只說「誰不能用」，而是推薦天下的賢才來回答齊桓公，那麼齊桓公即使昏庸，也會有賢人在朝輔佐，三奸根本不足為患。可是管仲只說三奸「不合人情」，卻沒有推薦一個真正可用的人才。更何況，以齊桓公這樣的君主，區區臨終遺言，怎麼可能約束得了他？管仲可說是不知根本的人啊。
>
> 看看歷史上的正面例子：衛國大夫史鰌，因為生前未能推薦賢人，死後以屍諫的方式完成使命；漢相蕭何臨死之前，推薦與自己有矛盾的曹參接替自己。作為大臣，本來就應該這樣做——不悲傷自己的死亡，而要憂慮國家的衰敗，所以一定要找到賢能的接班人，然後才能安心離世。管仲呢？他憑什麼安心去死？
