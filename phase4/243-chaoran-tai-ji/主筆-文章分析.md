# 〈超然臺記〉— 文章分析

> 蘇軾（1037–1101）
>
> 本檔案由主筆系統生成

## 1. 文章結構分析

〈超然臺記〉全文約四百餘字，是蘇軾「亭臺記」文體中最具哲學深度的一篇。不同於傳統臺記先敘後議的寫法，蘇軾反其道而行——先破題立論，再敘事驗證，最後收束於命名。全文以「樂」字為骨、以「超然」為眼，結構如珠走盤，環環相扣。

### 起承轉合

| 結構 | 段落 | 內容 |
|:----:|------|------|
| **起** | 第一段：破題立論——快樂不需條件 | 開門見山提出核心命題「凡物皆有可觀，苟有可觀，皆有可樂」，以「哺糟啜醨皆可醉、果蔬草木皆可飽」的生活常識論證：快樂不是由外部條件決定的，而是由內心態度決定的。結以反問「吾安往而不樂？」——將問題拋給讀者。 |
| **承** | 第二段：哲學深化——為什麼人會不快樂 | 從反面論證「求樂反而失樂」的道理。提出全文最核心的哲學概念——「遊於物之內」與「遊於物之外」的區分。指出人之所以不快樂，是因為「遊於物之內」——被外物遮蔽，將小事放大如「隙中之觀鬥」。這一段是全文的哲學核心，為後文的「超然」提供了理論基礎。 |
| **轉** | 第三段：敘事驗證——以親身經歷證明超然之樂 | 從抽象議論轉入具體敘事。蘇軾以自己從杭州到密州的巨大落差為例——「釋舟楫之安，而服車馬之勞；去雕牆之美，而庇采椽之居」。眾人皆疑其不樂，但他「貌加豐，髮反黑」——用身體的好轉來證明超然哲學的真實效力。這是全文唯一「敘事」的部分，卻是最具說服力的一筆。 |
| **合** | 第四、五段：登臺見證——快樂當下即是 + 點題收束 | 第四段寫修臺、登臺、遠眺歷史遺跡（姜太公、齊桓公、韓信），以「樂哉遊乎！」四字達到全文情感高潮。第五段引出弟弟蘇轍命名「超然」，並以「蓋遊於物之外也」七字收束全文——完美呼應開頭的「吾安往而不樂」。 |

### 起承轉合結構圖解

```
起（第1段）：快樂不需條件 ──→ 承（第2段）：為何人不快樂（遊於物之內）
                                        │
                                        ↓
合（第4-5段）：以「樂哉遊乎！」收束 ←── 轉（第3段）：親身經歷證明（貌加豐髮反黑）
      │
      └──→ 點題：「蓋遊於物之外也」（閉環）
```

### 分段解析

**第一段：破題——「凡物皆有可觀」**

> 凡物皆有可觀。苟有可觀，皆有可樂，非必怪奇偉麗者也。哺糟啜醨，皆可以醉；果蔬草木，皆可以飽。推此類也，吾安往而不樂？

開篇如平地驚雷，沒有鋪墊、沒有引言，直接拋出全文的核心命題。三句話構成完美的邏輯鏈條：前提（「凡物皆有可觀」）→ 推論（「苟有可觀，皆有可樂」）→ 反思（「非必怪奇偉麗者也」）。這是在完成一次「快樂的民主化」——把快樂從少數擁有「怪奇偉麗」之物的特權階級手中，解放給每一個普通人。

「哺糟啜醨，皆可以醉；果蔬草木，皆可以飽」——十二個字寫盡了蘇軾的人生態度：真正的「醉」不在於喝什麼酒，而在於內心的愉悅；真正的「飽」不在於吃什麼菜，而在於知足的心。這是全文的「宣言」，也是蘇軾貶謫哲學的基石。

**第二段：哲學深化——「遊於物之內」vs「遊於物之外」**

> 夫求禍而辭福，豈人之情也哉？物有以蓋之矣。彼遊於物之內，而不遊於物之外。物非有大小也，自其內而觀之，未有不高且大者也。彼挾其高大以臨我，則我常眩亂反覆，如隙中之觀鬥，又烏知勝負之所在？是以美惡橫生，而憂樂出焉。可不大哀乎！

這一段是全文最深刻的哲學論述，也是最難讀懂的部分。蘇軾提出了三個層層遞進的哲學概念：

1. **「物有以蓋之」——遮蔽**。人不是故意追求災禍，而是被外物遮蔽了雙眼——被困在事物的表象中，看不到更高的真相。

2. **「遊於物之內」vs「遊於物之外」——視角**。這對概念是理解全文的鑰匙。「遊於物之內」是站在事物內部看世界——一件小事被無限放大，壓得人喘不過氣；「遊於物之外」是跳出事物本身、從更高的角度審視——你會發現今天讓你痛苦的事，放在更長的時間維度裡，不過是滄海一粟。

3. **「隙中之觀鬥」——比喻**。這是全段最精彩的比喻——就像從門縫中看人打架，什麼都看不清楚，只能看到兩個模糊的影子在晃動，根本不知道誰贏誰輸。人被困在「物之內」的時候，就是這種狀態：被眼前的得失弄得「眩亂反覆」，卻看不清大局。這正是心理學所謂的「認知扭曲」——當你陷入困境時，視野變得狹窄，把眼前的問題看成全世界。

**第三段：敘事——「貌加豐，髮反黑」的身體證明**

> 余自錢塘移守膠西，釋舟楫之安，而服車馬之勞；去雕牆之美，而庇采椽之居；背湖山之觀，而行桑麻之野。始至之日，歲比不登，盜賊滿野，獄訟充斥；而齋廚索然，日食杞菊。人固疑余之不樂也。處之期年，而貌加豐，髮之白者，日以反黑。余既樂其風俗之淳，而其吏民亦安予之拙也。

這一段是全文中唯一「敘事」的部分，卻是全文最具說服力的論證。蘇軾描寫了從杭州到密州的巨大落差——六個「而」字句形成了一連串的對比：舟楫→車馬、雕牆→采椽、湖山→桑麻。密州的困境更是雪上加霜：災荒、盜賊、獄訟、飢餓——「日食杞菊」四個字寫盡了物質匱乏的極致。

但蘇軾緊接著來了個驚人逆轉：「人固疑余之不樂也。處之期年，而貌加豐，髮之白者，日以反黑。」——你們都以為我肯定不快樂？錯了！我在這裡住了一年，反而長胖了，白頭髮都變黑了！

「貌加豐，髮反黑」這六個字帶有一種「調皮」的自信——是一種用身體變化來證明哲學真理的高級幽默。讀到這裡，讀者會不由自主地笑出來，同時也被深深打動：原來快樂真的可以戰勝環境。

最後「余既樂其風俗之淳，而其吏民亦安予之拙也」——蘇軾用「拙」這個字來自嘲。在一個講究「政績」的時代，他承認自己「笨拙」，但這種「拙」恰恰是一種不折騰、不擾民的治理智慧。

**第四段：登臺——「樂哉遊乎！」的情感高潮**

> 於是治其園圃，潔其庭宇，伐安丘、高密之木，以修補破敗，為苟全之計。而園之北，因城以為臺者舊矣，稍葺而新之。時相與登覽，放意肆志焉。南望馬耳、常山，出沒隱見，若近若遠，庶幾有隱君子乎？而其東則廬山，秦人盧敖之所從遁也。西望穆陵，隱然如城郭，師尚父、齊威公之遺烈，猶有存者。北俯濰水，慨然太息，思淮陰之功，而弔其不終。臺高而安，深而明，夏涼而冬溫。雨雪之朝，風月之夕，余未嘗不在，客未嘗不從。擷園蔬，取池魚，釀秫酒，瀹脫粟而食之，曰：「樂哉遊乎！」

這一段是全文的「敘事高潮」。蘇軾登臺遠眺，所見的不是一般的風景，而是一幅「歷史地圖」：向南望馬耳、常山——彷彿有隱君子；向東看廬山——秦人盧敖隱居之處；向西望穆陵關——姜太公、齊桓公的遺跡；向北俯濰水——韓信的功業與悲劇。

這些歷史人物曾經叱吒風雲：姜太公輔佐周王、齊桓公稱霸諸侯、韓信戰無不勝——但現在呢？只剩下遺跡供人憑弔。這不就是〈凌虛臺記〉的主題嗎？「一切都會消逝」。

但蘇軾在這裡完成了一次對「凌虛臺記」的終極反轉。他筆鋒一轉，寫道：

**「臺高而安，深而明，夏涼而冬溫。雨雪之朝，風月之夕，余未嘗不在，客未嘗不從。」**

這幾句話是對「凌虛臺記」的溫柔反駁：是的，一切都會消逝——姜太公不見了、齊桓公不見了、韓信也不見了。但是！這座臺現在就在這裡，它很安穩、很明亮。無論是雨雪天還是風月夜，我和朋友們都在這裡喝酒、吃菜、聊天。這種快樂是真實的、當下的——它不需要永恆來證明。

最後，蘇軾用一個極具生活氣息的場景將快樂推向高潮：「擷園蔬，取池魚，釀秫酒，瀹脫粟而食之」——摘自己種的菜、撈自己養的魚、喝自己釀的酒、吃自己煮的飯——然後大喊一聲：「樂哉遊乎！」（多麼快樂啊！）

這四個字是全文的情感高峰。它不是抽象的哲學命題，而是活生生的生命體驗。

**第五段：點題——「蓋遊於物之外也」**

> 方是時，余弟子由適在濟南，聞而賦之，且名其臺曰「超然」。以見余之無所往而不樂者，蓋遊於物之外也。

全文的結尾乾淨利落，形成了一個完美的閉環：
- 開頭提問：「吾安往而不樂？」
- 中間論證：「遊於物之內」則苦，「遊於物之外」則樂
- 結尾回答：「無所往而不樂者，蓋遊於物之外也」

「超然」二字的含義在這裡被最清晰地揭示——所謂「超然」，就是「遊於物之外」。而這個名字是弟弟蘇轍取的，兄弟二人在這篇文章中完成了一次精神上的合作：蘇軾建臺、蘇轍命名、蘇軾作文——三人（建臺者、命名者、作文者）共同鑄造了這座精神紀念碑。

## 2. 寫作技巧分析

### 2.1 「先議後敘」的破體寫法

傳統的「臺記」文體通常按照「地點→修建→風景→感慨」的順序書寫。但蘇軾在〈超然臺記〉中大膽突破，採用了「先議後敘」的結構：

| 傳統臺記寫法 | 〈超然臺記〉寫法 |
|:-----------:|:--------------:|
| 先寫在哪裡、誰建的 | **先發議論**：「凡物皆有可觀」 |
| 再寫建築的樣子 | **再論哲學**：「遊於物之內外」 |
| 後發表感慨 | **後敘經過**：從杭州到密州 |
| 最後抒發志向 | **最後登臺**：「樂哉遊乎」 |

這種寫法的好處是：開篇即抓住讀者注意力，以哲學命題代替建築描述，使「臺」從物理存在昇華為精神象徵。蘇軾不是在介紹一座建築——他是在發表一份「快樂宣言」。

### 2.2 正反論證法——從兩面夾擊

全文的論證結構呈現為清晰的正反交替：

```
正面（第1段）：凡物皆有可觀 → 吾安往而不樂？
反面（第2段）：遊於物之內 → 求福反得禍（可不大哀乎！）
正面（第3段）：親身經歷 → 貌加豐、髮反黑
正面（第4段）：登臺 → 樂哉遊乎！
合題（第5段）：超然 → 遊於物之外
```

第一段從正面建立快樂不需要條件的命題；第二段從反面論證「為什麼人會不快樂」（被外物遮蔽）；第三段用親身經歷從正面證明超然哲學的有效性；第四段用登臺的快樂場景再次回到正面。正反交替之間，論證力度不斷加強。

### 2.3 身體可感的論證策略——「貌加豐，髮反黑」

這是蘇軾全文最精彩的論證技巧之一。他不是用抽象的邏輯來證明自己很快樂——他說自己的臉色紅潤了、白頭髮變黑了。

這種「身體化論證」的效果極好：
- **直觀可感**：讀者無法反駁「臉色好轉」這樣的事實
- **幽默自信**：帶著一種「你們想不到吧」的調皮
- **反轉落差**：從「日食杞菊」的艱苦到「貌加豐」的好轉，戲劇性極強

### 2.4 歷史與當下的對照——對〈凌虛臺記〉的超越

第四段登臺遠眺，蘇軾寫了四方歷史遺跡：姜太公、齊桓公、韓信——每一個都是名垂千古的人物。但蘇軾的用意不是感慨歷史，而是襯托當下：

| 歷史人物 | 功業 | 結局 | 蘇軾的態度 |
|:-------:|:----:|:----:|:---------:|
| 姜太公 | 輔佐周王開國 | 功成身退 | 遺跡猶存，人已不在 |
| 齊桓公 | 春秋五霸之首 | 餓死宮中 | 霸業成空 |
| 韓信 | 百戰百勝 | 被殺未央 | 功高震主，不得善終 |

這些曾經改變歷史的人物，如今只剩「穆陵隱然如城郭」「濰水北流」的遺跡。但蘇軾沒有停留在這種悲涼中——他緊接著寫「臺高而安，深而明……雨雪之朝，風月之夕，余未嘗不在，客未嘗不從」——把讀者從歷史的蒼涼中拉回當下的溫暖。這一段的情感節奏控制堪稱完美：先放鬆（寫景）→ 再沉重（憑弔歷史）→ 再溫暖（回到當下）→ 最後歡快（「樂哉遊乎」）。

### 2.5 「樂」字作為貫穿全文的線索

「樂」字在全文出現五次，每一次的層次不同：

| 出現位置 | 原文 | 層次 | 功能 |
|:-------:|:----:|:----:|:----:|
| 第1段 | 「皆有可樂」 | 理論層次 | 提出快樂不需要條件 |
| 第1段 | 「吾安往而不樂」 | 懸念層次 | 以反問引發讀者好奇 |
| 第3段 | 「樂其風俗之淳」 | 實踐層次 | 在密州找到了真快樂 |
| 第4段 | 「樂哉遊乎」 | 情感最高峰 | 在臺上發出快樂的呼喊 |
| 第5段 | 「無所往而不樂」 | 結論層次 | 呼應開頭，完成閉環 |

從理論上的「可樂」，到實踐中的「樂其風俗」，再到情感高潮的「樂哉遊乎」，最後哲學總結的「無所往而不樂」——「樂」字像一條紅線貫穿全文，把抽象哲學、個人經歷、生活場景完美串聯。

### 2.6 修辭手法一覽

| 手法 | 原文 | 效果 |
|:----|:-----|:-----|
| **排比** | 「釋舟楫之安，而服車馬之勞；去雕牆之美，而庇采椽之居；背湖山之觀，而行桑麻之野」 | 三組「A之B，而C之D」的平行結構，節奏整齊，將從杭州到密州的巨大落差一氣呵成地展現 |
| **排比** | 「南望……東則……西望……北俯……」 | 四個方位詞引領的並列句，構建出立體的空間感，將密州四周的歷史人文景觀一網打盡 |
| **對比** | 杭州之樂 vs 密州之苦（第3段） | 以極端對比凸顯蘇軾「超然」的可貴——從天堂到窮鄉，反而更快樂 |
| **對比** | 歷史功業 vs 當下快樂（第4段） | 姜太公、齊桓公、韓信的功業已成往事，但臺上的當下快樂卻是真實的 |
| **反問** | 「吾安往而不樂？」 | 開篇反問，以不容置疑的語氣建立全文基調——到哪裡都快樂，這是肯定的 |
| **反問** | 「豈人之情也哉？」 | 第二段開頭反問——人難道會故意求禍辭福嗎？引導讀者思考不快樂的真正原因 |
| **反問** | 「可不大哀乎！」 | 反問加強嘆——「這難道不是最大的悲哀嗎？」強化對「遊於物之內」的批判 |
| **比喻** | 「如隙中之觀鬥」 | 以「從門縫中看人打架」比喻「遊於物之內」的視角局限，形象生動，一語中的 |
| **頂針** | 「凡物皆有可觀。苟有可觀，皆有可樂」 | 「可觀」重複出現，形成因果鏈條，邏輯嚴密 |
| **頂針** | 「遊於物之內，而不遊於物之外」 | 一正一反的概念重複對比，強化核心哲學概念 |
| **層遞** | 「可觀→可樂→安往不樂」 | 從觀察到快樂再到普遍的快樂，步步推進 |
| **層遞** | 正（第1段）→ 反（第2段）→ 正（第3-4段）→ 合（第5段） | 全文結構的層層推進 |

| **用典** | 姜太公、齊桓公、韓信、盧敖 | 以歷史人物增加文章的厚度和視野 |

## 3. 佳句賞析

### 佳句一

> **「凡物皆有可觀。苟有可觀，皆有可樂，非必怪奇偉麗者也。」**

**白話翻譯：** 任何事物都有值得觀賞的地方。只要有值得觀賞的地方，就有值得快樂的地方——不一定非得是稀奇古怪、宏偉壯麗的東西。

**賞析：** 這是全文最著名的句子，也是蘇軾人生哲學的核心表達。三句話構成一個完整的邏輯三段論：大前提（「凡物皆有可觀」）→ 小前提（「苟有可觀，皆有可樂」）→ 結論（「非必怪奇偉麗者也」）。但它的力量不在於邏輯，而在於態度——蘇軾用極其肯定的語氣告訴我們：快樂是每個人的權利，它不取決於你擁有什麼，而取決於你如何看待你所擁有的。三句話層層遞進，最後一句「非必怪奇偉麗者也」既是反思也是批判——那些以為只有「怪奇偉麗」才值得快樂的人，是蘇軾要糾正的對象。

### 佳句二

> **「哺糟啜醨，皆可以醉；果蔬草木，皆可以飽。」**

**白話翻譯：** 吃酒糟、喝薄酒，都可以讓人醉；水果蔬菜、草木野果，都可以讓人飽。

**賞析：** 這兩句是對「凡物皆有可觀」的具體化。蘇軾用最樸素的生活經驗（喝酒、吃飯）來說明最深刻的哲學道理。這不是「將就」或「湊合」，而是「知足」——真正的「飽」不在於吃什麼山珍海味，而在於知道什麼時候「夠了」；真正的「醉」不在於喝什麼瓊漿玉液，而在於內心的愉悅。蘇軾用這十二個字完成了對「快樂標準」的重新定義：不是把低標準的快樂說成「也還不錯」，而是從根本上認識到——粗茶淡飯自有粗茶淡飯的滋味。

### 佳句三

> **「彼遊於物之內，而不遊於物之外。物非有大小也，自其內而觀之，未有不高且大者也。彼挾其高大以臨我，則我常眩亂反覆，如隙中之觀鬥，又烏知勝負之所在？」**

**白話翻譯：** 那些人被困在事物內部，而不能跳出事物之外來看待問題。事物本身沒有大小之分，但如果你站在事物內部來看它，沒有什麼事物不是高大無比的。當事物挾著它的「高大」來壓迫你時，你就會眼花繚亂、反覆不定，就像從門縫中看人打架，又怎麼知道誰勝誰負呢？

**賞析：** 這是全文最深刻的哲學段落，也是中國古代散文中最精彩的「認知心理學」論述之一。蘇軾用「隙中之觀鬥」這個比喻，精準地描述了「認知扭曲」的心理機制——當你陷入困境時，你的視野會變得狹窄，把眼前的問題放大到不可逾越的程度。一千年前的蘇軾，已經洞察了現代心理學的核心發現：痛苦不是來自於困難本身，而是來自於你如何看待困難。而他的解決方案「遊於物之外」——拉高視角、拓寬視野——到今天仍然是最有效的心理調節策略。

### 佳句四

> **「處之期年，而貌加豐，髮之白者，日以反黑。余既樂其風俗之淳，而其吏民亦安予之拙也。」**

**白話翻譯：** 在這裡住了一年，我的臉色反而紅潤了，頭髮中的白髮也一天天變黑了。我喜歡上了這裡淳樸的民風，而官吏和百姓也習慣了我這個笨拙的官員。

**賞析：** 這六個字（「貌加豐，髮反黑」）是全文最幽默也最有說服力的一筆。蘇軾用身體變化作為快樂的「證據」——這比任何理論論證都更有效。而「吏民亦安予之拙也」更是一句精妙的自嘲：一個知州用「拙」來形容自己——不是真的無能，而是一種「不折騰」的治理智慧。在一個追求「政績工程」的時代，蘇軾用「拙」來對抗「巧」——這種低調的自信，比高調的宣言更有力量。

### 佳句五

> **「臺高而安，深而明，夏涼而冬溫。雨雪之朝，風月之夕，余未嘗不在，客未嘗不從。擷園蔬，取池魚，釀秫酒，瀹脫粟而食之，曰：『樂哉遊乎！』」**

**白話翻譯：** 這座臺又高又穩固，又深又明亮，夏天涼爽冬天溫暖。無論是下雨下雪的早晨，還是颳風月明的夜晚，我沒有不在這裡的，客人們也沒有不跟隨的。摘園子裡的蔬菜，撈池塘裡的魚，釀一點秫酒，煮一點糙米飯來吃，然後大喊一聲：「多麼快樂啊！」

**賞析：** 這是全文情感的最高峰，也是對「凌虛臺記」最溫柔的反駁。前兩句「臺高而安……雨雪之朝……余未嘗不在」寫出了超然臺的日常溫暖——它不高大宏偉，但「夏涼而冬溫」；它不華麗壯觀，但「余未嘗不在，客未嘗不從」。然後一連串簡潔有力的動詞句——「擷」「取」「釀」「瀹」「食」——五個動詞寫出了最樸素也最快樂的生活場景。最後「樂哉遊乎」四字如天外飛來，將全文的情感推向最高點。這四個字不是哲學推論的結果，而是生命實踐的吶喊——這正是蘇軾要告訴我們的：快樂不是想出來的，是活出來的。

### 佳句六

> **「以見余之無所往而不樂者，蓋遊於物之外也。」**

**白話翻譯：** 以此來表明我之所以在任何地方都能快樂，是因為我能夠超脫於事物之外（不為外物所束縛）。

**賞析：** 全文的結尾，也是對開頭「吾安往而不樂」的完美回答。從開頭的提問，到中間的論證、敘事、登臺，到最後的點題——全文形成一個嚴密的邏輯閉環。「蓋遊於物之外也」七個字，既是對「超然」二字的定義，也是全文哲學思想的總結。至此讀者恍然大悟：原來「超然」不是高高在上的姿態，不是逃離現實的消極，而是「遊於物之外」——在塵世中保持內心的從容與自由。

## 4. 綜合心得

〈超然臺記〉是蘇軾人生哲學的「奠基之作」。如果說〈赤壁賦〉是蘇軾貶謫哲學的巔峰，那〈超然臺記〉就是這座高峰的「第一級臺階」——在這裡，三十八歲的蘇軾第一次系統性地提出了他的「快樂哲學」。

這篇文章最動人之處，不在於它的文采（雖然文采確實燦然），而在於它的**真實性**。蘇軾不是在書齋中憑空想出來的哲學——他是真的被貶到了密州這個「齋廚索然，日食杞菊」的窮地方，真的從「夕陽西下」走到了「貌加豐、髮反黑」。這篇文章的每一個字，都有生命的重量。

從思想價值上說，蘇軾完成了一次對莊子哲學的「世俗化改造」。莊子的「逍遙遊」要求人脫離一切社會關係和物質束縛——這對普通人來說幾乎不可能。蘇軾把「逍遙」拉回了人間：你不需要隱居山林，不需要斷絕慾望，你只需要「遊於物之外」。他讓莊子的哲學從「精英的奢侈品」變成了「每個人都可以實踐的方法」。

從文學成就上說，〈超然臺記〉最精彩的技巧是**「以身體證哲學」**。蘇軾不說「我很快樂」，而是說「我長胖了、白頭髮變黑了」——用最直觀的身體變化來證明最抽象的哲學道理。這種寫法讓這篇文章在千百年後仍然能打動人心，因為它訴諸的是每一個人都能體會的生命經驗。

從歷史地位上說，〈超然臺記〉是蘇軾對〈凌虛臺記〉的終極回答。十年前他在〈凌虛臺記〉中說「一切都會消逝」——這是一種智者的清醒，但清醒如果沒有超脫，就會變成虛無。十年後他在〈超然臺記〉中補上了下一塊拼圖：是的，一切都會消逝——但正因為如此，我們才更應該在當下找到快樂。快樂不需要永恆來證明，快樂就是快樂本身。

「超然」不是「不在乎」，而是「不困住」。蘇軾仍然關心百姓、關心政務、關心家人——但他不讓這些關心變成內心的枷鎖。這是一種「帶著愛而不執著」的智慧，一種「身在紅塵、心在雲端」的境界。正如他在結尾所說：「以見余之無所往而不樂者，蓋遊於物之外也。」

這，就是〈超然臺記〉千百年來打動無數人的原因——當你讀完這篇文章，你會覺得：原來即使在最困難的時候，我們也可以選擇快樂。
