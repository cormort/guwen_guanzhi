# 〈范增論〉文章分析

> **主筆分析 · 古文觀止卷十一·宋文·蘇軾**

---

## 1. 文章結構與起承轉合

### 1.1 全文結構總覽

〈范增論〉全文約五百餘字，是蘇軾翻案史論中篇幅最短、結構最精悍的一篇。蘇軾以「去」字為全文之骨，以「義帝」為全文之眼，層層追問一個核心問題：**范增到底應該在什麼時候離開項羽？**

全文六個自然段，構成一個精密推進的邏輯鏈條：

| 段落 | 核心內容 | 功能 |
|:----:|----------|:----:|
| **第1段** | 敘事：陳平反間計 → 項羽疑范增 → 范增憤而去 → 疽發背死 | 交代事實，建立問題的背景 |
| **第2段** | 開門見山給出結論：「增之去，善矣。不去，羽必殺增。獨恨其不早耳。」→ 追問：「然則當以何事去？」→ 自答：「增之去，當於羽殺卿子冠軍時也。」 | 拋出核心命題，製造懸念 |
| **第3段** | 義帝之立與亡：范增為謀主 → 義帝存亡即范增禍福 → 「物必先腐也，而後蟲生之」——讒言生效的前提是君臣之間已有嫌隙 | 建立哲學基礎，揭示讒言運作規律 |
| **第4段** | 論義帝之賢：獨遣沛公入關、識宋義於稠人之中 → 項羽殺宋義後，義帝與項羽勢不兩立 | 抬高義帝，進一步論證項羽「不義」 |
| **第5段** | 范增最初與項羽「比肩而事義帝」→ 「力能誅羽則誅之，不能則去之」→ 「陋矣！」——蘇軾對范增最凌厲的批評 | 從君臣名分角度徹底論證范增的失誤 |
| **第6段** | 收束：「增，高帝之所畏也。增不去，項羽不亡。嗚呼！增亦人傑也哉！」 | 表面讚美，實為惋惜，深化翻案結論 |

### 1.2 起承轉合分析

| 結構 | 對應段落 | 核心內容 | 論證策略 |
|:----:|:--------:|----------|:---------|
| **起** | 第1–2段 | 敘事入題 → 直接拋出結論 → 追問「何時該走」→ 自答「當在羽殺卿子冠軍時」 | **開門見山，結論先行**。蘇軾不鋪墊，不迂迴，第一句話就告訴你范增走了是對的，但走晚了。然後馬上追問「什麼時候該走」並自答——讀者還來不及反駁，就已經被帶進了蘇軾的論證節奏 |
| **承** | 第3–4段 | 建立哲學前提（「物必先腐而後蟲生」）→ 論證義帝之賢 → 說明項羽殺宋義即「弒義帝之兆」 | **層層推進，建立標準**。先用一個無可反駁的比喻（腐爛生蟲）作為邏輯基礎，再用義帝的賢明反襯項羽的不義，為後文的凌厲批評鋪墊 |
| **轉** | 第5段 | 筆鋒急轉——「方羽殺卿子冠軍，增與羽比肩而事義帝，君臣之分未定也」→ 范增當時和項羽是「同事」而非「君臣」→ 「力能誅羽則誅之，不能則去之」→ 「陋矣！」 | **最凌厲的一轉**。蘇軾突然從君臣名分的角度切入，指出在項羽殺宋義的那一刻，范增和項羽是平級的「義帝之臣」，不是項羽的下屬——范增完全可以誅殺項羽！這翻轉了所有讀者對范增與項羽關係的預設 |
| **合** | 第6段 | 收束全篇——承認范增的才能（高帝之所畏），感嘆其結局的可惜（增亦人傑也哉） | **以退為進，餘音繞梁**。結尾看似在肯定范增，實際上是在強化翻案：正因為范增是「人傑」，他更應該有能力判斷何時離開——他卻沒有做到，這恰恰說明他的悲劇是自己造成的 |

### 1.3 起承轉合結構圖解

```
起（第1-2段）：敘事 → 結論先行 → 追問「何時該走」
    │
    ▼
承（第3-4段）：哲學基礎（物必先腐）→ 論義帝之賢 → 項羽不義
    │
    ▼
轉（第5段）：君臣名分逆轉——「比肩事義帝」→ 范增本可誅羽 →「陋矣！」
    │
    ▼
合（第6段）：讚其人傑 → 強化惋惜 → 深化翻案
```

### 1.4 結構特點

1. **節奏極快**：五百字完成「事實→結論→哲學→反轉→收束」的完整閉環，沒有一句廢話
2. **結論先行**：不按常規的「先論證再結論」，而是先拋出結論（「增之去，善矣……獨恨其不早耳」）再論證，製造懸念
3. **一轉致命**：第五段「君臣名分論」是全篇最強的反轉，將范增從「忠臣」翻轉為「判斷失誤的謀士」
4. **結尾凌厲**：「嗚呼！增亦人傑也哉！」——九字結尾，表面讚美，實為定讞

## 2. 修辭與論證手法

### 2.1 修辭手法分析

| 修辭手法 | 原文例證 | 作用與效果 |
|:---------|----------|:-----------|
| **比喻（設喻）** | 「物必先腐也，而後蟲生之；人必先疑也，而後讒入之」 | **全文最著名的比喻**。用自然界的規律（腐爛生蟲）來說明政治規律（讒言生效的前提是已有嫌隙）。這個比喻之所以不朽，在於它揭示了人性中一條普遍真理：外部破壞力量永遠需要內部裂縫才能生效 |
| **設問** | 「然則當以何事去？」 | 自問自答，引導讀者進入蘇軾的論證軌道。設問後緊接答案「增之去，當於羽殺卿子冠軍時也」——這是全文最核心的命題 |
| **設問** | 「陳平雖智，安能間無疑之主哉？」 | 以反問的形式給出答案：陳平不厲害，厲害的是項羽本身已經懷疑范增。強化「先腐後蟲」的論證 |
| **設問** | 「不賢而能如是乎？」 | 反問論證義帝之賢——逼迫讀者承認一個「被項羽殺死的君主」其實是賢明的 |
| **設問** | 「豈不毅然大丈夫也哉？」 | 反問強化理想期待——范增本可以像大丈夫一樣果斷行事（誅羽或離去），但他選擇了留下 |
| **對比** | 范增「建議立義帝」vs. 項羽「殺義帝」 | 揭示范增與項羽的根本矛盾——范增的政治貢獻（立義帝）被項羽親手摧毀 |
| **對比** | 范增「年七十，合則留不合則去」的理想 vs. 「依羽以成功名」的現實 | 以蘇軾設定的「智者標準」對比范增的實際選擇，凸顯其判斷失誤 |
| **層遞** | 全文六段：事實→結論→哲學→義帝論→名分論→收束 | 論證不斷加壓、翻轉，說服力逐層增強 |
| **層遞** | 「增之去，善矣。不去，羽必殺增。獨恨其不早耳。」 | 三句話三個層次：肯定行為（善矣）→ 指出危險（必殺）→ 指出失誤（不早）。層層推進到核心論點 |
| **引用（用典）** | 陳涉以項燕、扶蘇得民；項氏以立楚懷王孫心而興；諸侯叛之因弒義帝 | 用歷史先例建立論證基礎——得民心與失民心都與「義帝」有關，為後文范增與義帝的命運綁定鋪墊 |
| **諷刺** | 「陋矣！」 | 一個「陋」字，極度輕蔑——蘇軾用這個字徹底否定了范增留在項羽身邊的選擇 |
| **誇飾** | 「羽不殺增」 | 簡潔而強烈的判斷——如果范增不走，下場就是死。這是全文的前提之一 |
| **感嘆** | 「嗚呼！增亦人傑也哉！」 | 以感嘆收束全篇，表面讚美，實際是嘆息——人傑卻做出不智的選擇，可悲可嘆 |

### 2.2 論證策略

蘇軾的論證之所以無法反駁，在於他用了三層策略：

**第一層：偷換評價標準（翻案的核心方法論）**

傳統敘事問的是「范增忠不忠？」——答案是「忠，但項羽不用他」。蘇軾不回答這個問題，而是換了一個更刁鑽的問題：「范增什麼時候該走？」這個問題假設的前提是：范增不是「該不該走」的問題，而是「什麼時候走」的問題。一旦讀者接受了這個提問方式，就已經接受了蘇軾的翻案前提——范增的失敗不是因為項羽不聽話，而是因為他自己走晚了。

**第二層：建立不可反駁的哲學前提**

「物必先腐也，而後蟲生之」——這句話太符合常識，沒有人能反駁。但蘇軾利用這個「常識性真理」來論證一個遠非顯然的結論：陳平的反間計之所以成功，不是因為陳平高明，而是因為項羽早就懷疑范增了。一旦接受了這個前提，蘇軾的結論就變得順理成章。

**第三層：用「君臣名分論」進行致命反轉**

這是全文最精彩的一擊。蘇軾在第5段提出：在項羽殺宋義的時候，范增和項羽的關係是「比肩事義帝」——他們是同事，不是君臣。因此，范增在那個時候有兩個正當選擇：要麼誅殺項羽（為義帝除害），要麼離開（不以不義之人為伍）。但他兩者都沒有做——他留下來依附項羽以求成功名。蘇軾用一個字評價：「陋矣」。

這個反轉的厲害之處在於：它從君臣名分的倫理高度出發，讓范增失去了任何辯解的可能。你不是不能離開，而是你根本就不應該留下——你本可以站在道德高地上殺掉項羽，你卻選擇了依附他。

### 2.3 節奏控制

〈范增論〉的節奏極具辨識度：

- **開篇急促**：三句話交代完反間計、范增離去、范增死亡——沒有一句多餘的描寫，讀者還來不及反應，就已經進入了蘇軾的論證
- **中間從容**：第3–4段節奏放緩，逐層展開論證——先說哲學、再論義帝、再談名分，從容不迫
- **結尾凌厲**：「嗚呼！增亦人傑也哉！」——九字收束，乾淨利落，餘音繞梁

這種「急促→從容→凌厲」的節奏控制，是蘇軾「以論帶文」風格的核心體現。

## 3. 名句深度解析

### 3.1 「物必先腐也，而後蟲生之；人必先疑也，而後讒入之。」

**白話：** 物體一定是先腐爛了，然後才會生出蟲子；人一定是先有了疑心，然後讒言才能滲入。

**賞析：**

這是〈范增論〉乃至蘇軾全部史論中流傳最廣的名句，也是中國文化中最深刻的政治哲學格言之一。其核心洞見在於揭示了「內因決定外因」的普遍規律：

- 所有外部破壞力量（讒言、誤會、挑撥）都需要一個內部條件（信任裂縫）才能生效
- 陳平的反間計之所以成功，不是因為陳平有多高明——任何一個謀士都能想出這個計策——而是因為項羽對范增的信任早已動搖
- 項羽的懷疑從哪裡來？從他殺宋義的那一刻就開始了——范增是建議立義帝的人，而宋義是義帝的將領，項羽殺宋義就是打范增的臉

這句話的生命力遠遠超越了范增的個人故事，成為一個關於人際關係的普遍真理：任何關係的破裂，都不是因為外部的「第三者」，而是因為內部的裂縫已經大到足以被入侵。

### 3.2 「增之去，善矣。不去，羽必殺增。獨恨其不早耳。」

**白話：** 范增的離開是對的。如果不離開，項羽一定會殺了他。只遺憾他走得還不夠早。

**賞析：**

這是全文的「結論句」，也是蘇軾翻案論證的起點。三句話構成一個精密的邏輯推進：

1. **「增之去，善矣」**——肯定范增離開這一行為的正確性。這句話同時也是在反駁那些認為范增不該離開的傳統觀點。蘇軾先給范增一個「台階」：我承認你離開這個決定是對的
2. **「不去，羽必殺增」**——指出范增處境的危險性。這是一個不容置疑的判斷：如果范增當時不走，項羽不會因為他留下而恢復信任，反而會以「通敵」的罪名殺了他。這個判斷的依據是「物必先腐而後蟲生」——項羽的懷疑已經根深蒂固
3. **「獨恨其不早耳」**——轉折。這才是蘇軾真正想說的話。一個「獨」字，看似只是「唯一的遺憾」，實際上否定了整個傳統敘事對范增的同情。范增最大的錯誤不是「忠誠不被賞識」，而是「判斷時機失誤」

這三句話的順序是蘇軾刻意設計的：先肯定（消除敵意）→ 再指出危險（引發同情）→ 最後給出致命一擊（翻案）。讀者被引導到第三句時，已經來不及反對了。

### 3.3 「方羽殺卿子冠軍，增與羽比肩而事義帝，君臣之分未定也。為增計者，力能誅羽則誅之，不能則去之。豈不毅然大丈夫也哉？增年已七十，合則留，不合則去。不以此時明去就之分，而欲依羽以成功名，陋矣！」

**白話：** 當項羽殺宋義的時候，范增和項羽還是並肩侍奉義帝的同事，君臣名分還沒有確定。為范增考慮，有能力誅殺項羽就殺了他，沒有能力就離開。這難道不是大丈夫的果斷之舉嗎？范增已經七十歲了，合得來就留，合不來就走。不在這個時候明白去就的選擇，反而想依附項羽來成就功名，真是太淺陋了！

**賞析：**

這是全文最凌厲、最精彩的一段。蘇軾在這裡完成了一個令人瞠目的「名分逆轉」——在主流歷史敘事中，范增一直是「項羽的謀士」，但蘇軾指出：在項羽殺宋義的時候，這個君臣關係還沒有確立！范增當時的身份是「義帝的臣子」，與項羽是「同事」。

這個論斷的意義在於：范增的選擇不是「下屬是否該離開君主」，而是「同事是否該與叛徒為伍」。蘇軾給范增設計了兩個選項——誅殺項羽（大義滅親）或離開項羽（潔身自好），但范增選擇了第三條路：依附項羽以求成功名。「陋矣」二字，是蘇軾對范增最嚴厲的批評——不是說他壞，而是說他格局太小、判斷力太差。

### 3.4 「雖然，增，高帝之所畏也。增不去，項羽不亡。嗚呼！增亦人傑也哉！」

**白話：** 即使如此，范增是劉邦所畏懼的人。范增如果不離開，項羽不一定會滅亡。唉！范增也算是一代豪傑啊！

**賞析：**

這是全文的結尾，也是蘇軾最精妙的「以退為進」的手法。表面上看，蘇軾在肯定范增的才能——劉邦都怕他，他不走項羽就不會亡——所以他是「人傑」。但仔細品味，這其實是對翻案結論的深化：

如果范增只是一個平庸的謀士，他「走得太晚」或許可以原諒。但正因為他是「人傑」——連劉邦都畏懼他——他的判斷失誤就更加不可原諒。一個「人傑」級別的智者，不應該犯這種低級錯誤。所以蘇軾的「人傑」讚美，本質上是「這麼厲害的人物卻做出這麼愚蠢的選擇，真是太可惜了」——表面上讚美，骨子裡是惋惜，而惋惜就是另一種形式的批判。

## 4. 白話改編

### 原文故事

漢高祖劉邦採用陳平的計策，用反間計離間項羽和他的謀士范增。項羽果然中計，懷疑范增與劉邦私下勾結。范增一怒之下對項羽說：「天下大勢已定，您自己好好幹吧！請讓我這把老骨頭回家養老。」於是范增離開項羽，啟程回彭城。還沒走到目的地，背上長了毒瘡，就這麼死了。

表面上看，這是陳平用計高明的結果。但蘇軾提出了完全不一樣的看法。

### 蘇軾的翻案論證

**一、范增走對了，但走晚了**

蘇軾劈頭就說：范增離開項羽，這個決定是對的。如果不走，項羽遲早會殺了他。只可惜，范增走得還不夠早——他應該在項羽殺宋義（卿子冠軍）的時候就離開。

**二、讒言之所以有效，是因為君臣之間早有裂痕**

蘇軾用了一個著名的比喻：「物必先腐也，而後蟲生之；人必先疑也，而後讒入之。」——東西一定是自己先爛了，蟲子才會長出來；人一定是自己先起了疑心，讒言才能滲進去。陳平的反間計之所以成功，不是因為陳平有多厲害，而是因為項羽對范增的懷疑早就種下了。從什麼時候開始？從項羽殺宋義那一刻就開始了。

**三、義帝之賢**

宋義是誰？他是楚懷王（義帝）的將領。義帝選擇宋義當上將軍，又在眾人中識拔了劉邦，讓他去攻打關中——可見義帝是一位賢明的君主。項羽殺了宋義，這不僅是弒將，更是對義帝權威的直接挑戰。范增呢？范增正是當初建議項氏立楚懷王後代為義帝的人——范增的政治生命與義帝緊緊綁在一起。項羽殺宋義，就等於與義帝翻臉，而范增作為義帝的支持者，處境必然尷尬。

**四、最強的一擊：君臣名分之論**

蘇軾最精彩的論證在這裡：當項羽殺宋義的時候，范增和項羽是什麼關係？——他們都是義帝的臣子，是「同事」，不是「君臣」！君臣的名分在那一刻還沒有確定。所以蘇軾說，范增當時有兩個正當選擇：

- **有能力就殺了項羽**——為義帝除掉這個心腹大患
- **沒能力就離開項羽**——不與這種不義之人為伍

這才是大丈夫該做的事。可是范增怎麼做的？他既沒有殺項羽，也沒有離開項羽，而是選擇留下來依附項羽，指望跟著項羽成就一番功名。蘇軾只用了一個字評價：「陋矣！」——太淺陋了！

**五、結尾的讚美，其實是惋惜**

最後蘇軾說：范增是劉邦最畏懼的人。如果范增不走，項羽不一定會滅亡。唉！范增也算是一代豪傑啊！——表面上是讚美，但仔細一想：正因為他是「人傑」，他的判斷失誤才更令人惋惜。一個連劉邦都怕的頂級謀士，卻在最重要的時刻做了最錯誤的選擇，這不是他的悲劇，是什麼？

### 核心結論

蘇軾的〈范增論〉把范增從一個「忠心耿耿卻被項羽辜負的悲劇謀士」，翻案為一個「能力超群但在去留時機上判斷失誤的失敗者」。范增的悲劇不是項羽不聽他的話，而是他自己走得太晚了。

### 4.3 層層翻剝的論證技巧

蘇軾此文最精彩之處不在於提出一個新觀點，而在於**層層翻剝**——像剝洋蔥一樣，每翻一層，范增的「罪證」就更加清晰：

1. **第一層（事實）**：陳平反間計 → 項羽疑范增 → 范增去 → 范增死
2. **第二層（結論）**：去是對的，但太晚了 → 應該在殺宋義時就去
3. **第三層（哲學）**：物必先腐後蟲生 → 項羽早就懷疑范增了
4. **第四層（義帝論）**：義帝是賢主 → 項羽殺宋義就是殺義帝的先兆 → 范增作為義帝之臣，必受牽連
5. **第五層（名分論）**：范增和項羽是同事，不是君臣 → 范增本可誅殺項羽 → 選擇依附項羽是「陋矣」
6. **第六層（收束）**：范增是「人傑」→ 人傑卻走錯了這一步 → 無限惋惜

### 4.4 蘇軾史論的獨特方法

〈范增論〉集中體現了蘇軾史論的方法論精華：

| 方法論特徵 | 在〈范增論〉中的體現 |
|:-----------|:--------------------|
| **翻案思維** | 將范增從「悲劇忠臣」翻轉為「去留失時的失敗謀士」——不否定忠誠，只追問判斷力 |
| **時間節點分析** | 精確定位范增該走的時間點：「當以羽殺卿子冠軍時」——提供一個可檢驗的標準 |
| **因果鏈條重建** | 范增立義帝 → 項羽殺宋義/義帝 → 范增立場尷尬 → 項羽疑范增 → 范增不得不走——環環相扣 |
| **君臣名分論** | 從「比肩事義帝」角度論證范增與項羽並非君臣，而是同事關係——最精彩的角度切入 |
| **比喻論證** | 「物必先腐而後蟲生」——用自然規律論證政治規律，簡潔有力 |
| **凌厲收束** | 「嗚呼！增亦人傑也哉！」——以讚美包裝批判，以惋惜深化翻案 |

蘇軾的史論與父親蘇洵的「剛硬」不同。蘇洵的文章鋒利如刀，邏輯扼殺讀者；蘇軾則更加「從容不迫」——他先建立一個常識性的標準（「物必先腐而後蟲生」），然後層層推進到一個令人驚訝的結論（「增之去，當於羽殺卿子冠軍時也」），最後用一個看似溫和的讚美收束全文（「增亦人傑也哉」）。讀者被他引導著，不知不覺就接受了他的判斷，甚至還覺得自己是被說服的。
