# 〈信陵君救趙論〉文章分析

> **主筆分析 · 古文觀止卷十二·明文·唐順之**

---

## 1. 文章結構與起承轉合

### 1.1 全文結構總覽

〈信陵君救趙論〉是明代唐宋派古文大家唐順之的翻案史論代表作，收於《荊川先生文集》。全文約四百餘字，篇幅精悍，卻完成了從「破題」到「定讞」的完整論證閉環。唐順之以「心術」為核心武器——不去爭論竊符行為本身的是非，而是從信陵君的內心動機入手，將一個千古美談徹底翻轉為一場「以私義害公義」的政治事故。

全文五個自然段，構成一個精密推進的邏輯鏈條：

| 段落 | 核心內容 | 功能 |
|:----:|----------|:----:|
| **第1段** | 承認世人對竊符的批評 → 但否定竊符是真正的罪 → 從戰略高度肯定救趙（救趙即救魏、救六國） | 破題——先退一步，再進三步 |
| **第2段** | 轉折——「然則信陵果無罪乎？」→ 真正該批評的是信陵君的心 → 趙國只知有信陵而不知有王 | 提出翻案核心論點 |
| **第3段** | 假設推理——如果不是趙國、如果不是平原君，信陵君會救嗎？→ 結論：趙王與社稷不如平原公子一人 | 翻案的最強一擊 |
| **第4段** | 批評侯生、如姬也「知有信陵而不知有王」→ 給出三種更好的方案（諫諍、死諫）→ 但他們都沒選 | 深化論證，提出替代方案 |
| **第5段** | 收束——「信陵君知有義之可為，而不知有君之不可不事也」→ 最終判決 | 定讞——全文一句話總結 |

### 1.2 起承轉合分析

| 結構 | 對應段落 | 核心內容 | 論證策略 |
|:----:|:--------:|----------|:---------|
| **起** | 第1段 | 先承認世人觀點（竊符是罪）→ 再否定（此未足以罪信陵）→ 從戰略高度肯定救趙（救趙即救魏、救六國）→ 反問「夫奚不可者？」 | **以退為進，先立後破**。唐順之開篇不做攻擊，反而先替信陵君辯護——「竊符有什麼不可以的？」讀者正以為他要為信陵君翻案，卻不知這只是設下的一個陷阱 |
| **承** | 第2段 | 「然則信陵果無罪乎？曰：又不然也。」→ 鎖定真正目標：「余所誅者，信陵君之心也」→ 指責趙國「知有信陵，不知有王」→ 指責信陵君「知有婚姻，不知有王」 | **翻轉方向，鎖定目標**。從竊符的「事」轉向心術的「心」——這是最關鍵的一步。一旦讀者接受了「心術」這個戰場，信陵君就無法防禦了 |
| **轉** | 第3-4段 | 第3段：假設性推理——如果不是趙國、如果不是平原君，信陵君不會救 → 「趙王與社稷之輕重，不能當一平原公子」→ 第4段：批評侯生、如姬亦「知有信陵而不知有王」→ 提出三種更好方案（諫於王、死諫） | **最凌厲的一轉**。用兩個「使……則……」的假設推理將信陵君的「義」從天下大義降格為婚姻私義，再用三種替代方案反襯信陵君選擇的錯誤 |
| **合** | 第5段 | 「信陵君知有義之可為，而不知有君之不可不事也。余所誅者，信陵君之心也。」 | **一句定讞，首尾呼應**。與開頭的「此未足以罪信陵也」形成首尾閉環——開頭說「竊符不足罪」，結尾說「罪在心術」，回答了自己開篇提出的問題 |

### 1.3 起承轉合結構圖解

```
起（第1段）：先退一步——「竊符不足罪信陵」
    │
    ├─ 肯定救趙的戰略正確性（救趙即救魏、救六國）
    ├─ 反問——「夫奚不可者？」（引君入甕）
    │
    ▼
承（第2段）：突然翻轉——「余所誅者，信陵君之心也」
    │
    ├─ 將戰場從「行為」轉向「心術」
    ├─ 趙國：知有信陵，不知有王
    └─ 信陵君：知有婚姻，不知有王
    │
    ▼
轉（第3-4段）：最強翻案 + 替代方案
    │
    ├─ 第3段：假設推理 → 「為一平原公子耳」
    └─ 第4段：三種更好方案 → 卻一個都沒選
    │
    ▼
合（第5段）：最終判決——「知有義而不知有君」
    └─ 與開頭呼應，完成論證閉環
```

### 1.4 結構特點

1. **以退為進的破題術**：開篇不攻擊，反而先替信陵君辯護——「竊魏之符以紓魏之患……夫奚不可者？」這是中國論說文中最經典的開局策略：先承認對手的合理性，再從更高層次推翻這個合理性。讀者跟著唐順之走進第一層（竊符合理），卻不知道後面還有三層在等著。

2. **「心術」作為翻案槓桿**：全文的結構核心是一個「心」字。唐順之巧妙地繞開了「竊符」這個事實（因為救趙的戰略合理性確實無法否認），轉而從「為什麼竊符」這個動機問題切入。一旦把戰場從「事」轉向「心」，信陵君就失去了辯解的餘地——因為動機永遠比行為更難自證清白。

3. **假設推理的連環衝擊**：第3段的兩個「使……則……」假設構成了全文最強的打擊點。這種假設推理之所以有力，是因為它不需要推翻任何歷史事實——它只是在問一個問題：「如果情境換了，信陵君還會這樣做嗎？」而答案（不會）直接證明了信陵君的「義」是有條件的、是私人的。

4. **首尾呼應的閉環結構**：開頭「此未足以罪信陵也」與結尾「余所誅者，信陵君之心也」形成精妙的呼應——開頭否定了一種罪（竊符之罪），結尾確定了另一種罪（心術之罪）。這種結構讓全文成為一個完整的論證閉環，無懈可擊。

## 2. 修辭與論證手法

### 2.1 修辭手法分析

| 修辭手法 | 原文例證 | 作用與效果 |
|:---------|----------|:-----------|
| **設問** | 「然則信陵果無罪乎？」 | 全文最重要的結構轉折點。設問後緊接否定答案「又不然也」——用自問自答的方式將讀者從第一層（竊符合理）引導到第二層（心術有罪） |
| **設問** | 「夫奚不可者？」 | 反問式的設問——「這有什麼不可以的呢？」語氣中帶著一種「理所當然」的自信，引導讀者先接受救趙的合理性 |
| **設問** | 「吾不知信陵何以謝魏王也」 | 以推測語氣發出的設問——信陵君如果失敗了，要怎麼向魏王交代？這個問題讓讀者自行想像信陵君的尷尬處境 |
| **對比** | 「救趙者，亦以救魏；救一國者，亦以救六國也」vs.「其竊符也，非為魏也，非為六國也，為趙焉耳」 | 全文的核心對比：第一段說救趙就是救魏（戰略效果），第二段說竊符不是為魏而是為趙（動機）——效果與動機的分離，構成了唐順之翻案的邏輯基礎 |
| **對比** | 「趙王與社稷之輕重」vs.「一平原公子」 | 一國之君與一個公子的對比——趙國的江山社稷不及信陵君的一個姻親，這種不對等凸顯了信陵君動機的私人性 |
| **層遞** | 全文五段：破題→鎖定心術→假設推理→替代方案→最終判決 | 論證不斷加壓，從最寬鬆的「竊符不足罪」到最嚴厲的「知有義而不知有君」，層層遞進 |
| **層遞** | 「非為魏也，非為六國也，為趙焉耳。非為趙也，為一平原君耳。」 | 雙重否定疊加：先否定「為魏」和「為六國」，再否定「為趙」，最終鎖定「為一平原君」——範圍不斷縮小，動機不斷暴露 |
| **層遞** | 三種替代方案：「激諫於王」→「死於王前」→魏王「必亦悟矣」 | 從勸諫（溫和）到死諫（極端），層層遞進——每一種都比信陵君的竊符更高尚、更符合君臣之義 |
| **假設推理（counterfactual）** | 「使禍不在趙，而在他國……信陵亦必不救」 | 全文最核心的修辭武器。用一個假設情境（如果不是趙國）推導出信陵君的真實動機——不是為天下大勢，而是為特定對象 |
| **假設推理** | 「使趙無平原……雖趙亡，信陵亦必不救」 | 第二個假設情境（如果沒有平原君）。兩個假設疊加，將信陵君的動機鎖定在「婚姻私義」上，不留任何反駁空間 |
| **排比** | 「救趙者，亦以救魏；救一國者，亦以救六國也」 | 兩個「救……者，亦以救……」的排比句式，節奏明快，強化了救趙的戰略合理性——這正是為後文翻案鋪墊的「誘餌」 |
| **排比** | 「侯生教公子以竊符，如姬為公子竊符於王之臥內，是二人亦知有信陵，不知有王也」 | 將侯生和如姬的行為並列，指出他們與信陵君犯的是同樣的錯誤——心中無君。這種排比將批評範圍從信陵君一人擴大到整個「救趙團隊」 |
| **排比** | 三種替代方案中的「曷若……不聽，則以其欲死信陵君者死於……王亦必悟矣」 | 三重排比——對信陵君、對侯生、對如姬各給出一種更好的選擇，語氣越來越急迫，反襯出他們實際選擇的荒唐 |
| **諷刺** | 「趙王與社稷之輕重，不能當一平原公子」 | 全文最凌厲的諷刺。「不能當」三字輕描淡寫，但意味極深——趙國一國之君的份量，竟然比不上平原君這個人。這句諷刺直接摧毀了信陵君「為天下」的崇高形象 |
| **諷刺** | 「魏之兵甲所恃以固其社稷者，只以供信陵君一姻戚之用」 | 將國家軍隊（保衛社稷的武器）諷刺為信陵君的「個人禮物」——「只以供……之用」的句式帶有強烈的輕蔑 |
| **反問** | 「夫奚不可者？」 | 開篇結尾的反問——用「奚……者」的句式引導讀者自問自答，讓讀者在不知不覺中接受了「救趙合理」的前提 |
| **反問** | 「何計不出此？」 | 第4段結尾的反問——三種更好的方案都擺出來了，信陵君為什麼一個都不選？這個反問充滿了遺憾與不解，引導讀者與唐順之一起搖頭嘆息 |
| **感嘆** | 「夫強秦之暴亟矣，今悉兵以臨趙，趙必亡。」 | 開篇的感嘆語氣——「亟矣」二字帶有緊迫感，突顯了秦國的威脅之嚴重，為後文救趙的合理性做鋪墊 |
| **引用（用典）** | 信陵君竊符救趙的歷史事件（出自《史記·魏公子列傳》） | 全文的討論基礎。唐順之完全依賴司馬遷提供的史料，但在同一套史料上得出完全相反的結論——這是翻案文章的最高境界 |

### 2.2 論證策略——「三步剝皮法」

〈信陵君救趙論〉最精彩的論證策略是唐順之的「三步剝皮法」——像剝洋蔥一樣，每剝一層，信陵君的形象就更加不堪：

**第一步：先替對手說話（第1段）**

> 「論者以竊符為信陵君之罪，余以為此未足以罪信陵也。」

唐順之開篇先承認世人對信陵君的指控（竊符），然後否定這個指控。他從戰略角度論證救趙的合理性——「趙，魏之障也。趙亡，則魏且為之後」——進而得出結論：救趙就是救魏，救一國就是救六國。

這種策略的高明之處在於：唐順之建立了一個「客觀公正」的形象。讀者會想：「這個作者不是來罵信陵君的，他反而在為信陵君辯護。」這種印象讓讀者在後面的「翻案」中失去了防備。

**第二步：提出自己的指控（第2段）**

> 「然則信陵果無罪乎？曰：又不然也。余所誅者，信陵君之心也。」

用一個設問完成方向的轉折——「信陵君真的沒有罪嗎？不，還是有罪的。」但罪不在竊符，而在「心」。唐順之將戰場從「事」（竊符）轉向了「心」（動機），這是一個無法反駁的轉移——因為沒有人能證明信陵君的內心。

然後他從三個角度論證信陵君「心中無王」：趙國求救於信陵而非魏王 → 平原君以婚姻關係刺激信陵 → 信陵君自認因婚姻而救 → 最終結論：「信陵知有婚姻，不知有王」。

**第三步：推導出更嚴重的結論（第3-5段）**

第三步分為兩個層次：

第一層（第3段）：用假設推理將信陵君的「義」從「大義」降格為「私義」——如果不是趙國、如果不是平原君，信陵君不會救。結論：「則趙王與社稷之輕重，不能當一平原公子」。

第二層（第4-5段）：不僅信陵君有罪，侯生、如姬也有同樣的罪（知有信陵而不知有王）。唐順之還提出三種更好的替代方案（勸諫、死諫），反襯出信陵君等人的選擇是多麼錯誤。最終判決在第5段：「信陵君知有義之可為，而不知有君之不可不事也」。

### 2.3 節奏控制

〈信陵君救趙論〉的節奏控制極具辯論色彩：

- **開篇從容（第1段）**：不急著批評，先從容論證救趙的戰略意義——「趙，魏之障也。趙亡，則魏且為之後。趙、魏，又楚、燕、齊諸國之障也……」。這段節奏舒緩，像在和朋友聊天，讓讀者放下戒心
- **轉折急促（第2段開頭）**：「然則信陵果無罪乎？曰：又不然也。」——用短促的設問和否定，突然切換節奏，從從容變為凌厲
- **核心攻擊密集（第3-4段）**：假設推理、層層遞進、排比反問——節奏越來越快，論證越來越密，讀者來不及喘息
- **結尾乾淨利落（第5段）**：「信陵君知有義之可為，而不知有君之不可不事也。余所誅者，信陵君之心也。」——兩句話完成全文收束，沒有多餘的修飾，像法官落槌

### 2.4 與司馬遷的對話

〈信陵君救趙論〉最深刻的地方在於：**唐順之是在和司馬遷辯論。**

司馬遷在〈史記·魏公子列傳〉中將信陵君塑造成一個完美的英雄——禮賢下士、高義薄雲、救趙存魏。司馬遷說信陵君「名冠諸侯，不虛耳」。唐順之要做的就是推翻司馬遷的結論，但他不是捏造事實，而是用司馬遷自己提供的史料得出完全相反的結論：

- 司馬遷說信陵君「高義」→ 唐順之說：他的高義是為姻親，不是為國家
- 司馬遷說信陵君「救趙」→ 唐順之說：他救的是平原君，不是趙國
- 司馬遷說信陵君「賢」→ 唐順之說：他「賢」到心中無君

這是翻案文章的最高境界——不否定史料，只重新解釋動機。唐順之的做法堪稱「以子之矛攻子之盾」的典範。

## 3. 名句深度解析

### 3.1 「論者以竊符為信陵君之罪，余以為此未足以罪信陵也。」

**白話：** 評論者把竊取兵符當作信陵君的罪過，我認為這還不足以給信陵君定罪。

**賞析：**

這是全文的開篇句，也是唐順之「三步剝皮法」的第一步。其修辭藝術極其精妙：

- **「論者以……余以為……」的對立結構**：將「世人」與「我」並列，建立一個「眾人皆醉我獨醒」的話語姿態。唐順之不是要與世人爭論，而是要告訴世人：你們看問題的角度錯了。

- **「此未足以罪信陵也」的否定策略**：不是直接說「竊符無罪」，而是說「還不足以定罪」。這個「不足以」為後面的翻案留下了空間——「不足以」不代表「沒有罪」，只是「罪名不對」。「不足以」暗示還有更大的罪在後面。

- **語氣的微妙控制**：這句話的語氣不是攻擊性的，而是裁判式的。唐順之不像一個辯論者，而像一個高明的法官——先駁回一個指控（竊符），再提出新的指控（心術）。這種語氣讓讀者覺得他很客觀、很公正。

### 3.2 「余所誅者，信陵君之心也。」

**白話：** 我所譴責的，是信陵君的存心（內心動機）。

**賞析：**

這是全文的「文眼」——全文所有的論證，歸根結底就是這一句話。這句話出現在第2段開頭和第5段結尾，形成了全文最重要的首尾呼應。

- **「誅」字的凌厲**：「誅」本義是「殺戮」，引申為「譴責、批判」。用「誅」而不是「論」「評」「議」，顯示了唐順之的態度——不是溫和的討論，而是嚴厲的審判。但「誅」的對象是「心」而不是「人」，這又保留了倫理的空間：我殺的不是你這個人，而是你心中的那個念頭。

- **「心」字的關鍵定位**：唐順之選擇「心」作為全文的核心概念，有深刻的理學背景。宋明理學的核心命題之一就是「心術」——一個人的行為是否正確，關鍵不在於行為本身，而在於行為背後的動機。陽明心學講「致良知」，唐順之這裡用的是同一套話語體系：批評信陵君不是因為他做了什麼（救趙），而是因為他的「心」不純（為婚姻而不是為國家）。

- **全文的邏輯樞紐**：這句話出現在第2段，正是全文從「起」到「承」的轉折點。前一段在談「事」（竊符、救趙），從這一句開始轉向「心」（動機、心術）。一旦「心」成為戰場，信陵君就再也無法為自己辯護了——沒人能證明自己的動機是純粹的。

### 3.3 「使禍不在趙，而在他國，則雖撤魏之障，撤六國之障，信陵亦必不救。使趙無平原，而平原亦非信陵之姻戚，雖趙亡，信陵亦必不救。」

**白話：** 假如禍患不在趙國，而在其他國家，那麼即使這個國家是魏國的屏障、六國的屏障，信陵君也一定不會去救。假如趙國沒有平原君，或者平原君不是信陵君的姻親，那麼即使趙國滅亡，信陵君也一定不會去救。

**賞析：**

這是全文最核心的論證段落，也是唐順之翻案的最強一擊。其論證力量來自兩個方面：

- **雙重假設的疊加效應**：第一個假設（不是趙國）排除「為天下大勢」的可能；第二個假設（不是平原君）排除「為趙國」的可能。兩個假設疊加後，唯一的解釋就是「為平原君個人」。這種疊加假設的論證手法在邏輯學上叫做「排除法」——當所有其他可能性都被排除後，剩下的就是真相。

- **「必不救」的絕對語氣**：「必不救」三個字斬釘截鐵——不是「可能不救」，而是「一定不救」。這種絕對化的表述顯示了唐順之對自己判斷的極度自信。讀者會想：他為什麼這麼肯定？因為事實已經很清楚——信陵君和趙國沒有其他關係，唯一的連結就是平原君這個姻親。

- **修辭上的層層遞進**：第一個假設從「在他國」開始，推到「撤魏之障」（魏國的屏障），再推到「撤六國之障」（六國的屏障）——範圍不斷擴大，情況越來越嚴重，但結論始終不變：「信陵亦必不救」。這種「越嚴重越不救」的對比，將信陵君的「自私」暴露無遺。

### 3.4 「則是趙王與社稷之輕重，不能當一平原公子，而魏之兵甲所恃以固其社稷者，只以供信陵君一姻戚之用。」

**白話：** 這樣看來，趙國的國君和江山社稷的份量，還抵不上平原君一個人；而魏國的軍隊——原本是用來保衛國家的——只不過是給信陵君用來送給一個姻親的人情。

**賞析：**

這是全文最具殺傷力的一句話，可以說是唐順之對信陵君的「判詞」。其修辭藝術達到了巔峰：

- **「趙王與社稷」vs.「一平原公子」的強烈對比**：一國之君 + 整個江山社稷（最大的份量）vs. 一個人（最小的份量）——「不能當」三個字將這種不對等關係徹底暴露。讀者讀到這裡，會不由自主地倒吸一口冷氣：真的有這麼嚴重嗎？但回過頭看前面的論證，又不得不承認唐順之說得有理。

- **「只以供……之用」的諷刺語氣**：「只以」二字帶有強烈的輕蔑——這麼大的國家軍隊，這麼重要的軍事力量，竟然只是信陵君用來送人情的一件禮物。「供……之用」的句式將國家軍隊降格為日常用品，嘲諷意味十足。

- **「姻戚」二字的精準打擊**：唐順之不用「友人」「知己」「義氣」等詞，而用「姻戚」（婚姻親戚）。「姻戚」是一個中性的、甚至是帶有功利色彩的詞——它不是基於道義的結合，而是基於婚姻的血緣關係。用這個詞來描述信陵君與平原君的關係，從根本上消解了「義」的崇高性。

### 3.5 「信陵君知有義之可為，而不知有君之不可不事也。」

**白話：** 信陵君只知道「義」是可以做的事，卻不知道「君」是不可以不侍奉的。

**賞析：**

這是全文的最終判決，也是唐順之整篇文章的主旨所在。

- **「知有……而不知有……」的對立結構**：這個句式將「義」與「君」對立起來——不是說信陵君不知道「義」，而是說他只知「義」而不知「君」。在唐順之看來，對「義」的認知沒有問題，對「君」的認知才是問題的關鍵。

- **「有義之可為」vs.「有君之不可不事」的深層含義**：前半句是肯定——「義」是可以做的（好事）；後半句是否定——「君」是不可以不侍奉的（絕對義務）。兩者之間是「可選擇」與「不可選擇」的差別：「義」是你選擇做的事，「君」是你必須做的事。信陵君的問題在於：他把「可選擇」放在「必須」之前。

- **唐順之的儒家立場**：這句話體現了宋明理學最核心的政治倫理——「君臣之義無所逃於天地之間」。在儒家看來，君臣關係不是一種「選擇」，而是一種「天命」。你可以選擇不做「義」的事情（不救趙），但你不能選擇不侍奉「君」（不能繞過君主自行其事）。信陵君的問題在於，他把「義」當成了一種可以超越「君臣」的更高價值——這在理學家看來是絕對錯誤的。

- **與開頭的呼應**：這句話與開頭的「此未足以罪信陵也」形成了全文最關鍵的首尾呼應。開頭說「竊符不足罪」，結尾說出了真正的罪：「知有義而不知有君」。全文從否定一個罪名開始，到確立一個罪名結束，結構完整而有力。

### 3.6 「夫竊符之計，蓋出於侯生，而如姬成之也。」

**白話：** 那竊取兵符的計謀，是出自侯生（侯嬴），而由如姬完成的。

**賞析：**

這句話雖然不如其他名句那麼震撼，但在論證結構中有其特殊作用：

- **擴大批評範圍**：唐順之不是只批評信陵君一個人，而是將侯生、如姬也納入批評對象。這不僅讓批評更全面，而且暗示了一個更深層次的問題：不只是信陵君「心中無王」，他周圍的人（門客、寵妃）也都「心中無王」。這說明「心中無王」不是信陵君一個人的問題，而是整個魏國權力結構的問題。

- **為替代方案做鋪墊**：緊接著這句話，唐順之提出了對信陵君、侯生、如姬三人的三種替代方案——他們本可以用諫諍、死諫的方式來救趙，但他們選擇了最極端的方式。這種「先陳述事實，再提出更好方案」的論證結構，讓批評更加有力：不是說救趙不對，而是說你們救趙的方式可以更好。

- **「蓋……而……成之」的敘述語氣**：這句話的語氣像是一個冷靜的歷史敘述者——不急不緩，不怒不罵。但正是這種冷靜，讓後面的批評顯得更加客觀有力。

## 4. 白話改編

以下為〈信陵君救趙論〉全文分段的現代白話翻譯，便於對照原文理解文意。

### 原文：第一段（破題——先退一步）

> 論者以竊符為信陵君之罪，余以為此未足以罪信陵也。夫強秦之暴亟矣，今悉兵以臨趙，趙必亡。趙，魏之障也。趙亡，則魏且為之後。趙、魏，又楚、燕、齊諸國之障也。趙亡，則魏且為之後；魏亡，則楚、燕、齊諸國為之後。救趙者，亦以救魏；救一國者，亦以救六國也。竊魏之符以紓魏之患，借一國之師以分六國之災，夫奚不可者？

**白話：** 評論者把竊取兵符當作信陵君的罪過，我認為這還不足以給信陵君定罪。秦國的暴虐已經到了極點，如今傾全國兵力進逼趙國，趙國必將滅亡。趙國是魏國的屏障，趙國滅亡了，魏國就會跟著成為下一個目標。趙國和魏國，又是楚國、燕國、齊國等國的屏障。趙國滅亡，魏國就會步其後塵；魏國滅亡，楚國、燕國、齊國等國也會相繼淪陷。所以救趙國，也就是救魏國；救一個國家，也就是救六個國家。竊取魏國的兵符來解除魏國的禍患，借一個國家的軍隊來分擔六國的災難——這有什麼不可以的呢？

### 原文：第二段（翻案——鎖定心術）

> 然則信陵果無罪乎？曰：又不然也。余所誅者，信陵君之心也。趙王請救於魏，非請救於信陵也。信陵亦無自請救於趙之理也。乃不請於王，而私竊符以救趙，是趙王知有信陵，不知有王也；平原君知有信陵，不知有王也；而信陵君亦自以為救趙者，非為魏也，非為六國也，為趙焉耳。非為趙也，為一平原君耳。使禍不在趙，而在他國，則雖撤魏之障，撤六國之障，信陵亦必不救。使趙無平原，而平原亦非信陵之姻戚，雖趙亡，信陵亦必不救。

**白話：** 那麼信陵君真的沒有罪嗎？回答是：又不是這樣。我所譴責的，是信陵君的存心（內心動機）。趙王是向魏王求救，不是向信陵君求救。信陵君也沒有自己向趙國請求救趙的道理。但他不向魏王請示，就私自竊取兵符去救趙國——這說明趙王只知道有信陵君，不知道有魏王；平原君只知道有信陵君，不知道有魏王；而信陵君自己也認為救趙，不是為了魏國，不是為了六國，而是為了趙國罷了。不僅不是為了趙國，只是為了一個平原君罷了。假如禍患不在趙國，而在其他國家，那麼即使這個國家是魏國的屏障、六國的屏障，信陵君也一定不會去救。假如趙國沒有平原君，或者平原君不是信陵君的姻親，那麼即使趙國滅亡，信陵君也一定不會去救。

### 原文：第三段（最凌厲一擊——大小之辨）

> 則是趙王與社稷之輕重，不能當一平原公子，而魏之兵甲所恃以固其社稷者，只以供信陵君一姻戚之用。幸而戰勝，可也；不幸戰不勝，為虜於秦，是傾魏國數百年社稷以殉姻戚，吾不知信陵何以謝魏王也。

**白話：** 這樣看來，趙國的國君和江山社稷的份量，還抵不上平原君一個人；而魏國的軍隊——原本是用來保衛國家的——只不過是給信陵君用來送給一個姻親的人情。僥倖打贏了，也就罷了；如果不幸打敗了，被秦國俘虜，這簡直是把魏國數百年的江山社稷拿去為一個姻親殉葬——我不知道信陵君要如何向魏王交代。

### 原文：第四段（擴大批評——替代方案）

> 夫竊符之計，蓋出於侯生，而如姬成之也。侯生教公子以竊符，如姬為公子竊符於王之臥內，是二人亦知有信陵，不知有王也。余以為信陵之自為計，曷若以脣齒之勢激諫於王，不聽，則以其欲死信陵君者死於王前，王必悟矣。侯生為信陵計，曷若見王而說之救趙，不聽，則以其欲死信陵君者死於王前，王亦必悟矣。如姬有意於報信陵，曷若乘王之隙而日夜勸之救，不聽，則以其欲為公子死者死於王前，王亦必悟矣。如此，則信陵君不負魏，亦不負趙；二人不負王，亦不負信陵君。何計不出此？

**白話：** 那竊取兵符的計謀，是出自侯生（侯嬴），而由如姬完成的。侯生教公子竊取兵符，如姬為公子從魏王的臥室內竊取兵符——這兩個人也是只知道有信陵君，不知道有魏王。我認為，信陵君為自己打算，不如用「脣亡齒寒」的道理去激諫魏王，如果魏王不聽，就用他原本打算為信陵君赴死的那條命死在魏王面前——魏王一定會醒悟。侯生為信陵君出主意，不如去見魏王勸說他救趙，如果魏王不聽，就用他打算為信陵君赴死的那條命死在魏王面前——魏王也一定會醒悟。如姬有心報答信陵君，不如趁魏王有空的時候日夜勸他救趙，如果魏王不聽，就用她打算為公子赴死的那條命死在魏王面前——魏王也一定會醒悟。這樣一來，信陵君既沒有辜負魏國，也沒有辜負趙國；侯生和如姬既沒有辜負魏王，也沒有辜負信陵君。為什麼就沒有人想到這個辦法呢？

### 原文：第五段（最終判決）

> 信陵君知有義之可為，而不知有君之不可不事也。余所誅者，信陵君之心也。

**白話：** 信陵君只知道「義」是可以做的事，卻不知道「君」是不可以不侍奉的。我所譴責的，是信陵君的存心（內心動機）。

4. **第四層（假設推理）**：如果不是趙國 → 不救；如果不是平原君 → 不救 → 結論：所謂高義，不過是婚姻私義
