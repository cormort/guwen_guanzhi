# 〈親政篇〉— 文章分析

> 王鏊（1450–1524），字濟之，號守溪，諡文恪，蘇州吳縣人。明成化十一年進士第一（探花），官至戶部尚書兼文淵閣大學士。明代唐宋派古文大家，文章「醇正博雅」，與李東陽齊名。《古文觀止》收錄其文二篇，〈親政篇〉即其中之一。
>

## 1. 文章結構分析

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| **起**（第1段） | 以經典為據，揭示「君臣隔絕」為亂源——引用《易經·泰卦》「上下交而其志同」與《否卦》「上下不交而天下無邦」，直接點明：君臣之間的溝通品質，決定國家的治亂存亡。開篇不談具體制度，先確立一個無可辯駁的道義前提，為全文的制度改革論證鋪好理論地基。 |
| **承**（第2段） | 追述古代「三朝」制度——外朝（大朝，禮儀性）、治朝（常朝，日常政務）、內朝（燕朝，君臣面議），並指出內朝才是「上下交」的關鍵場合。舉周成王、漢宣帝、文帝為例，說明古人如何透過內朝實現君臣之間的有效溝通。王鏊在此段建立了全文論證的歷史參照系——古代有完善的制度，今人卻遺忘了它。 |
| **轉**（第3段） | 對比古今——明朝的朝會只剩外朝（大朝）與名存實亡的早朝，內朝完全廢弛。結果是皇帝「深居宮中」與大臣如隔天淵，奏章往來經過宦官之手，宦官因此壟斷資訊與決策權。王鏊在此完成從「制度描述」到「問題診斷」的關鍵轉折：不是宦官壞，而是制度留下了權力真空。 |
| **合**（第4段） | 提出務實的改革方案——恢復內朝：皇帝每隔三五日在文華殿或便殿，召集內閣大臣面議政事，各衙門要事當面呈奏、當面裁決，臺諫官亦得面陳意見。最後以「如祖宗之舊」收束——這不是標新立異，而是恢復被遺忘的祖制。話說得溫和，改革力度卻極大。 |

### 結構特點

1. **經典開篇、史論為骨**：開篇引《易經》，建立最高權威；中段引周、漢史例，建立歷史正當性；結尾回歸明代祖制，避免激進之嫌。三層權威（經典、歷史、祖制）層層加持，論證無懈可擊。
2. **「古今對比」的核心框架**：全篇以古代「三朝」制度為正面參照，與明代「廢內朝」的現實形成強烈對比，讓讀者直觀感受「失去的東西」有多麼重要。
3. **從診斷到處方的節奏**：先擺經典（問題的判斷標準）→ 再講古代對策（理想狀態）→ 再指出明代偏離（問題所在）→ 最後提出矯正方案（解答）——四段層層遞進，邏輯嚴密。

## 2. 寫作技巧分析

### 修辭手法

| 手法 | 原文 | 效果 |
|:----|------|------|
| **經典引用（開篇定調）** | 「《易》之《泰》曰：『上下交而其志同。』其《否》曰：『上下不交而天下無邦。』」 | 以《易經》為全文的最高論證權威。〈泰〉〈否〉二卦的對比，精準對應「君臣溝通」與「君臣隔絕」兩種狀態——不是王鏊在說教，而是聖人經典早已告誡。一句話就讓反對者無法開口。 |
| **古今對比（貫穿全文）** | 古代內朝「君臣皆坐」vs 明代「君臣隔絕」 | 全文最核心的修辭手段。古代的「坐而論道」象徵著君臣之間的信任與平等交流，明代的「隔絕」則象徵疏離與猜忌。對比越強烈，改革的正當性就越強。 |
| **引經據典（三朝制度）** | 「蓋古者有內朝而後有外朝，有治朝而後有內朝。」 | 引用《周禮》等經典中的三朝制度，為自己的改革方案建立歷史先例。王鏊的聰明之處在於：他不說「我要改革」，而說「我要恢復」——恢復祖制比創立新制更容易被接受。 |
| **歷史舉例（以史為證）** | 周成王「嚴恭寅畏」→ 漢宣帝「五日一聽朝」→ 漢文帝「臨朝諮訪」 | 連舉三個成功案例，涵蓋周、西漢兩個黃金時代，證明內朝的有效性。案例的時間跨度越大，論證的普世性越強。漢文帝「臨朝諮訪」的細節尤其生動——「咫尺天顏，無不盡見」，為後文描寫明代隔絕的負面狀況埋下伏筆。 |
| **對比反襯（古今落差）** | 「今內朝罕復臨御，常朝之後，人臣無復進見。三殿高閎，鮮或窺焉。」 | 用「罕」「無復」「鮮或」一系列否定性詞彙，勾勒出明代朝會的荒廢景象。前文描寫古代內朝的興盛越詳細，此處明代朝會的冷清就越刺眼——這種「先揚後抑」的節奏控制極為精準。 |
| **因果分析（制度診斷）** | 「上下之間，章奏批答相關接，刑名法度相維持而已。」 | 一句話概括了明代政務運作的全部問題：君臣之間只剩下書面往來（奏章批紅），沒有面對面的交流。用「而已」二字收尾，語氣輕巧卻含義沉重——堂堂國家大事，居然只剩紙上作業？ |
| **設問引導** | 「豈非所謂『上下不交』之患耶？」 | 在描述完明代朝會荒廢的狀況後，以設問將前文的描述與開篇的經典命題串聯起來。讀者心中早已有了答案，但由作者正式提出，形成前後呼應的論證閉環。 |
| **排比建議（改革方案）** | 「如：……三日一臨……五日一臨……諸司面奏……臺諫面陳……」 | 一連串具體建議排比而出，形成明確的行動綱領。從頻率（三日/五日）→ 場合（文華殿/便殿）→ 對象（內閣/諸司/臺諫），層層細化，顯示王鏊絕不是空談，而是經過深思熟慮的具體設計。 |
| **卒章敬語（奏議體收束）** | 「臣不勝至願。」 | 奏議體的標準結尾格式。用極其恭敬的語氣總結全文——雖然前面提出了激烈的制度改革意見，但結尾保持臣子謙遜的姿態，既不失改革者的銳氣，也不失臣子的禮節。 |

### 寫作特色總評

1. **制度分析取代道德譴責**：全文沒有一句罵劉瑾、罵宦官的話。王鏊完全繞開了「誰壞」的爭論，直接進入「為什麼壞」的制度分析。這種寫法在明代奏議中極為罕見——正因如此，它比所有罵太監的奏章都深刻。

2. **經典敘事框架的巧妙運用**：王鏊將自己的制度改革方案包裝成「恢復祖制」——這在儒家政治話語中是最安全的立場。他不說「我要改革」，而說「我要恢復如祖宗之舊」，降低了改革的政治風險。

3. **「正名」策略**：全篇反覆辨析「什麼才是真正的朝會」——不是大典上的排場，而是君臣面對面的交流。通過重新定義「朝會」的內涵，王鏊完成了對明代政治制度的根本批判。

## 3. 佳句賞析

> **「《易》之《泰》曰：『上下交而其志同。』其《否》曰：『上下不交而天下無邦。』蓋言上下之義，猶天地然。」**

**賞析**：全文破題句，以《易經》泰否二卦為論證的根本支點。〈泰卦〉象徵天地交融（上→下，下→上），萬物通暢；〈否卦〉象徵天地隔絕，萬物閉塞。王鏊把君臣關係比作天地關係——君臣溝通順暢，國家就像春天一樣繁榮；君臣隔絕，國家就像冬天一樣死寂。這不是修辭，這是世界觀。用一個卦象對比，就為全文的論證建立了不可動搖的哲學基礎。

> **「蓋古者有內朝而後有外朝，有治朝而後有內朝。」**

**賞析**：這句話的核心在於「而後有」三個字——這不是時間順序，而是因果關係。王鏊的邏輯是：內朝（君臣面議）是政務運作的起點，有了內朝的充分討論，外朝的儀式才有意義。這直接反轉了一般人對朝會的理解——一般人認為大朝是最重要的（場面最大），王鏊卻說內朝才是根本。這種「顛覆常識」的洞察力，是政論文的最高境界。

> **「君臣相見，止於視朝數刻；上下之間，章奏批答相關接，刑名法度相維持而已。」**

**賞析**：一句話勾勒出明代政務的全部悲哀。「數刻」強調時間之短（古代朝會可能只有幾刻鐘），「而已」強調功能之弱（只剩文書往來）。王鏊用輕描淡寫的語氣說出了最嚴重的指控：堂堂大明的政務運作，不過是紙上作業罷了。語調越是平靜，批判力道越強。

> **「深居宮中，而大臣不得一見；批答奏章，而宦官得以代之。」**

**賞析**：全文最直接、最鋒利的一句診斷。兩組對仗：「深居」對「批答」——一個是行動，一個是儀式；「不得一見」對「得以代之」——一個是被排除，一個是取而代之。十四個字說盡了明代宦官專權的制度根源：不是劉瑾太壞，而是皇帝不見大臣，中間的真空自然被宦官填補。這是〈親政篇〉全文的「文眼」。

> **「如祖宗之舊，此天下所以望於陛下也。」**

**賞析**：全篇收束句，語氣謙卑而力量萬鈞。「如祖宗之舊」——王鏊把自己的改革方案包裝成「恢復祖制」，這是明代政治中最有力也最安全的論證方式；「此天下所以望於陛下也」——用天下百姓的期待來給皇帝施加壓力，卻又說得極其恭敬。全句謙卑中藏著鋒芒，溫和中帶著堅定，是奏議體結尾的典範。

## 4. 白話改編

> 《易經》的〈泰卦〉說：「君臣上下互相溝通，他們的意志就能一致。」〈否卦〉說：「君臣上下不溝通，天下就沒有國家的樣子。」——大概說的是君臣之間的關係，就像天和地的關係一樣。
>
> 古代的聖明天子，設立了「三朝」的制度。在宮門之外的是「外朝」，那是舉行大典的地方；在宮門之內、宮殿之外的是「治朝」，那是日常處理政務的朝會；再往內的是「內朝」，也叫「燕朝」，那是君臣坐在一起討論國事的地方。由此可見，古人是先有內朝，然後才有外朝；先有治朝，然後才有內朝。
>
> 周成王的時候，莊重恭敬，每天上朝聽政，讓群臣從容地各抒己見。後來禮儀越來越繁瑣，但朝會制度中君臣坐而論道的內朝精神，始終沒有廢除。到了漢代還保存著這個傳統：漢宣帝每隔五天在便殿聽一次朝，丞相以下的官員各自匯報職責上的大事。漢文帝更勤奮，上朝時詔令大臣們暢所欲言，連侍衛軍官（虎賁、期門）和各部門的長官（郎、將），都有機會回答皇帝的詢問——咫尺之間，皇帝可以看見所有官員，官員可以看見皇帝。君臣之間，從來沒有被隔絕的憂患。
>
> 如今，內朝已經很少舉行了，早朝之後，大臣們再也沒有機會進見皇帝。宮殿那麼高大宏偉，官員們幾乎無法窺見其中。再加上有些官員品級太低，根本無法上殿，只能跪伏在殿前的石階之下。皇帝和臣子見面，只在早朝那短短幾刻鐘；上下之間，不過是靠奏章和批紅來聯繫、靠法律法規來維持而已。這難道不就是〈否卦〉所說的「君臣不溝通」的禍患嗎？
>
> 這就是為什麼我希望陛下在處理政事的空閒時間，恢復古代的內朝制度。具體來說：比如每隔三五天，在文華殿或便殿，召集內閣大臣來討論政務。各衙門有事需要上奏的，可以當面向皇帝呈奏、當面裁決。臺諫官可以隨時當面陳述意見。這樣一來，皇帝就可以知道整個天下的情況，宦官就無法從中作梗了。從前流傳的「上下溝通、意志一致」的局面，就可以實現了。
>
> 恢復祖宗曾經實行過的舊制——這就是天下百姓對陛下的期望啊。
