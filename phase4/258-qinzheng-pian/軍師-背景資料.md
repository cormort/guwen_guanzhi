## 二、作者：王鏊其人

王鏊（1450–1524），字濟之，號守溪，晚號拙叟，蘇州吳縣人。他是明代中期最負盛名的名臣與文章家之一。

- **仕途履歷**：成化十一年（1475）進士第一（探花），授編修。歷任侍講學士、少詹事、吏部左侍郎，官至戶部尚書兼文淵閣大學士（入閣為相），後晉少傅、太子太傅。
- **正直品格**：王鏊以剛正不阿著稱。武宗寵信宦官劉瑾（「八虎」之首），朝政敗壞，王鏊雖在內閣，屢次力諫不可，終因劉瑾專權而憤然辭官歸里。
- **文學成就**：王鏊是明代唐宋派古文大家，文章「醇正博雅」，與李東陽齊名。《古文觀止》收錄其文二篇，〈親政篇〉即其中之一。
- **歷史評價**：《明史》本傳稱其「以文章名世，而德業亦相稱」，是明代少數能兼顧道德、文章、事功的士大夫。

王鏊寫作〈親政篇〉時，正處於武宗荒嬉、劉瑾亂政的黑暗時期。這不是一篇書齋裡的空論，而是一位身居內閣、親眼目睹朝政崩壞的名臣，從制度根源提出的一劑苦口良藥。

## 三、時代背景：明武宗朝的亂局

### 3.1 明武宗朱厚照（在位1505–1521）

武宗是明代歷史上最具爭議的皇帝之一。他十五歲繼位，天性貪玩、厭惡朝政，幾乎不上朝，終日沉溺於：

- **豹房**：他在宮中另建豹房，畜養虎豹、美女、樂工，在此嬉樂，甚至把朝政搬到豹房處理。
- **微行出遊**：屢次化裝出宮，遠赴宣府、大同等地，擾民不已。
- **寵信宦官**：將政事全權交給劉瑾等宦官，「批答奏章，皆不經御覽」。

武宗在位十六年，正規朝會（早朝）形同虛設，大臣常年見不到皇帝，國家大事由宦官劉瑾在司禮監中專斷。

### 3.2 宦官劉瑾與「八虎」

明武宗寵信的宦官集團以劉瑾為首，人稱「八虎」。劉瑾的權力達到明代宦官專權的頂點：

- **控制批紅**：明代內閣票擬後，皇帝用硃筆批紅。武宗不管，劉瑾代批，等同實際掌握最終決策權。
- **設立內行廠**：劉瑾在錦衣衛、東廠之外另設「內行廠」，監視百官，恐怖統治。
- **打擊朝臣**：正德元年（1506），劉瑾矯旨將劉健、謝遷等顧命大臣逼退；尚書韓文等數十人被廷杖、削籍；內閣大學士焦芳投靠劉瑾，朝政糜爛。

王鏊正是在這樣的環境中撰寫〈親政篇〉——他深知宦官專權不是因為劉瑾一個人壞，而是因為制度設計讓皇帝與大臣隔絕，中間的空隙自然被宦官填補。

## 四、歷史根源：明代君臣隔絕的制度設計

明代的政治制度，從朱元璋時代就刻意設計成「君臣隔絕」的模式。這是理解〈親政篇〉論證的關鍵背景：

### 4.1 朱元璋廢宰相（1380）

洪武十三年，朱元璋以胡惟庸案廢除宰相制度，皇帝直接管理六部。此舉讓皇權達到頂峰，但也產生一個問題：

- **宰相的橋樑功能消失**：以前由宰相定期與皇帝會面、傳達百官意見，現在這條管道斷了。
- **皇帝獨攬政務的能力跟不上**：朱元璋本人勤政，可以日批數百奏章；但後代皇帝做不到，於是奏章審閱的權力逐步落入皇帝身邊最親近的人——宦官手中。

### 4.2 內閣制度與司禮監的此消彼長

- **內閣**（文淵閣）本質上是皇帝的祕書班子，位低權輕，大學士品秩僅五品。後來雖權重，但名義上仍不是宰相，沒有制度化的面見皇帝的管道。
- **司禮監**（宦官衙門）反而因為掌握「批紅」權（代皇帝批答奏章），成為事實上的最高決策機構。
- **結果**：明代後期出現「內閣票擬、司禮監批紅」的雙軌體制，皇帝不見內閣大學士，卻天天見司禮監太監——宦官的地位自然高於大臣。

### 4.3 朝會制度的萎縮

明代祖制有三種朝會：

1. **大朝**（正旦、冬至、萬壽聖節）——禮儀性質，不議政。
2. **常朝**（早朝）——理論上每日舉行，但到中期已成形式。
3. **內朝**——即皇帝在內廷與大臣面議政事，最有效，但明代幾乎廢弛。

王鏊指出：古代有「內朝」，君臣坐而論道；明朝皇帝連「早朝」都不願上，更別說「內朝」。於是君臣之間只剩下「奏章—批紅」的書面往來，中間完全由宦官代理。

## 五、明代宦官專權的制度根源（王鏊的核心診斷）

王鏊的論證極具洞見——他不罵宦官，而是從制度面分析：

### 5.1 關鍵因果鏈

```
皇帝不上朝 → 不見大臣 → 資訊來源只剩宦官 → 宦官壟斷資訊與決策
                                     ↓
                            不是宦官太壞，
                            而是制度留下權力真空
```

### 5.2 「三朝」制度之辨

王鏊梳理古代朝會制度：

- **外朝**（大朝）：隆重但形式化，不議政。
- **內朝**（燕朝、路寢之朝）：君臣坐而論道，才是真正的政務討論場合。

明代廢除了內朝，只剩下大朝和名存實亡的早朝，導致皇帝永遠處在深宮，與大臣如隔天淵。

### 5.3 恢復內朝的改革方案

王鏊提出的具體對策非常務實：

1. 皇帝每隔三五日，在便殿（如文華殿）與內閣大臣面議政事。
2. 各衙門有重要奏章，當面呈奏、當面裁決。
3. 讓臺諫官亦得隨時面陳意見。
4. 如此則宦官無法從中作梗，君臣上下交流通暢。

這不是激進的改革，而是一個極其理性、可操作的制度調整方案——從形式上看，就是恢復「內朝」這個被遺忘的祖制。

## 六、〈親政篇〉的歷史影響與評價

### 6.1 當時效果

- 武宗並未採納王鏊的建議。〈親政篇〉上呈後石沉大海，劉瑾的權勢反而更大。
- 王鏊眼見忠言不用，於正德四年（1509）辭官歸隱，嘉靖三年（1524）卒於家中。

### 6.2 後世解讀

- 明末清初思想家反思明朝滅亡的原因時，〈親政篇〉成為關鍵文獻——顧炎武、黃宗羲論政治制度時均指向「皇帝與大臣隔絕」此一病灶。
- 黃宗羲《明夷待訪錄·置相》明確繼承了王鏊的思路：廢宰相導致皇帝孤立、宦官專權，必須設置一個能夠與皇帝面對面交流的職位。

### 6.3 古文觀止的選文意義

《古文觀止》選入〈親政篇〉，不是因為它的文學修辭多麼華麗（雖然王鏊確實寫得典雅簡潔），而是因為：

- 它是「經世致用」文章的典範——用古文的語言，討論最現實的政治問題。
- 它代表了明代士大夫對皇權制度最深刻的反省之一。
- 它的論證結構（古今對比、制度分析、提出方案）是議論文寫作的教科書級範本。

## 七、關鍵詞彙與概念對照

| 關鍵詞 | 解釋 |
|--------|------|
| **親政** | 皇帝親自治理政事，非委政於宦官或外戚 |
| **內朝** | 皇帝在內廷便殿與大臣面議政事的制度，先秦稱「路朝」、「燕朝」 |
| **外朝** | 國家大典時的朝會，儀式性強，不議政 |
| **三朝** | 外朝、治朝（常朝）、內朝並存的古代制度體系 |
| **上下交** | 君臣之間良性溝通、資訊暢通的狀態 |
| **上下不交** | 君臣隔絕、資訊壟斷的危險狀態，語出《易經》 |
| **批紅** | 皇帝用硃筆批示內閣票擬的奏章；明代中後期常由司禮監太監代行 |
| **票擬** | 內閣對奏章提出處理意見，貼在奏章上供皇帝參考 |

> **軍師評**：明朝的宦官問題，從來不是因為某個宦官「壞」，而是因為制度讓皇帝與大臣之間形成真空，宦官自然填補。王鏊〈親政篇〉的高明之處，在於他繞開了道德譴責，直接向制度動刀——恢復內朝，讓皇帝重新看見大臣的臉。這一洞察，比後世無數罵宦官的奏章都深刻。只可惜，武宗連「看見大臣的臉」都不願意。
