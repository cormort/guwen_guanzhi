# 〈深慮論〉— 文章分析

> 方孝孺（1357–1402），字希直，號遜志，浙江寧海人。明初大儒，師從宋濂，以「明王道、致太平」為己任，建文帝時任侍講學士。靖難之役後拒絕為朱棣起草即位詔書，被凌遲處死，誅十族，是中國歷史上唯一一次「誅十族」的記載。
>

## 1. 文章結構分析

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| **起**（第1段） | 開篇即拋出一個悖論——開國君主無不深謀遠慮（「圖其所難，備其所可畏」），但禍亂總是發生在他們「所忽」「所不疑」之處。隨即以一個反問引出全文核心命題：「豈其慮之未周歟？蓋慮之所能及者，人事之宜然；而出於智力之所不及者，天道也。」——不是慮得不夠，而是慮的範圍有限，天道在智力範圍之外。 |
| **承**（第2段） | 以五個歷史案例逐一論證上述命題——秦（廢封建設郡縣→亡於庶民劉邦）、漢（大封同姓王→七國之亂、王莽篡漢）、晉（懲魏孤立→八王之亂）、唐（殺武氏嫌疑者→武則天就在身邊）、宋（杯酒釋兵權→子孫困於外敵）。每個案例都遵循相同邏輯：開國君主的「深慮」→恰好敗於該「深慮」之外的變數——形成一種歷史的黑色幽默：「慮切於此，而禍興於彼」。 |
| **轉**（第3段） | 從歷史案例上升為哲學診斷——「智可以謀人，而不可以謀天」。方孝孺用「良醫之子多死於病，良巫之子多死於鬼」這個類比，將抽象的天命觀化為日常經驗。良醫不是不會治病，而是太自信於自己的專業，反而忽略了平凡的風險；開國君主不是不聰明，而是太專注於「謀人」（設計制度防範內亂），反而忽略了「謀天」（天意不可測）。 |
| **合**（第4段） | 提出解決方案——捨棄「私謀詭計」，回歸「積至誠、用大德，以結乎天心」。用「慈母保赤子」比喻上天對有德之君的眷顧，即使子孫愚鈍，上天也不忍心立即滅亡這個王朝。最後以反問收束：「欲以區區之智，籠絡當世之務，而必後世之無危亡，此理之所必無者，而豈天道哉！」——擲地有聲，完成論證閉環。 |

### 結構特點

1. **由果推因的歷史歸納結構**：不從理論出發，而從具體的歷史案例出發——先呈現「深慮反而亡國」的現象，再歸納出「智可謀人不可謀天」的命題，最後提出「積德結天」的對策。這種「現象→診斷→解方」的三段式非常清晰。
2. **案例的遞進排列**：秦→漢→晉→唐→宋（以及元），時間順序排列，但每個案例都證明同一個邏輯——排比的氣勢營造出歷史的必然感，讀者讀完五個案例，幾乎無法反駁。
3. **從歷史到哲學再到倫理的上升**：文章不滿足於歷史分析，而是從歷史歸納躍升到哲學命題（謀人vs謀天），最終導向倫理實踐（積至誠、用大德）——層層遞進，格局不斷放大。

## 2. 寫作技巧分析

### 修辭手法

| 手法 | 原文 | 效果 |
|:----|------|------|
| **反問（破題）** | 「豈其慮之未周歟？」 | 開篇的關鍵反問，方孝孺先擺出悖論（深慮仍亡國），再用反問否定表面答案（不是慮得不够），逼迫讀者思考真正的答案。反問之妙在於：看似開放，實則引導。 |
| **排比（歷史案例）** | 秦→漢→晉→唐→宋（五個朝代、相同的邏輯結構） | 五個案例在句式上形成排比——每段皆以朝代開頭，描述其「深慮」的內容，再指出其「亡」的結果。排比營造歷史的規律感，讓讀者覺得這不是偶然，而是普遍的歷史法則。 |
| **對比（貫穿全文）** | 「圖其所難而忽其所易，備其所可畏而遺其所不疑」 | 開篇四組對比（難/易、可畏/不疑、所忽/所備、此/彼），每一組都是一個矛盾，將「深慮的失敗」濃縮為一連串的認知盲點。句型工整，力量集中。 |
| **對比（最高層次）** | 「私謀詭計」vs「至誠大德」 | 結尾的最終對比，將全文提升到道德層次——前文所有的制度分析、歷史案例，最終收束到一個最根本的價值選擇：權謀 vs 德行。 |
| **比喻（良醫良巫）** | 「良醫之子，多死於病；良巫之子，多死於鬼」 | 全文最經典的比喻。將抽象命題（專注導致盲點）化為具體的日常經驗——醫生家的孩子死於病，法師家的孩子死於鬼。這個比喻的力量在於：人人都能理解，而且無法反駁。它不是哲學推導，而是生活常識。 |
| **比喻（慈母保赤子）** | 「若慈母之保赤子而不忍釋」 | 將天道擬人化為慈母——上天對有德之君的眷顧，就像母親疼愛自己的孩子。這個比喻讓冷峻的天道有了溫暖的感性色彩，使讀者更容易接受「積德結天」的結論。 |
| **層遞（三層轉折）** | 第一層：歷史現象（秦漢唐宋皆如此）→ 第二層：哲學診斷（謀人vs謀天）→ 第三層：倫理解方（積德結天） | 全文的結構本身就是一個巨大的層遞——從具體到抽象，從現象到本質，從問題到對策，層層深入，不可逆轉。 |
| **諷刺（反語）** | 「此理之所必無者，而豈天道哉！」 | 結尾的反諷——你們以為靠區區智謀就能保證萬世太平？這種想法根本違背天道！語氣從前文的冷靜分析轉為凌厲的批判，產生強烈的情感衝擊。 |
| **設問（自問自答）** | 「何哉？蓋智可以謀人，而不可以謀天」 | 在五個歷史案例之後，方孝孺自問自答——「為什麼深慮反而導致滅亡？」→ 因為智可以謀人，不可以謀天。設問引導讀者的思緒，同時毫不含糊地給出答案。 |
| **引用（用典）** | 全文以歷史為論據——秦郡縣、漢封王、晉削宗室、唐殺武氏、宋收兵權 | 所有引用都服務於同一個論證目的，每一個案例都是「深慮失敗」的同一邏輯的不同變體。案例的選擇極具代表性——涵蓋了不同類型的制度設計（行政、宗室、兵權），使論證無懈可擊。 |
| **互文（前後呼應）** | 開篇「慮之所能及者，人事之宜然」→ 結尾「欲以區區之智，籠絡當世之務」 | 開篇承認「人事之宜然」是智慮所能及的範圍，結尾批評「區區之智」的局限——首尾對應，形成論證閉環。開篇的懸念（「豈其慮之未周歟？」）在結尾得到了完整的解答（「此理之所必無者，而豈天道哉！」）。 |

## 3. 佳句賞析

> **「圖其所難而忽其所易，備其所可畏而遺其所不疑。然而禍常發於所忽之中，而亂常起於不足疑之事。」**

開篇破題句，也是全文最精煉的悖論表述。方孝孺用四組對比（難/易、可畏/不疑、所忽/所備、此/彼）濃縮了王朝興衰的亙古規律——人類總是關注那些明顯的威脅，卻忽略了那些看似平常、不值得懷疑的事物。但歷史的諷刺在於：真正的災難從來不來自你嚴陣以待的方向，而來自你認為「不足疑」的地方。這兩句話奠定了全文的基調：對人類「理性自信」的深刻懷疑。

> **「蓋慮之所能及者，人事之宜然；而出於智力之所不及者，天道也。」**

全文的核心命題，也是方孝孺對「深慮」問題的最終診斷。這句話的精妙在於它的「界線思維」——方孝孺不否定人類智慮的價值（「人事之宜然」），但明確劃出了一條界線：人事之內，智慮有效；人事之外，天道主導。這種「劃界」的論證方式既避免了全盤否定人類努力的虛無主義，又點出了理性設計的根本局限，是儒家「盡人事、聽天命」思想最精確的哲學表述。

> **「智可以謀人，而不可以謀天。」**

全文的「文眼」，九個字說盡千古興亡。這是最簡潔的表述，卻涵蓋了最深刻的道理——人類社會的一切制度、權謀、算計，都只能在「人」的層面運作（你可以算計對手的行為、設計制度的約束），但無法控制「天」的層面（歷史的走向、時代的變遷、不可預見的偶然）。這句話讓所有的「深慮」者都必須面對一個殘酷的事實：你算計了所有的變數，卻算不準天意。

> **「良醫之子，多死於病；良巫之子，多死於鬼。」**

全文最生動的比喻，也是將哲學命題化為日常經驗的典範。方孝孺這句話的力量來自三個層次：
- **經驗層**：這是生活常識——醫生的孩子也生病，法師的孩子也撞鬼——人人都有共鳴
- **邏輯層**：揭示了「專業即盲點」的悖論——越是擅長某個領域，越容易忽視該領域的常規風險，因為自信產生了盲區
- **象徵層**：良醫象徵擅長「謀人」的開國君主，他們太專注於制度的設計，反而忽略了真正的威脅

一個比喻承載了三層意義——這就是方孝孺作為頂尖散文家的功力所在。

> **「唯積至誠、用大德，以結乎天心；使天眷其德，若慈母之保赤子而不忍釋。」**

全文的解決方案。方孝孺用「慈母保赤子」這個溫暖的比喻，將冷峻的「天道」轉化為有溫度的倫理訴求。這句話的邏輯層次是：
- 人類的智慮有界 → 所以不能只靠制度 → 必須讓「天」站在你這邊 → 讓天站在你這邊的唯一方法不是權謀，而是道德
- 天對有德者的眷顧，就像母親對孩子的愛——即使孩子愚笨，母親也不忍拋棄

這段話將全文從冷峻的歷史分析提升到溫暖的道德勸說，讀者讀到這裡，不再感到「天道難測」的絕望，而是感受到「積德行善」的希望。

> **「欲以區區之智，籠絡當世之務，而必後世之無危亡，此理之所必無者，而豈天道哉！」**

全文收束句，凌厲的反問，擲地有聲。方孝孺在前文用冷靜的歷史分析、精妙的哲學推導、溫暖的道德比喻說服了讀者，最後一句卻突然轉為凌厲的批判——「區區之智」「豈天道哉」——帶著強烈的諷刺語氣。這種從平和到凌厲的節奏轉換，製造了全文最強烈的情感衝擊，讓讀者在結尾處被狠狠地「敲醒」。這句反問也完成了全文的論證閉環：開篇提出問題（為什麼深慮仍會亡國？），結尾給出終極答案（因為天道不允許）。

## 4. 白話改編

> 考慮天下大事的人，常常謀劃那些困難的部分，卻忽略了簡單的部分；防備那些可怕的事情，卻遺漏了那些看似不用懷疑的小事。然而，災禍往往發生在他們忽略的地方，動亂常常起因於他們認為不足懷疑的事情——難道是他們的考慮不够周全嗎？不是——因為智慮所能到達的，只是人事中合乎常理的部分；而超出智力範圍之外的，叫做天道。

> 在秦朝的時候，秦始皇消滅諸侯、統一天下，他心想：周朝之所以滅亡，是因為諸侯太強大了。於是廢除封建制，改為郡縣制，以為從此可以不用打仗了，天子之位可以世代相傳——卻不知道漢高祖從田野中崛起，最終滅掉了秦朝的江山。

> 漢朝吸取了秦朝孤立無援的教訓，大力分封同姓諸侯王，認為同姓的親人，可以互相扶持，不會發生變亂——結果七國諸侯發動了篡位弒君的陰謀。漢武帝、漢宣帝以後，漸漸削弱諸侯勢力，分掉他們的權力，以為沒事了——結果王莽最終篡奪了漢朝的江山。

> 東漢光武帝吸取了西漢哀帝、平帝的教訓，曹魏吸取了東漢的教訓，晉朝吸取了曹魏的教訓——每個朝代都針對前朝滅亡的原因來做防備，而它們自己的滅亡，卻恰恰出於防備措施之外。

> 唐太宗聽說姓武的人將會殺害他的子孫，就在可疑的人當中搜捕要殺掉，而武則天每天侍奉在他身邊，他卻沒有察覺。宋太祖看到五代時期藩鎮割據足以挾制君主，就把武將的兵權全部解除，使他們力量削弱、容易控制——卻不知道自己的子孫最終被敵國所困。

> 這些人都有超人的智慧、蓋世的才能，對於治亂存亡的關鍵，思考得周詳、防備得嚴密了——他們考慮在這邊，禍患卻發生在那邊，最終導致混亂滅亡。為什麼呢？因為智慧可以用來謀劃人事，卻不能用來謀劃天道。

> 就像高明的醫生，他的兒子往往死於疾病；高明的巫師，他的兒子往往死於鬼怪——難道是他們善於救治別人、卻不善於保護自己的兒子嗎？不是——是因為他們善於謀劃人事，卻不善於謀劃天道。

> 古代的聖人知道天下後世的變化，不是智慮所能完全顧及、不是法令制度所能完全控制的——所以不敢放縱自己的私心和詭計，而是累積真誠、施行大德，來結交上天的心意；使上天眷顧他的德行，就像慈母保護嬰兒一樣不忍心放手。所以他們的子孫，即使有極其愚笨不堪、足以亡國的後代，上天最終也不忍心立刻滅亡他們——這才是真正長遠的考慮。

> 如果一個人不能讓上天護佑自己，卻想靠著區區的智慧，來籠絡當代的事務、保證後世沒有危險——這是道理上絕對不可能的事情，難道合乎天道嗎！
