# 〈刑賞忠厚之至論〉— 文章分析

> 蘇軾（1037–1101），字子瞻，號東坡居士，眉州眉山（今四川眉山）人。北宋文學家，「唐宋八大家」之首。嘉祐二年（1057）以二十二歲應禮部試，此文即其省試考卷，獲主考官歐陽修驚嘆，取為第二。此後歷經仁宗、英宗、神宗、哲宗、徽宗五朝，官至翰林學士、禮部尚書，一生雖屢遭貶謫，卻在詩、詞、文、書、畫五個領域均達頂尖水準。

## 一、文章結構分析

〈刑賞忠厚之至論〉全文約六百餘字，結構極其嚴謹，以「罪疑惟輕，功疑惟重」為核心命題，層層推進，從歷史視野到具體論證再到理論總結，一氣呵成。

**第一層：開門見山——以歷史視野立基（開篇至「見於虞、夏、商、周之書」）**

> 堯、舜、禹、湯、文、武、成、康之際，何其愛民之深，憂民之切，而待天下之以君子長者之道也！有一善，從而賞之，又從而詠歌嗟嘆之，所以樂其始而勉其終。有一不善，從而罰之，又從而哀矜懲創之，所以棄其舊而開其新。故其吁俞之聲，歡休慘戚，見於虞、夏、商、周之書。

文章開篇即展現極大的格局——從堯舜至周成王、康王，歷代聖王無一例外，都以「君子長者之道」對待人民。對善行，不僅賞賜還要歌頌（樂其始而勉其終）；對惡行，不僅懲罰還要哀憐（棄其舊而開其新）。蘇軾一開頭就將「刑」與「賞」統一在「教化」的目的之下——兩者都是為了讓人變好，而非為了懲罰或獎勵本身。

**第二層：引入經典——亮出核心論點（「成、康既沒」至「盡之矣」）**

> 成、康既沒，穆王立，而周道始衰。然猶命其臣呂侯，而告之以祥刑。其言曰：「罰疑從去。」然則聖人之所以為聖人者，蓋在於此矣。
>
> 《書》曰：「與其殺不辜，寧失不經。」——嗚呼！盡之矣！

蘇軾在此拋出全文的理論依據。他先引周穆王告呂侯的「祥刑」（善用刑罰）與「罰疑從去」（有疑問的罰款從輕處理），然後引出《尚書·大禹謨》中最核心的一句話——「與其殺不辜，寧失不經」。這句話是全篇文章的「題眼」：與其殺害無辜的人，寧可違反常規（放走有嫌疑的罪犯）。蘇軾以一句「嗚呼！盡之矣！」收束——用極強的感嘆語氣告訴讀者：經典的精神已經把話說盡了，真理就在於此。

**第三層：理論辨析——「仁可過，義不可過」（「可以賞」至「義不可過也」）**

> 可以賞，可以無賞，賞之過乎仁；可以罰，可以無罰，罰之過乎義。過乎仁，不失為君子；過乎義，則流而入於忍人。故仁可過也，義不可過也。

這是全文最有哲學深度的一段。蘇軾將「賞」與「刑」分別對應「仁」與「義」，並指出：在兩可的情況下，多賞（過乎仁）頂多是做了一個過於寬厚的人，仍然不失為君子；但多罰（過乎義），就會流為殘忍之人。因此「仁可過，義不可過」——寬厚再多也不為過，嚴厲多一分就是殘忍。這六個字是全文的邏輯中樞。

**第四層：高潮段落——「皋陶曰殺之三，堯曰宥之三」**

> 當堯之時，皋陶為士，將殺人。皋陶曰殺之三，堯曰宥之三。故天下畏皋陶執法之堅，而樂堯用刑之寬。

這是中國科舉史上最著名的段落。蘇軾創造了一個歷史上從未發生的場景——皋陶三次堅持判死，堯三次寬恕赦免——用一個充滿戲劇張力的故事來說明「罪疑惟輕」的道理。皋陶的「殺之三」代表法律的尊嚴，堯的「宥之三」代表仁政的溫度，兩者相互制衡，才構成理想的司法制度。這個虛構典故的妙處在於：它把一個抽象的司法哲學命題變成了讀者腦海中揮之不去的畫面。

**第五層：理論深化——賞刑的教化本質（「古者賞不以爵祿」至「忠厚之至也」）**

> 古者賞不以爵祿，刑不以刀鋸……先王知天下之善不勝賞，而爵祿不足以勸也；知天下之惡不勝刑，而刀鋸不足以裁也。是故疑則舉而歸之於仁。

蘇軾進一步深化論證：賞賜的本質不是給爵祿，刑罰的本質不是用刀鋸——因為爵祿和刀鋸的覆蓋範圍有限，無法真正勸善止惡。聖王的做法是：遇到疑難，一律歸結於「仁」。用君子長者之道對待天下人，天下人也自然趨向於君子長者之道。最後點題：「故曰：忠厚之至也。」

**第六層：總結論證——引《詩經》《春秋》收束全文（「《詩》曰」至結尾）**

> 《詩》曰：「君子如祉，亂庶遄已；君子如怒，亂庶遄沮。」夫君子之己亂，豈有異術哉？時其喜怒，而無失乎仁而已矣！《春秋》之義，立法貴嚴，而責人貴寬。因其褒貶之義，以制賞罰，亦忠厚之至也。

蘇軾引用《詩經·小雅·巧言》的句子——君子行賞（祉）則禍亂將止，君子發怒（怒）則禍亂將消——將「賞」與「刑」統一於「喜怒」之下。他點明：君子平息禍亂沒有什麼祕訣，不過是恰當表達喜怒，而不離開「仁」的根本。最後以《春秋》的義理收束——立法要嚴厲，責備人要寬厚——以《春秋》的褒貶原則來制定賞罰，這才是忠厚的極致。

### 起承轉合

| 結構 | 對應段落 | 內容 |
|:----:|:--------:|------|
| **起** | 第1層 | 以堯舜至周的歷史視野開篇——聖王皆以「君子長者之道」對待人民，賞罰皆為教化 |
| **承** | 第2-3層 | 引出經典論據——「與其殺不辜，寧失不經」，並推導出「仁可過，義不可過」的核心論點 |
| **轉** | 第4-5層 | 以「皋陶曰殺之三，堯曰宥之三」的虛構典故將抽象道理具象化——轉入賞刑之教化本質的深度論證 |
| **合** | 第6層 | 引《詩經》《春秋》總束——喜怒不離乎仁，忠厚之至，完成論證閉環 |

## 二、寫作技巧分析

### 核心寫作策略

**一、「以意逆志」——不引經而用經**

蘇軾全文幾乎沒有直接引用經典原文（除末尾引《詩經》外），但每一句話都符合經典的精神。他讀《尚書》讀的不是文字，而是「罪疑惟輕」的原則；他創造「皋陶曰殺之三」的典故，是為了說出經典沒有直接說出的、但符合經典精神的道理。這種「用自己的語言重說經典」的能力，是蘇軾日後成為一代文豪的根本——他不做經典的奴隸，而做經典的代言人。

**二、虛構典故——以想像力征服說理**

「皋陶曰殺之三，堯曰宥之三」是全文最具創造力的段落。蘇軾虛構了一個場景，卻比任何真實史料都更能說明「忠厚」的道理。這個故事的說服力來自於它的「具體性」——三次對話的對峙，讓讀者親眼看到了「仁厚如何在司法決策中發揮作用」。抽象的道理（罪疑惟輕）被具象化為一個充滿張力的場景，這是最高明的修辭。

**三、層層遞進的論證節奏**

全文的論證節奏由寬到窄、由遠到近：
- 寬闊的歷史視野（堯舜至周）→ 具體經典依據（《尚書》） → 哲學辨析（仁可過義不可過） → 生動的典故（皋陶與堯） → 理論深化 → 經典總結
- 這種「放大→縮小→聚焦→展開→收束」的節奏，讓讀者在不知不覺中跟隨蘇軾的邏輯前進，毫無滯礙。

**四、以「勸百諷一」的方式處理刑罰**

傳統論文中討論刑罰往往偏向嚴厲（法家傳統），蘇軾卻用大量篇幅談賞賜（正面引導）而少談刑罰（反面警示）。他刻意將「賞」放在前面反覆論證，再以「刑」作為對照——這種「先談賞、後說刑、以賞統攝刑」的結構安排，本身就是對「忠厚」的實踐：先讓讀者感受到仁厚的溫暖，再討論刑罰的謹慎。

### 修辭手法

| 手法 | 原文 | 效果 |
|:----|------|------|
| **排比** | 「有一善，從而賞之，又從而詠歌嗟嘆之……有一不善，從而罰之，又從而哀矜懲創之」 | 善與惡、賞與罰的對稱排比，節奏感極強，展現聖王對待善惡的截然不同態度 |
| **反覆對仗** | 「皋陶曰殺之三，堯曰宥之三」 | 三次「殺」與三次「宥」的對峙，形成極強的戲劇張力，讀者腦海會反覆浮現這個場景 |
| **遞進** | 「可以賞，可以無賞……可以罰，可以無罰」 | 從兩可的情況出發遞進到「過乎仁」與「過乎義」的結果，邏輯層層深入 |
| **對比** | 「過乎仁，不失為君子；過乎義，則流而入於忍人」 | 「仁」與「義」、「君子」與「忍人」的對比鮮明，讓讀者直觀感受「仁可過、義不可過」的合理性 |
| **感嘆** | 「嗚呼！盡之矣！」 | 閱讀至此，讀者也跟著感嘆——經典說得真好、真徹底！一句感嘆強化了經典引用的權威性 |
| **設問（隱含）** | 「夫君子之己亂，豈有異術哉？」 | 先拋出問題引起讀者思考，再用「時其喜怒而無失乎仁而已矣」給出答案，增強說服力 |
| **引用** | 引《書》曰「與其殺不辜，寧失不經」、引《詩》「君子如祉，亂庶遄已」、引《春秋》之義 | 三處引用分佈在文章的前、中、後，形成經典依據的三重支撐，層層強化 |
| **比喻（隱含）** | 「以君子長者之道待天下」 | 將「忠厚」比作「君子長者」的形象——溫和、寬厚、有耐心，讓抽象概念具體化 |
| **層遞** | 「賞以爵祿……刑以刀鋸……先王知天下之善不勝賞……知天下之惡不勝刑……是故疑則舉而歸之於仁」 | 從具體刑賞手段（爵祿刀鋸）推到其局限性，再推到聖王的解決方案（歸之於仁），一氣呵成 |
| **總收排比** | 「立法貴嚴，而責人貴寬」 | 六個字，一組對仗，收束全文核心思想——法律要嚴格，但責備人要寬厚 |

### 全篇最核心的技巧：不引經而用經

這是〈刑賞忠厚之至論〉在科舉史上最令人驚嘆的成就。全篇文章雖以儒家經典為依據，但幾乎沒有直接引用經文——唯一明確的引用是結尾處的《詩經》和《尚書》各一處。蘇軾的論證方式，是「用自己的話說出經典的精神」：

- 「與其殺不辜，寧失不經」——雖然出自《尚書》，但蘇軾沒有簡單地列舉，而是大嘆「嗚呼！盡之矣！」——讓這句話從經文變成了自己的感嘆
- 「皋陶曰殺之三，堯曰宥之三」——完全虛構，但完美詮釋了《尚書·大禹謨》的「罪疑惟輕」
- 全篇的「君子長者之道」——不見於任何一部經典的字面，卻是所有經典的共同精神

這種「不引經而用經」的寫法，滿足了科舉考試「以經解經」的要求，卻超越了「引經據典」的規矩——它不是從經典找證據，而是讓自己成為經典的發言人。歐陽修讀完驚為天人，正是因為他看到了這種「化經為己」的能力。

## 三、佳句賞析

### 佳句一：「與其殺不辜，寧失不經。」——嗚呼！盡之矣！

**現代漢語：** 與其殺害無辜的人，寧可違反常規（放走有嫌疑的罪犯）。——唉！這話把道理說盡了！

**賞析：**
這是全篇文章的「題眼」，也是思想核心。蘇軾引用《尚書·大禹謨》的這句話後，緊接一句「嗚呼！盡之矣！」——他不是在引用經典，而是在和經典對話。這句感嘆讓「與其殺不辜，寧失不經」這句古文當中的古文，忽然活了起來，變成了蘇軾自己的聲音。

從法學思想的角度看，這句話是「疑罪從無」原則的中國古典表達。蘇軾在近一千年前就明確提出：司法判決中最可怕的不是放走了罪犯（失出），而是冤枉了好人（失入）。因為放走罪犯只是讓一個可能的壞人逃脫，但冤枉好人，就是國家用暴力懲罰了一個無辜的人——這比放任犯罪更嚴重。

### 佳句二：「當堯之時，皋陶為士，將殺人。皋陶曰殺之三，堯曰宥之三。」

**現代漢語：** 在堯的時代，皋陶擔任法官，將要判處一個人死刑。皋陶三次說「殺了他」，堯三次說「寬恕他」。

**賞析：**
這是中國科舉史上最著名的「神句」。它的力量來自三個層面：

1. **想像力的勝利**：一個虛構的場景，比任何真實史料都更能說明「忠厚」的道理。蘇軾用「三次殺、三次宥」的對峙，把抽象的司法哲學變成了讀者腦海中揮之不去的畫面。
2. **戲劇張力**：「殺之三」和「宥之三」形成強烈的節奏衝擊——越堅持要殺，越堅持寬恕；越對峙，越凸顯仁厚的力量。
3. **科舉冒險**：在決定命運的科舉考場上創造一個不存在的典故，需要極大的自信。蘇軾回答歐陽修「想當然耳」——這三個字比文章本身更傳奇，展現了一個二十二歲年輕人對自己學問的絕對把控。

這個句子之所以傳頌千年，不僅因為寫得好，更因為它背後的故事——歐陽修的驚嘆、蘇軾的從容、以及那句敢於挑戰一切規則的「想當然耳」。

### 佳句三：「可以賞，可以無賞，賞之過乎仁；可以罰，可以無罰，罰之過乎義。過乎仁，不失為君子；過乎義，則流而入於忍人。故仁可過也，義不可過也。」

**現代漢語：** 可以賞賜也可以不賞賜的情況下，賞賜了是出於仁厚；可以懲罰也可以不懲罰的情況下，懲罰了是出於道義。出於仁厚而過頭，仍然不失為君子；出於道義而過頭，就會流為殘忍的人。所以仁厚可以過分，道義不可以過分。

**賞析：**
這是全文最有哲學深度的一段，也是蘇軾論證的核心邏輯樞紐。他將「賞」對應「仁」、「刑」對應「義」，然後提出一個大膽的命題：仁可以過，義不可以過。這在當時是一個極具挑戰性的觀點——儒家傳統講「仁義並重」，蘇軾卻說「仁重於義」。

更精彩的是他的推論邏輯：如果賞多了（過乎仁），最多被人說太好說話（不失為君子）；但如果罰多了（過乎義），就會變成殘忍之人（流而入於忍人）。「仁可過，義不可過」六個字，是蘇軾對「忠厚」原則的最高提煉——他用最精煉的語言，說出了全文的核心判斷標準。

### 佳句四：「古者賞不以爵祿，刑不以刀鋸……先王知天下之善不勝賞，而爵祿不足以勸也；知天下之惡不勝刑，而刀鋸不足以裁也。是故疑則舉而歸之於仁。」

**現代漢語：** 古時候賞賜不靠爵位俸祿，刑罰不靠刀鋸刑具……先王知道天下的善行是賞賜不完的，爵祿不足以勸人行善；知道天下的惡行是懲罰不完的，刀鋸不足以制止罪惡。因此遇到疑難，一律歸結於仁厚。

**賞析：**
這段話展現了蘇軾對「賞罰制度局限性」的深刻洞察。他指出了一個根本性的矛盾：善行無窮，但爵祿有限；惡行無窮，但刀鋸有限。有限的工具無法控制無限的社會——因此，單純依靠物質獎勵和肉體懲罰是不夠的。聖王的解決方案是超越制度層面的「歸之於仁」——用道德教化來彌補制度的不足。

這段論證的現代意義極強：任何制度都有邊際效應遞減的問題。法律無法解決所有的社會問題，最終還是要靠人心與道德來補充。蘇軾不是反法治，而是指出法治的邊界——這在北宋的政治哲學中是非常先進的觀念。

### 佳句五：「《詩》曰：『君子如祉，亂庶遄已；君子如怒，亂庶遄沮。』夫君子之己亂，豈有異術哉？時其喜怒，而無失乎仁而已矣！」

**現代漢語：** 《詩經》說：「君子如果行賞，禍亂差不多就要停止了；君子如果發怒，禍亂差不多就要平息了。」君子平息禍亂，難道有什麼特別的方法嗎？不過是恰當地表達喜怒，而不偏離仁的根本罷了！

**賞析：**
這是全文的收束段落，也是最精彩的一處經典活用。蘇軾引用《詩經·小雅·巧言》的句子，將「賞」與「刑」統一於「喜怒」這對日常概念之下——賞就是「喜」，刑就是「怒」。然後他用一個反問句：「夫君子之己亂，豈有異術哉？」——君子平息禍亂有什麼祕訣嗎？答案平淡到令人意外：「時其喜怒，而無失乎仁而已矣。」

這句話的力量正在於它的「平淡」。蘇軾告訴我們：最好的治理沒有什麼玄妙的技巧，不過是在該高興的時候高興（賞），該生氣的時候生氣（罰），而所有的喜怒都不離開「仁」的根本。這就是「忠厚之至」的最終解釋——不是複雜的制度設計，而是執政者內心對「仁」的堅守。

## 四、白話改編

堯、舜、禹、湯、文王、武王、成王、康王的時代——他們對百姓的愛護是多麼深厚，憂慮是多麼懇切，用君子長者的態度對待天下人是多麼真誠！有人做了一件好事，他們不但要賞賜他，還要歌頌讚美他，這是為了讓他的善行有一個好的開始，並鼓勵他堅持到底。有人做了一件壞事，他們不但要懲罰他，還要同情哀憐他，這是為了讓他拋棄過去的錯誤，開闢新的人生。因此，那些贊同和反對的聲音、歡樂和悲傷的表情，都記載在虞、夏、商、周的文獻之中。

成王、康王去世之後，穆王繼位，周朝開始衰落了。但即使如此，穆王仍然告誡他的大臣呂侯，要善用刑罰。他說：「對有疑問的罰款，從輕處理。」那麼，聖人之所以成為聖人，大概就在於此吧。

《尚書》說：「與其殺害無辜的人，寧可違反常規（放走有嫌疑的罪犯）。」——唉！這句話把道理說盡了！

可以賞賜也可以不賞賜的情況下，賞賜了是出於仁厚；可以懲罰也可以不懲罰的情況下，懲罰了是出於道義。出於仁厚而過頭，仍然不失為君子；出於道義而過頭，就會流為殘忍的人。所以仁厚可以過分，道義不可以過分。

古時候賞賜不靠爵位俸祿，刑罰不靠刀鋸刑具。如果賞賜只靠爵祿，那麼賞賜的效果就只能局限於爵祿所及的人身上；如果刑罰只靠刀鋸，那麼刑罰的威嚴就只能施加於刀鋸所及的地方。先王知道天下的善行是賞賜不完的，爵祿不足以勸導所有人向善；知道天下的惡行是懲罰不完的，刀鋸不足以制止所有的罪惡。因此，遇到疑難的情況，一律歸結於仁厚。用君子長者的態度對待天下人，讓天下人相互引導、共同歸向君子長者的道理。所以說：這才是忠厚的極致啊。

在堯的時代，皋陶擔任法官，將要判處一個人死刑。皋陶三次說「殺了他」，堯三次說「寬恕他」。因此，天下人敬畏皋陶執法的堅定，卻讚許堯用刑的寬厚。

《詩經》說：「君子如果行賞，禍亂差不多就要停止了；君子如果發怒，禍亂差不多就要平息了。」君子平息禍亂，難道有什麼特別的方法嗎？不過是恰當地表達喜怒，而不偏離仁的根本罷了。《春秋》的義理是：制定法律要嚴格，但責備人要寬厚。按照《春秋》褒貶的義理來制定賞罰制度——這也是忠厚的極致啊。
