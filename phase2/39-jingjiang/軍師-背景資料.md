# 軍師分析：〈敬姜論勞逸〉背景資料

---

## 座標定位

| 維度 | 定位 |
|------|------|
| **典籍** | 《國語·魯語下》 |
| **編選** | 《古文觀止》卷三 |
| **西曆** | 春秋中晚期，約公元前六世紀初 |
| **地域** | 魯國（今山東曲阜） |
| **人物關係** | 敬姜（公父文伯之母，季悼子之子公父穆伯之妻）、公父文伯（魯國大夫，季氏後裔） |
| **文體** | 諫訓體——母親訓誡兒子的對話記錄 |

此文出自《國語》，乃先秦國別史之祖，以記言為主，所錄多為春秋時期各國卿大夫的諫諍議論之辭。敬姜之言，實為中國古代勞動倫理思想最早的系統表述之一。

---

## 時代壓力：春秋貴族的奢靡危機

### 禮崩樂壞，貴族腐化

春秋中晚期，周室衰微，諸侯兼併加劇，而各國內部亦出現深刻的階層變動：

1. **世卿世祿之弊**——魯國自三桓（季孫、孟孫、叔孫）專政以來，公族大夫世代把持朝政，不事生產，養尊處優。貴族子弟從小錦衣玉食，不知稼穡艱難，更不懂民間疾苦。

2. **勞動被鄙視**——在周禮等級秩序中，「勞心者治人，勞力者治於人」的觀念已逐漸固化。君子以「不親庶務」為貴，體力勞動被視為賤役，僅屬庶人、奴隸之事。公父文伯退朝見母親猶在紡織，第一反應便是：「以歜之家而主猶績，懼忤季孫之怒也……」（以我們這樣的家世，母親還在紡織，恐怕會觸怒季孫氏吧？）——這句話本身就暴露了當時貴族階層的普遍心態：**體面人家不該親手勞作。**

3. **魯國的特殊危機**——魯為周公封地，號稱禮樂之邦。然而到了春秋晚期，魯國內部「公室卑，三桓強」，禮樂制度日益空心化，徒存形式而失其精神。敬姜以母親之身對兒子發出勞動倫理的警告，實則是在禮樂崩壞的時代呼聲中，以最樸素的方式呼籲回歸周初「勤儉興國」的古訓。

### 敬姜的個人背景：一位特殊的貴族女性

- 敬姜出身莒國（紀國後裔），嫁入魯國季氏家族，為公父穆伯之妻。
- 丈夫早逝，她以寡母之身獨立教養兒子公父文伯，使其學成出仕，官至魯國大夫。
- 她雖為貴族命婦，卻畢生堅持紡織勞作，不因身分改變而荒廢。
- 孔子的老師——一說孔子曾問禮於敬姜（《禮記》載孔子讚敬姜「知禮」）；另有說法稱孔子曾派人向敬姜求教，敬姜以治家之道喻治國之禮，孔子歎服。
- 她的言行不僅是母教的典範，更被儒家吸收為「修身齊家治國」理念的具體實踐。

## 關鍵人物

### 敬姜

- **身分：** 公父穆伯之妻、公父文伯之母。季悼子之子公父穆伯早逝，敬姜守寡，獨力教子。
- **核心主張：** 「勞則思，思則善心生；逸則淫，淫則忘善，忘善則噁心生。」
- **行動即論證：** 她不以言說教，而是身體力行——兒子當了大夫，她仍在家中紡織。這不是貧困所致，而是刻意為之的身教。她的針線，本身就是一篇文章。
- **戰略思維：** 敬姜深知「由奢入儉難」的人性規律。她擔心的不是兒子今日的品性，而是權力與財富滋生安逸、安逸滋生腐化的長期趨勢。她的教訓不是一次性的訓話，而是一套預防性的國策性思維——以勞動為防腐劑，以勤儉為免疫力。
- **地位：** 她在先秦典籍中被多處提及（《國語》《禮記》《列女傳》），是中國古代最具典範意義的賢母之一。後世「孟母三遷」「岳母刺字」等故事，所承襲的正是敬姜的母教傳統。

### 公父文伯（名歜，一作公父歜）

- **身分：** 敬姜之子，魯國大夫，季氏家族的旁支。
- **性格：** 從他退朝後的反應來看，他並非紈絝子弟——他對母親仍親手紡織感到不安，是因為他已在貴族社交圈中內化了「貴族不應勞動」的價值觀，而非對母親不敬。這恰恰說明了社會風氣對人的浸染有多深。
- **困境：** 他是一個「體面的人」落入「體面的陷阱」的案例。他的擔憂來自於階級壓力（怕季孫氏不高興），而非對母親勞動價值的否定。他需要的是對勞動意義的重新認知——而敬姜給了他這個認知。

### 配角：季孫氏

- 魯國實際掌權者，文伯所屬的宗族領袖。文中雖未直接登場，卻是文伯的參照點：「懼忤季孫之怒也」——季孫對勞動的態度，即是貴族圈對勞動的態度。敬姜駁斥的，正是這個以「體面」為名的階級偏見。

## 寫作意圖：為什麼一位母親對兒子的教訓能成為千古名文？

### 1. 勞動倫理的最早系統表述

敬姜之言，是中國思想史上最早將勞動提升到人性修養根本層次的論述。她的邏輯鏈極其清晰：

```
勞動 → 勤勉思考（思） → 善心生
安逸 → 放縱享樂（淫） → 忘善 → 噁心生
```

這不是道德的隨感，而是一套嚴謹的因果推理：**勞動是善的心理機制，安逸是惡的心靈溫床。** 這個論述比西方新教倫理的「勞動即天職」早了近兩千年。

### 2. 治家即治國的隱喻

敬姜並非只在談家庭倫理。她將紡織勞作的微物，一路引向天子、諸侯、卿大夫治國安邦的大道：

> 「天子大采朝日……諸侯朝修天子之業命……卿大夫朝考其職……士朝受業……庶人以下，明而動，晦而休，無日以怠……」

她構築了一套「人人皆須勞動」的社會秩序圖景——上至天子，下至庶人，無一人可以怠惰。勞動不是下賤，而是所有階層共同的生存法則。**治家的織機與治國的綱紀，不過是一體兩面。**

### 3. 女性聲音的珍稀典範

在男性書寫主導的先秦文獻中，能以完整論述、系統邏輯、深遠影響流傳後世的女性話語極為罕見。敬姜是極少數以女性身分、以母親角色、以勞動倫理為核心，在正統文獻中佔有一席之地的人物。這篇文章不僅是古文選本的常客，更是中國女性教育史和母教傳統的源頭文獻之一。

### 4. 對「君子遠庖廚」的階級虛榮心的批判

文伯那句「以歜之家而主猶績，懼忤季孫之怒也」，是一個時代的註腳——它暴露了貴族階層將勞動視為恥辱的階級偏見。敬姜的反駁，實質上是對這種虛榮心的一記重擊：**真正的體面不在於不做什麼，而在於知道什麼才是一個人該做的事。**

### 5. 儒家思想的先聲

敬姜的勞動倫理與儒家「敬業樂群」「克勤克儉」「先勞而後祿」的精神完全一致。孔子讚她「知禮」，絕非偶然。她以母教傳遞的勞動價值觀，後來被儒家系統化，成為中國文化中最深層的倫理基因之一。

## 原文精要

> 公父文伯退朝，朝其母，其母方績。
>
> 文伯曰：「以歜之家而主猶績，懼忤季孫之怒也。其以歜為不能事主乎！」
>
> 其母歎曰：「魯其亡乎！使童子備官而未之聞耶？居，吾語女。
>
> 昔聖王之處民也，擇瘠土而處之，勞其民而用之，故長王天下。夫民勞則思，思則善心生；逸則淫，淫則忘善，忘善則噁心生。
>
> 沃土之民不材，逸也；瘠土之民莫不嚮義，勞也。」

——此為全文立意所在，語出《國語·魯語下》。敬姜以「沃土瘠土」為喻，以自然界的現象論證勞動與品德的因果關係，樸素而深刻。

---

## 一句話總結

> **敬姜以織機為講壇，以勞動為經、倫理為緯，在春秋貴族奢靡頹廢的風氣中，為兒子同時為後世編織出一套樸素而堅韌的勞動倫理——「勞則思，思則善心生」，這八個字是中國文化對人性修養最深層的認知之一，而它出自一位母親縫紉時對兒子的叮囑。**
