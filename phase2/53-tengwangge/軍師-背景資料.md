## 🌪️ 時代壓力：初唐文學的過渡之戰

### 文學戰場——從六朝餘燼到盛唐號角

西元 675 年，唐帝國建國已逾半世紀。太宗貞觀之治（627–649）為帝國打下軍事與經濟根基，但**文學領域仍處於漫長的過渡期**。

初唐文學面臨的核心矛盾是：

> **六朝駢文的華麗遺產 vs. 盛唐雄渾氣象的呼喚**

隋及唐初，文壇主流仍是齊梁以來的駢文傳統——講究對仗、典故、聲律、辭藻。上官儀的「上官體」風靡一時，「綺錯婉媚」成為時尚。但在這種精緻之下，**內容往往空洞，情感往往虛浮**。

到了高宗朝，一股新的力量正在醞釀：

- **王勃**與楊炯、盧照鄰、駱賓王並稱「初唐四傑」，率先喊出「**反上官體**」的口號，主張文章要有真情、有骨氣。
- 但四傑仍大量使用駢文形式——他們的革命不是形式革命，而是**精神革命**：用駢文的殼，裝盛唐的魂。
- 真正的突破要等到陳子昂（約比王勃晚 20 年）高呼「**文章道弊五百年矣**」，才徹底開啟盛唐之音。

**王勃站在兩個時代的斷崖上**：他的身後是六朝駢文的溫柔鄉，前方是盛唐詩文的萬里晴空。他要在駢文格式的限制之內，把文章寫出超越格式的力量。

### 政治背景——帝國的穩定與暗流

高宗朝的政治相對穩定，但並非毫無壓力：

- **武后（武則天）勢力崛起**：上元元年（674 年），高宗稱「天皇」，武后稱「天后」，二人並稱「二聖」。朝中權力結構正在悄然傾斜。
- **科舉制度的成熟**：唐代以詩賦取士的制度逐步完善，文人通過文章獲得政治地位的管道越來越明確。這意味著——**文章不僅是文學，更是政治資本**。
- **邊疆戰爭持續**：吐蕃、突厥不斷騷擾邊境，「關山難越」不僅是文學修辭，更是帝國的真實地理圖景。

王勃本人的仕途極為坎坷。他曾在沛王府任修撰，因戲作〈檄英王雞文〉觸怒高宗，被逐出長安。後任虢州參軍，又因殺死官奴（一說是被陷害）而獲罪，甚至連累父親王福畤被貶為交趾（今越南北部）令。**西元 675 年，王勃正是前往交趾探望父親的途中，路過洪州**。

一個仕途全毀、前途未卜的 26 歲天才，走進了一場滿是權貴的宴會。

## 👥 關鍵人物

### 王勃（650–676）——主角·流星

| 項目 | 內容 |
|------|------|
| **字** | 子安 |
| **籍貫** | 絳州龍門（今山西河津） |
| **家世** | 出身儒學世家，祖父王通為隋末大儒 |
| **天才表現** | 六歲能文，九歲讀顏師古注《漢書》即能指出訛誤，十六歲應幽素科舉及第，授朝散郎——**大唐最年輕的命官之一** |
| **致命傷** | 恃才傲物，不懂政治分寸。一篇〈檄英王雞文〉斷送前程 |
| **生命終點** | 作〈滕王閣序〉後不久，前往交趾途中渡海溺水，驚悸而卒——**年僅 27 歲** |

王勃的一生是一場**天才的連續引爆**：16 歲入仕，26 歲作千古絕唱，27 歲死於海上。他的作品不多（因早逝），但每一篇都像最後一篇——用盡全身力氣地寫。

### 閻伯嶼（閻都督）——舞台搭建者

洪州都督，滕王閣重修的主持者。史載：

- 重修滕王閣後舉辦盛大宴會，欲藉此讓自己的女婿吳子章展示文采（一說預先讓女婿寫好序文）。
- 宴會上假意請眾賓客作序，眾人皆辭讓——**唯獨王勃「不辭」，提筆便寫**。
- 閻都督起初大怒（「迎新客」為由拂袖而去），但派人監看王勃所寫。當聽到「**落霞與孤鶩齊飛，秋水共長天一色**」時，閻都督驚呼：「**此真天才，當垂不朽矣！**」遂請王勃完成全文，賓主盡歡。

**閻都督的角色極具戲劇性**：他本想為女婿搭台，卻意外為一個路過的落魄天才搭建了歷史最高規格的舞台。他的「發怒→震驚→折服」三部曲，本身就是〈滕王閣序〉傳奇的一部分。

### 與會賓客——見證者群像

在場的賓客包括洪州當地的文武官員、文人名士。他們是：見證一次傳奇即興創作的**現場觀眾**。

根據序中「**都督閻公之雅望，棨戟遙臨；宇文新州之懿範，襜帷暫駐**」，可知名單中至少還有：

- **宇文新州**：一位路過的刺史級官員
- **孟學士**：「孟學士之詞宗」，當地名士
- **王將軍**：「王將軍之武庫」，本地武將

這些人構成了〈滕王閣序〉的**社交語境**——王勃面對的不是歷史，而是一屋子審視的目光。

## ✍️ 寫作意圖：26 歲的遺書，寫在真正死亡之前

### 表面意圖——宴會應酬之作

從最表層看，〈滕王閣序〉是一篇標準的宴會序文，功能包括：

1. **讚美東道主**——「都督閻公之雅望」
2. **描寫景色**——滕王閣與贛江的壯麗風光
3. **記錄盛會**——「四美具，二難并」
4. **以詩作結**——序末附七律一首

如果僅止於此，它不過是唐代無數宴會應酬文中比較華麗的一篇。

### 深層意圖——命運的預言

但王勃在標準格式之下，塞進了一份**極度私人的宣言**。請注意這幾段文字的順序：

```
層巒聳翠，上出重霄；飛閣流丹，下臨無地。
           ↓ （風景由外轉內）
天高地迥，覺宇宙之無窮；興盡悲來，識盈虛之有數。
           ↓ （由宴會的歡樂轉入個人的孤獨）
關山難越，誰悲失路之人？萍水相逢，盡是他鄉之客。
           ↓ （由個人擴大到所有「失路」之人）
馮唐易老，李廣難封。屈賈誼於長沙，非無聖主；竄梁鴻於海曲，豈乏明時？
```

**從極樂寫到極悲，從滕王閣寫到自己的命運。**

這就是王勃最驚人的地方：在一場慶祝重修落成的歡樂宴會上，他當著滿屋子權貴的面，說出了自己——一個仕途失敗、父親遭貶、前途未卜的26歲青年——的孤獨與委屈。

更驚人的是，他寫的不是「我多慘」的哭訴，而是：

> **「老當益壯，寧移白首之心？窮且益堅，不墜青雲之志。」**

這不是一個勝利者的宣言，而是一個失敗者在所有人面前**拒絕承認失敗**的宣言。

### 為什麼說它是「遺書」？

王勃在作序後不久渡海溺水身亡。從結果回看，〈滕王閣序〉中反覆出現的**水意象**和**終點感**令人不寒而慄：

- 「**漁舟唱晚，響窮彭蠡之濱；雁陣驚寒，聲斷衡陽之浦**」——每一句都在說「結束」。
- 「**天高地迥，覺宇宙之無窮**」——宇宙無窮，而人之生命短暫。
- 「**嗚呼！時運不齊，命途多舛**」——他對自己命運的預判完全正確。

**他彷彿在死前就已知道自己要死了。** 這不是迷信，而是一個天才對自身生命能量的極限感知——他知道自己快燒完了，所以要在熄滅之前，把所有的光一次釋放。

### 歷史的現場：即興表演的權力遊戲

不要忘記：這是一篇**即興之作**。在眾目睽睽之下，王勃沒有打草稿，沒有修改空間。他需要在二十秒內決定下一句的對仗、典故、平仄。

這不僅是文采的問題，更是**心理素質的問題**：

- 如果寫砸了，他就是一個在長官面前丟臉的路人。
- 如果他拒絕寫，他就是一個不敢接招的文人。
- 如果他寫得好，他就在一個全權貴的場合證明了：**一個沒有政治地位的天才，也可以碾壓所有有政治地位的人。**

王勃選擇了第三條路。他用〈滕王閣序〉向整個唐代官場證明了一個真理：**權力可以決定你的官位，但決定不了你在歷史上的位置。**
