# 屈原列傳 · 背景資料

> **軍師按：** 以下為研讀〈屈原列傳〉前之戰略情資，請參閱。

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **出處** | 《史記》卷八十四·屈原賈生列傳第二十四 |
| **選本** | 《古文觀止》卷五·漢文 |
| **作者** | 司馬遷（太史公） |
| **定位** | 屈原與賈誼合傳，以「不遇」為經，以「忠而見疑」為緯，雙星並耀 |

> 此文位於《史記》列傳之第二十四卷。司馬遷將屈原與賈誼合為一傳，非僅時序相近，實因二人命運如出一轍——皆懷經世之才，皆遭讒被放，皆以文字鑄千古之悲。此乃太史公精心之布局。

## 二、時代壓力

### 宏觀局勢：戰國後期 · 合縱連橫之終局

| 時間軸 | 事件 | 對楚國之影響 |
|--------|------|------------|
| 楚懷王前期（約前328－前313） | 楚為合縱長，與齊結盟抗秦 | 屈原受重用，修明法度，聯齊抗秦——楚國國勢達短期高點 |
| 秦惠王遣張儀入楚（前313） | 張儀詐諾割商於之地六百里，離間楚齊 | 懷王貪而信之，絕齊——此為楚國命運之**致命轉捩點** |
| 懷王受騙怒而攻秦（前312） | 丹陽之戰、藍田之戰，楚軍大敗 | 楚國元氣大傷，喪失漢中之地 |
| 懷王入秦被扣（前299） | 秦昭王邀懷王會武關，懷王不聽屈原勸諫，終被劫持 | 楚國淪為天下笑柄，懷王客死咸陽 |
| 頃襄王即位（前298） | 親秦派當權，屈原被放逐江南 | 屈原流放，楚國加速腐敗 |
| 秦將白起拔郢（前278） | 秦軍攻破楚國都城郢 | **楚國滅亡已成定局**，屈原懷石投江 |

### 軍事態勢解讀

```
        秦（虎狼之師）
          │
    ┌─────┴─────┐
    │            │
  楚（合縱）   齊（盟友）
    │            │
    └──張儀離間──┘  ← 外交敗筆在此
          │
    楚孤立無援
          │
    秦步步進逼
          │
    郢都淪陷 → 屈原投江
```

### 楚國內部結構性腐敗

- **貴族專權**：昭、屈、景三大氏把持朝政，盤根錯節
- **讒佞當道**：上官大夫靳尚、令尹子蘭、鄭袖（懷王寵姬）結黨營私
- **外交搖擺**：合縱派（屈原）與親秦派（上官大夫等）內鬥激烈，政策反覆
- **法治敗壞**：屈原訂立法令遭奪稿篡改，貴族利益凌駕國家之上

> **軍師判斷：** 楚國之亡，非亡於秦之強，而亡於內政之腐、外交之愚、人才之棄。屈原之放逐，乃楚國自我摧折棟樑之象徵。

## 三、關鍵人物

### ① 屈原（約前340－前278）

| 屬性 | 內容 |
|------|------|
| **身份** | 楚國貴族，與楚王同姓（羋姓屈氏） |
| **職位** | 三閭大夫（掌管王族三姓事務），左徒（僅次於令尹之高官） |
| **核心特質** | 博聞強識，明於治亂，嫻於辭令。內修法度以強國，外聯齊國以抗秦 |
| **文學成就** | 開創楚辭體，《離騷》《九章》《九歌》《天問》等不朽之作 |
| **政治悲劇** | **忠而被謗，信而見疑**——盡忠報國，反遭流放 |
| **最後一役** | 秦破郢都，懷石自沉汨羅江，以身殉國 |

> **軍師評語：** 屈子之才，足以安天下；屈子之忠，足以感天地。然生不逢時，遇主不明，終以江水洗冤。後世文人讀至此，無不扼腕。

### ② 楚懷王（？－前296）

| 屬性 | 內容 |
|------|------|
| **初始表現** | 早年曾重用屈原，聯齊抗秦，有中興氣象 |
| **性格致命傷** | 貪圖小利、耳根軟、無定見、好大喜功 |
| **三大敗筆** | ① 信張儀而絕齊——自斷臂膀 ② 怒而伐秦——不計代價 ③ 入秦武關——自投羅網 |
| **最終下場** | 被秦扣留三年，客死咸陽，屍體運回楚國時「楚人皆憐之，如悲親戚」 |

> **軍師評語：** 懷王非暴君，乃庸主也。庸主之害，甚於暴君——暴君有綱，庸主無斷。屈原遇此君，可謂英雄無用武之地。

### ③ 上官大夫（靳尚）

| 屬性 | 內容 |
|------|------|
| **角色定位** | 奸臣之典型，讒佞之化身 |
| **關鍵行動** | ① 企圖奪屈原草稿邀功，遭拒後進讒：「王使屈平為令，眾莫不知。每一令出，平伐其功，曰以為非我莫能為也。」 ② 聯同鄭袖、子蘭構陷屈原 |
| **破壞力** | 一言而致忠臣被疏，一語而令國策翻覆 |

> **軍師評語：** 上官大夫之讒，僅一句話，卻改變了楚國命運。讒言之力，往往大於百萬雄兵——因為它從內部摧毀了國家的決策中樞。

### ④ 頃襄王（橫）

| 屬性 | 內容 |
|------|------|
| **即位背景** | 懷王被扣後繼位，為保住王位而倒向親秦派 |
| **對屈原態度** | 聽從令尹子蘭（其弟）之讒，將屈原放逐江南，終生不召 |
| **歷史評價** | 懦弱無能，苟且偷安，畏秦如虎 |

> **軍師評語：** 頃襄王之不肖，更甚於其父。父雖庸，尚知用賢；子則懦，唯恐賢者奪其權。屈原之絕望，有一半來自此君。

### ⑤ 司馬遷（約前145－前86）——寫作者！重點

| 屬性 | 內容 |
|------|------|
| **身份** | 西漢太史令，中國最偉大的史學家 |
| **個人遭遇** | 因李陵之禍，被漢武帝處以宮刑——**身為史官而受此辱，生不如死** |
| **與屈原之共鳴** | **司馬遷在屈原身上，看見了自己的影子。** 兩個相隔一百七十餘年的人，命運卻如拷貝：皆盡忠於君，皆遭小人讒害，皆被君主拋棄，皆以文字寄託千古悲憤 |
| **寫作姿態** | 本篇傳記的情感強度在《史記》中極為突出。結尾「太史公曰」引屈原〈懷沙〉之賦，並感嘆：「餘讀《離騷》《天問》《招魂》《哀郢》，悲其志。適長沙，觀屈原所自沉淵，未嘗不垂涕，想見其為人。」 |

> **軍師評語：** 太史公寫屈原，實則寫自己。〈屈原列傳〉表面是史，骨子裡是血——是司馬遷用筆蘸著自己的傷口，為天下所有被冤枉的忠臣良將所寫的一封長長的血書。

## 五、寫作意圖（核心戰略判斷）

### 司馬遷為什麼要寫〈屈原列傳〉？

表面上，這是《史記》列傳體例，為重要歷史人物立傳。

但深層來看——**這篇傳記是司馬遷為自己寫的辯護詞，也是為天下所有被冤枉的人寫的控訴書。**

| 司馬遷 | 屈原 |
|--------|------|
| 為李陵辯護，遭漢武帝下獄 | 為楚國抗秦，遭楚懷王疏遠 |
| 對朝廷一片忠心 | 對楚國一片忠心 |
| 被處以宮刑——士大夫最大之恥 | 被放逐江南——忠臣最痛之遇 |
| 「腸一日而九迴，居則忽忽若有所亡」 | 「長太息以掩涕兮，哀民生之多艱」 |
| 忍辱負重完成《史記》 | 含悲忍淚寫作《離騷》 |

### 司馬遷在文中的情感注入

1. **極高的評價**：稱〈離騷〉「兼詩風雅，可與日月爭光」
2. **強烈的同情**：對屈原「忠而見疑」的反覆強調，實為自己發聲
3. **引賦入傳**：大量引用屈原作品原文，讓屈原自己說話
4. **結尾的涕泣**：「未嘗不垂涕」——這在《史記》列傳中極罕見，是司馬遷極少數直接在傳記中流露個人感情之處
5. **暗線貫穿**：「舉世皆濁我獨清，眾人皆醉我獨醒」——這十四字，既是屈原的絕唱，也是司馬遷的獨白

> **軍師判斷：** 太史公筆下的屈原，不完全是真實的屈原，而是經過司馬遷自己傷痛過濾後的屈原。這篇傳記之所以千古傳誦，正是因為它在史筆之外，多了一份「同是天涯淪落人」的血肉情感。**它不是一篇冷靜的史傳，而是一聲從歷史深處傳來的、響徹兩千年的吶喊。**

## 七、一句話總結

> **〈屈原列傳〉不僅是一篇史傳，更是司馬遷以自身血淚為墨、為天下所有「忠而見疑、信而被謗」之人所寫的千古辯護詞——屈原投的是汨羅江，司馬遷投的是《史記》；前者以身殉國，後者以筆殉志，兩道光芒在歷史的長河中交匯成同一聲不屈的嘆息。**
