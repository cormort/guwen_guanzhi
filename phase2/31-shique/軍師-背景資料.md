# 〈石碏諫寵州吁〉背景資料

> 軍師曰：讀此文，須先觀全局、察地勢、辨人心，方知一字一盾，句句藏鋒。

---

## 一、座標定位

| 項目 | 定位 |
|------|------|
| **典籍** | 《左傳》·隱公三年（卷三） |
| **時間** | 魯隱公三年，西元前七二〇年 |
| **空間** | 衛國都城朝歌（今河南淇縣） |
| **原文起訖** | 「衛莊公娶于齊東宮得臣之妹，曰莊姜……」「……石碏使其宰獳羊肩涖殺石厚于陳。」 |
| **事件跨度** | 隱公三年（諫寵）→ 隱公四年（州吁弒君）→ 隱公四年（石碏大義滅親） |

**軍師點位**：此文非孤立諫言，實為一場政治災難的預警與善後報告。前因在隱公三年，後果在隱公四年。讀者須跨越兩年戰線，方能見石碏之全謀。

---

## 二、時代壓力

### 2.1 周室衰微，諸侯裂土

- 平王東遷（前七七〇年）後五十年，王室威信已如落日殘燈。
- 諸侯征伐自專，禮樂征伐不再自天子出。
- 衛國乃姬姓封國，武王之弟康叔之後，理論上應為周室屏障，實則內鬥連綿。

### 2.2 衛國國勢：內憂外患交織

- **內部**：莊公在位，寵妾嬖子，嫡庶不明。莊姜無子，以戴媯之子完為己子（後為桓公）。州吁為嬖人之子，恃寵而驕。
- **外部**：衛國位於黃河北岸，東有齊、西有晉、南有鄭、北有戎狄，四戰之地也。莊公晚年諸侯爭霸格局初成，衛國若不整肅內政，必為他國所乘。

### 2.3 嫡庶制度：周禮的底線

- 西周以來的宗法制度，核心在「立嫡以長不以賢，立子以貴不以長」。
- 莊公寵州吁，等於以個人好惡動搖國本。石碏之諫，本質上是維護周禮秩序，與莊公的人治思維對撞。
- 州吁日後的弒君自立，正是對「禮壞樂崩」趨勢的一次加速——連同姓諸侯國都無法守住嫡庶之防，天下焉能不亂？

**軍師研判**：衛國處境，好比一屋簷漏四壁裂，莊公不去修樑柱，反而在屋頂跳舞——寵州吁而壓桓公，是把火種塞進梁柱間，只待風起。

## 三、關鍵人物

### ● 石碏 —— 全篇之謀主

| 屬性 | 分析 |
|------|------|
| **身分** | 衛國上卿，資深大夫 |
| **謀略** | 諫、隱、縱、斷四策連環 |
| **性格** | 剛直而深沉，果決而悲痛 |
| **核心困境** | 殺子存國 vs. 存子滅國——他選了前者 |

**軍師評**：石碏是整齣悲劇中唯一清醒的人。他看得見火苗，澆不熄火勢，最後只能用最烈的手段——砍掉自己的手——來保住身體。他的諫言是良藥，莊公不受；他的沉默是等待，州吁不悟；他縱子從賊，是誘敵深入；他殺子於陳，是斷腕求生。此人有大局觀、有執行力、有犧牲的覺悟，可謂「知不可為而為之」的極致。

**名局復盤**：石碏殺石厚之計，精妙處不在殺，而在借力。他不動衛國一兵一卒，遣使赴陳，以「二賊可擒」說服陳侯，借陳國之手制伏州吁、石厚，再由衛國派人至陳行刑。此計一石三鳥：一不動國內兵戈，免生變亂；二借友邦正義之名，減輕衛國內部反彈；三將石厚引至境外處決，使父子恩義不與國法直接衝突於朝堂之上。

### ● 衛莊公 —— 敗局之源

| 屬性 | 分析 |
|------|------|
| **身分** | 衛國國君 |
| **致命傷** | 以私情廢公義，以寵愛亂秩序——生前溺愛庶子州吁，未料身後釀禍 |
| **性格** | 剛愎自用，目光短淺 |
| **歷史判詞** | 愛子不以義，終害其子、其國、其家 |

**軍師評**：莊公不是昏君，卻是不負責任的父親和國君。他聽得懂石碏的話，卻不願執行——這比聽不懂更糟。他寵幸庶子州吁，不是因為州吁賢能，而是因為私愛。他讓州吁將兵（兵權入手），讓他驕縱不法（民心離散），卻從未想過：你把一個人捧到可以取代嫡子的位置，又不給他合法的繼承權，這不就是逼他造反嗎？

**典型錯誤**：
1. 愛之不以禮——寵愛而不教以規矩。
2. 任之不以法——給兵權而不設約束。
3. 諫之而不改——石碏言切如此，莊公仍「不聽」。

### ● 州吁 —— 禍端之刃

| 屬性 | 分析 |
|------|------|
| **身分** | 莊公寵妾之子，庶子 |
| **性格** | 驕、狠、急 |
| **致命缺陷** | 有野心而無格局，有勇力而無人心 |
| **終局** | 弒兄自立，旋被石碏設計捕殺於陳 |

**軍師評**：州吁是典型的「被寵壞的野心家」。石碏預言他「驕而能降，降而不憾，憾而能眕」——這一句形同八字批命。州吁從小被寵，從未學過克制；長大後擁兵自重，從未學過敬畏。他的驕是莊公給的，他的狠是天性，他的急——急於上位——是驕與狠匯流後必然的爆炸。

他的致命錯誤：
1. 殺兄篡位，在名義上已輸（挑戰嫡長制，諸侯不服）。
2. 即位後不修德政，反而發動戰爭以樹威（伐鄭），加速垮台。
3. 信任石厚，卻不知石厚是石碏之子——暴露他識人不清。

### ● 石厚 —— 悲劇之鏈

| 屬性 | 分析 |
|------|------|
| **身分** | 石碏之子，州吁死黨 |
| **性格** | 愚忠州吁，不識大義 |
| **位置** | 連接石碏與州吁的鏈條，也是斷裂點 |
| **終局** | 被親父設計，斬於陳國 |

**軍師評**：石厚是全篇中最悲劇的角色——不是死於敵手，而是死於父親的計謀。從石碏的角度，他是必須剪除的禍根；從州吁的角度，他是唯一的忠臣；從歷史的角度，他站在了錯誤的一邊且無法回頭。石碏的「大義滅親」，滅的不只是兒子的命，更是自己作為父親的心。

## 四、寫作意圖

### 4.1 《左傳》作者的史筆

左丘明作《左傳》，意在「以史為鑑」。此篇不是單純記敘諫言，而是透過一次失敗的諫諍，展示一個精準的預言如何逐步應驗，以及一個智者如何在最惡劣的局勢中執行最艱難的決策。

### 4.2 三大主題

1. **預警的價值**：石碏的諫言是一份政治風險評估報告，精確指出從「驕、奢、淫、佚」走向禍亂的因果鏈條。莊公的拒絕採納，證明了「預警不執行等於無效」。

2. **制度與人治的衝突**：石碏代表制度（禮法、嫡庶、秩序），莊公代表人治（私愛、特權、例外）。制度輸了一次，但最終由石碏以更殘酷的方式補回了制度的尊嚴。

3. **大義滅親的慘烈正義**：這不是歌頌，是記錄。石碏的選擇是正義的，但正義的背後是父親殺子。作者不美化、不簡化，只是冷冷寫下「石碏使其宰獳羊肩涖殺石厚于陳」——讓管家去殺，因為父親下不了手。

### 4.3 修辭兵法

- **層遞法**：驕→奢→淫→佚，四字四階，步步沉淪。
- **對比法**：石碏的冷靜（諫言有條不紊）vs. 莊公的漠然（「弗聽」二字如刀）。
- **留白法**：殺石厚一段，不寫石碏的反應，不寫石厚臨終之言，空白處承載最多的悲劇張力。

---

## 五、一句話總結

> **石碏以「驕、奢、淫、佚」四字預言了一場宮廷政變，又以殺子之舉完成了對秩序的終極守護——他所諫的是制度，他所犧牲的是人性，留給後世的是「大義滅親」這道永遠淌血的傷口。**

> **軍師後記**：此文可與〈鄭伯克段于鄢〉對讀。鄭莊公「養惡」使弟自斃，石碏「除惡」殺子以衛國。一個用計不動手，一個動手不留情——但都是同一個時代的縮影：宗法制度已在崩潰邊緣，所有人都在用最極端的方式，試圖堵住禮樂的堤防。石碏之諫，不是他一個人對莊公的進言，而是整個周代制度對末世的絕望通牒。
