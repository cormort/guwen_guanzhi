## 1. 文章結構分析

**全文總覽：** 四段結構，先以自貶開場，再辯解退隱並非怨望，接著以田歌暗諷朝廷，最後以「道不同不相為謀」作結。看似謙卑，實則步步反擊。

### 第一段（自述身世，看似自貶）

> 惲材朽行穢，文質無所底，幸賴先人餘業，得備宿衛。遭遇時變，以獲爵位，終非其任，卒與禍會。

開篇自稱「材朽行穢」，姿態極低——這是在模仿傳統的貶官謝罪書。但緊接著「幸賴先人餘業」「遭遇時變，以獲爵位」等語，暗示自己因揭發霍氏有功而得官，並非無能之輩。低姿態之下藏著不甘：我本不該落到這步田地。

### 第二段（辯解隱居，並非怨望）

> 身率妻子，勤力耕桑，灌園治產，以給公上。不意當復用此為譏議也。

孫會宗來信勸他低調，楊惲的回應非常巧妙：他把自己的隱居生活描述為「勤力耕桑，灌園治產，以給公上」——意思是我不僅沒犯法，還在種田納稅、奉養朝廷。你孫會宗憑什麼說我這樣不對？這一段的邏輯是：**我過得很好，我沒有錯。**

### 第三段（田歌諷喻，全信核心）

> 田彼南山，蕪穢不治。種一頃豆，落而為萁。人生行樂耳，須富貴何時！

這是全文的轉折點，也是楊惲的催命符。前兩句寫田園荒蕪、種豆得萁，字面寫農事，實則暗喻朝廷賢人不用、小人當道。最後一句「人生行樂耳，須富貴何時」則徹底撕掉了自貶的面具——我何必等富貴？我現在就很快樂。對漢宣帝而言，這句話等於在說：你的懲罰在我眼裡沒有任何分量。

### 第四段（起舞狂歌，徹底決裂）

> 是日也，拂衣而喜，奮袖低昂，頓足起舞……籲！道不同，不相為謀！

結尾以拂衣起舞的狂放形象收束，楊惲把自己塑造成一個躬耕自足、飲酒高歌的田園浪子。最後引用孔子「道不同不相為謀」，既是對孫會宗的告別，也是對朝廷的告別——你們走你們的富貴路，我走我的田園路。這在專制下是最致命的一句話：**拒絕合作，比反抗更讓統治者恐懼。**

### 起承轉合

| 結構 | 內容 |
|:----:|------|
| **起** | 自述身世貶官，表面自貶「材朽行穢」，實則暗示「遭遇時變，以獲爵位」，為後文的不甘埋下伏筆 |
| **承** | 辯解隱居生活並非怨望——「身率妻子，勤力耕桑……以給公上」，以納稅奉公證明自己無罪 |
| **轉** | 以田歌「田彼南山，蕪穢不治」暗諷朝廷，轉入「人生行樂耳，須富貴何時」——從防禦轉為進攻 |
| **合** | 拂衣起舞、頓足高歌，以「道不同不相為謀」斷然收束：我不會變成你們要的樣子 |

## 2. 寫作技巧分析

### 修辭手法

| 手法 | 原文 | 效果 |
|:----|------|------|
| **反語（正話反說）** | 「材朽行穢，文質無所底」 | 表面自貶，實則自傲。通篇以謙卑姿態包裹憤怒，讓朝廷無法直接指責「他態度不好」，卻又能感受到字裡行間的不滿 |
| **比喻（田園隱喻政治）** | 「田彼南山，蕪穢不治。種一頃豆，落而為萁」 | 全信最核心的修辭。南山之田隱喻朝廷，蕪穢不治暗指朝政腐敗，種豆得萁暗喻賢人不得其用。這是司馬遷史筆的遺傳——用最樸素的意象說最鋒利的話 |
| **反問** | 「不意當復用此為譏議也？」 | 故作驚訝：我種田納稅有什麼好批評的？看似質疑孫會宗，實際是質疑整個朝廷的邏輯——被貶者連種田都是錯嗎？ |
| **排比／意象疊加** | 「拂衣而喜，奮袖低昂，頓足起舞」 | 連續三個動作描寫，製造出狂放不羈的畫面感。用身體語言傳達「我很快樂，我很自由」——這是對權力懲罰的無聲否定 |
| **引用經典** | 「道不同，不相為謀」 | 引孔子之言為自己的選擇背書。引用聖人語錄，等於是以聖人之道對抗朝廷之道——你的道不是聖人之道，所以我拒絕 |
| **對比** | 「勤力耕桑」vs「奮袖起舞」 | 前半段寫躬耕的勤苦（姿態端正），後半段寫飲酒的狂放（情緒自由），兩相對照，把「被貶官員」的形象從「可憐蟲」翻轉為「洒脫人」 |

### 結構技巧

1. **層層遞進的語氣轉折：** 第一段謙卑 → 第二段自辯 → 第三段暗諷 → 第四段狂放。語氣逐步升溫，像一鍋慢慢煮沸的水，讀者到最後才發現楊惲從頭到尾都沒有低頭。

2. **偽書信體的政治表演：** 名義上是寫給孫會宗的私人書信，實際上楊惲知道這封信會被呈報朝廷。所以他寫給「一個朋友」的語氣，其實是寫給「皇帝」看的。整封信是一場精心設計的政治表演——用友人間的坦率，包裹對最高權力的譏諷。

3. **以退為進的修辭策略：** 不說「我沒錯」，而是說「我過得很好」；不說「朝廷不對」，而是說「田園荒蕪」。不說抱怨的話，卻句句都是抱怨。這套策略最終失敗了——因為漢宣帝讀懂了。

## 3. 佳句賞析

### 佳句一

> **「田彼南山，蕪穢不治。種一頃豆，落而為萁。人生行樂耳，須富貴何時！」**

**白話翻譯：** 南山上有塊田，荒蕪沒人耕種。種了一頃豆子，收穫的卻只是豆萁（豆稈）。人生不過是及時行樂罷了，何必等那不知何時才來的富貴！

**賞析：** 這是中古中國最早的「田園政治隱喻」之一。陶淵明之前，楊惲已經用田園意象來批判政治。前兩句明寫農事失收，實寫朝廷失政——賢人（豆子）埋沒，小人（豆萁）當道。後兩句則從諷喻一轉而為生命態度：「人生行樂耳」——五個字，把專制權力最重視的「功名富貴」踩在腳下。這句話放在今天叫豁達，放在前54年叫「藐視朝廷」。楊惲的不在乎，是統治者最恐懼的東西。

### 佳句二

> **「身率妻子，勤力耕桑，灌園治產，以給公上。」**

**白話翻譯：** 我親自帶著妻子兒女，努力耕田種桑、澆灌園圃、經營產業，以此供應國家的稅收。

**賞析：** 楊惲的辯護策略極為高明——他不說「我沒做錯事」，而是說「我在納稅」。在西漢的政治邏輯裡，納稅是臣民的義務，也是忠誠的表現。楊惲把自己塑造成一個勤懇納稅的良民，反問孫會宗：這樣做，哪裡不對？從修辭學上說，這叫**佔據道德制高點**——你不是說我怨望嗎？我在為國家生產，我在奉養朝廷，你憑什麼說我？一個被貶的庶人，用自己的勞動養活自己還納稅，這在邏輯上確實無懈可擊。但問題是，專制權力不講邏輯——它講感受。

### 佳句三

> **「是日也，拂衣而喜，奮袖低昂，頓足起舞，誠淫荒無度，不知其不可也。」**

**白話翻譯：** 那一天，我高興地抖抖衣服，揮起衣袖上下舞動，跺著腳跳起舞來——實在是放縱到了極點，完全不覺得這樣做有什麼不可以。

**賞析：** 這段文字極具畫面感。楊惲用三個連續動作描寫——拂衣、奮袖、頓足——構建出一個狂放起舞的形象。最關鍵的是最後七個字：「不知其不可也」。這是一句帶著挑釁的辯解：我不知道這樣有什麼不對。這句話擺明了告訴孫會宗（以及背後的皇帝）：我的快樂是真心的，我拒絕為我的快樂道歉。專制權力最無法接受的，就是被懲罰者對處罰的「不在乎」。楊惲以舞蹈和歌聲，完成了對權力最優雅的蔑視。

### 佳句四

> **「道不同，不相為謀。」**

**白話翻譯：** 所走的道路不同，就沒有辦法一起謀劃事情。

**賞析：** 引用《論語》名句收束全信，楊惲在此處完成了最後的自我定位。他把自己和孫會宗（以及孫會宗所代表的朝廷）劃分為「道不同」的兩個陣營。你的道是功名富貴、謹小慎微；我的道是躬耕自足、及時行樂。但問題在於，對皇帝來說，天下只有一條「道」——皇帝的意志。楊惲說「道不同」，等於是說「我不走你的路」。在專制政治中，拒絕與朝廷同道，本身就是一種政治表態。這句話是他的最後通牒，也是他的墓誌銘。

## 4. 白話改編

楊惲恭敬回覆孫會宗大人：

我這個人才德淺薄、行為卑下，學問和品行都沒有什麼成就。僥倖依靠先人留下的餘蔭，得以在宮中擔任侍衛官。後來碰上時局變故（指霍氏謀反案），因而獲得了爵位，但終究不是我能勝任的，最後還是惹來了禍事。

您來信教導我，說做人的道理在於朋友之間互相規勸、互相幫助，這份心意我非常感激。但是，恐怕您的用意太深了——您其實是想說，我這種不檢點的行為，會給自己惹來更大的麻煩吧？

我仔細想了想我的過錯，卻自己想不通。我現在帶著妻子兒女，努力耕田種桑，灌溉園圃、經營產業，以此繳納國家的賦稅。萬萬沒想到，這樣做居然也會被拿來批評議論！

人的感情，不就是想把日子過好嗎？那些違背禮法、放縱慾望的人，固然可恥；但如果真的能夠做到通達情理、隨遇而安的人，也不應該受到非議。我現在種田納稅，有什麼不對呢？

說起來，我這些年確實學壞了。我跟著百姓一起勞動，不自覺地染上了農夫們的習慣。每年伏日、臘月的祭祀之後，殺一頭羊、烤熟了配酒喝，我端起酒碗，仰著頭看天上的雲氣——心裡想的只有：田裡的那座南山啊，荒蕪沒人耕種；種了一頃的豆子，收穫的卻全是豆稈。人生不過是及時行樂罷了，富貴要等到什麼時候才算數！

那一天，我開心得抖抖衣裳，揮著袖子上下舞動，跺著腳跳起舞來——實在是放縱得不像話，但我一點也不覺得有什麼不對。我喝了酒，就忍不住要放聲高歌。唱的是南山種田，荒蕪不治；唱的是種豆得萁，人生苦短。我的話越說越放肆，簡直像是醉了說胡話。

唉！古人說：走的不是同一條路，就沒辦法一起謀劃事情。您就讓我這樣下去吧，何必再用功名富貴來打擾我這個在田裡種豆子的人！
