# 〈原道〉背景資料 — 軍師策論

---

## 一、座標定位

| 軸線 | 定位 |
|------|------|
| **時間** | 唐 · 貞元末至元和初（約 800–805 年），中唐轉折期 |
| **作者** | 韓愈（768–824），時年約三十五至四十，正值思想成熟、仕途顛簸的中年 |
| **地理** | 長安。天子腳下，佛道交織、儒門淡薄的核心戰場 |
| **文體** | 論說文 — 不是抒情、不是記事，是宣戰書 |
| **戰略性質** | **古文運動的總綱領，宋明理學的發令槍** |

> 此非一人一時之文，乃千年道統之存亡戰。

---

## 二、時代壓力：三股力量席捲中唐

### 🟥 佛教 — 最強大的敵人

- **寺院經濟**：天下寺廟數千，「僧房占天下之半」；僧侶免稅免役，大量人口逃入佛門避賦。
- **皇室崇佛**：憲宗迎佛骨（〈諫迎佛骨表〉背景）、代宗、德宗皆佞佛，宮中設內道場。
- **思想壟斷**：禪宗已徹底漢化，士大夫談禪成風。儒生不懂佛理，就無法在知識圈立足。
- **韓愈的判斷**：佛教「棄君臣、去父子、禁生養」——夷狄之法，亂中國之教。

### 🟨 道教 — 暗中的隱患

- 李唐皇室自認老子後裔，尊道教為國教。
- 道士煉丹求長生，上自帝王下至士族，服丹致死不計其數（韓愈晚年亦服硫磺，死於丹毒——這是他的悲劇，也是時代的縮影）。
- 韓愈批判：道教「清靜無為」之說，實為「去仁義」之論。

### 🟦 儒學 — 已入絕境

- **科舉浮華**：進士科考詩賦，士人重聲律對偶，不讀經義。
- **經學破碎**：南北朝以來，義疏之學越講越煩，越注越亂，儒生只會註經，不會行道。
- **沒有信仰**：儒家淪為官樣文章、儀節制度，失去了打動人心的力量。

> **戰局總評**：佛有信仰，道有修行，儒只剩考試。誰能讓儒家再次成為中國人的精神支柱，誰就是這個時代的拯救者。韓愈說：我能。

## 三、關鍵人物

### 🧑‍🎓 韓愈（768–824）— 主帥

- **字**：退之。自稱「郡望昌黎」，世稱韓昌黎。
- **職位**：監察御史→陽山令→國子博士→比部郎中→史館修撰→潮州刺史→吏部侍郎。
- **性格**：狂、直、硬、不認輸。「前古之興亡，未嘗不經於心也；當世之得失，未嘗不留於意也。」
- **武器**：文章。他不用刀劍，用古文打天下。
- **核心信念**：孟軻死，道統絕；道統絕，天下亂；我韓愈，來接。

### 👑 道統序列 — 韓愈心中的「正統傳承」

韓愈在〈原道〉中首次明確列出這條傳承線：

```
堯 → 舜 → 禹 → 湯 → 文、武、周公 → 孔 → 孟
                                   ↕        ｜
                                 （韓愈）←———┘
```

| 人物 | 戰略定位 |
|------|----------|
| **堯、舜** | 禪讓天下，公天下也。道之始。 |
| **禹** | 治水定九州。以人力抗天災，非靠禱告。 |
| **湯** | 革命。夏桀無道，湯伐之。天道在民不在君。 |
| **文王、武王** | 以周代商，制禮作樂。王道政治的制度化。 |
| **周公** | 制禮作樂的執行者。儒家政治秩序的真正奠基人。 |
| **孔子** | 刪述六經，傳道於後。道不在政權，在師承。 |
| **孟子** | 闢楊墨，正人心。最後一位道統傳人。 |
| **（斷層）** | 秦火、漢儒、魏晉清談、南北朝佛道——一千二百年無人接上。 |
| **韓愈** | 「軻之死，不得其傳焉。」——然後是他自己，捨我其誰。 |

> **軍師註**：這個序列是韓愈最大的原創。它在說：儒家不是一套學術，不是一套制度，而是一條血脈。誰接上這條血脈，誰就是中國人。

## 四、寫作意圖

### 表面目標：駁佛老

- 佛教「必棄而君臣，去而父子，禁而相生養之道」——毀棄人倫。
- 道教「必棄而君臣，去而父子，禁而相生養之道」——與佛同罪。
- 結論：佛老之教，是滅絕人類社會的學說。

### 深層目標：重新定義「道」

韓愈做了中國思想史上最大膽的一件事——**他直接重新定義了「道」這個字**。

> 「博愛之謂仁，行而宜之之謂義，由是而之焉之謂道。」

- 道不是老子的「道可道非常道」。
- 道不是佛家的「明心見性」。
- **道就是仁義。離開仁義，無所謂道。**

這是一記精準的斬首行動：他沒說佛老不對——他說佛老說的「道」根本就不是「道」。

### 終極目標：復興的不是學術，是信仰

韓愈要做的不是學術改革，不是文學運動，不是科舉改良。

**他要讓儒家重新成為中國人的精神支柱。**

- 佛教有生死輪迴？儒家有「仁者壽」「殺身成仁」。
- 佛教有極樂世界？儒家有「天下大同」。
- 佛教有功德？儒家有「立心、立命、繼絕學、開太平」。

他不是在寫一篇文章——他是在重建一個文明的信仰體系。

> **宋儒**讀了〈原道〉後，發現原來儒家可以這麼硬、這麼深、這麼有戰鬥力。沒有〈原道〉，就沒有程朱理學。

---

## 五、一句話總結

**韓愈以〈原道〉這篇宣戰書，在佛教道教席捲天下的中唐，重新建構了儒家的「道統」傳承體系，把「道」還原為「仁義」，宣告儒家不僅是一套學術，更是一條血脈、一種信仰——而這條血脈，從他韓愈開始，重新流動。**

---

> **軍師按**：後人讀古文觀止，以為〈原道〉只是韓愈諸多名篇之一。錯。〈原道〉是中國文明的自救宣言，是中唐思想界的赤壁之戰。從這篇文章之後，儒家不再只是伏案註經的書生，而是可以與佛道正面對決的戰士。一千兩百年後的中國人罵「儒家保守」，卻不知道，在〈原道〉寫成的年代，儒家是最大的革命者。
