# 石鐘山記 · 文章分析

## 1. 文章背景

〈石鐘山記〉為北宋蘇軾（1037–1101）於元豐七年（1084）六月由黃州團練副使移任汝州團練副使途中，遊歷江西鄱陽湖口石鐘山後所著之山水遊記。此時蘇軾因「烏台詩案」貶謫黃州已近五年，政治生涯遭遇重大打擊，心境趨於沉潛內斂，轉而寄情山水、考證名物。

石鐘山位於鄱陽湖入長江之口，分上鐘山與下鐘山。蘇軾對其得名之由提出質疑：酈道元《水經注》謂其「風水相搏，有聲如洪鐘」；唐李渤則謂扣石其聲如鐘。蘇軾親往考察，夜乘小舟至絕壁下，驗證聲響來源，最終提出「空中而多竅，與風水相吞吐，有窾坎鏜鞳之聲」乃真因之說。此文不僅為山水遊記，更是一篇考證辨偽的科學考察筆記，體現了宋代文人「格物致知」的理性精神。

## 2. 內容結構

全文可分為四大段落，層層遞進，結構嚴謹：

**第一段：設疑——舊說之辨**
開篇即引酈道元與李渤兩說，指出酈說雖近是卻未詳，李渤扣石之說「尤為謬矣」。蘇軾以「人常疑之」「餘尤疑之」二語，分別表達對兩說的態度，層次分明。此段為全文伏筆，製造懸念，引出下文親身考證的必要性。

**第二段：親證——夜探石鐘山**
此為全文核心，記敘夜乘小舟至絕壁下的親歷。文筆由敘入景、由景入聲，可分三層：
- **環境鋪陳**：大石側立千尺、如猛獸奇鬼、棲鶻驚飛、鸛鶴叫嘯——營造陰森詭譎的夜探氛圍。
- **水石相搏**：大聲發於水上，噌吰如鐘鼓不絕。此為石鐘山命名之真源。
- **小港細察**：入港口，有大石中空多竅，與風水吞吐發出窾坎鏜鞳之聲，與先前噌吰之聲相應。

**第三段：釋疑——命名真相**
蘇軾綜合考察結果，得出結論：「噌吰者，周景王之無射也；窾坎鏜鞳者，魏莊子之歌鐘也。」比喻精妙，將自然聲響比作古樂器，以證石鐘山確因水石相激之聲得名。進而譏笑李渤「目見耳聞」之淺陋。

**第四段：議論——為學之道**
末段昇華至治學方法論：「事不目見耳聞，而臆斷其有無，可乎？」強調親身實證、審慎求真的態度。感嘆酈道元「簡而陋」，士大夫「不肯以小舟夜泊絕壁之下」，批評漁工水師「知而不能言」，層層推演，從個案上升到普遍學術方法，使遊記具備哲理深度。

## 3. 寫作手法

1. **記敘、描寫、議論三體交融**
   全文由記遊敘事入手，穿插絕壁夜景的生動描寫，最終歸結為議論。三者渾然一體，無割裂之痕，是北宋古文運動「文以載道」理念的典範實踐。

2. **感官渲染與聲景營造**
   蘇軾極擅聽覺描寫。夜探一段：「大聲發於水上，噌吰如鐘鼓不絕」「窾坎鏗鞺，如樂作焉」，以擬聲詞貫串全段，並以古鐘樂器（無射、歌鐘）作譬，將抽象的自然聲響具體化、雅化。視覺上則以「大石側立千尺」「如猛獸奇鬼」營造恐怖氣氛，聲景交融，驚心動魄。

3. **對比手法**
   - 酈道元之「簡」與李渤之「陋」的對比
   - 士大夫不肯夜泊與漁工水師無力言說的對比
   - 前人「臆斷」與親身「目見耳聞」的對比
   層層對比，凸顯實證精神的可貴。

4. **設問與反詰**
   「事不目見耳聞，而臆斷其有無，可乎？」一句反詰，力道千鈞，直指為學之本。前文「人常疑之」「餘尤疑之」的設問鋪墊，亦為全篇蓄勢。

5. **比喻與典故**
   以「無射」「歌鐘」兩種古樂器喻石鐘之聲，不僅準確傳神，更彰顯作者學養深厚，使文章典雅含蓄。

## 4. 思想情感

1. **格物致知的科學精神**
   蘇軾不以權威之說為滿足，不憚親歷險境以求真知，展現了宋代文人「格物致知」的實學傳統。此精神與宋儒「即物窮理」的理學追求相表裡，在傳統遊記中獨樹一幟。

2. **貶謫之痛與山水之寄**
   文中雖未直接言及貶謫之苦，但「夜泊絕壁」的險境，實暗合其身處政治驚濤駭浪的處境。蘇軾以考證石鐘山為寄託，在追求物理真知的過程中，尋求心靈的安頓與超越。山水之遊，實為自我療癒之途。

3. **對小民智慧的同情與惋惜**
   「漁工水師雖知而不能言」一句，流露對勞動者的同情，亦暗示知識精英有責任為無聲者發聲。這種關懷，與蘇軾一生「民胞物與」的胸懷一脈相承。

4. **為學當求實的終極關懷**
   文章最終昇華為對學術方法的反省，反對空談臆斷，提倡實地考察。這種「實學」精神，不僅是對漢唐注疏之學的反動，更是宋代新儒學「經世致用」思想的具體實踐。

綜上，〈石鐘山記〉雖篇幅不長，卻兼具遊記之景趣、考證之嚴謹與哲理之深遠，是蘇軾貶謫時期文學創作的傑出代表，亦是中國古代散文「格物—致知—明理」書寫傳統的典範之作。
