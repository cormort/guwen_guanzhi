# 子魚論戰 — 文章分析

> 左丘明《左傳·僖公二十二年》
>

## 1. 文章結構

**背景：** 宋襄公與楚軍在泓水交戰。

**關鍵場景：**
- 楚軍還沒完全渡河 → 子魚：請擊之 → 宋襄公：不可
- 楚軍渡河後還沒列陣 → 子魚：請擊之 → 宋襄公：不可
- 楚軍列陣完畢 → 開戰 → 宋軍大敗，宋襄公重傷

**戰後宋襄公的辯解：**
> 「君子不重傷，不禽二毛。古之為軍也，不以阻隘也。寡人雖亡國之餘，不鼓不成列。」

君子不傷害已經受傷的人，不俘虜頭髮斑白的人……我即使亡國，也不攻擊沒有列好陣的敵人。

**子魚的反駁：**
> 「君未知戰！」——然後用一套精彩的戰爭邏輯徹底駁斥。

## 2. 寫作技巧

**正反論辯：** 宋襄公講的是「禮」，子魚講的是「勝」，兩套價值觀的正面衝突。

**子魚的三層反駁：**
1. 敵人在險境中，正是我軍的機會
2. 戰爭的目的是殺敵，不是講禮儀
3. 如果一定要講禮，那就乾脆投降算了

## 3. 佳句賞析

### 「君未知戰。」
三個字推翻宋襄公的全部理論——你根本不懂戰爭。

### 「若愛重傷，則如勿傷；愛其二毛，則如服焉。」
如果不忍心再傷害已受傷的人，不如一開始就不要傷人；如果憐惜老人，不如直接投降算了。用極端的類比戳破宋襄公的矛盾邏輯。

### 「三軍以利用也，金鼓以聲氣也。」
軍隊是為了取勝而用的，金鼓是為了鼓舞士氣的。戰爭的本質——不是禮儀表演。

## 4. 白話改編（節要）

宋襄公和楚軍在泓水交戰。宋軍已經列好陣，楚軍還在渡河。子魚說：「敵眾我寡，趁他們沒渡完趕緊打！」襄公說不行。

楚軍渡完河但還沒列好陣，子魚又請求出擊。襄公又說不行。等楚軍列好陣，宋軍開戰——大敗。

襄公說：「君子不傷害傷兵，不俘虜老人……我即使亡國也不攻擊沒列好陣的敵人。」

子魚說：「您根本不懂戰爭！如果敵人在險地，正是我們的機會……如果憐惜受傷的敵人，不如不要傷害他們；如果憐惜老人，不如直接投降！」
