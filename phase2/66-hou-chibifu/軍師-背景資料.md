# 〈後赤壁賦〉背景資料

> 軍師按：本文為北宋文豪蘇軾（東坡）七OO篇傳世散文中最具夢幻感的一篇。若〈前赤壁賦〉是白天的理性超脫，〈後赤壁賦〉就是夜晚的感性迷茫——蘇軾在告訴我們：即使想通了，還是會孤獨。

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **時間** | 北宋神宗元豐五年十月十五日（西元1082年），歲次壬戌 |
| **地點** | 黃州赤壁磯（今湖北黃岡市黃州區長江畔） |
| **作者** | 蘇軾（字子瞻，號東坡居士），時年四十七歲 |
| **體裁** | 賦，以夜遊紀實為表、夢幻寓託為裏的哲理散文 |
| **相對位置** | 距〈前赤壁賦〉之作恰好三個月——前賦寫於同年七月既望（七月十六日），後賦寫於十月望日（十月十五日） |

---

## 二、時代壓力——從「七月既望」到「十月之望」

### 《前、後赤壁賦》對照軍情圖

| 維度 | 前赤壁賦 | 後赤壁賦 |
|------|----------|----------|
| **季節** | 初秋（七月） | 深秋／初冬（十月） |
| **天候** | 「清風徐來，水波不興」——風平浪靜 | 「霜露既降，木葉盡脫」——蕭瑟凋零 |
| **月色** | 「月出於東山之上，徘徊斗牛之間」——朗月高懸 | 「人影在地，仰見明月」——清冷孤照 |
| **心緒** | 曠達超脫，以「變與不變」之理自我寬解 | 蒼涼孤寂，登山長嘯後「悄然而悲，肅然而恐」 |
| **結局** | 理性昇華——「相與枕藉乎舟中，不知東方之既白」 | 夢幻消散——道士不答，開戶視之「不見其處」 |
| **關鍵轉折** | 客吹洞簫，蘇子答客——理性說服 | 孤鶴橫江，道士入夢——感性迷離 |

### 軍師分析

寫完〈前赤壁賦〉三個月後，蘇軾的心境不但沒有持續「曠達」，反而轉入更深沉的孤獨與虛無。這是一個極其關鍵的心理信號：

**前賦的「超脫」是理性的努力；後賦的「蒼涼」是感性的真實。**

前賦中的蘇軾可以對客人說出一套「變與不變」的哲理——萬物與我皆無盡，何須羨長江之無窮？這是理悟。但三個月後，同樣的場景、同樣的赤壁，他不再說理，而是用行動呈現另一種真實：我看到一隻孤鶴，我夢到一個道士，醒來後什麼都沒有。

這說明了什麼？**理性可以說服別人，卻說服不了深夜的自己。**

烏臺詩案（1079年）被貶黃州已三年，蘇軾的創傷從未真正癒合。前賦是寫給世人的回答——「我很好，我想通了」；後賦是寫給自己的獨白——「其實我還是不安」。

## 三、關鍵人物——角色分析

### 1. 蘇軾（主體——夜的獨行者）

後賦中的蘇軾與前賦判若兩人。前賦中他是從容的智者，與客泛舟、飲酒樂甚；後賦中他獨自「攝衣而上，履巉巖，披蒙茸，踞虎豹，登虬龍」，攀登山壁、大聲長嘯——這是一個人在黑夜中自我掙扎的意象，是對孤獨的主動擁抱，也是對命運的不甘。

登山一段是全賦最激烈的行動描寫：「劃然長嘯，草木震動，山鳴谷應，風起水湧。」——然而嘯聲過後，換來的是「悄然而悲，肅然而恐，凜乎其不可留也」。這像極了中年困境：你可以對著命運吼叫，但吼完之後，世界仍然沉默。

### 2. 客人（陪襯——世俗的旁觀者）

後賦中的客人形象遠比前賦模糊。前賦中的客人會吹洞簫、會感慨「哀吾生之須臾」，是個有血有肉的對話者；後賦中的客人幾乎沒有台詞，只是跟著蘇軾走、看著蘇軾做。他們在蘇軾登山時「不能從焉」，被留在山下。

這是一個極具象徵意義的安排：**人在最深的孤獨時刻，是沒有人能跟上來的。** 客人代表的是世俗關係——朋友、同僚、家人——他們可以在岸邊等你，但在你必須獨自攀爬的山路上，他們無法同行。

### 3. 道士／孤鶴（關鍵——虛實之間的象徵）

這是全賦最精妙的筆法。四更時分，一隻鶴「橫江東來，翅如車輪，玄裳縞衣，戛然長鳴」，掠過小舟而去。當夜夢中，一位道士羽衣蹁躚，問蘇軾：「赤壁之遊樂乎？」蘇軾問其姓名，道士「俛而不答」。蘇軾猛然醒悟：「噫！疇昔之夜，飛鳴而過我者，非子也耶？」道士回首一笑，蘇軾驚醒，開門去看——「不見其處」。

軍師解讀：

- **鶴即道士，道士即鶴**——這是靈魂的隱喻。蘇軾的白日自我與夜晚靈魂在夢中對話。
- **「赤壁之遊樂乎？」**——全賦唯一的提問，來自道士／鶴，而非蘇軾自己。這是靈魂在問意識：你快樂嗎？你真的快樂嗎？
- **「俛而不答」→「不見其處」**——答案不在語言中，在消散中。真正的孤獨是無法回答的，真正的虛無是無處可尋的。

**這隻鶴／道士就是蘇軾心中的另一個自己。** 如果前賦的「客」是外在的對話者，後賦的「道士」就是內在的審問者。前賦的答案是蘇軾對世界說的；後賦的問題是靈魂對蘇軾問的，而且蘇軾無法回答。

## 四、寫作意圖——軍師戰略分析

### 核心命題：理性與感性的分野

〈前赤壁賦〉的結局是「不知東方之既白」——理性戰勝了黑夜，天亮了。
〈後赤壁賦〉的結局是「不見其處」——感性吞沒了理性，道士消失了，孤鶴不見了，什麼都沒有留下。

這不是否定前賦，而是補完前賦。蘇軾在告訴讀者一個殘酷的事實：

> **人是分裂的。白天想通的事，晚上還是會痛。**

### 三個層次的寫作意圖

1. **文學層次**：用夢幻筆法打破賦體傳統。前賦還是標準的賦——敘事、對話、說理；後賦則完全走向象徵主義，以鶴為魂、以夢為境，開創了中國散文前所未有的夢幻敘事風格。

2. **心理層次**：記錄被貶黃州三年後的真實精神狀態。前賦是「向外界展示的韌性」，後賦是「向內心坦承的脆弱」。兩篇合起來，才是一個完整的蘇軾——既曠達又孤獨，既超脫又執著。

3. **哲學層次**：提出一個比「變與不變」更深的命題——「夢與醒」。前賦談的是時間（逝者如斯，而未嘗往也）；後賦談的是真實（到底是夢見道士，還是夢見人生？）。莊周夢蝶的命題在這裡被推向極致：你怎麼知道此刻的「醒」不是一個更大的夢？

### 與前賦的結構張力

```
前賦：    夜遊 → 樂 → 悲（簫聲）→ 理悟（變與不變）→ 天亮
後賦：    夜遊 → 悲（蕭瑟）→ 激烈（登山長嘯）→ 夢幻（鶴/道士）→ 消失
```

前賦是**有解的**——蘇軾用一篇賦回答了人生的困頓。
後賦是**無解的**——蘇軾用一篇賦承認了困頓仍在。

這就是為什麼〈後赤壁賦〉通常被認為比〈前赤壁賦〉更難解讀，也更耐人尋味。前賦給了你答案，後賦讓你帶著問題離開。

---

## 五、一句話總結

> **〈前赤壁賦〉是蘇軾寫給世界的答案，〈後赤壁賦〉是蘇軾寫給自己的提問——當理性在白天完成了說服，孤獨總在深夜重新登船。**

---

*軍師註：元豐五年十月，黃州江上。一個四十七歲的中年人，帶著無法排遣的寂寞，在月光下看著一隻孤鶴飛過。他不知道這隻鶴會飛進中國文學史，成為夢與醒之間最優美的隱喻。*
