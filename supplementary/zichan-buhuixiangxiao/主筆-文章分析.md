## 1. 文章結構分析

**背景：** 鄭國子產執政，百姓常在鄉校聚會議論國政。大夫然明建議毀掉鄉校。

**核心對話：** 子產以「是吾師也」為根本立場，斷然拒絕。他認為百姓的議論正是施政的鏡子——好的意見就推行，壞的就改正。這種以民為師的胸襟，在春秋時代極為超前。

**孔子讚評：** 孔子聽聞此事後說：「以是觀之，人謂子產不仁，吾不信也。」以權威之口為子產背書，強化全文說服力。

| 結構 | 內容 |
|:----:|------|
| **起** | 然明提議毀鄉校——矛盾設立，百姓議論是否該禁 |
| **承** | 子產反對，闡述理由——民意如師，可資施政 |
| **轉** | 然明心服口服，自認淺陋——反對者被說服，立場翻轉 |
| **合** | 孔子讚子產不仁吾不信——以聖人之言作結，一錘定音 |

## 2. 寫作技巧分析

### 修辭手法表

| 手法 | 原文 | 效果 |
|:----|------|------|
| **設問** | 「何為？夫人朝夕退而遊焉，以議執政之善否。」 | 以反問開場，氣勢凌厲，直接打斷然明的提議 |
| **對比** | 其所善者行之 vs 其所惡者改之 | 正反對照，涵蓋所有施政面向，滴水不漏 |
| **比喻** | 「然猶防川……不如小決使導」 | 將民意比作河流——堵則潰決，導則通暢，形象精準 |
| **層遞** | 善者行之 → 惡者改之 → 是吾師也 | 從具體措施上升到根本原則，層層推進 |
| **引用權威** | 孔子「吾不信也」 | 借聖人之口為子產背書，結論無可辯駁 |

### 其他技巧

**翻轉期待：** 一般統治者討厭被議論，子產卻主動把議論當作老師——這種反差製造了強大的說服力。全文的兵法就是一句話：**把敵人（民意）變成老師（盟友）。**

**以退為進的論證：** 子產沒有直接反駁然明，而是先順著對方的邏輯走進具體情境——「夫人朝夕退而遊焉」——然後再翻轉。這種先接招再出招的手法，比正面衝突更高明。

**結尾的權威加持：** 引孔子之言收束全文，類似現代文章結尾用專家背書——子產的道理已經說完了，再由孔子蓋章確認，雙重權威疊加，令人無話可說。

## 3. 佳句賞析

### 「其所善者，吾則行之；其所惡者，吾則改之。是吾師也。」

**全文的靈魂。** 三句話完成一套完整的施政邏輯：接收民意的標準（善/惡）、政府的行動（行之/改之）、最終的定性（是吾師也）。七個字「是吾師也」是全篇最強的一擊——當你把罵你的人叫老師，你已經立於不敗之地。這是中國古代最早的「以民為師」思想，比現代公共治理中的「公民回饋機制」早了兩千五百年。

### 「我聞忠善以損怨，不聞作威以防怨。」

**對仗工整的千古名言。** 「忠善」對「作威」，「損怨」對「防怨」——一正一反，一開一合。子產用十四個字道破了權力與民意的永恆關係：靠做好事來減少怨恨，而不是靠權力來壓制怨恨。清人評此句「義正詞嚴，千古名言」，毫不為過。

### 「然猶防川：大決所犯，傷人必多，吾不克救也；不如小決使導。」

**以治水喻治國，精妙絕倫。** 這是全篇最精彩的比喻——民意如河流，堵（防川）只會釀成大災，導（小決使導）才能讓水安全流走。這個比喻來自召公諫厲王（《國語·周語》），子產在此化用，既展現了深厚的歷史素養，又用一個具體畫面說透抽象的政治道理。

## 4. 白話改編

鄭國人在鄉校聚會，議論朝政的得失。大夫然明對子產說：「把鄉校毀了吧，怎麼樣？」

子產回答：「為什麼要毀？人們早晚在那裡聚會，議論施政的好壞——他們認為好的我就推行，認為不好的我就改正。這是我的老師啊，為什麼要毀掉？」

「我只聽說忠誠善良可以減少怨恨，沒聽說作威作福可以防止怨恨。靠權力堵住大家的嘴，難道不難嗎？這就像堵河流一樣。大面積決堤，傷人一定很多，我沒辦法挽救——不如開個小口子讓水慢慢疏導。把聽到的議論當作治病的良藥。」

然明聽完這番話，慚愧地說：「我現在才真正知道，您確實是可以成就大事的人。小人確實沒有見識——如果照我說的去做，恐怕鄭國就完了！」

孔子後來聽說這件事，感嘆道：「從這件事來看，有人說子產不仁，我絕不相信。」

子產死後，孔子流著淚說：「古之遺愛也。」——這位春秋名相身上，還保留著古代仁人愛民的風範。
