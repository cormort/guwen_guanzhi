召公諫厲王止謗

左丘明 · 國語·周語

📜 時代背景與寫作意旨

〈召公諫厲王止謗〉背景資料

一、座標定位

項目 內容
出處 《國語·周語上》
作者 左丘明（傳統說法，約與孔子同時）
史學體裁 國別史（分國記言），以「語」為體
事件背景 西周第十位君主——周厲王（姬胡）在位期間
諫言時間 約西元前 845 年
時代歸屬 西周晚期，共和行政前夕
對應文章 《古文觀止》卷三·周文

二、時代壓力：西周晚期——王室威信崩壞的前夜

厲王在位時（約前 877–前 841），西周已走過二百餘年，內憂外患交織：

  • 經濟矛盾引爆：厲王推行「專利」政策——將山川林澤之利全部收歸王室，斷絕貴族與平民的傳統生計來源。此舉等同於王室與全民爭利，經濟壟斷成為政治危機的導火線。
  • 夷狄威脅加劇：西北犬戎、東南淮夷頻繁入侵，軍事壓力掏空國庫，周王室已無力維持文武成康時期的絕對威權。
  • 政治氣候惡化：厲王不信貴族，不聽諫言，轉而倚賴特務監控（衛巫監謗），以高壓手段壓制異議——這不是強盛王朝的自信，而是末代君主式的恐慌統治。

一句話總結時代氣氛：西周已經站在歷史斷崖的邊緣，厲王親手把最後一塊基石抽掉。

三、關鍵人物解碼

召公（召穆公，姬虎）

  • 角色定位：清醒的顧問、制度的守門人、貴族理性的代表
  • 戰略姿態：他不是反對厲王，而是在救厲王。他的諫言本質是風險管理報告——告訴你堵住訊息流通的成本遠大於收益
  • 核心論證：「防民之口，甚於防川。」——用治水隱喻訊息管理，指出壓制只會讓壓力累積，最終潰堤
  • 歷史下場：厲王被流放後，召公與周公合力代理國政（共和行政），並用自己的兒子換下厲王太子的性命——此人不是空談者，是會收拾局面的人

周厲王（姬胡）

  • 角色定位：以權力堵住訊息的獨裁者、風險感知失能的統治者
  • 認知偏誤：聽到「國人不敢言，道路以目」，居然高興地對召公說：「吾能弭謗矣，乃不敢言。」——這不是鎮壓成功，而是統治者完全失去了現實感
  • 致命失誤：把批評當成叛亂，把沉默解讀為服從。他建立的不是秩序，是訊息真空——而真空的盡頭就是崩潰
  • 最終命運：西元前 841 年，國人暴動，厲王逃亡至彘（今山西霍州），病死於流放地。他是中國歷史上第一位被記載「被人民趕下台」的君主

衛巫

  • 角色定位：打小報告的特務系統原型
  • 運作邏輯：厲王派衛巫「監謗者」——巫師集團成為最早的政治監控機構，以告密為核心職能，不問是非，只問忠誠
  • 系統效應：一旦告密系統建立，人人自危，「道路以目」——熟人社會中最恐怖的沉默就此誕生
  • 歷史意義：衛巫模式是所有後世特務政治（如明朝廠衛、東德史塔西）的最早藍本。當統治者開始用祕密警察代替公共討論，結局從未改變過

四、寫作意圖：言論自由的寓言？還是統治者傲慢的教材？

這是中國古代政治思想中最常被引用、也最容易被誤讀的一個段落。至少有三層解讀：

層次一：中國最早的「言論自由」寓言？

從現代視角看，召公的論證確實觸及公共意見的正當性：

「民之有口，猶土之有山川也，財用於是乎出……口之宣言也，善敗於是乎興。」

人民的嘴巴，就像大地的山川一樣——自然、不可剝奪、而且有生產力。允許人民說話，國家才能興盛；不讓他們說話，國家就會敗亡。

這與現代言論自由理念有表面上的相似，但必須注意時代差異：召公的出發點不是天賦人權，而是統治利益——讓人民說話是因為對統治者有好處，不是因為人民有權力。然而，正因為如此，這套論證反而更務實、更接近政治學而非道德哲學。

層次二：統治者傲慢導致政權崩潰的經典教材

這才是《國語》作者真正的寫作意圖。全篇的敘事結構極其清楚：

厲王虐（原因）
  → 國人謗王（反應）
    → 王怒，殺人（錯誤決策）
      → 國人莫敢言，道路以目（恐怖平衡）
        → 王喜，以為成功（致命誤判）
          → 三年，乃流王於彘（最終懲罰）

這是一則傲慢統治者的標準劇本：虐政 → 批評 → 報復 → 沉默 → 自滿 → 崩潰。六個步驟，一步不差。

《國語》作者左丘明（或史官集團）記載此事，不是為了記錄一件歷史軼事，而是為了給後世君主一個直接的政治教訓：你不聽人民的聲音，人民就會用行動回答你。

層次三：治水隱喻——中國政治智慧的原型

「防民之口，甚於防川」這句話之所以流傳兩千多年，是因為它觸及了中國政治思想中一個核心隱喻：統治像治水。

  • 禹的成功在於「疏導」而非「堵塞」
  • 召公把這個原理從水利工程轉移到政治管理
  • 後世的納諫傳統、科舉取士、甚至御史制度，都或多或少繼承了這個邏輯

所以這不僅是一篇諫文，它是中國政治學中「訊息流通與政權穩定」這一命題的開山之作。

五、一句話總結

一個拒絕傾聽的統治者，最終只能聽見自己倒下的聲音。

1. 文章結構分析

分段大意

本文篇幅簡短，結構清晰，可分為三大段：

第一段：厲王弭謗，百姓敢怒不敢言

「厲王虐，國人謗王。召公告曰：『民不堪命矣！』王怒，得衛巫，使監謗者。以告，則殺之。國人莫敢言，道路以目。」

開門見山點出衝突：厲王暴虐→百姓批評→厲王鎮壓→殺人止謗→百姓「道路以目」（在路上相遇只敢用眼神交流）。短短四句話，一個高壓統治的畫面躍然紙上。

第二段：召公進諫——核心說理

「王喜，告召公曰：『吾能弭謗矣，乃不敢言。』召公曰：『是障之也。防民之口，甚於防川。川壅而潰，傷人必多，民亦如之。……』」

厲王得意洋洋炫耀自己「成功」了，召公卻以「防民之口，甚於防川」的比喻告誡他：堵住百姓的嘴，就像堵住河流，總有一天會決堤。 接著召公提出正面論述：統治者應該讓百姓「宣之使言」——讓他們把想法說出來，好的採納、壞的防範。

第三段：厲王不聽，結局印證

「王弗聽，於是國人莫敢出言。三年，乃流王於彘。」

厲王拒絕諫言，繼續高壓統治。三年後，百姓發動暴動，把厲王流放到彘地。結局用事實證明了召公的預言。

2. 寫作技巧與修辭

一、召公的說理策略

召公的諫言雖短，說理結構卻極為完整：

第1層：破題——直接點出問題本質

「是障之也。」

厲王說「吾能弭謗矣」，召公沒有拐彎抹角，直接說：你這不是解決問題，只是在堵塞問題。

第2層：比喻——用自然現象說明社會道理

「防民之口，甚於防川。川壅而潰，傷人必多，民亦如之。」

這是中國政治哲學中最經典的比喻之一。河流堵住會決堤，百姓的嘴巴堵住會暴動——道理完全一致。用具體的自然現象，說明抽象的政治風險。

第3層：正面論述——為君之道

「是故為川者決之使導，為民者宣之使言。」

對偶句：治理河川要疏通，治理百姓要讓他們說話。正面提出解決方案。

第4層：民意有用——列舉百姓建言的好處

「民之有口也，猶土之有山川也，財用於是乎出……口之宣言也，善敗於是乎興。」

從功利的角度說明：讓百姓說話，不是統治者的恩賜，而是了解施政得失的唯一途徑。好的政策可以推行，壞的政策可以修正。

第5層：警告——不聽話的下場

「夫民慮之於心而宣之於口，成而行之，胡可壅也？若壅其口，其與能幾何？」

民意是自然而然的產物，就像江河必然奔流一樣。堵住它，能撐多久？言下之意：遲早會爆炸。

三、修辭手法

  • 比喻：「防民之口，甚於防川」——成為千古名言
  • 對偶：「為川者決之使導，為民者宣之使言」——工整有力
  • 反問：「胡可壅也？」——語氣強烈
  • 層遞：從「民不堪命」→「道路以目」→「流王於彘」，一步步推向毀滅

3. 佳句摘錄與賞析

佳句一：「防民之口，甚於防川。」

白話： 堵住百姓的嘴巴，比堵住河流還要危險。

賞析： 中國政治思想史上最重要的比喻之一。用防洪的常識來談治國——每個人都知道河流不能硬堵，但厲王偏偏不懂（或不願懂）這個道理。七個字的反問句格局極大：從一條河聯想到一個國家的命運。這句話後來成為歷代諫官反對言論鉗制的最強武器。

佳句二：「道路以目。」

白話： 在路上相遇，只能互使眼色。

賞析： 四個字寫出一個國家的窒息感。沒有哭喊、沒有抗議，只有無聲的眼神交流——這是比任何悲慘描述更令人心寒的畫面。左丘明用最少的文字，寫出最深的恐懼。

佳句三：「是故為川者決之使導，為民者宣之使言。」

白話： 所以治理河川的人要疏導它，治理百姓的人要讓他們說話。

賞析： 對偶句，前句是比喻（為川者），後句是正題（為民者），兩者並列產生無可辯駁的說服力。這不僅是政治主張，更是一種治理哲學：堵塞只能短期見效，疏導才是長治久安之道。

佳句四：「民之有口也，猶土之有山川也，財用於是乎出。」

白話： 百姓有嘴巴，就像大地有山川一樣，財富是用品都是從這裡產出的。

賞析： 召公把「讓百姓說話」從道德義務提升為經濟利益——百姓的意見就是國家的財富。這個角度非常高明：你不聽百姓的話，損失的是你自己。

佳句五：「三年，乃流王於彘。」

白話： 三年後，人民就把厲王流放到彘地去了。

賞析： 全文最冷的結局。沒有評論、沒有感嘆，只是一個事實陳述。但這個事實本身就是最有力的論證——召公的話全部應驗。七個字寫完一場王朝悲劇，筆力千鈞。

4. 白話改編

原文（分段）⇢ 白話文

第一段：

厲王虐，國人謗王。召公告曰：「民不堪命矣！」王怒，得衛巫，使監謗者。以告，則殺之。國人莫敢言，道路以目。

周厲王暴虐無道，國內的百姓都在公開批評他。召公向厲王報告說：「百姓受不了您的政令了！」

厲王大怒，找來衛國的巫師，派他們去監視批評朝政的人。只要巫師來報告誰說了什麼，就把那個人處死。從此百姓沒有人敢開口說話了，在路上相遇，也只能互使眼色。

第二段：

王喜，告召公曰：「吾能弭謗矣，乃不敢言。」

召公曰：「是障之也。防民之口，甚於防川。川壅而潰，傷人必多，民亦如之。是故為川者決之使導，為民者宣之使言。故天子聽政，使公卿至於列士獻詩，瞽獻曲，史獻書，師箴，瞍賦，矇誦，百工諫，庶人傳語，近臣盡規，親戚補察，瞽、史教誨，耆、艾修之，而后王斟酌焉，是以事行而不悖。

「民之有口也，猶土之有山川也，財用於是乎出；猶其有原隰衍沃也，衣食於是乎生。口之宣言也，善敗於是乎興。行善而備敗，其所以阜財用、衣食者也。夫民慮之於心而宣之於口，成而行之，胡可壅也？若壅其口，其與能幾何？」

厲王非常高興，對召公說：「我能制止誹謗了，他們現在都不敢說話了！」

召公說：「您這只是堵住他們的嘴罷了。堵住百姓的嘴巴，比堵住河流還要危險。河流堵塞就會決堤，傷害的人一定很多；管百姓也一樣。所以，治理河川的人要疏導它，治理百姓的人要讓他們暢所欲言。

「因此，天子處理政務的時候，要讓從公卿大夫到列士的人都進獻詩歌，讓盲人樂師進獻樂曲，讓史官進獻史書，讓少師進獻箴言，讓盲人朗誦，讓各種工匠規勸，讓百姓的意見傳達上來，讓身邊的近臣盡力規諫，讓宗室親戚補過糾偏，讓樂師和史官進行教誨，讓年高德劭的人修訂整理，然後由天子仔細斟酌取捨。這樣政事才能推行而不違背情理。

「百姓有口有嘴，就像大地有山川一樣，財富用度都是從這裡產出的；就像大地有平原、窪地、低溼地、肥沃的田地一樣，衣服食物都是從這裡生出來的。百姓用嘴巴發表意見，施政的好壞就從這裡反映出來。推行好的、防範壞的，這才是增加財富、豐富衣食的根本啊。百姓心裡想什麼，嘴上就說什麼，這是自然而然的事情，怎麼能堵得住呢？如果硬要堵住他們的嘴，這樣能維持多久呢？」

第三段：

王弗聽，於是國人莫敢出言。三年，乃流王於彘。

厲王不聽從召公的勸諫。從此，國內的百姓沒人敢開口說話。過了三年，人民就把厲王流放到彘地去了。
