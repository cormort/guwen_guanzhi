子產論尹何爲邑

左丘明 · 《左傳·昭公十九年》

時代背景與寫作意旨

〈子產論尹何爲邑〉背景資料

座標定位

時間：  魯襄公三十一年（西元前 542 年）
空間：  鄭國都城
事件：  執政子皮欲任命年輕寵臣尹何治理城邑，
        子產以「美錦學裁」為喻諫阻，子皮從之
體裁：  諫言對話（以日常比喻行政治說理之實）
出處：  《左傳·襄公三十一年》／古文觀止卷二

子皮時為鄭國上卿，掌握人事大權，想提拔自己喜愛的年輕人尹何去管理一個城邑，讓他「學為政」——邊做邊學。子產當面反對，說出一段千古名言：你捨得拿自己的高級錦緞給人練習裁縫嗎？大官大邑是你身家性命所託，怎能讓一個生手拿來練習？

子皮聽完，從善如流。

這一年，也是子產剛剛執政（前 543 年上任）、推行「作封洫」土地改革的關鍵節點。同一年他還幹了另一件與「用人」同樣知名的事——不毀鄉校。可以說，襄公三十一年的鄭國正處於子產改革的高密度期，每一場對話都關乎國運。

時代壓力：改革深水區的「人事風暴」

2.1 鄭國的結構性困境

鄭國在春秋後期是一個「小而逼」的危邦：

壓力維度 現狀
地緣 介於晉、楚兩大霸權之間，七十年來反覆倒戈，年年備戰
內政 貴族世卿（七穆）尾大不掉，內亂頻仍——前 563 年尉止之亂、前 554 年子孔被殺
改革 子產剛上任，正在推行「作封洫」重新劃分田畝，觸動無數既得利益
信任 民間與貴族對新政充滿疑慮，隨時可能反彈

2.2 為什麼這樁人事任命是「火藥桶」？

子皮想提拔尹何，表面上是「培養年輕人」，但在這個時間點：

  • 改革剛起步：新法執行需要最可靠的官員——派一個生手去管城邑，萬一搞砸了，被貴族抓住把柄，整場改革可能前功盡棄
  • 貴族虎視眈眈：子產的改革已經得罪了許多世族，任何一個行政失誤都會被放大為「子產新政無能」的證據
  • 人事權是核心權力：誰來管城邑，決定新政能否落地。子皮雖然是執政，但這不是私人家事，而是國家戰略部署

軍師觀察：子產反對的不是「培養年輕人」，而是在「改革關鍵期把一個沒有受過訓練的人放到必須零失誤的位置上」。時機不對，代價太大。

關鍵人物

子產（公孫僑）—— 說破天機的人

  • 角色定位：鄭國正卿（執政），春秋第一流的改革家與外交家
  • 此時狀態：剛執政一年，四面楚歌，正在推動激進的土地改革
  • 核心信條：「苟利社稷，死生以之」——不計個人毀譽，只問國家利害
  • 論辯風格：永遠不用權威壓人，而是用對方的利益來說服對方。跟子皮講話，他搬出「美錦」——一個子皮這個貴族完全能體會的日常比喻
  • 歷史地位：孔子譽為「古之遺愛」。本篇中的「美錦學裁」是中國政治哲學中關於「用人須謹慎」最經典的比喻

軍師筆記：子產在這場對話中扮演的角色，不是「反對提拔新人」的保守派，而是「風險管理者」。他看出來了——子皮把國家職位當作教育訓練的場地，這在改革深水區是致命的。

子皮（罕虎）—— 能聽進逆耳忠言的執政

  • 角色定位：鄭國上卿，鄭國七穆之一罕氏的族長，前一任執政
  • 重要性：他是主動讓賢給子產的人（前 543 年），格局極大
  • 性格特質：能用人、能放權、能聽諫——這在春秋貴族中極其罕見
  • 本角色迷思：他是在位者，想提拔自己喜歡的年輕人，這在任何時代都是人之常情。但他最可貴的是——被子產說服後，不僅不生氣，反而公開承認自己錯了

關鍵一句（聽完子產的話後）：「善哉！虎不敏。……微子之言，吾不知也。」

  • 歷史定位：如果沒有子皮的格局，就沒有子產的舞台。他是「知人善任」的最佳範例——既能讓賢，也能聽諫

尹何 —— 被動的「試錯成本」

  • 角色定位：子皮的年輕寵臣，二十出頭，無治理經驗
  • 命運：全文只作為「被討論的對象」出現，沒有發言
  • 象徵意義：他代表了一個永恆的組織困局——「我喜歡他，所以給他機會」vs.「他還沒準備好，所以不能讓他上」
  • 軍師點評：尹何本身沒有對錯。問題不在他是好人壞人，而在制度不該用國家城邑來練兵。子產反對的是「任命機制」而非尹何本人

寫作意圖：用一個日常比喻說明白一個大道理

3.1 命題拆解

本篇的核心命題極其簡單，卻蘊含了中國政治思想史上最深刻的「用人哲學」之一：

你願意用自己最珍貴的東西，讓一個生手去練習嗎？

3.2 比喻的威力：「美錦學裁」

子產的論證結構如下：

層次 子產的話 邏輯操作
① 設喻 「子有美錦，不使人學製焉」 用貴族最熟悉的錦緞裁縫作類比——你絕不會拿高級布料讓學徒練手
② 推論 「大官、大邑，身之所庇也，而使學者製焉，其為美錦不亦多乎？」 城邑是身家性命所託，比錦緞貴重千百倍，更不能拿來練習
③ 反證 「僑聞學而後入政，未聞以政學者也」 先學習再當官，沒聽說過拿當官來學習的——這是一句顛覆性的命題
④ 舉例 打獵也罷、射箭也罷，都要先練基本功 任何技藝都需要訓練，何況治理國家
⑤ 歸結 「人心之不同，如其面焉」——你不可能在不了解一個人的情況下讓他去治民 治理需要經驗、需要識人、需要判斷——這些都不是憑空得來的

3.3 為什麼這個比喻如此經典？

因為子產沒有從「權威」或「階級」的角度說話——他沒有說「尹何地位太低不配當官」，也沒有說「你子皮亂用人」。

他用的是 「資產管理」的邏輯：

  • 錦緞是你的私產，你捨不得給學徒練手
  • 城邑是國家的資產，同時也是你身家性命的庇護所
  • 你怎麼反倒捨得拿更重要的東西去冒險？

這是一記精準的「逆向心理打擊」：把對方的日常常識（愛惜錦緞）抽出來，反過來打他自己（不愛惜官職）。

每個人讀到「子有美錦，不使人學製焉」都會心一笑——因為每個人都知道不能浪費好料子。然後子產說：「城邑比美錦重要得多，你怎麼就捨得了？」——瞬間把讀者從「我覺得尹何不錯」拉到了「我是不是在用國家給我的寵臣練兵？」

這就是高手寫文章的手法：用一個所有人都能體會的日常場景，推倒一個所有人都沒想過的深層道理。

3.4 在中國政治思想史中的地位

「美錦學裁」與以下經典隱喻並列，構成中國古代「用人政治學」的核心比喻群：

比喻 出處 核心命題
美錦學裁 本篇·子產 不可用國家重任來訓練生手
器使 《論語》 用人如用器，各取其長
匠石斫堊 《莊子·徐無鬼》 知音難覓，能人須配能用的環境
千里馬常有 韓愈〈馬說〉 人才需要被辨識、被珍惜

其中「美錦學裁」的特點是：它不討論「這個人好不好」，只討論「這個時機對不對」——它是一個風險管理模型，不是一個人才鑑定模型。

一句話總結

子產用一塊錦緞說破了千百年來最難的組織問題：你愛一個人，不等於你應該把國家交給他——學習治理的成本，遠比學裁縫昂貴得多，而這個代價不該由全城百姓來買單。

軍師戰術拆解：這場對話的本質是什麼？

如果把子產和子皮的對話拆成一個決策模型：

子皮的初始狀態：
  「我喜歡尹何 → 給他一個城邑 → 他邊做邊學 → 人才成長 → 雙贏」

子產的還原：
  「你喜歡尹何 → 給他一個城邑 → 他把城邑搞砸 → 百姓受害、改革受挫、
    貴族反撲、你失去威信 → 全輸」

子產的解法：
  「先用美錦比喻讓你停下來想 → 再用『未聞以政學』建立原則 →
   最後用『人心不同如其面』堵死『他其實可以』的僥倖心理
   → 讓你自主得出結論：現在不是時候」

軍師結論：子產不是在反對尹何這個人，而是在反對一種「把組織命運綁在個人感情上」的決策模式。而子皮最了不起的地方是——他聽懂了。

延伸閱讀線索

  • 本篇與〈子產不毀鄉校〉同出《左傳·襄公三十一年》——同一年、同一個人、兩套截然不同的說理策略（一個用比喻、一個用治水），讀者可對照觀摩
  • 子皮讓賢給子產的故事見《左傳·襄公三十年》——先有子皮的格局，才有子產的舞台
  • 「美錦學裁」的命題在後世反覆被引用：從《戰國策》到《資治通鑑》，歷代論「用人當慎」者無不引此為據
  • 明代張居正《陳六事疏》中「用人必考其終，授任必求其當」——正是子產「學而後入政」思想的延續

子產論尹何為邑 — 文章分析

1. 文章結構分析

分段大意

第一段：子皮提議——讓尹何邊做邊學

「子皮欲使尹何為邑。子產曰：『少，未知可否。』子皮曰：『愿，吾愛之，不吾叛也。使夫往而學焉，夫亦愈知治矣。』」

子皮想把城邑交給年輕的尹何治理。子產說：「他太年輕了，不知道行不行。」子皮說：「他很忠厚老實，我很喜歡他，他不會背叛我的。讓他邊做邊學，他會越懂治理的。」

第二段：子產的比喻——「美錦學裁」

「子產曰：『不可。人之愛人，求利之也。今吾子愛人則以政，猶未能操刀而使割也，其傷實多。子之愛人，傷之而已，其誰敢求愛於子？子於鄭國，棟也。棟折榱崩，僑將厭焉，敢不盡言？』」

子產用一個絕妙的比喻反駁：你有一塊高級錦緞，絕不會讓學徒拿來練裁縫。城邑是身家性命所託，比錦緞貴重千百倍——你怎麼反而捨得讓生手拿去練手？

第三段：子皮從善如流——公開認錯

「子皮曰：『善哉！虎不敏。聞君子務知大者遠者，小人務知小者近者。我，小人也。衣服附在吾身，我知而慎之；大官大邑，身之所庇也，我乃而慢之。微子之言，吾不知也。』」

子皮聽完後不但沒有生氣，反而坦誠自己錯了：我知道愛惜衣服，卻不知道愛惜官位——如果不是你說了，我根本不明白這個道理。

起承轉合

結構 內容
起 子皮想讓尹何為邑——邊做邊學
承 子產反對——用「美錦學裁」比喻
轉 以「棟折榱崩」警告子皮——用人不當會連累自己
合 子皮認錯，從善如流

2. 寫作技巧分析

「美錦學裁」——中國政治哲學最經典的比喻之一

這個比喻的威力來自三個層次：

層次 內容 效果
日常化 高級布料不給學徒練手——人人都懂 瞬間抓住聽眾的共鳴
類推 城邑比錦緞貴重百倍——同理可證 用日常經驗推導政治道理
反諷 你愛惜布料卻不愛惜官職 揭示認知矛盾

「未聞以政學」——全篇的核心命題

子產說：「僑聞學而後入政，未聞以政學者也。」——先學習再當官，沒聽說過拿當官來學習的。這句話是中國政治思想史中關於「用人須謹慎」最原始的表述之一。它顛覆了「邊做邊學」的直覺——政府不是實習機構，百姓不是練習對象。

「棟折榱崩」——從組織利益到個人利益

子產在結尾加了一句極其聰明的一句話：「子於鄭國，棟也。棟折榱崩，僑將厭焉。」——您在鄭國就像大梁，大梁斷了屋頂就塌了，我也會被壓在下面。他把自己和子皮的命運綁在一起——我不是在反駁你，我是在保護我自己（同時也保護你）。

子皮的可貴——能聽諫的執政

歷史上多少君臣關係毀於一句「你這話什麼意思」。子皮聽完後，沒有翻臉、沒有辯解、沒有記恨——他只說了一句「我錯了」。這種胸懷，是子產能大展拳腳的根本保障。

修辭手法一覽

手法 原文 效果
比喻 「美錦學裁」 化抽象為具體，人人都能理解
誇張 「猶未能操刀而使割也，其傷實多」 生動展現後果
類比 「棟折榱崩」 把個人風險與組織命運掛鉤
反問 「其誰敢求愛於子？」 讓對方反思自己的做法
名言 「未聞以政學者也」 一句話立下千年用人準則

3. 佳句賞析

「子有美錦，不使人學製焉。大官、大邑，身之所庇也，而使學者製焉，其為美錦不亦多乎？」

你有高級錦緞，不讓學徒拿來練裁縫。大官、大邑是您身家性命所託——您讓一個生手去治理，這塊錦緞不是太珍貴了嗎？錦緞是物質，官位是性命——這個對比讓子皮完全無法反駁。

「僑聞學而後入政，未聞以政學者也。」

我聽說先學習再從政，沒聽說過拿從政來學習的。這句話是全文的理論高峰。它把「邊做邊學」這個看似合理的願望，還原成了它的荒謬本質：你敢讓一個不會開刀的人在你的身體上練習外科手術嗎？

「人心之不同，如其面焉。吾豈敢謂子面如吾面乎？」

人的內心就像人的面孔一樣——各不相同。我怎麼敢說您的面孔和我的面孔一樣呢？這句話告訴我們：用人不是用「我喜歡他」就可以判斷的，必須用「他是否能勝任」來判斷。感情不能取代判斷。

4. 白話改編

子皮想讓尹何去治理一個城邑。子產說：「他年紀太輕，不知道行不行。」

子皮說：「他很忠厚，我很喜歡他，他不會背叛我的。讓他去了邊做邊學，他會越來越懂得治理的。」

子產說：「不可以。你愛一個人，總是要對他有利。現在你愛一個人卻讓他從政——這就像一個人還不會用刀就讓他去切東西，只會傷到自己。你愛一個人，結果卻害了他——那以後誰還敢求你愛他呢？」

「你在鄭國，就像屋子的棟樑。棟樑一折，屋頂就會塌——我也會被壓在裡面。我怎麼敢不把話說清楚呢？」

「就拿打獵來說——射箭、駕車都不熟練的人，去追飛禽走獸，連車都翻不好，還談什麼收穫？」

「國家大事，就像天上的陰晴一樣變幻莫測。你讓一個生手去治理——就像讓他駕車去追變幻的風雲——下場可想而知。」

子皮說：「說得太好了！我真是不夠聰明。你說得對——我愛惜衣服，卻不知道愛惜官位——如果不是你說了這番話，我根本不明白這個道理。從今以後，就算我們鄭國的事，我也不一定自己做主，都要先聽你的意見。」

子產說：「人心的不同，就像人的面孔各不相同一樣——我怎麼敢說你的面孔和我的面孔一樣呢？不過，你心裡覺得危險的事，我把它告訴你——這大概就是我的心意吧。」
