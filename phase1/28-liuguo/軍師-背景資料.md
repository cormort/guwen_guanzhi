| 項目 | 內容 |

## 時代壓力：北宋的戰略困局

蘇洵提筆之時，北宋朝廷正面臨兩大外患，情勢與戰國末年驚人地相似：

```
         ┌──────────────────────┐
         │       遼（契丹）       │ ← 歲幣銀絹，和親納貢
         │    （北方巨獸）        │
         └──────────┬───────────┘
                    │ 燕雲十六州
         ┌──────────┴───────────┐
         │       北宋朝廷        │ ← 主流意見：花錢買和平
         │     「納貢求和」       │
         └──────────┬───────────┘
                    │ 西北邊境
         ┌──────────┴───────────┐
         │       西夏（李元昊）    │ ← 歲賜絹銀，屢戰屢敗
         │    （西北狼）          │
         └──────────────────────┘
```

- **澶淵之盟（1005）**：北宋每年向遼國輸送歲幣銀十萬兩、絹二十萬匹。
- **慶曆和議（1044）**：北宋又向西夏歲賜銀七萬兩、絹十五萬匹、茶三萬斤。
- 朝廷主流共識：「以幣帛買和平」，認為花錢比打仗划算。
- 蘇洵眼中：這根本是「**抱薪救火**」—— 今日割五城，明日割十城，終有一日無城可割。

## 關鍵人物

### 一、作者 — 蘇洵（表面說書人，實為諫臣）

> 「蘇洵不是歷史學家，他是披著史論外衣的**戰略參謀**。」

- 時年約 40-50 歲，以布衣之身攜二子進京，文章轟動文壇。
- 歐陽修讀其文，讚曰：「**賈誼、劉向不能過也。**」
- 蘇洵深知：直接批評朝廷求和政策是死罪，所以借戰國故事，拐彎罵人。

### 二、被批判對象 — 六國君主（戰國傀儡）

蘇洵筆下的六國君主，全是「**短視近利、苟且偷安**」的負面教材：

| 國家 | 罪狀 | 史實對照 |
|------|------|----------|
| **韓** | 率先割地賄秦 | 韓桓惠王獻宜陽、南陽於秦 |
| **魏** | 割地求和，自毀長城 | 魏安釐王割地予秦，魏遂衰 |
| **楚** | 割地苟安，終被滅 | 楚懷王受騙入秦，客死異鄉 |
| **燕** | 遠交近攻，袖手旁觀 | 燕王喜坐視韓魏被滅，終自食惡果 |
| **趙** | 兵強而無援，獨木難支 | 長平之戰後趙已無力抗秦 |
| **齊** | 隔岸觀火，不助五國 | 齊王建聽信秦間，不修戰備，五國亡後齊亦亡 |

蘇洵的評語一針見血：

> **「賂秦而力虧，破滅之道也。」**
> （割地賄賂秦國，削弱自己實力，就是滅亡之路。）

> **「不賂者以賂者喪，蓋失強援，不能獨完。」**
> （不割地的國家也因為割地的國家而滅亡——失去了強援，撐不住的。）

### 三、真正目標讀者 — 北宋朝廷（被罵的人）

這是整篇文章最有意思的地方：

```
┌────────────────────────────────────────────┐
│           表面的讀者（戰國故事）              │
│   → 六國君主：你們這些白痴，割地自滅          │
├────────────────────────────────────────────┤
│           真正的讀者（北宋朝堂）              │
│   → 宋仁宗 & 宰相們：你們跟六國一樣白痴！     │
│       遼國 = 秦國                           │
│       歲幣 = 割地                           │
│       求和 = 自殺                           │
└────────────────────────────────────────────┘
```

## 寫作意圖：項莊舞劍，意在沛公

### 表面意圖
分析六國滅亡的原因，探討歷史興衰之理。

### 真實意圖
**「借六國的棺材，裝北宋的屍體。」**

蘇洵的論證邏輯是一套完美的**歷史類比**：

| 戰國 | 北宋 |
|------|------|
| 秦國貪得無厭 | 遼國/西夏得寸進尺 |
| 六國割地賄秦 | 北宋納貢求和 |
| 割地而力虧，終至滅亡 | 歲幣而國弱，終將崩潰 |
| **└→ 結論：以地事秦，猶抱薪救火，薪不盡，火不滅** | **└→ 結論：以幣事遼，猶抱薪救火，薪不盡，火不滅** |

> **「以地事秦，猶抱薪救火，薪不盡，火不滅。」**
> 這句話是全文的**核彈頭**。
> 表面說戰國，實際在罵北宋朝廷：「你們以為給遼國送錢就能換來和平？做夢！對方只會越要越多，直到把你們吃乾抹淨！」

## 戰略教訓（軍師觀點）

蘇洵在文章中其實提出了三條戰略路徑：

```
選擇一：賂秦（割地／納貢）
  └→ 抱薪救火，慢性滅亡 ← 北宋現行政策（錯！）

選擇二：不賂秦，但不團結（各自為政）
  └→ 被各個擊破，加速滅亡 ← 六國實際作為（更錯！）

選擇三：不賂秦，且團結抗秦（合縱抗秦）
  └→ 唯一生路「以天下之大，而從六國破亡之故事，
                    是又在六國下矣！」
    ← 北宋應該走的正確戰略（卻不走！）
```

蘇洵的潛台詞：

> 北宋比六國更大、更強、更有條件打持久戰。
> 如果你們連六國都不如，那是恥辱中的恥辱。

---

## 一句話總結

> **蘇洵借六國割地亡國的歷史教訓，痛罵北宋朝廷「以幣買和」的綏靖政策，指出納貢求和猶如抱薪救火——今日割五城，明日割十城，敵欲未足而己力已竭，最終難逃亡國命運。**

## 延伸線索

- 蘇洵另著有《權書》、《衡論》等政論，〈六國論〉原收錄於《權書》第八篇。
- 同一時期，其子蘇軾亦寫有〈六國論〉，觀點與乃父不同——蘇軾認為六國滅亡在於「養士」政策失效。父子對讀，趣味盎然。
- 蘇轍亦有〈六國論〉，兄弟三人各寫一篇六國論，是文學史上一段佳話。
- 北宋最終的結局，恰恰印證了蘇洵的預言：靖康之恥（1127），金兵南下，北宋滅亡。歷史不給第二次機會。
