# 〈桃花源記〉背景資料

> 作者：陶淵明（約 365–427）｜體裁：虛構遊記／寓言｜收錄：《古文觀止》卷七·六朝文

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **地理座標** | 武陵（今湖南常德一帶），一處「不復得路」的峽谷盆地。精確位置不可考——這是故意的。 |
| **時間座標** | 晉太元中（376–396），東晉孝武帝年號，正是司馬氏政權搖搖欲墜之時。 |
| **歷史源頭** | 居民「自云先世避秦時亂」——秦末（~209 BC）至東晉末（~400 AD），與世隔絕約六百年。 |
| **定位判斷** | 這不是地理上的真實地點，而是政治地理上的**烏托邦投影**。武陵是現實邊界，桃花源是理想與現實之間的一條裂縫。 |

---

## 二、時代壓力——東晉亂世與陶淵明的理想投射

### 東晉末期：史上最難生存的時代之一

- **政治**：門閥把持朝政，司馬氏皇室形同傀儡。桓玄篡位（403）、劉裕崛起（420篡晉建宋），朝局如旋轉門。
- **軍事**：淝水之戰（383）雖勝，但北方依舊是五胡十六國——前秦、後燕、後秦、南燕⋯⋯戰火從未停歇。
- **社會**：賦役沉重，土地兼併嚴重，農民「或斃於饑寒，或死於兵役」。戶口銳減，大批百姓逃入深山（**時人稱「山民」**）。
- **文化氛圍**：玄學清談盛行，士大夫避世談虛，卻少有真正關心民間疾苦者。

> 陶淵明活在一個「有桃花源的需要，卻沒有桃花源的可能」的時代。

### 陶淵明的個人處境

- 出身東晉名將陶侃之後，但家道中落。
- 曾任彭澤令，八十餘日即辭官——「不為五斗米折腰」。
- 歸隱田園，親身經歷饑荒、火災、戰亂。他的隱逸不是風雅遊戲，是**生存選擇**。
- 晚年劉裕篡晉（420），陶淵明從此不書年號，僅以甲子紀年——**他的「避秦」從未停止**。

### 投射：桃花源就是陶淵明的精神祖國

桃花源裡「黃髮垂髫，並怡然自樂」，對比東晉「白骨露於野，千里無雞鳴」。桃花源中「不知有漢，無論魏晉」，正是陶淵明「不知有宋」的自我寫照。

## 三、關鍵人物

### 漁夫——敘事之眼，也是失職的見證者

| 面向 | 分析 |
|------|------|
| 身份 | 武陵人，捕魚為業——一個邊緣人、勞動者，非士大夫。這不是巧合：**陶淵明選擇一個沒有話語權的底層人做見證者。** |
| 行為 | 發現→受邀→款待→告辭→標記→失信→再尋不復得。 |
| 關鍵一筆 | 「處處誌之」卻「遂迷不復得路」——**他做了標記，卻仍然失敗。這不是記憶問題，是命運問題。** |
| 批判意義 | 漁夫代表**現實世界的人——永遠在旁觀理想，卻無法真正進入，也無法帶領別人回去。** |

### 桃花源居民——集體的沉默英雄

| 面向 | 分析 |
|------|------|
| 起源 | 「自云先世避秦時亂」——一個集體逃亡的決定。他們的祖先在秦始皇的暴政下選擇了**消失**。 |
| 社會結構 | 無君王、無賦稅、無兵役。人際關係單純：「設酒殺雞作食」——食材就是他們的貨幣。 |
| 世界觀 | 「不知有漢，無論魏晉」——他們對外界王朝更迭一無所知，也**毫不關心**。這是對歷史進步論的最大諷刺。 |
| 對漁夫的態度 | 熱情而克制——「鹹來問訊」，「各復延至其家」，但最後「不足為外人道也」。**他們不拒絕客人，但拒絕被發現。** |

## 四、寫作意圖——這不是童話，是政治的匕首

### 表面：一篇優美的虛構遊記

語言乾淨、簡練、樸素，像一篇真實的見聞記錄——這也是它騙過無數讀者的原因。

### 深層：六層諷刺

1. **反秦即反東晉**：秦朝暴政導致人民「避秦」，東晉亂世同樣逼民為「山民」。「避秦時亂」四個字，罵的不只是秦始皇，還有所有逼民自絕於世的統治者。

2. **「不知有漢，無論魏晉」是歷史虛無的炸彈**：六百年王朝更迭——楚漢相爭、王莽篡漢、三國鼎立、西晉統一、八王之亂、五胡入華——這一切在桃花源居民眼中等於不存在。**陶淵明對所謂「正統」的蔑視，已經到了連討論都不屑的程度。**

3. **「不足為外人道也」是最溫柔的威脅**：居民對漁夫說這句話時的口吻，像是叮囑，但結合後文——漁夫回報太守、郡下遣人尋訪——這句話成了預言式的警告。桃花源不想被發現，因為被發現就等於被毀滅。

4. **太守介入是對權力的嘲諷**：漁夫回報太守，「太守即遣人隨其往」——官僚系統的運作速度在這裡諷刺地快。但快有什麼用？找不到。**權力想要佔有桃花源，但它不配。**

5. **南陽劉子驥——高尚士也——未果**：連品格高尚的人也找不到。陶淵明把最後一絲希望也掐滅了。**不是品德問題，是這個世界不配擁有桃花源。**

6. **「後遂無問津者」——全篇最殘酷的一句**
   - 最初，漁夫回報，太守遣人——還有人在找。
   - 劉子驥前往——還有人聽說後想去。
   - 然後⋯⋯沒有然後了。**再也沒有人問津了。**
   - 人們忘記了。不是因為找不到，而是因為**不在乎了**。
   - 這比「找不到」更絕望。找不到，至少還在找；無人問津，是連尋找的念頭都消失了。
   - 陶淵明的時代沒有桃花源，他的後人也不會再有——**因為沒有人真正想要一個理想世界。**

### 文體策略：以「遊記」為偽裝

整篇使用第一人稱視角（漁夫），細節具體（「忘路之遠近」→「忽逢桃花林」→「從口入」→「土地平曠，屋舍儼然」），讓讀者以為這是真實記述。直到最後「不復得路」「後遂無問津者」，讀者才恍然大悟——**這從頭到尾都是一個寓言。**

陶淵明一生都在寫——寫歸園田居，寫飲酒，寫五柳先生傳——但只有〈桃花源記〉同時做到了三件事：
- **建構了一個完美的理想國**
- **宣告了它的不可到達**
- **嘲笑了所有試圖到達的人**

---

## 五、一句話總結

**「桃花源從來不是一個地方，而是一個對現實絕望的人寫給未來的遺書——內容是『這世界不配擁有理想』，落款是『後遂無問津者』。」**
