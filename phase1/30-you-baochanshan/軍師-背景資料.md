# 〈遊褒禪山記〉背景資料

> **風格**：軍師推演 · **體裁**：說理遊記 · **編次**：《古文觀止》卷十一·宋文

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **時間** | 北宋·神宗熙寧年間，約 1054–1055 年（一說至和元年，即 1054 年） |
| **作者** | 王安石（1021–1086），字介甫，號半山，臨川人 |
| **官職** | 舒州通判任滿返京途中／或云被貶後所作 |
| **地點** | 安徽含山縣北·褒禪山（今稱褒山或華陽山） |
| **地理意義** | 長江沿岸丘陵地貌，多石灰岩溶洞，正合文中「有穴窈然」之景 |

### 更精確的時局標定

至和元年（1054），王安石三十三歲，任舒州通判期滿，即將返京述職。此時仁宗在位已三十餘年，天下承平日久，但官僚冗濫、財政空虛、邊患叢生——北宋的「積貧積弱」已經全面發作。

王安石本人正處於仕途上升期，尚未進入權力中樞，但其變法思想已在地方施政中不斷試煉。舒州任上他推行青苗法雛形（貸糧於民、秋收償還），成效顯著。這篇文章表面上寫一次失敗的洞穴探險，實質上是他政治哲學的第一次完整書面陳述——比〈上仁宗皇帝言事書〉（1058）早了四年。

## 二、時代壓力：變法前夜的醞釀與掙扎

**北宋中葉三大病灶：**

1. **三冗危機**——冗官、冗兵、冗費。宗室恩蔭、科舉取士逐年膨脹，禁軍達百四十萬而戰鬥力空虛，養兵養官的開銷佔財政收入十分之七八。
2. **邊防困局**——西夏元昊稱帝（1038）後連年交戰，宋軍屢敗。慶曆四年（1044）簽訂「慶曆和議」，歲賜銀絹二十五萬五千，換來名義上的稱臣，實則屈辱。
3. **慶曆新政失敗**——范仲淹在仁宗支持下推行「慶曆新政」（1043），不到兩年即因保守派反撲而廢止。改革派的信譽與動能遭到重挫，朝中瀰漫「多一事不如少一事」的姑息之氣。

**王安石在此時的心境：**

他親眼見證范仲淹的失敗，深知改革之難不在於「方案對不對」，而在於「人能不能堅持」。褒禪山之遊中，那個舉火把入洞、卻半途折返的同行者，正是他對改革同僚最深刻的隱喻——你們明明有火把（資源）、有志願（初心），為什麼不再多走幾步？差的就是那一點「盡吾志也而不能至者，可以無悔矣」的決絕。

> 這是王安石對自己、也是對未來所有變法志士的告誡：**不是做不到，是走不到底。**

## 三、關鍵人物

### 核心人物：王安石（自述者）

- 三十三歲，思想已經成熟，行事風格日趨峻厲
- 舒州任上的施政經驗讓他堅信「變法可救天下」，但同時也讓他看清人性的弱點——支持改革的人，往往在壓力面前第一個退縮
- 文中以「予」自稱，語氣冷靜、理性，帶有克制的不甘

### 同行四人（同遊者）

| 人物 | 身份 | 在文中的角色 | 隱喻意義 |
|------|------|-------------|----------|
| **蕭君圭（蕭珪）** | 王安石友人，時任縣尉 | 不主動退出，但也不堅持前進 | 中立的旁觀者，改革中的沉默多數 |
| **王回（深父）** | 長樂人，學者型人物 | 僅記名，未描述其行為 | 象徵「志」的擁有者——但志有餘而力不足？ |
| **王安國（平父）** | 王安石之弟 | 同行，未表態 | 血親，代表「跟自己最親近的人也未必能走到最後」 |
| **某僧**（名字未記） | 褒禪山僧人，當地導遊 | 舉火把者、帶路者，也是**提議退出者** | **這是全文的關鍵人物**——他掌握火把（物），但率先動搖。正是因為他退，整團人跟著退。 |

### 軍師視角的關鍵洞察：那個退縮的僧人

很多人讀此文只注意「志、力、物」的哲學三段論，但忽略了一個細節——

> 既其出，則或咎其欲出者，而予亦悔其隨之，而不得極夫遊之樂也。

「欲出者」就是那個僧人。他在隊伍中處於關鍵位置——拿火把（資源）的人就是話事人。他一說「火把要滅了」，其他人即便心有不甘，也只能跟隨。這和變法完全一樣：**掌握資源的人一旦失去信心，整個進程就會崩潰。**

王安石隱隱點出：慶曆新政失敗，不是因為方案不對，而是因為范仲淹身邊的關鍵人物（或是仁宗本人）先動搖了。

## 四、寫作意圖：以遊山喻改革，以洞穴喻天下

### 表層結構：一次失敗的探險

一群人興致沖沖要去探洞，走了沒多遠，帶路的僧人說「火把不夠了」，於是全軍撤退。出來後大家互相埋怨，但王安石最終把檢討的矛頭指向自己——「悔其隨之」。他不是怪那個僧人，而是怪**自己為什麼不堅持一下、說服大家繼續走**。

### 深層結構：變法的條件論

文中「於是予有嘆焉」一段才是正文的核心。三層邏輯：

```
至極之境  ───┬── 志（決心）    ──→ 沒有志願，一步都走不了
             ├── 力（能力、毅力）──→ 有志還要能走，能走還要能撐
             └── 物（火把、資源）──→ 沒資源就是送死，但有資源不等於能到
```

這三者是 **AND** 關係——缺一不可。

- **志** = 改革的意願與方向（王安石不缺這個）
- **力** = 執行力、堅持力、面對打壓不退縮的韌性（他懷疑同僚缺這個）
- **物** = 皇帝的信任、執政的職權、朝廷的資源（當時他還沒有，但很快就會有了）

### 隱喻的具體對照

| 遊山元素 | 變法對應 |
|----------|----------|
| 褒禪山深穴 | 北宋積弊之深、改革之險 |
| 火把 | 皇帝的授權、改革的資源（預算、人才） |
| 「火且盡」的判斷 | 資源有限論——保守派喊的「國庫空虛、不可妄動」 |
| 半途退出 | 改革半途而廢——范仲淹的教訓 |
| 「悔其隨之」 | 王安石對自己的自我要求：下次絕不妥協 |
| 「盡吾志也而不能至者，可以無悔矣」 | **變法的核心綱領——不是成功與否的問題，而是有沒有走到最後一刻的問題** |

---

## 五、一句話總結

> **〈遊褒禪山記〉不是一篇遊記，而是一篇改革宣言——王安石用一次失敗的探洞，建構了變法的條件哲學：志（方向）、力（執行）、物（資源），三者並舉，缺一不可，而最致命的是「中途退縮的帶路人」——他的變法預言在十五年後全面應驗，也因為同樣的原因崩於中途。**

---

## 六、附論：此文的歷史回響

王安石後來拜相（1069），推行熙寧變法，正如他早在褒禪山中預言的那樣：反對者不是沒有資源，也不是否定變法的方向，而是「在關鍵時刻退了」。

司馬光、呂惠卿、韓琦——這些人在不同階段都當過那個「帶火把的僧人」。有的退得太早（司馬光），有的走到一半反過來搶火把（呂惠卿）。王安石晚年退居金陵，回憶一生，不知是否想起褒禪山那個幽暗的洞穴，和那個說「火且盡」的聲音。

**成也火把，敗也火把。**
**遊山如此，變法亦然。**
