# 五柳先生傳 · 軍師背景分析

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **時代** | 東晉末至南朝宋初期（約 365–427 年） |
| **文體** | 自傳體散文，以第三人稱虛構人物作傳 |
| **典出** | 《陶淵明集》· 卷六 |
| **古文觀止定位** | 卷七·六朝文，與〈歸去來兮辭〉、〈桃花源記〉並稱陶氏三大名篇 |
| **核心命題** | 隱逸人格的自我建構與宣言 |

此文寫作時間眾說紛紜，一般推定在陶淵明**棄彭澤令歸隱之後（約 406 年後）**，屬晚年作品，是其一生志趣的總結性文字。

---

## 二、時代壓力

> **「天下大勢，合久必分，分久必合——然此一分，便是三百年亂世。」**

### 2.1 政治版圖：魏晉南北朝的裂谷

東晉末年，司馬氏政權已名存實亡。桓玄篡位（403–404）、劉裕崛起（420 年代晉建宋），政權更迭如走馬燈。士族門閥把持朝政，寒門無路，所謂「上品無寒門，下品無勢族」。陶淵明身處江州潯陽（今江西九江），地處南北交鋒要衝，戰亂頻仍，百姓流離。

### 2.2 思想戰場：仕 vs 隱 的永恆兩難

魏晉是中國史上「仕隱矛盾」最尖銳的時代：

- **儒家入世**：士大夫以兼濟天下為天職，孔子曰「學而優則仕」。
- **道家出世**：老莊思想復興，玄學清談蔚然成風，嵇康〈與山巨源絕交書〉為「不仕」立下宣言。
- **現實壓力**：劉宋代晉之際，知識分子站在政治十字路口——仕則可能卷入篡位鬥爭，隱則被視為不合作姿態，兩邊都不安全。

陶淵明正是在這種**「仕則危身，隱則貧困」**的夾縫中，為自己尋一條出路。

### 2.3 社會氣候：虛偽的名教與門閥之累

魏晉名士標榜「風度」，但真正實踐者少。陶淵明看不慣官場的逢迎朮——「不為五斗米折腰」一句，道盡東晉官僚體制的繁文縟節與人格屈辱。

## 三、關鍵人物：陶淵明（以自己為樣本）

> **「先生不知何許人也」——這是史上最狡猾、最坦誠、最深情的開頭。**

### 3.1 基本履歷

| 項目 | 內容 |
|------|------|
| **本名** | 陶淵明（一名潛，字元亮），私諡靖節 |
| **生卒** | 約 365–427 年 |
| **籍貫** | 潯陽柴桑（今江西九江） |
| **家世** | 曾祖陶侃為東晉大司馬，祖父陶茂為武昌太守；但父親早逝，家道中落 |
| **仕宦經歷** | 五次出仕：州祭酒 → 桓玄幕僚 → 劉裕鎮軍參軍 → 劉敬宣建威參軍 → 彭澤令（僅八十餘日辭官）|
| **終局** | 隱居田園，晚年貧病交加，拒劉宋徵召，卒年約六十三 |

### 3.2 性格特徵（從文本反推）

1. **嗜酒成癮**：是缺點也是品質。歷來注家多謂淵明「性嗜酒」非好酒本身，而是藉酒避世、以酒養真。酒是其精神自由的最後防線。
2. **好讀書，不求甚解**：讀書不為考據、不為功名，純粹為精神愉悅。此八字實為對漢儒章句之學的無聲反擊。
3. **安貧樂道**：環堵蕭然、短褐穿結、簞瓢屢空——這是真實的貧窮，不是文人雅士的「田園濾鏡」。
4. **「不戚戚於貧賤，不汲汲於富貴」**：全文精神核心，也是中國士人獨立人格的宣言。

### 3.3 為何選自己當主角？

陶淵明不是第一個隱士，但**他是第一個為自己的隱逸生活寫「人物專訪」的人**。他把自己當作一個「案例研究」來分析——這需要極大的自信與自省。

## 四、寫作意圖：為什麼用第三人稱寫自己？

> **「這不是自傳——這是中國文學史上的第一次『人設建構』。」**

### 4.1 戰略分析：第三層偽裝

用第一人稱自傳有兩個天然缺陷：
- **主觀性過強**，讀者天然防備——「你當然說自己好」。
- **缺乏距離感**，無法將自己「客體化」，難以塑型為可被傳誦的理想典範。

陶淵明的天才策略：**以第三人稱為自己立傳。**

這等同於：
- **將自己從「肉身陶淵明」抽離為「典範五柳先生」**——你不是在讀一個人講自己的故事，而是讀一篇「經典傳記」。
- **開頭「先生不知何許人也」**：刻意抹消真實身分，讓文本脫離個人傳記的限制，變成**所有人都可以代入的「隱士原型」**。
- **以史筆寫自我**：傳記是史書體例，通常寫死人或聖賢。陶淵明為自己活著時就用史傳體裁，等於宣告：「我已將此生視為歷史。」這是一種驚人的超越感。

### 4.2 中國最早的人設建構

現代「人設」（persona）一詞看似新潮，實則陶淵明在一千六百年前就熟練操作：

| 層次 | 操作手法 | 現代類比 |
|------|----------|----------|
| **命名** | 自號「五柳先生」，用宅邊五棵柳樹建立視覺記憶點 | 品牌識別 |
| **形象** | 性嗜酒、好讀書、不慕榮利 | 核心人設標籤 |
| **故事** | 環堵蕭然、簞瓢屢空、曾不吝情去留 | 情境塑造 |
| **金句** | 不戚戚於貧賤，不汲汲於富貴 | Slogan/Signature line |
| **權威背書** | 模仿史傳體裁（以「贊曰」作結，仿《史記》太史公曰） | 格式權威 |

這是**自覺的自我品牌建構**，而非偶發的隨筆。

### 4.3 寫作目的總結

1. **為自己辯護**：向世人解釋「為什麼我不做官」——不是因為我無能，而是因為我選擇。
2. **為隱逸正名**：建立一個有尊嚴、有格調、有智識深度的隱士形象，對抗「隱士等於失敗者」的社會偏見。
3. **為自己立碑**：在物質世界赤貧到「簞瓢屢空」的時候，在文字世界為自己建造一座不朽的紀念碑。
4. **設下標準**：讓後世所有想當隱士的人，都得先跟五柳先生比較一下——你有沒有這個覺悟？有沒有這個氣度？

### 4.4 關鍵張力：理想 vs 現實

有趣的是，陶淵明寫的「五柳先生」是否完全等於他自己？答案是否定的。

- **真實的陶淵明**：曾任官五次，晚年因餓肚子寫過〈乞食〉詩，對兒子的愚鈍發過牢騷，對自己「誤落塵網中」感到悔恨。
- **理想的五柳先生**：生來就不知官場為何物，天性自然，純粹得像一棵柳樹。

五柳先生是**陶淵明想成為的那個人**——一個沒有歷史包袱、沒有現實妥協、完完整整從出生就活在自由裡的理想自我。這篇傳記，是陶淵明對自己人生的一次**終極改稿**。

---

## 五、一句話總結

> **〈五柳先生傳〉是中國史上第一次「以第三人稱為自己立傳」的戰略級人設建構——陶淵明用史筆把自己從血肉之軀提煉為「隱逸典範」，在文字的世界裡，完成了一場對貧窮、亂世與遺忘的華麗反殺。**
