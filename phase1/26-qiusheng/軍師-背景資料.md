# 〈秋聲賦〉背景資料

---

## 一、座標定位

| 項目 | 內容 |
|------|------|
| **所處時代** | 北宋·仁宗至神宗時期（約 1050s–1070s） |
| **作者** | 歐陽脩（1007–1072），字永叔，號醉翁、六一居士 |
| **創作階段** | 晚年之作。寫於嘉祐四年（1059）之後，最可能在治平年間（1064–1067）左右 |
| **載體位置** | 《古文觀止》卷十·宋文 |
| **文體** | 賦體抒情散文，以秋聲起興，以議論收束 |

---

## 二、時代壓力——戰場形勢分析

歐陽脩下筆之際，背後是一整片戰場廢墟。

### 2.1 官場風雨——從慶曆新政到全面潰敗

- **慶曆新政（1043–1045）**：歐陽脩與范仲淹聯手推動改革，力圖整頓吏治、精簡官僚。結果是保守派反撲，新政崩盤，范仲淹、韓琦、富弼接連被貶。
- **歐陽脩本人**：因支持新政，被政敵羅織罪名，貶謫滁州（琅琊山上的〈醉翁亭記〉就是這時的產物）。此後雖歷任要職，但銳氣已磨。
- **晚年重回中樞**：官至參知政事（副宰相），位極人臣——但此時的他已不是當年那個熱血改革的年輕人。他看著朝廷依然腐朽，同僚朋黨傾軋，自己卻無力回天。

### 2.2 身體衰老——肉體的崩壞與時間的壓迫

- **身體狀況**：歐陽脩晚年多病，視力衰退、體力大不如前。多次上表請求致仕（退休），朝廷不准。
- **內心狀態**：「修年來衰病，百事俱廢，唯文字不能忘。」——這是他在信中給友人的自白。他清楚感受到時間不多了。
- **心理壓力**：寫〈秋聲賦〉時，歐陽脩約六十歲上下。對古人來說，六十已是暮年。他看著鏡中的自己，再想想當初的豪情壯志，心中只有一個字：**「嘆」**。

### 2.3 時代的集體焦慮

- 北宋中葉，表面上國泰民安，實際上：
  - 冗官、冗兵、冗費——「三冗」問題日益嚴重
  - 外有遼、西夏虎視眈眈
  - 內部士大夫階層瀰漫著「盛極必衰」的末世感
- 歐陽脩是那個時代最有政治敏感度的人之一。他聽到的秋聲，不僅是風聲，更是國運的隱喻。

## 三、關鍵人物——戰場上的角色分佈

### 歐陽脩——感嘆者／主角

| 屬性 | 說明 |
|------|------|
| **身份** | 歷經滄桑的老臣，當代文壇宗主 |
| **心境** | 敏感、通透、沉重。他不是在寫風景，是在寫自己的生命 |
| **行動** | 夜讀聞秋聲，感而賦之 |
| **象徵位置** | 中年以上的知識分子，對時間流逝有切膚之痛 |

### 童子——不懂事的對照組

| 屬性 | 說明 |
|------|------|
| **身份** | 侍奉歐陽脩的少年僕役 |
| **反應** | 「童子莫對，垂頭而睡。」——他根本沒聽到秋天的聲音 |
| **象徵意義** | 這就是年輕人的寫照。沒有經歷過官場起伏、沒有被歲月磨過的人，聽不到秋聲。歐陽脩獨自醒著，童子安穩睡著——**同一間屋子，兩個世界** |
| **戰術功能** | 透過童子的「不懂」，反襯歐陽脩的「懂」。沒有這個對照組，整篇賦的張力會少一半 |

## 四、寫作意圖——這不是單純寫秋天

### 表面文章（童子看到的層次）

寫秋天的聲音——「初淅瀝以蕭颯，忽奔騰而澎湃，如波濤夜驚，風雨驟至。」

### 深層結構（歐陽脩真正在寫的）

> **秋天是隱喻，對象是人生。**

歐陽脩用了一整套戰術鋪排來完成這個隱喻：

#### 第一層：秋天的「四字戰陣」

原文寫道：「蓋夫秋之為狀也，其色慘澹，煙霏雲斂；其容清明，天高日晶；其氣慄冽，砭人肌骨；其意蕭條，山川寂寥。」

- **色**（視覺）→ 慘澹
- **容**（視覺）→ 清明
- **氣**（觸覺）→ 慄冽
- **意**（感受）→ 蕭條

這不是自然描寫，這是**用四季比擬人生階段**——春天是少年，夏天是壯年，秋天是暮年。秋天有其肅殺之理，這是自然的法則，誰也逃不掉。

#### 第二層：草木與人的對比

> 「草木之無情，有時飄零。人為動物，惟物之靈；百憂感其心，萬事勞其形；有動于中，必搖其精。」

歐陽脩說：
- **草木**無情，秋天來了該凋就凋，沒有痛苦
- **人**有情，有憂慮、有勞苦、有感傷——**所以人比草木還慘**

這是全篇的戰術轉折點。前面寫秋天多恐怖，後面全部收束到：**恐怖的不是秋天，是時間本身。**

#### 第三層：童子聽不懂，正是全文的關鍵

童子睡著了。歐陽脩獨自醒著。

這不是一個隨意的細節。這是歐陽脩在說：
> **你們年輕人聽不到。等你們聽得到的時候，你們也老了。**

這是一種極其孤獨的聲音——只有經歷過失去的人，才聽得見秋聲。

### 一句話總結

**歐陽脩的〈秋聲賦〉是北宋暮年老臣在夜半時分，對時間的處決令所做的獨白——童子聽不見秋聲，是因為年輕人還不曾被歲月斬過。**
