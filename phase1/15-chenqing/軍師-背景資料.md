# 〈陳情表〉背景資料

> **軍師評曰**：一篇用「孝」字寫成的政治避險文件，堪稱「拒絕皇帝說明書」之典範。

---

## 一、座標定位

| 維度 | 定位 |
|------|------|
| **時間** | 西晉泰始三年（公元267年）前後 |
| **地理** | 犍為武陽（今四川彭山）→ 洛陽（西晉都城） |
| **政局** | 司馬炎篡魏建晉（265年），蜀漢滅亡（263年）後第四年 |
| **身分** | 李密 — 蜀漢舊臣，曾任蜀尚書郎，多次出使吳國，有辯才 |
| **事件** | 晉武帝下詔徵李密為太子洗馬（東宮侍從官，太子近臣），李密上表辭官 |
| **文體** | 表 — 臣子呈給皇帝的正式奏章，屬公文，直接影響皇帝決策 |

## 二、時代壓力 — 棋盤上的致命局

### 2.1 司馬炎的政治算盤

司馬炎於265年逼迫魏元帝曹奐禪讓，建立西晉。篡位後的最大難題是：**如何讓天下人心服？**

> **戰略分析：**
>
> 蜀漢雖亡（263年），但蜀地士族與舊臣仍心懷故國。若這群人聚眾反抗，西晉的南方防線將立刻告急。司馬炎的策略是 **「徵召—籠絡—收編」** 三部曲：把蜀漢有聲望的舊臣召到洛陽，授予官職，一來展示新朝寬容大度，二來將潛在反對者置於眼皮底下監視，三來瓦解蜀地地方勢力網絡。
>
> 李密以孝聞名，且有外交使節經驗，正是司馬炎心目中理想的 **「樣板人物」**——若李密乖乖來洛陽當官，蜀地其他舊臣便沒有藉口抗拒。

### 2.2 李密的致命困境 — 進退皆死局

李密面對的是典型的中國古代 **「伴君如伴虎」陷阱**：

| 選項 | 表面結果 | 實際風險 |
|------|----------|----------|
| **奉詔赴任** | 加官晉爵，風光上任 | 政治風險極大：① 背負「背主求榮」罵名，被蜀人唾棄；② 入洛陽後身處司馬氏核心，朝中曹魏舊臣與司馬氏新貴鬥爭激烈，隨時可能被捲入政爭；③ 失去在地影響力，成為無根浮萍 |
| **直接拒絕** | 保全名節 | 殺身之禍：新朝初立，最忌有人公開不合作。「辭官=不給皇帝面子=心懷異志=可殺」。司馬炎連曹魏皇帝都敢逼退，殺一個蜀漢降臣何足掛齒？嵇康就是前車之鑑——拒絕合作，被司馬昭處死（262年）。 |
| **上表陳情（本策略）** | 以孝辭官，暫緩赴任 | 這是唯一能在 **「保全性命」** 與 **「不傷皇帝顏面」** 之間取得平衡的方案 |

> **軍師研判：**
>
> 李密的處境＝**高風險棋局中的孤子**。皇帝的一道詔書，對他而言不是恩寵，而是催命符。他需要的不是做官，而是 **「如何安全地說『不』」**。〈陳情表〉正是在這種刀尖上寫成的。

### 2.3 時間壓力 — 詔書的步步進逼

根據表中線索，李密感受到的壓力是遞增的：

1. **「逮奉聖朝，沐浴清化」** — 第一次詔書，用詞客氣，李密按兵不動
2. **「尋蒙國恩，除臣洗馬」** — 第二次，直接任命官職，壓力升級
3. **「詔書切峻，責臣逋慢」** — 第三次，語氣嚴厲，斥責拖延抗命
4. **「郡縣逼迫，催臣上道；州司臨門，急於星火」** — 第四次，地方官上門押送

> 從「客氣邀請」到「急於星火」，朝廷的耐心正在耗盡。李密必須在 **最後期限之前** 遞交一份讓司馬炎無法拒絕、不忍拒絕的答覆。

## 三、關鍵人物

### 李密（224—287）

- **字**：令伯
- **出身**：蜀漢犍為武陽人，父早亡，母何氏改嫁，由祖母劉氏撫養成人
- **仕履**：蜀漢時任尚書郎，曾出使吳國，以辯才著稱。吳主孫權曾讚：「蜀士若李令伯者，乃真才也。」
- **性格**：至孝（史載「祖母有疾，涕泣側息，不解衣帶」）、謹慎、深謀遠慮
- **困境**：蜀漢亡國時四十一歲，正值壯年卻無處施展；面對晉廷徵召，既不能背棄養育之恩，也不敢貿然踏入政治漩渦

### 祖母劉氏

- **關係**：李密祖父之妾，李密父親死後，劉氏獨力撫養孫兒
- **年齡**：表中自述「祖母劉愍臣孤弱，躬親撫養……祖母今年九十有六」
- **角色**：全篇的道德槓桿——李密所有拒官的理由，最終都繫於祖母一人身上
- **軍師點評**：祖母是李密手中 **最強的一張牌**。晉朝以「孝治天下」（司馬氏自己就以孝為名奪權），李密以孝辭官，等於用皇帝自己的盾牌擋皇帝自己的矛。「孝」在當時具有無可反駁的道德正當性。沒有祖母，就沒有〈陳情表〉。

### 晉武帝司馬炎（236—290）

- **身分**：西晉開國皇帝，司馬昭之子
- **性格**：史載其「寬厚」而「沉毅」，但篡位初期極度敏感
- **政治需求**：急需收買人心，尤其是蜀吳兩地士族
- **對李密的態度**：既有政治算計（利用李密當樣板），也有真實賞識（李密確有才幹）
- **為何准奏**：① 李密給了完美台階——不是不忠，而是至孝，符合朝廷提倡的價值；② 強迫孝子做官反而會損害名聲；③ 李密在祖母死後仍出仕，證明最終仍臣服晉朝，達成了籠絡的目的

> **軍師評點：**
>
> 司馬炎讀完此表的反應，史書記載四個字——**「帝覽之曰：『士之有名，不虛然哉！』」**。這句話的潛台詞是：**「李密這人果然名不虛傳——這份報告寫得滴水不漏，朕想殺你都找不到理由。」**

## 四、寫作意圖 — 以孝為盾的政治求生術

### 4.1 表面文本 vs. 深層策略

| 層次 | 內容 | 策略功能 |
|------|------|----------|
| **表面** | 陳述身世孤苦，祖母年邁，需侍奉終老，無法赴任 | 情感攻勢，引發同情 |
| **中層** | 強調「聖朝以孝治天下」，自己的行為完全符合朝廷最高價值 | 道德防禦，建立正當性 |
| **深層** | 表明「臣不勝犬馬怖懼之情」，姿態極度卑微，絕無反抗之意 | 政治表態，消除猜疑 |
| **潛臺詞** | 「我不是不聽你的話，我是太聽你的話——聽你提倡的『孝』。」 | 邏輯閉環，無從反駁 |

### 4.2 四大修辭戰略

**戰略一：先弱後強，博取同情**
開篇不談政治，而是從「臣以險釁，夙遭閔凶」開始，細述自己孤苦無依的童年。「零丁孤苦，至於成立」——八個字勾勒出一個無父無母、與祖母相依為命的悲慘畫面。皇帝也是人，讀到這裡，殺氣已消三分。

**戰略二：以子之矛，攻子之盾**
「伏惟聖朝以孝治天下」是全篇的關鍵一擊。司馬氏篡魏，最怕被人指責「不忠」，於是大力提倡「孝」——因為忠是對君主的，孝是對家族的；你如果批評司馬氏不忠，他可以用孝來反擊。李密正是利用這一點：既然朝廷以孝為本，我為祖母盡孝，正是朝廷價值的最高體現。

**戰略三：先捧後請，不留把柄**
李密在婉拒之前，先用大量筆墨感謝皇恩：「尋蒙國恩，除臣洗馬……非臣隕首所能上報。」姿態低到塵埃裡。這讓皇帝即使生氣，也無法說他傲慢、無禮、不識抬舉。

**戰略四：開空頭支票，預留後路**
「臣密今年四十有四，祖母劉今年九十有六，是臣盡節於陛下之日長，報養劉之日短也。」——這是一句極高明的 **遠期承諾**：祖母總會去世，到那時我一定來為陛下效力。這句話等於告訴司馬炎：「我不是永遠拒絕你，只是現在不行。」既給了皇帝面子，也為自己留下了後路（祖母死後，李密確實出仕晉朝）。

### 4.3 這為什麼是「如何安全地拒絕皇帝」的教科書

| 不安全的拒絕方式 | 安全的拒絕方式（李密的策略） |
|------------------|------------------------------|
| 「我不去，我不屑與司馬氏為伍」 | 「我不是不去，我是有苦衷」 |
| 沉默、拖延、不理詔書 | 主動上表，態度誠懇卑微 |
| 批評朝政或表達不滿 | 讚美朝廷，感謝皇恩 |
| 以「忠於蜀漢」為理由 | 以「孝於祖母」為理由 |
| 不給任何承諾 | 給出明確的後續安排（祖母終後出仕） |

> **軍師總結戰術：**
>
> 這是一篇用 **眼淚包裝的邏輯論證**。每一個讓你感動的句子，背後都是一道讓你無法反駁的邏輯牆。李密沒有跟皇帝硬碰硬，而是把皇帝拉到同一套價值體系中，然後用這套體系的最高標準來保護自己。

## 五、一句話總結

> **〈陳情表〉是一篇以「孝」為外衣、以「保命」為核心的政治避險文件——李密用眼淚澆熄了皇帝的怒火，用邏輯堵住了皇帝的刀，完美示範了「如何在權力面前優雅地說『不』」。**

## 六、延伸觀察 — 歷史的後續

- 祖母劉氏去世後，李密服喪期滿，**依約出仕西晉**，先後任太子洗馬、溫縣令、漢中太守，證明他當年的承諾不是空話
- 晚年李密在官場仍不得志，因在漢中太守任上作詩「人亦有言，有因有緣；官無中人，不如歸田」，觸怒晉武帝被免官
- 〈陳情表〉後來與諸葛亮〈出師表〉並稱，有 **「讀諸葛亮〈出師表〉不墮淚者不忠，讀李密〈陳情表〉不墮淚者不孝」** 的經典評價——然而李密若地下有知，恐怕只會苦笑：這篇表當初是拿來 **保命** 的，不是拿來賺眼淚的
